# 蒙古语族与东胡

## 概括

以东胡、乌桓、鲜卑、柔然、契丹、室韦、蒙古等为主线，主要活动于辽西、东北西部、大兴安岭、蒙古高原和漠北。这里采用“蒙古语族 / 旁蒙古语相关线索”的谨慎表述。

## 起源

东胡是战国秦汉文献中的东北草原族群称谓，乌桓、鲜卑常被视为其后续分支。契丹、室韦、蒙古等与蒙古语族或旁蒙古语关系密切，但具体族群形成包含多部族整合。

## 变迁

这一大类的演进通常不是单线血缘继承，而是多部族联盟、迁徙、征服、内附、语言转用和文化融合的结果。
