# 华夏周边民族

本目录整理华夏周边民族的起源、变迁和演进。这里的分类采用“历史族群 + 语言谱系 + 活动区域”的综合办法，用来组织笔记，不表示纯粹生物血缘；古代族名也常常是外部称谓、政权名称或部落联盟名称。

本 README 只承担总览和入口功能：先看大类与地区索引，再进入各大类 README、二级目录 README 和具体民族笔记查看详细演进图、世系、起源与变迁。

## 起源

- **突厥语族与北方草原**：以蒙古高原、漠北、阿尔泰、叶尼塞和中亚草原为主要空间。匈奴是多族群草原帝国，语言归属未定；丁零 / 铁勒、突厥、回鹘、葛逻禄、黠戛斯以后，突厥语草原和西域线索才更清晰。
- **蒙古语族与东胡**：以东胡、乌桓、鲜卑、柔然、契丹、室韦、蒙古为主线。它更适合看成东北西部、辽西、兴安岭和蒙古高原之间不断重组的政治和语言共同体，而不是单一祖先直系延续。
- **通古斯语族与肃慎**：以肃慎、挹娄、勿吉、靺鞨、女真、满洲、满族为长时段线索。各阶段存在连续性，也存在部落重组、政权整合和八旗制度带来的新共同体形成。
- **东北濊貊与朝鲜**：以辽东、鸭绿江、松花江和朝鲜半岛北部为核心，包含濊貊、古朝鲜、扶余、高句丽传统及渤海国。该线索长期与通古斯、蒙古、汉地和半岛人群交错。
- **西戎羌氐与青藏**：戎、羌、氐是中原文献中的宽泛称谓，覆盖关陇、河湟、青海、川西北和高原边缘多种人群。吐蕃则从青藏高原内部兴起，重塑唐以后西部边疆格局。
- **西域绿洲与印欧**：月氏、大月氏、乌孙、楼兰代表河西、塔里木、伊犁和中亚之间的早期迁徙与绿洲国家。唐以后突厥化、伊斯兰化和蒙古时代重组，使早期印欧线索大多融入后世西域民族格局。
- **南方百越百濮苗瑶**：三苗、百濮、百越都是南方古代多群体泛称，涉及江汉、洞庭、云贵、岭南和东南沿海。它们不能直接等同任何单一现代民族，而是与汉族南方支系、壮侗、苗瑶、白、彝、黎、畲等多线相关。

## 变迁

### 按时期

**传说至夏商周**：华夏共同体逐步形成，周边多以方位性、泛称性名称出现。西部有戎、羌，南方有三苗、百濮、百越前身，东北出现肃慎、濊貊等称谓。

**春秋战国**：秦、赵、燕、楚等诸侯国向边地扩张，义渠、林胡、楼烦、东胡、百越等进入更清楚的政治叙事。东胡受匈奴压力后分化出乌桓、鲜卑等后续线索。

**秦汉**：匈奴建立跨区域草原帝国，月氏西迁为大月氏并进入中亚，乌孙、楼兰等西域国家在汉与匈奴之间变动。汉灭卫氏朝鲜后设置郡县，扶余、高句丽等东北线索逐渐清晰。

**魏晋十六国南北朝**：匈奴、羯、氐、羌、鲜卑诸部大量内迁、建政或融入华北；拓跋、慕容、宇文等鲜卑集团影响北方制度重组。柔然控制漠北，勿吉、靺鞨等东北山林集团名称增强。

**隋唐**：突厥、薛延陀、回纥、葛逻禄、黠戛斯相继主导草原和西域；回鹘西迁开启河西、吐鲁番和中亚绿洲的新线索。吐蕃、南诏、渤海国分别重塑青藏、西南和东北格局。

**辽宋金西夏**：契丹建辽，整合东北西部和草原边缘人群；女真建金，灭辽并进入华北；大理承接南诏传统。室韦、蒙古等部在草原东部逐渐走向蒙古统一前夜。

**元明清**：蒙古统一草原并建立蒙古帝国和元朝，女真诸部在明清之际被建州女真整合为满洲、满族。西域继续经历突厥化、伊斯兰化和蒙古影响，南方与西南则在郡县、土司和移民过程中形成多民族并存格局。

**近现代**：民族识别把长期流动的区域共同体固定为现代民族身份，但古代族名与现代民族不能简单一一对应。蒙古族、满族、维吾尔族、吉尔吉斯族、朝鲜族、藏族、羌族、白族、壮族、苗族、瑶族等都包含多源融合和近现代制度重塑。

### 连续线索

- **草原突厥线索**：匈奴、丁零 / 铁勒是重要背景，突厥、回鹘、葛逻禄、黠戛斯以后形成更明确的突厥语草原和西域线索，后续关联维吾尔、裕固、吉尔吉斯等。
- **东胡蒙古线索**：东胡分化为乌桓、鲜卑，鲜卑之后出现柔然、契丹、室韦等相关线索，蒙古统一前又整合塔塔尔、蔑乞儿、乃蛮、克烈、汪古等诸部。
- **肃慎通古斯线索**：肃慎、挹娄、勿吉、靺鞨、女真、满洲、满族构成东北山林和江河流域的长时段演进，但每一阶段都有新的政治整合。
- **濊貊朝鲜线索**：濊貊、古朝鲜、扶余、高句丽、百济叙事和渤海国之间存在交汇，近现代朝鲜族则主要来自朝鲜半岛移民和东北定居人群。
- **羌氐青藏线索**：戎、羌、氐经历周秦边地竞争、汉魏内迁和十六国建政；吐蕃从高原内部兴起，后来与藏族、羌族、党项、西北多民族格局相连。
- **西域绿洲线索**：月氏西迁、大月氏建贵霜，乌孙据伊犁，楼兰代表塔里木绿洲国家；唐以后回鹘、葛逻禄和中亚伊斯兰化力量重塑绿洲社会。
- **南方古族群线索**：三苗、百濮、百越通过楚、秦、汉以来的郡县化、移民和地方政权重组，分别与南方汉族支系、壮侗、苗瑶、白、彝等多线相连。
