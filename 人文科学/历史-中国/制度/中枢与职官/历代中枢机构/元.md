# 元代中枢机构

元代中央以中书省、枢密院、御史台为主要机构。

| 机构 | 职掌 |
| --- | --- |
| 中书省 | 掌全国政务，并总领吏、户、礼、兵、刑、工六部。 |
| 枢密院 | 掌军事，设知枢密院事、同知枢密院事、枢密副使等官；具体权力随时期和任官变化。 |
| 御史台 | 掌监察。 |

门下省不复置；尚书省曾短期设置，后废。
