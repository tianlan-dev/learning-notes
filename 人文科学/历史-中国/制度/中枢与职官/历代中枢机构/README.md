# 历代中枢机构总览

本目录按朝代整理中央权力结构、官署和职掌。地方区划另见 [历代地方区划](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2-%E4%B8%AD%E5%9B%BD/%E5%88%B6%E5%BA%A6/%E5%9C%B0%E6%96%B9%E8%A1%8C%E6%94%BF%E5%8C%BA%E5%88%92/%E5%8E%86%E4%BB%A3%E5%9C%B0%E6%96%B9%E5%8C%BA%E5%88%92/README.md)。

## 朝代顺序

| 顺序 | 朝代 / 时期 | 中枢机构概括 |
| --- | --- | --- |
| 1 | 先秦 | 夏世袭、商内外服、西周分封宗法，春秋战国官僚制和郡县制萌芽。 |
| 2 | 秦 | 皇帝制度、三公九卿。 |
| 3 | 西汉 | 三公九卿为外朝，汉武帝后中朝参与决策。 |
| 4 | 东汉 | 三公仍存，尚书台权力增强。 |
| 5 | 两晋 | 尚书、中书、门下三省形成，八公多为尊官。 |
| 6 | 隋唐 | 三省六部制成熟。 |
| 7 | 宋 | 二府三司分割政务、军政、财政。 |
| 8 | 元 | 中书省、枢密院、御史台并立。 |
| 9 | 明 | 废中书省和丞相，六部直属皇帝，内阁兴起。 |
| 10 | 清 | 内阁六部仍存，军机处处理机要军政。 |
