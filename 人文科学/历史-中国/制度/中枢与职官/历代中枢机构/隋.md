# 隋代中枢机构

隋朝中枢以三省为核心：内史省起草诏令，门下省审查政令，尚书省执行各项政务。

## 中央机构

| 机构 | 职掌 / 说明 |
| --- | --- |
| 三师三公 | 三师不主事，不置府僚，盖与天子坐而论道；三公参议国之大事。三公不常置，无其人则缺。 |
| 殿内省 | 隋炀帝废三师设置。 |
| 内史省 | 起草诏令。 |
| 门下省 | 审查政令；对内史省可审议、封驳。 |
| 尚书省 | 执行各项政务。 |
| 秘书省 | 图书籍藏整理。 |
| 内侍省 | 内廷侍奉。 |
| 御史台、谒者台、司隶台 | 监察、礼仪或巡察相关机构。 |

## 下设机构

| 上级 | 下设机构 |
| --- | --- |
| 门下省 | 城门、尚食、尚药、符玺、御府、殿内。 |
| 尚书省 | 吏部、礼部、兵部、工部、都官（刑部）、度支（户部）。 |
| 九寺 | 太常、光禄、卫尉、宗正、太仆、大理、鸿胪、司农、少府。 |
| 五监 | 国子、将作、都水、长秋、少府。 |

## 制度特点

- 隋代三省职责逐步明确，内史省起草、门下省审查、尚书省执行，为唐代三省六部制成熟奠定基础。
- 三省相互牵制，有助于削弱相权、加强皇权，并使中央政务分工更清晰。
- 科举制设分科考试，逐渐取代九品中正制，使选官不再完全依赖门第。
