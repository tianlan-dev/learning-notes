# 先秦中枢与政治结构

先秦政治制度经历夏代世袭、商代内外服、西周分封宗法，到春秋战国君主专制和官僚制增强的过程。

| 时期 | 制度 | 内容 |
| --- | --- | --- |
| 夏 | 世袭制 | 约公元前 2070 年夏建立，王位世袭取代禅让，形成“家天下”格局。世袭制内容包括父子相传、兄终弟及。 |
| 商 | 内外服制度 | 商王直接控制王畿内服，四周方国和部族为外服，由附属国管辖；已有较系统国家机构和分掌事务的官吏。 |
| 西周 | 分封制、宗法制 | 周天子分封同姓宗族、功臣和先代贵族；诸侯再分封卿大夫，卿大夫再分封士。政治权力与血缘关系结合。宗法制中周天子和各级贵族多实行嫡长子继承制。 |
| 春秋战国 | 君主专制加强 | 西周贵族等级分封制开始解体，郡县制、官僚制等封建政治体制开始产生。 |
