# 东汉中枢机构

东汉沿用西汉官制，以三公九卿为基本架构，但三公权力被削弱。尚书台逐渐成为实际行政中枢，尚书台和六曹机构权力增强，为后来的三省六部制奠定基础。

## 中央机构

| 机构 / 官职 | 内容 |
| --- | --- |
| 太傅 | 事归台阁，地位尊崇。 |
| 三公 | 太尉、大司徒、大司空。多保留高位名义和礼仪地位，权力大大削弱。 |
| 御史台 | 独立监察机构。 |
| 尚书台 | 实际行政中枢，名义上隶属少府，但逐渐掌握主要行政事务。 |
| 尚书六曹 | 吏曹、三公曹、二千石曹、民曹、南主客曹、北主客曹。 |
| 九卿 | 太常、光禄勋、卫尉、太仆、廷尉、大鸿胪、宗正、大司农、少府。 |

光武帝刘秀为加强集权，提升尚书台职权，削弱“三公”府的大部分权力，使尚书台成为实际中枢决策机构；但东汉时期尚书台名义上仍是少府寺的分支机构，尚未完全独立。
