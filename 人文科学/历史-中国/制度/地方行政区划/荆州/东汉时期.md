# 荆州：东汉时期

东汉汉灵帝中平五年（188 年）后，州成为正式的一级行政区域。据《后汉书》相关记载，荆州辖有南阳、南郡、江夏、长沙、桂阳、武陵、零陵等郡。

## 所辖郡

- **南阳**：河南西南。
- **南郡**：湖北西部。
- **江夏**：湖北东部。
- **长沙**：湖南东北。
- **桂阳**：湖南东南。
- **武陵**：湖南西北。
- **零陵**：湖南西南。

## 赤壁之后的分割

汉献帝建安十三年（208 年）赤壁之战后，短暂领有荆州的曹操主要保有北部荆州，并从**南阳郡**中分出**襄阳郡**、**南乡郡**。

孙权和刘备经过征战与协商后，荆州形成三分割据局面：

| 势力 | 控制区域 |
| --- | --- |
| 刘备 | **南郡、零陵、武陵**。 |
| 孙权 | **江夏、桂阳、长沙**。 |
| 曹操 | **南阳、襄阳、南乡**。 |

一般认为这阶段三家各占三郡的局面，是“荆襄九郡”一词的来源之一；但“荆襄九郡”是后世概括，具体所指随语境变化，不宜当作严格行政单位。

建安二十四年（219 年），荆州牧刘备的守将关羽被孙权、曹操联军击败。夷陵之战（221 年至 222 年）刘备又败于孙权势力。此后蜀汉再无力夺回原有荆州辖权，荆州成为曹魏与孙吴分领的局面，分治**南阳郡新野县**（今河南省新野县）与**南郡江陵县**（今湖北省荆州市江陵区）。
