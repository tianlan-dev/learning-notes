# 先秦地方结构

先秦地方结构从西周分封体系逐渐转向春秋战国的郡县制和官僚制。

| 时期 | 地方结构 |
| --- | --- |
| 商 | 内外服：王畿为内服，四周附属方国为外服。 |
| 西周 | 周天子分封诸侯，诸侯在国内分封卿大夫，卿大夫分封士；政治权力和血缘关系结合。 |
| 春秋战国 | 西周贵族等级分封制开始解体，郡县制、官僚制等封建政治体制开始产生。 |
