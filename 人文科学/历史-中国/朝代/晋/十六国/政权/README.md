## 概括

本目录收录十六国及若干同时期重要政权的单独笔记。每个政权页主要承担“政权概括 + 具体史事 + 君主世系”的功能；总览页和阶段页只保留导览、分期和横向比较，避免在多处重复维护同一批细节。
