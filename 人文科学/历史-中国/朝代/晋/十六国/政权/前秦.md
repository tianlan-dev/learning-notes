# 前秦

> 导航：[晋](../../README.md) / [十六国](../README.md) / [政权索引](README.md) / [淝水之战前](../%E6%B7%9D%E6%B0%B4%E4%B9%8B%E6%88%98%E5%89%8D.md) / [淝水之战后](../%E6%B7%9D%E6%B0%B4%E4%B9%8B%E6%88%98%E5%90%8E.md)


## 时间

351年—394年。

## 别称

- 苻秦

## 概括

前秦由氐族苻氏建立，苻坚时期灭前燕、前凉、代，基本统一北方。383年淝水之战败于东晋后瓦解，394年灭亡。

## 说明

- 350年，氐族苻洪占据关中，称三秦王。
- 352年，苻健称帝，定都长安，国号“秦”。
- 370年，前秦灭前燕。
- 373年，前秦攻取东晋梁、益二州。
- 376年，前秦灭前凉、代，基本统一北方。
- 382年，苻坚命吕光驻西域，前秦势力达到高峰。
- 383年，苻坚南征东晋，淝水之战失败，前秦瓦解。
- 394年，苻登被后秦姚兴擒杀；苻崇继位后攻西秦，被乞伏轲弹斩杀，前秦灭亡。
