# 翟魏

> 导航：[晋](../../README.md) / [十六国](../README.md) / [政权索引](README.md) / [淝水之战前](../%E6%B7%9D%E6%B0%B4%E4%B9%8B%E6%88%98%E5%89%8D.md) / [淝水之战后](../%E6%B7%9D%E6%B0%B4%E4%B9%8B%E6%88%98%E5%90%8E.md)


## 时间

388年—392年。

## 别称

- 丁零魏

## 概括

翟魏由丁零首领翟辽建立，位于黄河下游与滑台一带，周旋于东晋、后燕、西燕之间。它通常不列入传统十六国，但在淝水之战后北方格局中占有一席。

## 说明

- 383年，丁零人翟斌在新安起兵反抗前秦。
- 388年，翟辽自称魏天王，建立魏国，史称翟魏。
- 翟魏以黄河为屏障，周旋于东晋、后燕、西燕之间，并经常进攻后燕南部边疆。
- 392年，翟魏被后燕所灭。
