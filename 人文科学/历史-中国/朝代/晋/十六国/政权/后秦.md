# 后秦

> 导航：[晋](../../README.md) / [十六国](../README.md) / [政权索引](README.md) / [淝水之战前](../%E6%B7%9D%E6%B0%B4%E4%B9%8B%E6%88%98%E5%89%8D.md) / [淝水之战后](../%E6%B7%9D%E6%B0%B4%E4%B9%8B%E6%88%98%E5%90%8E.md)


## 时间

384年—417年。

## 别称

- 姚秦

## 概括

后秦由羌族姚氏建立，前秦瓦解后占据关中。姚兴时势力较盛，417年被东晋刘裕灭。

## 说明

- 384年，原降于前秦的羌族贵族姚苌在渭北叛秦。
- 385年，姚苌擒杀苻坚。
- 386年，姚苌在长安称帝，国号“秦”，史称后秦。
- 394年，姚兴杀前秦苻登。
- 399年，后秦乘东晋内乱攻陷洛阳。
- 416年，姚兴去世，姚泓继位，后秦内讧。
- 417年，东晋刘裕攻破长安，姚泓投降，后秦灭亡。
