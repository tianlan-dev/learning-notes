## 概括

这张时空图以时间为纵轴、地理区域为横轴，展示西晋末年至北魏统一华北之间北方诸政权的兴亡。图中主体是“十六国”，同时也列入了曹魏、西晋、东晋、南朝宋、北魏、代国、冉魏、西燕、翟魏、仇池、高句丽等相关政权，便于理解整个北方与南方的并行格局。

十六国可用口诀概括为：**两赵三秦，四燕五凉，成汉胡夏**。

| 类别 | 政权 |
|---|---|
| 两赵 | 汉赵 / 前赵、后赵 |
| 三秦 | 前秦、后秦、西秦 |
| 四燕 | 前燕、后燕、南燕、北燕 |
| 五凉 | 前凉、后凉、南凉、西凉、北凉 |
| 其他 | 成汉、胡夏 |

## 阅读方法

| 维度 | 读法 |
|---|---|
| 时间轴 | 纵向从上到下约为公元250年—450年，可观察政权持续时间与前后继承。 |
| 地理轴 | 横向从西北、关中、河套、山西、河南、河北、辽东、山东、淮北到长江流域和四川。 |
| 色块 | 表示政权控制区域在时间与空间上的大致范围。 |
| 竖线 | 常表示都城或重要政治中心，例如长安、洛阳、邺、建康、成都。 |
| 事件点 | 蓝点或标注常表示战役、迁都、灭国、政变等关键事件。 |
| 非十六国政权 | 图中也保留冉魏、代、西燕、翟魏、仇池等势力，以便展示北方实际格局。 |

## 时间分期

| 阶段 | 时间 | 主线 | 代表政权 |
|---|---|---|---|
| 西晋统一与边地割据萌芽 | 280年前后—303年 | 西晋统一后，北方和西南边地矛盾积累，巴蜀、河西、并州等地逐渐出现割据基础。 | 西晋、前凉前身、成汉前身 |
| 西晋崩溃 | 304年—316年 | 刘渊建汉赵，李雄建成汉；八王之乱后西晋中央崩溃，洛阳、长安相继陷落。 | 汉赵 / 前赵、成汉、西晋 |
| 后赵与前燕竞争 | 319年—351年 | 后赵灭前赵并控制北方大部，随后内乱；冉魏短暂出现，前燕入主中原。 | 后赵、冉魏、前燕、前凉、成汉 |
| 前秦统一北方 | 351年—383年 | 氐族苻氏据关中兴起，苻坚时灭前燕、前凉、代等政权，基本统一北方。 | 前秦、前凉、前燕、代 |
| 淝水之战后再分裂 | 383年—420年 | 前秦败于东晋后瓦解，关中、河北、辽东、河西、山东、陇西同时出现多个政权。 | 后秦、后燕、西秦、后凉、南凉、西凉、北凉、南燕、北燕、胡夏 |
| 北魏统一华北 | 386年—439年 | 拓跋魏从代国旧部重建北魏，逐步灭后燕、北燕、胡夏、北凉等，439年统一华北。 | 北魏、北燕、胡夏、北凉 |

## 地域格局

| 地域 | 主要政权 | 格局特点 |
|---|---|---|
| 关中 | 前赵、前秦、后秦、胡夏、北魏 | 长安是西北与中原的核心节点，多次成为称帝和统一北方的基地。 |
| 河西走廊 | 前凉、后凉、南凉、西凉、北凉 | 凉州政权密集，既控制丝路通道，也经常在前秦、后秦、北魏压力下分合。 |
| 河套与陕北 | 胡夏、北魏 | 胡夏以统万城为中心，后来被北魏和吐谷浑夹击而亡。 |
| 山西、河北、河南 | 后赵、冉魏、前燕、前秦、后燕、北魏 | 中原北部是争夺最激烈区域，邺、洛阳、中山等地反复易手。 |
| 辽东、河北东北 | 前燕、后燕、北燕、高句丽、北魏 | 慕容鲜卑在辽西、辽东起家，后燕和北燕延续其东北根基。 |
| 山东 | 南燕、东晋、北魏 | 南燕从后燕分出，占据山东一带，410年被东晋刘裕灭。 |
| 淮北与长江以南 | 东晋、南朝宋 | 东晋守住江南并多次北伐，420年后由南朝宋承接。 |
| 四川 | 成汉、东晋、谯蜀 | 成汉据巴蜀，347年被东晋灭；后来四川仍有短暂地方割据。 |

## 关键转折

| 时间 | 事件 | 影响 |
|---|---|---|
| 304年 | 刘渊称汉王；李雄据蜀称王 | 十六国时代开始，西晋周边割据政权正式成形。 |
| 311年 | 永嘉之乱，洛阳陷落 | 西晋中央遭毁灭性打击，北方秩序崩溃。 |
| 316年 | 长安陷落，西晋灭亡 | 晋室南渡，东晋与北方诸国对峙。 |
| 329年 | 后赵灭前赵 | 后赵成为北方强权。 |
| 350年—352年 | 冉魏取代后赵，又被前燕灭 | 后赵旧秩序瓦解，前燕进入中原。 |
| 370年 | 前秦灭前燕 | 前秦开始统一北方。 |
| 376年 | 前秦灭前凉、代 | 苻坚基本统一北方。 |
| 383年 | 淝水之战 | 前秦失败并瓦解，北方重新分裂。 |
| 410年 | 东晋刘裕灭南燕 | 东晋北伐取得阶段成果。 |
| 417年 | 东晋刘裕灭后秦 | 关中短暂落入东晋之手，随后又被胡夏夺取。 |
| 431年 | 胡夏灭西秦，随后胡夏灭亡 | 西北格局进一步收束。 |
| 436年 | 北魏灭北燕 | 东北纳入北魏控制。 |
| 439年 | 北魏灭北凉 | 北魏统一华北，十六国时代结束。 |
