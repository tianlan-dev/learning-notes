## 概括

商朝（约前1600年—约前1046年）是继夏之后的中原王朝，子姓。传统说法认为商经历十七代、三十一王。商汤在鸣条之战中灭夏后建立商朝；盘庚迁殷后，商朝后期也称殷。商代甲骨文和金文的发现，使商成为中国历史上较早拥有同时代文字材料证明的王朝。

## 核心线索

- **从部族到王朝**：契受封于商，后世商族逐步强大，至汤灭夏建商。
- **信史地位**：甲骨文和殷墟考古使商代成为中国早期较可靠的信史阶段。
- **迁都与殷墟**：商前期多次迁都，盘庚迁殷后政治中心相对稳定。
- **王权与神权**：商王通过祭祀、占卜和军事征伐维系统治。
- **商周更替**：帝辛时期社会矛盾和外部压力加剧，周武王牧野之战灭商。
