# 唐

> 导航：[隋](../%E9%9A%8B/README.md) / [唐](README.md) / 五代十国

## 概括

唐朝（618年—907年）上承隋朝、下接五代十国，是中国中古时期最重要的大一统王朝之一。618年李渊受隋恭帝杨侑禅让建唐，定都长安，并以洛阳为东都。唐朝前期完成统一并形成贞观、永徽、开元等治世；中期经历武周代唐与神龙复唐；755年安史之乱后，藩镇、宦官、党争和财政军事结构变化使中央权威持续削弱，907年朱温迫唐哀帝禅让，唐亡。

## 阶段导览

| 顺序 | 阶段 | 时间 | 简要概括 |
|---:|---|---|---|
| 1 | 唐建立与统一战争 | 618年—624年 | 李渊建唐后，唐军陆续平定薛举、刘武周、王世充、窦建德、萧铣、辅公祏等势力。 |
| 2 | 贞观之治 | 626年—649年 | 唐太宗整顿政治、纳谏任贤，对外击败东突厥，唐朝进入稳定扩张期。 |
| 3 | 高宗与武后时期 | 649年—690年 | 高宗前期延续治世，后期武后参政；唐势力扩展至西域、朝鲜半岛等方向。 |
| 4 | 武周 | 690年—705年 | 武则天称帝，改国号周；延续唐制并强化科举与官僚流动，705年神龙政变后复唐。 |
| 5 | 开元盛世 | 713年—741年 | 唐玄宗前期任用姚崇、宋璟、张九龄等，政治经济和文化发展达到高峰。 |
| 6 | [安史之乱](%E5%AE%89%E5%8F%B2%E4%B9%8B%E4%B9%B1.md) | 755年—763年 | 安禄山、史思明叛乱重创唐朝，战后藩镇割据成为长期结构性问题。 |
| 7 | 中晚唐 | 763年—875年 | 中央尝试削藩和财政重建，出现元和中兴、大中暂治，但宦官专权、党争和藩镇问题反复。 |
| 8 | 唐末崩解 | 875年—907年 | 黄巢之乱摧毁唐朝统治基础，朱温控制朝廷并于907年篡唐，五代十国开始。 |

## 核心线索

- **隋制唐承**：唐承接隋代三省六部、科举、均田租庸调、府兵等制度，并在实践中调整完善。
- **开放帝国**：唐朝与草原、西域、朝鲜半岛、日本、南亚、西亚等区域交流频繁，长安成为国际都市。
- **文化高峰**：唐诗、佛教、道教、史学、书法、绘画和音乐舞蹈高度繁荣，形成兼容并蓄的中古文化格局。
- **由府兵到藩镇**：均田与府兵基础瓦解后，募兵和节度使体系增强边镇军权，安史之乱后演变为藩镇割据。
- **灭亡原因**：藩镇割据、宦官掌禁军、牛李党争等内耗，叠加财政转型、边患和黄巢之乱，最终使朱温取得篡唐条件。
