# 南明

## 时间

1644年-1662年。

## 概括

南明是明朝北京失守后，明宗室和遗臣在南方建立的一系列延续政权。1644年，李自成攻入北京，崇祯帝自缢，清军随后入关。南京、福建、广东、广西、云南等地先后拥立明宗室称帝或监国，试图维持明朝法统并抗清。

南明内部存在党争、军阀割据、财政匮乏和战略分歧，未能形成稳定统一的抗清核心。1662年，永历帝朱由榔在缅甸被吴三桂俘获后遇害，南明作为宗室皇帝政权基本结束。郑氏仍以明朝名义据守台湾，至1683年降清。

## 说明

- 弘光政权以福王朱由崧为君，建都南京，1645年清军南下后覆灭。
- 隆武政权以唐王朱聿键为君，活动于福建，1646年覆灭。
- 绍武政权以朱聿𨮁为君，活动于广州，存在时间很短。
- 永历政权以桂王朱由榔为君，转战两广、贵州、云南，并一度依靠李定国等抗清力量。
- 1661年清军攻入云南，永历帝逃入缅甸；1662年被吴三桂杀害。
- 南明与明郑、农民军余部、地方军阀关系复杂，既合作抗清，也长期内耗。

## 统治结构

| 角色 | 说明 |
|---|---|
| 君主 | 明宗室皇帝或监国，承担延续明朝法统的政治象征。 |
| 文臣集团 | 东林、复社和地方士绅等参与拥立、筹饷和政务，但党争严重。 |
| 军事集团 | 江北四镇、郑氏、李定国等军政势力影响政权存亡。 |
| 地方基础 | 南京、福建、两广、云贵等地先后成为抗清核心，但地域割裂明显。 |
