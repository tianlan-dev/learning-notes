# 秦末群雄

## 时间

前209年-前202年

## 概括

秦末群雄指陈胜、吴广起义后，各地反秦力量、六国旧贵族后裔、地方豪杰和秦军将领在秦亡前后形成的复杂政治军事格局。其核心演变是：民变引爆反秦浪潮，六国旧贵族复起，项羽主导灭秦后分封诸侯，刘邦最终在楚汉相争中胜出，建立汉朝。

## 主要阶段

| 阶段 | 时间 | 说明 |
|---|---|---|
| 大泽乡起义 | 前209年 | 陈胜、吴广起义，陈胜称王，建立张楚，反秦浪潮扩散。 |
| 六国旧势力复起 | 前209年-前208年 | 魏、赵、齐、楚、燕、韩等旧贵族或地方豪杰相继拥立王号。 |
| 巨鹿之战 | 前207年 | 项羽率楚军击败秦将章邯、王离主力，秦军主力瓦解。 |
| 刘邦入关 | 前207年 | 刘邦先入咸阳，秦王子婴投降，秦朝灭亡。 |
| 项羽分封 | 前206年 | 项羽自立西楚霸王，分封十八路诸侯，政治秩序重新碎片化。 |
| 楚汉相争 | 前206年-前202年 | 刘邦与项羽争夺天下，最终刘邦胜出，建立汉朝。 |

## 主要势力

| 势力 | 代表人物 | 根据地或活动区域 | 说明 |
|---|---|---|---|
| 张楚 | 陈胜、吴广 | 陈郡一带 | 秦末第一波大规模起义，政治号召力很强，但内部组织薄弱，迅速失败。 |
| 楚系反秦力量 | 项梁、项羽、楚怀王熊心 | 江东、楚地 | 借楚国旧号召力起兵，项羽在巨鹿之战后成为反秦盟主。 |
| 沛县集团 | 刘邦、萧何、曹参、樊哙、周勃等 | 沛县、关中、汉中 | 从地方起兵发展为汉王集团，最终击败项羽。 |
| 秦降将 | 章邯、司马欣、董翳 | 关中及秦军系统 | 章邯降楚后被项羽封为雍王，关中被分为三秦。 |
| 赵地势力 | 赵王歇、张耳、陈馀 | 赵地 | 旧赵区域的重要反秦力量，后在楚汉战争中分裂。 |
| 齐地势力 | 田儋、田荣、田横等 | 齐地 | 齐国旧族复起，长期牵制项羽和刘邦。 |
| 魏、韩、燕等势力 | 魏豹、韩成、韩信、臧荼等 | 原六国区域 | 多数以旧国名义复兴，在楚汉相争中依附或反复易帜。 |

## 关键人物

| 人物 | 身份 | 作用 |
|---|---|---|
| 陈胜 | 张楚王 | 首先发动大规模反秦起义，提出“王侯将相宁有种乎”的政治号召。 |
| 吴广 | 起义领袖 | 与陈胜共同发动大泽乡起义。 |
| 项梁 | 楚系领袖 | 扶立楚怀王，整合楚地反秦力量。 |
| **项羽** | 西楚霸王 | 巨鹿之战击破秦军主力，灭秦后分封诸侯，是楚汉相争中的刘邦主要对手。 |
| **刘邦** | 汉王、汉高祖 | 先入关灭秦，后以汉中、关中为基础击败项羽，建立汉朝。 |
| 章邯 | 秦将、雍王 | 秦末最重要的秦军将领，降楚后被封于关中。 |
| 韩信 | 汉将 | 在楚汉战争中为刘邦开辟北方战场，是汉胜楚的重要军事人物。 |

## 说明

- 秦末起义不是单纯农民战争，也包括六国贵族复国、地方豪杰扩张和秦军系统瓦解。
- 项羽灭秦能力强，但分封秩序难以稳定；刘邦能够整合关中、汉中和各路反楚力量，最终取得天下。
