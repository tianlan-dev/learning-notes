# 秦

## 时间

- 秦国：约前9世纪-前221年
- 秦朝：前221年-前207年

## 概括

秦由西周、春秋时期的西陲诸侯发展为战国强国。秦王嬴政继承秦国数代积累的军政优势，前230年至前221年相继灭韩、赵、魏、楚、燕、齐，完成中国历史上第一次大一统，建立秦朝，自称始皇帝。

秦朝虽然存在时间很短，但在制度史上影响极深：皇帝制度、三公九卿中央官制、郡县制、统一文字、度量衡、货币和交通制度，都成为后世中央集权国家的重要基础。

## 历史阶段

| 阶段 | 时间 | 概括 |
|---|---|---|
| 秦国兴起 | 西周晚期至春秋 | 秦以西陲军事边防起家，逐渐取得诸侯地位。 |
| 变法强国 | 战国中期 | 商鞅变法强化法制、军功爵、县制和国家动员能力。 |
| 统一六国 | 前230年-前221年 | 秦王政先后灭韩、赵、魏、楚、燕、齐，结束战国。 |
| 秦朝统治 | 前221年-前210年 | 秦始皇推行皇帝制度、郡县制和统一政策，北击匈奴，南服百越。 |
| 秦末崩溃 | 前209年-前207年 | 陈胜、吴广起义引发全国性反秦浪潮，六国旧贵族和地方豪杰并起。 |
| 楚汉相争 | 前206年-前202年 | 秦亡后项羽分封诸侯，刘邦与项羽争夺天下，最终建立汉朝。 |

## 制度与政策

| 领域 | 内容 | 影响 |
|---|---|---|
| 皇帝制度 | 秦始皇废除“王”号，自称始皇帝。 | 奠定此后两千多年君主专制国家的最高权力形式。 |
| 中央官制 | 以丞相、太尉、御史大夫为核心，形成三公九卿体制。 | 强化中央对行政、军事、监察和礼制事务的控制。 |
| 地方制度 | 废封建，设郡县，由中央任命官吏治理地方。 | 打破世卿世禄和诸侯割据传统，是后世郡县国家的基础。 |
| 法制与军功 | 延续秦国法家治理和军功爵体系。 | 提高国家动员效率，但严刑峻法也加剧社会压力。 |
| 统一措施 | 统一文字、度量衡、货币、车轨，修驰道。 | 促进大一统国家内部的行政、经济和文化整合。 |
| 边疆经营 | 北击匈奴，修筑长城；南征百越，设置郡县。 | 扩展秦朝控制范围，但也加重徭役和军事负担。 |

## 灭亡原因

- 统一后没有给民众充分休养生息，徭役、兵役、赋税负担沉重。
- 战时形成的严刑峻法没有及时调整为和平治理模式，社会承压过高。
- 大规模工程、边疆战争和中央集权动员同时推进，基层负担失衡。
- 秦二世能力有限，赵高专权，统治集团陷入宫廷斗争。
- 地方武力和治理网络失效，陈胜、吴广起义后各地迅速响应。
- 六国旧贵族后裔复辟力量仍强，反秦起义从民变扩展为旧贵族与地方豪杰的联合反扑。
