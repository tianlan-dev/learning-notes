# 十国

## 概括

十国是五代十国时期较主要的地方割据政权统称，包括前蜀、后蜀、吴、南唐、吴越、闽、楚、南汉、荆南、北汉。它们主要分布在南方、四川、岭南和河东等地，存在时间并不完全重叠。

## 说明

- “十国”是后世归纳的史学概念，并非十个政权同时并立。
- 多数十国政权源于唐末地方节度使或军政集团。
- 南方诸国在战乱中维持区域经济与文化发展，也为宋代南方经济上升保留基础。
