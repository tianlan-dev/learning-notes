# 湘军

## 时间

1852年前后形成，太平天国战争时期达到高峰，1864年攻陷天京后主力逐步裁撤，但其将领和制度影响延续至晚清。

## 概括

湘军是曾国藩以湖南团练为基础组建的地方军队。它以乡土、师生、宗族和私人关系维系组织，重视营官自募、层级负责和士绅筹饷，是晚清地方军事集团兴起的代表。

湘军在镇压太平天国战争中发挥决定性作用，尤其是曾国荃部攻陷天京。湘军的成功推动淮军、楚军等地方军队发展，也使晚清督抚和地方军事首领权力上升。

## 说明

- 曾国藩受命办团练后，以湖南士绅和乡勇为基础扩编成军。
- 湘军水师和陆师配合作战，是攻取长江中下游的重要力量。
- 主要人物包括曾国藩、曾国荃、胡林翼、左宗棠、彭玉麟、杨载福等。
- 湘军镇压太平天国后，曾国藩主动裁撤部分部队，以缓解清廷对地方军权的疑惧。
- 湘军经验影响淮军、新军和近代中国军政关系。

## 组织特点

| 特点 | 说明 |
|---|---|
| 募兵方式 | 营官自募，依靠同乡、宗族、师友等私人关系。 |
| 经费来源 | 地方筹饷、厘金、捐输和督抚财政支持。 |
| 统属关系 | 名义上受清廷调度，实际高度依赖统帅和地方网络。 |
| 历史影响 | 推动晚清地方督抚坐大，并为淮军、北洋系统提供组织经验。 |
