# 准噶尔

## 时间

约17世纪上半叶-1757年。

## 概括

准噶尔是漠西蒙古卫拉特诸部中以准噶尔部为核心形成的草原政权，活动范围主要在天山以北、伊犁河流域和中亚部分地区。它在噶尔丹、策妄阿拉布坦、噶尔丹策零时期强盛，一度成为清朝在西北边疆最大的对手。

清朝与准噶尔的冲突贯穿康熙、雍正、乾隆三朝。康熙帝三征噶尔丹，雍正时期经营西北，乾隆时期趁准噶尔内乱出兵，1755年攻入伊犁，1757年前后平定准噶尔势力。此后清朝将新疆纳入更稳定的军政管理体系。

## 说明

- 准噶尔属于卫拉特蒙古体系，与和硕特、杜尔伯特、土尔扈特等部关系密切。
- 噶尔丹时期准噶尔势力向漠北蒙古扩张，引发清朝和喀尔喀蒙古的共同应对。
- 1690年乌兰布通之战、1696年昭莫多之战后，噶尔丹势力被严重削弱。
- 策妄阿拉布坦、噶尔丹策零时期，准噶尔仍能控制天山北路并影响西藏和中亚局势。
- 乾隆时期清军平定准噶尔后，又平定大小和卓叛乱，最终形成新疆治理格局。

## 演变关系

| 关系 | 说明 |
|---|---|
| 前一节点 | 卫拉特蒙古诸部。 |
| 并列关系 | 喀尔喀蒙古、青海和硕特、西藏政局、中亚诸汗国。 |
| 后一节点 | 清朝平定准噶尔并设置伊犁将军，新疆进入清朝边疆治理体系。 |
