# 淮军

## 时间

1862年前后形成，19世纪60年代至甲午战争前后影响最大。

## 概括

淮军是李鸿章以安徽淮河流域团练和湘军经验为基础组建的晚清地方军队。它在镇压太平天国、捻军和维护江南秩序中发挥重要作用，后来成为李鸿章经营洋务、海防和北洋系统的重要军事基础。

淮军比湘军更深地卷入近代军事技术和外交环境。它使用洋枪洋炮，参与洋务企业、北洋海防和对外战争，但在甲午战争中暴露出军事体制、训练、后勤和国家动员能力不足。

## 说明

- 1862年，李鸿章率淮军赴上海，与湘军、洋枪队等共同对抗太平军。
- 淮军将领多出自安徽、江苏及李鸿章私人网络，具有明显地方和人身依附特点。
- 淮军镇压太平天国和捻军后，成为李鸿章掌控直隶、北洋、海防和洋务事务的重要基础。
- 淮军系统与北洋水师、北洋机器局、天津武备学堂等晚清洋务体系关系密切。
- 甲午战争后，淮军和北洋体系威望受挫，清廷转向编练新军。

## 组织特点

| 特点 | 说明 |
|---|---|
| 来源 | 安徽淮系团练、湘军经验和地方募兵网络。 |
| 代表人物 | 李鸿章、刘铭传、张树声、周盛波、周盛传、叶志超等。 |
| 军事技术 | 较早使用西式枪炮和近代训练，依赖洋务体系补给。 |
| 历史影响 | 上承湘军，下启北洋军事集团和晚清新军改革。 |
