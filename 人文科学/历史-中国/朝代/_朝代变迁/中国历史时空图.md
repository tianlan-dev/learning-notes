# 中国历史时空图

![image1](_images/中国历史时空图-01.png)

## 概括

这张图把中国历史放在“时间 + 地域”的坐标中观察：纵轴是时间，从约前3000年到20世纪初；横向按区域展开，包括中亚、新疆、北方草原、东北、渭河平原与河西走廊、华北、长江中下游、四川盆地、珠江流域、云贵、越南北部、朝鲜北部等。

图中的色块表示政权或文化的大致存在范围，边界表示势力范围的变动。点、线和文字用于标出重要事件、人物、战争、都城、文化遗址和世界史参照。

## 读图方法

| 要素 | 含义 |
|---|---|
| 纵向时间轴 | 上古至近代，越往下越接近现代。 |
| 横向区域列 | 同一时期不同地理区域的文化、政权和族群分布。 |
| 大色块 | 主要王朝、政权、部族或区域势力的大致存在范围。 |
| 细长色块 | 边疆政权、割据政权、都护府、地方势力或短期政权。 |
| 红色文字 | 战争、起义、政权更替等冲突性事件。 |
| 蓝色文字 | 交通、制度、外交、科技文化等事件或说明。 |
| 绿色文字 | 人物、文化事项、制度或区域性标注。 |
| 点状标记 | 事件、人物活动或都城位置的大致时间点。 |

## 时间主线

| 时段 | 中原与核心王朝 | 并行区域与重要线索 |
|---|---|---|
| 约前3000年-前2000年 | 龙山文化、二里头文化等早期文明阶段 | 红山文化、马家窑文化、大汶口文化、良渚文化、石家河文化、宝墩文化、三星堆文化等区域文化并行。 |
| 约前1600年-前1046年 | 商 | 三星堆文化、岳石文化、吴城遗址等与商并行；甲骨文、青铜器和都邑体系出现。 |
| 前1046年-前256年 | 周，包括西周、东周 | 春秋战国时期诸侯并立，秦、晋、齐、楚、燕、吴、越等势力扩张；北方草原、西南、东南沿海仍有多种区域势力。 |
| 前221年-前207年 | 秦 | 完成统一，设郡县，北击匈奴，南征百越，修筑交通和防御体系。 |
| 前202年-220年 | 汉，包括西汉、新、东汉 | 西域都护府、匈奴、乌桓、鲜卑、羌、夜郎、南越、卫满朝鲜等与汉并行；张骞通西域、丝绸之路、汉匈战争是关键线索。 |
| 220年-280年 | 三国：魏、蜀、吴 | 北方、西南、江东三大政权鼎立；辽东、公孙氏、南中、山越等区域线索仍在。 |
| 265年-420年 | 晋，包括西晋、东晋 | 西晋短暂统一后进入十六国与东晋南北对峙；北方出现前赵、后赵、前秦、后秦、北凉、南燕等政权。 |
| 420年-589年 | 南北朝 | 南朝宋、齐、梁、陈与北魏、东魏、西魏、北齐、北周并行；柔然、高句丽、吐谷浑等活跃。 |
| 581年-618年 | 隋 | 重新统一南北，开通大运河，北方与西域仍有突厥、高句丽等重要对手。 |
| 618年-907年 | 唐 | 唐长期控制中原、关中、河西与西域部分地区；周边并行吐蕃、回鹘、渤海国、南诏、新罗、日本等。 |
| 907年-960年 | 五代十国 | 中原五代更替，南方十国并立；北方契丹辽崛起。 |
| 960年-1279年 | 宋，包括北宋、南宋 | 与辽、西夏、金、蒙古并行；南宋偏安江南，北方政权更替频繁。 |
| 1206年-1368年 | 蒙古帝国与元 | 蒙古兴起后统一中国，元代覆盖中原、草原、西域和青藏高原部分区域；同时有大理、吐蕃诸部等区域线索。 |
| 1368年-1644年 | 明 | 明控制中原、江南、华南和部分边疆，北方有北元、瓦剌、鞑靼，东北女真兴起；郑和下西洋、土木堡之变、倭寇、明末党争等被标出。 |
| 1616年-1912年 | 后金、清 | 清由东北入关，统一中原与边疆，西藏、新疆、蒙古、台湾等纳入不同形式的治理；晚清出现鸦片战争、太平天国、洋务运动、甲午战争、戊戌变法、义和团等事件。 |
| 1912年以后 | 中华民国及近现代转型 | 图的主体止于20世纪初，右侧世界史参照延伸到近代革命、工业化和列强扩张。 |

## 区域线索

| 区域 | 图中主线 |
|---|---|
| 中亚 | 大宛、康居、贵霜、嚈哒、突厥、阿拉伯、蒙古诸部、俄国等先后出现，是中原王朝与欧亚大陆交流、冲突的重要方向。 |
| 新疆 | 西域都护府、安西都护府、高昌、龟兹、回鹘、喀喇汗、准噶尔、清代新疆治理等连续出现。 |
| 北方草原 | 匈奴、鲜卑、柔然、突厥、回鹘、契丹、蒙古、女真、满洲等族群和政权反复成为中原王朝的北方变量。 |
| 东北 | 箕子朝鲜、卫满朝鲜、夫余、高句丽、渤海、辽、金、后金、清等政权构成东北历史主线。 |
| 渭河平原 / 河西走廊 | 周、秦、汉、唐等以关中为核心的王朝在此展开；河西走廊连接西域，是丝路和边防核心通道。 |
| 华北 | 商、周、战国诸侯、秦汉、魏晋、北朝、隋唐、宋辽金元明清等主线最密集，是王朝更替核心区。 |
| 长江中下游 | 吴、越、楚、东晋南朝、南唐、南宋、明清江南等线索突出，是南方政权和经济文化重心的重要区域。 |
| 四川盆地 | 三星堆、蜀、巴、成汉、前后蜀、宋元明清四川等线索连续，体现相对独立的盆地历史。 |
| 珠江流域 | 百越、南越、南汉及明清岭南治理等线索出现，是华南纳入王朝体系的重要区域。 |
| 云贵 | 滇、夜郎、南诏、大理、土司和明清改土归流等线索突出。 |
| 越南北部 | 南越、交趾、安南都护府、越南独立政权和阮朝等线索体现中原王朝与越南北部的长期互动。 |
| 朝鲜北部 | 箕子朝鲜、卫满朝鲜、汉四郡、高句丽、新罗、高丽、李氏朝鲜等线索显示东北亚秩序变迁。 |

## 重要事件标记

| 时期 | 事件或标记 | 说明 |
|---|---|---|
| 西汉 | 张骞两次通西域 | 打通中原与西域、中亚的长期交通和外交通道。 |
| 西汉 | 西域都护府 | 汉代经营西域的重要制度标记。 |
| 秦汉 | 南征百越、汉灭南越 | 华南逐步纳入中央王朝治理体系。 |
| 东汉末 | 黄巾起义、董卓入京、官渡之战、赤壁之战 | 东汉秩序崩解并导向三国格局。 |
| 西晋 | 八王之乱、永嘉之乱 | 西晋瓦解，北方进入十六国和民族政权并立阶段。 |
| 北魏 | 孝文帝改革 | 北朝政治制度和民族融合的重要节点。 |
| 隋唐 | 开通大运河、玄奘西行、安史之乱、两税法 | 展示隋唐帝国的制度建设、对外交流和中后期转折。 |
| 宋元 | 澶渊之盟、靖康之变、蒙古兴起、崖山海战 | 宋与辽金蒙古关系、南北对峙和元统一的关键节点。 |
| 明 | 土木堡之变、郑和下西洋、万历朝鲜战争、明末党争 | 显示明代对外、边防和内部政治变化。 |
| 清 | 康熙平定三藩、准噶尔、鸦片战争、太平天国、洋务运动、甲午战争、戊戌变法、义和团 | 展示清代统一、边疆治理和晚清危机。 |
