# 北宋

## 时间

960年-1127年。

## 概括

北宋是宋朝前期阶段，由赵匡胤代后周建立，定都东京开封。北宋基本结束五代十国以来中原和南方割据局面，但未能收复燕云十六州，因此长期面对辽、西夏等北方和西北政权的军事压力。

北宋以重文抑武、分权制衡为基本政治特征，强化中央集权和科举文官政治，经济、城市和文化高度发展。其弱点在于军政体系牵制较多，边防主动权不足。1127年金军攻陷汴京，徽宗、钦宗被俘，北宋灭亡。

## 说明

- 960年，赵匡胤发动陈桥兵变，黄袍加身，建立宋朝。
- 北宋先后平定南方诸国，并于979年灭北汉，基本结束五代十国分裂局面。
- 1005年，宋辽签订澶渊之盟，北宋以岁币换取辽宋边境长期和平。
- 宋仁宗时期政治较为宽缓，经济、科举和士大夫文化繁荣。
- 王安石变法试图解决财政、军事和社会治理问题，但也引发新旧党争。
- 1120年，宋金结成海上之盟夹攻辽；金灭辽后南下攻宋。
- 1127年，金军攻破汴京，掳走徽宗、钦宗，史称靖康之变，北宋灭亡。

## 统治结构

| 角色 | 说明 |
|---|---|
| 君主 | 赵氏皇帝，最高统治者。 |
| 宰执 | 宰相、参知政事等负责最高行政决策。 |
| 枢密院 | 掌军政，但与统兵将领分离，体现以文制武。 |
| 三司 | 掌财政，后在神宗元丰改制中调整。 |
| 地方官 | 知州、通判、转运使等互相牵制，加强中央控制。 |
