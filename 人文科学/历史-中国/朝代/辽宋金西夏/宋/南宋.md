# 南宋

## 时间

1127年-1279年。

## 概括

南宋是宋朝后期阶段。靖康之变后，宋高宗赵构在南京应天府即位，后南迁临安，重建宋朝。南宋失去黄河流域和华北核心区域，但依靠江南经济、海运、财政和军事防线维持了一个半世纪。

南宋前期围绕抗金与议和展开政治斗争，岳飞、韩世忠等将领一度取得战果，后以绍兴和议形成宋金对峙。13世纪蒙古兴起后，南宋先联蒙灭金，随后独自面对蒙古和元朝南下。1276年临安陷落，1279年崖山海战后南宋残余政权灭亡。

## 说明

- 1127年，赵构在南京应天府即位，重建宋朝。
- 南宋初期都城几经转移，最终以临安府为行在和事实首都。
- 高宗时期既有李纲、宗泽、岳飞、韩世忠等主战力量，也有秦桧等主和力量。
- 1142年前后，南宋与金形成绍兴和议后的长期对峙格局。
- 南宋经济重心在江南，城市、海外贸易、手工业和文化教育继续发展。
- 1234年，南宋联蒙古灭金，但蒙古随即成为最大威胁。
- 1276年，元军攻占临安；1279年，崖山海战后陆秀夫负幼主赵昺投海，南宋灭亡。

## 统治结构

| 角色 | 说明 |
|---|---|
| 君主 | 赵氏皇帝，南渡后延续宋朝法统。 |
| 宰执 | 中书门下、尚书省等中枢设置随改制变化，宰执掌政务。 |
| 军事统帅 | 抗金、抗蒙时期将领地位重要，但仍受中央文官体系节制。 |
| 财政与运输 | 江南税赋、盐茶收入、海运和漕运支撑朝廷长期抗战。 |
| 地方防线 | 淮河、长江、荆襄、四川等区域构成对金和蒙古的主要防御体系。 |
