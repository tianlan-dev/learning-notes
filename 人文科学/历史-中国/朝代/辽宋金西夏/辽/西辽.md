# 西辽

## 时间

1132年-1218年。

## 别称

喀喇契丹、哈剌契丹。

## 概括

西辽是辽亡后由契丹贵族耶律大石西迁中亚建立的政权。辽朝在金军打击下灭亡后，耶律大石率部向西发展，在楚河流域和中亚地区建立统治，延续契丹政治传统。西辽在中亚多族群、多宗教环境中统治，既保留契丹和辽朝制度记忆，也吸收当地政治关系。

1211年，乃蛮王子屈出律夺取西辽政权。1218年，蒙古西征过程中灭西辽，契丹在中亚的王朝延续结束。

## 说明

- 1125年辽亡后，耶律大石未归附金朝，而是率契丹残部西迁。
- 1132年，耶律大石称帝，建立西辽。
- 西辽统治区域大致包括七河流域、天山南北和中亚部分地区。
- 西辽在中亚史中常被称为喀喇契丹，与伊斯兰世界诸政权发生战争和宗主关系。
- 1211年，屈出律篡夺西辽政权。
- 1218年，蒙古灭西辽。

## 演变关系

| 关系 | 内容 |
|---|---|
| 前一节点 | [辽](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2-%E4%B8%AD%E5%9B%BD/%E6%9C%9D%E4%BB%A3/%E8%BE%BD%E5%AE%8B%E9%87%91%E8%A5%BF%E5%A4%8F/%E8%BE%BD/README.md)。 |
| 并列关系 | 与金、南宋、西夏处在同一大时代，但地理重心转向中亚。 |
| 后一节点 | 蒙古帝国西征扩张。 |
| 分支性质 | 西辽不是辽朝本土直接延续，而是契丹贵族西迁后建立的流亡与扩张型政权。 |

## 统治结构

| 角色 | 说明 |
|---|---|
| 君主 | 耶律氏皇帝，沿用辽朝皇帝称号和契丹政治传统。 |
| 摄政者 | 萧塔不烟、耶律普速完等女性统治者曾以皇后身份摄政或掌权。 |
| 地方统治 | 对中亚诸城邦和地方王公多采取宗主权、贡赋和间接统治。 |
| 后期实际统治者 | 屈出律篡位后控制西辽，直到蒙古灭亡该政权。 |
