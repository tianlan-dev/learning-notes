## 概括

南朝（420年—589年）指南方依次更替的宋、齐、梁、陈四个政权。除梁元帝短暂都江陵外，南朝大部分时间以建康（今江苏南京）为都城。南朝承接东晋江南政权传统，政治上有门阀士族影响，也有寒门武人和皇权上升；军事上长期与北魏、东魏、西魏、北齐、北周对峙。

## 核心线索

- **都城连续性**：建康长期是南方政权中心。
- **政权更替模式**：多由掌握军政实权的大臣逼迫前朝末帝禅让。
- **士族与寒门**：南朝前期仍有士族影响，但军功集团和寒门官僚逐渐重要。
- **南弱北强趋势**：侯景之乱后，南朝国力严重受损，北方统一压力不断增强。
