# 史前文化列表

![image1](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2-%E4%B8%AD%E5%9B%BD/%E6%9C%9D%E4%BB%A3/%E5%8F%B2%E5%89%8D%E6%97%B6%E6%9C%9F/_images/%E5%8F%B2%E5%89%8D%E6%96%87%E5%8C%96%E5%88%97%E8%A1%A8-01.png)

## 概括

这张图按时间和流域列出中国重要史前文化。纵轴大致是距今年代，从距今10000年至距今2000年；横向分为黄河上游、黄河中游、黄河下游、长江下游、长江中游等区域。图中还用图例标出石器、骨器、陶器、玉器、青铜器、聚落、城邑、建筑、农业、家畜、纺织、文字符号等考古特征。

## 图例要点

| 类型 | 图中含义 |
|---|---|
| 石器 | 打制、磨制石器。 |
| 骨器 | 粗制、精致骨器。 |
| 陶器 | 初始陶器、中级陶器、高级陶器。 |
| 玉器 | 普通或精美玉器。 |
| 青铜器 | 少量或大量青铜器。 |
| 聚落与建筑 | 村落、城邑、半地穴式建筑、土台基建筑、干栏式建筑。 |
| 农业 | 原始或成熟的粟黍、稻作。 |
| 家畜 | 狗、猪、牛、羊、鸡等。 |
| 手工业与符号 | 编织、纺织、类似文字的符号、骨卜。 |

## 主要文化表

| 区域 | 文化 / 遗址 | 大致时间 | 主要特征 |
|---|---|---|---|
| 黄河中游 | 南庄头遗址 | 距今约10000年前后 | 早期陶器、磨制石器和定居生活线索，是华北新石器时代早期遗存之一。 |
| 黄河中游 | 裴李岗文化 | 距今约9000年-7000年 | 粟作农业、磨制石器、陶器和聚落发展。 |
| 黄河中游 | 磁山文化 | 距今约8000年-7000年 | 粟作农业、家畜饲养和磨制石器较突出。 |
| 黄河上游 | 大地湾文化 | 距今约8000年-5000年 | 彩陶、聚落、早期农业和建筑遗存丰富。 |
| 黄河下游 | 后李文化 | 距今约8500年-7500年 | 山东地区早期新石器文化，陶器和定居生活明显。 |
| 辽河流域 | 兴隆洼、赵宝沟、红山文化 | 距今约8000年-5000年 | 聚落、玉器、祭祀遗址和红山文化礼仪系统突出。 |
| 黄河中游 | 仰韶文化 | 距今约7000年-5000年 | 彩陶传统、粟作农业和大型聚落发达，庙底沟类型影响广。 |
| 黄河上游 | 马家窑文化 | 距今约5300年-4000年 | 彩陶高度发达，分布于甘青地区。 |
| 黄河下游 | 北辛文化 | 距今约7300年-6100年 | 海岱地区新石器文化，承接后李并影响大汶口。 |
| 黄河下游 | 大汶口文化 | 距今约6300年-4500年 | 墓葬分化、陶器和玉器发展，社会分层增强。 |
| 长江下游 | 河姆渡文化 | 距今约7000年-5300年 | 稻作农业、干栏式建筑、木构技术和水乡聚落突出。 |
| 长江下游 | 马家浜文化 | 距今约7000年-6000年 | 长江下游早期稻作文化之一。 |
| 长江下游 | 崧泽文化 | 距今约6000年-5300年 | 连接马家浜与良渚，聚落和墓葬分化增强。 |
| 长江下游 | 良渚文化 | 距今约5300年-4300年 | 玉礼器、古城、水利系统和等级社会突出，是早期文明化的重要代表。 |
| 长江中游 | 彭头山文化 | 距今约9000年-7800年 | 稻作和早期陶器线索重要。 |
| 长江中游 | 城背溪文化 | 距今约8000年-7000年 | 长江中游早期新石器文化，稻作和聚落发展。 |
| 长江中游 | 大溪文化 | 距今约6500年-5300年 | 稻作、聚落和墓葬资料丰富。 |
| 长江中游 | 屈家岭文化 | 距今约5300年-4500年 | 长江中游区域中心发展，陶器和聚落形态明显。 |
| 长江中游 | 石家河文化 | 距今约4600年-4000年 | 城址、玉器和复杂社会发展突出。 |
| 黄河中游 | 龙山文化 | 距今约4500年-4000年 | 城址、黑陶、战争和社会分化明显，处于早期国家形成前夜。 |
| 黄河中游 | 陶寺文化 | 距今约4300年-3900年 | 大型城址、礼制、天文观测和社会分层线索突出。 |
| 黄河上游 | 齐家文化 | 距今约4200年-3600年 | 铜器、玉器和西北地区早期青铜时代线索重要。 |
| 黄河中游 | 二里头文化 | 距今约3800年-3500年 | 宫殿区、青铜礼器和都邑结构明显，常与夏代问题相关。 |
| 四川盆地 | 宝墩文化 | 距今约4500年-3700年 | 成都平原早期城址群。 |
| 四川盆地 | 三星堆文化 | 距今约4000年-3000年 | 青铜器、祭祀坑和区域文明特色突出。 |
| 中原 | 二里岗文化、殷墟文化 | 距今约3600年-3000年 | 商代青铜文明、甲骨文和王权国家高度发展。 |
| 关中 | 周原等先周、周文化 | 距今约3200年以后 | 周人兴起、西周礼制和封建秩序形成。 |

## 读图提示

- 图中“距今”是相对年代，和“公元前”换算时需要注意现代基准不同。
- 同一文化内部常有早中晚期，表中只列大致范围。
- “夏、商、周”在图中用于和传统王朝叙事对照；考古文化与王朝边界不一定完全重合。
