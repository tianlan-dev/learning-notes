# 史前时期

## 时间

约旧石器时代至夏、商、周形成以前。若按这张图的范围，重点覆盖距今约10000年至距今约2000年之间的中国新石器时代、铜石并用时代和早期青铜时代文化。

## 概括

中国史前时期不是单一王朝，而是多个区域文化并行、交流和演变的阶段。黄河流域、长江流域、辽河流域、四川盆地、华南和西北地区都出现了各具特征的考古学文化。它们共同构成早期中国文明形成的基础。

这一时期的主线包括：农业定居、陶器发展、玉器礼制化、聚落和城址出现、贫富分化、礼仪中心形成、早期铜器和青铜器出现，以及中原地区逐渐进入二里头、商、周等早期国家阶段。

## 阶段概览

| 阶段 | 大致时间 | 主要特征 | 代表线索 |
|---|---|---|---|
| 旧石器时代晚期 | 距今约数万年至约10000年 | 打制石器、狩猎采集为主，人群流动性较强。 | 山顶洞人等旧石器遗存。 |
| 新石器时代早中期 | 距今约10000年-约7000年 | 农业、陶器、磨制石器和定居聚落逐渐发展。 | 南庄头、裴李岗、磁山、大地湾、后李等。 |
| 新石器时代中晚期 | 距今约7000年-约5000年 | 区域文化繁荣，彩陶、玉器、稻作和粟作系统发展。 | 仰韶、河姆渡、大汶口、马家浜、崧泽等。 |
| 龙山时代 / 铜石并用阶段 | 距今约5000年-约4000年 | 城址、礼制、战争、社会分层加剧，区域中心增强。 | 龙山、良渚、石家河、宝墩、陶寺、三星堆早期等。 |
| 早期国家形成 | 约前2000年前后以后 | 中原进入二里头、商、周体系，青铜礼器和王权国家形成。 | 二里头、二里岗、殷墟、周原。 |

## 区域线索

| 区域 | 主要文化或遗址 | 说明 |
|---|---|---|
| 黄河上游 | 大地湾、马家窑、齐家等 | 彩陶传统突出，后期出现铜器和早期青铜器线索。 |
| 黄河中游 | 裴李岗、磁山、仰韶、庙底沟、龙山、陶寺、二里头等 | 逐渐成为早期国家形成的重要核心区。 |
| 黄河下游 | 后李、北辛、大汶口、龙山、岳石等 | 聚落、墓葬和礼器变化明显，与中原和东部海岱地区互动密切。 |
| 长江下游 | 河姆渡、马家浜、崧泽、良渚等 | 稻作农业、干栏式建筑、玉器和水乡聚落特征突出。 |
| 长江中游 | 彭头山、城背溪、大溪、屈家岭、石家河等 | 稻作、城址和区域中心发展明显。 |
| 四川盆地 | 宝墩、三星堆、十二桥等 | 成都平原早期城址和三星堆文明是西南史前至早期青铜时代的重要线索。 |
| 辽河流域 | 兴隆洼、赵宝沟、红山、小河沿等 | 玉器、祭祀遗址和女神庙等遗存体现东北地区复杂社会发展。 |
| 华南与东南 | 贝丘、百越相关史前文化等 | 与海洋、稻作、贝丘遗存和南方区域文化有关。 |

## 说明

- 史前文化的时间多为考古学分期，不等同于后世王朝纪年。
- 各区域文化并非单线继承关系，而是长期并行、互动、融合和替代。
- “母系、父系、神话时代、夏、商、周”等标签多是为了帮助对照传统历史叙事，不能简单等同于考古学文化本身。
