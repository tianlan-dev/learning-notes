# 汉

## 时间

- 西汉：前202年-8年
- 新朝：9年-23年
- 更始政权、赤眉汉等过渡政权：23年-27年
- 东汉：25年-220年

## 概括

汉是继秦之后的长期大一统王朝，分为西汉和东汉。西汉由汉高祖刘邦建立，定都长安；东汉由汉光武帝刘秀建立，定都洛阳。因为皇室姓刘，也称“刘汉”。

汉承秦制而有所调整，形成郡县与封国并行、中央官僚治理、儒学进入国家意识形态、对匈奴和西域长期经营等基本格局。汉武帝时期国力极盛，张骞通西域后丝绸之路逐渐成形。西汉平帝时期人口约六千万，汉代政治、制度、文化和族群认同对后世影响很深，“汉族”“汉字”等称谓也与此有关。

## 历史阶段

| 阶段 | 时间 | 概括 |
|---|---|---|
| 楚汉相争与西汉建立 | 前206年-前202年 | 刘邦击败项羽，前202年称帝，建立汉朝。 |
| 汉初休养生息 | 前202年-前141年 | 高祖、惠帝、吕后、文帝、景帝时期恢复经济，形成文景之治。 |
| 汉武扩张 | 前141年-前87年 | 汉武帝强化中央集权，打击匈奴，经营西域，推行盐铁、均输、察举等政策。 |
| 昭宣中兴 | 前87年-前49年 | 霍光辅政后政局稳定，汉昭帝、汉宣帝时期国力恢复。 |
| 西汉后期衰落 | 前49年-8年 | 外戚势力上升，土地兼并加剧，王莽最终篡汉。 |
| 新莽改制与天下动乱 | 9年-23年 | 王莽建立新朝，大规模改制失败，引发绿林、赤眉等起义。 |
| 东汉重建 | 25年-57年 | 刘秀称帝，削平群雄，定都洛阳，开创光武中兴。 |
| 东汉前中期稳定 | 57年-125年 | 明章之治后，外戚、宦官与幼主政治逐渐成为结构性问题。 |
| 东汉后期危机 | 125年-189年 | 外戚宦官轮流专权，党锢之祸、土地兼并和黄巾起义削弱中央。 |
| 汉末割据 | 189年-220年 | 董卓乱政后群雄并起，曹操控制汉献帝，曹丕最终篡汉建魏。 |

## 重要制度与影响

| 领域 | 内容 | 影响 |
|---|---|---|
| 政治制度 | 承秦郡县制，同时保留诸侯王封国。 | 初期形成郡国并行，七国之乱后中央集权增强。 |
| 思想文化 | 汉武帝以后儒学地位上升。 | 儒家经典逐渐成为官学和政治伦理核心。 |
| 选官制度 | 察举制发展。 | 为士人进入官僚体系提供制度化途径。 |
| 对外关系 | 汉匈战争、张骞通西域、西域都护府。 | 中原与西域、中亚联系加强，丝绸之路形成。 |
| 经济政策 | 休养生息、盐铁官营、均输平准等。 | 兼具恢复经济和强化财政动员的双重作用。 |
| 族群认同 | 汉代长期统一和文化整合。 | “汉”成为后世重要族群、文字和文化标识。 |

## 灭亡原因

- 东汉中后期皇帝多幼年即位，太后临朝、外戚辅政成为常态。
- 宦官借皇权制衡外戚，外戚与宦官长期争斗，中央政治内耗严重。
- 土地兼并扩大，大量农民失去土地，豪强地主形成地方性力量。
- 党锢之祸打击士人政治参与，削弱朝廷整合能力。
- 黄巾起义促使地方州郡和豪强武装化，中央失去对地方军事的有效控制。
- 董卓乱政后中央权威崩溃，曹操挟汉献帝以令诸侯，最终曹丕篡汉。
