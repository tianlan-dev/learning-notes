# 七国之乱

## 时间

前154年

## 概括

七国之乱是西汉景帝时期以吴王刘濞为首的七个诸侯王发动的叛乱。其根源是汉初郡国并行体制下，诸侯王势力过大，与中央集权发生冲突。景帝采纳晁错削藩建议后，吴、楚等国以“诛晁错，清君侧”为名起兵。叛乱被周亚夫等平定后，诸侯王权力被进一步削弱，西汉中央集权明显加强。

## 背景

| 背景 | 说明 |
|---|---|
| 郡国并行 | 汉初既有中央郡县，也有同姓诸侯王封国。 |
| 诸侯坐大 | 吴、楚等国地广财足，拥有铸钱、煮盐等经济优势。 |
| 中央削藩 | 文帝、景帝时期逐步限制诸侯王权力。 |
| 晁错建议 | 晁错主张削夺诸侯王封地，触发诸侯反弹。 |

## 叛乱七国

| 诸侯国 | 代表人物 | 说明 |
|---|---|---|
| 吴 | 吴王刘濞 | 叛乱核心，实力最强。 |
| 楚 | 楚王刘戊 | 与吴合兵，是主要叛乱力量。 |
| 赵 | 赵王刘遂 | 北方参与叛乱的诸侯国。 |
| 济南 | 济南王刘辟光 | 齐地诸王之一。 |
| 菑川 | 菑川王刘贤 | 齐地诸王之一。 |
| 胶西 | 胶西王刘卬 | 齐地诸王之一。 |
| 胶东 | 胶东王刘雄渠 | 齐地诸王之一。 |

## 经过

| 阶段 | 说明 |
|---|---|
| 起兵 | 吴王刘濞联合楚、赵及齐地诸王，以“诛晁错”为名起兵。 |
| 中央应对 | 汉景帝诛晁错试图缓和局势，但叛军并未停止。 |
| 梁国牵制 | 梁王刘武坚守，牵制吴楚主力。 |
| 周亚夫平乱 | 周亚夫避其锋芒，断绝叛军粮道，最终击败吴楚。 |
| 叛乱失败 | 七国迅速败亡，主要叛乱诸侯被杀或自杀。 |

## 影响

- 中央证明有能力压制同姓诸侯王。
- 诸侯王军事、行政和财政权力被进一步削弱。
- 后续推恩令等政策继续拆分和削弱封国。
- 西汉从汉初的郡国并行，逐步走向更强的中央集权。
- 七国之乱为汉武帝时期进一步强化中央权力铺平道路。
