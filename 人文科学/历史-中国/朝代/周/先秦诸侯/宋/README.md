# 宋

## 时间

- 约前11世纪：商王族后裔微子启受封于宋。
- 前286年：齐、楚、魏共灭宋。

## 概括

宋是商王族后裔封国，位于今河南东部、山东西南和安徽北部一带。宋国既是周代安置殷商遗民的重要政治安排，也是春秋中原格局中的重要诸侯。宋襄公曾试图继齐桓公之后称霸，但宋国总体实力有限，战国时因宋康王扩张引发诸国围攻而灭亡。

## 说明

- 周初，微子启受封于宋，以奉商祀，是周代分封前代王族后裔的典型。
- 宋国地处中原交通要道，常处于齐、楚、晋、郑、鲁等国影响之间。
- 春秋时期宋襄公图霸，在泓水之战中败于楚，宋国霸业未成。
- 宋国在春秋战国之间仍保持一定实力，但难以与大国长期抗衡。
- 战国时期宋康王一度扩张，攻滕、灭薛，号称“五千乘之劲宋”。
- 前286年，齐、楚、魏共灭宋，瓜分宋地。
