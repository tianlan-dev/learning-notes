# 卫

## 时间

- 约前11世纪：周公弟康叔受封于卫。
- 前209年：秦二世废卫君角，卫国最终结束。

## 概括

卫是周初姬姓诸侯国，始封君为康叔。卫国早期地位较高，春秋以后逐渐衰落，长期夹在晋、齐、郑、宋等国之间。战国时卫降为附庸性小国，却一直延续到秦统一之后，是周代诸侯国中存续时间很长的一支。

## 说明

- 周公平定三监之乱后，康叔受封于殷商故地附近，负责安抚殷民并控制东方要地。
- 卫国都城曾在朝歌等地，后来因狄人入侵和政治变动迁徙。
- 春秋时卫国内乱频繁，国君与卿族、诸公子之间斗争不断。
- 卫地与孔子生平关系密切，孔子周游列国时多次至卫。
- 战国时期卫国国力微弱，先后依附魏、秦等强国。
- 秦统一六国后仍保留卫君角，前209年秦二世废卫君，卫国最终结束。

## 演变关系

| 关系 | 说明 |
|---|---|
| 前一节点 | 周初姬姓封国，承担控制殷商旧地的职能。 |
| 并列关系 | 与宋、郑、鲁、晋等中原诸侯联系密切。 |
| 后一节点 | 秦统一后仍短暂保留，前209年被废。 |
