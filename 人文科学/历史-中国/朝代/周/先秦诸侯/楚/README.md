# 楚

## 时间

- 西周初：熊绎受封于楚蛮，居丹阳。
- 前704年：楚子熊通自立为王，史称楚武王。
- 前223年：秦军攻破寿春，楚国灭亡。

## 别称

- 荆
- 荆楚
- 楚国

## 概括

楚是芈姓熊氏诸侯国，起源可追溯至商周之际，最早兴起于丹江流域。楚在中原诸侯眼中长期带有“蛮夷”色彩，却在春秋战国持续扩张，成为南方第一大国。楚庄王是春秋霸主之一，战国时期楚又是七雄之一，最终在秦统一战争中灭亡。

## 说明

- 楚人的活动历史可上追至商朝。商王武丁攻打鬼方时，曾经征讨楚人。
- 楚先祖鬻熊为殷商诸侯。西周初，周成王封熊绎于楚蛮，居丹阳，爵位为子爵。
- 前704年，楚子熊通自立为王，突破周代爵秩秩序，史称楚武王。
- 春秋早期，楚国大举扩张，灭亡江汉流域众多小国，成为南方第一大国。
- 前597年，楚庄王在邲之战中击败晋国，被列为春秋五霸之一。
- 春秋晚期，楚在吴楚争霸中连续受挫。前506年，吴王阖闾攻楚，柏举之战大败楚军，攻破郢都；楚国在秦国援助下复国。
- 战国中期，楚宣王、楚威王时期国势强盛，史称“宣威盛世”。
- 楚怀王后期，内受郑袖等人影响，外受张仪欺诈，蓝田、丹淅等战事失利，楚国逐渐衰落。
- 前278年，秦将白起攻占鄢郢，楚国元气大伤。
- 前223年，秦军攻破楚都寿春，俘楚王负刍，楚国灭亡。

## 演变关系

| 关系 | 说明 |
|---|---|
| 前一节点 | 商周之际南方部族与周初子爵封国。 |
| 并列关系 | 春秋时与晋、吴等争霸；战国时与秦、齐、燕、韩、赵、魏并列为七雄。 |
| 后一节点 | 前223年被秦灭，故地纳入秦统一体系。 |
