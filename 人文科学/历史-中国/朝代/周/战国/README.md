# 战国

## 时间

公元前475年或前403年－公元前221年。常见分界包括前476年、前453年三家灭智、前403年周威烈王册命韩赵魏为诸侯；结束于前221年秦统一六国。

## 概括

战国是东周后半段，也是由贵族封建秩序转向郡县官僚国家的关键时期。韩、赵、魏、齐、楚、燕、秦等大国在兼并战争中扩张，诸侯相继称王，周天子权威几乎只剩象征。各国通过变法、军功爵制、郡县制、户籍赋税、铁器与农业扩张增强国力，最终秦国凭借制度、地缘和战争资源优势完成统一。

## 说明

- “战国”一名来自西汉刘向编定的《战国策》，强调诸侯国之间长期兼并战争的时代特征。
- 战国七雄一般指秦、楚、齐、燕、韩、赵、魏；此外宋、中山、越、巴蜀等也在不同时段影响格局。
- 周显王三十五年（前334年）徐州相王、四十六年（前323年）五国相王后，诸侯称王常态化，周天子权威进一步下降。
- 东周在前256年周赧王死后已经灭亡，战国时代则一直延续到前221年秦统一六国。
