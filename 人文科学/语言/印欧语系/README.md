# 印欧语系

## 范围

印欧语系覆盖欧洲大部、伊朗高原、南亚北部和中部，并随近现代殖民、移民和国际交流扩散到全球。

## 概括

印欧语系是比较语言学研究最充分的语系之一，可重建共同祖语“原始印欧语”。本目录展开亚美尼亚、希腊、阿尔巴尼亚、印度-伊朗、意大利、日耳曼、波罗的-斯拉夫、凯尔特等主要分支。
