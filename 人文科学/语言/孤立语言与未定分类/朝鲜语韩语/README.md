# 朝鲜语 / 韩语

## 概括

朝鲜语/韩语是朝鲜半岛主要语言，现代标准书写使用谚文。

## 说明

朝鲜语/韩语常作为独立小语系或孤立语处理；不应因地理接近而直接归入汉藏语系或阿尔泰语系。
