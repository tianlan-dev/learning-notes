# 日语

## 概括

日语是日本语系的主要语言，标准书写系统混用汉字、平假名、片假名。

## 子系统

| 分支 / 语言 | 代表内容 | 说明 |
|---|---|---|
| 日语 | 汉字、平假名、片假名 | 不应因使用汉字而归入汉语族。 |

## 说明

日语与朝鲜语、阿尔泰诸语的关系有过假说，但未成为普遍接受的确定亲缘分类。
