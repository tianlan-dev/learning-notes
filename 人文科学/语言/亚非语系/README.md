# 亚非语系

## 范围

亚非语系主要分布在北非、西亚、非洲之角和撒哈拉周边地区。

## 概括

亚非语系包括闪米特、库希特、柏柏尔、乍得、埃及等分支。本目录先展开闪米特语族和库希特语族，其他分支保留在总览中。
