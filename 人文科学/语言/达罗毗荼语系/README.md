# 达罗毗荼语系

## 概括

达罗毗荼语系主要分布于南印度、斯里兰卡和南亚部分地区，代表语言包括泰米尔语、泰卢固语、卡纳达语、马拉雅拉姆语。

## 说明

达罗毗荼语系与印欧语系的印度-雅利安语言不同源，但长期接触产生许多区域共性。
