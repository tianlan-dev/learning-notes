# 闽语

## 概括

主要分布于福建、台湾、海南、粤东、浙南及海外华人社群。

| 分支 / 语言 | 代表内容 |
|---|---|
| 闽南语 | 厦门话、泉州话、台语、东南亚闽南语、潮州话、汕头话等。 |
| 闽东语 | 福州话、福清话、长乐话、福安话、蛮讲等。 |
| 琼文话 | 海口话、文昌话、万宁话、崖县话、昌江话等。 |
| 莆仙语 | 莆田话、仙游话、乌坵话等。 |
| 雷州话 | 雷城话等。 |
| 闽北语 | 建瓯话、建阳话、武夷山话等。 |
