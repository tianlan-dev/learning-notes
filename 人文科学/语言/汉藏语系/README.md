# 汉藏语系

## 范围

汉藏语系主要分布于中国、缅甸、喜马拉雅地区和东南亚大陆北部。

## 概括

汉藏语系通常包括汉语族和藏缅语族两大层级。汉语族内部可继续整理官话、吴语、粤语、闽语、客语、赣语、湘语、晋语、徽语等汉语分支；藏缅语族则包括藏语、缅甸语等大量语言。
