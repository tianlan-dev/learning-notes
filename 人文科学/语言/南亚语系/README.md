# 南亚语系

## 概括

南亚语系又称 Austroasiatic，主要分布于东南亚大陆和南亚局部，代表语言包括越南语和高棉语。

## 说明

南亚语系不要与“南亚地区所有语言”混淆；印度北部大量语言属于印欧语系，南印度大量语言属于达罗毗荼语系。
