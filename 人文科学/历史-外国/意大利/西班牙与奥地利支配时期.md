# 西班牙与奥地利支配时期

## 时间

1559年-1796年

## 别称

外国支配时期、哈布斯堡影响时期

## 概括

卡托-康布雷西和约后，西班牙哈布斯堡支配意大利大部；18世纪奥地利哈布斯堡影响增强。

## 王朝世系 / 统治结构

此时期意大利半岛仍分裂，南意和米兰长期受西班牙或奥地利哈布斯堡控制，部分城邦保持自治。

| 顺序 / 人物或阶段 | 时间 | 身份 / 关系 | 关键事件 / 说明 |
|---|---|---|---|
| 西班牙哈布斯堡支配 | 1559-1714 | 西班牙王室 | 控制米兰、那不勒斯、西西里、撒丁等关键地区。 |
| 威尼斯共和国 | 1559-1797 | 独立共和国 | 保持自治但海上优势逐渐衰落。 |
| 托斯卡纳大公国 | 1569-1859 | 美第奇、洛林家族 | 中意重要公国。 |
| 萨伏依公国 / 撒丁王国 | 1563-1796 | 萨伏依家族 | 从阿尔卑斯地区逐步转向意大利统一主力。 |
| 奥地利哈布斯堡支配 | 1714-1796 | 奥地利王室 | 西班牙王位继承战争后控制伦巴第等地。 |
| 教皇国 | 1559-1796 | 教皇 | 中意大利世俗政权继续存在。 |

## 重要事件

- 1559年以后，西班牙在意大利占优势。
- 1647年，那不勒斯爆发马萨涅洛起义，反映税负和西班牙统治矛盾。
- 1713-1714年，乌得勒支和拉施塔特和约后，奥地利取代西班牙成为北意重要支配力量。
- 1720年，萨伏依家族取得撒丁王国头衔。
- 18世纪，启蒙改革在托斯卡纳、伦巴第等地推进。
- 1796年，拿破仑入侵意大利，旧秩序开始瓦解。
