# 十字军东征

## 概括

十字军东征通常指11世纪末至13世纪末，拉丁基督教世界以收复、保卫或扩张圣地利益为名发起的一系列军事行动。狭义的“第1次至第9次十字军东征”主要围绕黎凡特、埃及、突尼斯和拜占庭展开；广义上还包括北方十字军、阿尔比十字军等针对异教徒、异端或政治敌人的战争。

最初的导火索是拜占庭帝国在安纳托利亚受到塞尔柱突厥压力，皇帝阿莱克修斯一世向西方求援；教宗乌尔巴诺二世在1095年克莱蒙会议上号召西欧骑士东征。此后，十字军运动逐渐从“援助拜占庭、收复耶路撒冷”的宗教军事行动，演变为夹杂教廷权威、封建贵族扩张、商业利益、朝圣保护、拜占庭与西欧关系、伊斯兰世界内部政治等多重因素的长期战争。

## 广义十字军运动

狭义的九次东征之外，教廷还把十字军赎罪、宣誓、特许和军事动员机制用于欧洲内部、波罗的海、伊比利亚半岛、地中海和巴尔干等地区。这些战争不一定以耶路撒冷为直接目标，但仍属于广义十字军运动。

## 结局

1291年，马木留克苏丹阿什拉夫·哈利勒攻陷阿卡，耶路撒冷王国在黎凡特的主要据点瓦解。此后拉丁基督教势力仍在塞浦路斯、罗得岛等地保持存在，也多次计划新十字军，但传统意义上围绕圣地的十字军国家体系基本终结。
