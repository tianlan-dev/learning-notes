# 乌克兰

## 时间

1991年至今

## 概括

1991年8月24日，乌克兰宣布脱离苏联独立，成为现代乌克兰国家。

## 说明

- 乌克兰的历史线索在图中主要由西罗斯、加利西亚-沃里尼亚王国、波兰-立陶宛联邦、哥萨克酋长国、俄罗斯帝国和乌克兰苏维埃政权等阶段构成。
- 1917年，乌克兰苏维埃政权成立。
- 1939年，西乌克兰与乌克兰苏维埃社会主义共和国合并。
- 1941年，乌克兰被德国占领。
- 1944年，乌克兰解放。
- 1991年8月24日，乌克兰宣布脱离苏联独立。
- 图中将乌克兰标注为“东部亲俄，西部亲欧”。
