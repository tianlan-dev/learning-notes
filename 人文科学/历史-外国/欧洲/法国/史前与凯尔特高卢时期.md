# 史前与凯尔特高卢时期

## 时间

约前180万年-前52年

## 别称

史前法国、凯尔特高卢

## 概括

从旧石器时代遗存、农业扩散到凯尔特高卢部落社会，形成罗马征服前法国地域的族群与地域基础。

## 王朝世系 / 统治结构

此时期没有统一王朝。

| 顺序 / 人物或阶段 | 时间 | 身份 / 关系 | 关键事件 / 说明 |
|---|---|---|---|
| 旧石器时代 | 无 | 狩猎采集群体 | 拉斯科洞窟等旧石器文化遗存；尼安德特人与现代智人先后活动。 |
| 新石器时代 | 无 | 农业共同体 | 农业、畜牧、陶器和巨石文化扩散。 |
| 青铜器-铁器时代 | 无 | 部落社会 | 金属器、贸易网络和地方酋邦发展。 |
| 凯尔特高卢 | 无 | 高卢部落联盟 | 阿维尔尼、埃杜维、塞夸尼等部落并立，维钦托利领导反罗马联盟。 |

## 重要事件

- 约前17000年左右，拉斯科洞窟壁画代表旧石器时代艺术高峰之一。
- 前1千纪后期，凯尔特语族高卢部落在今法国大部地区占优势。
- 前58年-前50年，凯撒发动高卢战争。
- 前52年，维钦托利在阿莱西亚战败，高卢进入罗马统治轨道。
