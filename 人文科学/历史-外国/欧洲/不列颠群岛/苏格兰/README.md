# 苏格兰

## 历史主线

苏格兰从皮克特、盖尔、布立吞和盎格鲁边缘势力交错中形成。阿尔巴王国发展为苏格兰王国，长期与英格兰竞争；1603年进入斯图亚特共主，1707年通过联合法案进入大不列颠和联合王国结构。
