# 爱尔兰

## 历史主线

爱尔兰历史不能简单并入英国史。它从盖尔诸王国和修道院文化出发，经历诺曼入侵、英格兰 / 英国控制、殖民和宗教冲突；1801年并入联合王国，20世纪通过独立战争形成爱尔兰自由邦 / 爱尔兰共和国，同时北爱尔兰留在联合王国。
