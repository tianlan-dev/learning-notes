# 联合王国

## 历史主线

联合王国主线从1707年大不列颠王国开始，1801年合并爱尔兰成为大不列颠及爱尔兰联合王国，1922年爱尔兰自由邦成立后形成现代“大不列颠及北爱尔兰联合王国”。因此，汉诺威王朝和温莎王朝更适合放在联合王国目录下，而非单独英格兰史。
