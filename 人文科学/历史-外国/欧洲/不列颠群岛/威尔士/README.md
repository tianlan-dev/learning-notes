# 威尔士

## 历史主线

威尔士主线以罗马后布立吞诸国和凯尔特语言文化延续为基础。中世纪威尔士诸王国长期与英格兰竞争，1283年后被英格兰征服，16世纪并入英格兰法制；现代威尔士则以语言复兴、工业化遗产和自治议会为关键线索。
