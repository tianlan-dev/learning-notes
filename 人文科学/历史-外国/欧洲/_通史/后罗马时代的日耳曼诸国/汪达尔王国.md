# 汪达尔王国

## 时间

439年-534年

## 概括

汪达尔王国是汪达尔人在北非建立的后罗马日耳曼政权，以迦太基为中心，控制北非粮食产区和西地中海海权，534年被拜占庭帝国灭亡。

## 王朝世系 / 统治结构

| 君主 / 阶段 | 时间 | 说明 |
|---|---|---|
| 盖塞里克 | 428-477 | 率汪达尔人进入北非，439年占领迦太基，建立海上强权。 |
| 匈纳里克 | 477-484 | 盖塞里克之子，延续北非王国统治。 |
| 冈塔蒙德、特拉萨蒙德 | 484-523 | 王国维持北非和西地中海影响。 |
| 希尔德里克 | 523-530 | 与拜占庭关系较缓和，后被盖利默推翻。 |
| 盖利默 | 530-534 | 末代国王，败于拜占庭将领贝利撒留。 |

## 说明

- 汪达尔人经高卢、伊比利亚进入北非。
- 439年占领迦太基后，掌握西罗马重要粮仓和地中海航运据点。
- 汪达尔海军曾威胁意大利和西地中海沿岸。
- 455年汪达尔军洗劫罗马，是西罗马衰亡象征性事件之一。
- 534年，拜占庭将领贝利撒留征服汪达尔王国。
