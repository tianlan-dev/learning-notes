# 东哥特王国

## 时间

493年-553年

## 概括

东哥特王国是东哥特人在意大利建立的后罗马日耳曼政权。狄奥多里克统治时期尽量维持罗马行政传统和意大利城市秩序，后在拜占庭查士丁尼的再征服战争中灭亡。

## 王朝世系 / 统治结构

| 阶段 / 君主 | 时间 | 说明 |
|---|---|---|
| 狄奥多里克大王 | 493-526 | 击败奥多亚塞，建立东哥特意大利王国，维持罗马行政传统。 |
| 阿塔拉里克与阿玛拉孙塔摄政 | 526-534/535 | 王室继承不稳，拜占庭介入意大利事务的机会增加。 |
| 狄奥达哈德 | 534-536 | 统治危机加深，查士丁尼发动哥特战争。 |
| 维蒂吉斯、托提拉等哥特战争时期诸王 | 536-552 | 与拜占庭长期战争，王国军事消耗严重。 |
| 提亚 | 552-553 | 末期国王，东哥特王国在拜占庭再征服中灭亡。 |

## 说明

- 493年，狄奥多里克击败奥多亚塞，控制意大利。
- 王国在军事上由东哥特统治集团主导，行政上大量沿用罗马官僚和法律传统。
- 拉文纳是重要政治中心。
- 535年以后，拜占庭发动哥特战争。
- 553年前后，东哥特王国在拜占庭再征服中灭亡。
