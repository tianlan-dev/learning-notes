# 伦巴德王国

## 时间

568年-774年

## 概括

伦巴德王国是伦巴德人在意大利北部建立的日耳曼王国。它与拜占庭残余势力、罗马教皇和意大利地方贵族长期并存，774年被查理曼征服，成为法兰克和加洛林意大利秩序的一部分。

## 王朝世系 / 统治结构

| 阶段 / 君主 | 时间 | 说明 |
|---|---|---|
| 阿尔博因 | 568-572 | 率伦巴德人进入意大利，开启伦巴德王国。 |
| 诸公爵时期 | 574-584 | 王权中断，各地伦巴德公爵割据。 |
| 奥塔里、阿吉卢尔夫等 | 584-7世纪初 | 王权恢复，伦巴德政权逐渐稳定。 |
| 李乌特普兰德 | 712-744 | 伦巴德王国强盛阶段，扩张并影响教皇政治。 |
| 艾斯图尔夫 | 749-756 | 夺取拉文纳，引发教皇转向法兰克求援。 |
| 德西德里乌斯 | 756-774 | 末代国王，被查理曼击败，王国并入加洛林秩序。 |

## 说明

- 568年，阿尔博因率伦巴德人进入意大利。
- 伦巴德王国以意大利北部为核心，帕维亚是重要中心。
- 意大利半岛同时存在伦巴德、拜占庭、教皇和地方势力。
- 伦巴德压力促使教皇更多寻求法兰克保护。
- 774年，查理曼征服伦巴德王国并兼任伦巴德国王。
