# 中法兰克王国

## 时间

843年-855年；后续洛林、勃艮第、意大利方向继续分化

## 概括

中法兰克王国是843年《凡尔登条约》后洛泰尔一世所得的中部王国，包含从北海、洛林、勃艮第到意大利北部的狭长区域。855年《普吕姆条约》后分为洛泰尔二世的洛塔林吉亚、路易二世的意大利王国和查理的普罗旺斯 / 勃艮第方向，成为中世纪西欧边界争夺最复杂的地带之一。

## 王朝世系 / 统治结构

| 阶段 | 时间 | 说明 |
|---|---|---|
| 洛泰尔一世统治 | 843-855 | 兼具皇帝称号，控制帝国中部地带。 |
| 洛塔林吉亚 | 855年后 | 位于西、东法兰克之间，长期成为争夺地带。 |
| 意大利王国方向 | 855年后 | 与皇帝称号、罗马教廷和北意大利权力格局相连。 |
| 勃艮第 / 普罗旺斯方向 | 855年后 | 后续分化为上勃艮第、下勃艮第等区域政治。 |

## 重要事件

- 843年，《凡尔登条约》形成中法兰克王国。
- 855年，《普吕姆条约》进一步分割洛泰尔一世领地。
- 9-10世纪，洛林、勃艮第和意大利方向分别卷入西法兰克、东法兰克和地方贵族竞争。
