# 勃艮第王国

## 时间

5世纪-534年

## 概括

勃艮第王国是勃艮第人在罗讷河流域建立的后罗马日耳曼政权。它位于高卢东南部和阿尔卑斯西侧，是法兰克、意大利和日耳曼方向之间的重要中间地带，534年被法兰克王国并入。

## 王朝世系 / 统治结构

| 阶段 / 君主 | 时间 | 说明 |
|---|---|---|
| 早期莱茵河流域勃艮第政权 | 5世纪前期 | 勃艮第人先在莱茵河流域活动，后转入罗讷河流域。 |
| 贡迪奥克及其后裔 | 5世纪中后期 | 在高卢东南部建立较稳定的勃艮第王国。 |
| 贡多巴德 | 473-516 | 勃艮第王国重要国王，整理《勃艮第法典》。 |
| 西吉斯蒙德、贡多马尔等末期诸王 | 516-534 | 面对法兰克压力，534年王国被法兰克并入。 |

## 说明

- 勃艮第人先后在莱茵河流域和罗讷河流域活动。
- 王国位于后来的勃艮第、萨伏依和罗讷河谷相关区域。
- 其政治文化同时受日耳曼军事集团、罗马地方社会和基督教影响。
- 534年被法兰克王国征服，但“勃艮第”作为地区和政治名称长期保留。
- 勃艮第王国的早期政权史放在后罗马日耳曼诸国中理解；后续“勃艮第”作为法国东部地区、勃艮第公国及法兰西王权整合的一部分，更多归入[法国历史](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2-%E5%A4%96%E5%9B%BD/%E6%AC%A7%E6%B4%B2/%E6%B3%95%E5%9B%BD/README.md)主线。
