# 西哥特王国

## 时间

418年-711年

## 概括

西哥特王国是西哥特人在西罗马晚期和后罗马时代建立的日耳曼王国。早期中心在高卢南部图卢兹，507年武耶战役后重心转向伊比利亚半岛，以托莱多为核心，711年在穆斯林征服中崩溃。

## 王朝世系 / 统治结构

| 阶段 / 君主 | 时间 | 说明 |
|---|---|---|
| 阿陶尔夫等早期西哥特首领 | 5世纪初 | 西哥特集团进入西罗马境内，在高卢南部形成同盟者政权基础。 |
| 图卢兹王国 | 418-507 | 以图卢兹为中心，控制高卢南部和伊比利亚部分地区。 |
| 阿拉里克二世 | 484-507 | 在武耶战役中败于克洛维，西哥特重心转向伊比利亚。 |
| 托莱多王国 | 6世纪-711 | 以托莱多为中心整合伊比利亚，逐渐形成西哥特西班牙主线。 |
| 罗德里克 | 710/711 | 末期国王，711年后王国在穆斯林征服中崩溃。 |

## 说明

- 早期作为罗马的同盟者进入帝国境内，逐渐在阿基坦和高卢南部形成政权。
- 图卢兹时期同时受罗马政治传统和日耳曼军事集团影响。
- 克洛维率法兰克人在507年武耶战役击败西哥特人后，西哥特势力主要退向伊比利亚。
- 托莱多时期逐渐整合伊比利亚半岛多数地区。
- 711年以后，倭马亚势力进入伊比利亚，西哥特王国瓦解。
