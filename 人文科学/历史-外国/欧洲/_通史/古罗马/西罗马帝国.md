# 西罗马帝国

## 时间

395年-476年

## 别称

Western Roman Empire

## 概括

罗马帝国西部在日耳曼迁徙、财政军事困境和军人政治中衰亡，476年西部皇帝被废。

## 统治结构 / 主要阶段

西罗马帝国是罗马帝国西部行政和皇权体系，在五世纪的财政、军事和迁徙压力中崩溃。

| 阶段 | 时间 | 身份 / 关系 | 说明 |
|---|---|---|---|
| 霍诺里乌斯时期 | 395-423 | 狄奥多西一世之子 | 西部帝国行政中心转向拉文纳，边境压力加剧。 |
| 蛮族入侵与军人政治 | 5世纪前中期 | 西哥特、汪达尔、匈人等 | 罗马军队和日耳曼将领影响朝政。 |
| 末期傀儡皇帝 | 455-476 | 多位短命皇帝 | 西部皇权依赖军事首领。 |
| 奥多亚克统治 | 476以后 | 日耳曼军政首领 | 废黜西罗马末帝，名义承认东罗马皇帝。 |

### 西部皇帝世系概览

| 顺序 / 皇帝或集团 | 在位时间 | 关系 / 权力基础 | 关键事件 / 说明 |
|---|---|---|---|
| 霍诺里乌斯 | 395-423 | 狄奥多西一世之子 | 西部长期由斯提里科等将领支撑，410年罗马被西哥特洗劫。 |
| 君士坦提乌斯三世、瓦伦丁尼安三世 | 421；425-455 | 狄奥多西王朝西部支系 | 埃提乌斯长期掌军，451年抵御匈人。 |
| 佩特罗尼乌斯·马克西穆斯、阿维图斯 | 455-456 | 贵族与军队拥立 | 汪达尔洗劫罗马后西部皇权更加不稳。 |
| 马约里安 | 457-461 | 里西默拥立后试图恢复西部 | 末期少数有实权的西部皇帝。 |
| 利比乌斯·塞维鲁、安特米乌斯、奥利布里乌斯 | 461-472 | 军人首领和东罗马影响下的皇帝 | 皇权受里西默等军政强人控制。 |
| 格利凯里乌斯、朱利乌斯·尼波斯 | 473-475；474-480 | 西部末期竞争皇帝 | 尼波斯获东罗马承认，后在达尔马提亚维持名义皇权。 |
| **罗慕路斯·奥古斯都** | 475-476 | 奥瑞斯特斯之子，西部末帝 | 被奥多亚克废黜，传统上作为西罗马灭亡标志。 |
| 奥多亚克 | 476-493 | 意大利军政首领 | 不再设置西部皇帝，名义承认东罗马皇帝。 |

## 重要事件

- 410年，西哥特人攻陷罗马。
- 451年，阿提拉和匈人在高卢被阻击。
- 455年，汪达尔人洗劫罗马。
- 476年，奥多亚克废黜罗慕路斯·奥古斯都。
