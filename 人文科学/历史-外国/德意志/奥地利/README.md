# 奥地利历史

奥地利是德意志世界的重要分支，长期处在神圣罗马帝国和德意志邦联的核心位置。1866年普奥战争后，奥地利被排除出普鲁士主导的德意志统一进程，随后转向奥匈帝国和现代奥地利共和国的发展路径。
