# 日本历史

这页是 `人文科学/历史-外国/日本` 目录的 README 导览页。详细说明已拆分到同目录下的各时代笔记；本页只保留演变总图、按时间顺序排列的导航、统治者世系入口，以及原总览中的中日时期对照。

## 历史演进流程图

```mermaid
flowchart TD
    paleolithic["旧石器时代<br/>约前14000年以前"] --> jomon["绳文时代<br/>约前14000年-前5/前3世纪"]

    yayoi --> kofun["古坟时代<br/>3世纪中期-7世纪初"]
    kofun --> asuka["飞鸟时代<br/>593-710"]
    asuka --> nara["奈良时代<br/>710-794"]
    nara --> heian["平安时代<br/>794-1185"]
    heian --> kamakura["镰仓时代<br/>1185-1333"]
    kamakura --> nanboku["南北朝时期<br/>1336-1392"]
    nanboku --> muromachi["室町时代<br/>1336-1573"]
    muromachi --> sengoku["战国时代<br/>1493-1590"]
    sengoku --> azuchi["安土桃山时代<br/>1573-1603"]
    azuchi --> edo["江户时代<br/>1603-1868"]
    edo --> meiji["明治时代<br/>1868-1912"]

```
