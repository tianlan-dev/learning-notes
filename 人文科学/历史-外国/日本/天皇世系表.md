# 天皇世系表

## 概括

本表按日本传统皇统编号整理历代天皇。早期天皇的年代多来自古代史书传统，史实性和年代可靠性与后世不同；南北朝时期以南朝为正统编号，北朝天皇另列附表。

## 南北朝时期北朝天皇

北朝天皇不列入传统正统编号，但在室町幕府政治秩序中具有重要地位。

| 顺序 | 北朝天皇 | 在位时间 | 说明 |

| 北1 | 光严天皇 | 1331-1333 | 镰仓幕府拥立，建武新政后被废。 |
| 北2 | 光明天皇 | 1336-1348 | 足利尊氏拥立，北朝正式展开。 |
| 北3 | 崇光天皇 | 1348-1351 | 观应扰乱中退位。 |
| 北4 | 后光严天皇 | 1352-1371 | 北朝重建后的天皇。 |
| 北5 | 后圆融天皇 | 1371-1382 | 北朝后期天皇。 |
| 北6 | 后小松天皇 | 1382-1392 | 北朝末代天皇；1392年南北朝合一后列为第100代天皇。 |
