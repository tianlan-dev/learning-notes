# 北亚自然地理、考古与早期人口

## 时间

旧石器时代至早期历史时期

## 概括

北亚由西西伯利亚平原、中西伯利亚高原、南西伯利亚山地、勒拿河与黑龙江流域、远东海岸、苔原和北极群岛等不同环境组成。冰期与间冰期、永久冻土、河流和草原森林带共同影响人口移动与生计。


## 关键线索

| 线索 | 说明 |
|---|---|
| 旧石器时代遗址 | 雅纳河、马尔他—布列季等遗址说明人群很早适应寒冷环境，但遗址人群不能直接等同现代民族。 |
| 白令陆桥 | 冰期海平面下降连接东北亚与阿拉斯加，是理解美洲早期人口的重要地理背景。 |
| 南西伯利亚 | 草原、山地和森林交界促进冶金、畜牧与跨欧亚交流。 |
| 大河体系 | 冬季冰道和夏季水运使鄂毕河、叶尼塞河、勒拿河和阿穆尔河成为交通轴线。 |
| 语言多样性 | 乌拉尔语系、突厥语族、蒙古语族、通古斯语族以及楚科奇—堪察加、叶尼塞等语言群并存。 |

## 关键辨析

- 考古文化、遗传群体、语言共同体和现代民族是不同概念。
- 北亚不是单一迁徙通道；人口也会停留、回流、融合和区域分化。
- “古西伯利亚语言”等名称多为方便归类的集合称呼，不表示一个已证实的共同语系。
