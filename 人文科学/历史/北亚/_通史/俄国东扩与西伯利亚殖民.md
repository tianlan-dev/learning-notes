# 俄国东扩与西伯利亚殖民

## 时间

16世纪中叶—19世纪末；边疆开发和殖民关系延续至20世纪。

## 概括

俄国越过乌拉尔山脉的扩张，不是一次由中央军队完成的征服，而是国家特许、商人投资、哥萨克远征、毛皮猎取、城堡驻防、农民移居、罪犯流放和东正教传教交错推进的长期过程。早期核心动力是毛皮及贡貂制度；鄂毕河、叶尼塞河、勒拿河和阿穆尔河等水系提供了快速移动的通道。俄国队伍在约一个世纪内从乌拉尔抵达太平洋，但地图上的占领远早于稳定行政、人口优势和日常控制。

扩张面对的不是“无人荒原”，而是西伯利亚汗国、森林与苔原诸民族、南西伯利亚草原集团以及阿穆尔河流域多种政治网络。地方社会既有武装抵抗，也会迁徙避贡、与俄国结盟、利用俄清竞争或借贸易争取利益。因而，俄国东扩同时是一部国家形成、商业扩张和定居殖民史，也是一部原住民族持续调整、承受灾难并保存主体性的历史。

## 建立背景与扩张条件

| 条件 | 具体表现 | 历史作用 |
|---|---|---|
| 伏尔加方向扩张 | 莫斯科国家先后征服喀山汗国和阿斯特拉罕汗国 | 削弱西部草原强权，扩大国家、商人和军役群体向乌拉尔活动的空间 |
| 毛皮市场 | 黑貂等优质毛皮在欧洲、奥斯曼和中国市场价值很高 | 为私人远征、国家征贡和边疆驻军提供财政动力 |
| 斯特罗加诺夫家族特许 | 商人获得乌拉尔一带土地、筑堡、招募武装和经营盐业的权利 | 把国家名义主权与私人资本、地方军事行动结合起来 |
| 哥萨克军事文化 | 小型武装队伍善于水路移动、筑寨、火器作战和边疆谋生 | 便于以有限人力深入大河流域，但纪律和国家服从程度并不稳定 |
| 河流交通 | 主要河流及支流构成可连续转运的水路网 | 降低跨越森林和冻土的运输成本，使据点能沿水系逐段前推 |
| 原有区域网络 | 汗国、部落联盟和贡赋贸易早已连接草原、森林与中亚 | 俄国往往接管、改造而非从零创造征贡和交换渠道 |
| 火器与堡垒 | 火绳枪、火炮和木制城堡形成局部优势 | 能抵御人数较多的围攻，却不能消除补给脆弱和地方反抗 |

### 西伯利亚汗国的瓦解

西伯利亚汗国以额尔齐斯河和托博尔河流域为中心，统治集团与金帐汗国、诺盖和中亚世界有历史联系，并向周边汉特、曼西等群体征取贡物。库楚姆汗掌权后试图巩固汗国，并与向东扩张的俄国势力发生冲突。

约1581—1582年，叶尔马克率哥萨克队伍越过乌拉尔。队伍得到斯特罗加诺夫家族支持，但并非一支编制完整的国家正规军。火器、战术机动以及汗国内部关系使其取得初期优势；1582年的楚瓦什角战斗后，哥萨克占领汗国都城卡什雷克。部分地方首领转而向沙皇纳贡，另一些继续支持库楚姆。

这不是汗国的即时终结。补给困难、人员不足和地方反击长期威胁俄军，叶尔马克于1585年在一次袭击中身亡。莫斯科随后直接派遣军队并修筑秋明、托博尔斯克等据点，才把一次冒险远征转化为持续扩张。库楚姆及其支持者继续抵抗，直到16世纪末其主要军事力量被击溃；汗国政治网络则以不同形式在地方社会中延续。

### 沿大河抵达太平洋

俄国据点先控制西西伯利亚水系，再向叶尼塞河和勒拿河推进。秋明建于1586年，托博尔斯克建于1587年，托木斯克建于1604年；叶尼塞斯克、克拉斯诺亚尔斯克和雅库茨克等城堡随后成为征贡、贸易、军事和司法中心。小队伍会在冬季驻留，次年沿支流侦察并要求新接触的群体“归顺”沙皇。

1630年代，俄国队伍以雅库茨克为基地向鄂霍次克海和东北方向活动，1639年到达太平洋沿岸。谢苗·杰日尼奥夫等人在1648年航经楚科奇半岛和白令海峡一带，说明欧亚大陆与北美之间存在海峡；其航行当时没有立即形成稳定的科学和商业成果，18世纪白令探险后这一路线才更系统地进入帝国知识体系。

### 阿穆尔河、清朝与南方边界

17世纪中叶，波雅尔科夫、哈巴罗夫等俄国远征者进入阿穆尔河流域。这里已有达斡尔、杜切尔、赫哲等社会，并处于清朝征贡、驻防与地方联盟网络中。俄国征粮、征贡和暴力行动引发人口迁移与抵抗，也把扩张带入与清朝的直接冲突。

俄国人在雅克萨等地筑堡，清军经过围攻迫使其谈判。1689年《尼布楚条约》划定部分边界，俄国撤出雅克萨并暂时放弃阿穆尔河中上游的据点。条约表明，俄国东扩并非所到之处都能永久占领；当面对组织和后勤能力相当或更强的帝国时，外交边界会约束军事前进。19世纪俄国才在新的国际条件下重新取得黑龙江以北和乌苏里江以东地区，详见[清俄边疆、东北亚与北太平洋联系](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E5%8C%97%E4%BA%9A/_%E9%80%9A%E5%8F%B2/%E6%B8%85%E4%BF%84%E8%BE%B9%E7%96%86%E3%80%81%E4%B8%9C%E5%8C%97%E4%BA%9A%E4%B8%8E%E5%8C%97%E5%A4%AA%E5%B9%B3%E6%B4%8B%E8%81%94%E7%B3%BB.md)。

### 堪察加、楚科奇与北太平洋

17世纪末以后，弗拉基米尔·阿特拉索夫等远征者推动俄国进入堪察加。当地伊捷尔缅、科里亚克等群体受到征贡、暴力和疫病冲击，也反复发动抵抗。楚科奇人凭借分散政治结构、机动作战和跨白令海贸易长期抵抗俄军；18世纪中叶多次军事行动成本高昂，帝国最终更多依靠贸易和名义臣属关系，而非完全军事征服。

堪察加和鄂霍次克成为进入阿留申群岛及阿拉斯加的基地。毛皮猎人和商人把西伯利亚的贡赋、债务和强迫劳动方式带入北太平洋，同时依赖阿留申等原住民的航海和海兽知识。这一跨海殖民网络后来发展为俄属美洲，但其人力、粮食和运输始终受距离限制。

## 殖民统治结构

| 机构或群体 | 权力与职责 | 实际局限 |
|---|---|---|
| 西伯利亚衙门及中央机构 | 任命官员、制定贡赋和边贸政策、处理军政与司法 | 距离遥远，信息滞后，难以约束所有地方人员 |
| 总督或地方军事长官 | 管理城堡、驻军、税贡、审判与外交 | 依赖少量军役人员和地方翻译，常受补给及贪腐影响 |
| 哥萨克 | 侦察、征贡、护送、驻防和镇压 | 可脱离命令私自贸易或施暴，国家需以俸给和赦免维持效忠 |
| 毛皮商与猎人 | 组织私人猎取、债务贸易和远征 | 与官员争夺毛皮，也高度依赖原住民劳动和知识 |
| 农民与城镇居民 | 农业、手工业、运输、粮食和市场供应 | 主要集中在较适合农业的南部及河谷，北部密度很低 |
| 流放者与苦役犯 | 服刑、定居、采矿、修路及行政辅助 | 身份和处境差异很大，不能与自愿移民合并理解 |
| 东正教传教士 | 洗礼、教堂、学校、翻译和文化传播 | 皈依可能出于信仰、税赋优惠或政治压力，地方传统没有因此消失 |
| 原住民首领与翻译 | 协商贡额、组织交换、调解或领导抵抗 | 可能借帝国认可强化权力，也可能被竞争者挑战 |

## 贡貂制度与地方社会

“贡貂”通常译自俄语“亚萨克”，主要以黑貂等毛皮缴纳。它具有税收和主权宣示双重含义：官员把纳贡登记视为臣服沙皇的证据，许多地方群体却可能把它理解为换取礼物、贸易机会或军事保护的互惠关系。双方对关系性质的不同理解经常引发冲突。

征收者会登记成年男性、指定贡额，并扣留首领亲属作为人质以保证缴纳。贡物往往伴随沙皇礼物或市场交换，但官员、商人私征、重复征贡和以暴力压价十分常见。一些群体迁往偏远支流、改变狩猎路线或同时向多个政权纳贡；另一些通过首领谈判降低贡额，或利用火器、铁器和粮食贸易增强自身地位。

高强度捕猎使部分地区黑貂资源减少，迫使征贡网络继续向东移动，也令地方生计更依赖市场。毛皮因此既是扩张的财政基础，又制造了资源耗竭和进一步扩张的压力。

## 重要事件

| 时间 | 事件 | 过程与影响 |
|---|---|---|
| 1552年、1556年 | 喀山与阿斯特拉罕被征服 | 莫斯科国家扩展至伏尔加中下游，为进入乌拉尔和西伯利亚创造条件 |
| 约1581—1585年 | 叶尔马克远征 | 击败汗国主力并占领卡什雷克，但叶尔马克战死，征服需由国家筑堡接续 |
| 1586—1587年 | 秋明、托博尔斯克建立 | 俄国形成西西伯利亚稳定驻防和行政支点 |
| 1598年 | 库楚姆主力败退 | 西伯利亚汗国作为区域霸权瓦解，地方反抗和汗族政治并未立即消失 |
| 1604—1632年 | 托木斯克至雅库茨克的据点链形成 | 扩张越过鄂毕—叶尼塞水系进入勒拿河流域 |
| 1639年 | 俄国队伍到达鄂霍次克海 | 国家势力首次横贯北亚抵达太平洋 |
| 1640—1650年代 | 阿穆尔河远征与冲突 | 俄国队伍征粮征贡，引发当地抵抗并与清朝势力相遇 |
| 1685—1686年 | 清军围攻雅克萨 | 证明俄国在阿穆尔河据点难以单靠军事维持 |
| 1689年 | 《尼布楚条约》 | 俄国撤出雅克萨，清俄以条约调整边界和交往 |
| 1690年代以后 | 堪察加征服展开 | 俄国获得北太平洋前进基地，当地社会经历长期战争、疫病与贡赋 |
| 18世纪中叶 | 对楚科奇战争转向协商 | 高昂军事成本迫使帝国以贸易和松散臣属替代直接征服 |
| 19世纪 | 移民、流放和行政整合扩大 | 从毛皮边疆逐渐转为农业、采矿、交通和定居殖民并重 |

## 扩张成功的机制

- **低成本的分段推进**：小队先建城堡和索贡，国家机构随后接管，无需一开始维持大规模常备军。
- **利用既有网络**：俄国接入原有贡赋、商路和地方首领体系，降低了全面重建行政的成本。
- **毛皮财政激励**：高价值、低体积的毛皮适合长距离运输，能支持远征和边疆官僚。
- **河流与季节交通**：夏季水运和冬季冰道使人员、粮食和讯息得以沿流域移动。
- **多种人口接续**：哥萨克之后有官员、商人、农民、教士和流放者，使临时据点逐步转为城镇。
- **地方分散与竞争**：许多地区不存在能统一调动全域资源的国家，俄国可分别结盟、征贡或攻击；但在清朝和楚科奇等案例中这一优势明显受限。

## 原住民抵抗、协商与殖民后果

武装抵抗包括攻击征贡队、围攻城堡、拒交贡物和结成联盟；非武装策略则包括迁徙、隐瞒人口和猎获、利用不同俄国官员间竞争、接受名义臣属但拒绝日常控制。部分首领借俄国武器和承认对付地方对手，部分社区则通过通婚、翻译和贸易形成新的边疆社会。

殖民冲击具有明显地区差异，但常见后果包括：

- 天花等传染病造成严重人口损失，具体规模因记录不完整而难以精确统计。
- 暴力征贡、人质和报复行动破坏地方政治与家庭生活。
- 商业捕猎和债务关系改变资源使用，某些动物数量显著下降。
- 农业移民占用南部草场、渔场和河谷，原住民活动空间被压缩。
- 东正教、俄语和帝国学校改变身份表达，但地方语言、宗教和习惯法持续存在。
- 通婚和文化互借形成俄罗斯老住民、混合家庭和多语社区，不能把边疆社会简单分为彼此隔绝的“殖民者”与“被殖民者”。

## 从征服边疆到定居殖民

到18—19世纪，西伯利亚的意义逐渐超出毛皮。矿山、冶炼、农业、城市市场、交通和刑罚流放扩大，南西伯利亚与远东的军事战略价值上升。19世纪末西伯利亚铁路把大规模移民、商品运输和国家军事能力连接起来，帝国对土地和人口的管理显著加深。

然而，人口和行政密度始终不均：铁路沿线、农业南部和矿区与北极、森林腹地存在巨大差异。扩张“完成”更多是国家疆域叙事，不能表示所有地方社会已经被同样程度地俄罗斯化或纳入市场。其后续工业化、强制迁徙和民族政策见[苏联开发、人口迁徙与当代北亚](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E5%8C%97%E4%BA%9A/_%E9%80%9A%E5%8F%B2/%E8%8B%8F%E8%81%94%E5%BC%80%E5%8F%91%E3%80%81%E4%BA%BA%E5%8F%A3%E8%BF%81%E5%BE%99%E4%B8%8E%E5%BD%93%E4%BB%A3%E5%8C%97%E4%BA%9A.md)。

## 关键辨析

- **“横贯西伯利亚”不等于“完全控制西伯利亚”**：早期据点之间存在广阔、难以日常干预的区域。
- **哥萨克不是单一民族**：它在此主要指承担军事服务、远征和边疆谋生的社群与队伍。
- **纳贡不总等于双方理解一致的臣服**：原住民、俄国官员和邻近帝国对同一行为可能有不同政治解释。
- **殖民并非只有海外形式**：跨越连续陆地的征服、土地占用、资源抽取和文化等级化同样具有殖民性质。
- **原住民不是被动背景**：帝国路线、贸易方式和边界结果都受到地方协商、知识和抵抗影响。
