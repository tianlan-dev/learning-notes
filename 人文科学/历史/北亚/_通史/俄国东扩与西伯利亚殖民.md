# 俄国东扩与西伯利亚殖民

## 时间

16世纪后期至19世纪

## 概括

俄国越过乌拉尔山脉的扩张由国家、商人、斯特罗加诺夫家族、哥萨克队伍、农民和传教者共同推动。毛皮利益和贡貂制度是早期动力，河流城堡网络使俄国势力在约一个世纪内抵达太平洋，但有效控制、移民密度和原住民回应因地区而异。

## 统治机制

| 机制 | 说明 |
|---|---|
| 贡貂 | 要求原住民以毛皮缴纳贡赋，常伴随人质、谈判和武力。 |
| 城堡与河运 | 秋明、托博尔斯克、克拉斯诺亚尔斯克、雅库茨克等据点控制交通和征税。 |
| 商业 | 毛皮商、私人远征和国家专卖把西伯利亚连接到欧洲与中国市场。 |
| 移民与农业 | 农民、逃亡者、流放者和官兵建立定居点，南部农业区扩张较快。 |
| 传教与行政 | 东正教传教、洗礼和地方行政逐步进入原住民社会，但覆盖不均。 |
| 暴力与疾病 | 镇压、人口迁移、疫病和资源竞争造成严重冲击。 |

## 重要节点

| 时间 | 节点 | 说明 |
|---|---|---|
| 1580年代 | 叶尔马克进攻西伯利亚汗国 | 不是一次完成征服，而是俄国长期东扩的象征性开端。 |
| 1632年 | 雅库茨克建立 | 成为勒拿河流域扩张的重要基地。 |
| 1640年代 | 俄国队伍进入阿穆尔河流域 | 与当地社会和清朝势力发生冲突。 |
| 1689年 | 《尼布楚条约》 | 清俄划定部分边界并调整俄国在阿穆尔河上游的据点。 |
| 18世纪 | 堪察加与北太平洋探险 | 俄国进入白令海、阿拉斯加和北太平洋毛皮网络。 |

## 关键辨析

- 俄国东扩不是单一中央军队预先规划的直线行动。
- 条约中的国家边界与当地实际控制、族群活动范围并不完全重合。
- “哥萨克”在不同地区可指军事服务群体、边疆社群或远征队伍，不能视为单一民族。
