# 西伯利亚与俄罗斯远东

## 范围与概括

本入口连接乌拉尔山脉以东至太平洋的区域历史。“西伯利亚”通常涵盖西西伯利亚、中西伯利亚、南西伯利亚及东西伯利亚等广阔空间；“俄罗斯远东”主要是近现代行政和战略概念，面向阿穆尔河、鄂霍次克海、日本海和白令海。两者有重叠，但不是同义词：雅库特常同时进入西伯利亚与远东叙事，西西伯利亚则通常不属于俄罗斯远东。

区域主线不能缩减为“俄国征服后才有历史”。西伯利亚汗国、草原与森林贡赋、阿穆尔河社会、北极海岸及原住民族网络都早于俄国国家到来。16—17世纪东扩把这些体系接入沙皇贡貂和堡垒行政；19世纪铁路、移民及阿穆尔河边界重划形成新的城市和港口；苏联工业化、劳改营、民族政策、军工与人口迁徙又深刻重组区域。1991年以来，资源经济、人口收缩、面向亚洲的贸易和气候变化成为共同主题。

## 历史主线概括

1. **早期人口与环境适应**：阿尔泰、南西伯利亚、北极圈和远东海岸保留古人类及现代人类活动证据，北亚不是单一迁徙通道。
2. **地方社会与跨生态网络**：草原牲畜、森林毛皮、大河鱼类、北极驯鹿和太平洋海兽构成相互依赖的交换系统。
3. **多层政治关系**：西伯利亚汗国、蒙古诸部、清朝贡赏和地方首领体系形成重叠权力，现代领土国家不能直接套用。
4. **俄国殖民扩张**：哥萨克、商人、官员和移民依靠河流、贡貂与堡垒推进，地方社会以抵抗、迁徙、贸易和协商回应。
5. **帝国整合**：铁路、流放、农业移民、采矿和太平洋港口提高国家控制，阿穆尔河与滨海边界在19世纪中叶重新划定。
6. **苏联重组**：计划经济把资源城市、交通、学校、医疗和军工扩展到偏远地区，同时伴随集体化、强制迁徙、劳改和环境损害。
7. **当代分化**：大城市、油气矿区和港口仍有战略投资，许多小城与村落人口外流；对华及亚太贸易、北方航路和冻土风险日益重要。

## 区域分区

| 历史空间 | 大致范围 | 核心线索 | 易混点 |
|---|---|---|---|
| 西西伯利亚 | 乌拉尔以东至叶尼塞河一带 | 西伯利亚汗国、鄂毕—额尔齐斯河、农业南部、油气区 | 帝国时期行政“西伯利亚”范围曾变化 |
| 中西伯利亚 | 叶尼塞河至勒拿河之间 | 针叶林、河运、原住民族、矿冶与诺里尔斯克等工业 | 人口和城市高度集中于少数节点 |
| 南西伯利亚 | 阿尔泰、萨彦、贝加尔周边及森林草原 | 突厥蒙古世界、牧业、冶金、铁路和民族共和国 | 与中亚、蒙古高原界线具有历史流动性 |
| 东西伯利亚 | 勒拿河、外贝加尔及邻近地区 | 萨哈社会、驯鹿、清俄边疆、矿产和冻土 | 日常用法与联邦经济区划不完全一致 |
| 阿穆尔河与滨海地区 | 黑龙江 / 阿穆尔河下游、乌苏里江及日本海沿岸 | 清俄条约、原住民族、移民、海参崴和东北亚贸易 | 河流既是交通轴，也是19世纪后形成的国界 |
| 堪察加与楚科奇 | 东北半岛和白令海沿岸 | 原住民抵抗、海兽、跨白令海、探险与北太平洋 | 国家边界晚于跨海亲族和贸易 |
| 萨哈林与千岛方向 | 萨哈林、邻海及群岛联系 | 尼夫赫、阿伊努等社会，俄日划界、渔业和战争 | 主权变化不能代替当地人口史 |
| 俄罗斯远东 | 近现代面向太平洋的广义行政战略区 | 港口、铁路、军工、资源、人口与亚洲联系 | 不等于全部西伯利亚，也不只指海岸 |

## 重要转折与时间节点

| 时间 | 转折 | 区域影响 |
|---|---|---|
| 约4.5万年前以后 | 现代人类深入北亚 | 多种寒地和河海适应逐步形成 |
| 前4—前1千纪 | 南西伯利亚牧业、冶金和草原网络扩大 | 北亚更深连接中亚、蒙古高原、东欧和中国北方 |
| 13—16世纪 | 蒙古及后继汗国、清朝前身和地方政权互动 | 形成俄国东扩前的多层贡赋与贸易秩序 |
| 约1581—1598年 | 西伯利亚汗国遭进攻并瓦解 | 俄国国家据点取代汗国区域霸权 |
| 1630—1650年代 | 俄国抵达太平洋并进入阿穆尔河 | 扩张横贯大陆，随后与清朝发生直接冲突 |
| 1689年 | 《尼布楚条约》 | 俄国南进受限，清俄以条约调整边界 |
| 1858—1860年 | 阿穆尔河与滨海边界重划 | 俄罗斯远东现代领土和港口体系形成 |
| 1891—1916年 | 西伯利亚铁路体系建设 | 大规模军运、移民和市场整合加速 |
| 1917—1922年 | 革命、内战与远东共和国 | 多方争夺后区域纳入苏维埃国家 |
| 1930年代 | 集体化、工业化、劳改与强制迁徙 | 国家控制深入生产和人口，社会代价巨大 |
| 1960—1980年代 | 油气、矿业、军工和贝阿铁路扩张 | 北亚成为苏联能源、原料与战略基地 |
| 1991—2026年 | 市场转型、人口分化和亚洲联系增强 | 资源出口仍重要，气候和地缘政治风险上升 |

## 关键辨析

- 西伯利亚是历史地理概念，俄罗斯远东更多是近现代行政与战略概念。
- “远东”从俄罗斯欧洲方向命名；讨论地方史时宜使用阿穆尔河、楚科奇、堪察加等具体区域。
- 俄国队伍抵达太平洋不等于同时完成全境行政和人口控制。
- 城市、铁路沿线和矿区的俄罗斯化程度不能代表所有乡村、北极和原住民族地区。
- 苏联城市化和公共服务是真实变化，强制劳动、迁徙和环境成本也必须进入同一叙事。
- 当代国家开发计划和资源储量不等于已经建成的经济能力；“至今”以2026年7月为截点。
