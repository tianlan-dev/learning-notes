# 吉尔吉斯斯坦的天山社会、突厥与吉尔吉斯传统

## 时间

古代—18世纪

## 概括

今吉尔吉斯斯坦位于天山、楚河—塔拉斯河谷、伊塞克湖盆地与费尔干纳东缘之间。高山夏牧场、低地冬牧场、绿洲农业和商队城市彼此依存，因此本区长期处在草原联盟、绿洲王朝与山地部族网络的交界处。萨卡、乌孙、突厥、葛逻禄、叶尼塞吉尔吉斯、蒙古和蒙兀儿斯坦都参与了这片历史，却没有哪一个古代名称能够直接等同于现代吉尔吉斯民族或现代国家。

## 地理与社会机制

- **山地牧业**：部族家庭按季节在高山夏牧场、河谷春秋牧场与低地冬牧场之间移动。牧道、饮水点和越山通道既是经济资源，也是冲突与结盟的焦点。
- **绿洲与城镇**：楚河、塔拉斯和费尔干纳的农业、手工业及市场为牧民提供粮食、织物和金属器；牧区则供应牲畜、马匹、皮毛和交通保护。
- **联盟政治**：多数时期不存在固定疆界的统一山地国家。首领、勇士、宗族长老和宗教权威依靠亲缘、婚姻、战利品分配与仲裁维持追随者，联盟会随外敌、牧地和继承争端而重组。
- **文化层积**：突厥语扩展、伊斯兰化、蒙古政治词汇与地方习惯法分别发生在不同阶段；它们不是一次完成的“民族形成事件”。

## 分阶段发展

### 早期草原—绿洲社会

前1千纪，希腊、波斯和中国史料以“萨卡”等外部名称描述天山及邻近草原的多种群体。墓葬、金属器、马具与长距离交换显示骑乘牧业和绿洲农业并存。约前2世纪，乌孙势力以伊犁河谷为中心，并与汉朝、匈奴和大月氏竞争；伊塞克湖与楚河一带处于其活动和交通范围，但不能把乌孙直接认定为某一现代民族的单一祖先。

### 突厥诸汗国、城市与伊斯兰化

6世纪，突厥汗国把天山纳入横跨欧亚的可汗政治与商路体系。603年前后东西突厥分立后，西突厥、突骑施和葛逻禄先后争夺碎叶、塔拉斯与七河。统治依靠阿史那等王族法统、部落分封和对商路城市的控制，粟特商人与多宗教社群仍是城市生活的重要组成。

751年怛罗斯战役改变了唐、阿拔斯与地方盟友的力量关系，却没有让天山社会立即全面伊斯兰化。10世纪喀喇汗王朝的统治使费尔干纳、楚河和伊塞克湖周边的突厥统治集团更深地进入穆斯林城市、税收与学术网络；牧区的宗教转变则延续数百年。

### 叶尼塞吉尔吉斯与天山吉尔吉斯

中国、突厥与穆斯林文献中的叶尼塞吉尔吉斯主要活动在南西伯利亚叶尼塞河上游。840年，其军队击败回鹘汗国并进入蒙古高原，但没有建立长期控制整个蒙古草原和现代吉尔吉斯斯坦的稳定帝国。10世纪契丹势力进入蒙古高原后，叶尼塞吉尔吉斯的外部影响收缩。

1207年，术赤代表蒙古帝国征服叶尼塞地区诸部；随后反抗、迁徙和再征服使北方吉尔吉斯群体分散。与此同时，天山本地的突厥语群体长期处于察合台汗国及其后继网络之中。14—15世纪史料开始更清楚地记录天山吉尔吉斯；1469—1470年前后又有东部群体进入蒙兀儿斯坦天山地区。现代吉尔吉斯共同体由北方名称传统、天山既有群体和后续联盟共同塑造，具体迁徙比例与连续关系仍有争议。

### 蒙兀儿斯坦、准噶尔与18世纪格局

13世纪蒙古征服后，天山被纳入察合台兀鲁思。14世纪中叶蒙兀儿斯坦形成后，汗室需要依赖朵豁剌惕等贵族以及山地部族；吉尔吉斯群体时而臣属，时而反抗或脱离。16世纪史家已把吉尔吉斯描述为蒙兀儿斯坦北部的重要力量，东费尔干纳、伊塞克湖和帕米尔—阿赖各支的政治经历并不完全相同。

17世纪起，卫拉特—准噶尔扩张迫使天山多支吉尔吉斯迁徙、结盟或接受临时宗主关系。1703年，叶尼塞吉尔吉斯部分统治集团被准噶尔强迁至其控制区；北方残留群体又与哈卡斯、图瓦等群体发生融合。1755—1758年清朝摧毁准噶尔汗国后，吉尔吉斯首领向清朝遣使并建立边境交往，但清朝在多数吉尔吉斯牧区并未建立类似内地州县的日常直接行政。18世纪后半，浩罕势力由费尔干纳向山口和楚河推进，新的堡垒—税收关系逐渐形成。

## 重要事件

| 时间 | 事件 | 具体影响 |
|---|---|---|
| 约前2世纪 | 乌孙在伊犁—七河活动 | 天山进入汉、匈奴与中亚多边外交和交通网络 |
| 552年以后 | 突厥汗国扩张 | 可汗制、突厥语政治名称与商路保护体系覆盖天山 |
| 657—766年 | 西突厥衰落，突骑施和葛逻禄相继上升 | 七河权力多次重组，城市与部族联盟并存 |
| 751年 | 怛罗斯战役 | 唐朝在中亚的直接军事影响减弱，伊斯兰化仍以渐进方式推进 |
| 10世纪 | 喀喇汗王朝伊斯兰化 | 城市行政、穆斯林学术和突厥语王朝传统结合 |
| 840年 | 叶尼塞吉尔吉斯击败回鹘汗国 | 北方吉尔吉斯政治声望达到高点 |
| 1207年 | 叶尼塞诸部被蒙古征服 | 北方政治结构纳入蒙古兀鲁思，后续反抗造成重组 |
| 13世纪 | 天山纳入察合台兀鲁思 | 蒙古分封、驿路和税收体系覆盖山地—绿洲通道 |
| 14—15世纪 | 蒙兀儿斯坦形成，天山吉尔吉斯记录增多 | 吉尔吉斯逐渐成为天山北部重要联盟名称 |
| 1469—1470年前后 | 部分东部吉尔吉斯进入天山 | 推动天山共同体重组，但规模与路线存在史学争议 |
| 17—18世纪初 | 准噶尔扩张 | 牧地丧失、人口迁徙和跨区域联盟增加 |
| 1755—1758年 | 准噶尔汗国被清朝击败 | 吉尔吉斯首领与清建立外交，浩罕亦向天山扩张 |

## 统治结构与文化演变

| 阶段 | 主要权力载体 | 实际运作 |
|---|---|---|
| 早期草原社会 | 部族联盟、战士首领、绿洲城邦 | 依靠牧地、贡赋、交换与军事保护，边界随联盟变化 |
| 突厥诸汗国 | 可汗王族、左右翼分封、地方吐屯 | 王族法统连接部落军力，城市官员征税并保护商路 |
| 察合台—蒙兀儿斯坦 | 成吉思汗系汗、贵族家族、山地部族首领 | 汗室名义权威与地方军事实力并存，臣属关系频繁变化 |
| 17—18世纪吉尔吉斯社会 | 宗族首领、比伊、巴特尔与长老会议 | 通过习惯法仲裁、婚姻和临时军事联盟协调；没有稳定统一王统 |
| 伊斯兰网络 | 毛拉、苏菲传承、圣裔与地方社群 | 从绿洲向牧区渐进传播，并与既有习俗长期并存 |

## 兴起、分化与转折原因

### 形成与壮大的条件

- 天山牧道和山口让吉尔吉斯联盟能够在强大邻国之间保持机动，并吸纳战争、迁徙中重新组合的人群。
- 突厥语共同性、口述谱系和《玛纳斯》传统等文化资源，加强了不同支系的认同联系，但口述传统不能直接替代具体年代史。
- 准噶尔压力和蒙兀儿斯坦解体使原有权力空间松动，促成更大规模的防御联盟和群体重组。

### 结构性限制

- 季节迁徙、分散牧地与宗族自主使持续征税、常备军和固定首都难以建立。
- 北部、南部和帕米尔—阿赖各支面对的绿洲政权与商路不同，政治利益并不总一致。
- 强邻常通过册封、贸易、堡垒和扶植首领取得间接影响，使“臣属”与“实际统治”不能简单等同。

### 18世纪末的直接变化

准噶尔覆亡消除了一个主要军事压力，却没有产生统一吉尔吉斯国家。清朝主要维持边境与贡使关系，浩罕则依靠费尔干纳的财政和堡垒网络逐步深入南部及楚河谷；不同部族首领在贸易、安全与牧地考虑下分别合作或反抗。这一多中心格局直接衔接19世纪的浩罕统治、奥尔蒙汗整合尝试与俄罗斯扩张。
