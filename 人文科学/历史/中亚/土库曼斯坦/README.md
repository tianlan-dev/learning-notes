# 土库曼斯坦历史

土库曼斯坦位于里海东岸、卡拉库姆沙漠和阿姆河—穆尔加布河绿洲之间。今日国土曾同时属于马尔吉亚纳、帕提亚、花剌子模、呼罗珊和草原—沙漠部落世界；梅尔夫、尼萨与科尼亚乌尔根奇等城市的历史都跨越现代国界。奥古兹语土库曼诸群体在中世纪以后逐步形成，现代民族边界和共和国机构则主要由俄国征服与苏维埃民族划界塑造。

## 历史主线

卡拉库姆边缘的社会依赖少数河流、泉水、坎儿井和绿洲渠道。绿洲城市提供粮食、税收、工艺和商路，沙漠与草原部落提供牲畜、军队和跨区移动；两者既合作也冲突。尼萨是帕提亚早期王权的重要中心，梅尔夫则在阿契美尼德、希腊化、萨珊、阿拉伯和塞尔柱时期反复成为东北边疆或帝国都会。中世纪奥古兹部众、塞尔柱扩张和后续人口流动为土库曼语群形成提供背景，但不能把所有古代居民直接称为现代土库曼人。

13世纪以后，蒙古战争、阿姆河改道、伊朗和花剌子模诸政权竞争，使政治长期分散。特克、尧穆特、埃尔萨里、萨雷克等群体依部落、氏族和地方联盟组织，未形成覆盖今日全国的单一王朝。19世纪俄国通过里海据点、格奥克捷佩战争、梅尔夫归附和铁路建立外里海州。1924—1925年民族划界形成土库曼加盟共和国，苏维埃政权推进定居、集体化、棉花灌溉、教育、工业和天然气开发，同时造成镇压、生态压力和对联盟计划的依赖。

1991年独立后，总统制、天然气出口和联合国承认的永久中立成为国家道路。尼亚佐夫、库尔班古力·别尔德穆哈梅多夫和谢尔达尔·别尔德穆哈梅多夫三代领导延续高度集中的行政体系。2023年人民委员会重新成为最高代表机构，由前总统库尔班古力主持；正式行政首脑仍是总统。能源收入、低人口密度和外交谨慎支持制度稳定，出口市场集中、水资源、经济多元化和政策透明度则构成长期约束。

## 跨区域世系分工

帕提亚和塞尔柱都是跨中亚—伊朗—两河的帝国，完整世系不在本国目录重复维护。帕提亚诸王、并立者及争议年代见[安息王世系表](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A/%E4%BC%8A%E6%9C%97/%E5%AE%89%E6%81%AF%E7%8E%8B%E4%B8%96%E7%B3%BB%E8%A1%A8.md)；大塞尔柱核心苏丹序列见[塞尔柱与突厥化时期](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A/%E4%BC%8A%E6%9C%97/%E5%A1%9E%E5%B0%94%E6%9F%B1%E4%B8%8E%E7%AA%81%E5%8E%A5%E5%8C%96%E6%97%B6%E6%9C%9F.md)；与花剌子模、希瓦有关的完整王序见[河中地区](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E4%B8%AD%E4%BA%9A/%E6%B2%B3%E4%B8%AD%E5%9C%B0%E5%8C%BA/README.md)。本目录专注这些政权在今日土库曼斯坦空间的绿洲、部落和国家形成。
