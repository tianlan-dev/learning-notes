# 土库曼斯坦的土库曼部落、俄国征服与苏维埃化

## 时间

13世纪—1991年

## 概括

蒙古征服后，今日土库曼斯坦长期分属花剌子模、伊朗、呼罗珊和草原政权的边缘。奥古兹语人口与更早的绿洲居民、游牧集团不断融合，特克、尧穆特、埃尔萨里、萨雷克、乔杜尔等土库曼群体逐渐形成。它们有氏族、部落和临时联盟，却没有覆盖全境的固定王朝世系；谱系传统常服务于牧地、婚姻和政治认同，不能按近代国家君主表处理。

19世纪俄国从里海和北方推进。1879年第一次进攻格奥克捷佩失败，1881年斯科别列夫率军攻破要塞，造成大量军民死亡；1884年梅尔夫代表接受俄国统治，1885年潘杰德危机划定俄国与阿富汗势力边缘。外里海州、铁路和棉花经济把地区纳入帝国。革命和内战后，布尔什维克控制城市与铁路，土库曼反苏武装延续到1920年代。1924—1925年民族划界建立土库曼加盟共和国，苏维埃党国以定居、集体化、学校、运河、棉花和天然气工业重塑社会。

## 土库曼群体形成与近世政治

### 蒙古后人口与政治重组

1221年后，梅尔夫等绿洲遭破坏，人口迁移和渠道废弃扩大。南部先后受伊儿汗、卡尔提德、帖木儿和后续呼罗珊政权影响，北部和阿姆河下游则与金帐、花剌子模及希瓦体系相连。部落并非完全脱离国家：它们向不同王廷纳贡、提供骑兵、争取牧地，也在中央衰弱时保持高度自治。

### 部落与绿洲结构

| 群体或空间 | 主要活动区 | 政治特点 |
|---|---|---|
| 特克 | 阿哈尔、捷詹、梅尔夫一带 | 19世纪人口与军力较强，格奥克捷佩抵抗核心 |
| 尧穆特 | 里海东岸、阿特列克与花剌子模边缘 | 与伊朗、希瓦和俄国多方互动，内部支系众多 |
| 埃尔萨里 | 阿姆河中游与东部 | 河运、牧业和绿洲农业并存，常处布哈拉影响边缘 |
| 萨雷克 | 梅尔夫以南、穆尔加布上游 | 在俄、伊、阿边界形成中受挤压 |
| 乔杜尔等 | 北部和里海方向 | 与希瓦、卡拉卡尔帕克及草原网络相连 |
| 梅尔夫、阿哈尔绿洲 | 农业、市场和堡寨 | 部落首领、宗教人士、渠道共同体共同决策 |

领导权多由有威望的长老、军事首领、富户和宗教人物协商形成，战争时可能推举统一指挥，和平时又恢复分散。所谓“五大部落”是现代国家象征化的概括，不能抹去内部支系与地区差异。

### 与希瓦、布哈拉和伊朗的关系

希瓦王廷依赖并防范尧穆特等土库曼骑兵，18—19世纪屡有拥立、反叛和实际掌权；末期朱奈德汗即为尧穆特军事领袖。布哈拉影响东部埃尔萨里地区，卡扎尔伊朗则试图控制阿特列克、萨拉赫斯和梅尔夫。边境战争、掳掠和奴隶贸易构成近世暴力经济的一部分，但土库曼社会也广泛经营畜牧、地毯、盐、谷物和商队保护。希瓦完整可汗与实际掌权者序列见[布哈拉、希瓦与浩罕统治者表](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E4%B8%AD%E4%BA%9A/%E6%B2%B3%E4%B8%AD%E5%9C%B0%E5%8C%BA/%E5%B8%83%E5%93%88%E6%8B%89%E3%80%81%E5%B8%8C%E7%93%A6%E4%B8%8E%E6%B5%A9%E7%BD%95%E7%BB%9F%E6%B2%BB%E8%80%85%E8%A1%A8.md)。

## 俄国征服与殖民统治

### 推进过程

俄国1869年建立克拉斯诺沃茨克据点，以里海航运为补给。希瓦1873年战败后，俄军能够从北方和西方夹击阿哈尔。1879年洛马金仓促进攻格奥克捷佩失败，显示沙漠补给和土堡防御仍有效。1880—1881年斯科别列夫以铁路、炮兵、工兵坑道和更充分补给重新进攻，攻破邓吉尔捷佩／格奥克捷佩。死亡人数在不同记载中差异很大，但战役无疑摧毁了特克集体军事抵抗的核心。

1881年阿哈尔并入俄国，1884年梅尔夫土库曼代表接受统治，1885年俄军取得潘杰德使英俄濒临战争。帝国随后以外里海州军事长官治理，1898年起划入突厥斯坦总督区。外里海铁路1880年代由海岸延至阿什哈巴德、梅尔夫并通向布哈拉、撒马尔罕，既服务军队也改变商贸。殖民官署保留部分本地法庭与长老，却垄断军政、土地裁判和跨境关系。

完整外里海州长序列见[俄属、苏维埃与共和国领导人表](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E4%B8%AD%E4%BA%9A/%E5%9C%9F%E5%BA%93%E6%9B%BC%E6%96%AF%E5%9D%A6/%E4%BF%84%E5%B1%9E%E3%80%81%E8%8B%8F%E7%BB%B4%E5%9F%83%E4%B8%8E%E5%85%B1%E5%92%8C%E5%9B%BD%E9%A2%86%E5%AF%BC%E4%BA%BA%E8%A1%A8.md)。

## 革命、内战与民族划界

1917年后，阿什哈巴德苏维埃、社会革命党人、铁路工人、白军、土库曼武装和英属印度派遣部队相互竞争。1918年外里海临时政府反对布尔什维克并接受英军有限援助；内部清洗和财政困难削弱其基础。红军1920年控制阿什哈巴德、克拉斯诺沃茨克等中心，地方反苏武装仍借沙漠、伊朗和阿富汗边界活动。抵抗者目标从保卫部落自治、反征粮到恢复旧秩序不一，不能统一解释为单一民族运动。

1924年苏联进行中亚民族划界，从原突厥斯坦自治共和国及布哈拉、花剌子模苏维埃共和国划出土库曼地区；1925年正式成为联盟共和国。边界试图兼顾语言人口、牧地、灌溉、铁路和党政可治理性，留下跨境部落与水资源问题。阿什哈巴德成为首都，标准土库曼语、学校和干部体系逐渐建立。

## 苏维埃化的分阶段过程

| 阶段 | 时间 | 主要政策与影响 |
|---|---|---|
| 建国与本土化 | 1924—1929年 | 建立共和国机关，推广土库曼语教育和本地干部，同时压制武装抵抗。 |
| 集体化与清洗 | 1929—1941年 | 牧民定居、农业集体化、反宗教运动；饥荒、逃亡和1937—1938年精英清洗。 |
| 战争与重建 | 1941—1953年 | 人员参战、后方生产；1948年阿什哈巴德地震后大规模重建。 |
| 运河、棉花与工业 | 1954—1980年代 | 卡拉库姆运河分段建成，棉花农业扩大；石油、天然气、化工和城市就业增长。 |
| 改革与主权化 | 1985—1991年 | 尼亚佐夫任第一书记，党政本地化；1990年主权宣言和总统制，1991年独立。 |

## 苏维埃权力结构

| 角色 | 法定或实际功能 |
|---|---|
| 土库曼共产党第一书记 | 共和国实际最高领导，关键任免受苏共中央影响 |
| 最高苏维埃主席团主席 | 法定国家元首，主要承担立法签署与礼仪职责 |
| 部长会议主席 | 政府首脑，执行计划、预算、农业和工业政策 |
| 州、区党委 | 控制地方干部、集体农庄、企业和安全动员 |
| 联盟部委 | 掌握大型能源、铁路、军工与跨共和国水利项目 |
| 集体农庄与国营企业 | 组织定居、劳动、福利和农产品征购，也是基层控制单元 |

## 重要事件

1. 1221年蒙古攻陷梅尔夫，古代城市与灌溉体系遭重创。
2. 14—15世纪，帖木儿战争及呼罗珊诸政权重组南部绿洲。
3. 16—18世纪，奥古兹语土库曼群体在里海、花剌子模和呼罗珊边缘逐渐形成。
4. 1740年纳迪尔沙征服希瓦、布哈拉，土库曼部落被卷入帝国战争。
5. 1804年后弘吉剌希瓦汗国加强对花剌子模边缘部落的征税与军事动员。
6. 1855年萨拉赫斯战役中，特克土库曼击败希瓦军，希瓦可汗穆罕默德·阿明阵亡。
7. 1869年俄国建立克拉斯诺沃茨克据点。
8. 1873年希瓦成为俄国保护国，俄国更接近土库曼绿洲。
9. 1879年第一次格奥克捷佩远征失败。
10. 1881年第二次格奥克捷佩战役中俄军攻破要塞，阿哈尔被吞并。
11. 1884年梅尔夫土库曼代表接受俄国统治。
12. 1885年潘杰德危机后，俄阿边界和英俄势力范围趋于固定。
13. 1880—1890年代外里海铁路贯通主要绿洲。
14. 1898年外里海州转归突厥斯坦总督区。
15. 1916年中亚劳役征调和战争经济加剧社会紧张，土库曼地区规模与形态有地区差异。
16. 1918年外里海临时政府成立，英军短期介入。
17. 1920年红军控制主要城市和铁路。
18. 1924—1925年土库曼苏维埃社会主义共和国建立。
19. 1929—1930年代集体化、定居化和反苏武装镇压同步展开。
20. 1937—1938年大清洗处决包括奈迪尔拜·艾塔科夫、盖吉瑟兹·阿塔巴耶夫在内的早期干部。
21. 1941—1945年，大量土库曼居民参战，后方生产纳入战争动员。
22. 1948年阿什哈巴德地震造成严重伤亡和城市毁坏，数字统计存在争议。
23. 1954年以后卡拉库姆运河分段建设，扩大棉田和城市供水，也增加蒸发、盐碱化与阿姆河压力。
24. 1960—1980年代，大型天然气田与联盟管网开发。
25. 1985年尼亚佐夫成为第一书记。
26. 1990年8月22日通过国家主权宣言，11月设总统制。
27. 1991年10月公投与独立宣言终结苏维埃阶段。

## 稳定、衰落与转型原因

### 部落秩序的韧性

沙漠环境使外来国家难以常驻，部落网络能通过迁徙、分支和跨境亲缘保存自主；绿洲农业、畜牧和商队贸易又提供多元生计。其弱点是难以形成常备财政与统一指挥，内部仇杀和王廷干预经常破坏联盟。

### 俄国征服的结构条件与直接触发

俄国拥有海运、工业炮兵、工程兵和持续财政，希瓦战败又消除北方缓冲。1879年失败促使俄军改善铁路和补给；1881年集中攻城直接摧毁格奥克捷佩防线。1884年梅尔夫选择归附，既受军事威慑，也考虑与伊朗、阿富汗和其他部落竞争。征服不是“技术必然”，而是多次战役、外交与地方选择的结果。

### 苏维埃稳定与代价

共和国边界、标准语言、普及教育、医疗和工业就业建立现代国家社会基础；党组织、集体农庄和安全机关深入基层。与此同时，强制定居、征粮、清洗和宗教压制破坏旧有制度。棉花—运河—天然气体系带来城市化和收入，却造成水资源依赖、盐碱化与单一经济。1980年代联盟财政和政治权威衰退时，第一书记兼总统的制度安排使权力平稳转入独立国家。
