# 乌兹别克斯坦历史

乌兹别克斯坦的历史核心是阿姆河—锡尔河之间的河中绿洲、费尔干纳谷地和花剌子模南缘。撒马尔罕、布哈拉、塔什干、希瓦等城市先后处于粟特城邦、伊朗与突厥王朝、蒙古—帖木儿体系、乌兹别克诸汗国、俄罗斯帝国和苏联之中。现代边界、国语和“乌兹别克民族国家”主要由19—20世纪的殖民统治、民族划界与独立建国制度化，不能直接倒推到古代。

## 历史主线

绿洲灌溉、草原—城市交换和跨欧亚商路构成长期底层结构。古代粟特与花剌子模并非统一国家；伊斯兰化也不是一次征服后立即完成，而是在阿拉伯驻军、地方贵族、城市法学与波斯语文化的长期互动中推进。蒙古征服破坏若干城市和灌溉区，却又把河中纳入更广阔的帝国交通。帖木儿以撒马尔罕为中心建立军事帝国，昔班尼后裔随后把“乌兹别克”草原政治名号带入河中。近世布哈拉、希瓦、浩罕并立，各自依靠王族、部落、宗教精英和城市税收。

19世纪俄国把塔什干、撒马尔罕和费尔干纳直接纳入突厥斯坦总督区，同时保留布哈拉、希瓦为保护国。革命、巴斯玛奇战争与1924年民族划界形成乌兹别克加盟共和国；棉花专门化、灌溉扩张、工业化与教育普及并存，也造成强制动员、政治镇压和咸海危机。1991年独立后，强总统制维持国家连续性；2016年以后经济开放、行政改革与中亚合作加快，但水资源、人口就业、地区差距和政治问责仍是长期议题。

## 重要转折与时间节点

| 时间 | 转折 | 长期意义 |
|---|---|---|
| 前6—前4世纪 | 粟特、花剌子模进入阿契美尼德体系 | 绿洲税赋、帝国交通与地方王权结合 |
| 705—715年 | 屈底波征服河中主要城市 | 阿拉伯军政控制建立，伊斯兰化长期展开 |
| 892—999年 | 萨曼王朝以布哈拉为中心 | 新波斯语文化、逊尼学术与城市经济繁荣 |
| 1219—1221年 | 蒙古征服 | 王朝秩序、人口与灌溉网络剧烈重组 |
| 1370年 | 帖木儿取得河中最高权力 | 撒马尔罕成为跨区域帝国中心 |
| 1500—1501年 | 昔班尼占领布哈拉、撒马尔罕 | 草原乌兹别克王族进入河中 |
| 1709年前后 | 浩罕在费尔干纳兴起 | 三大汗国格局逐渐形成 |
| 1865—1876年 | 俄国占领塔什干并吞并浩罕 | 直接殖民地与保护国并存 |
| 1917—1924年 | 革命、战争和民族划界 | 帝国制度瓦解，现代共和国框架建立 |
| 1924年 | 乌兹别克苏维埃社会主义共和国成立 | 行政边界、标准语言与国家机构制度化 |
| 1960年代以后 | 棉花灌溉高峰与咸海萎缩 | 计划经济的生态成本集中显现 |
| 1991年8月31日 | 宣布国家独立 | 苏维埃加盟共和国转为主权国家 |
| 2016年以后 | 新一轮经济与行政改革 | 区域开放和国家治理进入调整期 |
