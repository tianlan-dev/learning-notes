# 中亚突厥化、伊斯兰化与语言文化

## 时间

6世纪以后

## 概括

中亚的突厥化和伊斯兰化均是长期、多方向过程。突厥诸汗国的政治扩张、草原迁徙和军队编制扩大了突厥语影响；伊斯兰则经由阿拉伯征服、萨曼王朝、商人、学者、苏菲社团和地方统治者逐步扎根。波斯语、粟特语、突厥语和阿拉伯语在不同层面共存。

## 主要阶段

| 阶段 | 特征 |
|---|---|
| 西突厥与突骑施时期 | 草原联盟控制七河和河中边缘，城市仍保有伊朗语文化 |
| 阿拉伯征服 | 8世纪进入河中，政治征服先于社会全面伊斯兰化 |
| 萨曼王朝 | 9—10世纪以布哈拉为中心，波斯语伊斯兰文化繁盛 |
| 喀喇汗王朝 | 10—12世纪突厥王朝改宗伊斯兰，突厥书写文化发展 |
| 花剌子模与察合台传统 | 波斯语城市文化和突厥军事精英并存 |
| 近世汗国与苏菲网络 | 乌兹别克、哈萨克、土库曼、吉尔吉斯等身份在不同时期形成与重组 |

## 辨析

- 语言转移不等于人口完全替换；许多城市、山地与绿洲社群保留或重构了伊朗语传统。
- 伊斯兰化并未消除地方习俗和前伊斯兰圣地，而是常通过朝圣、圣裔与苏菲网络重新诠释它们。
- 现代“民族”名称大多在近世和苏维埃时期被标准化，不能直接投射到早期中世纪。
