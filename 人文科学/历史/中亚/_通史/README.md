# 中亚通史

## 概括

本目录用于放置跨越现代中亚国家边界的通史主题，例如粟特与丝绸之路、突厥化、伊斯兰化、蒙古帝国、帖木儿帝国、汗国体系和俄罗斯 / 苏联中亚。

## 主题导航

| 主题 | 时间 | 说明 |
|---|---|---|
| 绿洲城邦与丝绸之路 | 古代-中古 | 撒马尔罕、布哈拉、粟特商人和东西交通网络。 |
| 突厥化与伊斯兰化 | 6世纪以后；8世纪以后 | 草原政治和伊斯兰文明共同重塑中亚。 |
| 蒙古与帖木儿传统 | 13世纪-16世纪 | 蒙古帝国、察合台汗国和帖木儿帝国构成关键转折。 |
| 汗国与俄罗斯扩张 | 16世纪-20世纪 | 布哈拉、希瓦、浩罕等汗国逐步被俄罗斯帝国和苏联重组。 |
