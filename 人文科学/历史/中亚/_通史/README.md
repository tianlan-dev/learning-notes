# 中亚通史

中亚通史不是五个现代国家历史的简单相加，而是绿洲、草原、山地和沙漠边缘不断重组的共同史。阿姆河、锡尔河、泽拉夫尚河、天山、帕米尔、七河与里海东岸形成多个相互依赖的历史空间；粟特商人、突厥联盟、伊斯兰学术网络、蒙古兀鲁思、俄罗斯帝国和苏维埃共和国都跨越今日国界。

本目录只维护跨区域机制和共同转折。具体城市、国家、王朝细节留在河中、草原汗国与现代国家页面；完整统治者序列只在规范专表维护。

## 历史主线

- 灌溉城市与牧业草原不是相互隔绝的两种世界；粮食、马匹、军队、市场和水源把它们连接起来。
- “丝绸之路”是可改道的陆海网络，不是一条由单一帝国长期控制的道路；粟特商人依靠侨居、信用和多语能力发挥中介作用。
- 突厥化主要改变政治精英、人口分布和日常语言，伊斯兰化主要改变法统、制度、信仰与社会网络；两者速度和地区范围并不一致。
- 蒙古征服造成严重人口和城市破坏，随后又建立驿传与跨洲流动。察合台汗国的草原—城市张力最终形成西察合台与蒙兀儿斯坦。
- 帖木儿借成吉思汗系傀儡汗、联姻和军事实力掌权；撒马尔罕与赫拉特的文化繁荣和征服暴力属于同一帝国结构的两面。
- 俄国先控制哈萨克草原，后在19世纪中叶进入河中；苏维埃又以民族共和国、计划经济和党国体系重组殖民遗产。
- 1991年五国继承苏维埃边界和机构成为独立国家，政治、经济与对外路线分化，水源、飞地、劳务和安全仍要求跨境协调。

## 历史空间比较矩阵

| 历史空间 | 生态与经济 | 前伊斯兰核心 | 中世纪重组 | 俄苏时期与现代分属 |
|---|---|---|---|---|
| 河中与泽拉夫尚 | 密集灌溉、城市、手工业 | 粟特城邦、撒马尔罕、布哈拉 | 萨曼、喀喇汗、察合台、帖木儿、布哈拉 | 主要进入乌兹别克斯坦，部分进入塔吉克斯坦及邻国 |
| 花剌子模与阿姆河下游 | 河道与大型渠系、绿洲—草原贸易 | 花剌子模王国 | 花剌子模沙阿、蒙古、希瓦 | 分属乌兹别克斯坦、土库曼斯坦，咸海危机跨境 |
| 七河、伊犁与天山 | 牧地、河谷农业、山口 | 萨卡、乌孙及多种草原集团 | 西突厥、喀喇汗、察合台、蒙兀儿斯坦 | 主要进入哈萨克斯坦、吉尔吉斯斯坦及相邻中国地区 |
| 哈萨克草原 | 季节牧业、河湖与商镇 | 萨卡及草原联盟 | 突厥、蒙古、金帐、乌兹别克—哈萨克汗国 | 俄国移民和苏维埃定居影响深，主体进入哈萨克斯坦 |
| 费尔干纳盆地 | 高密度灌溉、山口和市场 | 大宛及地方绿洲 | 喀喇汗、帖木儿、浩罕 | 划分于乌兹别克斯坦、吉尔吉斯斯坦、塔吉克斯坦 |
| 梅尔夫与里海东岸 | 沙漠绿洲、乌古斯—土库曼牧业 | 马尔吉亚纳、帕提亚联系 | 塞尔柱、花剌子模、土库曼部落 | 主体进入土库曼斯坦，天然气与水资源塑造现代经济 |
| 巴达赫尚与帕米尔 | 高山牧业、河谷农业、跨山通道 | 大夏与东伊朗语传统 | 伊斯兰山地政权、察合台和帖木儿竞争 | 分属塔吉克斯坦、阿富汗及邻区，语言宗教多样性突出 |
| 阿富汗北部 | 巴尔赫绿洲、山口与草原边缘 | 大夏、贵霜及吐火罗斯坦 | 加兹尼、古尔、察合台、帖木儿和乌兹别克 | 未进入苏联，但与中亚边境、迁徙和战争长期相连 |

## 关键转折

| 时间 | 转折 | 跨区域意义 |
|---|---|---|
| 前6—前4世纪 | 阿契美尼德统合东部区域 | 粟特、大夏、花剌子模进入跨高原帝国网络 |
| 约前2世纪 | 汉使、月氏迁徙与贵霜形成 | 中国、草原、印度和伊朗联系扩大 |
| 6世纪 | 突厥汗国兴起 | 草原军政、突厥语和粟特商业形成合作体系 |
| 705—715年 | 阿拉伯征服河中主要城市 | 政治进入哈里发体系，长期伊斯兰化开始 |
| 9—10世纪 | 萨曼繁荣与喀喇汗改宗 | 波斯语伊斯兰文化与突厥穆斯林王权结合 |
| 1219—1221年 | 蒙古征服 | 花剌子模帝国崩溃，城市与人口遭重创 |
| 1340年代 | 察合台东西分裂 | 西部埃米尔掌权，东部形成蒙兀儿斯坦 |
| 1370年 | 帖木儿控制河中 | 非成吉思汗系埃米尔以傀儡汗和联姻建立帝国 |
| 1500—1512年 | 昔班尼占领河中 | 帖木儿中亚主线终结，汗国时代展开 |
| 1731年以后 | 俄国逐步控制哈萨克草原 | 堡垒、宣誓和行政改制先于南部全面征服 |
| 1865—1885年 | 俄国征服南部中亚 | 塔什干、三大汗国与土库曼地区进入殖民秩序 |
| 1916—1924年 | 起义、革命、内战与划界 | 帝国秩序崩溃，苏维埃民族领土体系建立 |
| 1929—1936年 | 五个加盟共和国框架完成 | 今日五国边界与国家机构的主要前身形成 |
| 1991年 | 苏联解体 | 加盟共和国边界转为国际边界 |
| 2025年 | 吉尔吉斯斯坦—塔吉克斯坦边界条约 | 长期未定边界开始以双边、三边协议收束 |

## 跨阶段机制比较

| 机制 | 古代与中世纪 | 俄苏与现代 |
|---|---|---|
| 水与土地 | 渠系维护决定绿洲兴衰，牧场和农田互相依赖 | 棉花、跨境水库、咸海和边界把资源治理国家化 |
| 法统 | 帝国册封、成吉思汗血统、伊斯兰称号与宗教精英 | 沙皇主权、苏维埃民族共和国、现代宪制主权 |
| 人口移动 | 商人侨居、牧民迁徙、征服和工匠强制迁移 | 定居殖民、集体化、战争疏散、劳务移民 |
| 语言文化 | 粟特语、波斯语、突厥语、阿拉伯语功能分层 | 民族语言标准化、俄语共同语和独立后文字政策 |
| 边界 | 牧地、贡赋与城市控制范围重叠且流动 | 帝国行政线经苏维埃划界成为现代国际边界 |
| 区域网络 | 商队、驿站、寺院和苏菲网络 | 铁路、能源管线、区域组织与跨境市场 |
