# 中亚绿洲、粟特与丝绸之路

## 时间

古代—近世

## 概括

中亚的绿洲城市依托雪山融水、灌溉渠和商队路线发展。撒马尔罕、布哈拉、潘吉肯特、梅尔夫、塔拉兹和费尔干纳等并非孤立的“驿站”，而是农业、手工业、税收、宗教机构与跨区域贸易的政治中心。粟特商人尤以多语能力和侨居网络著称。

## 核心节点

| 节点 | 特征 |
|---|---|
| 粟特 | 以撒马尔罕、布哈拉和泽拉夫尚河谷为中心的伊朗语城邦网络 |
| 大夏与花剌子模 | 阿姆河两岸的灌溉王国，与伊朗、希腊化和草原力量互动 |
| 费尔干纳 | 山地与绿洲之间的农业谷地，连接中国、河中和草原 |
| 梅尔夫 | 马尔吉亚纳绿洲大城，连通伊朗、花剌子模与河中 |
| 七河地区 | 天山北麓的绿洲、牧地与商路，连接中国西域与草原 |
| 商队网络 | 丝、马匹、金属、香料、纸、宗教文本、技术和人口流动 |

## 关键辨析

- “丝绸之路”是近代概念，历史上并不存在一条固定、单向的道路，而是可替换的多条陆路和海路网络。
- 粟特不是单一国家名称，而是由多个城邦、语言共同体和商人侨居网络构成。
- 绿洲城市与草原并非对立：牧民提供马匹、畜产与军事力量，城市提供粮食、手工业、税收和市场。
- 佛教、摩尼教、祆教、景教和伊斯兰教都经由这些网络传播或重组。
