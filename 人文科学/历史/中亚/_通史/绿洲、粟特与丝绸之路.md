# 中亚绿洲、粟特与丝绸之路

## 时间

约前4千纪—16世纪（陆路网络此后仍延续）

## 概括

中亚的历史基础不是一条横穿大陆的“道路”，而是绿洲农业、草原牧业、山地通道和沙漠边缘共同组成的节点网络。雪山融水与河流冲积扇支持灌溉城市，草原和山地社会提供马匹、畜产品、护卫与季节性运输；城镇则向牧民提供粮食、织物、金属器、信贷和市场。两者既交换也冲突，任何一方都不能脱离另一方独立维持区域秩序。

“丝绸之路”是近代学术概念。历史上的商人通常分段转运，路线会随战争、关税、河道、驿站安全和市场需求改道。丝只是高价值货物之一；马匹、棉布、毛皮、金属、宝石、香料、纸张、奴隶、粮食和宗教文本同样重要。粟特商人之所以突出，不是因为建立了统一帝国，而是因为他们以多语能力、亲族信用、侨居社区和跨政权外交，把撒马尔罕等故乡城市与草原、中国、印度、伊朗和地中海市场连接起来。

## 网络运行机制

| 机制 | 具体作用 | 脆弱点 |
|---|---|---|
| 灌溉与绿洲农业 | 以渠系把阿姆河、锡尔河及山前河流引向农田，供养城市、军队和商队 | 河道改道、失修、战争毁渠会迅速压缩定居区 |
| 分段贸易 | 商人通常在撒马尔罕、布哈拉、梅尔夫、塔拉兹、费尔干纳等节点换货、换畜和结算 | 多重关税、盗掠和边界封锁增加成本 |
| 侨居与信用 | 同乡、亲族、寺院和商团提供翻译、借贷、仓储、诉讼与信息 | 政权排斥外来社群或战争切断通信时网络收缩 |
| 草原—城市互补 | 牧民提供马匹、肉乳、毛皮、军事力量；城市提供粮食、工艺品和铸币 | 牧场压力、征税和继承斗争会转化为劫掠或围城 |
| 帝国保护与驿传 | 大国维护关隘、道路和驿站，统一部分税制与度量 | 帝国分裂后地方势力可能重复征税，但分裂并不等于贸易必然停止 |
| 宗教与知识网络 | 僧侣、传教士、学者和朝圣者随商路传播文本、语言、艺术与技术 | 改宗和国家政策会改变机构，却很少一次清除全部旧传统 |

## 分阶段过程

### 早期城市化与区域文明

泽拉夫尚河谷的萨拉兹姆在前4—前3千纪已经出现灌溉、冶金和远距离交换；阿姆河上游与马尔吉亚纳的青铜时代聚落则显示大型建筑、手工业分工和跨山联系。由于这一时期缺乏连续文字年表，考古文化不能直接等同于后来的某个现代民族。

前6—前4世纪，粟特、大夏、花剌子模和马尔吉亚纳进入阿契美尼德帝国的行省与贡赋体系，但各绿洲并非完全同质。前329—前327年，亚历山大在大夏、粟特遭遇持续抵抗；其后塞琉古与希腊—大夏政权把希腊城市、铸币和本地政治结合起来。约前2世纪的萨卡、月氏等迁徙重组了草原与绿洲权力，最终促成贵霜政权跨越大夏、犍陀罗和北印度的交通体系。

### 汉代交通扩大与粟特商人网络

前2世纪末张骞出使使汉朝更系统地认识大宛、大夏和安息，但这不是某一天“开通”一条道路。既有地方贸易、草原路线与印度—伊朗联系早已存在；汉朝的外交、军屯和丝马交换只是扩大了网络规模。

约4—8世纪，粟特商人建立从河西走廊到中国内地、蒙古高原和七河地区的侨居社区。敦煌粟特古信札证明商人利用书信、合伙和同乡代理维持远程生意。6世纪突厥汗国为跨草原交通提供军事框架，并借用粟特书吏与外交人才；粟特人则从突厥的护送和对外谈判中获利。双方是互相利用的伙伴，不是单向“游牧征服—商人服从”。

### 宗教、技术与艺术的流动

佛教主要从印度经大夏、犍陀罗和塔里木盆地向东传播；祆教是许多粟特社群的重要传统，摩尼教、景教也依赖商人和侨居社区扩展。宗教传播不是简单复制：寺院形制、神像、葬俗和故事会在潘吉肯特、七河、中国城市等地重新组合。

葡萄与果树栽培、金属工艺、玻璃、纺织纹样、造纸和医药知识也沿网络移动。751年怛罗斯之战可能加速俘虏工匠和技术流动，但把中亚造纸或伊斯兰化全部归因于一场战役并不可靠；政治征服、市场需求和工匠迁徙是更长期的机制。

### 阿拉伯征服后的网络重组

705年以后，屈底波等将领逐步控制布哈拉、撒马尔罕等地；720年代粟特诸城反抗失败，地方贵族与商人开始在新的税制、驻军和伊斯兰法律环境中调整身份。粟特语并未立即消失，伊斯兰化也经过数代才深入社会。

9—10世纪萨曼王朝以布哈拉为中心，波斯语伊斯兰文化、银币贸易和城市手工业繁荣；撒马尔罕成为货物集散地，布哈拉织物、河中纸张、草原毛皮与奴隶贸易进入更广阔的伊斯兰经济圈。10世纪以后喀喇汗等突厥王朝改宗伊斯兰，商路的政治精英更替，但绿洲生产、波斯语文书和商人网络延续。

### 蒙古与帖木儿时代

1219—1221年的蒙古战争严重破坏讹答剌、布哈拉、撒马尔罕、花剌子模和部分渠系，人口死亡、迁徙与工匠强制转移不可淡化；不同城市的破坏程度和恢复速度却不相同。蒙古帝国随后设置驿站、护照和商人合伙，使使节、工匠与货物跨洲移动更频繁。所谓“蒙古和平”只描述部分路线的可通行性，不能抹去征服暴力、重税和地方战争。

14—15世纪，帖木儿以征服所得人口、税收和工匠建设撒马尔罕；沙哈鲁时期赫拉特与河中形成新的学术、工艺和商业中心。16世纪以后，海运扩张、欧亚帝国边界变化和区域战争改变了长途陆运的相对地位，但商队贸易并未突然终止，俄国、布哈拉、印度、伊朗与中国之间的陆路交换一直延续到近代。

## 重要事件

| 时间 | 事件 | 过程与长期意义 |
|---|---|---|
| 前4—前3千纪 | 萨拉兹姆等聚落发展 | 灌溉、冶金和跨区域交换显示中亚早期城市化远早于“丝绸之路”名称。 |
| 前6—前4世纪 | 阿契美尼德统合东部行省 | 粟特、大夏、花剌子模进入帝国贡赋与军事网络，同时保留地方差异。 |
| 前329—前327年 | 亚历山大征服粟特、大夏 | 城镇围攻与地方抵抗并行，希腊化政治和铸币随后与本地传统融合。 |
| 约前130年代 | 张骞抵达中亚 | 汉朝外交扩大丝马与使团交通，但建立在既有区域路线之上。 |
| 1—3世纪 | 贵霜政权兴盛 | 大夏—犍陀罗—北印度通道促进贸易、佛教和多语铸币传播。 |
| 313年前后 | 敦煌粟特古信札 | 直接证明中国西北已有粟特侨商、代理和跨城通信。 |
| 560年代 | 突厥与萨珊击败嚈哒 | 新的草原帝国与粟特商人合作，重组伊朗—拜占庭—中国间贸易。 |
| 6—8世纪 | 粟特商业文化鼎盛 | 撒马尔罕、潘吉肯特及侨居社区成为外交、艺术和宗教传播枢纽。 |
| 705—715年 | 阿拉伯军队控制河中主要城市 | 政治征服先于社会全面伊斯兰化，旧城邦精英逐步被纳入新秩序。 |
| 751年 | 怛罗斯之战 | 唐军败退是区域力量变化的一环；不能视为伊斯兰化或造纸传播的单一开端。 |
| 9—10世纪 | 萨曼时代城市复兴 | 波斯语伊斯兰文化、银币、纺织和商队网络连接草原与阿拔斯世界。 |
| 1219—1221年 | 蒙古征服河中与花剌子模 | 城市、人口和渠系遭重创，随后又被纳入蒙古驿传和跨洲人员流动。 |
| 1370年以后 | 帖木儿以撒马尔罕为中心 | 强制迁徙工匠与帝国赞助共同造就建筑、学术和工艺中心。 |
| 16世纪以后 | 海陆网络重新配比 | 海运与新帝国改变远程贸易成本，陆路转向区域化但继续运作。 |

## 地区差异

| 地区 | 经济与社会基础 | 网络角色 | 主要差异 |
|---|---|---|---|
| 粟特与泽拉夫尚河谷 | 密集灌溉、城市手工业、商人和地主 | 跨欧亚侨商与文化中介 | 政治常由多个城邦构成，商业共同体不等于统一国家 |
| 大夏 / 吐火罗斯坦 | 阿姆河上游农业、山口与巴尔赫中心 | 联系印度、呼罗珊和河中 | 佛教与希腊—贵霜遗产较强，伊斯兰化部分地区较晚 |
| 花剌子模 | 阿姆河下游大型渠系 | 连接咸海、钦察草原、河中与伊朗 | 高度依赖河道与灌溉，政治中心多次因水系和战争迁移 |
| 费尔干纳 | 封闭而肥沃的山间盆地 | 马匹、农业与通往喀什的通道 | 地方政权韧性较强，控制谷口比控制全盆地更关键 |
| 七河与天山北麓 | 牧地、河谷农业和混合聚落 | 草原—绿洲换货及东西通道 | 突厥、粟特和后来的蒙古社群长期混居 |
| 梅尔夫与马尔吉亚纳 | 穆尔加布河灌溉大绿洲 | 呼罗珊通往河中、花剌子模的门站 | 更直接受伊朗高原帝国与呼罗珊政治影响 |

## 兴盛、收缩与转型的原因

- **兴盛的结构条件**：融水灌溉产生农业剩余，城市手工业与草原畜牧互补；多语商人和侨居社区降低跨政权交易成本。
- **政治保护**：阿契美尼德、贵霜、突厥、萨曼、蒙古和帖木儿等不同政权都曾通过道路、驿站、铸币和关税制度扩大交流，但收益分配并不平等。
- **收缩的结构因素**：河道变迁、渠系失修、瘟疫、市场转移和重复征税会长期削弱节点；并非每次衰落都由外族入侵单独造成。
- **外部压力**：帝国战争和草原权力竞争可能造成围城、屠杀和强制迁徙，尤其蒙古征服的直接破坏十分严重。
- **直接触发**：具体城市的衰败常由破城、断水或商路改道触发；但其他节点可能同时兴起，因此不能把整个中亚写成同步“繁荣”或同步“衰落”。
- **网络转型**：伊斯兰化、突厥化和海运扩张改变通用语言、制度与路线，却没有终止中亚作为区域交换中心的作用。

## 争议与认识边界

- 古代政治边界主要由考古、钱币、外部史书和零散铭文重建，绝对年代与统治范围常只能写“约”。
- “粟特人”既指故乡区域的伊朗语居民，也可指跨地区侨居商人；不能把所有中亚商人都称为粟特人。
- 丝绸之路不是从中国单向通往罗马，也不是只有奢侈品长途贸易；大量交换发生在相邻节点之间。
- 751年并未立即终结唐朝一切中亚联系，伊斯兰在塔里木盆地等地取得稳定优势也晚得多。
- 路线变化不等于“丝绸之路消失”。陆路、海路、草原路与山口路长期竞争并互相衔接。
