# 中亚金帐汗国、乌兹别克与哈萨克汗国

## 时间

13—18世纪

## 概括

蒙古帝国在钦察草原和河中建立新的汗国体系。金帐汗国统治伏尔加—钦察草原，察合台后裔活动于中亚腹地。15世纪昔班尼后裔阿布海尔汗整合部分草原部落，其分离者在克烈和贾尼别克领导下形成哈萨克汗国；“乌兹别克”和“哈萨克”身份在此后数百年持续重组。

## 主要政治体

| 政权 | 时间 | 特征 |
|---|---|---|
| 金帐汗国 | 13—15世纪 | 钦察草原、伊斯兰化与欧亚商路 |
| 阿布海尔汗国 | 15世纪 | 昔班尼后裔在东部草原建立联盟 |
| 哈萨克汗国 | 15—18世纪 | 三玉兹、牧业迁徙与草原外交 |
| 布哈拉与希瓦乌兹别克王朝 | 16世纪以后 | 乌兹别克军事精英进入绿洲城市政治 |
| 准噶尔汗国 | 17—18世纪 | 天山和伊犁草原强权，与哈萨克、清朝和俄国互动 |

## 重要节点

- 1460年代前后，克烈与贾尼别克离开阿布海尔汗，后世常视为哈萨克汗国开端。
- 16世纪卡西姆汗时期哈萨克汗国扩张并形成法律传统记忆。
- 18世纪准噶尔战争、俄国堡垒和清朝西进同时改变草原权力平衡。
- 三玉兹是历史形成的草原联盟与地域组织，不等同于现代行政区。
