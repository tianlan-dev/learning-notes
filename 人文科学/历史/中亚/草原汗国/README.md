# 草原汗国

本目录整理中亚及相邻草原的游牧政治传统。草原社会并非静止的“部落世界”：牧业、季节迁徙、部族联盟、城市市场、宗教网络和军事组织共同塑造从萨卡到哈萨克汗国的政治形态。俄罗斯国家史归入欧洲东斯拉夫目录，中国北方及西域归入中国史，本目录侧重其与中亚绿洲的互动。
