# 草原汗国

## 概括

本目录用于整理中亚及其相邻草原方向的草原汗国和游牧政治传统，包括斯基泰、萨尔马特、匈人、突厥、西突厥、蒙古、钦察、察合台、哈萨克等与中亚绿洲互动密切的草原政权。这里保留草原政治对中亚的影响；俄罗斯国家史归入欧洲东斯拉夫目录。

## 主线提示

| 主线 | 说明 |
|---|---|
| 斯基泰、萨尔马特与早期草原传统 | 黑海—里海—中亚草原的早期游牧和半游牧政治背景。 |
| 匈人、突厥与西突厥 | 中亚突厥化和草原帝国扩张的重要政治背景。 |
| 蒙古与察合台体系 | 将绿洲和草原纳入蒙古帝国秩序，并与金帐汗国、钦察草原相互关联。 |
| 钦察、乌兹别克、哈萨克 | 草原联盟和汗国重塑中亚北部。 |
| 草原—绿洲关系 | 游牧军事力量与绿洲农业、城市贸易互相依存。 |
