# 哈萨克斯坦历史

哈萨克斯坦历史以草原牧业、萨卡与突厥传统、哈萨克汗国、俄罗斯帝国扩张和苏维埃化为主线。现代国家继承苏维埃边界与资源工业，也持续面对草原饥荒、核试验、人口迁徙和语言文化重建的历史遗产。

## 历史坐标

- 地理范围：里海以东、阿尔泰至天山北麓的哈萨克草原与绿洲边缘。
- 现代国家：边界和民族名称多在俄罗斯帝国、苏联或近现代战争中制度化，不能直接倒推为古代疆域。
- 区域联系：[中亚历史](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E4%B8%AD%E4%BA%9A/README.md)；跨国背景见[中亚通史](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E4%B8%AD%E4%BA%9A/_%E9%80%9A%E5%8F%B2/README.md)。

## 关键节点

| 时间 | 事件 | 意义 |
|---|---|---|
| 前1千纪 | 萨卡文化 | 草原早期历史的重要证据 |
| 约15世纪 | 哈萨克汗国形成 | 现代民族政治传统的关键来源 |
| 18—19世纪 | 俄罗斯扩张 | 草原自治逐步丧失 |
| 1931—1933年 | 饥荒 | 苏维埃改造的巨大社会代价 |
| 1991年 | 独立 | 哈萨克斯坦共和国成立 |
