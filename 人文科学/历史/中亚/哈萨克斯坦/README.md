# 哈萨克斯坦历史

哈萨克斯坦横跨里海、哈萨克草原、阿尔泰与天山北缘，其历史并不是一条从古代民族直接延伸到现代国家的单线谱系，而是草原游牧联盟、绿洲贸易网络、成吉思汗诸兀鲁思、哈萨克汗国、俄罗斯帝国边疆和苏维埃民族共和国层层叠合的结果。现代哈萨克斯坦既继承了草原政治传统，也承受强制集体化饥荒、人口迁徙、塞米巴拉金斯克核试验和资源型工业留下的长期影响。

## 历史主线

- **草原文明的形成：**青铜时代冶金、季节性迁徙和远距离交换奠定草原经济基础；萨卡、乌孙、康居等名称分别来自考古材料和不同传统的文献，不能简单等同于单一现代民族。
- **突厥化与蒙古时代：**6世纪以后突厥语政治共同体不断重组，钦察草原网络进一步改变语言格局；13世纪蒙古征服又把本区纳入术赤、察合台等兀鲁思体系。
- **哈萨克汗国：**15世纪克烈与贾尼别克脱离阿布海儿汗，形成新的政治联盟。“哈萨克”身份在迁徙、战争、婚姻和汗权整合中逐渐稳定，后来出现大、中、小三玉兹的地域性结构。
- **帝国兼并与殖民改造：**18世纪起各玉兹在准噶尔压力和内部竞争下分别接受俄国保护；俄国随后通过堡垒、行政改革、移民与土地征用逐步取消汗权，保护关系演变为直接统治。
- **革命、自治与苏维埃化：**1917年阿拉什自治尝试未能延续。苏维埃政权完成领土和民族行政建制，却以没收牲畜、定居化和集体化造成1931—1933年大饥荒与人口外逃；工业化、流放迁徙和核试验又重塑人口与环境。
- **独立与权力转型：**1991年独立后，纳扎尔巴耶夫以总统制、资源出口与多向外交维持国家整合，同时形成高度集中的权力网络。2019年托卡耶夫接任总统，2022年“一月事件”成为精英重组和宪制改革的转折点；2026年新宪法进一步调整国家机构，但政治开放程度和权力制衡仍需由实际运行检验。

## 重要转折与时间节点

| 时间 | 转折 | 历史意义 |
|---|---|---|
| 前8—前3世纪 | 萨卡文化兴盛 | 骑马、冶金与远距交换塑造早期草原社会，但“萨卡”并非现代国族的直接同义词。 |
| 552年以后 | 突厥诸汗国扩张 | 突厥语政治和文化网络逐渐成为草原主流。 |
| 13世纪 | 蒙古征服 | 本区被纳入成吉思汗诸兀鲁思，后来的汗权合法性深受“黄金家族”传统影响。 |
| 约1465—1466年 | 克烈、贾尼别克建立哈萨克汗国 | 标志新的草原政治共同体形成。 |
| 1723年起 | 准噶尔大举入侵 | 造成严重人口损失，也促使各玉兹寻求外援和重新结盟。 |
| 1731—1847年 | 从俄国保护到汗权终结 | 帝国以分阶段行政和军事扩张取代草原政治自主。 |
| 1916年 | 中亚起义 | 征调令与殖民土地政策引发大规模反抗和镇压。 |
| 1917—1920年 | 阿拉什自治 | 近代民族自治和宪政方案的重要尝试。 |
| 1931—1933年 | 强制集体化饥荒 | 大量人口死亡或外逃，牧业社会结构遭到破坏。 |
| 1949—1989年 | 塞米巴拉金斯克核试验 | 造成跨世代健康与环境损害，并催生反核运动。 |
| 1986年 | “十二月事件” | 反映本地干部任用、民族尊严与苏维埃中央控制之间的矛盾。 |
| 1991年12月16日 | 宣布独立 | 建立哈萨克斯坦共和国，并接手苏维埃领土、工业和核遗产。 |
| 1997—1998年 | 首都迁往阿克莫拉并更名阿斯塔纳 | 强化北部整合和新国家象征，也体现总统主导的国家工程。 |
| 2019年 | 纳扎尔巴耶夫辞职、托卡耶夫接任 | 首次总统权力交接，但旧精英影响并未立即消失。 |
| 2022年1月 | 一月事件 | 燃料抗议扩展为全国危机，集安组织部队短期介入，随后发生精英重组。 |
| 2022—2026年 | 修宪与机构改组 | 政府以“新哈萨克斯坦”名义重塑制度；改革效果取决于竞争、监督和司法实践。 |
