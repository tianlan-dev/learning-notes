# 哈萨克斯坦的早期草原、萨卡与突厥汗国

## 时间

青铜时代至15世纪中叶；重点为前1千纪至蒙古后继时期。

## 概括

今哈萨克斯坦跨越里海—咸海、钦察草原、阿尔泰、七河和锡尔河城镇带，从来不是单一游牧空间。北部和中部草原适合季节牧业，南部绿洲与河谷发展农业、城镇和商路；牧人、农民、工匠与商队构成相互依赖的网络。萨卡、乌孙、康居等名称来自波斯、希腊、中国和考古资料，不能直接等同现代民族。

6世纪以后突厥可汗政治和突厥语逐渐扩展，西突厥、突骑施、葛逻禄、乌古斯、基马克和钦察先后控制不同地区。13世纪蒙古征服把草原分入术赤和察合台体系，金帐汗国、白帐、蒙兀儿斯坦、诺盖和阿布海尔汗国的分化，最终为哈萨克汗国及现代中亚民族政治提供背景。

## 环境与社会机制

| 条件 | 作用 | 需要避免的误解 |
| --- | --- | --- |
| 草原与半荒漠 | 季节迁徙调节降水、积雪和牧草差异，马、羊、骆驼构成运输及财富 | 游牧不是无边界随机移动，而有固定冬牧地、夏牧地和权利关系。 |
| 七河与天山北麓 | 水草、山地—平原垂直迁徙和绿洲城镇共存 | “游牧”与“城市”并非两个隔绝世界。 |
| 锡尔河城镇 | 粮食、手工业、税收和宗教中心，连接草原与河中 | 草原汗国争夺城镇不是放弃牧业，而是争夺财政和贸易。 |
| 阿尔泰与中部矿区 | 铜、锡、金和铁冶金支持武器、装饰与远距离交换 | 金属风格相似不等于所有遗址属于同一民族。 |
| 东西商路 | 与伊朗、河中、中国、伏尔加及黑海相连 | “丝绸之路”是多条变化的网络，不是一条固定道路。 |

## 分阶段发展

### 青铜时代牧业与冶金网络

博泰遗址约前4千纪显示北哈萨克斯坦已存在密集马匹利用；马是否已被现代意义“完全驯化”、博泰马与后来的家马关系仍有争论。前2千纪辛塔什塔—安德罗诺沃相关社会发展轮式车辆、畜牧、农业和冶金，中部晚期别加兹—丹迪拜文化出现大型墓葬与区域精英。考古文化名称表示物质组合，不是可直接写成某一现代民族的王朝。

气候、牧群和冶金促成更远距离移动，但定居点、冬季聚落与耕作长期存在。社会分层可从墓葬和金属品推测，不能据此重建完整君主世系。

### 萨卡世界（约前8—前3世纪）

阿契美尼德铭文把中亚多种游牧群体称为萨卡，希腊作者常使用“斯基泰”作更宽泛称呼。伊塞克、别斯沙特尔、奇利克塔等墓葬显示金器、动物纹样、武器和等级差异；著名“金人”墓主的性别、身份和是否为王并无定论。

大流士一世铭文记载对“尖帽萨卡”等群体作战，波斯帝国也吸收萨卡军队。亚历山大进入河中后在锡尔河一线与草原力量冲突。萨卡政治可能由多个首领和联盟构成，不能排成一张统一国王表。

### 乌孙、康居与匈人时代（约前2世纪—公元5世纪）

中国史籍记载乌孙在伊犁河、伊塞克湖和七河地区建立昆弥领导的联盟，与汉朝通过使节、婚姻和马匹贸易合作，以对抗匈奴及其他力量。乌孙内部有大、小昆弥分立，汉文所载统治者并不能覆盖全部地方社会。

康居以锡尔河中游及塔什干绿洲北缘为核心，控制草原—绿洲商路，与粟特城镇和咸海区域联系。考古学上的康居文化分区与史书“康居国”范围不能完全重合。匈奴及后来被称为“匈人”的迁徙影响草原权力和人口，但从匈奴到欧洲匈人的直系谱系仍有争议。

### 突厥与西突厥汗国（6—7世纪）

552年阿史那氏突厥反抗柔然，迅速建立横跨蒙古高原和中亚的汗国。汗国东西分治逐渐制度化，约603年分裂为东、西两部。西突厥以七河为核心，十姓部落联盟分为咄陆和弩失毕，碎叶、塔拉兹等城镇连接粟特商人与草原王庭。

统叶护可汗时期西突厥势力达到高峰，介入萨珊—拜占庭战争和河中政治；继承斗争、十姓内战和唐朝西进削弱汗国。657年唐军击败阿史那贺鲁后设置羁縻体系，但唐朝控制依赖地方可汗、城镇和驻军，并未把整个草原变成内地式郡县。

### 突骑施、葛逻禄与伊斯兰化（7—10世纪）

突骑施在乌质勒、娑葛和苏禄时期以七河为中心对抗唐朝、倭马亚和河中对手。苏禄死后黄姓、黑姓内斗削弱联盟。751年怛罗斯之战中唐军与阿拔斯军队及其葛逻禄盟友交战，唐军失败；战役是区域权力转折之一，但中亚伊斯兰化是此后数百年城市、贸易、王朝和宗教网络累积的过程，不是一战完成。

766年前后葛逻禄控制碎叶、塔拉兹等地。10世纪喀喇汗王朝在突厥王权和伊斯兰制度结合下兴起，南部城镇、文字与宗教进一步转变。草原和北部仍保持多种信仰与生活方式。

### 乌古斯、基马克与钦察（9—13世纪初）

乌古斯联盟以锡尔河下游和咸海北部为核心，牙护与部落贵族分享权力；部分部众西迁形成塞尔柱力量。基马克联盟在额尔齐斯河及东北草原发展，钦察逐渐从中崛起。11世纪后广阔草原在伊斯兰和俄文资料中被称为钦察草原。

钦察不是单一中央帝国，而是多个部落及王族集团；与花剌子模、罗斯、格鲁吉亚和黑海贸易相连。东南七河又受喀喇契丹控制，形成佛教、景教、伊斯兰和地方信仰并存的政治空间。

### 蒙古征服与术赤—察合台体系（13—15世纪）

1218年讹答剌总督扣押并杀害蒙古商队和使者，成为成吉思汗进攻花剌子模的直接触发。1219—1221年蒙古军攻克讹答剌、锡尔河城镇和河中，战争、迁徙与税制重组造成破坏。征服后，草原大部归术赤后裔，七河和南部部分地区属察合台兀鲁思；边界与牧地会随王族权力变化。

术赤兀鲁思西部发展为金帐汗国，东部白帐诸王在锡尔河和草原活动。14世纪伊斯兰化加深，贸易和城镇恢复；同时瘟疫、王位内战和帖木儿战争削弱金帐统一。东南蒙兀儿斯坦、北部西伯利亚汗国、西部诺盖联盟和阿布海尔汗国并立。克烈、贾尼别克脱离阿布海尔并迁入七河，进入下一阶段。

## 主要政治体比较

| 政治体 / 网络 | 大致时期 | 权力机制 | 转型原因 |
| --- | --- | --- | --- |
| 萨卡诸群体 | 前8—前3世纪 | 区域首领、武士精英、牧业与金属交换 | 与帝国战争、区域迁徙及后续联盟重组；无单一“灭亡日”。 |
| 乌孙 | 前2世纪—约5世纪 | 昆弥、王族分支和部落联盟 | 内部分立、区域政权与突厥化前长期重组。 |
| 康居 | 前2世纪—约4世纪 | 锡尔河绿洲和草原属地网络 | 贵霜、匈人及中亚政治变化，名称逐渐消失。 |
| 西突厥 | 约603—657/8世纪 | 阿史那可汗、十姓部落与粟特城镇 | 继承内战、部落分裂和唐朝军事介入。 |
| 突骑施 | 约699—766年 | 可汗、黄黑两姓和七河据点 | 内斗、阿拔斯与唐竞争、葛逻禄取代。 |
| 葛逻禄—喀喇汗体系 | 8—12世纪 | 部落联盟转向伊斯兰王朝和绿洲财政 | 分支王权、塞尔柱、喀喇契丹和花剌子模压力。 |
| 乌古斯 / 基马克 / 钦察 | 9—13世纪 | 牙护、可汗和松散部落联盟 | 部众西迁、内部分化与蒙古征服。 |
| 术赤、察合台后继诸国 | 13—15世纪 | 成吉思汗王族分封、军政税收与地方城镇 | 王位争夺、区域化和帖木儿战争，分化为多个汗国。 |

## 演变与因果

早期草原政治的兴起通常依赖牧场调节、骑兵动员、王族合法性和绿洲税收；衰落则往往来自继承争斗、部落联盟转移、生态压力和外部帝国介入的组合。名称更替不等于人口整体消失：萨卡、乌孙、突厥、钦察与蒙古时期之间存在迁徙、融合和文化延续，却不能画成单线“祖先—后代”关系。
