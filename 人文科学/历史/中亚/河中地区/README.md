# 河中地区

河中地区通常指阿姆河与锡尔河之间的中亚绿洲地带，核心城市包括撒马尔罕、布哈拉、塔什干、费尔干纳和花剌子模周边。它是粟特、萨曼、喀喇汗、花剌子模、帖木儿和近世汗国的共同历史空间，今日主要分属乌兹别克斯坦、塔吉克斯坦、土库曼斯坦、哈萨克斯坦南部和吉尔吉斯斯坦费尔干纳边缘。
