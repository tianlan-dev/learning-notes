# 河中地区

## 概括

河中地区通常指阿姆河与锡尔河之间的中亚绿洲地带，是撒马尔罕、布哈拉等城市所在区域。它长期连接伊朗世界、突厥草原、伊斯兰文明和丝绸之路贸易，是中亚历史的核心地带之一。

## 主线提示

| 阶段 | 说明 |
|---|---|
| 粟特与绿洲城邦 | 以商业、城市和多语文化为特征。 |
| 阿拉伯征服与伊斯兰化 | 河中地区逐步纳入伊斯兰世界。 |
| 萨曼、喀喇汗与花剌子模 | 波斯—伊斯兰文化与突厥政治相互交织。 |
| 蒙古、察合台与帖木儿 | 蒙古征服后，撒马尔罕成为帖木儿帝国中心。 |
| 布哈拉、希瓦、浩罕 | 近世汗国体系延续河中城市政治传统。 |
