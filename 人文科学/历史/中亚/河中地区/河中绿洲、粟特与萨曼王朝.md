# 河中绿洲、粟特与萨曼王朝

## 时间

古代—10世纪

## 概括

河中绿洲依赖山地冰雪融水灌溉。粟特城邦以撒马尔罕、布哈拉、潘吉肯特等为中心，长期在更大帝国之间保持商业和文化活力。8世纪阿拉伯征服改变政治与宗教格局，9—10世纪萨曼王朝以布哈拉为中心，形成波斯语伊斯兰文化的重要枢纽。

## 主要阶段

| 阶段 | 特征 |
|---|---|
| 阿契美尼德与希腊化时期 | 大夏、粟特、花剌子模进入帝国和跨区域贸易体系 |
| 粟特城邦时期 | 多语商人、祆教等宗教和丝绸之路侨居网络 |
| 阿拉伯征服 | 8世纪倭马亚军队进入河中，城市精英与地方势力抵抗、妥协并存 |
| 萨曼王朝 | 819—999年，布哈拉成为波斯文学、伊斯兰法学与贸易中心 |

## 重要节点

- 751年怛罗斯之战发生在河中与七河边缘，不能简单解释为“造纸术单向传入”的唯一事件。
- 萨曼时期布哈拉与撒马尔罕的繁荣，依赖灌溉农业、商路和城市精英。
- 伊斯兰化并非在征服后立即完成，粟特语、地方习俗和多种宗教长期共存。
