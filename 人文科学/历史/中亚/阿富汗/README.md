# 阿富汗历史

阿富汗位于中亚、伊朗高原和南亚的交会处。巴尔赫面向阿姆河与草原，赫拉特属于呼罗珊城市网络，喀布尔和开伯尔通道连向印度河，坎大哈与赫尔曼德沟通伊朗东缘。历史上这些区域经常分属不同帝国；1747年杜兰尼王权才成为近代阿富汗国家的主要起点，19世纪末的中央化与帝国划界又塑造了大体延续至今的边界。

## 历史主线

- **跨区域走廊**：古代和中世纪的强权往往分别以巴尔赫、赫拉特、加兹尼或喀布尔为核心，不能把现代国界倒推为其疆域。
- **帝国边疆转为本地王权**：希腊总督、月氏部落、萨曼古拉姆、古尔山地首领和萨法维边疆部族都曾在宗主衰弱时建立新政权。
- **1747年的国家重组**：杜兰尼帝国以部族军、波斯语行政和印度方向贡赋迅速扩张；继承战争导致分裂，巴拉克宰随后以喀布尔为中心重建王权。
- **缓冲国与中央集权**：三次英阿战争、英俄划界和阿卜杜勒·拉赫曼的强制中央化共同形成近代领土国家，代价包括征服、迁徙与边界两侧社群分离。
- **改革与政变循环**：阿曼努拉、达乌德、人民民主党和2001年后的共和国都试图快速改造国家，却受财政、地方联盟、外部依赖和政治参与不足制约。
- **长期国际化战争**：1979—2021年的不同战争把苏联、美国、巴基斯坦、伊朗、海湾国家及跨国武装卷入，政权更替与难民、援助和战争经济相互塑造。
- **2021年后的事实统治**：塔利班重建高度集中的酋长国结构。至2026年7月，领土控制较稳定，但政治排他、女性权利限制、经济与人道危机、极端组织和边境紧张仍未解决。

## 重要转折与时间节点

| 时间 | 事件 | 意义 |
|---|---|---|
| 约前250年 | 希腊—巴克特里亚独立 | 巴克特里亚成为希腊化中亚政治中心 |
| 约1—3世纪 | 贵霜帝国 | 连接中亚、印度与远距离贸易的多元帝国 |
| 977—1215年 | 加兹尼与古尔先后兴起 | 阿富汗本地政治中心向呼罗珊和北印度扩张 |
| 1220年代 | 蒙古征服 | 城市与既有王统遭毁灭性冲击 |
| 15世纪 | 帖木儿赫拉特繁荣 | 赫拉特成为波斯语文化、建筑与书画中心 |
| 1709—1738年 | 霍塔克王朝 | 坎大哈起义推翻萨法维边疆统治并一度占领伊斯法罕 |
| 1747年 | 杜兰尼帝国建立 | 近代阿富汗国家的主要起点 |
| 1839—1842年 | 第一次英阿战争 | 英国扶立政权失败，巴拉克宰复位 |
| 1880—1901年 | 阿卜杜勒·拉赫曼统治 | 以强制征服、官僚与英援塑造中央集权国家 |
| 1893年 | 杜兰线协议 | 形成后来阿富汗—巴基斯坦边界争议核心 |
| 1919年 | 第三次英阿战争 | 阿富汗取得外交自主 |
| 1929年 | 阿曼努拉改革政权崩溃 | 王朝支系更替，快速改革路线受挫 |
| 1964年 | 新宪法 | 开启有限议会政治的“民主十年” |
| 1973年 | 达乌德政变 | 王国终结，共和国开始 |
| 1978—1979年 | 四月革命与苏联入侵 | 国内危机转为国际化战争 |
| 1989—1992年 | 苏军撤离与喀布尔政权瓦解 | 援助中断和民兵倒戈开启圣战者政权 |
| 1996年 | 塔利班占领喀布尔 | 第一次伊斯兰酋长国建立 |
| 2001年 | 美国及盟军介入、《波恩协议》 | 塔利班失去城市政权，新的共和国进程启动 |
| 2020—2021年 | 多哈协议、撤军与共和国崩溃 | 塔利班重新控制全国 |
| 2025年 | 俄罗斯正式承认塔利班政府 | 第二次酋长国首次获联合国会员国正式承认 |
