# 阿富汗

## 概括

阿富汗位于伊朗高原、印度次大陆和中亚之间，是山地走廊和帝国边疆。其历史包括巴克特里亚、大夏、贵霜、伊斯兰王朝、加兹尼、古尔、帖木儿、杜兰尼帝国、英俄大博弈和现代阿富汗国家。

## 主线提示

| 阶段 | 说明 |
|---|---|
| 大夏、巴克特里亚与贵霜 | 连接希腊化、中亚和印度文明。 |
| 伊斯兰王朝与突厥—伊朗传统 | 加兹尼、古尔等政权向印度和伊朗方向扩展。 |
| 杜兰尼帝国 | 近代阿富汗国家形成的重要前身。 |
| 英俄大博弈与现代国家 | 夹在俄国、英属印度和区域势力之间。 |
