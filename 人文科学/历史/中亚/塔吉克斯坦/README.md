# 塔吉克斯坦历史

塔吉克斯坦历史横跨粟特、巴克特里亚—吐火罗斯坦、帕米尔山地和波斯语伊斯兰世界。今国家领土从未在古代长期构成单一政治单位：北部胡占德、泽拉夫尚潘吉肯特、南部希萨尔—瓦赫什、东部帕米尔分别隶属不同城市、王朝和边疆体系。现代塔吉克民族名称与共和国边界在苏维埃民族划界中制度化，但其语言文化来源远早于苏维埃时期，也不能简单等同于某一个古代王朝。

## 历史主线

古代粟特是泽拉夫尚—卡什卡河城市与商人网络，巴克特里亚则以阿姆河两岸和兴都库什以北绿洲为中心；两者都超出现代国界。阿契美尼德、亚历山大与希腊化王国、月氏—贵霜、嚈哒和突厥等力量先后进入。705年后阿拉伯军队系统征服河中，潘吉肯特在722年失去本地王权；伊斯兰化与从东伊朗语向新波斯语的转换历时数世纪。

819—999年的萨曼王朝以布哈拉为中心，不是现代塔吉克斯坦疆域国家，却因其波斯语行政、文学和伊斯兰文化在现代塔吉克认同中具有核心象征。其后，喀喇汗、喀喇契丹、花剌子模、蒙古、察合台、帖木儿、昔班尼和布哈拉政权相继统治不同地区。19世纪，胡占德等北部地区被俄罗斯直接吞并，东布哈拉与帕米尔则经历保护国、边界划定和分层行政。

1924年塔吉克自治共和国先置于乌兹别克共和国之内，1929年升格为加盟共和国并获得胡占德地区；撒马尔罕和布哈拉仍在乌兹别克斯坦。苏维埃集体化、棉花专业化、教育、工业和党国干部体系把山地与河谷纳入统一国家。1991年独立后，旧共产党精英、民主与民族主义者、伊斯兰复兴力量和不同地区网络的竞争在1992年升级为内战。1997年和平协议结束大规模战争，随后权力逐渐集中于总统和安全体系。

## 重要转折与时间节点

| 时间 | 转折 | 意义 |
|---|---|---|
| 前6—前4世纪 | 粟特、巴克特里亚进入阿契美尼德体系 | 绿洲、帝国道路与地方贵族结合 |
| 前329—前327年 | 亚历山大征服粟特、巴克特里亚 | 希腊化统治和长期文化混合开始 |
| 1—3世纪 | 贵霜帝国兴盛 | 阿姆河上游连接伊朗、印度和丝路网络 |
| 705—722年 | 阿拉伯系统征服河中、潘吉肯特本地王权终结 | 政治纳入哈里发，宗教和语言转变仍属长期过程 |
| 819—999年 | 萨曼王朝 | 新波斯语伊斯兰文化繁荣，成为现代认同的重要锚点 |
| 1219—1221年 | 蒙古征服河中 | 城市、灌溉和人口受重创，兀鲁思秩序重组 |
| 1866—1868年 | 俄国占领胡占德并使布哈拉成为保护国 | 今国土分属俄国直辖区与布哈拉附庸区 |
| 1924年 | 塔吉克自治共和国建立 | 现代民族行政单位形成 |
| 1929年 | 塔吉克加盟共和国成立 | 胡占德并入，共和国边界和机构升级 |
| 1991-09-09 | 宣布独立 | 苏维埃共和国转为主权国家 |
| 1992—1997年 | 塔吉克内战 | 国家崩解、人口流离失所与地区权力重组 |
| 1997-06-27 | 签署和平与民族和解总协定 | 建立权力分享、解除武装与难民返回框架 |
| 2015—2016年 | 伊斯兰复兴党被取缔、宪法再修订 | 和平后多元安排进一步收缩 |
| 2025-03-13 | 与吉尔吉斯斯坦签署国界条约 | 长期未定边界和武装冲突出现重大转折 |
