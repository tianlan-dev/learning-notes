# 萨赫勒帝国与跨撒哈拉贸易

## 概括

萨赫勒是撒哈拉与稀树草原之间的过渡带。骆驼商队、黄金产区、盐矿和尼日尔河农业使这里成为连接北非与西非的贸易走廊。国家控制的重点往往不是连续领土，而是城市、河道、矿区、市场和贡赋关系。

## 主要帝国

| 政权 | 大致时期 | 中心与特征 |
|---|---|---|
| 瓦加杜—加纳帝国 | 约8—13世纪 | 位于今毛里塔尼亚、马里交界，控制黄金与撒哈拉商路 |
| 马里帝国 | 13—16世纪 | 松迪亚塔建立，曼萨·穆萨时期以黄金、朝觐和廷巴克图闻名 |
| 桑海帝国 | 15—16世纪末 | 以加奥为中心，松尼·阿里和阿斯基亚·穆罕默德时期扩张 |
| 卡涅姆—博尔努 | 约9—19世纪 | 乍得湖区域，连接撒哈拉、尼罗河与豪萨地区 |
| 豪萨城邦 | 中世纪—19世纪 | 卡诺、卡齐纳等发展手工业、商业与伊斯兰学术 |

## 重要事件

- 伊斯兰首先在商人和宫廷中传播，乡村与地方宗教长期并存。
- 1324—1325年曼萨·穆萨朝觐强化马里在地中海世界的声望。
- 廷巴克图、杰内和加奥兼具市场、清真寺与学术中心功能。
- 1591年摩洛哥军队在通迪比击败桑海，尼日尔河大帝国分裂。
- 19世纪富拉尼圣战建立索科托哈里发、马西纳等国家，既是宗教改革也是政治社会革命。

## 辨析

古代“加纳帝国”位于今毛里塔尼亚和马里，不等于现代加纳共和国。现代加纳以这一名称表达对西非帝国传统的认同。
