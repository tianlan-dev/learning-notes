# 毛里塔尼亚历史

毛里塔尼亚连接马格里布、萨赫勒和塞内加尔河谷。桑哈贾柏柏尔联盟、奥达戈斯特商镇和加纳帝国边缘贸易构成中世纪主线，后来阿拉伯马吉尔人迁入，形成摩尔人与河谷非洲人并存的社会。

## 历史坐标

- 地理范围：撒哈拉西部、塞内加尔河下游与大西洋之间。
- 国家形成：现代边界主要在殖民统治和去殖民化中确立，不能倒推为古代统一疆域。
- 阅读方法：先理解区域王国与商路，再理解殖民制度如何重组社会，最后进入独立后的国家建构。
