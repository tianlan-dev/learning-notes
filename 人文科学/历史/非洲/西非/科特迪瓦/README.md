# 科特迪瓦历史

科特迪瓦南北生态差异塑造多种社会。北部与曼德商贸和孔帝国相连，东南部受阿坎迁徙及金、可乐果贸易影响，西南由克鲁等社会组成。海岸缺少大型天然港口，使欧洲早期据点少于黄金海岸。

## 历史坐标

- 地理范围：几内亚湾沿岸、森林可可区与北部稀树草原。
- 国家形成：现代边界主要在殖民统治和去殖民化中确立，不能倒推为古代统一疆域。
- 阅读方法：先理解区域王国与商路，再理解殖民制度如何重组社会，最后进入独立后的国家建构。
