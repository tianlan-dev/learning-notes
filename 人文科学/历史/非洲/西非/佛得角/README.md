# 佛得角历史

佛得角群岛在葡萄牙人15世纪抵达前没有已知永久居民。圣地亚哥岛定居点成为大西洋航线、奴隶贸易和基督教传播节点，欧洲人和被强迫迁来的非洲人共同形成克里奥尔语言与社会。

## 历史坐标

- 地理范围：塞内加尔以西大西洋火山群岛。
- 国家形成：现代边界主要在殖民统治和去殖民化中确立，不能倒推为古代统一疆域。
- 阅读方法：先理解区域王国与商路，再理解殖民制度如何重组社会，最后进入独立后的国家建构。
