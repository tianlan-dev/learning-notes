# 西非

## 概括

西非历史以尼日尔河、萨赫勒和大西洋沿岸为核心，主线包括加纳帝国、马里帝国、桑海帝国、豪萨城邦、伊斯兰贸易、大西洋奴隶贸易和殖民独立。

## 整理建议

- 先按区域通史、主要王国 / 帝国、殖民阶段和现代国家四层组织。
- 注意把非洲内部社会发展、跨撒哈拉 / 印度洋 / 大西洋贸易和欧洲殖民影响区分开。
