# 布基纳法索历史

布基纳法索地区以莫西诸王国最著名。约15世纪起，瓦加杜古、亚滕加、滕科多戈等王国由莫戈纳巴和地方纳巴统治，凭骑兵、贡赋与村社农业维持数百年；西部还有博博、洛比、古鲁西等社会。

## 历史坐标

- 地理范围：沃尔特河上游、萨赫勒与西非草原交界。
- 国家形成：现代边界主要在殖民统治和去殖民化中确立，不能倒推为古代统一疆域。
- 阅读方法：先理解区域王国与商路，再理解殖民制度如何重组社会，最后进入独立后的国家建构。
