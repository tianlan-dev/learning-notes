# 多哥历史

多哥区域处于阿散蒂、达荷美、奥约和北部古尔马世界之间，埃维、卡布列、特姆等人群通过迁徙形成众多地方社会。海岸阿内霍等港口参与棕榈油和奴隶贸易，但没有覆盖全国的统一前殖民国家。

## 历史坐标

- 地理范围：贝宁湾狭长走廊，从海岸延伸至北部草原。
- 国家形成：现代边界主要在殖民统治和去殖民化中确立，不能倒推为古代统一疆域。
- 阅读方法：先理解区域王国与商路，再理解殖民制度如何重组社会，最后进入独立后的国家建构。
