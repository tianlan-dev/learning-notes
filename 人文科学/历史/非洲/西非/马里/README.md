# 马里历史

现代马里位于瓦加杜—加纳、马里和桑海三大萨赫勒帝国的核心或边缘地带。尼日尔河农业、黄金盐贸易和廷巴克图—杰内城市网络，使这里长期成为西非政治与伊斯兰学术中心。

## 历史坐标

- 地理范围：尼日尔河中游、萨赫勒与撒哈拉南缘。
- 国家形成：现代边界主要在殖民统治和去殖民化中确立，不能倒推为古代统一疆域。
- 阅读方法：先理解区域王国与商路，再理解殖民制度如何重组社会，最后进入独立后的国家建构。
