# 塞拉利昂历史

塞拉利昂内陆由泰姆奈、门德、林巴等社会组成，16世纪马内人迁徙促进军事和政治重组。沿海参与大西洋贸易，班斯岛成为英国奴隶贸易据点；18世纪末弗里敦则被规划为获释黑人定居地。

## 历史坐标

- 地理范围：上几内亚海岸、弗里敦半岛与内陆高地。
- 国家形成：现代边界主要在殖民统治和去殖民化中确立，不能倒推为古代统一疆域。
- 阅读方法：先理解区域王国与商路，再理解殖民制度如何重组社会，最后进入独立后的国家建构。
