# 伊斯兰改革、殖民征服与独立

## 时间

18世纪—20世纪后期

## 概括

18—19世纪西非出现多场伊斯兰改革运动。富塔贾隆、富塔托罗、索科托、马西纳和图库洛尔国家试图净化信仰、重建统治并控制贸易。与此同时，法国自塞内加尔向尼日尔河推进，英国从海岸和尼日利亚扩张，德国、葡萄牙也在沿岸建立殖民地。

## 主要转折

| 时间 | 事件 | 影响 |
|---|---|---|
| 1804年起 | 奥斯曼·丹·福迪奥圣战 | 建立索科托哈里发，重组豪萨地区 |
| 1850—1860年代 | 奥马尔·塔勒扩张 | 图库洛尔帝国控制塞内加尔河与尼日尔河部分区域 |
| 1880—1890年代 | 法军击败萨摩里·杜尔等抵抗 | 法属西非领地连接成片 |
| 1897—1903年 | 英国征服贝宁、阿散蒂和索科托 | 英属西非殖民框架成形 |
| 1945年后 | 工会、退伍军人与群众政党兴起 | 自治和独立要求迅速扩大 |
| 1957年 | 加纳独立 | 撒哈拉以南独立浪潮的重要开端 |
| 1960年 | 多个法属西非国家独立 | 法属联邦瓦解为主权国家 |

## 殖民遗产

法国殖民地多被纳入法属西非联邦，英国更常保留不同保护国和地方权力结构，葡萄牙则长期拒绝去殖民化。这些差异影响独立后的行政语言、货币、边界和政党制度，但各国政治不能仅由殖民宗主国解释。
