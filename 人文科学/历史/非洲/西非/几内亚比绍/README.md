# 几内亚比绍历史

今几内亚比绍内陆曾属马里和卡布帝国网络，沿海与群岛由巴兰特、佩佩尔、比热戈等社会控制。葡萄牙商人15世纪后进入，但数百年主要依靠沿岸据点和非洲中间人进行贸易。

## 历史坐标

- 地理范围：塞内冈比亚南部海岸与比热戈斯群岛。
- 国家形成：现代边界主要在殖民统治和去殖民化中确立，不能倒推为古代统一疆域。
- 阅读方法：先理解区域王国与商路，再理解殖民制度如何重组社会，最后进入独立后的国家建构。
