# 塞内加尔历史

塞内加尔河谷是跨撒哈拉贸易和伊斯兰传播的重要通道，沃洛夫、塞雷尔、富拉尼和曼丁等社会建立卓洛夫、瓦洛、卡约尔与富塔托罗等国家。沿岸岛屿和河口后来成为法国在西非最早的据点。

## 历史坐标

- 地理范围：塞内加尔河下游、萨赫勒西端与大西洋沿岸。
- 国家形成：现代边界主要在殖民统治和去殖民化中确立，不能倒推为古代统一疆域。
- 阅读方法：先理解区域王国与商路，再理解殖民制度如何重组社会，最后进入独立后的国家建构。
