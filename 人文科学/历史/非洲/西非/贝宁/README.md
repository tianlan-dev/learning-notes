# 贝宁历史

贝宁共和国与尼日利亚的贝宁王国不是同一政治体。今贝宁南部由阿贾、丰和约鲁巴国家组成，阿波美的达荷美王国17世纪后扩张；北部博尔古地区则有尼基等巴里巴王国。

## 历史坐标

- 地理范围：贝宁湾沿岸、阿波美高原与北部博尔古。
- 国家形成：现代边界主要在殖民统治和去殖民化中确立，不能倒推为古代统一疆域。
- 阅读方法：先理解区域王国与商路，再理解殖民制度如何重组社会，最后进入独立后的国家建构。
