# 冈比亚历史

冈比亚河把内陆萨赫勒与大西洋相连。曼丁、沃洛夫、富拉尼、乔拉等人群参与马里、卡布和塞内冈比亚国家网络，河口自15世纪起成为欧洲黄金、象牙和奴隶贸易据点。

## 历史坐标

- 地理范围：冈比亚河两岸、被塞内加尔大部环绕。
- 国家形成：现代边界主要在殖民统治和去殖民化中确立，不能倒推为古代统一疆域。
- 阅读方法：先理解区域王国与商路，再理解殖民制度如何重组社会，最后进入独立后的国家建构。
