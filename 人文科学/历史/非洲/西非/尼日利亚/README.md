# 尼日利亚历史

尼日利亚是现代殖民边界整合的多中心历史空间。诺克文化、伊费和贝宁的城市艺术、约鲁巴奥约帝国、尼日尔三角洲城邦、豪萨城市、卡涅姆—博尔努及索科托哈里发，代表彼此不同的政治文化传统。

## 历史坐标

- 地理范围：尼日尔河—贝努埃河流域、乍得湖西南与几内亚湾。
- 国家形成：现代边界主要在殖民统治和去殖民化中确立，不能倒推为古代统一疆域。
- 阅读方法：先理解区域王国与商路，再理解殖民制度如何重组社会，最后进入独立后的国家建构。
