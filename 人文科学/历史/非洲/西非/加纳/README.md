# 加纳历史

现代加纳并非古代加纳帝国所在地，而是以黄金海岸命名的殖民地继承国。阿坎人国家、北方莫莱—达贡巴王国、加—阿当梅城市和沿海芳蒂联盟共同塑造其历史，阿散蒂在18—19世纪成为内陆强国。

## 历史坐标

- 地理范围：黄金海岸、沃尔特河流域与北部草原。
- 国家形成：现代边界主要在殖民统治和去殖民化中确立，不能倒推为古代统一疆域。
- 阅读方法：先理解区域王国与商路，再理解殖民制度如何重组社会，最后进入独立后的国家建构。
