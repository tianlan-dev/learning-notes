# 利比里亚历史

利比里亚地区原有克佩勒、巴萨、克鲁、格雷博、马诺等多种社会，其政治从分散宗族到区域首领联盟不等。19世纪美国殖民协会在海岸安置自由黑人移民，形成与本地多数人口区隔的美国—利比里亚人共同体。

## 历史坐标

- 地理范围：上几内亚海岸与西非森林带。
- 国家形成：现代边界主要在殖民统治和去殖民化中确立，不能倒推为古代统一疆域。
- 阅读方法：先理解区域王国与商路，再理解殖民制度如何重组社会，最后进入独立后的国家建构。
