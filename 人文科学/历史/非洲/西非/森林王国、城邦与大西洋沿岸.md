# 森林王国、城邦与大西洋沿岸

## 概括

西非森林带并非萨赫勒帝国的边缘。尼日尔河下游、黄金海岸和贝宁湾发展出伊费、奥约、贝宁、达荷美、博诺曼和阿散蒂等城市与王国。它们控制农产、黄金、可乐果、布匹和区域市场，并在大西洋贸易到来后重新调整战争和外交。

## 主要政治体

| 政治体 | 区域 | 特征 |
|---|---|---|
| 伊费 | 今尼日利亚西南 | 约鲁巴宗教与王权中心，以自然主义雕塑著称 |
| 奥约帝国 | 尼日利亚西南—贝宁北部 | 骑兵、贡赋和约鲁巴城市网络 |
| 贝宁王国 | 尼日利亚南部 | 奥巴王权、城墙、铜牌与象牙艺术 |
| 达荷美 | 今贝宁 | 中央集权、军事组织与大西洋奴隶贸易 |
| 阿散蒂 | 今加纳中部 | 金凳象征、联邦结构、黄金与战争 |
| 富塔贾隆 | 今几内亚高地 | 富拉尼伊斯兰政权和区域贸易节点 |

## 大西洋转型

葡萄牙人15世纪抵达海岸后，欧洲商人最初寻求黄金、胡椒和象牙，奴隶贸易随后扩大。沿海堡垒并不等于欧洲立即控制内陆；非洲国家长期规定交易条件。枪械、奴隶需求与继承竞争改变战争规模，也促使阿散蒂、达荷美等国家扩张。

19世纪废奴和“合法贸易”时期，棕榈油、花生、橡胶与可可出口增长。新的商业精英、传教学校和殖民军事据点为之后的直接征服铺路。
