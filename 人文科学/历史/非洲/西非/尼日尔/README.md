# 尼日尔历史

尼日尔横跨撒哈拉和萨赫勒。西部尼日尔河谷受桑海帝国影响，南部有豪萨城邦，东部连接卡涅姆—博尔努，北部艾尔高原由图阿雷格联盟控制。商队、盐、牲畜和农业使不同生态区相互依赖。

## 历史坐标

- 地理范围：撒哈拉中部、艾尔高原、尼日尔河弯与乍得湖西缘。
- 国家形成：现代边界主要在殖民统治和去殖民化中确立，不能倒推为古代统一疆域。
- 阅读方法：先理解区域王国与商路，再理解殖民制度如何重组社会，最后进入独立后的国家建构。
