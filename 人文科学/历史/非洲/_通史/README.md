# 非洲通史

## 概括

本目录整理跨越西非、东非、中非和南部非洲分区以及现代国家边界的共同历史专题。各分区与国家笔记维护地方政权和社会的具体脉络；这里关注大陆尺度的人口与技术传播、贸易网络、殖民体系、独立进程和泛非合作。
