# 阿克苏姆、埃塞俄比亚与非洲之角

## 概括

非洲之角连接尼罗河、红海、阿拉伯半岛和印度洋。厄立特里亚与埃塞俄比亚高原的阿克苏姆王国控制内陆农产与红海贸易，4世纪接受基督教。其后高原王朝、穆斯林苏丹国和牧民社会长期竞争、贸易与交流。

## 主要政权

| 政权 | 大致时期 | 特征 |
|---|---|---|
| 达马特及早期高原社会 | 前1千纪 | 南阿拉伯联系与本地农业王权 |
| 阿克苏姆 | 约1—7世纪 | 铸币、红海贸易、格尔兹文与基督教国家 |
| 扎格维王朝 | 约12—13世纪 | 拉利贝拉岩石教堂与高原基督教 |
| 所罗门王朝 | 1270—1974年 | 以所罗门谱系建构正统，疆域和中央权力不断变化 |
| 阿达尔等穆斯林苏丹国 | 中世纪—16世纪 | 控制东部商路，与高原王国战争与贸易并存 |
| 索马里苏丹国 | 中世纪—19世纪 | 阿居兰、格莱迪、马吉尔廷等控制河谷、港口或牧民网络 |

## 关键转折

16世纪阿达尔统帅艾哈迈德·格兰发动战争，葡萄牙与奥斯曼势力介入；随后奥罗莫迁徙与地方重组改变高原政治。19世纪特沃德罗斯二世、约翰内斯四世和孟尼利克二世推动再中央化。1896年埃塞俄比亚在阿德瓦击败意大利，成为反殖民象征，但意大利仍控制厄立特里亚并在1936—1941年短暂占领埃塞俄比亚。
