# 大湖王国、殖民统治与独立

## 概括

维多利亚湖、坦噶尼喀湖与高原农业带人口密集，班图语、尼罗特语和中苏丹语人群长期互动。布尼奥罗、布干达、卢旺达、布隆迪、卡拉圭等王国通过土地、牛群、军队和宗族建立中央权力，但社会身份远比殖民时期固定化的“族群”分类复杂。

## 主要王国

| 王国 | 区域 | 特征 |
|---|---|---|
| 布尼奥罗—基塔拉 | 今乌干达西部 | 与巴奇韦齐传说相连，控制盐场和区域贸易 |
| 布干达 | 维多利亚湖北岸 | 卡巴卡王权、舰队和地方官制，19世纪迅速扩张 |
| 卢旺达 | 高原内陆 | 姆瓦米王权、牛群关系和军事扩张 |
| 布隆迪 | 坦噶尼喀湖东北 | 王权与地方氏族平衡，社会等级具有流动性 |
| 卡拉圭 | 今坦桑尼亚西北 | 牛牧王权和大湖贸易节点 |

## 殖民重组

英国把布干达等王国纳入乌干达保护国，德国建立德属东非，比利时在一战后取得卢旺达—乌隆迪托管地。殖民政府通过首领制、人口登记、学校和“种族”理论固化社会分类，并以税收、棉花、咖啡和劳工迁移重塑经济。

## 独立与冲突

坦噶尼喀和乌干达较早通过群众政党独立，肯尼亚经历定居殖民与茅茅战争，卢旺达和布隆迪的去殖民化则伴随王权崩溃、流亡和族群暴力。大湖地区现代战争跨越国界，必须与刚果东部、难民流动和殖民身份制度一并理解。
