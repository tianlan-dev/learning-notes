# 厄立特里亚历史

厄立特里亚高原与阿克苏姆文明相连，沿海阿杜利斯、马萨瓦连接红海贸易。中世纪米德里巴赫里等高原政治体、奥斯曼控制的沿岸港口和阿法尔、贝扎、提格雷等社会并存。现代领土由意大利殖民边界整合。

## 历史坐标

- 地理范围：红海西岸、厄立特里亚高原与达纳基勒低地。
- 国家形成：现代边界由区域历史、殖民分割和去殖民化共同塑造。
- 阅读方法：把内陆社会、海洋网络和跨境人群放在国家边界之前理解。
