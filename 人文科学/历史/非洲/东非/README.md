# 东非

## 概括

东非历史包括尼罗河上游、埃塞俄比亚高原、阿克苏姆、斯瓦希里海岸和印度洋贸易网络，连接非洲内陆、阿拉伯、印度和欧洲殖民体系。

## 整理建议

- 先按区域通史、主要王国 / 帝国、殖民阶段和现代国家四层组织。
- 注意把非洲内部社会发展、跨撒哈拉 / 印度洋 / 大西洋贸易和欧洲殖民影响区分开。
