# 东非历史

东非跨越非洲之角、尼罗河上游、大湖区和印度洋岛屿。埃塞俄比亚高原形成阿克苏姆与所罗门王朝传统；斯瓦希里城市把非洲内陆商品连接到阿拉伯、波斯和印度洋；大湖区发展布干达、卢旺达、布隆迪等王国；马达加斯加则由南岛与非洲人群共同塑造。
