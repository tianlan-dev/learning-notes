# 毛里求斯历史

毛里求斯在欧洲航海者到来前没有永久居民。荷兰、法国和英国先后殖民，引入非洲与马达加斯加奴隶、欧洲定居者和印度契约劳工，形成克里奥尔、印度裔、华裔与法裔等多元社会。

## 历史坐标

- 地理范围：马斯克林群岛毛里求斯本岛及附属岛屿。
- 国家形成：现代边界由区域历史、殖民分割和去殖民化共同塑造。
- 阅读方法：把内陆社会、海洋网络和跨境人群放在国家边界之前理解。
