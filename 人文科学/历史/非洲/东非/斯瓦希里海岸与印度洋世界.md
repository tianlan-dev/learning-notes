# 斯瓦希里海岸与印度洋世界

## 概括

斯瓦希里海岸从索马里南部延伸到莫桑比克北部及近海岛屿。班图语居民通过季风航海与阿拉伯、波斯、印度和更远地区贸易，形成以斯瓦希里语和伊斯兰文化为核心的港市社会。基尔瓦、蒙巴萨、马林迪、拉穆、桑给巴尔等城市各自独立，未构成单一帝国。

## 贸易与城市

| 时期 | 特征 |
|---|---|
| 公元1千纪前期 | 沿海聚落扩大，铁器、农业、捕鱼与海贸结合 |
| 10—15世纪 | 石造清真寺与商人住宅出现，黄金、象牙输入港口，陶瓷与布匹输入内陆 |
| 16—17世纪 | 葡萄牙以舰队和堡垒控制部分航路，地方抵抗持续 |
| 18—19世纪 | 阿曼布赛义德王朝驱逐葡萄牙势力，桑给巴尔成为丁香、象牙与奴隶贸易中心 |
| 19世纪末 | 德国、英国、意大利和葡萄牙殖民边界切割海岸与内陆 |

## 文化形成

斯瓦希里语属于班图语支，吸收大量阿拉伯语词汇。沿海居民的伊斯兰认同和海外联系并不意味着其社会是“阿拉伯殖民地”；考古与语言证据显示本地非洲连续性占核心。港市精英通过婚姻、宗教谱系和进口商品表达地位。

## 影响

印度洋贸易把东非内陆、刚果东部和南部非洲黄金产区连接到全球市场。19世纪商队扩展也加剧奴隶捕掠和象牙采集，随后成为欧洲殖民者进入内陆的路线。
