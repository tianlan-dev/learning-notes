# 乍得历史

乍得跨越撒哈拉商路、乍得湖农牧区和南部草原。萨奥文化留下城镇与金属传统，卡涅姆—博尔努、巴吉尔米和瓦达伊等伊斯兰国家控制商队、贡赋和奴隶贸易；南部萨拉等社会则有不同政治组织。

## 历史坐标

- 地理范围：撒哈拉、萨赫勒、乍得湖与南部草原交界。
- 国家形成：现代边界主要继承法、比、葡、德、西等殖民体系，不等同于任何单一旧王国。
- 阅读方法：区分森林与草原社会、内陆贸易与殖民资源飞地，再理解独立后的国家危机。
