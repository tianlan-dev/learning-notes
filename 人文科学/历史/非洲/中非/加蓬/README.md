# 加蓬历史

加蓬森林和河流长期由巴卡等采集狩猎者及米耶内、芳、普努等班图语族社会共同居住。沿岸奥伦古王国和洛安果贸易网络参与象牙、奴隶与欧洲商品交换，奥果韦河是进入内陆的主要通道。

## 历史坐标

- 地理范围：赤道大西洋海岸、奥果韦河流域与刚果森林。
- 国家形成：现代边界主要继承法、比、葡、德、西等殖民体系，不等同于任何单一旧王国。
- 阅读方法：区分森林与草原社会、内陆贸易与殖民资源飞地，再理解独立后的国家危机。
