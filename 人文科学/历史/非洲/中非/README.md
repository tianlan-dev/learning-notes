# 中非

## 概括

中非历史围绕刚果盆地、森林地带和内陆王国展开，重点包括刚果王国、卢巴、隆达、奴隶贸易、比利时和法国殖民重组。

## 整理建议

- 先按区域通史、主要王国 / 帝国、殖民阶段和现代国家四层组织。
- 注意把非洲内部社会发展、跨撒哈拉 / 印度洋 / 大西洋贸易和欧洲殖民影响区分开。
