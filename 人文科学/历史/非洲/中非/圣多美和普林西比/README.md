# 圣多美和普林西比历史

圣多美和普林西比在葡萄牙人15世纪抵达前没有已知永久居民。葡萄牙以罪犯、犹太儿童和定居者建立据点，并强迫大量非洲奴隶种植甘蔗，形成欧洲人、自由黑人、奴隶和逃亡者组成的克里奥尔社会。

## 历史坐标

- 地理范围：几内亚湾赤道附近火山岛屿。
- 国家形成：现代边界主要继承法、比、葡、德、西等殖民体系，不等同于任何单一旧王国。
- 阅读方法：区分森林与草原社会、内陆贸易与殖民资源飞地，再理解独立后的国家危机。
