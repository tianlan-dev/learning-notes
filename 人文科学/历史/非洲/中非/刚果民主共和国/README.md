# 刚果民主共和国历史

刚果民主共和国覆盖庞大刚果盆地，历史上没有单一统一前身。西部连于刚果王国，南部有卢巴、隆达和加丹加贸易国家，中部库巴形成联邦王权，东部则与斯瓦希里—阿拉伯商队和大湖区相连。

## 历史坐标

- 地理范围：刚果盆地、东非大湖区西缘与加丹加矿区。
- 国家形成：现代边界主要继承法、比、葡、德、西等殖民体系，不等同于任何单一旧王国。
- 阅读方法：区分森林与草原社会、内陆贸易与殖民资源飞地，再理解独立后的国家危机。
