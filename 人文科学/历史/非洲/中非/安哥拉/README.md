# 安哥拉历史

安哥拉北部属于刚果王国文化圈，中部高原形成本格拉、比耶等奥温本杜国家，宽扎河流域有恩东戈和马坦巴。葡萄牙1575年建立罗安达后以奴隶贸易和军事联盟向内陆扩张，但直到19世纪末至20世纪初才控制大部分现代疆域。

## 历史坐标

- 地理范围：大西洋中南部海岸、刚果河下游南侧与内陆高原。
- 国家形成：现代边界主要继承法、比、葡、德、西等殖民体系，不等同于任何单一旧王国。
- 阅读方法：区分森林与草原社会、内陆贸易与殖民资源飞地，再理解独立后的国家危机。
