# 殖民资源体系、独立与中非冲突

## 时间

19世纪末至今

## 概括

中非殖民统治高度依赖橡胶、象牙、木材、铜、铀、石油和种植园。刚果自由邦的特许公司和武装征收造成大规模暴力；法属赤道非洲也以特许公司、税役和刚果—海洋铁路劳工著称。德国殖民失败后，喀麦隆被英法分割，形成持久的语言与联邦问题。

## 主要殖民单元

| 殖民单元 | 宗主国 | 特征 |
|---|---|---|
| 刚果自由邦、比属刚果 | 比利时 | 私人国王领地转为殖民地，矿业与家长式统治 |
| 法属赤道非洲 | 法国 | 加蓬、中刚果、乌班吉沙立、乍得的联邦框架 |
| 德属喀麦隆及英法托管地 | 德国、后英法 | 一战后分治，法区与尼日利亚联系不同 |
| 西属几内亚 | 西班牙 | 比奥科种植园与大陆木材，殖民统治延续至1968年 |
| 葡属安哥拉、圣多美 | 葡萄牙 | 强制劳工、种植园与长期解放战争 |

## 独立与冷战

1960年前后多数中非国家迅速独立，但殖民行政薄弱、军队与地方权力失衡。刚果危机成为联合国、比利时、美国、苏联和地区力量介入的冷战焦点。安哥拉独立后内战持续至2002年。1990年代卢旺达大屠杀后，难民与武装进入刚果东部，推动第一次、第二次刚果战争。

## 结构性问题

资源丰富并不自动转化为公共能力。矿区飞地、外债、跨国公司、军队商业和边境走私长期影响国家财政；但地区社会也通过教会、工会、城市组织和跨境市场维持治理与生计。
