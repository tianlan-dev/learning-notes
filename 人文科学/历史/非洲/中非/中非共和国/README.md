# 中非共和国历史

中非共和国所在区域长期连接乍得湖、尼罗河与刚果盆地。班达、格巴亚、曼贾、赞德等社会受19世纪北方奴隶捕掠和武装商人扩张影响，达尔库提等苏丹国在东北形成；殖民边界将这些差异地区纳入乌班吉沙立。

## 历史坐标

- 地理范围：乌班吉河以北、乍得盆地以南和刚果林边缘。
- 国家形成：现代边界主要继承法、比、葡、德、西等殖民体系，不等同于任何单一旧王国。
- 阅读方法：区分森林与草原社会、内陆贸易与殖民资源飞地，再理解独立后的国家危机。
