# 博茨瓦纳历史

博茨瓦纳长期由桑人、科伊人与茨瓦纳语族社会共同居住。18—19世纪夸纳、恩瓦凯采、恩瓦托、塔瓦纳等茨瓦纳酋邦形成大型城镇，由科西与集会制度协调土地、牛群和劳役；北部奥卡万戈还有不同河流社会。

## 历史坐标

- 地理范围：卡拉哈里盆地、奥卡万戈三角洲与南部高原。
- 国家形成：殖民边界、矿业劳工体系和区域解放战争共同塑造现代国家。
- 阅读方法：把内陆王国、定居殖民和南非区域经济放在同一时间线上理解。
