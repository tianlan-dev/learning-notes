# 祖鲁、索托、茨瓦纳与十九世纪国家重组

## 时间

18世纪末—19世纪

## 概括

19世纪初南部非洲发生广泛国家重组、战争、迁徙与贸易变化。祖鲁王国在恰卡统治下强化年龄团军制和中央王权；莫舒舒一世在山地建立巴苏陀国家；恩德贝莱、斯威士、加沙等政权通过迁徙建立新中心。过去常用“姆费卡内”把一切归因于祖鲁扩张，现代研究更强调奴隶贸易、干旱、竞争和殖民边疆的共同作用。

## 主要国家

| 政权 | 建立者或核心 | 特征 |
|---|---|---|
| 祖鲁王国 | 恰卡及其继承者 | 军事年龄团、牛群与中央王权 |
| 巴苏陀王国 | 莫舒舒一世 | 山地防御、吸收难民与外交平衡 |
| 恩德贝莱王国 | 姆齐利卡齐 | 从高地草原迁至今津巴布韦西部 |
| 斯威士王国 | 索布扎一世、姆斯瓦蒂二世 | 王族双重中心和地区整合 |
| 加沙帝国 | 索尚加内 | 控制今莫桑比克南部及周边 |
| 茨瓦纳酋邦 | 多个科西王权 | 城镇化聚落、牛牧和贸易网络 |

## 殖民碰撞

布尔人大迁徙者、开普殖民军、葡萄牙势力和传教士进入同一空间。国家间战争与殖民征服相互叠加，最终形成纳塔尔、奥兰治自由邦、德兰士瓦、巴苏陀兰、贝专纳等殖民政治单元。
