# 南部非洲

## 概括

南部非洲历史包括大津巴布韦、班图社会、科伊桑族群、祖鲁、布尔人、英国殖民、南非联邦、种族隔离和现代南部非洲国家。

## 整理建议

- 先按区域通史、主要王国 / 帝国、殖民阶段和现代国家四层组织。
- 注意把非洲内部社会发展、跨撒哈拉 / 印度洋 / 大西洋贸易和欧洲殖民影响区分开。
