# 南部非洲历史

南部非洲历史包括科伊桑采集狩猎与牧业社会、班图语族农业扩散、马蓬古布韦和大津巴布韦等高原国家，以及祖鲁、索托、恩德贝莱等19世纪政权。葡萄牙、荷兰和英国殖民扩张、钻石黄金矿业与白人定居政权最终形成南非主导的区域劳工体系。
