# 科尔基斯、伊比利亚与基督教化

## 时间

古代至10世纪

## 概括

现代格鲁吉亚的古代基础主要来自黑海东岸科尔基斯和东部高加索伊比利亚。前者连接黑海贸易与希腊殖民网络，后者在罗马和伊朗之间发展王权。4世纪伊比利亚接受基督教后，格鲁吉亚教会、文字和君主传统成为跨越政治分裂的重要纽带。

## 主要政体

| 政体或阶段 | 区域 | 特点 |
|---|---|---|
| 科尔基斯 | 黑海东岸 | 港口、金属资源与希腊城邦贸易联系密切 |
| 高加索伊比利亚 | 今格鲁吉亚东部 | 在罗马与伊朗势力之间保持地方王权 |
| 拉齐卡 | 格鲁吉亚西部 | 6世纪成为拜占庭—萨珊战争的重要战场 |
| 阿拉伯统治与地方公国 | 第比利斯及周边、山地公国 | 埃米尔、地方王族与教会并存 |
| 巴格拉季昂诸公国 | 西南与中部格鲁吉亚 | 为后来的王国统一奠定王朝基础 |

## 重要事件

- 希腊殖民城市在黑海沿岸建立贸易据点，但地方社会保持自身政治文化。
- 罗马与安息 / 萨珊围绕高加索通道和伊比利亚王位长期竞争。
- 格鲁吉亚传统将圣尼诺与4世纪王权基督教化联系起来。
- 6世纪拉齐卡战争显示黑海东岸在拜占庭—萨珊竞争中的战略地位。
- 7世纪起阿拉伯势力进入高加索，第比利斯成为穆斯林统治中心之一。
- 格鲁吉亚文字、教会和修道院网络在晚期古代与中世纪早期巩固文化共同体。
