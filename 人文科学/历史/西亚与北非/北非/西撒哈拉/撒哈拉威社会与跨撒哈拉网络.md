# 西撒哈拉的撒哈拉威社会与跨撒哈拉网络

## 时间

古代—1884年

## 概括

西撒哈拉在殖民划界前是大西洋沿岸、干旱草场、绿洲和商队通道组成的流动空间。桑哈贾等阿马齐格联盟、后来迁入的阿拉伯部族、宗教家族和附属群体在长期互动中形成哈桑尼亚语和撒哈拉威社会。亲族、保护关系、放牧权和水源使用比固定领土边界更重要。

## 社会与政治结构

| 结构 | 作用 |
|---|---|
| 部族和支系 | 组织亲族认同、资源分配、防卫与赔偿 |
| 战士群体 | 提供武装保护，参与商路和地区竞争 |
| 宗教家族与学者 | 教育、仲裁、宗教声望和跨地区联系 |
| 牧民与商队 | 在水源、草场、盐场和市场之间流动 |
| 绿洲与沿海聚落 | 提供农业、渔业、补给和贸易节点 |
| 受支配群体和奴隶 | 承担劳动与家内生产，反映历史社会不平等 |

## 重要事件与过程

- 古代西撒哈拉受气候干旱化影响，人口和贸易围绕水源、山地与沿岸重新分布。
- 骆驼运输扩大后，西撒哈拉成为摩洛哥南部与萨赫勒之间的重要通道。
- 桑哈贾联盟参与黄金、盐和奴隶贸易，并在11世纪穆拉比特运动中发挥核心作用。
- 马吉尔等阿拉伯部族长期迁徙推动哈桑尼亚阿拉伯语传播，但阿马齐格传统并未完全消失。
- 苏菲教团和宗教学者通过教育、仲裁和朝觐建立超越地方的权威。
- 摩洛哥苏丹与撒哈拉部族之间存在宣誓、贡赋、贸易和宗教联系，但强度与连续性因时期和群体而异。
- 欧洲渔民和商人逐渐在大西洋沿岸活动，19世纪后殖民竞争转向正式占领。
