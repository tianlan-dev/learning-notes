# 摩洛哥的古代毛里塔尼亚、伊德里斯与早期国家

## 时间

古代—约11世纪中叶

## 概括

今摩洛哥地区在古代由多种阿马齐格社会构成，并与腓尼基、迦太基和罗马地中海网络互动。罗马在北部建立毛里塔尼亚廷吉塔纳行省，但其边界以外仍有独立地方共同体。这里的“毛里塔尼亚”是古代地理名称，不等于现代[毛里塔尼亚](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E9%9D%9E%E6%B4%B2/%E8%A5%BF%E9%9D%9E/%E6%AF%9B%E9%87%8C%E5%A1%94%E5%B0%BC%E4%BA%9A/README.md)。

7—8世纪阿拉伯军队进入马格里布后，伊斯兰逐步传播。伊德里斯王朝以非斯为中心，将先知后裔的宗教声望、阿马齐格部族支持与城市国家建设结合，成为摩洛哥早期王朝传统的重要来源。

## 统治结构

| 政治阶段 | 核心区域 | 统治特征 |
|---|---|---|
| 阿马齐格地方社会 | 山地、平原和绿洲 | 部族联盟、农业聚落、牧业与跨区贸易并存 |
| 毛里塔尼亚王国 | 摩洛哥北部与阿尔及利亚西部 | 地方王权与罗马结盟、竞争 |
| 罗马毛里塔尼亚廷吉塔纳 | 廷吉斯、沃吕比利斯等 | 城市、行省行政和边疆驻军 |
| 早期伊斯兰地方政权 | 北部城市与山地 | 阿拉伯总督、哈瓦利吉运动和地方联盟交错 |
| 伊德里斯王朝 | 非斯及周边 | 谢里夫合法性、城市中心与部族支持结合 |

## 重要事件

- 腓尼基商站将大西洋和地中海沿岸纳入海上交换网络。
- 前1世纪至1世纪，罗马逐步控制毛里塔尼亚王国北部并建立行省。
- 7世纪末至8世纪，阿拉伯征服带来新的行政与宗教网络，但地方反抗和自主政权持续出现。
- 740年代的阿马齐格起义削弱倭马亚对远西马格里布的直接控制。
- 788年伊德里斯一世在当地部族支持下建立统治；其后非斯发展为政治、商业与学术中心。
- 10—11世纪，多股地方与外来势力竞争，最终为穆拉比特运动兴起创造条件。
