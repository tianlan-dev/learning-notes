# 摩洛哥的保护国、独立与现代国家

## 时间

1912年至今

## 概括

1912年《非斯条约》建立法国保护国，西班牙控制北部与南部部分区域，丹吉尔后来形成国际区。苏丹制度仍被保留，但殖民当局掌握军事、财政和行政实权。民族主义者与王室逐步形成合作，苏丹穆罕默德五世被流放反而强化独立运动，摩洛哥于1956年恢复主权。

独立后的君主制通过军队、行政体系、政党竞争和宗教合法性整合国家。西撒哈拉问题、社会经济不平等、城市化和政治改革长期影响国家发展。

## 统治结构

| 阶段 | 国家权力结构 | 主要特征 |
|---|---|---|
| 法国与西班牙保护区 | 苏丹名义统治，殖民总督掌握实权 | 土地、城市和基础设施被殖民经济重组 |
| 独立初期 | 穆罕默德五世与民族运动协作 | 合并原保护区、建设中央行政与军队 |
| 哈桑二世时期 | 强势君主制、议会和政党并存 | 王权集中，经历军事政变企图、反对派压制与有限开放 |
| 1999年以后 | 君主制主导下的制度改革 | 家庭法、人权和基础设施改革并行，社会抗议持续 |

## 重要事件

- 1912年，《非斯条约》确立法国保护国；西班牙保护区随后形成。
- 1921—1926年，里夫共和国和阿卜杜勒·克里姆领导的战争挑战西班牙与法国统治。
- 1944年，独立党发表独立宣言，民族主义进入更明确的主权诉求阶段。
- 1953年，法国流放穆罕默德五世，引发更广泛抵抗；1955年其回国。
- 1956年，法国和西班牙结束主要保护安排，摩洛哥恢复独立。
- 1961年，哈桑二世即位；1962年颁布宪法。
- 1975年绿色进军发生，西班牙撤离后摩洛哥控制西撒哈拉大部分地区，但其最终地位未解决。
- 1999年，王位更替开启新的改革阶段。
- 2011年抗议推动宪法修订，但国王仍保留广泛权力。
