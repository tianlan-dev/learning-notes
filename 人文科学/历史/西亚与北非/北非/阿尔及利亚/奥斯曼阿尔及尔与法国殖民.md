# 阿尔及利亚的奥斯曼阿尔及尔与法国殖民

## 时间

1516—1954年

## 概括

16世纪初，阿尔及尔面对西班牙海上扩张，引入巴巴罗萨兄弟的军事力量并承认奥斯曼宗主权。阿尔及尔摄政成为具有高度地方自主性的奥斯曼属地，以港口舰队、军政集团、税收和与内陆部族的盟约维系统治。

1830年法国占领阿尔及尔，随后通过数十年战争征服内陆。土地被大量转移给欧洲移民，穆斯林多数居民受不平等法律体系约束。改革、同化和有限代表无法化解结构性矛盾，民族主义最终转向独立革命。

## 统治结构

| 阶段 | 权力结构 | 主要特征 |
|---|---|---|
| 奥斯曼摄政早期 | 贝勒贝伊、帕夏与禁卫军 | 名义服从奥斯曼苏丹，地方海军和军队掌握实权 |
| 代伊统治 | 由军政集团推举的代伊 | 阿尔及尔自治程度提高，地方贝伊治理各省 |
| 法国军事征服 | 总督、军队与移民殖民机构 | 长期战争、土地没收和人口迁徙 |
| 法属阿尔及利亚 | 法国行政区与欧洲移民政治优势 | 北部划为法国省份，本地穆斯林法律身份受限 |
| 民族运动时期 | 改良派、乌里玛、民族主义政党与地下组织 | 改革、文化复兴和独立路线竞争 |

## 重要事件

- 1516年巴巴罗萨兄弟控制阿尔及尔；1518年后与奥斯曼帝国关系制度化。
- 17—18世纪，阿尔及尔以地中海私掠、赎金、贸易和地方贡赋维持财政。
- 1830年法国军队攻占阿尔及尔。
- 1832—1847年，阿卜杜勒·卡迪尔建立抵抗政权并组织长期战争。
- 1871年莫克拉尼起义失败后，殖民土地剥夺进一步扩大。
- 1881年的《原住民法典》强化对穆斯林居民的特别惩罚和行政控制。
- 第一次世界大战后，退伍军人、工人、宗教学者和城市精英推动多种政治组织。
- 1945年塞提夫等地的示威、暴力与大规模镇压成为独立运动激进化的重要节点。
- 1954年前，和平改革和选举参与屡受限制，为武装起义创造背景。
