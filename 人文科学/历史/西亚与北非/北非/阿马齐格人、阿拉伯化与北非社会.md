# 阿马齐格人、阿拉伯化与北非社会

## 时间

古代至今

## 概括

阿马齐格人是北非一系列语言、地方社群和历史共同体的统称，旧文献常称“柏柏尔人”。从摩洛哥大西洋岸、阿特拉斯山区和阿尔及利亚卡比利亚，到突尼斯、利比亚与撒哈拉绿洲，不同群体长期参与农业、牧业、城市贸易、帝国军队和王朝建构。

北非的伊斯兰化与阿拉伯化并非一次完成。7—8世纪征服建立新的政治框架，皈依、城市教育、行政语言、部族迁徙和跨地区贸易则在数百年中不断改变语言与身份结构。

## 主要历史阶段

| 阶段 | 主要变化 | 辨析 |
|---|---|---|
| 古代地方社会 | 努米底亚、毛里塔尼亚、加拉曼特等政治体与地方联盟形成 | 这些名称不是现代民族国家的直接前身 |
| 罗马与基督教时期 | 城市罗马化、拉丁语和基督教扩展，山区与沙漠保持多样结构 | 罗马文化与地方传统并存 |
| 阿拉伯征服以后 | 伊斯兰传播，阿拉伯语进入行政、宗教与城市生活 | 皈依伊斯兰不等于立即放弃阿马齐格语言 |
| 中世纪王朝 | 穆拉比特、穆瓦希德等王朝由阿马齐格联盟建立 | 阿马齐格力量本身也是伊斯兰国家建构者 |
| 11世纪以后 | 希拉勒等阿拉伯部族迁徙推动部分平原和牧区阿拉伯化 | 迁徙影响因地区而异 |
| 近现代 | 殖民分类、民族国家教育与文化运动重塑身份 | 语言权利和地方自治成为现代议题 |

## 重要事件与机制

- 罗马帝国以城市、公民权、军役和税制整合北非，但地方语言和山地共同体长期存在。
- 哈瓦利吉派等早期伊斯兰政治宗教运动曾在马格里布地方社会中获得支持，并促进独立王朝形成。
- 穆拉比特和穆瓦希德把撒哈拉商路、马格里布和安达卢斯连接在一个政治网络中。
- 殖民政权常把“阿拉伯”与“柏柏尔”塑造成僵化对立，服务于行政分类和分化治理。
- 独立后的阿拉伯化教育强化共同国家语言，同时也推动阿马齐格文化和语言复兴运动。
