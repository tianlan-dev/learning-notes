# 迦太基海上霸权

## 时间

约前6世纪-前3世纪

## 概括

迦太基在西地中海形成以商业、海军、殖民据点、同盟城市和北非腹地为基础的海上霸权。它继承并重组腓尼基西方殖民网络，与希腊殖民城邦长期竞争，并在罗马崛起前成为西地中海最重要的强权之一。

## 统治结构

| 层面 | 说明 |
|---|---|
| 城市核心 | 迦太基本城是政治、商业、宗教和海军中心。 |
| 北非腹地 | 提供农业、贡赋、人力和战略纵深。 |
| 海上网络 | 连接西西里、撒丁、科西嘉、伊比利亚南部和北非港口。 |
| 军事体系 | 依赖舰队、同盟、雇佣军和贵族将领家族。 |
| 对外竞争 | 主要与希腊殖民城邦、叙拉古和后来的罗马竞争。 |

## 重要事件

- 迦太基逐渐成为西地中海腓尼基殖民据点的协调中心。
- 在西西里，迦太基与希腊城邦尤其叙拉古长期争夺岛屿控制权。
- 在撒丁、科西嘉和伊比利亚沿海，迦太基势力通过贸易据点、军事基地和地方联盟扩张。
- 罗马统一意大利后，双方在西西里问题上发生直接冲突，进入布匿战争阶段。
