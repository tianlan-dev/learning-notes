# 独立、复兴党统治、内战与政治过渡

## 时间

1946年至今

## 概括

独立后的叙利亚起初实行议会共和制，但军队、政党和地区精英竞争引发连续政变。1958年叙利亚与埃及组成阿拉伯联合共和国，1961年退出；1963年复兴党政变后，国家逐步形成军队、党组织和安全机构主导的体制。哈菲兹·阿萨德1970年掌权，其子巴沙尔·阿萨德于2000年继任。

2011年反政府抗议遭镇压后，冲突迅速军事化，政府军、不同反对派、库尔德主导力量、极端组织及多国外部力量同时介入。战争造成大规模人员伤亡、难民与境内流离失所，并使领土控制长期碎片化。2024年末反对派迅速推进并进入大马士革，阿萨德政权垮台；此后叙利亚进入由临时和过渡机构主导的新阶段，但全国性政治和解、少数群体安全、武装整合、宪政安排与重建仍未完成。

## 统治结构

| 阶段 | 权力结构 | 主要特征 |
|---|---|---|
| 1946—1958年 | 总统、议会、内阁与军队 | 文官政治薄弱，军人政变和阿拉伯民族主义竞争频繁。 |
| 阿拉伯联合共和国 | 以开罗为中心的统一国家机构 | 叙利亚政党被限制，中央集权引发不满。 |
| 1963—2024年复兴党时期 | 总统、复兴党、军队和安全机构 | 党国体制与总统权力集中，阿萨德家族长期主导。 |
| 2011年后冲突格局 | 政府、反对派、地方自治力量、武装组织及外国军力 | 领土、行政和安全权力碎片化。 |
| 2024年后过渡 | 临时和过渡机构、地方力量及待整合的武装体系 | 制度仍在重建，政治代表性和国家统一是核心问题。 |

## 重要事件

- 1949年，叙利亚发生多次军事政变，开启军队长期干政。
- 1958年，与埃及组成阿拉伯联合共和国；1961年叙利亚政变后退出。
- 1963年，复兴党掌权并实施紧急状态。
- 1970年，哈菲兹·阿萨德通过“纠正运动”掌权。
- 1982年，哈马起义遭政府军镇压，造成严重伤亡。
- 2000年，巴沙尔·阿萨德继任总统。
- 2011年，抗议、镇压和武装冲突相继扩展。
- 2014年前后，“伊斯兰国”控制叙利亚东部大片地区，后被多方逐步击退。
- 2015年后，俄罗斯直接军事介入显著改变战场态势；伊朗、土耳其、美国及其他外部力量也以不同方式介入。
- 2024年12月，阿萨德政府垮台，叙利亚进入政治过渡。
