# 西亚与北非

## 概括

西亚与北非是连接地中海、两河流域、伊朗高原、阿拉伯半岛、埃及和北非的历史区域。这里既包含古代近东文明、波斯帝国和希腊化 / 罗马-拜占庭秩序，也包含伊斯兰兴起后的阿拉伯帝国、突厥与蒙古重组、奥斯曼和近现代民族国家。

本目录先建立几条核心主线：古代西亚文明源头之一的[两河流域文明](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A%E4%B8%8E%E5%8C%97%E9%9D%9E/%E4%B8%A4%E6%B2%B3%E6%B5%81%E5%9F%9F/README.md)、连接东地中海与西亚帝国的[黎凡特](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A%E4%B8%8E%E5%8C%97%E9%9D%9E/%E9%BB%8E%E5%87%A1%E7%89%B9/README.md)、跨区域的[迦太基](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A%E4%B8%8E%E5%8C%97%E9%9D%9E/_%E9%80%9A%E5%8F%B2/%E8%BF%A6%E5%A4%AA%E5%9F%BA/README.md)和[阿拉伯帝国](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A%E4%B8%8E%E5%8C%97%E9%9D%9E/_%E9%80%9A%E5%8F%B2/%E9%98%BF%E6%8B%89%E4%BC%AF%E5%B8%9D%E5%9B%BD/README.md)通史、以安纳托利亚和奥斯曼继承为主的[土耳其](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A%E4%B8%8E%E5%8C%97%E9%9D%9E/%E5%9C%9F%E8%80%B3%E5%85%B6/README.md)，以尼罗河流域为核心、贯通古代文明与现代国家的[埃及](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A%E4%B8%8E%E5%8C%97%E9%9D%9E/%E5%9F%83%E5%8F%8A/README.md)主线，以及以伊朗高原和现代伊朗为核心的[伊朗](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A%E4%B8%8E%E5%8C%97%E9%9D%9E/%E4%BC%8A%E6%9C%97/README.md)主线。

## 关键辨析

- “阿拉伯帝国”主要指 7-13 世纪以阿拉伯征服、哈里发制度、倭马亚和阿拔斯为主的帝国体系，不等于整个伊斯兰世界。
- “伊朗”适合覆盖古代伊朗高原、波斯帝国传统和现代伊朗国家；“波斯”更常用于古典与前现代外称。
- 阿拉伯征服后的伊朗既属于伊斯兰世界，也保留强烈的伊朗语言、王权和文化传统，因此单独放在伊朗主线中维护。
- “两河流域文明”是古代西亚文明主线，跨越现代伊拉克、叙利亚、土耳其东南和伊朗西部等区域，已单独放在“两河流域”目录中维护。
- “黎凡特”用于整理现在以色列、巴勒斯坦、黎巴嫩及相邻东地中海地区的历史；古代阶段优先按迦南、腓尼基、以色列 / 犹大和帝国统治阶段组织，近现代再按现代政治实体拆分。
