# 腓尼基城邦

## 时间

约前12世纪-前6世纪起；城邦传统延续至波斯、希腊化和罗马时期

## 概括

腓尼基城邦主要位于今天黎巴嫩沿海及其相邻海岸，以推罗、西顿、比布鲁斯等城市为代表。它们在青铜时代崩溃后发展海上贸易、工艺生产、字母文字和殖民网络，并通过迦太基等殖民城市深刻影响地中海世界。

## 城邦与网络

| 城邦 / 节点 | 位置 | 说明 |
|---|---|---|
| 比布鲁斯 | 黎巴嫩沿海 | 与埃及和东地中海贸易关系密切，名称与书写材料传统相关。 |
| 西顿 | 黎巴嫩沿海 | 腓尼基重要港口和工商业中心。 |
| 推罗 | 黎巴嫩南部沿海 | 海上扩张和殖民网络的重要中心，与迦太基传统关系密切。 |
| 迦太基 | 北非 | 腓尼基人在西地中海建立的殖民城市，后来发展为独立强权；完整主线见[迦太基](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A%E4%B8%8E%E5%8C%97%E9%9D%9E/_%E9%80%9A%E5%8F%B2/%E8%BF%A6%E5%A4%AA%E5%9F%BA/README.md)。 |

## 重要事件

- 腓尼基字母对希腊字母、拉丁字母等后续文字系统影响深远。
- 腓尼基城邦通常不是统一帝国，而是在强邻压力下维持贸易自治和贡赋关系。
- 亚述、巴比伦、波斯、希腊化王国和罗马先后把腓尼基沿海纳入更大帝国秩序。
