# 现代以色列与巴勒斯坦

## 时间

1948年至今

## 概括

现代以色列与巴勒斯坦史围绕以色列建国、巴勒斯坦民族运动、阿以战争、难民问题、占领与定居点、和平进程、自治安排和持续冲突展开。这个节点属于现代政治史，和古代以色列 / 犹大王国有关联，但不能把古代王国、宗教传统和现代民族国家简单等同。

## 统治结构

| 主题 | 说明 |
|---|---|
| 以色列国家 | 1948年建国后形成议会制国家，并经历多次战争、移民吸收和安全治理。 |
| 巴勒斯坦政治 | 包括巴勒斯坦民族运动、巴勒斯坦解放组织、巴勒斯坦民族权力机构及不同政治派别。 |
| 争议地区 | 约旦河西岸、加沙、东耶路撒冷、边界、安全、定居点和难民问题是核心争议。 |
| 国际维度 | 美国、阿拉伯国家、伊朗、联合国和欧洲等外部力量长期参与。 |

## 重要事件

- 1948年以色列建国和第一次中东战争改变巴勒斯坦委任统治地政治结构。
- 1967年战争后，约旦河西岸、加沙、东耶路撒冷等问题成为长期冲突核心。
- 奥斯陆进程后形成有限自治安排，但最终地位问题长期未解决。
- 21世纪以来，加沙、定居点、安全治理和地区代理冲突持续影响局势。
