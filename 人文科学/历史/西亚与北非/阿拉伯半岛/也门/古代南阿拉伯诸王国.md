# 古代南阿拉伯诸王国

## 时间

约前1千纪-7世纪

## 概括

古代也门高地和东部河谷形成萨巴、米奈、卡塔班、哈德拉毛等王国。它们利用季风降水、梯田和大型水利发展农业，并通过乳香、没药与印度洋货物贸易连接地中海、东非和南亚。古代晚期希木叶尔统一也门大部，随后卷入阿克苏姆、拜占庭和萨珊之间的红海竞争。

## 主要政权

| 政权 | 大致时期 | 中心或特点 |
|---|---|---|
| 萨巴王国 | 约前1千纪初-3世纪 | 以马里卜为核心，水利农业和乳香商路发达。 |
| 米奈王国 | 约前8-前2世纪 | 活跃于焦夫地区和远程商贸网络。 |
| 卡塔班王国 | 约前8世纪-2世纪 | 控制南部内陆商路和农业区。 |
| 哈德拉毛王国 | 古代至3世纪 | 连接东部产香区与阿拉伯海港口。 |
| 希木叶尔王国 | 约前1世纪-6世纪 | 逐步统一南阿拉伯，古代晚期宗教和国际竞争突出。 |
| 阿克苏姆与萨珊统治 | 6世纪 | 红海对岸与伊朗势力先后干预也门。 |

## 重要事件

- 马里卜水利系统支撑高地和河谷农业，也成为王权与公共工程的象征。
- 骆驼商队与海运共同承担香料、乳香、没药和奢侈品贸易。
- 希木叶尔时期王权与犹太教关系增强，基督教社群也在南阿拉伯扩展。
- 6世纪阿克苏姆征服和萨珊干预使也门成为红海国际政治焦点。
- 7世纪也门各地接受伊斯兰，古代王国秩序转入新的宗教政治体系。
