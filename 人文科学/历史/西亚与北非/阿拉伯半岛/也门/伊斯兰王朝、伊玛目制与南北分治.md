# 伊斯兰王朝、伊玛目制与南北分治

## 时间

7世纪-1990年

## 概括

伊斯兰化后的也门长期保持强烈地方性。拉苏勒王朝等政权依托塔伊兹、扎比德和亚丁经营红海—印度洋贸易，北部高地则从9世纪起发展宰德派伊玛目制。奥斯曼两次进入北也门，英国自1839年控制亚丁并扩展保护地，最终形成北部王国／共和国与南部殖民地／人民共和国的分治。

## 主要政治阶段

| 阶段 | 时间 | 统治结构与特点 |
|---|---|---|
| 早期伊斯兰与地方王朝 | 7-12世纪 | 哈里发总督、地方部族和区域王朝并存。 |
| 宰德派伊玛目制 | 897年起 | 伊玛目依靠宗教学识、血统资格和高地部落支持，权力范围时有伸缩。 |
| 拉苏勒王朝 | 1229-1454年 | 以塔伊兹为中心，农业、学术和红海贸易繁荣。 |
| 奥斯曼与卡西姆王朝 | 16-19世纪 | 奥斯曼在低地和城市驻军，高地伊玛目多次恢复独立。 |
| 北也门王国 | 1918-1962年 | 哈米德丁家族伊玛目统治穆塔瓦基利亚王国。 |
| 北也门共和国 | 1962-1990年 | 革命后经历内战、军政与部落协商，后参与统一。 |
| 英属亚丁与保护地 | 1839-1967年 | 亚丁殖民港与周边苏丹国、酋长国保护地并存。 |
| 南也门国家 | 1967-1990年 | 独立后建立人民共和国，后成为阿拉伯世界主要马克思主义国家。 |

## 重要事件

- 897年叶海亚·本·侯赛因在萨达建立宰德派伊玛目制。
- 拉苏勒王朝时期塔伊兹和亚丁成为区域政治与贸易中心。
- 1839年英国占领亚丁，南部港口转为帝国航运和补给基地。
- 1918年奥斯曼撤离后，北也门伊玛目建立独立王国。
- 1962年北部革命引发共和派与王党长期内战，并有埃及、沙特等外部介入。
- 1967年南部民族解放运动迫使英国撤离。
- 1970年代至1980年代，两个也门既竞争又逐步谈判统一。
