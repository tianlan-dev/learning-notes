# 古代阿拉伯与伊斯兰圣地

## 时间

古代-18世纪初

## 概括

现代沙特阿拉伯疆域横跨汉志、内志、哈萨、阿西尔和北部沙漠边缘。古代这些地区没有统一国家：绿洲农业、商队贸易、游牧和地方圣地构成多中心社会。7世纪伊斯兰在麦加和麦地那兴起，此后汉志保持全球穆斯林朝觐中心地位，而帝国政治中心先后转向大马士革、巴格达、开罗和伊斯坦布尔。

## 统治结构

| 区域或阶段 | 主要结构 | 说明 |
|---|---|---|
| 汉志城市 | 商人、部落保护与圣地管理 | 麦加、叶斯里卜等城市连接红海和内陆商路。 |
| 早期穆斯林共同体 | 先知领导、盟约与部落结盟 | 麦地那成为政治中心，随后统一半岛大部。 |
| 哈里发时期 | 总督与朝觐治理 | 圣地重要性延续，但帝国行政中心外移。 |
| 麦加谢里夫 | 哈希姆家族地方统治 | 在阿拔斯、法蒂玛、马穆鲁克和奥斯曼宗主权下维持地方权力。 |
| 内志与哈萨 | 绿洲城镇、部落联盟和地方政权 | 中央帝国控制通常较间接，地方联盟变化频繁。 |

## 重要事件

- 麦加依托克尔白、年度集会和商贸网络形成宗教中心。
- 622年穆罕默德迁往麦地那并建立共同体，随后逐步整合半岛。
- 630年麦加纳入穆斯林共同体，朝觐制度成为伊斯兰共同实践。
- 倭马亚以后政治中心外移，汉志由总督、谢里夫和地方精英共同治理。
- 1517年后奥斯曼取得汉志宗主权，通过埃及和叙利亚朝觐路线维系圣地。
- 内志相对分散的政治环境为18世纪德拉伊耶联盟兴起提供条件。
