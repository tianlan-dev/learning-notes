# 海湾部落、港口与珍珠贸易

## 时间

古代-19世纪初

## 概括

今阿联酋地区包含哈杰尔山地、阿莱因绿洲、阿曼湾和波斯湾沿岸。农业绿洲、季节性牧业、渔业、海运和珍珠采集形成互补经济。18世纪后，巴尼亚斯联盟在利瓦绿洲、阿布扎比和迪拜发展，卡瓦西姆联盟则以拉斯海马和沙迦为中心经营海上贸易并与阿曼、波斯和英国竞争。

## 统治结构

| 社会或政治单元 | 主要区域 | 特点 |
|---|---|---|
| 绿洲社群 | 阿莱因、利瓦等 | 椰枣农业、法拉吉灌溉和部落土地权。 |
| 巴尼亚斯联盟 | 阿布扎比、利瓦，后及迪拜 | 纳哈扬家族在阿布扎比、马克图姆家族在迪拜建立统治。 |
| 卡瓦西姆联盟 | 哈伊马角、沙迦及海上据点 | 商船和武装船队活跃，与阿曼和英国长期竞争。 |
| 港口商人 | 迪拜、沙迦、阿布扎比等 | 经营珍珠、转口、造船和印度洋贸易。 |
| 珍珠船队 | 波斯湾浅海 | 依靠船主、潜水员、商人与信贷网络组织季节性生产。 |

## 重要事件

- 古代乌姆纳尔等文化显示阿曼半岛与两河、印度河方向存在贸易。
- 伊斯兰化后港口继续连接巴士拉、霍尔木兹、印度西岸和东非。
- 18世纪巴尼亚斯联盟从内陆绿洲向海岸扩展。
- 1790年代至1810年代，英国与卡瓦西姆冲突升级，英国把对方描述为“海盗”，而当地政治也涉及贸易竞争和区域战争。
- 1819年英国远征拉斯海马，为1820年条约体系铺路。
- 珍珠贸易成为沿海酋长国最重要的现金经济之一。
