# 早王朝时期

## 时间

约前3100-前2686年。

## 概括

早王朝时期是上下埃及统一后，埃及国家制度、法老王权、孟菲斯政治中心和早期行政体系成型的阶段。传统上常把那尔迈或美尼斯与埃及统一联系在一起，但具体过程可能是较长时间的政治整合。

## 王朝世系 / 统治结构

| 层面 | 说明 |
|---|---|
| 王朝范围 | 通常包括第1、第2王朝。 |
| 王权中心 | 王权以统一上下埃及为合法性核心，孟菲斯成为重要政治中心。 |
| 行政体系 | 王室、官吏、仓储、税收和书写管理逐渐发展。 |
| 宗教象征 | 荷鲁斯名、王衔、王陵和祭祀传统强化法老权威。 |

## 重要事件

- 上下埃及统一的王权叙事形成。
- 王名、王衔和早期圣书体行政记录发展。
- 阿拜多斯、萨卡拉等地出现早期王陵和贵族墓葬。
- 国家行政、赋税征收和劳役组织能力增强。
