# 独立、族群冲突与岛屿分治

## 时间

1960年至今

## 概括

1960年塞浦路斯共和国成立。宪法为希腊族和土耳其族规定固定的总统、副总统、议会、公务员和安全部队配额，并由英国、希腊和土耳其担任保证国。制度设计缺少跨社群政党和有效争端解决机制，1963年修宪争议后暴力升级，土耳其族成员退出或被排除出中央机构，双方居住区更加分隔。联合国维和部队自1964年起驻岛。

1974年，希腊军政府支持塞浦路斯国民警卫队发动政变，试图推动并入希腊；土耳其依据保证条约主张采取军事行动，并控制岛屿北部。军事行动和族群暴力造成希腊族、土耳其族大规模流离失所，南北事实分治。1983年北部宣布建立“北塞浦路斯土耳其共和国”，仅获土耳其承认。多轮联邦统一谈判至今未达成全面协议。

## 统治结构

| 地区或机构 | 地位 | 说明 |
|---|---|---|
| 塞浦路斯共和国 | 国际承认的主权国家和欧盟成员 | 实际控制岛屿南部；宪法名义涵盖全岛。 |
| 北部事实政权 | 自称“北塞浦路斯土耳其共和国” | 拥有自身行政、议会和安全机构，仅受土耳其承认并高度依赖土耳其。 |
| 联合国缓冲区 | 联合国维和部队巡逻区域 | 沿1974年停火线延伸，将南北主要控制区分隔。 |
| 英国主权基地区 | 阿克罗蒂里和泽凯利亚 | 1960年独立安排中由英国保留，不属于塞浦路斯共和国领土。 |
| 保证国体系 | 英国、希腊、土耳其 | 依据1960年条约承担特定保证责任，其解释和运用存在争议。 |

## 重要事件

- 1960年，塞浦路斯共和国成立。
- 1963年，总统马卡里奥斯提出宪法修改，族群冲突扩大，权力分享制度实际崩解。
- 1964年，联合国安理会设立塞浦路斯维和部队。
- 1967年，新的族群冲突促使国际调停和撤军安排。
- 1974年7月，希腊军政府支持的政变推翻马卡里奥斯；土耳其随后发动两阶段军事行动。
- 1975年后，双方人口交换和迁移使南北居住分隔进一步固化。
- 1983年，北部单方面宣布独立；联合国安理会认为该宣布在法律上无效并要求撤回。
- 2003年，部分南北通行点开放，双方居民接触增加。
- 2004年，联合国安南方案在北部公投获多数支持、在南部被多数否决；同年塞浦路斯共和国加入欧盟。
- 此后多轮以建立两区、两族联邦为目标的谈判未能达成最终解决。
