# 巴勒斯坦民族运动、占领与自治治理

## 时间

1949年至今

## 概括

1949年后，西岸和东耶路撒冷由约旦控制，加沙由埃及管理，巴勒斯坦人分散在本地、以色列境内、邻国难民营和全球侨民社会。1964年巴勒斯坦解放组织成立，逐渐成为民族运动主要代表。1967年战争后，以色列占领西岸、东耶路撒冷和加沙，占领、定居点、土地与水资源、行动限制和军事治理成为日常政治核心。

1987年第一次大起义推动巴勒斯坦内部群众动员。1988年巴勒斯坦国宣言和1993年《奥斯陆协议》使巴解组织转向以谈判和有限自治推进建国；巴勒斯坦民族权力机构随后在西岸和加沙部分地区成立。最终地位谈判未能解决边界、耶路撒冷、难民、定居点和安全问题。2007年后法塔赫主导的权力机构与哈马斯在西岸、加沙形成长期政治分裂，封锁和周期性战争进一步削弱统一治理。

## 统治结构

| 机构或力量 | 角色 | 说明 |
|---|---|---|
| 巴勒斯坦解放组织 | 民族代表与谈判主体 | 由多个派别组成，获广泛国际承认，其执行委员会承担政治代表职能。 |
| 巴勒斯坦国机构 | 总统、政府及外交代表体系 | 1988年宣布成立，获许多国家承认，2012年取得联合国非会员观察员国地位。 |
| 巴勒斯坦民族权力机构 | 有限自治行政机关 | 依据奥斯陆安排成立，在西岸部分地区行使有限行政和安全权限。 |
| 哈马斯及加沙事实治理 | 政党、武装组织与行政体系 | 2006年选举获胜，2007年后长期在加沙实行事实治理，并同以色列多次战争。 |
| 以色列占领与控制体系 | 军事、民政、定居点及边境控制 | 对不同地区实施不同程度控制，其合法性和具体措施受到持续国际争议。 |

## 重要事件

- 1950年，约旦正式合并西岸；加沙继续由埃及管理。
- 1964年，巴勒斯坦解放组织成立。
- 1967年，以色列占领西岸、东耶路撒冷和加沙。
- 1974年，阿拉伯联盟确认巴解组织为巴勒斯坦人民唯一合法代表；联合国大会给予其观察员地位。
- 1982年，以色列入侵黎巴嫩，巴解组织主力撤离贝鲁特。
- 1987年，第一次大起义爆发，哈马斯在此背景下成立。
- 1988年，巴勒斯坦全国委员会宣布巴勒斯坦国；约旦解除与西岸的行政和法律联系。
- 1993年，以色列与巴解组织相互承认并签署《奥斯陆协议》。
- 1994年起，巴勒斯坦民族权力机构在加沙和西岸部分地区运作。
- 2000年，第二次大起义爆发。
- 2005年，以色列撤出加沙境内定居点和常驻地面部队。
- 2006年，哈马斯在立法委员会选举中获胜；2007年巴勒斯坦政治分裂固化，以色列实施加沙封锁，埃及也严格限制口岸通行。
- 2012年，巴勒斯坦取得联合国非会员观察员国地位。
- 2023年10月，哈马斯等组织袭击以色列；以色列随后在加沙发动大规模军事行动，造成严重平民伤亡、基础设施破坏、流离失所和人道危机。
