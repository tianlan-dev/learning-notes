# 哈希姆王国与现代约旦

## 时间

1946年至今

## 概括

1946年独立后，外约旦哈希姆王国于1949年改称约旦哈希姆王国。1948年战争中，阿拉伯军团控制约旦河西岸和东耶路撒冷，王国于1950年正式合并西岸；1967年战争后，以色列占领西岸和东耶路撒冷，大量巴勒斯坦人再次迁入约旦。王国既要维护哈希姆君主制和部落—军队联盟，也要整合庞大的巴勒斯坦裔人口。

1970—1971年王国军队同巴勒斯坦武装组织爆发冲突，巴解组织主力转移到黎巴嫩。1988年约旦放弃对西岸的行政与法律联系，1994年同以色列签署和平条约。现代约旦实行君主立宪形式，但国王在任命政府、外交、安全和立法过程中拥有广泛权力；国家还长期接纳巴勒斯坦、伊拉克和叙利亚等地难民，水资源、就业和财政压力突出。

## 统治结构

| 机构 | 角色 | 说明 |
|---|---|---|
| 国王 | 国家元首与最高政治权威 | 任命首相和参议院成员，在军事、外交和立法方面权力广泛。 |
| 内阁 | 行政机关 | 由首相领导，对国王负责并接受议会监督。 |
| 国民议会 | 众议院与参议院 | 众议院经选举产生，参议院由国王任命。 |
| 武装部队与安全机构 | 国防与政权稳定 | 同哈希姆王权和东岸部落网络关系密切。 |
| 地方与社会组织 | 市镇、部落、专业团体、政党和慈善组织 | 在公共服务、政治动员和社会调解中发挥作用。 |

## 重要事件

- 1946年，外约旦独立。
- 1948—1949年，阿拉伯军团参加第一次中东战争并控制西岸和东耶路撒冷。
- 1950年，约旦正式合并西岸；这一行为的国际承认有限。
- 1951年，国王阿卜杜拉一世在耶路撒冷遇刺。
- 1952年宪法确立王国现行制度框架。
- 1967年，约旦在战争中失去西岸和东耶路撒冷。
- 1970—1971年，“黑九月”冲突后巴解组织主力撤出约旦。
- 1988年，约旦宣布解除与西岸的行政和法律联系。
- 1989年，经济抗议后恢复议会选举，政治有限开放。
- 1994年，约旦与以色列签署和平条约。
- 2011年前后，地区抗议浪潮推动政府更迭和若干制度调整。
- 2010年代，大批叙利亚难民进入约旦，加重公共服务和资源压力。
