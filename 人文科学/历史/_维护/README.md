## 维护约定

- 维护文档只定义可长期复用的目录、内容与验收规则。
- 知识笔记不得依赖维护文档作为唯一入口。
- 调整历史笔记时必须保留原有事实、例子、资源和有效链接，并完成内容与导航验收。
