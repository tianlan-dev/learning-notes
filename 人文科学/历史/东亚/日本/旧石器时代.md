# 旧石器时代

## 时间

约前3.8万年—前1.4万年

## 概括

日本列岛较可靠的人类活动证据大致始于距今3.8万年前。末次冰期海平面较低，北海道经萨哈林接近东北亚大陆，九州与朝鲜半岛之间的海峡变窄，琉球弧各岛也形成不同距离的海上跳板；但列岛并非在所有方向都与大陆连成一片，人群进入同时包含陆路移动和有计划的水上穿越。此期居民以流动狩猎、采集和沿海资源利用为主，尚无国家、王权或文字记录。

## 证据边界与分期

- 日本旧石器考古以地层、火山灰、石器类型和放射性碳年代为主要依据。约前3.8万年以前的少数“更早遗址”仍有较大争议。
- 1949年相泽忠洋等人在岩宿遗址确认关东壤土层中的石器，推翻“日本没有无陶旧石器文化”的旧看法。
- 2000年旧石器遗址造假事件暴露部分“早期石器”被人为埋入，学界此后重新审查遗址地层与年代；它不否定全部旧石器遗存，但使前期主张必须逐项验证。
- 约前1.4万年以后陶器、定居和资源利用发生显著变化，通常据此转入绳文时代；不同地区并非同年完成转型。

## 环境、生计与技术

| 方面 | 主要证据 | 说明 |
|---|---|---|
| 气候与地貌 | 冰期海平面、火山灰层、古植被 | 寒冷草原、森林和海岸线随阶段变化，影响猎物与迁徙路线 |
| 石器 | 石叶、尖状器、细石叶、局部磨制石斧 | 原料和技术存在明显地区差异，晚期细石叶与东北亚联系密切 |
| 原料流通 | 黑曜石、燧石等 | 长野、北海道、神津岛等原料被远距离搬运；神津岛黑曜石说明至少存在海上航行 |
| 生计 | 大型与中小型动物骨、植物利用痕迹、洞穴与营地 | 保存条件使植物与木器证据偏少，不能把生计简化为只猎猛兽 |
| 聚落 | 临时营地、石器制作场、洞穴 | 多为季节性流动网络，人口规模和社会组织难以直接重建 |

## 人群进入与地区差异

- 北海道受到萨哈林—东北亚技术传统影响，细石叶和寒地猎食适应较突出。
- 本州、四国、九州通过山地、河谷和海岸形成多条流动路线；火山喷发也会迫使人群迁移或重建资源网络。
- 琉球弧的港川人等晚更新世人骨说明南方岛链有人群活动，海峡穿越能力不可忽视。
- 考古文化、古代人群和现代民族不是同一概念；不能把某种石器类型直接等同于“日本人”的单一祖先。

## 重要发现与转折

1. **约前3.8万年的人类活动**：较可靠石器组合显示现代人已进入列岛。
2. **海上取得神津岛黑曜石**：原料跨海运输表明旧石器人群并非只沿陆地被动迁徙。
3. **火山环境适应**：姶良火山约前2.9万年的大喷发在广域留下火山灰，成为断代标志，也重塑西日本生态。
4. **细石叶技术扩散**：末次冰期晚期，复合工具提高狩猎与维修效率，并显示与东北亚的交流。
5. **岩宿遗址确认**：20世纪中叶建立日本旧石器考古的可靠地层基础。
6. **2000年造假事件后的重审**：促成更严格的发掘记录、理化测年和可重复验证。

## 向绳文时代的转型

气候转暖、森林扩张、海岸线变化和动植物资源多样化，使部分群体减少长距离迁徙。陶器便于加工坚果、鱼贝等食物，竖穴住居和季节性回访点逐渐稳定。这个过程持续很久，并非“冰期一结束就突然出现定居民族”。
