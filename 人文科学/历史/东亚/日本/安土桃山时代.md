# 安土桃山时代

## 时间

1573—1603年。1573年足利义昭被逐标志室町幕府终结，1603年德川家康受任征夷大将军、江户幕府成立。织田信长的统一活动早在1568年入京后已经展开，丰臣政权的最后军事覆灭则迟至1615年大阪之阵，因此阶段边界属于通史约定。

## 概括

安土桃山时代是区域战国政权被整合为全国性军事—领主秩序的阶段。织田信长打破畿内旧权力网络，丰臣秀吉以关白、太阁身份完成统一并用检地、刀狩、转封和人质制度固定统治；侵朝战争与继承安排失败又耗损丰臣政权。秀吉死后，德川家康借五大老地位和最大领地取得优势，关原之战后建立新幕府。

## 分阶段发展

### 织田政权（1573—1582）

信长放逐足利义昭后不建立新幕府，而以朝廷官位、直属家臣团、近畿城郭和大名联盟统治。安土城连接京都、琵琶湖和东国通路；解除部分关卡、保护市场和控制堺等商港扩大军费来源。对延历寺、长岛一向一揆等敌对宗教共同体的镇压极其残酷。1582年信长完成甲州征伐，却在本能寺之变中被明智光秀逼死。

### 丰臣继承与全国统一（1582—1590）

秀吉在山崎之战击败光秀，在清洲会议和贱岳之战中控制织田继承。他修筑大阪城、压服四国，1585年任关白并获丰臣姓；1587年九州征伐迫使岛津氏降服，1590年小田原征伐灭后北条氏，再以奥州仕置重排东北领地。德川家康被转封关东，既离开传统东海领地，也获得以江户为中心的大领国。

### 统一政权制度化与侵朝战争（1590—1598）

太阁检地以土地生产力和耕作者登记重定年贡、军役，刀狩和身份政策削弱农民武装并推动兵农分离。大名领地通过朱印状确认，妻子和继承人常留在中央政治空间。秀吉两次出兵朝鲜，企图以半岛为通道征服明朝；日军初期快速推进，但朝鲜水军与义兵抵抗、明军参战、漫长补给线和谈判破裂使战争陷入消耗。1598年秀吉死后日军撤退，朝鲜社会遭受人口、城市和文化财产的巨大破坏，日本与明朝也付出重费。

### 丰臣继承危机与德川胜出（1598—1603）

秀吉以幼子秀赖为继承人，设置五大老、五奉行协调，但机构依赖个人誓约，没有能约束最大大名的常设最高权力。家康扩张婚姻与联盟，引发石田三成等反对。1600年关原之战中东军获胜，家康没收和转封西军领地；1603年受任征夷大将军，建立[江户时代](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E4%B8%9C%E4%BA%9A/%E6%97%A5%E6%9C%AC/%E6%B1%9F%E6%88%B7%E6%97%B6%E4%BB%A3.md)的德川幕府。丰臣家仍在大阪保有领地，至1615年才被消灭。

## 统治结构与连续掌权者

天皇维持任官、改元和政治正统性，本期主要为正亲町天皇、后阳成天皇，详见[天皇世系表](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E4%B8%9C%E4%BA%9A/%E6%97%A5%E6%9C%AC/%E5%A4%A9%E7%9A%87%E4%B8%96%E7%B3%BB%E8%A1%A8.md)。

| 顺序 | 掌权者 / 权力结构 | 实际主导时间 | 权力依据与交接 |
| ---: | --- | --- | --- |
| 1 | **织田信长** | 1573—1582 | 以畿内军事实力、朝廷官位、直属家臣和大名同盟主导统一；本能寺之变中死亡。 |
| 2 | **丰臣秀吉** | 1582—1598 | 以织田继承战争胜利、关白 / 太阁职位、检地和大名转封完成统一；病逝后留下幼主。 |
| 3 | 丰臣秀赖与五大老、五奉行 | 1598—1600 | 秀赖为名义继承人，前田利家死后制衡迅速瓦解。 |
| 4 | **德川家康** | 1600—1603 | 关原之战后成为全国最强权力者，受任将军后把霸权转为幕府制度。 |

| 机构 / 政策 | 作用 |
| --- | --- |
| 关白、太阁与朝廷官位 | 秀吉不是源氏出身的将军，以摄关官职和丰臣姓取得全国合法性。 |
| 五大老 | 德川家康、前田利家等大大名组成，原意是保护幼主并处理最高政务。 |
| 五奉行 | 石田三成、浅野长政等负责行政、财政和司法执行。 |
| 太阁检地 | 统一土地丈量、石高与耕作者登记，决定大名军役和年贡基础。 |
| 刀狩、身份固定 | 收缴农民武器、限制身份转换，推动兵农分离，但执行程度存在地区差异。 |
| 转封与城郭 | 调动大名领地、拆除或控制城堡，把潜在对手与战略节点重新配置。 |

## 统一成功与丰臣失势的原因

### 统一成功

- 战国大名已在地方完成土地、军役和家臣组织，织田、丰臣能够通过降服和转封接管，而非逐村重建。
- 畿内人口、市场、港口、朝廷官位和交通网络让控制者同时获得财政与合法性。
- 信长、秀吉把歼灭、围城、赦免、婚姻、官位和领地保留结合，降低持续占领成本。
- 检地与石高制提供跨地区可比较的征税和军役尺度，增强全国资源调配。

### 丰臣失势

- **结构因素：** 权力围绕秀吉个人、关白身份和战功联盟形成，没有成熟的世袭幕府机构；秀赖年幼，大老领地差距悬殊。
- **资源压力：** 侵朝战争消耗军费、兵力和大名合作意愿，且没有获得可持续征服收益。
- **联盟分裂：** 武断派与文治派、东国大名与丰臣直臣之间存在利益冲突；前田利家去世后，能牵制家康的资深大老减少。
- **直接触发：** 家康的婚姻和政治扩张、上杉征伐及石田三成起兵把矛盾转为关原决战。
- 丰臣政权在1600年失去全国主导权、1603年失去最高制度地位，但丰臣家到1615年大阪之阵才灭亡，三者不可混为同一日期。
