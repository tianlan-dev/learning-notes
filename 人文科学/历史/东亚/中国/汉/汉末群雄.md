# 汉末群雄

## 时间

184年-220年

## 概括

汉末群雄指东汉末年黄巾起义、董卓乱政之后，地方州牧、刺史、豪强和军阀掌握兵权，形成割据混战的局面。东汉中央名义仍在，但实权逐渐转移到曹操、袁绍、孙策、刘备、刘表、刘璋、马腾、韩遂、公孙瓒等地方势力手中。最终曹操统一北方，孙权据江东，刘备据益州，形成三国格局的基础。

## 主要阶段

| 阶段 | 时间 | 说明 |
|---|---|---|
| 黄巾起义 | 184年 | 太平道发动大规模起义，东汉中央依赖地方武装平乱。 |
| 董卓乱政 | 189年-192年 | 董卓入洛阳，废少帝、立献帝，迁都长安，中央权威崩溃。 |
| 关东诸侯并起 | 190年前后 | 袁绍、曹操、袁术、公孙瓒等以讨董为名起兵。 |
| 群雄兼并 | 190年代 | 各地军阀互相攻伐，弱小势力被吞并。 |
| 官渡之后 | 200年以后 | 曹操击败袁绍，逐步统一北方。 |
| 赤壁之后 | 208年以后 | 曹操南下受阻，孙权、刘备势力巩固，三分趋势形成。 |
| 曹丕篡汉 | 220年 | 汉献帝禅位，东汉灭亡，曹魏建立。 |

## 主要势力

| 势力 | 代表人物 | 区域 | 结局或影响 |
|---|---|---|---|
| 曹操集团 | 曹操、曹丕 | 兖州、豫州、冀州、关中等北方地区 | 统一北方，控制汉献帝；曹丕于220年代汉建魏。 |
| 袁绍集团 | 袁绍 | 冀州、青州、并州、幽州 | 官渡之战败于曹操，后继者被曹操消灭。 |
| 袁术集团 | 袁术 | 淮南 | 称帝失败，迅速败亡。 |
| 孙氏集团 | 孙坚、孙策、孙权 | 江东 | 建立江东基业，后发展为孙吴。 |
| 刘备集团 | 刘备、关羽、张飞、诸葛亮 | 徐州、荆州、益州 | 后占益州，建立蜀汉。 |
| 刘表集团 | 刘表 | 荆州 | 赤壁前后瓦解，荆州成为曹、孙、刘争夺核心。 |
| 刘璋集团 | 刘璋 | 益州 | 被刘备取代。 |
| 公孙瓒集团 | 公孙瓒 | 幽州 | 败于袁绍。 |
| 马腾、韩遂集团 | 马腾、韩遂、马超 | 凉州、关中 | 与曹操长期对抗，后被曹魏体系吸收或击败。 |
| 董卓集团 | 董卓、李傕、郭汜 | 凉州、关中 | 董卓被杀后，部将继续控制长安，进一步破坏中央秩序。 |

## 关键事件

| 事件 | 时间 | 影响 |
|---|---|---|
| 黄巾起义 | 184年 | 地方武装兴起，中央控制力下降。 |
| 董卓入京 | 189年 | 废立皇帝，控制朝廷，激起关东诸侯讨董。 |
| 迁都长安 | 190年 | 洛阳遭破坏，东汉中央权威进一步受损。 |
| 挟天子以令诸侯 | 196年 | 曹操迎汉献帝至许，取得政治合法性优势。 |
| 官渡之战 | 200年 | 曹操击败袁绍，北方统一趋势确立。 |
| 赤壁之战 | 208年 | 曹操南下失败，孙刘联盟保住江东和荆州方向。 |
| 刘备入蜀 | 214年 | 刘备取得益州，三国鼎立基础形成。 |
| 曹丕篡汉 | 220年 | 东汉灭亡，进入三国时代。 |

## 说明

- 汉末群雄的出现，根源不只是黄巾起义，也包括东汉后期外戚宦官斗争、豪强扩张和地方军事化。
- 曹操的优势在于控制汉献帝和北方人口、粮食、军政资源；孙权和刘备则依靠长江防线、江东与益州的区域条件。
- 后续进入三国，曹魏、蜀汉、孙吴分别继承汉末三大政治军事集团。
