## 时间

291年—306年。

## 概括

八王之乱是西晋晋惠帝时期爆发的宗室诸王内战。它并不是只有八位诸王参与，而是以汝南王司马亮、楚王司马玮、赵王司马伦、齐王司马冏、长沙王司马乂、成都王司马颖、河间王司马颙、东海王司马越等八人为代表。乱事持续十余年，反复围绕皇后、外戚、辅政权、禁军和皇帝控制权展开，最终使西晋中央权威崩溃，为永嘉之乱和西晋灭亡铺路。

## 背景、阶段与转折

| 层次 | 内容 |
|---|---|
| 制度背景 | 晋武帝以宗室镇守要地，各王既有封国资源，又可出任都督、掌握军队；中央一旦失去裁决者，宗室便能把宫廷冲突军事化。 |
| 权力背景 | 晋惠帝无法主导朝政，杨氏外戚、贾后、太子集团和诸王都争夺诏令、禁军与皇帝人身。 |
| 第一阶段（291年） | 贾后借司马玮清除杨骏，又处死司马亮、司马玮，短期垄断朝廷；这仍主要是洛阳宫廷政变。 |
| 第二阶段（300年—301年） | 太子被害后，司马伦废杀贾后并篡位；司马冏、司马颖、司马颙等以“复位”名义动员地方军队，冲突扩大为全国性内战。 |
| 第三阶段（302年—304年） | 司马冏、司马乂先后掌权，洛阳被围，司马颖控制惠帝；皇帝被各方劫持，中央行政和粮运几近停顿。 |
| 第四阶段（305年—306年） | 司马越联合地方力量击败司马颖、司马颙，形式上结束乱事，但胜者无法重建统一军政体系。 |

## 结果与长期影响

- **直接结果**：洛阳、邺城、关中等核心地区反复征战，禁军和诸州精锐大量损耗，朝廷诏令失去公信力。
- **结构后果**：诸王为补充兵力而吸纳流民和各族武装，地方军事首领获得独立行动空间；司马越掌权后仍与皇帝、地方军镇冲突。
- **外部连锁**：304年刘渊建立汉政权，成汉亦在巴蜀成形；西晋已无力同时维持北方防线、首都供给与地方秩序。
- **人口与区域影响**：战乱、饥荒和掳掠推动人口南迁，改变江淮和江南的社会结构，也为东晋政权提供了侨民与士族基础。
- **直接通向灭亡的触发**：311年司马越死后，其主力在苦县宁平城一带被石勒击溃；同年洛阳陷落，内战造成的军事真空转化为王朝崩溃。

## 说明

- **背景一：宗室分封**：晋武帝司马炎大封同姓诸王，并给予诸王兵权，希望以宗室拱卫中央。
- **背景二：继承危机**：晋惠帝司马衷能力有限，无法有效裁决朝政，外戚、皇后和宗室都试图控制朝局。
- **背景三：后族干政**：杨骏、贾南风等围绕辅政权发生冲突，成为乱事导火索。
- **291年**：贾南风联合楚王司马玮清除杨骏；随后汝南王司马亮、卫瓘等被杀，司马玮又被贾后处死。
- **300年**：贾后害死太子司马遹，赵王司马伦起兵废杀贾后。
- **301年**：司马伦篡位称帝，齐王司马冏、成都王司马颖、河间王司马颙等起兵讨伐，司马伦失败被杀。
- **302年—303年**：齐王司马冏专权后被长沙王司马乂等击败，诸王之间的争夺继续扩大。
- **304年**：成都王司马颖一度控制朝政；同年刘渊起兵称汉王，北方民族政权开始兴起。
- **305年—306年**：东海王司马越逐渐取得优势，司马颖、司马颙相继失败，乱事基本结束。
- **后果**：西晋军政系统被严重破坏，洛阳周边残破，地方与少数民族武装坐大，西晋失去抵御汉赵等政权的能力。

## 演变关系

- 前一节点：[西晋](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E4%B8%9C%E4%BA%9A/%E4%B8%AD%E5%9B%BD/%E6%99%8B/%E8%A5%BF%E6%99%8B.md)统一后的制度矛盾与继承危机。
- 后一节点：永嘉之乱、西晋灭亡，以及[十六国](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E4%B8%9C%E4%BA%9A/%E4%B8%AD%E5%9B%BD/%E6%99%8B/%E5%8D%81%E5%85%AD%E5%9B%BD/README.md)格局形成。
- 与君主关系：主要发生在[晋君主世系](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E4%B8%9C%E4%BA%9A/%E4%B8%AD%E5%9B%BD/%E6%99%8B/%E4%B8%96%E7%B3%BB.md)中晋惠帝、晋怀帝交替前后。
