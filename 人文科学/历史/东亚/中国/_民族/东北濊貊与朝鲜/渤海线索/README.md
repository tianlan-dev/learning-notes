# 渤海线索

本目录是“东北濊貊与朝鲜”下的二级线索，用于收纳渤海线索相关民族、部族或政权笔记。

## 演进图

```mermaid
flowchart LR
  Goguryeo["高句丽遗民"] --> Bohai["渤海国"]
  Mohe["粟末靺鞨等靺鞨诸部"] --> Bohai
  Bohai --> Liao["契丹灭渤海后辽东丹国体系"]
  Bohai --> Jurchen["部分融入女真与东北诸部"]
  Bohai --> Goryeo["部分南迁入高丽"]
```
