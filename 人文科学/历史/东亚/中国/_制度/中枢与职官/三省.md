# 三省

三省指中书省、门下省、尚书省。其渊源可追溯到汉魏以来由宫廷近臣和文书机关发展出的中书、门下和尚书机构，至隋统一后重新整合，唐前期形成较成熟的中央政务框架。常见概括“中书出令、门下审驳、尚书执行”抓住了程序分工，但三省并非始终地位相等，也不是近代意义上的三权分立。

## 机构与职掌

| 机构 | 核心职能 | 主要长官 | 实际运行要点 |
| --- | --- | --- | --- |
| **中书省** | 参与最高决策，起草制敕。 | 中书令；副长官中书侍郎。 | 起草权使其接近皇帝，但重要政策通常经过宰相集议，并非中书省独断。 |
| **门下省** | 审议诏令、封驳不当文书，侍从顾问。 | 侍中；副长官门下侍郎。 | “封驳”可使文书退回重拟，是程序性纠错；能否有效行使取决于政治环境与皇帝态度。 |
| **尚书省** | 承受并执行政令，统领六部及日常行政。 | 尚书令；唐代常虚位，以左右仆射主省事。 | 事务量最大，但执行权不等于对政策拥有最终决定权。 |

## 一项政令的常规过程

```mermaid
flowchart LR
    E["皇帝提出方向或裁定议案"] --> P["宰相在政事堂等处集议"]
    P --> Z["中书省起草制敕"]
    Z --> M["门下省审读"]
    M -->|封驳、涂归| Z
    M -->|通过并署敕| S["尚书省分付六部"]
    S --> L["地方官署或专门机关执行"]
    L -->|奏报与执行反馈| P
    E -->|特旨、内廷文书等非常渠道| S
```

程序的意义在于让议论、文书和执行相互校验。唐代政事堂把三省长官及获授“同中书门下三品”“同中书门下平章事”等衔者组织为宰相群体，说明真正的决策中心不是三座官署简单串联，而是皇帝与宰相集议、文书审署和行政分付的组合。

## 形成与变化

| 时期 | 变化 |
| --- | --- |
| 汉魏 | 尚书由宫廷文书机关走向行政中枢；曹魏设置中书省以掌机密诏令，侍中、散骑等门下近臣参与顾问与审核。 |
| 两晋南北朝 | 中书、门下、尚书三套机关逐渐并立，名称、长官和职权仍多变；门阀、权臣和军府也会越过常制。 |
| 隋 | 以内史省、门下省、尚书省为核心，六部归尚书省，决策—审核—执行的分工趋于清晰。 |
| 唐前期 | 三省长官原则上为宰相，政事堂协调集议；尚书令因位尊多不授，左右仆射是否为宰相也随制度变化。 |
| 唐中后期 | 宰相资格与三省长官进一步分离，翰林学士、内廷枢密使和各种使职分走机要、财政与军政事务。 |
| 宋以后 | 宋元明清仍沿用部分三省名号或制度遗产，但实际中枢分别转为中书门下、枢密院、中书省、内阁和军机处等组合。 |

## 制度效果与成本

- **程序化**：把政策形成拆成议论、起草、审核、执行，减少单一官署垄断全部环节的风险。
- **协同成本**：三省职责重叠、文书往返会造成迟滞，因而常出现政事堂或临时差遣以加快协调。
- **皇权与相权**：分工既能形成官僚内部的纠错，也可被皇帝用来分散宰相权力；两种效果并存。
- **正式制度与实际政治**：皇帝特旨、近臣传宣、宦官或军府掌权都可能绕过常规程序，不能把制度图当作任何时期都完整执行的事实。
