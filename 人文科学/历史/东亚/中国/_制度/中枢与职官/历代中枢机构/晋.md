# 两晋中枢机构

两晋以“三省八公”为核心，三公九卿基本被三省取代，但九卿并未完全消失，而是地位下降、职务并省。

## 八公

八公包括：太宰、太傅、太保、太尉、司徒、司空、大司马、大将军。太尉、司徒、司空沿汉魏仍有宰相之称，但除司徒掌管中正品第职权外，与其他五公一样，多为尊宠虚衔。开府仪同三司者为位从公，列卿为九卿。

## 三省与监察

| 机构 | 职掌 |
| --- | --- |
| 尚书台 / 尚书省 | 掌管行政，是朝廷内外、中央地方各项政务的集汇处。太常等九卿及地方官员均奉尚书台命令行事。 |
| 中书省 | 魏文帝时期正式成立，掌管诏令、文书撰定，并参议政事；西晋开始地位、声望较曹魏提高。 |
| 门下省 | 晋哀帝始置，是皇帝近侍顾问机构，涉及重要政令和军国大事；有审查尚书机构上行下达文案，以及“驳奏”“驳诏”之权。 |
| 御史台 | 完全独立，架构完整，直接由皇帝控制，主要维护皇帝威严，并非扫荡贪污。 |

## 尚书省下设机构

不同阶段尚书省所辖机构有所变化：

| 阶段 | 机构 |
| --- | --- |
| 西晋初 | 吏部、三公、客曹、驾部、屯田、度支。 |
| 后改置 | 吏部、殿中、五兵、田曹、度支、左民。 |
| 东晋设 | 吏部、五兵、左民、度支、祠部。 |
