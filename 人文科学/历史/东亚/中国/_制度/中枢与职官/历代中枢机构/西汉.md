# 西汉中枢机构

西汉承秦制，中央行政体制仍以三公九卿为基础。汉武帝为削弱相权，设立“中朝 / 内朝”，尚书等近臣参与决策，外朝仍以三公九卿处理常规政务。

## 中央结构

| 层级 / 机构 | 内容 |
| --- | --- |
| 上公 | 太师、太傅、太保，位在三公之上，但多为荣誉性职位，不常置。 |
| 外朝 | 太尉、丞相、御史大夫及九卿。丞相为外朝首长。 |
| 中朝 / 内朝 | 侍中、尚书令、常侍等近臣参与决策，尚书原为少府属官，负责文书传达命令。 |
| 加官 | 侍中、诸吏、常侍、黄门令、散骑、给事中、左右曹等。 |
| 兼官 | 领尚书事、省尚书事、视尚书事、平尚书事等，武帝后期以宦官掌尚书改称“中书谒者”。 |

## 九卿及相关官署

九卿包括太常、光禄勋、卫尉、太仆、廷尉、大鸿胪、宗正、大司农、少府。尚书分曹可见常侍曹、二千石曹、户曹、主客曹、三公曹等。

## 选官

察举制：凡丞相、列侯、刺史、守、相等推举，经过考核，任以官职。
