# 唐代中枢机构

唐代前期以中书、门下、尚书三省和六部为法定行政骨架，宰相在政事堂集议，形成诏令起草、门下审议、尚书执行的程序。中后期因战争、财政与机密文书需求，翰林学士、内枢密使、盐铁转运使等新渠道兴起，三省六部仍存在，却不再包揽全部实际政务。

## 三省六部与宰相

| 机构 | 主要职能 | 实际运行 |
| --- | --- | --- |
| 中书省 | 参与决策、起草制敕。 | 中书令、中书侍郎可入宰相班；中书舍人负责制诰，重要文书仍受集议与门下审署。 |
| 门下省 | 审议、封驳，侍从顾问。 | 侍中、门下侍郎可为宰相；给事中等也能封还不当制敕，效力随政治环境而变。 |
| 尚书省 | 统领六部执行常务。 | 尚书令因位尊通常不授，左右仆射主省事；唐中宗以后，仆射未加宰相衔者不当然入政事堂。 |
| 六部 | 吏、户、礼、兵、刑、工分掌行政。 | 各部下分四司；与九寺五监多为政令指导、承办和复核关系，不是所有事项上的简单上下级。 |
| 政事堂 / 中书门下 | 宰相集议和综合协调。 | 三省长官之外，获授“同中书门下三品”“同中书门下平章事”等衔者也可成为宰相。 |

“参知政事”等早期加衔在唐代的具体使用和宰相资格随时期变化，不能把宋代固定副相官名倒推到整个唐代。

## 前期政令流程

```mermaid
flowchart LR
    E["皇帝提出议题或裁定"] --> P["政事堂：宰相集议"]
    P --> Z["中书省起草制敕"]
    Z --> M["门下省审议"]
    M -->|封驳| Z
    M -->|署敕通过| S["尚书省分付六部"]
    S --> D["九寺五监、州县等执行"]
    D -->|奏报与考课| P
    H["翰林学士：机密诏制"] --> E
    E --> H
    N["内枢密使：承奏与传宣"] --> E
```

这张图反映的是制度理想与常见流程，而非所有诏令都依次通过。皇帝手诏、内廷文书、宰相处分和使职文牒可以形成旁路；门下封驳也可能被政治压力削弱。

## 中后期新中枢

- **翰林学士**：翰林院原容纳文学、艺术和技术供奉，玄宗时翰林学士逐渐承担机密诏书起草，形成区别于中书舍人的“内制”渠道。
- **内枢密使**：代宗时期以后由宦官担任，主要承受章奏、宣达帝命，后来介入军机。它是内廷使职，不宜直接称作已经成熟的宋代枢密院。
- **财政使职**：度支、盐铁、转运等职因军费和两税法运行而重要，部分财政权脱离户部常规体系。
- **藩镇与神策军**：安史之乱后，节度使掌握地方军政财权；宦官控制神策军和宫廷禁军，改变中枢与地方、文官与军事力量的平衡。

## 阶段与关键转折

| 阶段 | 特征 |
| --- | --- |
| 高祖、太宗至高宗前期 | 承隋制并调整三省六部，政事堂强化宰相集议；科举、门荫与荐举并存。 |
| 武周至玄宗 | 官名和宰相加衔多次变化；翰林学士、诸使逐步增加，皇帝使用更灵活的近臣渠道。 |
| 755—763 年安史之乱 | 战时授予节度使更广军政财权，中央财政和军事组织被迫重构。 |
| 德宗以后 | 780 年两税法与财政使职相配合；中书门下、翰林、内枢密使、宦官禁军共同影响决策。 |
| 晚唐 | 藩镇、宦官、朝官与强势军阀争夺皇帝和军队控制，三省六部无法单独维系统一政治秩序。 |

## 选官、监察与社会基础

科举在唐代扩大，进士科地位逐渐上升，但门荫、流外入仕、荐举和军功仍是重要途径；高门士族在前中期仍有影响。御史台、谏官和门下封驳构成多层监督，既可纠错，也会卷入党争。官僚制度的成熟提高了文书与人事管理能力，却带来手续复杂、官与职分离、使职侵夺常官等协调成本。

## 盛衰原因的制度视角

唐前期的成功来自较稳定的税役基础、交通仓储、府兵与募兵的阶段性组合、开放的统治联盟及较有效的中枢协调。中后期危机并非“三省六部失效”单一造成，而是均田租庸调基础变化、长期边战、安史之乱、地方军事财政化、宫廷军权和继承冲突相互作用。新使职提高应急能力，却也使责任分散、常制与临时权力重叠。
