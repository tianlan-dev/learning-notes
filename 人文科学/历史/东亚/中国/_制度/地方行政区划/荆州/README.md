# 荆州

“荆州”在中国历史上至少有三种含义：先秦经典中的宏观地理区，汉代的刺史监察部，东汉末以后具有行政和军政实权的州，以及隋唐以后逐渐缩小、以江陵附近为核心的具体州府。名称连续并不等于辖境和机构连续，判断某条史料时必须先确认时代、治所、辖郡和实际控制者。

## 地理基础

早期荆州大体覆盖长江中游及汉水、洞庭湖流域，涉及今湖北、湖南大部，并延伸至河南南部、江西西北、贵州东部和两广北缘的部分地区。襄阳—樊城控制汉水南北通道，江陵扼长江与入蜀、下江东路线，洞庭湖与湘江水系通向岭南；这种交通位置使荆州兼具粮运、兵员和战略缓冲价值。

## 名称与制度变化

| 时段 | “荆州”的性质 | 主要变化 |
| --- | --- | --- |
| 先秦经典 | “九州”之一的地理文化概念。 | 边界是理想化天下分区，不应直接当作后世行政区地图。 |
| 西汉 | 十三刺史部之一。 | 刺史巡察南阳、南郡、江夏及两湖诸郡国，州本身主要是监察范围。 |
| 东汉前中期 | 仍以监察为主，刺史职权逐渐扩大。 | 州部有治所和属吏的情况逐步发展，具体阶段需区别。 |
| 东汉末 | 州牧、刺史成为区域军政长官。 | 刘表据荆州建立区域政权；赤壁后曹、孙、刘反复争夺并分割辖郡。 |
| 三国 | 曹魏、孙吴各设荆州。 | 同名州同时存在，魏治北部、吴治南部；蜀汉在 219 年后不再实际控制荆州。 |
| 晋南北朝 | 州郡不断分置、侨置。 | 湘州、郢州等分合使传统大荆州继续缩小，名称与辖境频繁改变。 |
| 隋唐以后 | 以江陵及今湖北荆州附近为核心的州府名。 | 宏观“荆楚”地理概念仍存在，但行政荆州已远小于汉代刺史部。 |

## 为什么荆州反复分合

- **军事需要**：北方政权沿南阳盆地、襄阳南下，江东政权依长江上游防线，益州政权需要经三峡或荆襄出入。
- **财政交通**：江汉平原、洞庭湖流域的粮食和水运可支持大军，江陵又连接巴蜀与江东。
- **行政可控性**：汉代荆州范围过广，人口开发和战争后不断分郡置州，以缩短治理距离或切割强势方镇。
- **名号竞争**：三国政权都沿用汉制以证明合法性，因此同名“荆州”可以在不同政权并存，不能把两个州的辖郡合成一张行政表。

- [东汉时期](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E4%B8%9C%E4%BA%9A/%E4%B8%AD%E5%9B%BD/_%E5%88%B6%E5%BA%A6/%E5%9C%B0%E6%96%B9%E8%A1%8C%E6%94%BF%E5%8C%BA%E5%88%92/%E8%8D%86%E5%B7%9E/%E4%B8%9C%E6%B1%89%E6%97%B6%E6%9C%9F.md)：州行政化、刘表经营及赤壁后控制权变化。
- [三国时期](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E4%B8%9C%E4%BA%9A/%E4%B8%AD%E5%9B%BD/_%E5%88%B6%E5%BA%A6/%E5%9C%B0%E6%96%B9%E8%A1%8C%E6%94%BF%E5%8C%BA%E5%88%92/%E8%8D%86%E5%B7%9E/%E4%B8%89%E5%9B%BD%E6%97%B6%E6%9C%9F.md)：魏、吴各自分郡和同名州并立。

“荆襄九郡”是后世常用概括，具体九郡组合随叙事时点不同，不是东汉政府正式、长期固定的法定名称。对辖郡数字应注明年份和政权，避免把 208、215、219 年后的格局混在一起。
