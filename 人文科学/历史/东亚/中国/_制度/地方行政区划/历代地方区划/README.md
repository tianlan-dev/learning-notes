# 历代地方区划总览

本目录按朝代比较正式行政层级、监察使职、军事辖区和基层治理。地方制度并非从“一级少”直线发展到“层级多”：王朝反复在控制幅度、行政成本与地方应变之间调整，把若干郡县交给高层区域机构可提高协调，却也可能形成新的地方权力中心。

中央机构部分见[历代中枢机构](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E4%B8%9C%E4%BA%9A/%E4%B8%AD%E5%9B%BD/_%E5%88%B6%E5%BA%A6/%E4%B8%AD%E6%9E%A2%E4%B8%8E%E8%81%8C%E5%AE%98/%E5%8E%86%E4%BB%A3%E4%B8%AD%E6%9E%A2%E6%9C%BA%E6%9E%84/README.md)。

## 跨朝比较

| 问题 | 常见做法 | 结构代价 |
| --- | --- | --- |
| 中央如何取得地方信息 | 上计、奏章、刺史巡察、转运提刑、密折。 | 信息层层过滤，监察官也可能行政化或结成地方利益。 |
| 如何防止地方割据 | 官员回避与调任，军财刑分署，监司交叉，中央控制高级任免。 | 权责分散、协调变慢，危机时需再设强势区域长官。 |
| 如何治理基层 | 以县为官僚末端，借乡里、保甲、里甲、胥吏和地方精英执行。 | 国家成本较低，但基层负担易被转嫁，中央难掌握真实情况。 |
| 如何治理边疆 | 都护府、羁縻、土司、盟旗、军府、驻扎大臣和改土归流。 | 适应地方差异，却形成法律、财政与权力多重结构。 |
