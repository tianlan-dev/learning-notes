# 转折、改革开放与现代化建设

## 时间

1976年-2012年

## 概括

“文化大革命”结束后，国家工作重点逐步转向经济建设。农村家庭联产承包、经济特区、城市经济体制改革、价格与财政改革、国有企业调整和加入世界贸易组织等进程推动经济高速增长、城市化和社会流动，也带来地区差距、环境压力和社会保障转型等问题。

## 重要事件

| 时间 | 事件 | 影响 |
|---|---|---|
| 1978年 | 中共十一届三中全会 | 成为改革开放和工作重心转移的重要标志。 |
| 1978年以后 | 农村改革 | 家庭联产承包责任制逐步推广，农业生产和乡村结构变化。 |
| 1980年起 | 经济特区与沿海开放 | 对外贸易、投资、产业转移和城市发展加快。 |
| 1982年 | 现行宪法通过 | 重新设置国家主席等国家制度，并经后续修正延续。 |
| 1989年 | 政治风波 | 对国内政治和此后改革路径产生深远影响。 |
| 1992年 | 改革开放进一步加快 | 社会主义市场经济目标逐步确立。 |
| 1997年、1999年 | 香港、澳门回归 | 分别成立香港和澳门特别行政区。 |
| 2001年 | 加入世界贸易组织 | 中国更深地进入全球生产、贸易和投资网络。 |
| 2008年 | 北京奥运会与国际金融危机 | 国际影响力上升，同时以宏观刺激应对外部经济冲击。 |
