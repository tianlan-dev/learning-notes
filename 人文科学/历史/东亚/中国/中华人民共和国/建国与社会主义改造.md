# 建国与社会主义改造

## 时间

1949年-1956年

## 概括

中华人民共和国成立后，中央政权在战争遗留、经济恢复和社会重组的背景下建立新的国家制度。土地改革、抗美援朝、国民经济恢复、第一个五年计划和社会主义改造共同塑造了这一阶段。

## 重要事件

| 时间 | 事件 | 意义 |
|---|---|---|
| 1949年 | 中华人民共和国成立 | 中央人民政府建立，新的国家政权开始运行。 |
| 1950-1953年 | 土地改革 | 在新解放区改变土地占有关系，具体进程和影响存在地区差异。 |
| 1950-1953年 | 抗美援朝战争 | 深刻影响国防、外交、财政和国内政治动员。 |
| 1953年 | 第一个五年计划开始 | 以重工业和基础工业为重点推进国家主导的工业化。 |
| 1954年 | 第一届全国人民代表大会与1954年宪法 | 确立人民代表大会制度等国家制度框架。 |
| 1953-1956年 | 农业、手工业和资本主义工商业社会主义改造 | 公有制和计划经济成为经济制度的主要基础。 |
