# 社会主义建设探索与曲折

## 时间

1956年-1976年

## 概括

这一时期在计划经济框架下推进工业、国防、教育、卫生和基础设施建设，同时经历反右运动、“大跃进”、人民公社化、三年困难时期和“文化大革命”等重大政治与社会曲折。不同地区、行业和群体的经历差异明显。

## 重要事件

| 时间 | 事件 | 影响 |
|---|---|---|
| 1956年 | 中共八大召开 | 对社会主义改造后的国内形势和建设任务作出判断。 |
| 1957年 | 整风与反右运动 | 政治和知识界环境发生重大变化。 |
| 1958年起 | “大跃进”和人民公社化 | 高指标、组织方式变化和资源配置失衡造成严重后果。 |
| 1959-1961年 | 严重经济困难 | 农业减产、政策失误和自然等因素叠加，造成广泛生活困难和人口损失。 |
| 1964年 | 第一颗原子弹试验成功 | 国防科技和战略能力建设的重要节点。 |
| 1966-1976年 | “文化大革命” | 国家政治、教育、文化、经济与社会生活受到长期冲击。 |
| 1971年 | 恢复中华人民共和国在联合国的合法席位 | 对外关系和国际地位发生重要变化。 |
| 1972年 | 中美关系开始正常化进程 | 冷战外交格局出现重大调整。 |

## 领导与统治结构

### 国家元首层级

| 人物或安排 | 时间 | 说明 |
|---|---|---|
| 毛泽东 | 1954-1959年 | 中华人民共和国主席。 |
| 刘少奇 | 1959-1968年 | 中华人民共和国主席；后被撤销党内外职务。 |
| 职位空缺及制度调整 | 1968-1975年 | 国家主席无人担任。 |
| 国家主席职务取消 | 1975-1982年 | 1975年宪法取消国家主席设置。 |
