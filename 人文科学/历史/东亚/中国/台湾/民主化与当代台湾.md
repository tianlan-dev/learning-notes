# 民主化与当代台湾

## 时间

1987年至今

## 概括

1987年解严后，台湾通过解除党禁报禁、终止动员戡乱时期、国会全面改选、总统直选和政党轮替完成民主化。高科技产业、全球供应链、社会运动、转型正义、人口老龄化和两岸关系持续塑造当代政治社会。

## 重要事件

| 时间 | 事件 | 说明 |
|---|---|---|
| 1987年 | 解除戒严 | 政治组织、媒体和社会运动空间扩大。 |
| 1991年 | 终止动员戡乱时期 | 宪政体制和两岸政策进入新阶段。 |
| 1992年 | 立法院全面改选 | 长期未改选的中央民意代表退出。 |
| 1996年 | 首次总统直接选举 | 总统政治授权方式发生根本变化。 |
| 2000年 | 首次政党轮替 | 民进党候选人陈水扁当选总统。 |
| 2008年、2016年 | 再次政党轮替 | 国民党、民进党先后重新执政。 |
| 2024年 | 赖清德、萧美琴就任正副总统 | 民进党连续第三次赢得总统选举。 |

## 政党领导与实际权力结构

### 主要政党与执政阶段

| 政党 | 总统执政时期 | 说明 |
|---|---|---|
| 中国国民党 | 至2000年、2008-2016年 | 从威权时期执政党转型为选举竞争政党。 |
| 民主进步党 | 2000-2008年、2016年至今 | 1986年成立，成为主要执政党之一。 |

### 实际权力结构

| 机构或角色 | 说明 |
|---|---|
| 总统 | 由直接选举产生，在外交、国防和行政院长任命等方面具有重要权力。 |
| 行政院院长 | 政府首脑，领导行政院；与总统和立法院多数之间的关系影响施政。 |
| 立法院 | 通过立法、预算和监督制衡行政部门。 |
| 政党与社会运动 | 政党竞争、选举、公民团体和社会运动共同影响公共政策。 |

## 关键辨析

- 民主化是持续的制度和社会过程，不只等于1987年解除戒严或1996年总统直选。
- 台湾内部对身份、两岸关系和宪政定位存在多种立场，不能用单一标签概括全体居民。
- 半导体产业重要，但当代台湾经济还包括传统制造、服务业、中小企业和全球贸易网络。
