# 清代台湾

## 时间

1683年-1895年

## 概括

清军击败郑氏后，台湾最初隶属福建省。随着汉人移民、土地开垦和市场网络扩大，清廷逐步增设行政区，但对山区和东部的有效控制长期有限。不同籍贯移民之间、移民与原住民族之间以及民众与官府之间的矛盾多次引发冲突。

## 行政结构

| 阶段 | 建制 | 说明 |
|---|---|---|
| 1684年起 | 福建省台湾府 | 初设一府三县，行政范围随后多次调整。 |
| 18-19世纪 | 增设厅县与分区治理 | 随人口、开发和边疆压力扩大行政覆盖。 |
| 1885年决定建省、1887年完成主要调整 | 福建台湾省 | 中法战争后加强海防、铁路、电报等近代建设。 |

## 重要事件

| 时间 | 事件 | 说明 |
|---|---|---|
| 1721年 | 朱一贵事件 | 大规模民变，反映地方治理和社会矛盾。 |
| 1786-1788年 | 林爽文事件 | 天地会组织、移民社会和清军镇压相互交织。 |
| 1874年 | 日本出兵台湾 | 牡丹社事件后日本出兵，推动清廷加强台湾治理。 |
| 1884-1885年 | 中法战争台湾战事 | 基隆、淡水和澎湖成为战场。 |
| 1885-1887年 | 台湾建省 | 刘铭传等推动防务、交通和行政改革。 |
| 1895年 | 《马关条约》 | 清政府将台湾、澎湖割让日本。 |

## 关键辨析

- “纳入版图”不等于行政力量立即覆盖全岛，山区、东部和原住民族地区长期保持不同程度的自主性。
- “熟番”“生番”等是清代治理分类，带有国家和移民社会视角，不应直接作为现代民族分类。
- 开垦和移民带来经济发展，也伴随土地转移、武装冲突和原住民族空间压缩。
