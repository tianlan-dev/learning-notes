# 战后接收、威权统治与冷战

## 时间

1945年-1987年

## 概括

日本战败后，中华民国政府接收台湾。接收初期的行政、经济和社会矛盾在1947年二二八事件中爆发。1949年中华民国中央政府迁台后，台湾进入长期戒严和一党优势的威权统治；白色恐怖、土地改革、美国援助、出口导向工业化和社会结构变化共同构成这一时期。

## 重要事件

| 时间 | 事件 | 说明 |
|---|---|---|
| 1945年 | 中华民国政府接收台湾 | 战后政权、法律和行政体系转换。 |
| 1947年 | 二二八事件 | 冲突与镇压造成大量伤亡，长期影响政治与族群记忆。 |
| 1949年 | 中央政府迁台、台湾戒严 | 两岸分治格局形成，戒严延续至1987年。 |
| 1950年代 | 土地改革与美国援助 | 改变农业结构，并支持财政、工业和安全体系。 |
| 1960年代以后 | 出口导向工业化 | 加工出口、基础教育和中小企业推动快速经济增长。 |
| 1971年 | 中华人民共和国恢复联合国席位 | 中华民国失去联合国中国代表权，外交环境改变。 |
| 1979年 | 美国与中华民国断交、美丽岛事件 | 对外关系和岛内民主运动的重要转折。 |
| 1987年 | 台湾地区解除戒严 | 威权体制向民主化转型的重要节点。 |

## 国家元首

| 人物 | 在台湾行使总统职权的主要时期 | 说明 |
|---|---|---|
| 蒋中正 | 1950-1975年 | 1948年就任总统，1949年一度引退，1950年在台复行视事。 |
| 严家淦 | 1975-1978年 | 蒋中正逝世后继任总统。 |
| 蒋经国 | 1978-1988年 | 后期解除戒严并推动部分政治开放。 |

## 政府首脑

| 主要人物 | 任职时期 | 说明 |
|---|---|---|
| 陈诚 | 1950-1954年、1958-1963年 | 行政院院长；参与土地改革和经济政策。 |
| 严家淦 | 1963-1972年 | 行政院院长，后任副总统、总统。 |
| 蒋经国 | 1972-1978年 | 行政院院长，后任总统。 |
| 孙运璿 | 1978-1984年 | 推动产业和基础设施政策。 |
| 俞国华 | 1984-1989年 | 本阶段列至1987年。 |

## 政党领导与实际权力结构

### 执政党领导

| 人物 | 职务与时期 |
|---|---|
| 蒋中正 | 中国国民党总裁，长期掌握党政军核心权力。 |
| 蒋经国 | 中国国民党中央委员会主席，1975-1988年。 |

### 实际权力结构

| 层级 | 说明 |
|---|---|
| 总统与国民党领导 | 总统、执政党主席和军事安全体系高度集中。 |
| 行政院 | 负责行政和经济政策，但重大方向受党国领导体系制约。 |
| 戒严与安全机关 | 以戒严法制、情治和军事审判限制政治组织与言论。 |
| 地方选举与社会组织 | 威权体制下仍存在地方选举、党外运动和逐渐扩大的社会参与。 |
