# 日本统治时期

## 时间

1895年-1945年

## 概括

《马关条约》后，日本建立台湾总督府。早期以军事镇压平定抵抗，随后通过警察、保甲、土地调查、公共卫生、学校和交通体系加强殖民治理。糖、稻米等产业和基础设施发展，与政治不平等、文化同化、原住民族征服和战时动员并存。

## 统治结构

| 层级 | 机构 | 说明 |
|---|---|---|
| 最高行政 | 台湾总督 | 由日本中央政府任命，兼具广泛行政、立法与警察权力。 |
| 地方治理 | 州、厅、街庄等地方制度 | 建制多次调整，警察和保甲深入基层。 |
| 殖民经济 | 总督府、专卖机构和日本资本 | 土地调查、糖业、铁路和港口建设服务殖民财政和帝国市场。 |
| 原住民族治理 | 警察、隘勇线与“理蕃”体系 | 以军事征服、迁居和行政控制扩张对山地的统治。 |

## 重要事件

| 时间 | 事件 | 说明 |
|---|---|---|
| 1895年 | 日本接管与武装抵抗 | 台湾民主国及各地抗日行动先后被镇压。 |
| 1915年 | 西来庵事件 | 成为早期武装抗日后期的重要事件。 |
| 1920年代 | 文化与议会设置请愿运动 | 知识界和社会团体争取文化、政治和自治权利。 |
| 1930年 | 雾社事件 | 赛德克族等反抗殖民统治，遭到严厉镇压。 |
| 1937年以后 | 皇民化与战争动员 | 日语、姓名、宗教和兵役政策加强，台湾被纳入总体战。 |
| 1945年 | 日本战败 | 日本在台湾的殖民统治结束。 |

## 关键辨析

- 基础设施与公共卫生改善不能抵消殖民制度中的政治排斥、资源控制和暴力。
- 日本统治并未使台湾社会完全同质化；族群、阶层、城乡和世代经验差异显著。
- “皇民化”是战争时期加强的同化与动员政策，不等于所有居民形成相同认同。
