# 三家分晋

## 时间

前453年三家灭智；前403年周威烈王册命韩、赵、魏为诸侯。

## 概括

三家分晋是春秋向战国转变的标志性事件。晋国国君在春秋末年已被卿大夫架空，韩、赵、魏三家联合消灭智氏后，进一步瓜分晋国公室土地。前403年，周威烈王正式册命三家为诸侯，传统上常被作为战国开端。

## 过程、转折与结果

```mermaid
    A["晋国公室衰弱，六卿掌政"] --> B["范氏、中行氏败亡"]
    B --> C["智、韩、赵、魏四家并立"]
    C --> D["前455—前453年：智氏围晋阳"]
    D --> E["韩、魏倒向赵氏，共灭智氏"]
    E --> F["韩、赵、魏取得晋国实权与主要土地"]
    F --> G["前403年：周威烈王册命三家为诸侯"]
    G --> H["晋君仅余少量领地"]
    H --> I["前376年：三家尽分晋地，晋国终结"]
```

| 阶段 | 具体过程 | 转折与意义 |
|---|---|---|
| 卿族坐大 | 晋国以异姓卿族统军、治邑，卿族在长期战争中积累封邑、兵力和附属人口。 | 国君对军政资源的直接控制不断缩小，晋国由公室政治转向卿族竞争。 |
| 六卿兼并 | 赵、魏、韩、智、范、中行六卿彼此攻伐，范氏和中行氏先败，形成四家角逐。 | 晋国不再只是“君弱臣强”，而是进入领地重新分配阶段。 |
| 晋阳之战 | 智伯联合韩、魏围赵氏于晋阳，并引水灌城；赵襄子说服韩、魏反戈，三家在前453年灭智氏。 | 智氏覆亡使韩、赵、魏成为晋国内无可抗衡的三大权力中心。 |
| 名分合法化 | 三家继续占有晋公室土地，前403年由周威烈王正式册命为诸侯。 | 既成事实获得周王室承认；《资治通鉴》以此年开篇，因此常被用作战国起点。 |
| 晋国余绪终结 | 晋君一度只保有绛、曲沃一带，前376年三家分尽其余地。 | “分晋”是历时数十年的政权替代过程，而非某一天一次完成。 |

## 原因与历史影响

- **结构因素**：晋国军政长期依赖卿族，封邑、家臣和私属武装不断扩大，公室却因内乱和君权衰退失去制衡能力。
- **竞争机制**：六卿通过结盟、倒戈和兼并重组权力；韩、魏在晋阳之战中改变阵营，是智氏由最强转为覆亡的直接转折。
- **直接结果**：韩、赵、魏分别建立领土国家，晋国的旧政治空间被三国继承。
- **长期影响**：周王册命显示王室仍能提供名分，却已无力逆转诸侯国内部的权力转移；世卿取代公室成为春秋政治秩序解体的集中表现。
- **年代口径**：“三家分晋”可分别指前453年灭智、前403年受命或前376年晋国最后领地被分尽。不同史书对晋末君主与个别年份也有差异，阅读时应先说明所用口径。

## 说明

- 晋文公回国即位时，许多随从成为晋国世袭贵族，晋国国政逐渐落入士大夫手中。
- 春秋末期，晋国国君已近傀儡，实权由卿大夫家族掌握。
- 晋国六卿中，智氏、范氏、中行氏与韩、赵、魏相互兼并。
- 范氏、中行氏被消灭，封地被其他四家瓜分。
- 前455年，智伯瑶联合韩、魏合兵攻赵，围赵襄子于晋阳，并决汾水灌城。
- 晋阳将破时，韩、魏倒向赵氏，于前453年共同消灭智氏，瓜分其土地，史称“三家灭智”。
- 之后韩、赵、魏又瓜分晋公室土地和人民，晋国国君只保有曲沃、绛等少量地区。
- 晋君地位降到三家之下，甚至要朝见三家大夫。
- 前403年，周威烈王册立韩、赵、魏三家为侯国，这是《资治通鉴》所取春秋战国分界点。
