# 秦灭巴蜀之战

## 时间

前316年，周慎靓王五年。

## 概括

秦灭巴蜀之战是秦国取得西南战略纵深的关键战争。秦国以援救苴、巴为出兵契机，先灭蜀，继而控制巴、苴；此后一度保留蜀侯和地方首领，同时逐步设置巴、蜀等郡县。巴蜀后来成为秦国重要粮赋基地，也为秦国由西南进攻楚国提供条件。

```mermaid
    H --> I["开发水利与交通，成为秦的粮赋和战略后方"]
```

| 阶段 | 具体过程 | 战略意义 |
|---|---|---|
| 出兵背景 | 蜀攻苴，苴侯逃往巴；巴、苴求助于秦。秦廷同时讨论向东伐韩和向西取蜀，司马错主张先取地广民富而防备较弱的巴蜀。 | 地方冲突为秦提供介入机会，秦的目标则超出单纯“救援”。 |
| 灭蜀 | 秦军通过蜀道南下，击败蜀军；传世记载称蜀王及太子、国相先后败亡。 | 秦首先消灭区域内力量较强的蜀，避免在巴蜀之间长期周旋。 |
| 取巴、苴 | 蜀亡后，秦继续向东控制巴国和苴地，原有王侯政权被解除。 | 秦取得长江上游、嘉陵江流域及通往楚地的交通节点。 |
| 分层治理 | 秦初期并非只用一种制度，而是一度分封蜀侯、设置守相，同时在核心地区推行县级行政，并与部分地方首领合作。 | 渐进接管降低了新征服区的统治成本，郡县化随后不断加深。 |
| 长期整合 | 秦持续修筑交通、发展成都平原农业与水利，巴蜀逐渐成为粮赋基地和攻楚后方。 | 关中之外形成第二战略纵深，显著提升秦承受长期兼并战争的能力。 |

## 胜因、影响与材料边界

- **内部条件**：巴、蜀、苴相互冲突，未能形成共同防线，是秦迅速介入的直接背景。
- **秦国优势**：变法后的行政和军事动员、蜀道交通条件以及明确的资源战略，使远征能够转化为长期占领。
- **直接结果**：巴蜀原有王权覆亡，但地方文化和首领网络并未立即消失；秦采取分封、郡县和地方合作并用的方式逐步整合。
- **长期影响**：巴蜀粮食、人口、水运和矿产增强秦国国力，并使秦能由汉水、长江上游夹击楚国。
- **史料限制**：战争细节大量依赖成书较晚的《华阳国志》等材料；郡县设置和“若干县”的数字可能反映数十年的累积过程，不宜全部系于前316年一次完成。

## 说明

- 前316年，蜀王讨伐苴侯，苴侯出走巴国。
- 巴国因与秦惠文王关系较好，请求秦国出兵援救苴、巴。
- 秦惠文王派张仪、司马错率兵救援。
- 秦军讨伐蜀国，蜀王兵败退至武阳，被秦军杀害。
- 蜀国国相及太子皆死于白鹿山。
- 蜀亡后，秦军继续向东控制巴国与苴地，俘获或废除原有王侯；这反映秦的目标是接管整个长江上游战略区，而非单纯救援。
- 巴国、苴国随之灭亡。
- 秦在巴蜀采取分封蜀侯、设置守相、推行县级行政和与地方首领合作等多种方式，巴郡、蜀郡等行政区是在后续整合中逐步形成；“四十一县”等总数不宜视为前316年一次设置。
- 秦国灭巴蜀后，向西南拓展领土，达到司马错所说“广地”“富国”“强兵”的目标。
- 巴蜀地区成为秦国稳定后方，并为后来的黔中之战、鄢郢之战提供水道和战略基础。
