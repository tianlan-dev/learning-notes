# 齐桓公称霸

## 时间

前685年－前643年，齐桓公在位时期；前651年葵丘会盟为称霸标志之一。

## 概括

齐桓公任用管仲改革齐国，以“尊王攘夷”为号召组织诸侯会盟，干预诸侯秩序、抵御夷狄，成为春秋五霸之首。齐桓公称霸标志着春秋霸政正式成形。

## 称霸过程、运行机制与局限

| 阶段 | 具体过程 | 霸政作用 |
|---|---|---|
| 继承与用人 | 公子小白在齐国内乱后夺得君位，接受鲍叔牙建议任用曾辅佐对手的管仲。 | 把政敌转化为执政人才，稳定新政权。 |
| 国家整顿 | 管仲改革基层组织、军政动员、财政和商业管理；具体“井田”“盐铁”制度多来自后世整理，不能全部视为一次颁布。 | 齐的海滨经济和临淄城市资源转化为可持续军事财政。 |
| 尊王攘夷 | 齐以周王名义救援邢、卫等国，组织诸侯应对山戎、狄及内部秩序危机。 | 提供公共安全并换取盟主合法性。 |
| 遏制楚国 | 齐率多国军队南下，以楚不贡周为名施压，形成召陵之盟。 | 避免与楚决战而取得外交成果，显示联盟威慑。 |
| 葵丘高峰 | 葵丘会盟得到周王室认可，齐对诸侯盟誓、继承和战争规则施加影响。 | 霸主在王权衰落后代行部分秩序协调，但不拥有直接统治各国的权力。 |
| 霸业瓦解 | 桓公晚年近臣政治和继承安排失控，死后诸子争位。 | 个人—君相型联盟缺少稳定继承，齐霸迅速结束。 |

- 齐桓公霸业由**经济基础、管仲改革、周王名义和诸侯安全需求**共同构成。
- “尊王攘夷”既有维护共同秩序的一面，也服务齐国扩大影响，不能当作纯粹道德口号。
- 管仲改革细目见于《管子》等不同年代文本，宜区分春秋核心事实与后世制度化叙述。
- 齐霸未转化为帝国：诸侯仍保留军队、税收和外交，盟主权威随齐国内乱迅速下降。

## 说明

- 前685年，齐桓公继位，任用管仲为相。
- 传统文献把按土地条件征赋、基层编组、商业与盐铁管理等归于管仲；具体措施是否在同一时期由一次改革完成，仍需结合不同年代的文本判断。
- 齐国将基层行政组织和军事组织结合，寓兵于农，提高兵源和作战能力。
- 齐国迅速成为华夏诸国中富强的大国。
- 齐桓公以“尊王攘夷”为号召，多次大会诸侯，帮助或干预其他国家，抗击夷狄。
- 周惠王二十一年（前656年），齐桓公率八国联军陈兵楚境，质询楚国不向周王室朝贡，迫使楚国签订召陵之盟。
- 前651年，齐桓公举行葵丘会盟，成为春秋五霸之首。
