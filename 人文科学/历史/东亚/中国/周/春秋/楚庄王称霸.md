# 楚庄王称霸

## 时间

前613年－前591年楚庄王在位时期；前597年邲之战是关键节点。

## 概括

楚庄王称霸标志楚国在春秋中期突破南方区域强国身份，正式参与中原霸权竞争。楚庄王改革内政、平息暴乱、任用孙叔敖，并在邲之战击败晋国，使楚国一度成为中原诸侯尊奉的霸主。

## 称霸过程、机制与局限

| 阶段 | 具体过程 | 结果 |
|---|---|---|
| 国内整合 | 庄王初期面对贵族和地方力量，随后平息动乱、整顿政令并任用孙叔敖等。 | 楚王重新掌握军政主动，为北进提供后方。 |
| 北方示威 | 楚攻陆浑戎、陈兵周郊并询问九鼎，显示其不再接受周代爵秩的从属定位。 | “问鼎”挑战象征权威，但庄王没有直接灭周。 |
| 郑国争夺 | 郑处于晋楚之间，楚围郑迫其归附；晋军南下试图恢复盟国。 | 晋楚在邲形成决战。 |
| 邲之战 | 晋内部指挥意见不一，楚军抓住机会击败晋军。 | 晋霸受挫，楚获得中原诸侯的广泛服从。 |
| 围宋与霸权 | 楚长期围宋，晋无力有效救援，宋与楚议和。 | 楚霸达到高峰，但齐、晋等并未永久退出竞争。 |

- **崛起机制**是江汉资源、对小国兼并、王权整合和对郑宋等交通节点的控制。
- **邲战胜因**不仅是庄王个人决断，也包括晋军指挥分裂、楚军动员和长期经营北方盟国。
- 霸主地位依赖持续军力和诸侯服从，庄王死后晋楚竞争仍延续，说明霸权没有制度化为统一帝国。
- “问鼎”主要见于传世叙事，其对话细节可能带有后世政治寓意；核心事实是楚已能军事威胁王畿。
- 楚庄王时的直接控制中心仍在江汉与淮北，不能把后世楚文化影响范围全部当作其疆域。

## 说明

- 城濮之战后，楚国继续向东发展，灭亡许多小国。
- 楚庄王时期的核心控制区仍在江汉，势力向淮河流域和中原南缘扩展；军事影响、盟属范围与后世楚文化分布不能等同为直接疆域。
- 楚庄王改革内政，平息暴乱，启用贤臣孙叔敖。
- 孙叔敖兴修水利、改革军制，使楚国国力增强。
- 楚庄王攻克陆浑戎后，陈兵周郊。
- 楚庄王向周定王使者询问九鼎大小轻重，表现出挑战周王室象征权威的意图，“问鼎”一词由此而来。
- 周定王十年（前597年），楚、晋会战于邲，楚军大胜晋国。
- 前594年，楚围宋，宋向晋求救，晋不能救，宋遂与楚言和尊楚。
- 当时中原各国除晋、齐、鲁之外，多尊楚庄王为霸主。
