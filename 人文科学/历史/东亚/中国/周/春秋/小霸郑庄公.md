# 小霸郑庄公

## 时间

前8世纪末至前707年。

## 概括

郑庄公是春秋早期最先显示诸侯强权的人物之一。郑国本为周王室卿士，但郑庄公势力增强后逐渐不尊周王。周郑交质与𦈡葛之战显示周天子权威已难以约束强大诸侯。

## 崛起过程、权力机制与局限

| 环节 | 具体过程 | 意义 |
|---|---|---|
| 东迁基础 | 郑在西周末把人口、财货转移到中原，武公又扩展新郑周边。 | 建国较晚，却占据洛邑以东交通与农业要地。 |
| 国内整合 | 庄公平定共叔段之乱，控制继承竞争和重要城邑。 | 为对外扩张和兼任王室卿士提供稳定后方。 |
| 王臣双重身份 | 郑君既是诸侯又在周王朝执政，能够利用王室名义影响中原。 | 王室依赖郑国，也担心郑庄公独揽朝政。 |
| 从交质到战争 | 周郑互换人质显示互不信任；桓王削郑权后亲率陈、蔡、卫等伐郑。 | 王命不再自动获得所有诸侯服从。 |
| 𦈡葛结果 | 郑军以编组和协同击败王师，桓王负伤；庄公战后仍遣使慰问。 | 郑证明实力，却没有废除周王名义，形成“尊王而制王”的早期霸政特征。 |

- 郑庄公的**小霸**来自王室职位、东迁先机、交通区位和个人政治能力，并非国力已达到后来齐晋规模。
- 郑国缺乏战略纵深，庄公死后又出现继承斗争，早期霸势迅速消退。
- “小霸”是后世概括，不是当时正式称号；其意义在于揭示周王—诸侯权力关系的转折。
- 𦈡葛之战没有立即终结周王室，但使诸侯更倾向以实力和联盟而非单纯王命决定行动。

## 说明

- 郑庄公兼任周王室卿士，因郑国连打胜仗，势力日益增强。
- 周平王不愿让郑庄公独掌朝政，想把一半权力交给另一位卿士虢公忌父，引发郑庄公不满。
- 前720年，周平王将王子狐送往郑国为质，郑国也将公子忽送到雒邑为质，史称“周郑交质”。
- 周郑交质使周天子名义共主地位明显下降。
- 周桓王十三年（前707年），周桓王率陈、蔡、卫等国军队讨伐郑国。
- 郑庄公领兵抵抗，两军战于𦈡葛，史称“𦈡葛之战”。
- 郑军击败王师，周桓王中箭受伤。
- 战后郑庄公派人慰问周桓王，但周王室威信已严重受损。
- 小国为求自保，不再只依附王室，而更倾向依附实力强大的诸侯。
