# 宋国（先秦）

## 时间

- 约前11世纪：商王族后裔微子启受封于宋。
- 前286年：齐、楚、魏共灭宋。

## 概括

宋是商王族后裔封国，位于今河南东部、山东西南和安徽北部一带。宋国既是周代安置殷商遗民的重要政治安排，也是春秋中原格局中的重要诸侯。宋襄公曾试图继齐桓公之后称霸，但宋国总体实力有限，战国时因宋康王扩张引发诸国围攻而灭亡。

## 历史分期与关键过程

| 阶段 | 主要过程 | 政治位置 |
|---|---|---|
| 周初建立 | 周平定武庚势力后封商王族后裔微子启于宋，使其奉商祀并统合部分殷民。 | 具有前代王族封国的礼制地位，却受周王朝政治框架约束。 |
| 春秋早期 | 宋地位于中原交通核心，继承争端和周边诸侯干预频繁；宋与郑、齐、鲁、卫反复结盟交战。 | 有会盟影响力，但人口、疆域不足以压倒大国。 |
| 宋襄公图霸 | 齐桓公死后，宋襄公试图维持盟会秩序；泓水之战拘泥礼战叙事而败于楚。 | 图霸失败，宋转为晋楚竞争之间的中等国家。 |
| 春秋中后期 | 宋处于晋楚两强交界，也成为弭兵与外交活动的重要场所；国内卿族和君位斗争持续。 | 通过外交维持生存，难以建立稳定霸权。 |
| 战国末期 | 宋康王扩张滕、薛等地，并与齐、楚、魏发生利益冲突。 | 扩张引发联合遏制，前286年三国共灭宋。 |

## 兴衰与灭亡原因

- **早期合法性**：奉商祀与殷民基础使宋具有独特礼制身份和较强文化凝聚力。
- **区位双刃剑**：中原交通便利带来商贸和外交价值，也使宋持续暴露于齐、楚、晋、魏等强国压力。
- **实力上限**：国土和战略纵深有限，宋襄公的盟主诉求缺乏持久军力与盟邦约束能力。
- **生存机制**：会盟、调停和在大国间周旋延长了国祚，但不能阻止战国兼并规模升级。
- **外交孤立**：宋康王晚期对外扩张同时损害多个邻国利益，失去可制衡齐、楚、魏的联盟。
- **直接灭亡**：前286年齐主攻、楚魏参与，宋军无法承受多国夹击，领土遂被瓜分。

## 说明

- 周初，微子启受封于宋，以奉商祀，是周代分封前代王族后裔的典型。
- 宋国地处中原交通要道，常处于齐、楚、晋、郑、鲁等国影响之间。
- 春秋时期宋襄公图霸，在泓水之战中败于楚，宋国霸业未成。
- 宋国在春秋战国之间仍保持一定实力，但难以与大国长期抗衡。
- 战国时期宋康王一度扩张，攻滕、灭薛，号称“五千乘之劲宋”。
- 前286年，齐、楚、魏共灭宋，瓜分宋地。
