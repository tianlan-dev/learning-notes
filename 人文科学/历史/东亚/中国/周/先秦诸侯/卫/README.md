# 卫

## 时间

- 约前11世纪：周公弟康叔受封于卫。
- 前209年：秦二世废卫君角，卫国最终结束。

## 概括

卫是周初姬姓诸侯国，始封君为康叔。卫国早期地位较高，春秋以后逐渐衰落，长期夹在晋、齐、郑、宋等国之间。战国时卫降为附庸性小国，却一直延续到秦统一之后，是周代诸侯国中存续时间很长的一支。

## 历史分期与关键过程

| 阶段 | 主要过程 | 政权状态 |
|---|---|---|
| 周初建立 | 三监之乱平定后，康叔受封于殷商旧地，兼具安抚殷民和控制东方交通的任务。 | 依托王室近亲身份，早期爵位和政治地位较高。 |
| 春秋初内乱 | 州吁弑君、诸公子争位及外部诸侯干预反复发生，国君与卿族的权力边界不稳。 | 战略位置重要，但内耗削弱持续动员能力。 |
| 狄破与复国 | 前660年狄人攻破卫，卫懿公死；齐桓公等援助遗民，卫戴公、卫文公先后重建政权并迁都。 | 国家没有永久灭亡，却失去旧都和大片实力基础。 |
| 春秋中后期 | 卫在晋、楚、齐等强国之间反复选择依附方向，献公、出公等又经历放逐、复位和父子争国。 | 保留宗庙与国君，实际自主空间持续缩小。 |
| 战国至终结 | 卫降为侯、君，成为魏、秦势力范围内的小国；秦统一后仍保留名号。 | 缓冲和附庸身份延长国祚，前209年才被秦二世正式废除。 |

## 兴衰与存续原因

- **建立基础**：康叔的宗室身份、殷商故地的人口与交通价值，使卫成为周初东方秩序的重要支点。
- **结构弱点**：地处黄河中下游强国交界，缺乏安全纵深；频繁的继承斗争又使外部势力容易介入。
- **复国条件**：狄破卫后仍有宗族、遗民和诸侯援助，卫文公能够以迁都和恢复生产重新组织国家。
- **长期衰落**：领土缩减、卿族政治和对晋、魏、秦的依附使卫从诸侯国转为名义封君。
- **延续而非强盛**：强国把卫视为缓冲或附庸，保留其名号的成本较低；这解释了它为何弱而长存。
- **直接终点**：秦帝国不再需要保留诸侯名号，秦二世废卫君角，卫的政治世系遂告结束。

## 说明

- 周公平定三监之乱后，康叔受封于殷商故地附近，负责安抚殷民并控制东方要地。
- 卫国都城曾在朝歌等地，后来因狄人入侵和政治变动迁徙。
- 春秋时卫国内乱频繁，国君与卿族、诸公子之间斗争不断。
- 卫地与孔子生平关系密切，孔子周游列国时多次至卫。
- 战国时期卫国国力微弱，先后依附魏、秦等强国。
- 秦统一六国后仍保留卫君角，前209年秦二世废卫君，卫国最终结束。

## 演变关系

| 关系 | 说明 |
