# 越

## 时间

- 先秦传说时期：越人常被追溯为会稽一带古老族群。
- 前473年：越王勾践灭吴，越国称霸。
- 前306年左右：楚大败越，越国主体瓦解。

## 别称

- 于越
- 越国

## 概括

越是会稽一带的诸侯国，春秋晚期因吴越争霸进入中原政治视野。越王勾践卧薪尝胆，最终灭吴并北上会盟，成为春秋晚期霸主之一。战国时期越国内部分裂，后被楚大败，越国主体瓦解。

## 历史分期与关键过程

| 阶段 | 主要过程 | 结果 |
|---|---|---|
| 早期形成 | 越国以会稽一带为核心，早期世系多含传说，地方考古文化与后世王统不能简单一一对应。 | 形成适应河网、山地和海岸环境的区域政权。 |
| 吴越战争 | 允常、勾践时期越与吴竞争；勾践战败后入吴臣服，越保留国内统治。 | 军事失败转化为长期休养和复仇政治。 |
| 勾践霸业 | 越整顿人口、农业、军队和外交，利用吴北上争霸造成的空隙连续反攻。 | 前473年灭吴，取得长江下游主导权并北上会盟。 |
| 霸权消退 | 勾践以后越王位争夺频繁，政治中心和控制范围变化。 | 个人霸业未转化为稳定跨区域制度，诸支势力分化。 |
| 楚击越与余绪 | 战国中期楚向东扩张，约前306年击败越国主体；具体灭越过程和疆域仍有史料争议。 | 越王国中心瓦解，但东南沿海各越人共同体并未因此消失。 |

## 崛起与瓦解原因

- **区域基础**：会稽山海、河网交通和地方人口支持越在吴国东南建立持续压力。
- **危机动员**：勾践政权以战败为契机恢复生产、控制资源并经营对吴情报和外交。
- **战略耐心**：越避免在国力未复时决战，等待吴两线扩张和内部疲敝。
- **制度局限**：灭吴后的广大领地依赖勾践个人权威和军事成果，继承冲突削弱中央整合。
- **外部竞争**：楚国拥有更大人口、纵深与战国式动员能力，逐步进入长江下游。
- **史料局限**：越后期世系和“楚灭越”年代存在不同解释，应区分王国主体瓦解与越人政权、文化延续。

## 说明

- 越国活动中心在今浙江绍兴一带，会稽是其重要政治与文化地标。
- 春秋晚期，越与吴长期对抗，形成吴越争霸主线。
- 越王勾践败于吴王夫差后称臣，后通过休养生息、整顿内政和等待时机复兴。
- 前473年，越灭吴，取得长江下游主导权。
- 勾践灭吴后北上会盟，一度被视为春秋霸主。
- 战国时期，越国王族内争，势力分散。
- 前306年左右，楚国大败越，越国主体瓦解，残余势力散布东南沿海。

## 演变关系

| 关系 | 说明 |
|---|---|
| 前一节点 | 会稽一带古越族群与地方政权。 |
| 并列关系 | 春秋晚期与吴长期争霸。 |
| 后一节点 | 战国中期被楚击破，主体瓦解。 |
