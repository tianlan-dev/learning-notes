# 秦国（先秦）

## 时间

- 前905年：非子受封秦邑，为周天子附庸。
- 前770年：秦襄公因护送周平王东迁有功，正式列为诸侯。
- 前221年：秦王嬴政统一六国，诸侯国秦转为秦朝。

## 别称

- 秦国
- 嬴秦

## 概括

秦是嬴姓诸侯国，早期活动于周朝西陲，与西戎长期作战。秦襄公护送周平王东迁后，获封岐山以西之地，正式成为诸侯。春秋时秦穆公称霸西戎；战国时秦孝公任用商鞅变法，秦国由西方强国转变为统一天下的主导力量。

## 历史分期与关键过程

| 阶段 | 主要过程 | 力量变化 |
|---|---|---|
| 附庸形成 | 非子因养马受封秦邑，嬴姓集团在周西陲承担畜牧、交通和边防职能。 | 依附周王室，尚非正式诸侯。 |
| 列侯与关中立足 | 秦襄公护送周平王东迁而列为诸侯，获许经营岐山以西；历代与西戎作战，逐步取得关中。 | 由边地附庸转为拥有稳定领土的诸侯国。 |
| 春秋西向霸业 | 秦穆公用百里奚等人，参与中原事务但在殽之战受晋阻击，转而扩张西戎。 | 控制西部交通和族群网络，形成独立强国基础。 |
| 战国制度重建 | 秦献公改革后，孝公任用商鞅推行军功爵、县制、户籍和土地制度等变法。 | 国家直接动员户口、赋税和军队的能力显著增强。 |
| 区域扩张 | 惠文王称王，灭巴蜀并争夺汉中、河西；昭襄王时期以远交近攻和连续战争重创韩赵魏楚。 | 关中、巴蜀构成互补后方，秦获得持久战争优势。 |
| 统一转型 | 庄襄王时期灭东周国，秦王政建立集中决策和大军团体系，前230—前221年依次灭六国。 | 诸侯国秦结束，转化为覆盖全国的秦朝。 |

## 崛起与统一原因

- **地理与后方**：关中易守难攻，郑国渠等水利和巴蜀粮源增强长期供给；地理优势须与国家建设结合才转化为军力。
- **制度动员**：商鞅变法削弱世卿特权，以军功、县制和严密户籍连接土地、赋税与兵役。
- **持续用人**：秦能吸收商鞅、张仪、范雎、白起、李斯等外来人才，并由君主与官僚体系执行长期战略。
- **外交与次序**：先巩固西部和巴蜀，再逐步夺取河西、汉中与三晋，避免同时与所有强国决战。
- **对手分裂**：六国改革深度、联盟互信和战略目标不一，合纵难以长期维持。
- **性质转变**：前221年不是秦国“灭亡”，而是秦王取消诸侯并称皇帝，秦的国家机器由一国制度扩展为帝国制度。

## 说明

- 秦的先祖传说可上溯至伯益。伯益因辅佐大禹治水有功，被帝舜赐嬴姓。
- 商代时，中衍、蜚廉、恶来等为秦、赵共同祖源中的重要人物。武王克殷时恶来被杀，恶来一系后来成为秦国祖先，季胜一系后来形成赵氏。
- 季胜后人造父在周穆王时为周王御者，受封赵城，遂为赵氏。
- 恶来后裔居于犬丘。非子因善于养马，于周孝王时受封秦邑，号为“秦嬴”。
- 秦最初处于周朝边缘地区，西接西戎，长期通过对戎战争扩张实力。
- 前771年，申侯联合犬戎攻打周王室，周幽王被杀。秦襄公率军救周，并支持周平王东迁洛邑。
- 秦襄公因功被周平王封为秦伯，并获赐岐山以西之地，秦由附庸转为正式诸侯国。
- 春秋时期，秦穆公任用百里奚等人，施德诸侯，向西拓展，称霸西戎。
- 战国时期，秦孝公任用商鞅变法，整顿田界与赋役、奖励耕战、推行军功爵和县制，并统一部分度量标准，为秦国持续扩张奠定制度基础。
- 秦惠文王正式称王，重用张仪等人，又命司马错伐蜀，灭巴、蜀两国，取得关中以外的重要战略后方。
- 秦武王举鼎而亡。
- 秦昭襄王时，宣太后诱杀义渠王，秦并入义渠；又重用魏冉、范雎、白起等人，持续削弱六国，并灭西周国与周朝。
- 秦庄襄王时期，秦相吕不韦灭东周国。
- 秦王嬴政继承秦国积累的制度与军事实力，完成统一，建立秦朝。

## 演变关系

| 关系 | 说明 |
|---|---|
| 前一节点 | 周天子附庸秦邑。 |
| 并列关系 | 春秋时与晋、楚、齐等争雄；战国时为七雄之一。 |
| 后一节点 | 前221年秦灭六国，秦国转入秦朝。 |
