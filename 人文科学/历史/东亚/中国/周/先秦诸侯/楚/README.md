# 楚国（先秦）

## 时间

- 西周初：熊绎受封于楚蛮，居丹阳。
- 前704年：楚子熊通自立为王，史称楚武王。
- 前223年：秦军攻破寿春，楚国灭亡。

## 别称

- 荆
- 荆楚
- 楚国

## 概括

楚是芈姓熊氏诸侯国，起源可追溯至商周之际，最早兴起于丹江流域。楚在中原诸侯眼中长期带有“蛮夷”色彩，却在春秋战国持续扩张，成为南方第一大国。楚庄王是春秋霸主之一，战国时期楚又是七雄之一，最终在秦统一战争中灭亡。

## 历史分期与关键过程

| 阶段 | 主要过程 | 国家形态变化 |
|---|---|---|
| 形成与受封 | 楚族在丹江—汉水流域发展，熊绎在周成王时受封；早期爵位低，需在周王朝边缘争取空间。 | 周封号与地方族群、文化网络逐渐结合。 |
| 江汉扩张 | 熊渠、楚武王、楚文王等向江汉、淮北扩张，灭并众多小国；前704年楚武王称王。 | 从子爵封国转为独立区域王国，并发展县等直接控制方式。 |
| 春秋争霸 | 楚成王北进受挫于城濮；楚庄王整顿内政，前597年邲之战击败晋。 | 进入中原盟会和霸权竞争，国土与人口优势扩大。 |
| 吴楚危机与复国 | 春秋晚期内部贵族矛盾和东部防线失衡，前506年吴军攻入郢；楚在秦援与地方抵抗下复国。 | 国家核心未被摧毁，但东南战略压力暴露。 |
| 战国改革与盛期 | 吴起在楚悼王时变法受贵族反扑而中断；宣王、威王时期楚仍凭广大疆域保持强势。 | 中央集权改革不彻底，封君贵族与地域差异继续制约动员。 |
| 秦压与灭亡 | 怀王时期外交失误和对秦战争失败，汉中等地丧失；白起攻取鄢郢后楚连续东迁。 | 战略纵深尚大但核心区、财政和指挥体系受损，前223年为秦所灭。 |

## 崛起与衰亡原因

- **崛起机制**：江汉水系、广阔土地、人口整合和对小国的持续兼并，为楚提供远超一般诸侯的资源。
- **制度弹性**：楚吸收周礼、地方传统和外来人才，以县制、封君和地方统治并用扩张国家。
- **文化与政治认同**：称王并不等于完全脱离中原体系；楚同时参与会盟、婚姻和外交，形成双重政治语言。
- **结构弱点**：疆域辽阔造成地域整合成本，强宗大族和封君势力使吴起式改革难以持续。
- **外部压力**：吴从东南突袭、秦从西北夺取汉中和郢都，迫使楚的战略中心不断东移。
- **直接灭亡**：秦在统一战争中以持续动员和分进合击消耗楚军，王翦大军最终攻破寿春、俘负刍。

## 说明

- 楚人的活动历史可上追至商朝。商王武丁攻打鬼方时，曾经征讨楚人。
- 楚先祖鬻熊为殷商诸侯。西周初，周成王封熊绎于楚蛮，居丹阳，爵位为子爵。
- 前704年，楚子熊通自立为王，突破周代爵秩秩序，史称楚武王。
- 春秋早期，楚国大举扩张，灭亡江汉流域众多小国，成为南方第一大国。
- 前597年，楚庄王在邲之战中击败晋国，被列为春秋五霸之一。
- 春秋晚期，楚在吴楚争霸中连续受挫。前506年，吴王阖闾攻楚，柏举之战大败楚军，攻破郢都；楚国在秦国援助下复国。
- 战国中期，楚宣王、楚威王时期国势强盛，史称“宣威盛世”。
- 楚怀王后期，内受郑袖等人影响，外受张仪欺诈，蓝田、丹淅等战事失利，楚国逐渐衰落。
- 前278年，秦将白起攻占鄢郢，楚国元气大伤。
- 前223年，秦军攻破楚都寿春，俘楚王负刍，楚国灭亡。

## 演变关系

| 关系 | 说明 |
|---|---|
| 前一节点 | 商周之际南方部族与周初子爵封国。 |
| 并列关系 | 春秋时与晋、吴等争霸；战国时与秦、齐、燕、韩、赵、魏并列为七雄。 |
| 后一节点 | 前223年被秦灭，故地纳入秦统一体系。 |
