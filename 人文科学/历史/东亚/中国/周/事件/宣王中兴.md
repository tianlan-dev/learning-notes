# 宣王中兴

## 时间

前827年－前782年，周宣王在位时期。

## 概括

宣王中兴是西周晚期的短暂复兴。周宣王前期任用大臣、整顿政治、恢复对外征伐，使周室威望有所回升；后期对外战争屡败，又干涉诸侯继承、改变旧制，反而加深王室危机。

## 分期、机制与局限

| 阶段 | 主要措施 | 成效与代价 |
|---|---|---|
| 即位整顿 | 宣王继承共和行政保存下来的王室体系，依靠召、尹、仲山甫等大臣重建决策和诸侯联系。 | 稳定厉王危机后的继承，恢复部分政治信任。 |
| 对外反攻 | 周师和诸侯军进攻𤞤狁、淮夷、徐、荆楚，并以筑城、驻军和财政管理巩固交通。 | 王畿安全和东方影响一度提高，战争也持续消耗人力。 |
| 南北布局 | 分封申伯于南方、封弟姬友于郑，利用亲族和诸侯构筑边防节点。 | 延伸王室网络，却增加地方诸侯的独立资源。 |
| 后期转折 | 对太原之戎、条戎、奔戎和姜戎等战争屡有失利，周军损失扩大。 | “中兴”未能转化为长期军事优势。 |
| 内政紧张 | 料民、赋役调整和干预鲁国继承反映王室试图直接恢复资源与裁判权。 | 政策触及旧制和诸侯自主，可能加剧不满。 |

- **中兴条件**包括共和时期保存的行政连续性、能臣将领和诸侯协作，并非宣王个人一项政策所致。
- **后期衰退**来自长期战争、兵源财政压力、边疆多线冲突和王室干预诸侯所引发的反作用。
- “不藉千亩”究竟是废除籍田礼、改变税制还是礼制批评，学界解释不一，不宜直接等同于一次完整土地私有化改革。
- 宣王时期材料包含金文、《诗经》《国语》《史记》等不同层次，具体纪年和战役对应仍需谨慎。

## 说明

- 前828年周厉王去世，太子姬静即位，是为周宣王。
- 周宣王前期励精图治，任用大臣辅佐朝政，并修建宫殿。
- 周宣王命仲山甫在齐国筑城加强防备，九年在成周洛邑大会诸侯。
- 军事上，周宣王任用将领与诸侯讨伐四周方国异族。
- 周宣王命召穆公率齐国、纪国、莱国等军伐淮夷获胜，并派尹吉甫管理成周至淮夷一带财政。
- 周宣王派南仲皇父率六师伐徐国，派方叔伐荆楚。
- 对外战事中，反攻𤞤狁最重要：五年周宣王率尹吉甫亲征𤞤狁，战于彭衙；又命南仲皇父至朔方筑城御敌。
- 十二年，周宣王派虢宣公伐𤞤狁于洛水，秦庄公可能也在洛水一带追击。
- 周宣王分封申伯于谢（今河南南阳），建立南申国，与吕国成为南方重镇。
- 前806年，周宣王封其弟友于郑，建立郑国。
- 周宣王后期对外屡战屡败，征伐太原之戎、条戎、奔戎惨败，对申戎虽胜，却又被姜戎击败于千亩。
- 制度方面，《国语·周语上》记有“不藉千亩”“料民于太原”“立戏伐鲁”等事。
- 周宣王废除藉礼，承认公田私有化，改行按亩征收实物的“彻”制，引发虢文公抗议。
- 周宣王不听仲山甫劝谏，在太原清查人口，以补充军队损失。
- 周宣王干涉鲁国继承，废鲁武公长子括，改立少子戏，导致鲁国内乱，后又派兵平乱。
- 前782年周宣王去世，其子姬宫湦继位，是为周幽王。
