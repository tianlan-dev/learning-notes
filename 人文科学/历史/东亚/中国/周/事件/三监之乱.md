# 三监之乱

## 时间

周成王初年，周公旦摄政时期。

## 概括

三监之乱是西周建国后最重要的内外叛乱。周成王年幼，周公旦摄政引发部分王叔不满；武庚试图复商，联合管叔鲜、蔡叔度以及东方方国、东夷、淮夷发动反周战争。周公东征平乱后，周室加强东方分封、营建洛邑，并奠定西周统治基础。

## 过程、原因与影响

| 环节 | 具体过程 | 历史作用 |
|---|---|---|
| 权力真空 | 武王克商后不久去世，成王年幼，周公摄政；宗室内部对摄政合法性和权力分配产生冲突。 | 新王朝尚未建立稳定继承与辅政惯例。 |
| 反周联盟 | 武庚利用复商号召，与管叔、蔡叔及奄、薄姑等东方势力结合；各参与者目标并不完全相同。 | 宗室政争、商遗民复国和东方方国反抗汇成大规模战争。 |
| 政治动员 | 周公先解释摄政正当性、争取召公等核心贵族，再调动周师东征。 | 平乱不仅依靠军事，也依靠宗族联盟和天命话语。 |
| 军事平定 | 战争延续约三年，周军先败三监与武庚，再攻奄、薄姑等东方中心。 | 周由克商联盟进一步转为直接经营东方的王朝。 |
| 战后重组 | 周处置叛乱者、迁徙部分殷民，封康叔于卫、微子于宋，并建设洛邑和东方封国网络。 | 把一次政权危机转化为扩大分封、驻军与行政控制的契机。 |
| 长期影响 | 周公还政成王，摄政危机暂告结束；东方控制仍需诸侯、军队和礼制长期维持。 | 奠定成康时期秩序，也说明西周并非灭商后立即稳固。 |

- 叛乱名称中的“三监”通常指管、蔡、霍，但霍叔在叛乱中的具体角色和三监制度细节存在史料差异。
- 战争年代、各方国位置及《尚书》篇章与后世叙事的对应并非完全确定，应把传统纪年与考古材料区分。
- “周公东征”不是单纯镇压宗室叛乱，而是西周国家从关中向东方扩展和重组统治的关键过程。

## 说明

- 周武王去世后，周成王尚未成年，周公旦以安定周室为由摄政。
- 召公奭、管叔鲜、蔡叔度等人对周公旦掌权不满，其中管叔、蔡叔与武庚联手。
- 武庚号召奄国、薄姑、徐国、熊盈等东方方国，以及东夷、淮夷诸国反周。
- 周公旦先取得召公奭谅解，发布《大诰》，稳定周朝宗室贵族。
- 周公东征历时三年：前两年平定三监与武庚之乱，第三年继续东征东方方国。
- 周公旦杀武庚、管叔鲜，放逐蔡叔度，废霍叔处为庶民。
- 周军消灭奄、蒲姑等东方大国，徐国也南迁至今江苏泗洪一带。
- 平乱后，周朝势力从中原扩展至东方、东北方。
- 周公旦通过分封宗室功臣、修建洛邑、建立礼乐制度、完善封建制度来加强周室控制。
- 周公旦最后还政于周成王。
