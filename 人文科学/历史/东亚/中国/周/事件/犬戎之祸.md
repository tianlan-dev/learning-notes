# 犬戎之祸

## 时间

前771年，周幽王末年。

## 概括

犬戎之祸是西周灭亡的直接事件。周幽王废申后和太子宜臼，改立褒姒及其子伯服，引发申侯强烈不满。申侯联合缯国、犬戎进攻镐京，周幽王、伯服、郑桓公被杀，镐京残破，西周灭亡。此后周平王东迁洛邑，东周开始。

## 原因、过程与结果

| 层次 | 内容 | 作用 |
|---|---|---|
| 结构因素 | 西周晚期对戎战争、王畿资源紧张和诸侯自主性增强，使王室难以独立应对内外危机。 | 为一次继承冲突演变为王朝中心崩溃创造条件。 |
| 宫廷与继承 | 幽王废申后、宜臼，改立褒姒、伯服，直接损害申侯及原太子支持集团。 | 宗法继承争议转化为诸侯反王联盟。 |
| 外部联盟 | 申侯联合缯与犬戎攻击周都，各方可能分别追求拥立、报复和掠夺。 | 王室无法像早期那样有效协调诸侯救援。 |
| 都城陷落 | 幽王等被杀，镐京宫室和府库受损；犬戎撤退后关中仍处于争夺。 | 西周王权核心终结，但周宗族和诸侯体系尚存。 |
| 政治后果 | 宜臼获立为平王，余臣被立为携王；平王迁洛邑，秦晋郑等护送诸侯崛起。 | 西周转入东周，王室从军事中心变为依赖诸侯的共主。 |

- “烽火戏诸侯”是最著名的道德化解释，但其早期文献依据、通信方式和与战争的关系均受质疑，不能把它当作唯一原因。
- 申侯既是受损外戚，也是反王军事联盟的组织者；用“犬戎突然入侵”会遮蔽周内部政治战争。
- 西周灭亡不是周文化、姬姓诸侯或关中周人同时消失，而是镐京王权中心和旧军事体系崩溃。
- 事件直接导致东迁，却还经过二王并立、诸侯护送和土地再分配等中间过程。

## 说明

- 周幽王三年（前779年），褒姒入宫，得到幽王宠爱，并生子伯服。
- 周幽王废正室申后与太子宜臼，改立褒姒为后、伯服为太子，破坏宗法继承秩序。
- 传统叙事中，周幽王为博褒姒一笑，屡次举烽火戏弄诸侯，导致诸侯不再信任王室警报。
- 周幽王任用虢石父为卿，也引发国人怨恨。
- 申后的父亲申侯因女儿与外孙被废而愤怒。
- 前771年，申侯联合缯国与西夷犬戎进攻周幽王。
- 周幽王、伯服、郑桓公在骊山下被杀，褒姒被掳后下落不明，犬戎掠取周室财物而去。
- 镐京残破，周王室无法继续以宗周为都城。
- 犬戎之祸成为西周与东周的分界事件。
- 钱穆《国史大纲》认为“烽火戏诸侯”可能是虚构叙事，理由包括幽王时代长城尚未出现、烽火难以及时召集诸侯。
- 另一种解释认为，申国本为靠近周都的强大诸侯，申侯立宜臼为“天王”与幽王对抗；幽王攻申失败，申国联合犬戎、缯国等击败周幽王。
