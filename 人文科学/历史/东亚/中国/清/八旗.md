# 八旗

## 时间

约1601年形成四旗，1615年前后扩为八旗；作为清代军政、户籍与身份制度延续至1912年。

## 概括

八旗并非单纯的八支军队，而是后金—清国家形成时把人口、户籍、赋役、生产、军事和政治忠诚编在一起的制度。努尔哈赤以女真牛录为基层重组各部，先设四色旗，再扩为正、镶八旗；皇太极又建立蒙古八旗和汉军八旗。入关后，旗制从征服型全民军事组织逐渐转为京师与各地驻防、俸饷和世袭身份体系，前期支撑扩张，后期则因脱离生产、经费僵化和训练衰退而失去主力地位。

| 阶段 | 制度变化 | 历史作用 |
|---|---|---|
| 牛录重组 | 努尔哈赤把归附人口编入牛录，打破部分旧部族边界，由额真统领。 | 统一建州与海西女真过程中，建立直接动员人口和军队的基础。 |
| 四旗到八旗 | 以黄、白、红、蓝四色及正、镶区别形成八旗。 | 扩大编制，也把征服所得人口重新分配给汗与诸贝勒。 |
| 皇太极改革 | 分设蒙古八旗、汉军八旗，吸纳蒙古盟友、降将、炮兵和汉人官兵。 | 提升攻城、火器和跨族群统治能力，使旗制成为国家框架。 |
| 入关与定鼎 | 八旗参加山海关战役、消灭大顺与南明、平定三藩等战争。 | 是清初征服和控制北京的核心武力。 |
| 驻防定型 | 北京设禁旅，各省会、战略城市和边疆要地设置满城或驻防城。 | 监控地方、拱卫交通与边疆，并与绿营分工。 |
| 身份化与俸饷化 | 旗籍世袭，旗人依赖钱粮、住房和官缺，民旗界限受到法律约束。 | 保持统治集团凝聚力，同时产生生计与财政压力。 |
| 晚清转型 | 内乱中八旗、绿营多难胜任野战，清廷依赖团练、湘军、淮军，后编练新军。 | 旗制仍有宫卫和身份作用，但不再是全国军事主力。 |

## 组织结构

| 层级 / 类别 | 说明 |
|---|---|
| 固山（旗） | 最高编组。清代常由都统、副都统处理日常旗务；早期旗主贝勒的政治权力更强。 |
| 甲喇 | 旗以下的中层编组，统辖若干牛录。 |
| 牛录 | 基层军政与户籍单位，额定规模曾随时期变化，不能机械理解为固定人数的现代营。 |
| 满洲八旗 | 以满洲旗人为主体，也包含早期编入的其他人口，是皇权和宗室政治的核心。 |
| 蒙古八旗 | 主要吸收归附蒙古人口；与外藩蒙古的盟旗制度不同，不能混为一体。 |
| 汉军八旗 | 由早期归附汉人、辽东军民及火器部队等发展而来，清初攻城和行政转换作用突出。 |
| 包衣 | 隶属皇室、王公或旗分的服务与生产人口，法律身份和隶属关系复杂，并非一概等同奴隶。 |

## 权力配置

- **上三旗与下五旗**：正黄、镶黄、正白构成上三旗，由皇帝直接控制；其余为下五旗，虽与宗室王公关系密切，旗务仍逐步纳入皇权官僚管理。
- **军户合一**：早期成年男丁承担兵役，家庭、土地和战利品分配均与旗籍相连。入关后专业驻防和固定俸饷增强，原有生产功能减弱。
- **多族群整合**：满洲、蒙古、汉军分别编旗，既保留类别差异，也共同依附皇帝；这帮助后金从女真联盟变成多族群征服国家。
- **旗民分治**：旗人由旗署和特定法律关系管理，地方民人由州县治理。实际生活中买卖、雇佣、婚姻和迁徙又不断冲击制度边界。
- **驻防网络**：北京八旗数量最多，各地驻防控制省城、运河、长江节点、西安、荆州、广州、杭州以及东北和西北要地。

## 军事作用与重要事件

1. 努尔哈赤依靠旗制统一女真诸部，并在萨尔浒之战中击败明军多路进攻。
2. 皇太极扩编蒙古、汉军旗，利用红衣炮等火器加强攻城能力。
3. 1644年八旗入关，随后参加对大顺、南明与明郑的战争。
4. 三藩之乱期间，八旗与绿营共同作战，但也暴露长途作战和供给问题。
5. 康雍乾时期，八旗参与对准噶尔、大小和卓及金川等战争，边疆驻防随版图扩大。
6. 太平天国战争中，江南大营等正规军屡遭失败，地方募兵取代八旗、绿营成为主要野战力量。
7. 1901年后清廷编练新军，旗军仅作局部整顿；1912年清亡后，原旗人俸饷与生计迅速失去制度保障。

## 衰退原因与历史影响

### 衰退原因

- 长期和平和驻防生活使训练、战马与实战经验下降，官兵额数和实际战力逐渐脱节。
- 旗人被限制从事若干职业并依赖固定俸饷，人口增长后出现普遍贫困，国家财政又难同步增加供给。
- 军饷、官缺和差使形成利益体系，整顿会触动既得关系；临时改革难以重建早期军户动员。
- 火器、后勤和近代编制迅速变化，而旗军改革分散，难与湘淮军及新军竞争。
- 八旗的政治身份功能使清廷不愿彻底取消，却也让军事效率与社会保障问题相互捆绑。

### 历史影响

- 八旗把部族联盟改造成能持续征兵、征税、迁徙人口和分配资源的早期国家机器。
- 它是清代满洲身份形成的重要制度环境，但旗人不等于所有满族，八旗内部也并非只有满洲人。
- 入关后的驻防城市、旗民空间和婚姻职业政策深刻影响地方社会。
- 晚清八旗失去主力地位与地方军兴同时发生，是中央军事权下移和近代军阀化的重要背景之一。
