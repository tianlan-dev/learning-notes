# 准噶尔

## 时间

约1634年-1755年为准噶尔汗国的主要统治期；阿睦尔撒纳反清及残余战争延续至1757—1758年。

## 概括

准噶尔汗国是卫拉特蒙古诸部中以绰罗斯氏准噶尔部为核心建立的游牧帝国，中心在伊犁河谷，势力曾覆盖天山南北、阿尔泰和中亚东部。它依靠骑兵、牧业税收、绿洲贡赋、跨中亚贸易和部分火器生产崛起，在噶尔丹、策妄阿拉布坦、噶尔丹策零时期成为欧亚内陆强权。它与清朝、喀尔喀蒙古、哈萨克诸部、西藏政权和俄罗斯帝国反复竞争，最终因继承内战、属部离散和清军多路进攻而覆亡。

## 统治结构

| 层面 | 运作方式 |
|---|---|
| 最高统治 | 绰罗斯氏珲台吉或汗依靠本部贵族、亲族与军队掌权，继承并无固定长子规则，政变和兄弟争位频繁。 |
| 部众体系 | 准噶尔、杜尔伯特、和硕特、土尔扈特、辉特等卫拉特集团既合作又竞争，中央对各部控制强弱不一。 |
| 军事 | 以骑兵为核心，吸收火器、铸炮和筑城技术；能够在草原、山地和绿洲间长距离作战。 |
| 财政 | 征收牧畜、贡赋和商税，控制伊犁、塔里木盆地与中亚交通；绿洲农业人口为汗国提供粮食、手工业品和税收。 |
| 宗教 | 藏传佛教格鲁派提供跨蒙古政治联系，僧侣与西藏局势也会卷入汗国内外竞争。 |
| 对外关系 | 在清、俄罗斯、哈萨克诸部和中亚汗国之间贸易、结盟与战争并行，既非封闭草原政权，也非清朝普通地方政权。 |

## 重要事件

1. **1640年卫拉特—蒙古会盟**：卫拉特与部分喀尔喀首领制定共同法典，试图规范部际秩序和佛教保护。
2. **征服叶尔羌汗国**：噶尔丹在17世纪后期控制塔里木盆地，使汗国获得绿洲税收与南疆通道。
3. **进攻喀尔喀**：1688年噶尔丹攻入漠北，喀尔喀诸部南迁并寻求清朝保护，清准冲突由此成为帝国级战争。
4. **乌兰布通与昭莫多**：1690年噶尔丹在乌兰布通与清军交战；1696年昭莫多战败、补给线崩溃，次年死亡。
5. **1717年进入西藏**：准噶尔军推翻拉藏汗，但其统治激起冲突；1720年清军进入拉萨并驱逐准噶尔。
6. **雍正时期西北战争**：1730年代双方在阿尔泰、漠北反复交战，均承受高昂补给成本，后来形成暂时边界与贸易安排。
7. **1745年继承危机**：噶尔丹策零死后，幼主、庶兄和不同部贵族互相政变，杜尔伯特等集团出走，汗国动员能力瓦解。
8. **1755年清军进伊犁**：清利用降附贵族引路，多路推进，迅速击败达瓦齐。
9. **阿睦尔撒纳反清**：清拒绝其统辖全部卫拉特的要求，试图分封四部；阿睦尔撒纳起兵，清军随后进行更彻底的镇压和人口重组。

## 崛起、衰落与灭亡原因

### 崛起条件

- 伊犁河谷兼具优良牧场、农业和东西交通位置，便于支持大规模政治中心。
- 卫拉特骑兵传统与火器、俘获工匠和绿洲资源结合，提高了攻城与远征能力。
- 明清之际蒙古各部、叶尔羌汗国和中亚政治分裂，为准噶尔扩张提供空间。
- 藏传佛教和部际婚盟帮助统治者跨越单一部族建立权威。

### 衰落因素

- 汗位继承缺乏稳定规则，强势君主死后容易发生亲族、部落与贵族内战。
- 领土横跨草原与绿洲，补给距离长，对降附部众和地方税源依赖很大。
- 清朝能调动内地财政、蒙古盟旗与降附卫拉特，从多个方向持续施压；俄罗斯和哈萨克关系也限制准噶尔退路。
- 1745年后连续政变使将领、属部和人口外逃，达瓦齐与阿睦尔撒纳内斗为清军提供介入机会。

### 直接灭亡与人口灾难

1755年达瓦齐被俘标志汗国中央政权结束。阿睦尔撒纳反清后，清廷把战争目标从更换统治者转向摧毁准噶尔抵抗基础。1756—1758年的战斗、处决、俘掠、逃亡和天花等疫病造成极严重人口损失，伊犁及准噶尔旧地人口结构被重塑。死亡比例与责任分配的具体数字有争议，不宜只归于单一因素；不少现代研究以“种族灭绝”讨论清廷有意消灭准噶尔部众的政策，亦应区分政策意图、战争杀戮与疫病死亡。

## 后续影响

- 清在天山南北建立更直接的军政体系，1762年设伊犁将军，统辖驻军、屯田和多类地方行政。
- 塔里木盆地大小和卓随后起兵，清于1759年前后平定，今日新疆的大范围政治整合由此形成。
- 清迁入或安置满洲、锡伯、索伦、察哈尔、回部与其他人口，改变伊犁等地族群分布。
- 准噶尔覆亡使清与哈萨克、中亚汗国和俄罗斯的边疆关系重新调整，也终结最后一个能够与清长期抗衡的内亚游牧帝国。
