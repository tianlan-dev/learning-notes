# 清末团练

## 时间

清代前期已有地方团练传统；19世纪中叶太平天国战争期间大规模军事化，影响延续至清末新军和民国地方武装。

## 概括

团练是官府授权或默许地方士绅按乡里、保甲、宗族和同乡关系组织的自卫武装。它原用于守村、缉盗和协助治安，不等同于国家常备军。太平天国战争爆发后，八旗、绿营难以控制广大地区，清廷鼓励各省办团，地方督抚又把团丁改造成跨区域作战的“勇营”，湘军、淮军由此产生。团练救助了濒临崩溃的清廷，也把募兵、筹饷和用人权转移给地方官绅，改变中央—地方和军队—社会关系。

## 团、练、勇与正规军

| 类型 | 主要来源 | 指挥与经费 | 典型任务 |
|---|---|---|---|
| 团 / 团练 | 本地户籍、宗族和乡村壮丁 | 士绅、地方官主持，地方捐输或按户筹集 | 守村、筑寨、巡逻、缉盗，通常不远征。 |
| 乡勇 | 临时招募的地方武装人员 | 官绅发给口粮、武器或少量薪饷 | 协助守城、侦察、运输和短期作战。 |
| 勇营 | 营官私人招募、训练的职业化部队 | 督抚与统帅通过地方税、厘金、捐输筹饷 | 跨地区野战和长期围攻，如湘军、淮军。 |
| 绿营 | 国家经制军，主要由汉人组成 | 兵部与地方督抚管理，国家固定兵额和饷项 | 驻防、治安与战时调遣；19世纪中叶战力普遍不足。 |
| 八旗 | 旗籍世袭军政身份体系 | 皇帝、旗署与国家俸饷 | 京师和要地驻防、王朝核心武力；晚清不再是主要野战军。 |

“团练发展为湘军”不是所有地方团丁自动改编，而是曾国藩等人在团练资源上另行建立募兵、营制和统帅负责体系。多数团练仍只在本地活动。

## 建立背景

- 白莲教起义、海盗与地方械斗时期，清廷已经多次依靠乡勇和士绅组织辅助官军，形成可调用的制度经验。
- 19世纪人口增长、土地紧张、灾荒、会党和跨地域流民增加，州县警力与绿营难以覆盖基层。
- 太平军快速穿越广西、湖南、湖北并进入长江流域，传统按营分驻的绿营来不及集中，江南大营等大型官军也屡败。
- 清廷财政无法立即建立一支统一新军，只能让地方自行募兵、筹饷，中央再以名义、官衔和奖惩加以控制。
- 科举士绅掌握宗族、书院、田赋与地方声望，能够比陌生官军更快识别人口、征集粮款和维持寨堡。

## 组织与运行机制

| 环节 | 实际运作 |
|---|---|
| 动员 | 由地方官发布告示，士绅按里甲、宗族、行业或村寨招集；忠诚首先指向家乡和招募者。 |
| 经费 | 依靠捐输、田亩摊派、商捐、关卡和后来的厘金；不同地区负担很不均衡。 |
| 武器 | 初期多为冷兵器、土枪和自备器械，职业勇营逐渐采购洋枪、火炮并设军械所。 |
| 指挥 | 普通团练由绅董、团总管理；勇营形成“统领—营官—哨官—什长”式私人层级。 |
| 纪律 | 乡土与师生关系提高内部互信，但也容易形成对统帅的人身依附和地域排他。 |
| 与官府关系 | 名义受州县、督抚或团练大臣节制；实际控制取决于谁掌握军饷、任免与补给。 |

## 重要事件与扩张过程

1. **咸丰初各地奉命办团**：太平军北上后，清廷广泛要求在籍官绅协助防守地方。
2. **曾国藩在湖南练勇**：1852年奉命帮办团练，随后在衡州建立更集中、职业化的湘军陆师与水师。
3. **厘金兴起**：1853年前后各地开征商业流通税，为长期战争和勇营提供持续财源，后来成为晚清重要财政制度。
4. **湘军跨省作战**：其任务从守湖南扩大到湖北、江西、安徽、江苏，团练由地方自卫转化为督抚统率的战略军队。
5. **淮军赴上海**：1862年李鸿章依湘军经验招募淮勇，在江南与洋枪队协同，地方勇营进一步结合西式军械。
6. **太平天国覆亡**：1864年天京陷落后，清廷裁撤部分湘军，但不少将领转任地方官或继续带兵。
7. **后续内外战争**：勇营网络参与镇压捻军、西北战事和边疆战争，并逐渐成为北洋、新军改革前的主要军事基础。

## 功能、局限与社会影响

### 对清廷的作用

- 用地方社会资源快速补上正规军缺口，守住县城、粮道和战略节点。
- 为湘军、淮军提供兵源、干部、筹饷和情报网络，使清廷最终压制多场内乱。
- 通过士绅介入，把保卫家园、维护宗族财产与保卫王朝结合起来。

### 内在局限

- 地方筹款加重民众负担，绅董可能借机摊派、报复或兼并，官府难以统一审计。
- 团练常卷入宗族、客土、族群和会党冲突，既维持治安，也可能扩大地方暴力。
- 私人募兵使军队忠于统帅和饷源，中央难直接调动；欠饷时容易哗变、扰民或解体。
- 各军装备、训练和号令不同，不利于建立全国统一军事体系。

## 对中央—地方关系的改变

- 督抚获得募兵、任将、筹饷、设局和处分地方官的临时权力，战争结束后其中相当部分被保留。
- 士绅从文化与赋税中介进一步成为治安和军事组织者，地方国家化与国家地方化同时发生。
- 湘淮将领通过军功进入总督、巡抚和外交、洋务职位，形成跨省人事网络。
- 清廷仍可用官位、调任、裁军和财政认可约束督抚，因此不能简单说中央立刻“失去地方”；但权力协商方式已显著变化。
- 清末新军试图以统一编制和学堂训练恢复国家军队，实际仍依赖各省督抚筹款和控制，辛亥革命后这种区域军事基础转入军阀政治。
