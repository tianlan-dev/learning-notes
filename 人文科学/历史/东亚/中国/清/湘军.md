# 湘军

## 时间

1852年曾国藩奉命办湖南团练，1853—1854年形成较完整的湘军陆师、水师；1864年攻陷天京后主力裁撤，人员与制度影响延续至清末。

## 概括

湘军是曾国藩在湖南团练基础上建立的职业勇营集团。它以营官亲自募兵、层层人身负责、儒家士绅教化和地方筹饷为组织特征，以陆军、水师协同沿长江逐城推进，最终成为镇压太平天国的主力。它挽救了清朝，却使军队与统帅、地域和督抚财政紧密结合，成为晚清地方军事化及淮军、楚军等后续集团的模板。

## 建立背景

- 太平军进入湖南、湖北后，绿营和临时调集的官军接连失败，地方自卫已经不能只靠零散团丁。
- 曾国藩以在籍京官身份获得清廷授权，能够联络湖南士绅、书院门生、地方官和富户筹集人力财力。
- 湖南长期存在宗族、乡勇和经世学风网络，为营官招募同乡兵员及以名教约束军纪提供条件。
- 长江战争要求水陆协同；湘军建立水师、船厂和炮队，使其不同于只守乡里的普通团练。
- 清廷财政紧张，允许地方开征厘金、接受捐输并自行筹饷，湘军由此获得相对独立的供给体系。

## 组织与实际权力

| 层面 | 运作方式 |
|---|---|
| 统帅 | 曾国藩负责总体战略、筹饷和人事；胡林翼等督抚提供湖北财政与地方行政支持。 |
| 募兵 | 营官自选哨官、哨官招募士兵，强调同乡、亲族、师生和保结，形成逐级私人责任。 |
| 编制 | 以营为基本作战单位，陆师、水师分工；规模、人数随战争调整，并非完全统一的现代军制。 |
| 军饷 | 依靠湖南、湖北、江西、安徽等地厘金、捐输和地方财政，饷额常高于绿营。 |
| 将领网络 | 曾国荃、彭玉麟、杨岳斌（杨载福）、李续宾、鲍超、罗泽南等各有部队；左宗棠和李鸿章亦由相关幕府与军务网络崛起。 |
| 与朝廷关系 | 名义奉旨作战并受官爵奖惩；实际调兵和任将多由统帅、督抚掌握，清廷通过分区、调任和财政认可加以制衡。 |

## 重要战役与阶段

1. **衡州练军（1853）**：曾国藩把团练改造成有固定营制、军饷和训练的勇营，并建立水师。
2. **湘潭与武汉争夺（1854）**：湘军取得湘潭胜利，随后沿江进入湖北、一度攻取武昌，证明地方军可以跨省作战。
3. **湖口挫败（1855）**：太平军切断水师内外湖联系，湘军遭重创，曾国藩战略一度陷入危机。
4. **再取武昌、围九江（1856—1858）**：借太平天国内乱重整攻势，控制长江中游重要节点。
5. **三河之败（1858）**：李续宾部被陈玉成等歼灭，显示湘军并非持续占优，逐城稳进成为更重要策略。
6. **安庆围攻（1860—1861）**：曾国荃等切断救援，城陷后湘军获得粮饷基地和进攻天京的上游支点。
7. **围困天京（1862—1864）**：吉字营修筑长壕、控制江面，1864年爆破城墙攻入；洪秀全已先死，太平天国主体政权覆亡。
8. **裁军与分流**：曾国藩为减轻财政和清廷疑忌裁撤大批部队，部分将领转入陕甘、新疆、长江水师和地方行政。

## 取胜机制

- **稳定的人身组织**：营官对自己招募的士兵负责，逃亡与失误可追究到具体层级；在旧军失序时提高凝聚力。
- **士绅与财政结合**：地方精英既提供兵员、粮饷和情报，也帮助湘军在占领区恢复税收与基层秩序。
- **水陆协同**：控制长江及支流后，湘军可运送重炮、粮食并封锁城池，逐渐压缩太平军机动空间。
- **围城战略**：避免长期追逐太平军主力，转而夺取武汉、九江、安庆等节点，再围困天京。
- **国家名义与地方执行**：清廷给予官爵、合法征税权和跨省协调，湘军并非完全私人叛军；地方自主又使决策较传统官军灵活。
- **对手内耗**：1856年天京事变及石达开出走削弱太平天国核心，湘军得以从防守转入长期反攻。

## 局限与问题

- 私人募兵提高忠诚，也使军队依赖统帅，中央难以建立统一任免、训练和后勤标准。
- 地方筹饷和厘金加重商业、乡村负担，军队在欠饷或攻城后可能掠夺；天京陷落后的大规模杀戮和破坏是战争结局的重要部分。
- 乡土关系可能排斥外地人员并强化派系，不同湘军将领之间也有资源和声望竞争。
- 湘军擅长内战中的围城和水陆作战，但并未形成持续、统一的近代国家军制。
- 依靠士绅恢复旧秩序有助于清廷续存，却未解决土地、财政和基层政治矛盾。

## 衰减与历史影响

### 主力为何裁撤

- 太平天国覆亡后维持庞大勇营费用过高，地方财政需要恢复。
- 清廷担心曾氏兄弟掌握重兵，曾国藩也以主动裁军表明政治服从。
- 部队长期作战后伤亡、老化和厌战，部分营需要解散或转为其他防军。
- 新的捻军、西北和沿海防务需要不同兵种，人员被分流而非湘军整体继续扩张。

### 长期影响

- 湘军证明督抚可通过地方财政、幕府和私人将领建立战略军队，改变清代兵部—经制军主导的格局。
- 李鸿章以其经验建立淮军，左宗棠组织楚军并经营西北，湘系人物广泛进入督抚与洋务机构。
- 安庆内军械所等尝试推动近代兵工和翻译，成为洋务运动早期环节。
- 地域军系与政治升迁结合，为清末新军和民国军事集团提供制度前史；但不能把后来的所有军阀直接等同湘军。
- 湘军镇压太平天国恢复了清朝统治，也造成深重战争暴力，其“中兴”叙事与地方社会遭遇需要并置理解。
