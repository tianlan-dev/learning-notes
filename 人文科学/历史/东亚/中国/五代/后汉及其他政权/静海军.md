# 静海军

## 时间

905年-938年

## 别称

- 交趾静海军
- 安南静海军

## 概括

静海军是唐末五代时期交趾地区的地方军镇。曲氏、杨氏、吴氏相继控制静海军，逐步摆脱中原王朝直接统治。938年白藤江之战后，吴权建立自主政权，越南历史进入独立发展阶段。

## 地方自主、南汉占领与白藤江转折

- **建立背景**：唐代静海军治交州，管理今越南北部。唐末中央权力衰退、外任节度使独孤损被贬后，本地大族曲承裕于905年掌握交州；906年唐廷追认其节度使身份，地方自主由事实转为名义册封。
- **曲氏治理**：907年曲颢继任，继续沿用中原王朝名号，同时调整基层行政和税役，以本地豪族网络替代唐朝外派官员的直接控制。曲承美继位后在后梁与南汉之间寻求承认，但缺乏足以抵抗岭南军队的外援。
- **南汉占领**：930年南汉发兵攻交趾，俘曲承美并设置驻军官署。占领依靠广州方面的军事输送，地方社会并未被完全整合；931年曲氏旧将杨廷艺从爱州起兵，围攻大罗，逐走南汉军。
- **内部政变**：杨廷艺掌权后仍使用节度使名号，既接受北方名义册封，也与南汉保持复杂关系。937年牙将矫公羡杀杨廷艺夺权，并向南汉求援；杨廷艺女婿、牙将吴权随即起兵。
- **白藤江之战**：938年吴权先杀矫公羡，再利用白藤江潮汐和预置木桩迎击南汉水军。南汉主力战败、统帅刘弘操阵亡，广州方面停止直接征服。
- **政权转型**：吴权获胜后不再以中原王朝授予的静海军节度使作为最高合法性，939年称王、定都古螺。静海军从唐朝边地军镇转化为本地自主王权，交趾由此长期脱离中原王朝直接行政控制。

|---|---|---|
| 905—906年 | 曲承裕掌权 | 本地豪族接管静海军，唐廷追认既成事实。 |
| 907年 | 曲颢继任 | 调整地方行政和税役，巩固曲氏自主。 |
| 930年 | 南汉灭曲氏 | 曲承美被俘，交趾短暂由南汉驻军直接控制。 |

| 938年 | 白藤江之战 | 吴权击败南汉舰队，直接征服尝试失败。 |
| 939年 | 吴权称王 | 废弃节度使式最高名号，建立本地王权。 |

## 说明

- 唐末中原衰乱后，交趾地方豪族曲氏掌握静海军实权。
- 曲承裕、曲颢、曲承美先后经营交趾，削弱中原对当地的直接控制。
- 南汉曾试图恢复对交趾的控制。
- 938年，吴权在白藤江击败南汉军队，交趾取得事实独立。

## 统治结构

| 角色 | 人物 / 机构 | 说明 |
|---|---|---|
| 地方统治者 | 曲氏、杨氏、吴氏 | 逐步推动交趾自主。 |
| 地域核心 | 交趾、安南 | 今越南北部一带。 |
| 外部压力 | 南汉 | 南汉南征失败后，交趾脱离中原直接控制。 |

## 演变关系

- 前一节点：唐代安南都护府、静海军节度使体系。
- 后一节点：吴权建立的越南自主政权。
- 并列关系：[南汉](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E4%B8%9C%E4%BA%9A/%E4%B8%AD%E5%9B%BD/%E4%BA%94%E4%BB%A3/%E5%8D%81%E5%9B%BD/%E5%8D%97%E6%B1%89.md)是静海军独立过程中的主要外部对手。
