# 中国

## 概括

## 读图提示

- 夏、商、周、秦、汉、隋、唐、元、明、清等可作为大一统或主要王朝主线阅读。
- 三国、晋以后到隋统一前，是长期分裂与南北互动阶段。
- 五代十国和辽宋金西夏时期，不能只看中原王朝；北方、东北、西北和江南政权共同构成时代格局。
- 民国以后进入现代国家与政党政治阶段，不再是传统王朝体制。
