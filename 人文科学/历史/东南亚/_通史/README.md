# 东南亚通史

本目录整理跨越中南半岛、海岛东南亚和现代国家边界的共同过程。东南亚的海峡、河谷、山地与群岛彼此连接：贸易带动港市和迁徙，宗教在地方社会中重构，殖民国家建立近代边界，战争和去殖民又形成现代主权国家。各王朝世系、殖民行政首脑和国家领导人仍由对应国家目录维护，本目录不重复长表。

## 历史主线

- **生态与网络：**季风、海峡、河谷和山地物产使贸易呈多中心结构；港口依赖内陆粮食和商品，内陆政权也依靠海上税收与武器。
- **宗教地方化：**印度教、佛教、伊斯兰教、基督教、儒学和华人宗教通过宫廷、商人、僧侣、婚姻与教育传播，并与祖先、地方神灵和习惯法结合。
- **移民与身份：**南岛语扩散、华人和印度人迁徙、殖民劳工及当代跨国就业不断重塑社群；现代民族分类并非史前身份的直接延续。
- **殖民国家：**欧洲势力先争港口和商品，19世纪才普遍转向领土征服、边界勘定、种植园和官僚行政；暹罗以改革与割地维持形式独立。
- **战争与独立：**日本占领削弱旧殖民权威，却造成强制劳动、征粮和屠杀。战后各地通过战争、谈判或两者结合独立。
- **区域合作：**东盟从五国安全与外交协调扩展到11国的经济、政治和社会合作；协商一致维持参与，也限制危机处理能力。

## 大陆与海岛比较矩阵

| 维度 | 中南半岛 | 海岛东南亚 | 共同点与例外 |
|---|---|---|---|
| 地理主轴 | 大河流域、稻作平原、山地通道与沿海港口 | 海峡、群岛航线、火山农业区与香料产地 | 两者都依赖内陆—海岸交换；越南、马来半岛等兼具大陆与海洋属性。 |
| 早期国家 | 河谷都城和灌溉税收较突出 | 港市联盟、海上贡赋和转口税较突出 | 吴哥也参与海贸，爪哇王国也依靠密集农业，不能二分过度。 |
| 宗教格局 | 上座部佛教覆盖缅泰老柬；越南以大乘、儒学和地方信仰并存 | 伊斯兰覆盖多数马来—印尼地区；菲律宾和东帝汶天主教突出，巴厘保留印度教 | 宫廷改宗与村社实践不同步，边疆和山地始终多样。 |
| 殖民经验 | 法属印度支那、英属缅甸；暹罗保持形式独立 | 荷属东印度、西/美属菲律宾、英属马来地区、葡属帝汶 | 殖民法域和间接统治方式在同一帝国内部也不同。 |
| 独立路径 | 谈判、革命战争、分裂和内战交错 | 共和国革命、宪制谈判、联邦重组与占领后独立并存 | 日本占领是共同转折，但并非每国独立的单一原因。 |
| 冷战主线 | 印度支那战争、缅甸内战、泰国军政 | 印尼1965年暴力、菲律宾戒严、东帝汶占领 | 大国介入与土地、族群、军队和国家建构问题叠加。 |
| 东盟 | 后加入的大陆国家扩大组织覆盖 | 五个创始国中四个位于海岛或半岛海域 | 东帝汶2025年加入后，东盟共有11个成员国。 |

## 重要转折

| 时间 | 转折 | 意义 |
|---|---|---|
| 前2世纪—公元4世纪 | 早期跨印度洋港口网络 | 东南亚主动参与有组织贸易，外来物品并不等于外来殖民建国。 |
| 7—9世纪 | 室利佛逝、吴哥前身及大型宗教中心兴起 | 港口、农业和宗教王权形成多种组合。 |
| 10—13世纪 | 宋元海贸扩大 | 陶瓷、铜钱、香料和人口流动推动港市繁荣。 |
| 13—16世纪 | 上座部佛教和伊斯兰网络扩展 | 大陆与海岛的主流宗教格局逐渐形成，但地方差异延续。 |
| 1511年 | 葡萄牙夺取马六甲 | 欧洲军事据点进入海峡，亚洲贸易转向多港竞争。 |
| 19世纪 | 领土殖民国家形成 | 现代边界、官僚、种植园和族群分类逐步制度化。 |
| 1941—1945年 | 日本占领 | 旧殖民权威崩塌，本地组织扩张，同时发生广泛战争暴力。 |
| 1945—1957年 | 第一轮独立高潮 | 印度尼西亚、越南、菲律宾、缅甸、柬老和马来亚走出不同建国路径。 |
| 1967年 | 东盟成立 | 从国家间冲突转向协商合作。 |
| 1975—1979年 | 印度支那与东帝汶剧变 | 战争结束、革命政权、种族灭绝、占领和地区战争相继发生。 |
| 1984—1999年 | 东盟扩大为十国 | 冷战结束前后逐步覆盖整个传统东南亚。 |
| 2007、2015年 | 东盟宪章与共同体 | 区域合作更制度化，但仍以成员主权和共识为核心。 |
| 2025年 | 东帝汶正式加入 | 东盟增至11国，重点转向法律、能力与经济整合。 |
