# 中南半岛大陆王国与上座部佛教

## 时间

9-19世纪。

## 概括

吴哥、蒲甘、阿瑜陀耶、东吁、贡榜、澜沧和越南诸王朝在河谷稻作、寺院网络和区域战争中发展。13世纪以后上座部佛教在大陆西部与中部更广泛传播；王权通过寺院捐赠、法统和功德积累获得正当性，但地方首领与僧团保留自身力量。

## 主要王国

| 王国 | 核心时期 | 主线 |
|---|---|---|
| 吴哥 | 9-15世纪 | 高棉帝国、水利城市与印度教—佛教王权。 |
| 蒲甘 | 11-13世纪 | 统一伊洛瓦底河谷，上座部佛教寺院网络发展。 |
| 阿瑜陀耶 | 1350-1767年 | 暹罗贸易与外交中心，同缅甸、高棉、马来世界互动。 |
| 东吁、贡榜 | 16-19世纪 | 缅甸扩张国家，多次与暹罗和周边政治体战争。 |
| 澜沧 | 14-18世纪 | 老挝王国传统，后分裂为多个中心。 |
| 越南诸王朝 | 10-19世纪 | 红河国家、儒学官僚、南进与占婆—高棉互动。 |

## 说明

- 王国扩张常以控制人口、贡赋和交通节点为核心，不等于完整控制所有领土。
- 佛教、印度教、地方精灵信仰和王权仪式长期并存。
- 战争与迁徙使族群、语言和手工业者跨国流动，现代民族叙事不应倒推至所有时期。
