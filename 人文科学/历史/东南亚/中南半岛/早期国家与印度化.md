# 中南半岛早期国家与印度化

## 时间

约公元前后至9世纪。

## 概括

中南半岛早期政治体依靠河口港市、稻作平原、山地物产和跨海贸易发展。梵文、巴利文、印度教与佛教王权观念被地方统治者采用，但并非印度殖民地。扶南、真腊、骠城邦、孟人政治体和红河地区政权各有不同结构。

## 主要政治体

| 政治体 / 区域 | 说明 |
|---|---|
| 扶南 | 湄公河下游港市网络，名称和边界主要由外部文献与考古重建。 |
| 真腊 | 6-8世纪高棉语政治体的统称，为吴哥国家发展提供前史。 |
| 骠城邦 | 缅甸中部城市网络，受印度洋贸易与佛教影响。 |
| 孟人政治体 | 下缅甸和泰国中部的佛教与城市传统。 |
| 红河地区 | 越南北部在中国王朝统治与地方社会互动下形成不同国家传统。 |

## 说明

- “印度化”是选择、翻译与本地创造过程，寺庙、文字和王号不能证明印度直接统治。
- 海港和内陆通过河流、森林产品与贡品连接。
- 早期政治体通常由多个中心和地方首领构成，边界并非现代地图式固定线。
