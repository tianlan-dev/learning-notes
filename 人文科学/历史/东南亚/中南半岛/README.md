# 中南半岛

## 概括

中南半岛包括越南、老挝、柬埔寨、泰国、缅甸等方向。历史主线包括印度化国家、汉文化影响、上座部佛教王国、山地与平原互动、法国和英国殖民以及现代国家形成。
