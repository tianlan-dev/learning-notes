# 中南半岛历史

## 概括

中南半岛由伊洛瓦底江、湄南河、湄公河、红河等河谷和山地边疆构成。骠、孟、高棉、越、缅、泰、老等政治共同体通过战争、贡赐、婚姻、贸易和宗教相互影响。上座部佛教在缅甸、泰国、老挝、柬埔寨形成重要传统，越南则长期受东亚文字与国家制度影响；现代边界主要在19-20世纪殖民与国家战争中固定。

## 关键辨析

- 古代“扶南”“真腊”等名称主要来自外部文献和铭文重建，不应直接等同于现代民族国家。
- “上座部佛教国家”不表示僧团、王权和地方社会始终一致。
- 山地族群、河谷国家和跨境亲属网络不以现代国界为界。
