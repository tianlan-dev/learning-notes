# 新加坡历史

新加坡位于马六甲海峡东端，早期淡马锡聚落属于马来海洋网络。1819年英国建立自由港后，移民、转口贸易与殖民行政把岛屿塑造成区域都会。经历日本占领、自治、加入马来西亚及1965年分离后，新加坡成为独立城市国家。

## 重要转折

| 时间 | 事件 | 意义 |
|---|---|---|
| 1819年 | 莱佛士与柔佛王室代表订约 | 英国贸易站建立 |
| 1867年 | 海峡殖民地改为王室殖民地 | 新加坡直接归伦敦殖民部 |
| 1942年 | 日军占领新加坡 | 英国亚洲威望遭重创 |
| 1959年 | 实现内部自治 | 李光耀领导的人民行动党执政 |
| 1963年 | 加入马来西亚 | 殖民统治终结 |
| 1965年 | 退出马来西亚并独立 | 城市国家建立 |
