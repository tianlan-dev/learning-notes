# 海岛东南亚

## 概括

海岛东南亚包括马来群岛、印尼、菲律宾、婆罗洲、苏门答腊、爪哇和周边海域。主线是海上贸易、印度化、伊斯兰化、港口苏丹国、欧洲殖民和现代民族国家。
