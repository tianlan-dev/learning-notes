# 海岛东南亚历史

## 概括

海岛东南亚由马来半岛南部、苏门答腊、爪哇、婆罗洲、苏拉威西、菲律宾和小巽他群岛等海域构成。南岛语族群、港市、海峡贸易、森林与香料产品形成流动网络。室利佛逝、满者伯夷和伊斯兰苏丹国是重要节点，欧洲公司统治、殖民群岛与现代国家边界则深刻改变海洋联系。

## 关键辨析

- “马来世界”是语言文化与海洋网络概念，不等同于现代马来西亚领土。
- 伊斯兰化主要通过港口、贸易、婚姻、学者和苏丹国扩展，不是一次统一征服。
- 荷属东印度、英属马来亚和西属 / 美属菲律宾的殖民制度差异很大。
