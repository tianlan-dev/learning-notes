# 殖民群岛与现代海岛东南亚

## 时间

16世纪至今。

## 概括

葡萄牙、西班牙、荷兰、英国和美国以港口、贸易公司、种植园和殖民行政控制海岛东南亚。现代印度尼西亚、马来西亚、菲律宾、新加坡、文莱和东帝汶的边界与这些殖民体系密切相关。日本占领、民族革命和战后建国又重新组织群岛国家。

## 主要殖民体系

| 体系 | 区域 |
|---|---|
| 荷属东印度 | 印度尼西亚群岛，公司统治后转为殖民国家。 |
| 英属马来亚与婆罗洲 | 马来半岛、新加坡、砂拉越、北婆罗洲和文莱的不同保护 / 殖民安排。 |
| 西属 / 美属菲律宾 | 西班牙殖民后由美国统治，二战后独立。 |
| 葡属帝汶 | 葡萄牙殖民延续至1975年，后遭印度尼西亚占领。 |

## 重要过程

- 种植园、矿业与港口引入大规模华人、印度人和爪哇劳工，形成多族群社会。
- 1942-1945年日本占领摧毁欧洲殖民权威，也造成强制劳动、饥荒和镇压。
- 印度尼西亚独立战争、马来亚紧急状态、菲律宾共和国与东帝汶抵抗体现不同国家形成路径。
- 1967年东盟成立后，海岛国家成为区域经济与安全合作的重要推动者。
