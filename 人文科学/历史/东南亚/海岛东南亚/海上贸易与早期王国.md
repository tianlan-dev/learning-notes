# 海岛东南亚海上贸易与早期王国

## 时间

公元前后至13世纪。

## 概括

季风航行把海岛东南亚连接至印度、中国和中东。港市通过控制海峡、征收贸易和组织贡赐发展，而内陆稻作、森林产品和海上族群支持其经济。室利佛逝是苏门答腊—马六甲海峡网络的重要力量，爪哇的夏连特拉、马打兰等王国则发展寺庙、稻作与跨海政治。

## 主要政治网络

| 政治体 | 主线 |
|---|---|
| 室利佛逝 | 7-13世纪海峡贸易与佛教学习中心，影响取决于港口联盟而非固定领土。 |
| 夏连特拉 / 古马打兰 | 中爪哇佛教与印度教王国，婆罗浮屠等建筑反映本地创造。 |
| 占婆与马来港市 | 连接大陆、南海与群岛贸易。 |
| 菲律宾早期港邦 | 吕宋、米沙鄢、棉兰老等地同中国、婆罗洲和马来世界交流。 |

## 说明

- 港市国家的力量依赖航道、商人和盟友，不能按现代国界绘成连续版图。
- 梵文、佛教与印度教传统经由本地语言和王权制度重新解释。
- 南岛语航海者和海上族群是贸易主体，而非帝国边缘人口。
