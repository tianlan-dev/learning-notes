# 海岛东南亚伊斯兰化与港口苏丹国

## 时间

约13-18世纪。

## 概括

伊斯兰教通过印度洋商人、苏菲学者、婚姻、港市统治者和朝圣网络逐步传播。马六甲、亚齐、淡目、马打兰、特尔纳特、蒂多雷、文莱和苏禄等苏丹国在贸易与地方社会中发展。伊斯兰化并未消除印度教、佛教、祖先和地方仪式，而形成地区差异明显的宗教社会。

## 主要苏丹国

| 苏丹国 / 地区 | 主线 |
|---|---|
| 马六甲 | 15世纪海峡贸易中心，1511年被葡萄牙攻占。 |
| 亚齐 | 苏门答腊北部贸易与伊斯兰学术中心。 |
| 爪哇苏丹国 | 淡目、万丹、马打兰等重组满者伯夷后的政治。 |
| 香料群岛 | 特尔纳特、蒂多雷围绕丁香贸易与欧洲势力竞争。 |
| 文莱、苏禄 | 婆罗洲和菲律宾南部的海洋苏丹国网络。 |

## 说明

- 改宗常从港市与王室开始，内陆和岛屿社会有不同节奏。
- 伊斯兰法、习惯法和王权仪式并存。
- 葡萄牙、荷兰、西班牙进入后，苏丹国既抵抗也结盟，殖民扩张不是单向过程。
