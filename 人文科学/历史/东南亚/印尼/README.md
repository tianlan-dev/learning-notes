# 印度尼西亚历史

印度尼西亚历史发生在连接印度洋与太平洋的巨大群岛。香料、稻米、海峡与季风贸易推动多个港市和内陆王国并存；伊斯兰化首先沿贸易网络展开，荷兰东印度公司和殖民政府再逐步把分散岛屿纳入共同行政疆域。20世纪民族主义以“印度尼西亚”整合多语言、多族群社会。

## 重要转折

| 时间 | 事件 | 意义 |
|---|---|---|
| 7—13世纪 | 室利佛逝活跃 | 控制马六甲海峡贸易并成为佛教学习中心 |
| 1293年 | 满者伯夷建立 | 后世群岛统一想象的重要来源 |
| 1602年 | 荷兰东印度公司成立 | 欧洲商业军事势力制度化进入群岛 |
| 1912—1928年 | 群众组织与青年誓言 | “一个祖国、一个民族、一种语言”的认同形成 |
| 1945年 | 宣布独立 | 共和国与荷兰展开四年革命战争 |
| 1965—1966年 | 政治暴力与权力转移 | 苏哈托“新秩序”形成 |
| 1998年 | 苏哈托下台 | 改革时代与民主化开启 |
