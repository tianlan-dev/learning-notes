# 缅甸历史

缅甸历史以伊洛瓦底江流域的国家整合为主线。骠人与孟人城市网络、蒲甘王国、东吁和贡榜王朝先后塑造低地政治；英国殖民统治把边疆族群地区纳入同一行政疆域，独立后中央建国、军人统治与族群冲突长期交织。

## 重要转折

| 时间 | 事件 | 意义 |
|---|---|---|
| 1044年前后 | 阿奴律陀建立蒲甘霸权 | 上座部佛教与缅文国家传统形成 |
| 16世纪 | 东吁王朝扩张 | 一度建立东南亚大陆最大帝国之一 |
| 1885年 | 第三次英缅战争 | 贡榜王朝灭亡，缅甸完全并入英属印度 |
| 1948年 | 缅甸独立 | 建立缅甸联邦 |
| 1962年 | 奈温政变 | 长期军人统治开始 |
| 2021年 | 军方再次夺权 | 政治危机和全国性武装冲突扩大 |
