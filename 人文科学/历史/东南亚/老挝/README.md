# 老挝历史

老挝历史以湄公河中游的河谷、山地与跨境人群为背景。澜沧王国在14世纪整合多个老族勐邦，并以琅勃拉邦、万象为中心发展上座部佛教文化；18世纪分裂后受暹罗和越南夹击，19世纪末被纳入法属印度支那。独立后的王国又因冷战和越南战争陷入内战，1975年建立老挝人民民主共和国。

## 重要转折

| 时间 | 事件 | 意义 |
|---|---|---|
| 1353年 | 法昂建立澜沧 | 形成长期老挝国家传统 |
| 1560年前后 | 塞塔提腊迁都万象 | 王国重心南移并应对缅甸扩张 |
| 1707年 | 澜沧正式分裂 | 多个老挝王国并存 |
| 1893年 | 法暹危机 | 湄公河东岸纳入法国保护范围 |
| 1953年 | 完全独立 | 老挝王国获得主权 |
| 1975年 | 人民民主共和国成立 | 君主制终结 |
