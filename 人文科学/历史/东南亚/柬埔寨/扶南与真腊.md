# 扶南与真腊

## 时间

约1—8世纪

## 概括

“扶南”和“真腊”主要是中国史籍对湄公河下游政治体的称呼，并不一定等同于两个疆界固定、中央集权且前后整齐继承的现代国家。扶南大致以湄公河三角洲、安江—茶胶低地和沿海港市为核心，通过运河、河道与季风航线连接印度洋和南海。6世纪以后，湄公河中游及洞里萨湖周边的高棉语政治中心逐渐扩大，隋唐史籍称其为真腊。两者的转换既包括征服和王族竞争，也包含人口、官员、宗教与贸易网络的延续。

## 史料范围与年代争议

- 扶南王名主要来自中国使节、僧侣和朝贡记录，常以汉字转写称号；把这些名字还原成梵语名和精确在位年具有不确定性。
- 考古学中的俄厄、吴哥波雷、波侬达等遗址证明下游贸易和宗教中心繁荣，却不能把所有遗址自动归给同一位“扶南国王”。
- 真腊资料较多来自梵语、古高棉语铭文。铭文常记土地捐献、神庙和王族关系，比后世编年更接近当时，但仍无法构成无缺口的全国王表。
- 唐代“陆真腊”“水真腊”是外部文献对多个中心的概括，不宜画成两个边界清楚、各有单一王统的国家。
- 下列世系表采用“史料明确出现的统治者与空缺”原则；未知区间不以虚构姓名填补。

## 扶南的形成与崛起机制

公元初期，湄公河三角洲已存在稻作、渔业、盐业和手工业聚落。季风航海发展后，来自印度、南海和中国沿岸的商人可在港市换装、修船和等待风向。地方首领通过控制河口、仓储、航道和内陆物产取得财富，再以礼物、婚姻和武力联合其他聚落。俄厄出土的外来钱币、宝石、玻璃、印章和本地作坊遗存，说明贸易繁荣建立在本地生产与转运能力上。

梵语、婆罗米系文字、印度教和佛教观念经商人、僧侣与宫廷专家传播。本地统治者采用“跋摩”等称号、神庙捐献和宇宙王权语言，以扩大跨地方合法性；这是一种选择性吸收和本地重构，而非印度人口对湄公河流域的直接殖民。中国史籍中的混填与柳叶女王婚姻故事，表达外来仪式知识同本地权力结合的政治记忆，其具体人物和年代不能当作已证实事实。

## 真腊的形成与统治结构

6世纪前后，湄公河中游的地方王族以河谷农业、陆路贸易和军事联盟为基础向南扩张。跋婆跋摩一世、质多斯那—摩诃因陀罗跋摩和伊奢那跋摩一世的铭文分布扩大，说明多个地方官、武士和寺庙捐献网络接受同一王权称号。伊奢那补罗（今三博波雷古）成为重要宫廷和宗教中心，但王国仍通过属王而非统一官僚直接治理全部地区。

| 层级 | 角色 | 运作方式 |
|---|---|---|
| 国王 | 战争、册封、神庙和最高土地捐献 | 以战功、王族关系及湿婆或毗湿奴崇拜证明合法性 |
| 王族与属王 | 管理区域中心 | 可同时是盟友、姻亲和潜在竞争者 |
| 婆罗门、僧侣与寺院 | 礼仪、书写、教育和土地经营 | 把梵语王权与本地社会连接起来 |
| 地方首领与村社 | 掌握人口、水田、劳役和贡品 | 对中央效忠程度随距离与军事形势变化 |
| 港市与商人 | 联结内陆物产和海上航线 | 扶南旧网络在真腊时期并未突然消失 |

## 真腊统治者与地方王权

前七位可视为6—7世纪较连续的广域王权；阇耶跋摩一世之后，铭文呈现多个并立中心。8世纪人物不得机械排列成统一全国世系。

| 顺序 | 统治者 | 约在位 / 活动时间 | 继承与关键说明 |
|---:|---|---|---|
| 先祖 | 首陀罗跋摩 | 年代不详，后世铭文追述 | 与室利湿多跋摩构成早期王族祖先记忆；未有可靠在位年。 |
| 先祖 | 室利湿多跋摩 | 年代不详，后世铭文追述 | 名称与上丁等北部中心相连，是否统治整个真腊不详。 |
| 1 | 毗罗跋摩 | 6世纪中期 | 王族先辈，具体疆域与在位年不详。 |
| 2 | **跋婆跋摩一世** | 约580—约600年 | 以武力和联盟扩大势力；与摩诃因陀罗跋摩的关系常被解释为兄弟或近亲。 |
| 3 | **摩诃因陀罗跋摩／质多斯那** | 约600—616年 | 延续向湄公河下游扩张；铭文见于广泛地区。 |
| 4 | **伊奢那跋摩一世** | 约616—635 / 637年 | 前王之子；以伊奢那补罗为中心，被中国史籍视为征服扶南的关键统治者。 |

| 6 | **阇耶跋摩一世** | 约657—681年 | 铭文分布广，常被视为最后一位能维持广域权威的统治者。 |
| 7 | 阇耶提／阇耶黛维女王 | 约681年后；713年仍见铭文 | 阇耶跋摩一世之女；其自述政治动荡，但统治范围不等于全部真腊。 |

| 地方王 | 商菩跋摩 | 约8世纪中期 | 地方中心统治者，不能确认统一“陆水真腊”。 |

## 重要事件与发展过程

| 时间 | 事件 | 历史意义 |
|---|---|---|
| 1—3世纪 | 湄公河三角洲港市和运河网络扩张 | 形成连接内陆与跨海贸易的扶南政治经济基础。 |
| 3世纪中期 | 扶南与孙吴互遣使节 | 中国文献开始提供较连续的宫廷信息。 |
| 4—5世纪 | 梵语铭文、印度教和佛教遗物增多 | 本地王权以跨区域宗教语言重塑合法性。 |
| 484—514年 | 憍陈如阇耶跋摩多次遣使南朝 | 反映扶南仍是活跃的外交和贸易中心。 |
| 514年后 | 留陀跋摩争位 | 王族冲突与北方政权扩张叠加。 |
| 6世纪末 | 跋婆跋摩一世与质多斯那扩大势力 | 真腊王权进入湄公河下游和洞里萨湖区域。 |
| 7世纪初 | 伊奢那跋摩一世营建伊奢那补罗 | 建立重要内陆都城与寺庙网络。 |
| 657—681年 | 阇耶跋摩一世广域统治 | 铭文分布显示较强区域整合。 |
| 7世纪末—8世纪 | 多个地方中心并立 | 唐代概括为“陆真腊”“水真腊”，实际关系更复杂。 |
| 8世纪末 | 爪哇 / 室利佛逝势力与大陆政治互动 | 海上强权介入的程度有争议，但重组了南部政治环境。 |
| 802年 | 阇耶跋摩二世王权仪式 | 后世吴哥王权以此追溯统一和摆脱外部控制。 |

## 鼎盛条件、转型与“灭亡”问题

### 扶南为何繁荣

- 湄公河三角洲能同时利用水稻、渔业、森林物产和海运。
- 季风贸易使港市成为印度洋—南海航线的中转点。
- 地方首领通过运河、贡赋和婚姻联盟整合多中心网络。
- 对梵语文字、宗教仪式和外来技术的选择性吸收增强跨区域交流。

### 扶南为何转为真腊主导

- **结构变化**：海上航线和港口兴衰使下游中心的贸易优势波动。
- **内陆崛起**：湄公河中游农业、人口和陆路网络支持新的军事王权。
- **王位竞争**：留陀跋摩时期的继承冲突可能削弱下游中心。
- **直接过程**：跋婆跋摩、质多斯那及伊奢那跋摩的扩张逐步控制原扶南地区，而非某一年一次性“灭国”。
- **连续性**：旧港市、官员、宗教传统和居民继续存在，考古层并不显示全面人口替换。

### 真腊为何分散又走向吴哥

- 阇耶跋摩一世去世后缺少公认男性继承人，王族和地方属王更易独立。
- 河谷地理、地方寺院土地和多重贸易路线不利于长期直接控制。
- 唐代的“陆”“水”分类反映外部观察，不等于正式分裂条约。
- 8世纪多个中心的竞争、海上势力影响与王族迁移促成重新组合。
- 阇耶跋摩二世通过巡行、联盟、军事与仪式逐步整合这些资源，形成[吴哥王朝](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E4%B8%9C%E5%8D%97%E4%BA%9A/%E6%9F%AC%E5%9F%94%E5%AF%A8/%E5%90%B4%E5%93%A5%E7%8E%8B%E6%9C%9D.md)的基础。

## 演变关系

扶南到真腊是政治重心、贸易模式和王族联盟的长期转型，不宜写成单一民族征服另一个民族。真腊晚期的多个中心也不是完全无序的“黑暗时代”，而是吴哥国家所继承的人员、寺院、称号与地方网络。后续见[吴哥王朝](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E4%B8%9C%E5%8D%97%E4%BA%9A/%E6%9F%AC%E5%9F%94%E5%AF%A8/%E5%90%B4%E5%93%A5%E7%8E%8B%E6%9C%9D.md)。
