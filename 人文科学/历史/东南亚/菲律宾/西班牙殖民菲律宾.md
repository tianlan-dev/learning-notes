# 西班牙殖民菲律宾

## 时间

1565—1898年。1898年8月马尼拉投降后，西班牙在伊洛伊洛、三宝颜等尚未移交地区短暂保留残余行政，至1899年撤离；这不再代表对整个菲律宾的有效统治。

## 概括

1565年莱加斯皮在宿务建立永久据点，1571年以马尼拉为殖民首府。西班牙没有先征服一个统一国家，而是逐个与地方首领结盟或交战，把低地聚落编入贡赋、强制劳役、迁村和堂区体系。总督、皇家审问院、修会与本地 principalia 共同构成统治网络；马尼拉—阿卡普尔科大帆船贸易则以美洲白银、中国商品和墨西哥财政补贴维持殖民中心。低地天主教社会逐渐形成，但苏禄、马京达瑙等苏丹国、高地和边远岛屿从未被同等程度控制。

## 分期

| 阶段 | 时间 | 主要过程 |
|---|---|---|
| 立足与征服 | 1565—1600年 | 从宿务转向马尼拉，建立总督区、贡赋和传教据点；以本地盟友参与远征 |
| 大帆船殖民体系 | 1600—1762年 | 马尼拉成为跨太平洋中转港；修会堂区、迁村和地方精英中介深入低地 |
| 英占冲击与改革 | 1762—1821年 | 英国占领马尼拉但未统治全群岛；收复后加强防务、专卖和经济开发 |
| 直接管辖与出口经济 | 1821—1872年 | 墨西哥独立后改由马德里直辖；港口开放、作物出口、教育与交通扩大 |
| 改良主义、革命与终结 | 1872—1898年 | 世俗化冲突和Gomburza记忆推动民族主义；宣传运动受挫后卡蒂普南发动革命，美西战争直接终结西班牙主权 |

## 殖民统治结构

| 层级 | 机构或角色 | 权力与限制 |
|---|---|---|
| 西班牙帝国 | 王室、印度事务机构；1821年前经新西班牙转辖 | 任命总督、制定贸易和教会政策；距离遥远导致地方裁量权很大 |
| 殖民中央 | 总督兼总司令 | 统辖行政、军队、财政和 patronato real 下的教会协调，并可发布具有立法效果的命令 |
| 司法与监察 | 马尼拉皇家审问院、任后审查 | 兼具最高法院、咨询和职位空缺时代政功能，但与总督也常冲突 |
| 省级行政 | alcalde mayor、corregidor，19世纪后为 gobernador civil 等 | 征税、治安、经济许可和司法；官商兼营容易滋生滥权 |
| 镇与村 | gobernadorcillo、cabeza de barangay、principalia | 本地精英负责户籍、贡赋和劳役，既是殖民中介也保留地方家族权势 |
| 教会网络 | 主教、奥古斯丁会、方济各会、耶稣会、多明我会等 | 经营堂区、学校、医院和地产；在许多地区掌握语言与基层信息 |
| 边疆 | presidio、传教站与军事远征 | 对苏丹国、高地和岛屿海域的控制常是断续的，名义主权大于实际行政 |

总督与代理任期的完整序列见[西班牙殖民菲律宾最高行政首脑表](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E4%B8%9C%E5%8D%97%E4%BA%9A/%E8%8F%B2%E5%BE%8B%E5%AE%BE/%E8%A5%BF%E7%8F%AD%E7%89%99%E6%AE%96%E6%B0%91%E6%9C%80%E9%AB%98%E8%A1%8C%E6%94%BF%E9%A6%96%E8%84%91%E8%A1%A8.md)。

## 贡赋、劳役、迁村与教会

西班牙早期用 encomienda 把特定地区贡赋授予征服者或机构，但理论上的受益权不等于土地所有权。殖民政府还征收贡赋，要求成年男性承担 polo y servicio 劳役，用于造船、道路、堡垒和远征。逃亡、隐匿户口、诉讼和起义都是地方社会回应负担的方式。

**reducción（迁村）** 把分散居民集中到以教堂、广场和镇政厅为核心的 poblacion，便于传教、征税和征发，却不可能在所有山区和岛屿彻底实行。洗礼、节庆、兄弟会和堂区生活塑造了新的低地天主教文化；祖灵信仰、地方仪式和语言传统则以改造、并存或地下延续等方式保留下来。

修会掌握大片地产和堂区任命，在缺乏足够西班牙官员的地区成为实际基层权力。菲律宾世俗神父要求接管堂区的“世俗化”运动，既涉及教会法和人事，也逐渐与殖民种族等级及政治代表问题结合。

## 马尼拉大帆船与经济转型

1565—1815年的马尼拉—阿卡普尔科航线把中国丝绸、瓷器和东南亚商品运往美洲，以白银支付。马尼拉华人商人和工匠不可或缺，却被集中于 Parián、课征特别税并多次遭驱逐或屠杀。殖民财政长期依靠墨西哥补贴，大帆船利润又偏向马尼拉少数商人，限制群岛内部生产的均衡发展。

18世纪末巴斯科改革设经济协会、发展烟草专卖与出口作物。1815年大帆船终止、1821年墨西哥独立后，马尼拉与伊洛伊洛等港口逐步加入全球糖、麻、烟草、咖啡贸易。华商— mestizo 中间群体、地主和受教育的 principalia 由此扩大。财富增长与土地集中同时发生，为19世纪知识精英和佃农冲突都提供了社会基础。

## 重要事件与具体过程

| 时间 | 事件 | 过程与影响 |
|---|---|---|
| 1565年 | 莱加斯皮建立宿务据点 | 以墨西哥为后方的持续殖民开始，并与部分米沙鄢首领结盟 |
| 1571年 | 西班牙占领马尼拉 | 击败或迫使马尼拉湾首领屈服，把天然良港建成首府 |
| 1574年 | 林凤袭击马尼拉 | 殖民者在本地盟军协助下守住首府，随后强化城防 |
| 1584年 | 马尼拉皇家审问院正式运作 | 殖民司法、咨询和职位空缺时代政制度成形 |
| 1603年 | 马尼拉华人起事与大规模杀戮 | 反映殖民者对华人经济依赖与安全恐惧并存；此后仍多次循环招徕与排斥 |
| 1610、1617年 | 普拉亚翁达海战 | 西班牙—菲律宾舰队阻止荷兰夺取马尼拉贸易中心 |
| 17世纪 | 对苏禄、马京达瑙的长期战争 | 西班牙建立若干要塞却未征服南部苏丹国，双方掠袭、俘虏和贸易交错 |
| 1762—1764年 | 英国占领马尼拉与甲米地 | 七年战争波及亚洲；西蒙·德安达在邦板牙维持西班牙抵抗，证明占领首都不等于控制全群岛 |
| 1781年前后 | 烟草专卖与经济改革 | 国家垄断增加财政收入，也以配额和强制收购压迫产区农民 |
| 1815年 | 大帆船贸易终止 | 跨太平洋垄断衰落，殖民经济转向区域和世界市场 |
| 1821年 | 新西班牙终结 | 菲律宾改由马德里直接处理，墨西哥财政和行政纽带中断 |
| 1834年以后 | 马尼拉及更多港口开放 | 出口农业、外商金融和华商网络增长，地方地主与 mestizo 精英壮大 |
| 1863年 | 教育法令 | 规划各镇男、女初等学校及师资培养；实施不均但扩大受教育群体 |
| 1869年 | 苏伊士运河通航 | 欧洲—马尼拉航程缩短，人员、报刊和自由主义思想流动加快 |
| 1872年 | 甲米地兵变与Gomburza被处决 | 小规模兵变被扩大解释，三位世俗神父之死成为民族主义象征 |
| 1880—1890年代 | 宣传运动 | 黎刹、德尔皮拉尔等 ilustrado 在菲律宾与西班牙要求代表、法治和教会改革 |
| 1892年 | 菲律宾联盟与卡蒂普南成立 | 改良组织遭镇压后，秘密革命组织转向武装独立 |
| 1896年 | 菲律宾革命爆发 | 卡蒂普南暴露后多地起兵；革命在甲米地壮大，也出现博尼法西奥与阿奎纳多派系冲突 |
| 1897年 | 特赫罗斯会议、博尼法西奥被处决、破石洞协议 | 革命领导权重组；阿奎纳多等流亡香港，但停战没有解决主权问题 |
| 1898年 | 美西战争、独立宣言与马尼拉投降 | 阿奎纳多回国恢复革命；6月12日宣告独立，8月美国占领马尼拉，12月西班牙在《巴黎条约》中把主权转让给美国 |

## 统治得以扩张和维持的原因

- **结构基础**：墨西哥提供船只、白银和人员；马尼拉良港连接中国商品与美洲市场。
- **地方合作**：殖民者吸收达图及其后裔进入 principalia，以税收豁免、职位和荣誉换取基层治理。
- **宗教与空间重组**：堂区、迁村、节庆和学校把居民纳入可见、可记录的共同体。
- **军事与外交**：火器、堡垒和跨洋海军与本地盟军结合；殖民者也利用地方政权之间的竞争。
- **有限覆盖**：统治集中于可航行河谷、低地和港口；对难以控制地区承认事实上的边界，降低了全面占领成本。

## 衰落与终结原因

### 结构性因素

- 官员短任、职位买卖和遥远决策造成腐败与政策反复；总督、审问院与修会之间又长期争权。
- 贡赋、劳役、土地集中和种族等级制造持续不满，修会地产与堂区人事尤其成为19世纪冲突焦点。
- 出口经济和教育培养了有财力、跨区域联系和共同政治语言的 ilustrado 与地方精英，他们要求平等公民权和代表权。
- 各地仍有不同诉求，革命内部也存在阶级、地区和领导权分歧；民族国家并非自然一次成形。

### 外部压力

- 拉丁美洲独立使帝国网络收缩，西班牙难以为遥远殖民地提供充分财政和军力。
- 苏伊士运河、报刊和留学加快自由主义、民族主义与反殖民信息传播。
- 1898年美国海军摧毁西班牙在马尼拉湾的舰队，使殖民政府失去海上与首都防务。

### 直接触发

1896年革命摧毁许多省份的日常行政，1897年停战仅使冲突暂歇。1898年美西战争让革命军重新扩张；马尼拉随后向美军而非革命军投降。西班牙并非把主权交给菲律宾共和国，而是在《巴黎条约》中转让给美国，殖民统治因此终结，却立即引出新的主权战争。
