# 美国统治与日本占领

## 时间

1898—1946年

## 概括

1898年《巴黎条约》使美国取得菲律宾。菲律宾第一共和国拒绝接受殖民主权，引发美菲战争。美国在军事镇压后建立文官政府，推广英语教育、公共卫生和有限选举，同时以土地、贸易和精英合作重塑殖民社会。1935年自治邦成立，原定十年后独立，却被日本占领打断。

## 统治结构

| 阶段 | 权力结构 | 说明 |
|---|---|---|
| 军事统治与美菲战争 | 美军政府 | 镇压共和国和地方游击队 |
| 美国文官政府 | 总督、菲律宾议会与地方精英 | 逐步“菲化”，但主权在美国 |
| 菲律宾自治邦 | 民选总统与美国高级专员 | 曼努埃尔·奎松任首任总统，准备独立 |
| 日本占领 | 日军与菲律宾第二共和国 | 劳雷尔政府名义独立，游击抵抗广泛存在 |

## 重要事件

- 1899—1902年美菲战争造成大量军民死亡，部分地区抵抗延续更久。
- 美国扩大公立学校和交通体系，但土地集中、佃农问题和对美经济依赖持续。
- 1916年《琼斯法》承诺未来独立；1934年《泰丁斯—麦克杜菲法》规定过渡安排。
- 1942年巴丹投降和死亡行军成为占领暴行象征。
- 1944—1945年盟军重返，马尼拉战役造成城市毁灭和大量平民死亡。

## 演变关系
