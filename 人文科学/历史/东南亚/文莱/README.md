# 文莱历史

文莱历史以婆罗洲西北岸的河口苏丹国为核心。15—16世纪文莱凭借海贸和伊斯兰王权扩大影响，后因内部争夺、西班牙和苏禄竞争及布鲁克家族扩张而收缩。1888年成为英国保护国，石油开发改变财政基础，1984年取得完全独立。

## 重要转折

| 时间 | 事件 | 意义 |
|---|---|---|
| 15—16世纪 | 苏丹国扩张 | 影响婆罗洲沿岸与菲律宾南部 |
| 1660年代 | 文莱内战 | 王权分裂并加速外部势力介入 |
| 1888年 | 成为英国保护国 | 外交受英国控制 |

| 1984年 | 完全独立 | 建立主权国家 |
