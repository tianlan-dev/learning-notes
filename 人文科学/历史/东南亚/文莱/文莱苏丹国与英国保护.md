# 文莱苏丹国与英国保护

## 时间

14世纪—1959年

## 概括

文莱河口聚落很早参与南海贸易，统治家族在马来—伊斯兰网络中采用苏丹称号。15—16世纪文莱影响婆罗洲北岸、苏禄与马尼拉湾部分港口，但这种影响多通过贡赋、亲族和贸易联系维持。17世纪后内部战争及外来势力使疆域不断收缩。

苏丹世系在保护国时期保持连续，但外交和部分行政权先后受英国政府与驻扎官制约；独立后苏丹同时处于君主、宗教领袖和行政权力核心的位置。

## 建立背景与崛起

文莱河口把婆罗洲内陆森林物产、金属与海产接入南海航线。14—15世纪统治者在马来商贸世界中采用伊斯兰苏丹称号，并借王族婚姻、港口首领和河流属地扩张。所谓“文莱帝国”更接近以贡赋、亲族与贸易节点连接的海上宗主网络，不是对婆罗洲和菲律宾沿海实行均质行政的领土国家。

## 统治结构与鼎盛条件

- 苏丹居于河口宫廷，四大维齐尔、王族贵族与地方首领管理不同河流和属民；王权依赖对税源、贸易和封地的协商。
- 15—16世纪的优势来自文莱河安全港、婆罗洲物产、马来语—伊斯兰商贸网络以及与中国、马六甲和菲律宾港口的外交。
- 外围港口承认文莱并不等于常驻官僚直接统治；地方首领在贸易转向或王室内战时很容易脱离。

## 重要事件

- 文莱与马六甲、爪哇、中国和菲律宾群岛保持贸易与外交联系。
- 1578年西班牙从马尼拉进攻文莱，短暂占领首都后撤退。
- 1841年苏丹承认詹姆斯·布鲁克在砂拉越的地位，此后割地扩大。
- 1888年保护条约未能立即阻止领土流失；1906年驻扎官制度强化英国控制。
- 1929年诗里亚油田投产后，石油成为财政支柱。
- 1941—1945年日本占领，油田成为重要军事资源。

## 衰落、保护与转型机制

- **结构因素**：封地和河流属地分散，王族内部对税源与继承的竞争使中央难以长期控制外围。
- **外部压力**：西班牙、苏禄、布鲁克家族和北婆罗洲公司分别争夺港口、土地与贸易；蒸汽航运和殖民边界削弱旧贡赋网络。
- **直接转折**：1661—1673年内战重创王权；1841年后砂拉越连续扩张，1888年保护条约仍未阻止林梦于1890年并入砂拉越。
- **保护国实质**：1906年驻扎官掌握除伊斯兰与王室习俗外的主要行政建议权，苏丹国保留法统却失去多数日常主权。
- **再生条件**：1929年石油发现提供新财政基础；1959年宪法恢复内部自治并规范继承，为后来的完全独立创造制度条件。
