# 东南亚历史

## 概括

东南亚位于中国、印度、印度洋和太平洋之间，可分为中南半岛和海岛东南亚两大历史空间。它的主线包括印度化国家、海上贸易、佛教与印度教传播、伊斯兰化、华人移民、欧洲殖民、二战与现代民族国家形成。
