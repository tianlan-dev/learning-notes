# 东南亚殖民、战争、独立与东盟

## 时间

16世纪至今，重点为19世纪殖民扩张、1941-1945年日本占领和战后独立。

## 概括

葡萄牙、西班牙、荷兰、英国、法国和美国在东南亚建立不同殖民体系；暹罗通过改革、外交与让地维持形式独立。日本占领摧毁欧洲殖民权威，也带来强制劳动、饥荒和战争暴力。战后印度尼西亚、越南等通过战争争取独立，马来亚、菲律宾、缅甸等沿不同路径建国。冷战把地区卷入印度支那战争、反共镇压和军政冲突；1967年东盟成立后逐步扩大区域合作。

## 比较

| 地区 | 殖民 / 战争路径 |
|---|---|
| 法属印度支那 | 越南、柬埔寨、老挝被纳入法属体系，战后独立战争与冷战高度重叠。 |
| 英属东南亚 | 缅甸、马来亚、新加坡、北婆罗洲等制度不同，独立与联邦形成路径不一。 |
| 荷属东印度 | 公司统治转为殖民国家，1945年宣布独立后经历四年战争。 |
| 菲律宾 | 西班牙殖民后转入美国统治，1946年独立。 |
| 葡属帝汶 | 1975年短暂宣布独立后被印度尼西亚占领，2002年恢复独立。 |
| 暹罗 / 泰国 | 未被正式殖民，但通过不平等条约、边界让步和国家改革应对帝国压力。 |

## 重要节点

- 1941-1945年日本占领重组行政与民族主义组织，也造成严重剥削与暴力。
- 1945年越南、印度尼西亚宣布独立，殖民复归尝试引发战争。
- 1967年印度尼西亚、马来西亚、菲律宾、新加坡、泰国签署《曼谷宣言》，建立东盟。
- 文莱、越南、老挝、缅甸和柬埔寨后来加入，东盟覆盖东南亚十国；东帝汶的加入进程另有时间线。
- “不干涉”与协商原则支持成员合作，也限制区域组织处理人权与国内冲突的能力。
