# 东南亚贸易、宗教与移民网络

## 时间

古代至今。

## 概括

季风航行、河谷稻作、山地物产和海峡港市使东南亚长期成为跨区域交换中心。佛教、印度教、儒学、伊斯兰教和基督教通过商人、僧侣、宫廷、婚姻、教育与殖民制度传播，并被地方社会重新解释。华人、印度人、阿拉伯人与区域内部移民推动港口、矿业、种植园和城市发展，也形成复杂的族群与公民政治。

## 主要网络

| 网络 | 作用 |
|---|---|
| 季风与海峡贸易 | 连接印度洋、南海、爪哇海和太平洋，形成马六甲、占婆、巨港等港市。 |
| 佛教与印度教 | 影响王权、文字、寺庙和仪式；本地传统始终参与重构。 |
| 伊斯兰化 | 主要通过港口贸易、苏菲网络、婚姻和苏丹国扩展，时间与地区差异显著。 |
| 华人移民 | 参与港口、矿业、税收承包、零售和现代工业，经历同化、自治与排斥政策。 |
| 殖民劳工 | 印度、华南、爪哇等地劳工被招募至种植园、矿区和城市。 |

## 说明

- 宗教传播不等于原有信仰消失；祖先、地方神灵和习惯法常与世界宗教并存。
- 港口国家与内陆农业国家并非截然分离，贡品、森林产品、稻米和人口流动把两者连接。
- 移民身份在殖民人口分类、独立后的公民制度和民族主义中被不断重塑。
