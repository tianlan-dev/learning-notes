# 越南历史

越南历史的主线，是红河三角洲的早期国家在中国帝国长期统治与交流中重组，10世纪建立持续自主的王朝国家，随后通过“南进”形成今日南北疆域；19—20世纪又经历法国殖民、日本占领、冷战分裂和长期战争，最终建立统一的社会主义共和国并转向革新开放。

## 历史主线

- 红河三角洲的东山文化与古螺城证明汉征服前已有复杂社会；文郎和雄王的极早年代仍属传统记忆与争议谱系。
- 前111年后，今越南北部长期处于郡县、州和都护府体系，但征氏姐妹、万春国及唐末地方军镇多次打断直接控制。
- 938年吴权击败南汉后，李、陈、后黎等王朝形成官僚国家；对宋、元、明既有战争，也有册封、朝贡和制度交流。
- 从李陈边境推进到后黎、阮主扩张，“南进”是战争、移民、贸易和边疆治理的累积过程，改变占婆与高棉地区的政治空间。
- 1802年阮朝完成统一；法国逐段夺取三圻，殖民统治又催生士绅、民族主义、共产主义和群众政治。
- 1945年独立后仍经历两次印度支那战争和南北分裂；1976年完成国家合并，1986年以后在一党政治延续下推进市场化与国际整合。

## 重要转折与时间节点

| 时间 | 事件 | 意义 |
|---|---|---|
| 前111年 | 汉灭南越 | 红河地区进入长期帝国郡县框架。 |
| 40—43年 | 征氏姐妹起兵 | 地方政治与反抗传统的重要象征。 |
| 544—602年 | 万春国 | 首次较长期以本地帝号和国号挑战北属秩序。 |
| 938年 | 白藤江之战 | 吴权击败南汉，开启持续自主时代。 |
| 1010年 | 李朝迁都升龙 | 红河王朝国家的政治中心趋于稳定。 |
| 1258—1288年 | 三次抗蒙古—元战争 | 陈朝以宗室、军队和村社动员守住独立。 |
| 1407、1427年 | 明属与蓝山复国 | 外来直接统治再现，后黎通过战争恢复国家。 |
| 1471年 | 后黎攻破占城毗阇耶 | 南进进入决定性阶段。 |
| 1527—1677年 | 黎莫、郑阮并立 | 名义王统和实际军政权长期分离。 |
| 1802年 | 阮朝统一 | 建立最后王朝，统合北中南。 |
| 1858—1887年 | 法国征服与印度支那联邦建立 | 越南三圻被纳入不同殖民法律结构。 |
| 1945年 | 日本政变、八月革命与独立宣言 | 阮朝和旧殖民总督体系终结，现代国家竞争开始。 |
| 1954年 | 奠边府与日内瓦停战 | 法国战争失败，南北进入临时分治。 |
| 1975—1976年 | 战争结束与国家合并 | 越南社会主义共和国成立。 |
| 1986年 | 革新开放 | 由高度计划体制转向社会主义定向市场经济。 |
| 1995年 | 加入东盟、越美建交 | 战后国际孤立基本结束。 |
| 2026年 | 十四大与国家换届 | 苏林连任总书记并兼国家主席，黎明兴任总理。 |
