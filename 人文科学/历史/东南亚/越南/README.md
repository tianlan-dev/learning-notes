# 越南历史

越南历史的主线，是红河三角洲国家在中国帝制影响下形成，又逐步取得自主、持续向南扩张，最后在殖民统治、冷战分裂与战争中建立统一共和国。

    A["文郎、瓯雒与南越"] --> B["北属时期与地方反抗"]
    B --> C["吴、丁、前黎与李陈王朝"]
    C --> D["后黎、莫氏与郑阮分治"]
    D --> E["西山与阮朝统一"]
    E --> F["法属印度支那"]

    G --> H["越南社会主义共和国"]

## 历史主线

- 红河三角洲是早期政治中心，东山文化、瓯雒等构成越南早期国家记忆。
- 前111年后，今越南北部长期处于汉至唐帝国的郡县体系之内，但地方社会持续保有自身传统。
- 10世纪取得稳定独立后，李、陈、后黎等王朝建立中央集权国家，并通过“南进”逐步吸收占城和湄公河三角洲地区。
- 19世纪法国将越南纳入法属印度支那；20世纪的民族主义、共产主义和冷战竞争，最终导向1975年的统一。

## 重要转折

| 时间 | 事件 | 意义 |
|---|---|---|
| 938年 | 白藤江之战 | 吴权击败南汉，开启长期自主时代 |
| 1010年 | 李朝定都升龙 | 红河三角洲王朝国家制度趋于稳定 |
| 1471年 | 后黎大破占城 | 南进进入决定性阶段 |
| 1887年 | 法属印度支那建立 | 越南三圻被纳入殖民联邦 |
| 1945年 | 八月革命与独立宣言 | 越南民主共和国成立 |
| 1975—1976年 | 战争结束并完成国家统一 | 南北政治合并为越南社会主义共和国 |
