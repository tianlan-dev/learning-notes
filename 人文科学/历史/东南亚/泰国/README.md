# 泰国历史

泰国历史由湄南河流域的孟人、泰人和高棉文化互动起步，经素可泰、阿瑜陀耶、吞武里和曼谷王朝形成暹罗国家。19世纪王室改革使暹罗在英法殖民扩张之间维持形式独立，20世纪则反复经历立宪、军人政治与社会动员。

## 重要转折

| 时间 | 事件 | 意义 |
|---|---|---|
| 1351年 | 阿瑜陀耶建立 | 湄南河下游形成长期政治中心 |
| 1767年 | 阿瑜陀耶被缅军攻陷 | 旧都时代终结，吞武里复国 |
| 1782年 | 曼谷王朝建立 | 曼谷成为首都，王朝延续至今 |
| 1855年 | 《鲍林条约》 | 暹罗更深卷入世界市场与不平等条约体系 |
| 1932年 | 立宪革命 | 绝对君主制终结 |
| 1939年 | 国名改为泰国 | 现代民族国家认同进一步制度化 |
