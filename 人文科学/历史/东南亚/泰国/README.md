# 泰国历史

泰国历史的主线不是一个民族或王朝自古连续扩张，而是湄南河、湄公河与北方盆地间的孟人城市、高棉网络、泰人勐邦和跨海贸易不断重组。素可泰、兰纳与阿瑜陀耶奠定泰语国家和上座部佛教传统，吞武里完成复国，1782年建立的却克里王朝则以曼谷为中心延续至今。

19世纪暹罗在英法殖民扩张之间通过领土让步、条约和中央集权改革保存核心独立。1932年后王朝转为立宪王室，军队、官僚、王室网络、法院、政党和社会运动持续争夺政治主导。截至2026年7月，国王为拉玛十世玛哈·哇集拉隆功，总理为阿努廷·参威拉军。

## 历史主线

- **多中心前史**：陀罗钵地是城市和文化网络，不宜虚构统一王统；素可泰与兰纳是并行发展的泰人政权。
- **阿瑜陀耶整合**：河网、稻米、港口贸易和萨克迪纳制度支撑四百余年王朝，期间经历1569年陷落、纳黎萱复国和17世纪国际贸易高峰。
- **复国与曼谷建都**：达信在1767年后迅速恢复国家；1782年却克里王朝建立并吸收阿瑜陀耶制度。
- **近代国家形成**：拉玛四世、五世时期以条约、外交平衡和省制改革应对殖民压力，代价包括主权受限、属地让渡与地方自主削弱。
- **立宪而非改朝换代**：1932年终结绝对君主制，却克里王朝继续作为立宪王室；总理、军政府与国王须分别维护。
- **竞争性政治与体制外否决**：选举、大众福利和社会运动扩大参与，军方政变、法院裁决、政党解散及任命机构又反复改变政府。

## 重要转折与时间节点

| 时间 | 事件 | 意义 |
|---|---|---|
| 约1238年 | 素可泰王权形成 | 泰人勐邦在上游盆地建立区域政权；具体年代和“第一王国”叙事须谨慎。 |
| 1351年 | 阿瑜陀耶建立 | 湄南河下游形成长期政治与商业中心。 |
| 1569、1584年 | 阿瑜陀耶陷落与纳黎萱宣布独立 | 东吁征服后重新取得自主，军事国家重建。 |
| 1767年 | 阿瑜陀耶被贡榜军攻陷 | 旧都时代终结，人口和工匠遭迁移，吞武里随即复国。 |
| 1782年 | 曼谷却克里王朝建立 | 曼谷成为首都，王朝延续至今。 |
| 1855年 | 《鲍林条约》 | 暹罗深度进入世界市场及不平等条约体系。 |
| 1893—1909年 | 殖民危机与边界条约 | 让渡老挝、柬埔寨及北马来属地，现代边界逐步定型。 |
| 1932年 | 人民党立宪革命 | 绝对君主制终结，国王、总理与议会进入新的宪制关系。 |
| 1957—1958年 | 沙立两次政变 | 冷战军人发展政权与王室—军方联盟形成。 |
| 1973、1976年 | 民主起义、惨案与政变 | 群众政治开放后遭暴力反扑，成为现代政治记忆核心。 |
| 1992年 | “黑色五月” | 军人直接担任总理的正当性受到重创，议会政治恢复。 |
| 1997年 | 金融危机与新宪法 | 经济结构重组，强政党与独立机构并行的制度成形。 |
| 2006、2014年 | 两次军事政变 | 他信政治与旧权力网络的冲突长期化，并产生2017年宪制。 |
| 2023年 | 前进党赢得最多席位但未组阁 | 显示选举授权与参议院、法院等制度门槛之间的张力。 |
| 2025—2026年 | 佩通坦被免、阿努廷组阁并赢得新授权 | 边境危机、法院裁决与联盟重组共同塑造新政府。 |
