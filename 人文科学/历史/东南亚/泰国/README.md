# 泰国

## 概括

泰国历史可按素可泰、阿瑜陀耶、吞武里、曼谷王朝和现代泰国整理，同时注意与高棉、缅甸、老挝和殖民时代边界的关系。

## 整理建议

- 先建立通史 README，再按主要王朝、殖民阶段或现代国家阶段拆分笔记。
- 注意区分文化影响、族群迁徙、宗教传播和现代国界，不把现代国家边界机械套用到古代阶段。
