# 阿瑜陀耶王朝

## 时间

1351—1767年

## 概括

阿瑜陀耶位于湄南河水系与暹罗湾贸易路线的交汇处，兼具内陆稻米国家和国际港市的性质。王朝通过战争、婚姻、属国体系和人口迁移整合湄南河流域，并与明清中国、琉球、波斯、印度、日本及欧洲商人往来。

## 王朝阶段

| 阶段 | 时间 | 代表统治者 | 特征 |
|---|---|---|---|
| 建国与扩张 | 1351—1569年 | 乌通王、波隆摩·戴莱洛迦纳 | 整合素可泰，发展官阶与属地制度 |
| 缅甸控制与复国 | 1569—1605年 | 纳黎萱 | 摆脱东吁控制并恢复扩张 |
| 商贸与宫廷政治 | 17世纪 | 那莱王等 | 国际贸易兴盛，外来社群活跃 |
| 晚期与灭亡 | 18世纪—1767年 | 波隆摩阁、厄伽陀 | 继承危机、地方压力与缅甸战争 |

## 分期过程

### 1351—1569年：河口王国扩张与制度整合

乌通王在湄南河、华富里河和巴萨河环绕的岛城建都，既能利用稻米腹地，又可经河网进入暹罗湾。早期王位在乌通系与素攀武里系之间反复转换，显示国家仍依赖王族领地与属城联盟。波隆摩罗阇二世向高棉地区扩张，波隆摩·戴莱洛迦纳又把素可泰王族与北方属地纳入阿瑜陀耶体系，区分军政与民政大臣，并以萨克迪纳等级把官位、土地收益和受役人口联系起来。制度整合提高动员能力，却没有消除兄弟、叔侄与外戚争位。

16世纪初葡萄牙商人和雇佣兵进入，火器、海贸与外国社群的重要性增加。东吁缅甸统一后向东扩张，1540年代起连续战争消耗暹罗。1569年都城在内部倒戈和长期围攻后陷落，摩诃·探玛罗阇被扶立，王朝进入缅甸宗主权阶段。

### 1569—1605年：附属、独立与纳黎萱复国

摩诃·探玛罗阇在缅甸支持下重建行政，其子纳黎萱以彭世洛为基地掌军。1584年东吁内部危机时，纳黎萱宣布不再臣属，随后迁移边境人口、抵御多次进攻并恢复属城。1593年农沙莱战役在泰国传统中以纳黎萱象战闻名，象战细节与具体地点存在史料争议，但这场战争确立了阿瑜陀耶摆脱东吁控制。复国后的军事扩张一度伸向德林达依、兰纳和柬埔寨。

### 1605—1688年：国际商贸与宫廷重组

17世纪阿瑜陀耶成为连接中国、日本、波斯、印度、马来世界与欧洲公司的国际港市。宫廷以港税、王室专卖和外商技术获利，并让华人、波斯人、日本人、葡萄牙人和孟人等群体承担贸易、军事与官僚职能。1629年巴沙通借宫廷军权夺位，说明商业繁荣与继承制度不稳定可以同时存在。

那莱王利用荷兰、英国与法国相互制衡，并让希腊裔官员康斯坦丁·华尔康管理部分外交和贸易。法国使团与驻军引发王族、僧侣和官僚对外来势力及华尔康集权的不安。1688年帕碧罗阇政变处死华尔康、排除那莱指定继承安排并限制法国驻军；但王朝并未因此全面“闭关”，对华和亚洲贸易继续扩大。

### 1688—1767年：晚期繁荣、继承危机与缅甸征服

18世纪前半叶稻米、林产和对华贸易仍带来繁荣，波隆摩阁时期寺院、文学与宫廷文化发展。问题在于中央军役体系逐渐松弛，贵族与地方首领掌握人口，王室又缺乏固定继承规则。1758年波隆摩阁死后，乌通蓬与厄伽陀之间的继承争议削弱统一指挥。

1759—1760年阿隆帕耶率贡榜军入侵未能攻下都城，但证明阿瑜陀耶边防脆弱。1765年缅军从北、西两路推进，先摧毁外围救援和粮源，再围困都城十余月。1767年4月城破，厄伽陀死亡，王族、工匠与人口被迁走；地方军队仍存，为达信迅速复国提供了基础。

## 建立背景、崛起与统治结构

乌通王在湄南河下游商业与稻作交汇处建都，吸收罗斛、素攀武里、素可泰和高棉政治传统。王朝以“曼荼罗”式属城、王族总督、萨克迪纳官阶与徭役人口组织扩张；港税、稻米和转口贸易使宫廷能维持军队与外交。王位没有稳定长子制，兄弟、王子、外戚和掌军大臣均可争位，因此复辟、弑君和短期君主必须保留在世系中。

## 国王完整世系

王系名称采用常见的乌通、素攀武里、素可泰、巴沙通和班普銮五系划分；早期婚姻与父系归属存在争议，因此“王系”主要用于说明宫廷集团和继承联盟，不等于近代固定姓氏。泰国常用官方口径为33位国王；下表按实际在位先后拆成37个统治期，因为拉梅萱、摩诃·查克拉帕和玛欣的复位分别成行，并保留通常不计入33位正式国王的短期夺位者沃拉翁萨。

## 鼎盛条件

- 河网与稻米剩余支持人口集中，港口又能连接中国、日本、印度洋和欧洲商人。
- 吸收孟、高棉、马来、华人、波斯和日本等群体的军政与商业技能，形成多族群都城。
- 15世纪制度改革把官阶、土地收益与劳役人口相连；纳黎萱复国和那莱外交分别代表军事与商贸两种高峰。

## 重要事件

| 时间 | 事件 | 过程与影响 |
|---|---|---|
| 1351年 | 乌通王建立阿瑜陀耶 | 利用河岛防御、稻米腹地和海贸路线建立新都，逐步压过罗斛、素攀武里等旧中心。 |
| 1431年前后 | 进攻吴哥 | 阿瑜陀耶军队攻入吴哥地区并迁移人口、工匠；高棉政治中心此后逐渐南移，但具体战事与影响范围仍有史料争议。 |
| 1448—1488年 | 波隆摩·戴莱洛迦纳改革 | 整合素可泰遗产，发展官阶、军民大臣和萨克迪纳制度，提高中央动员。 |
| 1511年起 | 葡萄牙等欧洲势力进入 | 火器、雇佣兵和欧洲贸易公司加入原有亚洲贸易网络。 |
| 1548年 | 与东吁战争、素里约泰王后战死传统 | 缅甸扩张进入湄南河流域；王后骑象救驾的细节主要来自后世编年，但成为重要历史记忆。 |
| 1569年 | 阿瑜陀耶第一次陷落 | 东吁军长期围攻并利用内部倒戈攻城，扶立摩诃·探玛罗阇，暹罗一度成为缅甸属国。 |
| 1584年 | 纳黎萱宣布独立 | 利用东吁继承危机脱离宗主权，重组人口与军队。 |
| 1593年 | 农沙莱战役 | 阿瑜陀耶击退缅军；传统象战叙事的具体细节存在争议，但复国地位由此巩固。 |
| 1629年 | 巴沙通夺位 | 权臣利用宫廷军队清除幼主，建立新王系，显示王位继承缺乏固定规则。 |
| 1685—1688年 | 法国外交、驻军与宫廷政变 | 那莱借法国制衡荷兰，华尔康权势和外国驻军激起反弹；帕碧罗阇夺位并驱逐法国力量。 |
| 1759—1760年 | 阿隆帕耶入侵 | 缅军逼近都城后因统帅重伤、死亡而撤退，阿瑜陀耶获得短暂喘息。 |
| 1765—1767年 | 贡榜缅甸围攻与王朝灭亡 | 缅军切断属城与粮道，十余月后攻破都城；中央王权终结，但地方网络很快由达信重新整合。 |

## 衰落与灭亡原因

- **结构因素**：王位继承竞争频繁，地方总督和贵族掌握兵员；波隆摩阁死后王室内斗进一步削弱协调。
- **外部压力**：贡榜缅甸完成内部统一后可集中更多兵力，暹罗则同时面对边境防务与属城离心。
- **军事过程**：1765年缅军由北、西多路推进，长期围困切断粮道；1767年4月攻破都城，迁走王族、工匠和人口。
- **直接终结**：厄伽陀在城破前后死亡，中央指挥消失。灭亡不是“闭关”或单一昏君造成，而是继承、动员与强敌攻势叠加。
- 阿瑜陀耶国家网络未随都城焚毁而消失；达信利用东部军队、华商资源和旧官僚，在数月内开始重建。
