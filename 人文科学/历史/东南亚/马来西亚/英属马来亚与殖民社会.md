# 英属马来亚与殖民社会

## 时间

1786—1957年

## 概括

英国先取得槟城、新加坡和马六甲，组成海峡殖民地；随后通过驻扎官制度控制半岛苏丹国，并在婆罗洲北部形成砂拉越布鲁克王朝、北婆罗洲公司和文莱保护国等不同体制。锡矿、橡胶和港口资本吸引大量华人、印度人移民，形成高度多族群的殖民社会。

## 统治结构

| 区域 | 殖民形式 | 说明 |
|---|---|---|
| 海峡殖民地 | 英国直辖殖民地 | 槟城、马六甲、新加坡由英国官员直接治理 |
| 马来联邦州 | 驻扎官制度 | 苏丹保留宗教与习俗地位，英国控制行政财政 |
| 马来属邦 | 顾问制度 | 形式自主较多，实际受英国影响 |
| 砂拉越 | 布鲁克家族“白人拉惹” | 1841—1946年家族王朝，后成为英国殖民地 |
| 北婆罗洲 | 特许公司，后英国殖民地 | 公司开发与地方行政结合 |

## 重要事件

- 1786年英国东印度公司取得槟城；1819年莱佛士建立新加坡贸易站。
- 1824年英荷条约划分势力范围，英国集中经营半岛，荷兰集中经营群岛。
- 1874年邦咯条约推动驻扎官制度扩展。
- 二战日本占领打破英国威望，也加剧族群之间不同的战争经验。
- 1946年马来亚联邦方案因削弱苏丹和放宽公民权而遭马来民族主义反对。
- 1948年马来亚联合邦成立；同年共产党领导的武装斗争促成紧急状态。
