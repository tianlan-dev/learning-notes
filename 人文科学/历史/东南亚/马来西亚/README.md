# 马来西亚历史

马来西亚历史跨越马来半岛与婆罗洲北部。马六甲海峡港市、马来苏丹国和伊斯兰网络塑造共同文化，葡萄牙、荷兰和英国的商业竞争又把各地纳入不同殖民体系。二战后，马来亚于1957年独立，并在1963年与沙巴、砂拉越、新加坡组成马来西亚；新加坡两年后退出。现代联邦结合九州世袭王室、轮任最高元首、议会内阁与东马特殊自治安排。

## 历史主线

- 海峡航运和港市竞争先后推动吉打、马六甲、柔佛—廖内及多个并立苏丹国发展，马来语、伊斯兰和王权礼仪由此扩散。
- 英国统治不是一个统一殖民地：直辖港口、间接统治的苏丹国、砂拉越布鲁克王朝和北婆罗洲公司长期并存。
- 锡、橡胶、港口和移民劳工塑造多族群社会；日本占领和战后宪制冲突促使民族政治与反殖民进程加速。
- 1957年半岛独立与1963年婆罗洲殖民地加入是两个相连而不同的步骤；沙巴、砂拉越的特殊保障仍是联邦政治核心。
- 1969年危机后，新经济政策和国民阵线体制重塑资源分配；2018年后，联邦进入政党轮替和多联盟政府阶段。

## 重要转折与时间节点

| 时间 | 事件 | 意义 |
| --- | --- | --- |
| 约1400年 | 马六甲建立 | 马来—伊斯兰港市秩序形成 |
| 1511年 | 葡萄牙占领马六甲 | 王统流亡并分支延续，海峡贸易中心分散 |
| 1824年 | 英荷条约 | 马来世界被划入英国与荷兰势力范围 |
| 1874年 | 邦咯条约 | 英国驻扎官制度开始系统扩展 |
| 1941—1945年 | 日本占领 | 英国威望崩塌，镇压、短缺和抵抗重塑民族政治 |
| 1948—1960年 | 马来亚紧急状态 | 反共战争、人口重组与自治谈判并行 |
| 1957年 | 马来亚联合邦独立 | 君主立宪联邦建立 |
| 1963年 | 马来西亚成立 | 马来亚、沙巴、砂拉越与新加坡组成新联邦 |
| 1965年 | 新加坡退出 | 联邦领土与政治结构定型 |
| 1969—1971年 | 族群危机与新经济政策 | 重构族群分配、发展政策和执政联盟 |
| 1997—1998年 | 亚洲金融危机与改革运动 | 经济治理和政治竞争进入新阶段 |
| 2018年 | 首次联邦政党轮替 | 国民阵线首次失去中央政权 |
| 2020—2022年 | 联盟重组与悬峙国会 | 王室裁量、议会多数和跨联盟政府作用上升 |
