# 阿尤布王朝统治下的埃及

## 时间

1171—1250年；1250—1254年仍有名义上的阿尤布共治者，但埃及实权已归马穆鲁克集团。

## 概括

萨拉丁以埃及的财政和军队为基础，终结法蒂玛哈里发并统一埃及、叙利亚和黎凡特多地。阿尤布国家是家族成员分掌诸城和领地的政治网络，并非固定边界下的单一中央王朝。对十字军战争、家族再分配和军人奴隶部队扩张共同塑造其兴衰；1250年突兰沙被杀后，埃及的阿尤布实权转入马穆鲁克之手。

## 统治结构

| 层面 | 说明 |
|---|---|
| 埃及苏丹 | 控制开罗、尼罗河财政、军队和最重要的王朝合法性资源。 |
| 家族分封 | 萨拉丁兄弟、子侄分掌大马士革、阿勒颇、哈马、外约旦等地，既协作又争夺宗主地位。 |
| 伊克塔与军队 | 军官以税收收益维持骑兵；库尔德、突厥及其他军事集团并存，后期马穆鲁克军团尤为关键。 |
| 宗教与司法 | 承认阿拔斯哈里发名义，扩建逊尼派经学院和四法学派司法，同时保留科普特等社群。 |
| 城市与贸易 | 开罗城堡强化军政中心；亚历山大里亚和红海港口连接地中海与印度洋。 |

## 埃及统治者世系

| 顺序 | 统治者 | 在位 | 继承关系与关键事件 |
|---:|---|---|---|
| 1 | **萨拉丁（萨拉赫丁·优素福）** | 1171—1193 | 纳吉姆丁·阿尤布之子；终结法蒂玛统治，1187年哈丁战役后收复耶路撒冷。 |
| 2 | 阿齐兹·奥斯曼 | 1193—1198 | 萨拉丁之子；统治埃及，与叔父阿迪勒及兄弟支系竞争。 |
| 3 | 曼苏尔·穆罕默德 | 1198—1200 | 阿齐兹幼子；由权臣和家族长辈争夺摄政，后被废。 |
| 4 | **阿迪勒一世** | 1200—1218 | 萨拉丁之弟；重新整合主要领地，以外交和贸易维持停战。 |
| 5 | 卡米勒 | 1218—1238 | 阿迪勒之子；抗击第五次十字军，后以条约暂时交出耶路撒冷。 |
| 6 | 阿迪勒二世 | 1238—1240 | 卡米勒之子；军政支持薄弱，被兄长萨利赫废黜。 |
| 7 | 萨利赫·阿尤布 | 1240—1249 | 卡米勒之子；重建埃及权力，扩充巴赫里马穆鲁克。 |
| — | 沙杰尔·杜尔 | 1249—1250 | 萨利赫遗孀；先隐瞒其死讯并摄政，突兰沙死后短期以女苏丹身份执政。 |
| 8 | 穆阿扎姆·突兰沙 | 1249/1250—1250 | 萨利赫之子；在第七次十字军战争期间抵埃及，后被马穆鲁克杀死。 |
| — | 阿什拉夫·穆萨 | 1250—1254 | 阿尤布幼年旁支，被艾贝克立为名义共治者；无独立实权，不能视为阿尤布恢复埃及统治。 |

## 发展过程与重要事件

| 时间 | 事件 | 影响 |
|---|---|---|
| 1171年 | 萨拉丁停止为法蒂玛哈里发祈祷 | 埃及转奉阿拔斯名义，法蒂玛国家终结。 |
| 1174—1186年 | 萨拉丁整合叙利亚诸城 | 埃及财政与叙利亚战略纵深结合。 |
| 1187年 | 哈丁战役、耶路撒冷被收复 | 阿尤布王朝声望达到高峰，引发第三次十字军东征。 |
| 1192年 | 与理查一世达成雅法和约 | 沿海部分留在十字军手中，耶路撒冷由阿尤布控制。 |
| 1193年 | 萨拉丁去世，领地在家族中分配 | 支系竞争加剧，阿迪勒最终重新统一核心。 |
| 1218—1221年 | 第五次十字军占领达米埃塔后失败 | 埃及水系、防御与谈判共同挫败入侵。 |
| 1229年 | 卡米勒与腓特烈二世订约 | 耶路撒冷一度以谈判交出，显示家族竞争和外交优先。 |
| 1244年 | 花剌子模军协助阿尤布夺回耶路撒冷 | 引发新的地区战争与第七次十字军。 |
| 1249—1250年 | 路易九世入侵、曼苏拉战役 | 十字军被击败，萨利赫之死使继承危机爆发。 |
| 1250年 | 突兰沙被杀，沙杰尔·杜尔和艾贝克掌权 | 埃及阿尤布实权终结，马穆鲁克国家形成。 |

## 崛起、鼎盛与衰落

王朝崛起依靠埃及可持续税收、萨拉丁在法蒂玛军政体系中的职位、赞吉王朝遗产和反十字军合法性。鼎盛并非全面中央集权，而是萨拉丁个人威望和家族分封合作。结构性弱点是继承时领地重新分配、支系以十字军或外来军队为盟友、埃及苏丹日益依赖由自身购买和训练的马穆鲁克。外部压力来自连续十字军战争与叙利亚竞争；直接触发是萨利赫死后突兰沙试图排挤其父旧部，巴赫里马穆鲁克在曼苏拉胜利后掌握军力并将其杀死。

阿尤布支系此后仍在叙利亚、哈马和其他地区延续，因而“1250年灭亡”只指埃及最高统治转换。开罗城堡、逊尼派经学院、伊克塔和军事奴隶制度都被马穆鲁克继承并扩大。
