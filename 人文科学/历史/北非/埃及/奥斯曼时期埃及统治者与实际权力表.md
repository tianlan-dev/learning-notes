# 奥斯曼时期埃及统治者与实际权力表

## 时间

1517—1914年

## 使用说明

埃及在1517—1805年主要作为奥斯曼行省运行；1805年以后穆罕默德·阿里家族取得世袭化本地统治，奥斯曼苏丹的名义宗主权仍延续到1914年。本表因此分为两部分：

1. 完整列出1517年征服至1914年法理关系终止期间的奥斯曼苏丹，复位另列；
2. 按阶段列出埃及本地实际权力中心。奥斯曼总督多为短任行政官，不把数百名代理、临时或重复总督误写成世袭王朝。

## 奥斯曼苏丹完整序列

| 顺序 | 苏丹 | 在位及与埃及关系 | 关键说明 |
|---:|---|---|---|
| 1 | **塞利姆一世** | 1512—1520；埃及自1517年 | 击败高里和图曼贝伊二世，将埃及、叙利亚和汉志宗教资源纳入帝国。 |
| 2 | **苏莱曼一世** | 1520—1566 | 镇压艾哈迈德帕夏叛乱，1525年规制行省；整合红海、朝觐和税务。 |
| 3 | 塞利姆二世 | 1566—1574 | 延续总督—底万—军团治理，地中海战争增加财政需求。 |
| 4 | 穆拉德三世 | 1574—1595 | 帝国战争与通货压力影响埃及驻军和税收，1580年代军团冲突加剧。 |
| 5 | 穆罕默德三世 | 1595—1603 | 驻军、贝伊与总督权力继续拉锯。 |
| 6 | 艾哈迈德一世 | 1603—1617 | 埃及军团哗变与财政整顿并存。 |
| 7 | 穆斯塔法一世（第一次） | 1617—1618 | 首次即位；短期统治，埃及仍由总督和军政集团运作。 |
| 8 | 奥斯曼二世 | 1618—1622 | 军政改革冲突波及帝国各行省，被禁卫军推翻。 |
| 9 | 穆斯塔法一世（第二次） | 1622—1623 | 复位；另列以显示帝位中断。 |
| 10 | 穆拉德四世 | 1623—1640 | 中央重新强化秩序，埃及地方军团和贝伊家户仍持续。 |
| 11 | 易卜拉欣 | 1640—1648 | 宫廷与财政危机时期，行省精英自主空间扩大。 |
| 12 | 穆罕默德四世 | 1648—1687 | 科普律吕时代中央改革与长期战争并行，埃及税农和贝伊权力增长。 |
| 13 | 苏莱曼二世 | 1687—1691 | 帝国战争危机和军队更替影响行省财政。 |
| 14 | 艾哈迈德二世 | 1691—1695 | 延续战争财政，埃及仍通过税收和粮食支援中央。 |
| 15 | 穆斯塔法二世 | 1695—1703 | 卡尔洛维茨和约前后，帝国重整税制与军政。 |
| 16 | 艾哈迈德三世 | 1703—1730 | 埃及马穆鲁克家户竞争加剧，开罗军团和贝伊政治成为实际核心。 |
| 17 | 马哈茂德一世 | 1730—1754 | 卡兹达格里家户逐步取得优势，中央总督任期普遍较短。 |
| 18 | 奥斯曼三世 | 1754—1757 | 地方家户与驻军继续分掌实权。 |
| 19 | 穆斯塔法三世 | 1757—1774 | 阿里贝伊·卡比尔扩大自主并一度停止向伊斯坦布尔输贡。 |
| 20 | 阿卜杜勒哈米德一世 | 1774—1789 | 阿布·宰海卜、易卜拉欣贝伊和穆拉德贝伊相继掌权；1786年中央远征短暂恢复控制。 |
| 21 | 塞利姆三世 | 1789—1807 | 1798年法国入侵；奥斯曼与英国联军于1801年驱逐法军。 |
| 22 | 穆斯塔法四世 | 1807—1808 | 穆罕默德·阿里已在埃及取得总督职位和本地实权。 |
| 23 | 马哈茂德二世 | 1808—1839 | 承认并约束穆罕默德·阿里；埃及军队扩张到阿拉伯、苏丹和叙利亚。 |
| 24 | 阿卜杜勒迈吉德一世 | 1839—1861 | 1840—1841年国际安排确认穆罕默德·阿里家族世袭埃及，奥斯曼宗主权保留。 |
| 25 | 阿卜杜勒阿齐兹 | 1861—1876 | 1867年承认伊斯玛仪为赫迪夫，埃及自治与欧洲债务同步扩大。 |
| 26 | 穆拉德五世 | 1876年 | 短暂在位；埃及处于债务监督与欧洲干预危机。 |
| 27 | 阿卜杜勒哈米德二世 | 1876—1909 | 1882年英国占领后仍保留名义宗主权，埃及赫迪夫继续以奥斯曼名义统治。 |
| 28 | 穆罕默德五世 | 1909—1918；埃及关系至1914年 | 1914年英国宣布保护国并废黜阿拔斯二世，埃及与奥斯曼的法理宗主关系终止。 |

## 埃及本地实际权力阶段

| 时段 | 名义结构 | 实际权力中心 | 说明 |
|---|---|---|---|
| 1517—1525 | 奥斯曼苏丹、埃及总督 | 总督、底万、驻军与被保留的马穆鲁克贝伊 | 图曼贝伊抵抗失败后仍需吸纳地方军政网络；1524年艾哈迈德帕夏叛乱后中央重订行省制度。 |
| 1525—约1580 | 规制化行省 | 总督与七个军团、税务官、贝伊并立 | 总督掌最高名义权力，地方治安、税收和朝觐事务依赖军团及贝伊。 |
| 约1580—1650 | 奥斯曼行省 | 军团长官、贝伊和税农权力上升 | 通货、军费与总督短任促使地方组织更能左右开罗政治。 |
| 1650—1760年代 | 奥斯曼行省 | 菲卡里、卡西米、卡兹达格里等马穆鲁克家户 | 家户通过购买、训练和释放马穆鲁克延续，不是1517年前苏丹国的简单复活。 |
| 1768—1773 | 奥斯曼名义主权 | **阿里贝伊·卡比尔** | 停止部分贡赋、铸币并向汉志和叙利亚扩张；其将领阿布·宰海卜倒戈导致失败。 |
| 1773—1775 | 奥斯曼名义主权 | 穆罕默德贝伊·阿布·宰海卜 | 击败阿里贝伊后掌权，远征叙利亚期间去世。 |
| 1775—1798 | 奥斯曼名义主权 | 易卜拉欣贝伊、穆拉德贝伊及其对手 | 两大贝伊分享军政；1786年奥斯曼远征短暂压制，随后恢复影响。 |
| 1798—1799 | 法国军事占领 | 拿破仑及法国东方军 | 金字塔战役后占领开罗；尼罗河海战切断舰队，开罗发生反抗。 |
| 1799—1800 | 法国军事占领 | 克莱贝尔 | 拿破仑离开后接任；在赫利奥波利斯战胜奥斯曼军，后被刺杀。 |
| 1800—1801 | 法国军事占领 | 梅努 | 面对奥斯曼—英国联军，亚历山大里亚投降。 |
| 1801—1803 | 奥斯曼恢复名义统治 | 胡斯雷夫帕夏、阿尔巴尼亚军团、马穆鲁克贝伊 | 英军逐步撤离，多支军队争夺薪饷和行政。 |
| 1803年 | 奥斯曼恢复名义统治 | 塔希尔帕夏与军团 | 阿尔巴尼亚军领袖短暂控制开罗，因欠饷冲突被杀。 |
| 1803—1804 | 奥斯曼恢复名义统治 | 阿里帕夏·杰扎伊尔利及对手 | 中央任命难以落实，地方军团继续争权。 |
| 1804—1805 | 奥斯曼恢复名义统治 | 胡尔希德帕夏与穆罕默德·阿里 | 开罗乌理玛和民众压力促使中央于1805年承认穆罕默德·阿里为总督。 |
| 1805—1841 | 奥斯曼宗主权 | 穆罕默德·阿里及其军政国家 | 本地实权高度集中，军事扩张一度威胁伊斯坦布尔。 |
| 1841—1882 | 奥斯曼宗主权与世袭总督制 | 穆罕默德·阿里家族 | 1841年敕令确认世袭，1867年改称赫迪夫；具体君主见王朝页。 |
| 1882—1914 | 奥斯曼宗主权名义保留 | 英国占领军、驻埃总领事与赫迪夫政府 | 奥斯曼无力在埃及行使实际统治；1914年英国保护国使名义关系终止。 |
