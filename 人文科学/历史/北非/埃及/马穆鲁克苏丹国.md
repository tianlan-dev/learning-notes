# 马穆鲁克苏丹国

## 时间

1250—1517年

## 概括

马穆鲁克苏丹国由受过军事训练的奴隶出身军人及其解放者集团掌权，以开罗为中心统治埃及、叙利亚和汉志圣城通道。它在阿因·贾鲁特阻止蒙古军继续向埃及推进，消灭黎凡特十字军最后的大据点，并把红海—地中海贸易、朝觐和宗教教育结合为跨区域强权。苏丹继承并非普通父系王朝：军团同袍、主人—门生关系和高级埃米尔会议往往比血缘更能决定废立。

## 统治结构

| 层面 | 作用 |
|---|---|
| 苏丹 | 最高军事与任命中心，但必须维系本人的马穆鲁克家户和高级埃米尔联盟。 |
| 高级埃米尔 | 掌军团、要塞、叙利亚行省和宫廷职位，可拥立、废黜乃至杀死苏丹。 |
| 马穆鲁克招募 | 青年奴隶被购买、改宗、训练、解放并编入主人家户；身份不能自动传给子孙。 |
| 伊克塔与财政 | 军官获分配特定地区税收收益以供养骑兵，中央财政、关税和王室垄断同时存在。 |
| 宗教司法 | 四大逊尼法学派法官、乌理玛、苏菲组织和开罗阿拔斯哈里发提供司法与象征合法性。 |
| 地方社会 | 村社承担土地税与灌溉，商人连接地中海和印度洋；科普特官员和社群在财政、手工业与乡村中持续存在。 |

## 建立与强盛

1250年萨利赫·阿尤布去世后，沙杰尔·杜尔和巴赫里马穆鲁克在曼苏拉挫败路易九世军队。突兰沙试图用自己的亲信取代父亲旧部，最终被杀；沙杰尔·杜尔短期称苏丹后与艾贝克结婚，让位解决女性统治和阿拔斯承认争议。艾贝克又扶立幼年阿尤布王阿什拉夫·穆萨作为名义共治者，说明国家建立是数年过渡，而非一日完成。

1258年蒙古攻陷巴格达，1260年库图兹与拜巴尔斯在阿因·贾鲁特击败蒙古军。拜巴尔斯即位后重建驿站、情报、要塞和海军，在开罗扶立阿拔斯哈里发后裔以取得宗教名义，并逐步攻取十字军城堡。嘉拉温建立更稳定的家户联盟，其子哈利勒于1291年攻陷阿卡，黎凡特主要十字军国家的大陆据点由此终结。

## 鼎盛与社会经济

纳赛尔·穆罕默德三次即位，第三次统治清除监护埃米尔，重新分配伊克塔并推动开罗建筑、贸易和外交。埃及粮食、糖、纺织品和过境税支撑军队；亚历山大里亚连接地中海商人，红海港口通向也门和印度洋。苏丹对商路征税并保护朝觐路线，开罗因经学院、医院、清真寺、商栈和慈善基金成为知识与商业中心。

马穆鲁克制度能不断吸收新军人，减少完全依赖血缘继承的风险，却也制造持续派系竞争。苏丹之子常被短暂拥立，用于给强势埃米尔争取时间；成熟军人随后夺位。这种机制既造成高频政变，也让军政精英在王位更换后保留共同制度。

## 危机、调整与晚期统治

1347年以后黑死病反复袭击埃及与叙利亚，人口、耕地、税收和马穆鲁克招募均受长期影响。纳赛尔后裔幼主接连被埃米尔控制，巴尔库克于1382年建立以切尔克斯马穆鲁克为主的布尔吉统治。帖木儿1400—1401年攻陷叙利亚重镇，进一步削弱北方防线。

布尔吉时代并非持续衰败。巴尔斯拜征服塞浦路斯并强化香料贸易垄断，盖特贝伊在1468—1496年维持长期统治、修建防御与宗教建筑，并与奥斯曼争夺安纳托利亚边境。但垄断和临时征收损害商人信心；瘟疫、灌溉衰退、贝都因冲突和军团薪饷压力使财政弹性降低。葡萄牙绕过好望角进入印度洋后，埃及并未立刻失去全部香料贸易，却面临海上武装竞争和航路分流。

## 衰落与灭亡原因

结构因素是人口和税基受瘟疫反复削弱、伊克塔和军团薪饷失衡、频繁废立造成长期规划困难；外部压力包括葡萄牙海上竞争、奥斯曼对叙利亚边界的推进和更有效的火器—步兵体系。马穆鲁克并非“拒绝一切火器”，但其骑兵家户、财政和训练难以迅速重组为奥斯曼式大规模常备火器军。

直接灭亡过程分两步：1516年塞利姆一世在马尔吉·达比克击败高里，叙利亚防线瓦解；1517年奥斯曼军在里达尼亚击败图曼贝伊二世并占领开罗。图曼贝伊转入游击抵抗后被出卖、俘获并处死。马穆鲁克苏丹国灭亡，但马穆鲁克家户和贝伊在奥斯曼埃及重新进入地方权力结构，并未随1517年全部消失。
