# 埃及共和国

## 时间

1953年至今；当代信息核验至2026年7月14日。

## 概括

1952年自由军官运动迫使法鲁克退位，1953年废除君主制。共和国从革命指挥委员会统治发展为以总统、军队、安全机构和大型官僚体系为核心的国家。纳赛尔时期建立阿拉伯民族主义与国家主导发展模式；萨达特以1973年战争、经济开放和对美—对以转向重组路线；穆巴拉克在长期紧急状态下维持稳定。2011年以后，SCAF、穆尔西、曼苏尔过渡与塞西时期接续出现，说明共和国制度连续而最高权力安排多次重构。

## 统治结构

| 机构 | 法定或实际作用 |
|---|---|
| 总统 | 国家元首和行政权首脑，任命总理、参与国防外交并拥有广泛紧急与任命权。 |
| 总理与内阁 | 负责政策执行、预算、公共服务和各部协调；权力范围取决于总统、宪法和政治阶段。 |
| 军队 | 共和国创建者和重要国家机构，在1952、2011、2013年权力转换中发挥决定性作用，并拥有广泛经济与行政影响。 |
| 议会 | 先后以国民议会、人民议会、众议院及参议院等形式存在，承担立法、预算和政府监督，但有效制衡随时期变化。 |
| 司法 | 普通法院、行政法院和最高宪法法院并行；在选举、解散议会和过渡合法性中多次扮演关键角色。 |
| 政党与社会 | 从单一政治组织到有限多党、群众政党和社会运动；工会、宗教组织、青年网络和街头抗议影响政治议程。 |

## 国家元首与政府首脑

全部总统、代理总统、革命指挥委员会和武装部队最高委员会等集体元首，以及1953年以来非连续总理任期、看守和职位空缺，见[埃及共和国国家元首与政府首脑表](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E5%8C%97%E9%9D%9E/%E5%9F%83%E5%8F%8A/%E5%9F%83%E5%8F%8A%E5%85%B1%E5%92%8C%E5%9B%BD%E5%9B%BD%E5%AE%B6%E5%85%83%E9%A6%96%E4%B8%8E%E6%94%BF%E5%BA%9C%E9%A6%96%E8%84%91%E8%A1%A8.md)。

截至2026年7月14日，总统为阿卜杜勒·法塔赫·塞西，总理为穆斯塔法·马德布利。塞西于2024年4月开始新的六年总统任期；马德布利自2018年任总理，2024年获再任命，2026年内阁改组后仍在任。

## 革命建国与纳赛尔时期

革命指挥委员会先保留婴儿国王和文官内阁，1953年才正式宣布共和国。首任总统纳吉布主张较早恢复议会政治，纳赛尔及委员会多数则强调先清除旧党、推进土地改革和巩固军人革命；1954年纳吉布被解除职务，总统职位一度由委员会集体取代。1956年新宪法和公投确立纳赛尔总统地位。

纳赛尔实施土地改革、国有化和公共部门扩张，建设阿斯旺高坝。1956年苏伊士运河国有化引发英法以三国军事进攻；在美苏和国际压力下撤军，埃及政治声望大增。1958年埃及与叙利亚组成阿拉伯联合共和国，1961年叙利亚退出，埃及仍沿用国号至1971年。1967年战争失去西奈，军队和政权遭受重创；纳赛尔以消耗战、军队重建和群众动员维持统治。

## 萨达特时期

萨达特在1970年接任后清除纳赛尔旧部，逐步推行“开放政策”，减少对苏联依赖并吸引阿拉伯和西方资本。1973年埃及军队渡过苏伊士运河、突破巴列夫防线，虽然后期战局复杂，却为外交谈判和恢复国家信心创造条件。1977年面包补贴削减引发大规模骚乱，显示开放政策、债务与社会保障之间的矛盾。

萨达特1977年访问耶路撒冷，1978年签署戴维营协议，1979年与以色列建交；西奈分阶段归还，埃及一度被阿拉伯联盟孤立。政治开放有限，反对派、伊斯兰组织和社会不满增长。1981年政府大规模逮捕不同派别人士，同年10月萨达特在阅兵式上被极端组织成员刺杀。

## 穆巴拉克时期

穆巴拉克维持紧急状态、总统主导的民族民主党体系和安全机构影响，同时恢复埃及在阿拉伯世界的外交地位。人口增长、城市化、补贴和公共就业使国家承压；1990年代与国际机构合作的经济改革扩大私人部门，也加剧失业、地区和阶层差异。伊斯兰武装暴力、卢克索袭击等事件促使安全政策收紧。

2000年代通信、媒体和青年运动扩大公共空间，警察暴力、腐败、接班猜测、物价和失业积累不满。2011年1月25日起的大规模抗议持续18天，军队拒绝以全面武力维持总统，穆巴拉克于2月11日辞职并把权力交给SCAF。

## 2011年后的多次重组

SCAF暂停宪法、管理公投与议会—总统选举，同时面对街头抗议和军政审判争议。穆尔西2012年当选，总统、法院、军方、伊斯兰主义与世俗反对派围绕制宪和权限激烈冲突。2013年6月出现大规模反政府示威，军方于7月3日解除穆尔西职务；支持者的抗议营地随后被强力清场，造成大量伤亡。

代理总统曼苏尔时期通过2014年宪法。塞西辞去军职参选并于2014年就任，2018年连任；2019年宪法修正调整总统任期、军队角色和议会结构。政府以大型基础设施、新行政首都、道路、能源和社会救助项目推动国家能力，同时面临债务、通胀、汇率、人口增长、人权和政治参与等争议。2024年塞西开始新的六年任期，现行政体继续以总统和国家机构高度集中为特征。

## 兴衰与转型机制

共和国各阶段并非不同“王朝”。其连续基础是军队、中央官僚、尼罗河—开罗高度集中、公共部门和国家安全体系；政策更替则由战争结果、经济资源、总统继承联盟与群众动员共同驱动。1954、2011和2013年的直接权力转换都涉及军队，但社会参与、宪法形式和国际环境不同。分析共和国不能只用个人领袖解释，也不能忽略土地改革、人口、补贴、外汇、苏伊士运河和地区战争等结构条件。
