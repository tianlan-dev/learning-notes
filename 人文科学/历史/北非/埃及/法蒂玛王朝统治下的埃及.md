# 法蒂玛王朝统治下的埃及

## 时间

969—1171年

## 概括

969年法蒂玛将领焦哈尔征服埃及，随后建立开罗并迎接哈里发穆伊兹迁入。埃及由边缘行省变成伊斯玛仪派哈里发帝国的政治、财政和宗教中心，连接地中海、红海与印度洋。王朝后期军团竞争、饥荒和维齐尔专权使哈里发逐渐成为象征，十字军与赞吉势力的介入最终为萨拉丁取代法蒂玛创造条件。

## 统治结构

| 层面 | 作用 |
|---|---|
| 伊玛目—哈里发 | 兼具王朝君主与伊斯玛仪派宗教领袖身份，以宫廷礼仪和传教网络维持合法性。 |
| 维齐尔 | 早期协调文官财政；11世纪后“执剑维齐尔”常掌军队和实际决策。 |
| 军队 | 柏柏尔、突厥、苏丹和黑人军团等来源多样，派系平衡既支撑扩张也制造内战。 |
| 文官与税务 | 延续埃及土地、灌溉和账簿传统，基督徒、犹太人及穆斯林官员都曾担任要职。 |
| 宗教与城市 | 爱资哈尔、达尔·伊尔姆和宣教官服务王朝宗教；逊尼派、基督徒、犹太人仍构成多元社会。 |

## 哈里发世系

> 在位年为整个法蒂玛哈里发任期；穆伊兹自969年起实际拥有埃及。1130—1132年属于继承危机，库泰法特是掌权维齐尔和僭位者，不列为法蒂玛哈里发。

## 发展阶段

### 建国与鼎盛

焦哈尔建立开罗作为宫廷—军营城，福斯塔特仍是商业和人口中心；两城后来逐渐连为一体。阿齐兹时期埃及税收、地中海舰队和叙利亚据点支撑帝国。爱资哈尔最初服务伊斯玛仪派教育，达尔·伊尔姆则汇集书籍和学者。王朝向不同宗教官员开放职位，但政策取决于宫廷、财政与具体时期，不能概括为始终宽容或始终迫害。

### 大危机与军事维齐尔

穆斯坦绥尔漫长统治后期，尼罗河低水位、军团冲突、道路失控和宫廷财政崩溃叠加，形成11世纪60—70年代的“大灾难”。这不是单纯自然灾害：粮运和治安崩坏使歉收转化为饥荒。巴德尔·贾马利率叙利亚军入埃及后兼掌军政，重修开罗城墙并恢复征收；此后维齐尔通常控制军队，哈里发实权收缩。

### 继承分裂与灭亡

1094年穆斯坦绥尔死后，阿夫达勒支持穆斯塔利而非年长的尼扎尔，伊斯玛仪派由此分裂。12世纪幼主与暗杀频繁，维齐尔职位成为军团和宫廷集团争夺目标。十字军从巴勒斯坦进逼，叙利亚赞吉王朝又以援助名义派希尔库入埃及；沙瓦尔在双方之间反复结盟。1169年希尔库任维齐尔后很快去世，萨拉丁继任并掌握军队。1171年阿迪德去世，礼拜名义改奉阿拔斯哈里发，法蒂玛国家无大规模决战即告终结。

## 重要事件

| 时间 | 事件 | 影响 |
|---|---|---|
| 969年 | 焦哈尔征服埃及、营建开罗 | 埃及成为法蒂玛帝国核心。 |
| 970—972年 | 爱资哈尔建立并启用 | 形成王朝宗教与教育中心。 |
| 973年 | 穆伊兹迁入开罗 | 哈里发宫廷与中央机构正式移驻埃及。 |
| 996—1021年 | 哈基姆统治 | 宫廷、宗教与行政政策剧烈变化。 |
| 1009年 | 圣墓教堂被毁 | 加剧跨宗教与地中海政治影响，后续政策又有调整。 |
| 1060—1070年代 | 军团内战、饥荒与财政崩溃 | 帝国领土收缩，中央秩序近乎瓦解。 |
| 1074年以后 | 巴德尔·贾马利重建军政 | 军事维齐尔成为实际权力中心。 |
| 1094年 | 尼扎尔与穆斯塔利继承争议 | 伊斯玛仪派发生持久分裂。 |
| 1163—1169年 | 沙瓦尔、十字军与赞吉军反复争夺埃及 | 外部力量直接介入维齐尔任命。 |
| 1171年 | 萨拉丁终结法蒂玛哈里发 | 埃及转入逊尼派阿尤布王朝。 |

## 兴衰原因

强盛来自埃及农业税、开罗宫廷、红海—印度洋贸易、舰队与跨地区宣教网络。结构性衰落来自军团派系和维齐尔权力膨胀、哈里发幼主与旁支继承、叙利亚领土丧失；外部压力包括十字军和赞吉王朝。直接终结不是一次全民宗派反叛，而是萨拉丁控制军队和行政后，在末代哈里发去世时改变礼拜名义。爱资哈尔、开罗城市格局、贸易与官僚制度继续被后继王朝使用。
