# 英国占领与埃及王国

## 时间

1882—1953年；英国军队最终于1956年按协定撤离运河区。

## 概括

1882年英国为保护债权、苏伊士航路和陶菲克政权而发动战争并占领埃及。1882—1914年埃及法理上仍属奥斯曼、由赫迪夫和内阁治理，英国驻埃总领事和军队却掌握关键决策，故常称“隐蔽保护国”。1914年英国正式建立保护国；1922年单方面宣布埃及独立后，国王、议会、内阁、英国代表与驻军继续争夺权力，直到1952年自由军官运动改变体制。

## 分阶段实际权力

| 时段 | 埃及名义机构 | 英国权力工具 | 实际关系 |
|---|---|---|---|
| 1882—1914 | 赫迪夫、埃及内阁；奥斯曼宗主权名义存在 | 占领军、驻埃总领事、财政顾问及英籍部门官员 | 总领事可决定外交、军务和重大财政；埃及官僚继续执行日常治理。 |
| 1914—1922 | 埃及苏丹、内阁；英国保护国 | 高级专员、戒严和庞大战时驻军 | 奥斯曼关系终止，英国拥有公开最高权力。 |
| 1922—1936 | 国王、议会、内阁与1923年宪法 | 高级专员；英国保留防务、帝国交通、外国利益和苏丹问题 | 主权不完整；国王也利用英国与政党冲突强化王权。 |
| 1936—1952 | 国王、议会和内阁 | 英国大使、苏伊士运河区驻军、英埃条约 | 英军撤出主要城市但保留战略基地；1942年仍能以武力威胁国王。 |
| 1952—1953 | 福阿德二世、摄政机构与文官内阁 | 英军仍驻运河区，但不再控制开罗政权更替 | 革命指挥委员会掌握实权，1953年废除君主制。 |

| 2 | **伊夫林·巴林（克罗默勋爵）** | 总领事，1883—1907 | 长期主导财政、灌溉和行政；以偿债与秩序为优先，限制政治参与和教育开支。 |

| 4 | **赫伯特·基奇纳** | 总领事，1911—1914 | 强化行政与农业政策；一战开始后离任。 |
| 5 | 亨利·麦克马洪爵士 | 高级专员，1914—1916 | 保护国和战时统治初期代表；戒严与军事动员扩大。 |
| 6 | 雷金纳德·温盖特爵士 | 高级专员，1917—1919 | 战争末期与1919年革命初期在任。 |
| 7 | **埃德蒙·艾伦比勋爵** | 高级专员，1919—1925 | 镇压与谈判并用，推动1922年单方面独立声明。 |
| 8 | 乔治·劳埃德勋爵 | 高级专员，1925—1929 | 倾向强硬干预，与埃及民族主义和内阁冲突。 |
| 9 | 珀西·洛兰爵士 | 高级专员，1929—1933 | 尝试以外交协调国王、政党与英国利益。 |
| 10 | **迈尔斯·兰普森爵士** | 高级专员1934—1936；大使1936—1946 | 参与英埃条约；1942年率军包围阿卜丁宫迫使法鲁克任命纳哈斯内阁。 |
| 11 | 罗纳德·坎贝尔爵士 | 大使，1946—1950 | 战后谈判、运河驻军和苏丹问题持续。 |
| 12 | 拉尔夫·史蒂文森爵士 | 大使，1950—1955 | 经历1951年埃及废约、运河区冲突、1952年革命及共和国初期谈判。 |

> 1936年以后英国代表的正式身份是大使，不再称高级专员；其影响来自条约、驻军和外交压力，不能与1914—1922年的保护国最高权力等同。

## 占领的建立与“隐蔽保护国”

债务危机使英法在1870年代控制埃及财政。军官与议会力量以艾哈迈德·乌拉比为中心反对外国控制、军队内部歧视和王室专断。1882年英国炮击亚历山大里亚，在泰勒凯比尔击败乌拉比军并占领开罗；陶菲克继续为赫迪夫，英国宣称占领临时，却没有设定撤军日期。

克罗默通过财政顾问、英籍部门官员和对内阁任命的影响推行偿债、灌溉和棉花出口政策。财政稳定和农业工程确有成效，但教育、工业及政治代表受限，乡村税役与土地不均延续。1906年丁舍瓦伊事件的严厉审判激化民族主义，显示地方冲突可迅速变成占领合法性危机。

## 保护国、革命与名义独立

一战爆发后，英国于1914年废黜亲奥斯曼的阿拔斯二世，拥立侯赛因·卡米勒为苏丹并宣布保护国。戒严、粮食和牲畜征用、埃及劳工军团及物价上涨使社会承压。战后英国拒绝萨阿德·扎格卢勒代表团赴巴黎并将其流放，引发1919年跨阶层、跨地区抗议；铁路破坏、罢工和镇压共同迫使伦敦重新评估统治。

1922年英国单方面宣布埃及独立，但保留四项事项。1923年宪法建立议会，华夫脱在选举中取得群众支持；国王可任免内阁和解散议会，英国又在安全、外交和苏丹问题上干预，三方由此反复冲突。1936年条约使英军撤出开罗和亚历山大里亚、集中运河区，并为埃及加入国际联盟铺路，但没有结束军事依赖。

## 战争、王国危机与退出

二战期间埃及是英军北非战场基地。1942年兰普森以装甲车包围阿卜丁宫，要求法鲁克任命华夫脱的纳哈斯，否则退位；事件既损害王室，也使华夫脱被视为借英国上台。1948年战争失利、武器丑闻、政治暴力和社会不平等进一步削弱制度。

1951年华夫脱政府废除1936年条约，运河区警察、游击队与英军冲突升级。1952年1月伊斯梅利亚警察冲突和开罗大火暴露国家失控；7月自由军官夺权，法鲁克退位。英国没有直接阻止政权转换，战略重心转为谈判撤军。1954年英埃协定规定撤军，1956年最后一批英军离开运河基地；因此“王国结束”和“英军完全撤离”不是同一年。

## 重要事件

| 时间 | 事件 | 影响 |
|---|---|---|
| 1882年 | 亚历山大里亚炮击与泰勒凯比尔战役 | 英国建立军事占领，赫迪夫和内阁保留。 |
| 1883年 | 克罗默开始长期主导 | “隐蔽保护国”行政结构稳定化。 |
| 1898—1899年 | 英埃共同征服苏丹并建共管名义 | 苏丹主权争议成为埃及民族主义核心问题。 |
| 1906年 | 丁舍瓦伊事件 | 严厉审判激化反占领舆论。 |
| 1914年 | 英国保护国成立 | 奥斯曼宗主关系终止，英国最高权力公开化。 |
| 1919年 | 全国革命 | 推动米尔纳调查与独立谈判。 |
| 1922年 | 单方面独立声明 | 埃及王国建立，但四项保留事项限制主权。 |
| 1923—1924年 | 宪法、选举与扎格卢勒内阁 | 群众政党进入执政，王室与英国制约仍强。 |
| 1936年 | 英埃条约 | 驻军集中运河区，埃及国际地位提高。 |
| 1942年 | 阿卜丁宫事件 | 英国军事压力公开决定内阁，王室和华夫脱信誉受损。 |
| 1948年 | 第一次中东战争 | 军队和王国统治合法性遭受重创。 |
| 1951年 | 埃及宣布废除英埃条约 | 运河区武装冲突升级。 |
| 1952年1月 | 伊斯梅利亚冲突与开罗大火 | 王国政治失序达到高点。 |
| 1952年7月 | 自由军官运动 | 法鲁克退位，实际政权转移。 |
| 1953年 | 共和国成立 | 王国与穆罕默德·阿里王朝正式终结。 |

## 演变关系

- 本页自1882年起与[穆罕默德·阿里王朝](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E5%8C%97%E9%9D%9E/%E5%9F%83%E5%8F%8A/%E7%A9%86%E7%BD%95%E9%BB%98%E5%BE%B7%C2%B7%E9%98%BF%E9%87%8C%E7%8E%8B%E6%9C%9D.md)重叠：王朝仍在位，英国取得实际控制。
- 1914年保护国取代奥斯曼名义宗主权；1922年王国建立后英国关键影响仍延续。
- 1952—1953年后接[埃及共和国](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E5%8C%97%E9%9D%9E/%E5%9F%83%E5%8F%8A/%E5%9F%83%E5%8F%8A%E5%85%B1%E5%92%8C%E5%9B%BD.md)。
