# 拜占庭统治下的埃及

## 时间

395—641年

## 概括

东罗马时期的埃及既是君士坦丁堡财政与粮食体系的重要组成部分，也是晚期古代基督教神学、修道生活和科普特文化的中心。451年迦克墩会议后的教会分裂，使帝国任命的迦克墩派宗主教与埃及非迦克墩派教会长期并立；宗教分歧与税役压力削弱统治，却不能简化为“科普特人一致欢迎阿拉伯征服”。

## 统治结构

| 层面 | 权力与职能 |
|---|---|
| 皇帝与中央政府 | 任命行省官员、制定税制和宗教政策，粮食更多供应君士坦丁堡与东部军队。 |
| 行政与军事官员 | 行省总督、财政官、边防将领和后期更集中的军政官共同治理，职责曾多次调整。 |
| 大地产与城市 | 城市议事机构衰落后，地主家族、官僚、神庙转化后的教会地产承担征税和地方调解。 |
| 教会 | 亚历山大宗主教区拥有广泛社会网络；迦克墩派与非迦克墩派各有领袖、修道院和支持者。 |
| 乡村社群 | 科普特语和希腊语文书并用，村落、修道院和地主庄园承担税役、灌溉与救济。 |

## 发展过程

### 税粮、土地与城市

尼罗河粮食仍是帝国财政基础，征收逐渐由实物与货币并行。亚历山大里亚保持港口与行政中心地位，上埃及城市和修道院则连接乡村生产。阿皮翁家族等大地产拥有雇员、运输和税务能力，但不能据此认为国家消失；帝国常借地方强者完成征收，再以法律和军队约束他们。

查士丁尼时期重整埃及行政，把部分民政与军权结合，并加强对亚历山大里亚粮运和教会的控制。鼠疫、战争和高税会造成逃亡与欠税，地方差异很大：三角洲、法尤姆、上埃及及边疆面对的市场、驻军和宗教格局并不相同。

### 教会分裂与科普特传统

451年迦克墩会议关于基督性质的定义遭埃及多数教士和信众反对。此后帝国支持的迦克墩派宗主教与本土非迦克墩派领袖并立，后者逐渐形成科普特正教会传统。争议既涉及神学，也牵涉宗主教任命、教产、城市社群和皇权。6—7世纪多次强制和解与迫害扩大不信任，但不同群体仍可能在地方合作，埃及社会不是两块完全分离的阵营。

### 波斯占领、恢复与阿拉伯征服

萨珊军队约619年占领埃及，切断帝国重要税粮来源；629年前后希拉克略战争胜利后恢复东罗马统治。短期反复征服破坏财政与驻军，宗教和解政策又未能消除分裂。639年阿姆鲁·伊本·阿斯率军进入埃及，先后攻取法尤姆通道、巴比伦堡等地；亚历山大里亚在642年前后交出，东罗马曾于645年短暂夺回，646年再度失败。

## 重要事件

| 时间 | 事件 | 影响 |
|---|---|---|
| 395年 | 埃及归入东罗马帝国 | 政治中心明确转向君士坦丁堡。 |
| 451年 | 迦克墩会议 | 埃及非迦克墩派与帝国教会的长期分裂加深。 |
| 536年以后 | 帝国强化迦克墩派宗主教与行政控制 | 双重教会领导和地方抵抗制度化。 |
| 541年以后 | 查士丁尼瘟疫波及东地中海 | 人口、税收和城市网络承压，影响程度因地而异。 |
| 619—629年 | 萨珊波斯占领埃及 | 东罗马失去税粮和港口，行政连续性受到冲击。 |
| 629年前后 | 希拉克略恢复统治 | 帝国重建税收与宗教统一，但资源已经疲敝。 |
| 639年 | 阿拉伯军队进入埃及 | 征服战争开始。 |
| 640—641年 | 巴比伦堡被围并投降 | 尼罗河三角洲和亚历山大里亚门户失守。 |
| 642年 | 亚历山大里亚交出 | 东罗马失去埃及主要城市和港口。 |
| 645—646年 | 东罗马短暂反攻后再败 | 重新征服失败，阿拉伯统治稳定。 |

## 衰落因素与直接终结

结构因素包括沉重而刚性的税役、中央与地方教会的合法性冲突、驻军和行政对地方精英的依赖；外部压力是与萨珊波斯数十年战争耗尽财力并造成占领创伤。直接触发则是639年以后阿拉伯军队抓住边防空虚，逐步控制巴比伦堡、河运和亚历山大里亚。部分科普特官员和社群与新统治者合作，另一些抵抗或观望；征服成功不能归结为单一宗教群体的态度。
