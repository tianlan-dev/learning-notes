# 法老世系表

## 使用说明

本表从前王朝与第 0 王朝一直列至托勒密王朝终结。古埃及年代依赖同时代铭文、王表、考古层位、天文资料和后世曼涅托体系；王朝越早，绝对年代误差通常越大。第 7—10、13—17、22—24 王朝的次序与归属尤其不稳定，因此表中以“约”“不详”“残名”或“归属争议”标示，不把并立王系误排成全国性的单线继承。

“王朝”是编年框架，不必然等同于单一家族。每名能够辨识的统治者均单列；共治、复位、僭位、女性法老、地方反王和学界争议写在备注中。王表破损而姓名完全不可恢复的席位只能标明缺口，不能虚构姓名。相同名字的编号是现代区分法，不同研究体系可能采用不同编号。
