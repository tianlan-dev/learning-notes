# 新王国时期

## 时间

约前1550-前1069年。

## 概括

新王国时期是古埃及帝国扩张和国际影响力最强的阶段。埃及向南控制努比亚，向东北进入黎凡特，与赫梯、米坦尼、巴比伦等近东强权互动。底比斯、阿蒙神庙、帝王谷和大型神庙建筑共同构成这一时期的鲜明特征。

## 王朝世系 / 统治结构

| 层面 | 说明 |
|---|---|
| 王朝范围 | 通常包括第18-第20王朝。 |
| 王权中心 | 底比斯宗教地位突出，行政中心和王室活动也可在孟菲斯、阿玛尔纳、拉美西斯城等地展开。 |
| 帝国治理 | 埃及在努比亚和黎凡特建立军事、贡赋和附庸体系。 |
| 宗教权力 | 阿蒙祭司集团势力增强，王权与神庙经济关系复杂。 |

## 重要事件

- 雅赫摩斯一世驱逐希克索斯，建立第18王朝。
- 哈特谢普苏特、图特摩斯三世推动贸易、建筑和军事扩张。
- 阿肯那顿推行阿吞崇拜改革，形成阿玛尔纳时期。
- 拉美西斯二世与赫梯发生卡迭石战役，后签订和平条约。
- 新王国后期受海上民族冲击、财政压力和神庙势力扩张影响而衰落。
