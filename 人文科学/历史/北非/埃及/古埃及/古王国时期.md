# 古王国时期

## 时间

约前2686-前2181年。

## 概括

古王国时期是埃及中央集权和法老神圣王权高度发展的阶段，通常被称为“金字塔时代”。国家能够组织大规模劳役、石材运输和祭祀经济，吉萨金字塔群是这一时期最具代表性的遗存。

## 王朝世系 / 统治结构

| 层面 | 说明 |
|---|---|
| 王朝范围 | 通常包括第3-第6王朝。 |
| 政治中心 | 孟菲斯及其周边是核心政治区域。 |
| 法老王权 | 法老被赋予强烈神圣性，王陵和祭祀体系支撑王权合法性。 |
| 行政体系 | 宰相、地方长官、书吏和神庙经济共同构成国家治理网络。 |

## 重要事件

- 左塞尔阶梯金字塔代表早期大型石构建筑突破。
- 胡夫、哈夫拉、孟卡拉等法老修建吉萨金字塔群。
- 太阳神拉崇拜增强，法老王权与太阳神信仰结合。
- 后期中央财政和地方贵族势力变化，国家统一秩序逐渐松动。
