# 后期埃及时期

## 时间

前664-前332年。

## 概括

后期埃及时期是埃及在亚述冲击后出现复兴、又多次受到波斯帝国控制的阶段。赛斯王朝一度恢复本土王权和国际贸易，但埃及逐渐处于西亚大帝国和地中海强权竞争之中。

## 王朝世系 / 统治结构

| 层面 | 说明 |
|---|---|
| 王朝范围 | 通常包括第26-第31王朝。 |
| 赛斯复兴 | 第26王朝以赛斯为中心，恢复本土王权和对外联系。 |
| 波斯统治 | 阿契美尼德波斯多次把埃及纳入帝国行省体系。 |
| 本土复辟 | 部分本土王朝短暂恢复独立，但难以长期抵御外部帝国。 |

## 重要事件

- 普萨美提克一世建立第26王朝，推动赛斯复兴。
- 埃及与希腊雇佣兵、商人和地中海贸易联系加深。
- 冈比西斯二世征服埃及，埃及成为波斯帝国重要地区。
- 前332年亚历山大进入埃及，结束波斯在埃及的统治，开启希腊化阶段。
