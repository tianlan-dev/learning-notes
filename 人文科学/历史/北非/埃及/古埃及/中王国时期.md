# 中王国时期

## 时间

约前2055-前1650年。

## 概括

中王国时期是古埃及重新统一和国家秩序恢复的阶段。底比斯王权完成统一后，埃及强化行政控制、发展文学和灌溉工程，并向努比亚和西亚扩展影响。

## 王朝世系 / 统治结构

| 层面 | 说明 |
|---|---|
| 王朝范围 | 通常包括第11王朝后期、第12王朝和第13王朝早期。 |
| 王权中心 | 底比斯兴起，第12王朝时期政治中心与行政布局进一步调整。 |
| 行政治理 | 中央重新控制地方长官，官僚体系和文书行政成熟。 |
| 对外关系 | 南向控制努比亚资源，东向与黎凡特保持贸易和军事联系。 |

## 重要事件

- 曼图霍特普二世完成埃及重新统一。
- 第12王朝加强中央行政和边疆防御。
- 法尤姆地区开发和水利工程增强农业能力。
- 文学作品、教谕文和官僚文化高度发展。
