# 罗马统治下的埃及

## 时间

前30—395年

## 概括

前30年屋大维吞并托勒密埃及后，埃及成为皇帝直接掌握的特殊行省。罗马保留许多希腊化时期的税务、村社和神庙制度，同时把尼罗河粮食、亚历山大里亚港与红海贸易纳入帝国财政。四个世纪间，居民的身份、语言和宗教不断变化；所谓“罗马化”并不是旧社会被一次替换。

## 统治结构

| 层面 | 权力与职能 |
|---|---|
| 罗马皇帝 | 掌握最高任命、驻军与粮食安全；元老未经许可不得进入埃及。 |
| 埃及长官 | 通常由骑士阶层出任，兼理行政、司法和军务，直接向皇帝负责。 |
| 诺姆与城市 | 诺姆行政、希腊城市机构、村级书记和税吏承担土地丈量、人口登记与征收。 |
| 军队 | 军团与辅助军守卫亚历山大里亚、尼罗河通道、东部沙漠和南部边疆。 |
| 神庙与教会 | 传统神庙先被纳入监督和财政体系；3—4世纪基督教会、修道团体逐渐形成新的社会网络。 |

## 具体过程

### 皇帝行省的建立

屋大维处死托勒密十五世后把埃及视为个人胜利所得，任命首任长官科尔内利乌斯·加卢斯。罗马没有废除所有旧制度，而是接收王室土地、仓储和书吏网络，以定期人口普查、土地税、人头税和实物粮税提高可预测性。亚历山大里亚仍是首府和最大港口，谷物由尼罗河运至港区，再送往罗马；粮运重要，却不能把埃及经济只理解为“帝国粮仓”。

### 社会等级与帝国整合

早期法律和税役区分罗马公民、亚历山大里亚等希腊城市公民、乡村埃及人及其他群体，身份影响人头税、婚姻和任官。希腊语仍是行政主语，埃及语演变为科普特语；拉丁语主要用于军政。212年卡拉卡拉颁布普授公民权令，扩大罗马公民身份，但土地税、职业义务和地方差异并未因此消失。

亚历山大里亚的希腊人、犹太社群和帝国当局之间多次爆发冲突。115—117年的基托斯战争重创埃及犹太社群；3世纪帝国危机又带来篡位、瘟疫、通货贬值和帕尔米拉短暂占领。这些震荡与乡村征税、城市政治和边防压力相互作用，而非单一族群冲突。

### 宗教转型

传统神庙、祭司与节庆在罗马时期长期延续，皇帝也以法老形象出现在神庙浮雕中。基督教约在1—2世纪进入亚历山大里亚，3世纪以后扩展到乡村。德西乌斯、戴克里先时期的迫害留下强烈殉道记忆；313年以后教会合法化，亚历山大主教区和埃及修道传统迅速发展。安东尼、帕科米乌斯等相关传统使沙漠修道生活影响整个基督教世界。

## 重要事件

| 时间 | 事件 | 影响 |
|---|---|---|
| 前30年 | 屋大维攻占亚历山大里亚 | 托勒密王朝灭亡，埃及成为皇帝直辖行省。 |
| 前26—前25年 | 罗马军队向努比亚推进，后与麦罗埃议和 | 南部边界与贸易关系重新划定。 |
| 1世纪 | 定期人口普查、土地和粮税体系稳定运行 | 村社与家庭被更紧密纳入帝国财政。 |
| 115—117年 | 埃及卷入基托斯战争 | 亚历山大里亚及乡村犹太社群遭受严重破坏。 |
| 212年 | 普授公民权令实施 | 大多数自由民取得罗马公民身份，旧等级部分重组。 |
| 270—272年 | 帕尔米拉控制埃及，随后被奥勒良收复 | 显示帝国危机对粮运和地方军政的冲击。 |
| 297—298年 | 戴克里先镇压叛乱并重组行政 | 行省被细分，税制和边防进一步军事化。 |
| 313年以后 | 基督教合法化并快速制度化 | 教会与修道团体成为重要社会力量。 |
| 395年 | 帝国政治分治 | 埃及归入东罗马体系，但行政与社会并未在一年内断裂。 |

## 转型原因与历史影响

罗马统治的稳定基础是尼罗河农业、成熟书吏网络、港口与驻军；其长期压力则来自高额税役、城市身份矛盾、边防和帝国继承危机。395年并非一次“灭亡”，而是帝国最高权力中心长期东移后的制度划分。希腊语行政、基督教会、科普特语社会和税粮体系都延续到拜占庭时期，并成为7世纪征服者接收埃及的制度基础。
