# 奥斯曼统治下的埃及

## 时间

1517—1805年为行省主导阶段；1805—1914年奥斯曼名义宗主权与穆罕默德·阿里家族统治重叠。

## 概括

塞利姆一世在1516—1517年击败马穆鲁克，将埃及纳入奥斯曼帝国。新制度以苏丹任命的总督、开罗底万、驻军军团和地方马穆鲁克贝伊共同治理，而非由总督单线控制。17—18世纪马穆鲁克家户与税农逐渐掌握本地实权，但仍使用奥斯曼官职、敕令和贡赋框架。法国入侵打破旧平衡，1801—1805年的多方军政竞争最终使穆罕默德·阿里崛起。

## 统治结构

| 层面 | 作用 |
|---|---|
| 奥斯曼苏丹 | 掌握法理主权、任命总督、确认重要官职并征收贡赋。 |
| 埃及总督 | 代表中央主持底万、财政和军务，但任期往往短，必须与驻军及贝伊协商。 |
| 开罗底万与驻军 | 多个军团掌治安、薪饷和政治表决，能迫使总督离任。 |
| 马穆鲁克贝伊家户 | 1517年后重新吸收新马穆鲁克，控制地方行政、税收承包和武装随从。 |
| 税农与村社 | 包税人把中央税额落实到地方；村落、科普特书吏、商人和宗教基金维持农业和城市经济。 |
| 宗教与朝觐 | 乌理玛、爱资哈尔与苏菲组织具有调解和合法性作用；埃及负责麦加朝觐商队及圣城粮饷。 |

## 统治者

1517—1914年全部奥斯曼苏丹、穆斯塔法一世复位，以及总督—军团—马穆鲁克贝伊—法国占领—穆罕默德·阿里各阶段实际权力，见[奥斯曼时期埃及统治者与实际权力表](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E5%8C%97%E9%9D%9E/%E5%9F%83%E5%8F%8A/%E5%A5%A5%E6%96%AF%E6%9B%BC%E6%97%B6%E6%9C%9F%E5%9F%83%E5%8F%8A%E7%BB%9F%E6%B2%BB%E8%80%85%E4%B8%8E%E5%AE%9E%E9%99%85%E6%9D%83%E5%8A%9B%E8%A1%A8.md)。短任总督是帝国派遣的行政职位，不当作世袭君主表处理。

## 征服与制度重建

1516年高里在马尔吉·达比克战败，1517年图曼贝伊二世又在里达尼亚失败。奥斯曼占领开罗后处死图曼贝伊，但保留许多马穆鲁克贝伊、税务官和地方书吏，以避免税粮系统崩溃。苏莱曼一世时期的1525年规制明确总督、底万、军团和贝伊之间的权责；这一“制衡”能防止本地强人迅速独立，也使权力长期碎片化。

埃及不再是独立苏丹国，却仍因尼罗河粮食、红海贸易和朝觐路线具有战略地位。苏伊士舰队参与红海对葡萄牙竞争，开罗商人连接亚历山大里亚、吉达、也门和印度洋。年度贡赋和圣城粮饷把埃及财政与帝国及汉志联系起来。

## 地方化与马穆鲁克家户复兴

16世纪末以后，帝国战争、通货变化和军团人数增加导致欠饷与哗变。总督任期短，难以建立持续本地网络；贝伊和税农则通过婚姻、家户、购买新马穆鲁克和控制乡村税区累积力量。17世纪后菲卡里、卡西米等集团竞争，18世纪卡兹达格里家户占优势。这里的“马穆鲁克复兴”是奥斯曼制度内部的地方精英重组，不等同于1250—1517年苏丹国复国。

阿里贝伊·卡比尔在1760年代末控制开罗，停止部分贡赋、铸币并派军进入汉志和叙利亚。他的将领阿布·宰海卜倒戈，阿里贝伊于1773年失败。其后易卜拉欣贝伊与穆拉德贝伊分享权力；1786年奥斯曼派军短暂恢复中央控制，却未消除家户政治。

## 法国入侵与1805年权力转换

1798年拿破仑以打击英国印度航路和马穆鲁克为名入侵。法国在金字塔战役击败马穆鲁克骑兵并占领开罗，但舰队在尼罗河海战中被英国摧毁，补给受困；开罗起义、上埃及抵抗和瘟疫持续消耗占领军。拿破仑离开后克莱贝尔、梅努相继指挥，1801年奥斯曼—英国联军迫使法国投降。

驱逐法国没有自动恢复稳定。胡斯雷夫等奥斯曼总督、阿尔巴尼亚军团、马穆鲁克贝伊、英国和开罗乌理玛各有目标。阿尔巴尼亚军官穆罕默德·阿里在薪饷冲突中保存军力，并联合城市领袖反对胡尔希德帕夏；1805年中央被迫承认其为总督。这一任命仍属奥斯曼法理，却开启世袭化的本地王朝与近代军政国家。

## 重要事件

| 时间 | 事件 | 影响 |
|---|---|---|
| 1516年 | 马尔吉·达比克战役 | 马穆鲁克失去叙利亚防线，高里战死。 |
| 1517年 | 里达尼亚战役、图曼贝伊被处死 | 独立苏丹国终结，埃及并入奥斯曼。 |
| 1524年 | 艾哈迈德帕夏叛乱 | 显示总督可能借地方资源割据，中央随后重订制度。 |
| 1525年 | 苏莱曼时期行省规制 | 总督、底万、军团和贝伊权责制度化。 |
| 1580—1600年代 | 开罗驻军多次冲突 | 地方军团和贝伊对总督的制约增强。 |
| 1768—1773年 | 阿里贝伊扩大自主 | 奥斯曼名义权威受到本地强人直接挑战。 |
| 1786年 | 奥斯曼远征埃及 | 中央短暂压制易卜拉欣、穆拉德两贝伊。 |
| 1798年 | 法国入侵与金字塔战役 | 旧军政平衡瓦解，开罗被占。 |
| 1798年 | 尼罗河海战 | 法国舰队被毁，占领军与本土及欧洲后方隔绝。 |
| 1798—1800年 | 开罗起义与上埃及抵抗 | 法国控制始终依赖军事镇压。 |
| 1801年 | 法国军队投降 | 奥斯曼名义统治恢复，多方争权开始。 |
| 1805年 | 穆罕默德·阿里获任总督 | 行省主导阶段结束，本地王朝实权形成。 |

## 转型原因与制度遗产

行省秩序的长期韧性来自奥斯曼法理、成熟税务、军团制衡和吸纳旧马穆鲁克精英；弱点则是包税加重地方负担、总督短任、军团欠饷和家户私兵化。法国入侵是外部冲击，1801年后中央、驻军和贝伊均无力垄断暴力，才给穆罕默德·阿里提供直接机会。奥斯曼宗主权并未在1805年消失，而以敕令、称号和国际法关系延续到1914年。
