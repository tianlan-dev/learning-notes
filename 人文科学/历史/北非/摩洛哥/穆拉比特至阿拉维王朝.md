# 摩洛哥的穆拉比特至阿拉维王朝

## 时间

约11世纪中叶—1912年

## 概括

从穆拉比特兴起到阿拉维王朝接受法国保护前，摩洛哥经历多个王朝的征服、分裂和重建。王朝权力依赖宗教合法性、部族与军队、城市税收、撒哈拉贸易和对港口的控制。穆拉比特与穆瓦希德跨越马格里布和安达卢斯；马林、瓦塔斯、萨阿德和阿拉维则在不同国际压力下重组摩洛哥国家。

## 王朝世系与统治结构

| 王朝 | 大致时间 | 核心与特征 |
|---|---|---|
| 穆拉比特王朝 | 约1040年代—1147年 | 源于西撒哈拉桑哈贾联盟，以马拉喀什为中心，控制跨撒哈拉商路和安达卢斯 |
| 穆瓦希德王朝 | 约1121—1269年 | 以宗教改革和山区动员兴起，建立横跨马格里布与伊比利亚的帝国 |
| 马林王朝 | 13世纪中叶—1465年 | 以非斯为中心，重建城市和宗教教育，反复介入安达卢斯 |
| 瓦塔斯王朝 | 15世纪后期—1549年 | 面对伊比利亚海上扩张，中央权力较弱 |
| 萨阿德王朝 | 16世纪—1659年 | 以谢里夫合法性抗击葡萄牙，1578年三王战役后强化，1591年远征桑海 |
| 阿拉维王朝 | 17世纪中叶—1912年以后延续 | 以塔菲拉勒特为根基，整合军队、税收、宗教声望与对外外交 |

## 重要事件

- 1062年前后马拉喀什建立，成为穆拉比特政治中心。
- 1086年萨拉卡战役后，穆拉比特逐步控制安达卢斯穆斯林诸国。
- 1147年穆瓦希德攻占马拉喀什并取代穆拉比特。
- 1212年托洛萨会战后，穆瓦希德在伊比利亚和马格里布的权威加速衰落。
- 1415年葡萄牙占领休达，伊比利亚势力开始长期介入摩洛哥沿海。
- 1578年三王战役挫败葡萄牙远征，增强萨阿德王朝声望。
- 1591年摩洛哥军队击败桑海，但远距离统治难以维持。
- 17世纪后期阿拉维统治者逐步恢复统一，并以常备军和地方任命强化王权。
- 19世纪欧洲军事、财政和商业压力不断上升，最终导致保护国体制。
