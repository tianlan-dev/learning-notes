# 摩洛哥的古代毛里塔尼亚、伊德里斯与早期国家

## 时间

约前1千纪—11世纪中叶

## 概括

今摩洛哥在古代不是地中海帝国的空白边缘。大西洋—地中海海岸、里夫与阿特拉斯山地、塞斯平原和撒哈拉绿洲各有阿马齐格农牧社会、城市与部族联盟；腓尼基—迦太基商站、毛里塔尼亚王国、罗马行省和后罗马地方共同体只是其中不同层次。罗马统治集中于北部城市与交通带，边界外的地方首领和山地社会始终保有自主空间。

7—10世纪的伊斯兰化也不是一次征服立即完成。阿拉伯军队、总督、商人、法学家、哈瓦利吉传教者和本地阿马齐格群体共同参与新政治秩序；740年代大起义后，远西马格里布摆脱倭马亚的稳定直接控制，形成奈库尔、巴尔加瓦塔、锡吉勒马萨和伊德里斯等并立国家。伊德里斯王朝把先知后裔声望、阿马齐格联盟和非斯城市网络结合，成为后世摩洛哥谢里夫王权的重要历史资源，却从未在所有时期垄断今摩洛哥全境。

## 一、地理网络与国家形成背景

| 空间 | 主要资源与联系 | 对政治形成的影响 |
|---|---|---|
| 丹吉尔海峡与北部海岸 | 地中海航路、渔业、港口与伊比利亚联系 | 廷吉斯、利克苏斯等城市成为跨海中介，也最易受罗马和后来的伊比利亚势力影响。 |
| 沃吕比利斯—塞斯平原 | 谷物、橄榄、葡萄、道路与城市市场 | 支撑毛里塔尼亚和罗马城市精英，后成为伊德里斯一世立足区。 |
| 里夫与阿特拉斯山地 | 牧业、山地农业、部族联盟与避难空间 | 中央权力通常依赖协商、盟约和宗教声望，无法仅靠城市官僚深入控制。 |
| 大西洋平原 | 农牧、盐场、港湾和塔马斯纳交通 | 巴尔加瓦塔等联盟在此长期维持独立。 |
| 塔菲拉勒特—撒哈拉边缘 | 绿洲农业、盐金贸易和驼运路线 | 锡吉勒马萨因跨撒哈拉贸易成为财政与宗教竞争中心。 |

国家权力往往表现为对城市、商路和贡赋节点的控制梯度，而非边界固定的领土。后世材料中的“摩洛哥”也不应倒投为古代统一民族国家。

## 二、毛里塔尼亚王国与罗马廷吉塔纳

### 沿海交换与本地王权

前1千纪，腓尼基和迦太基商人在利克苏斯、莫加多尔等海岸节点活动，交换金属、农牧产品、紫色染料原料和其他商品。商站没有取代本地社会；考古所见的布匿文字、陶器与地方器物说明海岸居民选择性吸收地中海技术和符号。

约前3世纪起，沃吕比利斯等中心进入可辨认的毛里塔尼亚王国体系。前2—1世纪的君主通过婚姻、军队、钱币和与罗马结盟维持权力。博库斯一世在朱古达战争中改变联盟并获得努米底亚西部；前1世纪中叶王国一度由博古德和博库斯二世分治。前25年，奥古斯都把王国授予在罗马成长的朱巴二世及其妻克娄巴特拉·塞勒涅二世，形成依赖罗马又保留本地王室的附属国。

### 罗马兼并、城市繁荣与边疆限制

40年，卡利古拉处死末王托勒密。罗马在平定反抗后把王国分成廷吉塔纳与凯撒里亚两省；今摩洛哥北部主要属毛里塔尼亚廷吉塔纳，行政中心为廷吉斯。沃吕比利斯的公共建筑、住宅、榨油设施、铭文和道路显示罗马法、市政与市场扩展，橄榄油和农产品进入帝国网络。

这种“罗马化”不等于人口整体变成罗马人。城市精英使用拉丁语和罗马身份，乡村与山地仍延续阿马齐格语言和地方组织。罗马边防线也随时期变化；3世纪末帝国收缩对内陆沃吕比利斯的正式控制，沿海据点和地方社会则继续交流。汪达尔人进入北非、拜占庭恢复部分海岸控制，都没有在今摩洛哥建立像突尼斯沿海那样密集的统治。后罗马时期仍可见基督教、犹太社群、拉丁城市传统与本地王权并存。

## 三、阿拉伯征服与地方伊斯兰国家

7世纪末至8世纪初，阿拉伯军队越过伊弗里基亚进入远西马格里布。穆萨·伊本·努赛尔时代的军事征服把部分本地军队纳入倭马亚体系，711年渡海进入伊比利亚的塔里克·伊本·齐亚德军中有大量阿马齐格士兵。然而税赋、战利品分配、奴役和身份不平等引起不满，伊斯兰信仰的传播也由多种派别竞争推动，不等同于对大马士革总督的服从。

739／740年起的阿马齐格大起义击败倭马亚军队，终结哈里发对远西地区稳定的直接行政。此后形成多个政治中心：

| 政权／网络 | 大致时间 | 统治基础 | 转型或终结 |
|---|---|---|---|
| 奈库尔萨利赫王朝 | 约8世纪初—1019年 | 里夫山地、地中海港口、早期逊尼派王权 | 859年遭维京袭击后重建；长期受科尔多瓦、法蒂玛及地方联盟夹击，最终被阿兹达吉势力摧毁。 |
| 巴尔加瓦塔联盟 | 约744—11世纪中叶 | 塔马斯纳部族联盟、自成宗教文本和首领王权 | 材料多来自敌对作者，教义描述须谨慎；经伊德里斯、法蒂玛代理和穆拉比特多次战争后消失。 |
| 锡吉勒马萨米德拉尔王朝 | 约757—976年，后有地方继承者 | 苏弗里哈瓦利吉合法性、绿洲农业、跨撒哈拉金盐贸易 | 909年遭法蒂玛远征，后在泽纳塔竞争中反复易手；11世纪被穆拉比特纳入。 |
| 伊德里斯王朝 | 788—974年，985年有最后复起 | 先知后裔声望、阿瓦拉巴等联盟、沃吕比利斯—非斯城市网络 | 王族分封、地方自主及法蒂玛—后倭马亚代理竞争导致分裂。 |
| 法蒂玛与科尔多瓦后倭马亚代理网络 | 10世纪 | 米克纳萨、马格拉瓦等泽纳塔联盟与城市驻军 | 忠诚随资源和军事形势变化，未形成持久统一政权。 |

这些政权说明早期摩洛哥同时存在逊尼、什叶、哈瓦利吉和地方宗教传统；“伊斯兰化”是几个世纪的制度、贸易、婚姻、教育和政治过程。

## 四、伊德里斯王朝的建立、鼎盛与分裂

### 建立机制

伊德里斯·伊本·阿卜杜拉在786年法赫起义失败后逃离阿拔斯追捕，788年抵达沃吕比利斯附近。作为哈桑支阿里后裔，他拥有谢里夫宗教声望；阿瓦拉巴等本地群体则需要能连接伊斯兰世界、调解联盟并抗衡对手的领袖。双方的结合构成伊德里斯一世权力基础。他通过结盟和军事行动扩大影响，791年去世，传统称其被阿拔斯代理人毒杀。

伊德里斯二世为遗腹子，幼年由拉希德等摄政，803年亲政。他吸收阿拉伯战士、法学家、工匠和安达卢斯、凯鲁万移民，在非斯两岸发展新的城市中心。城市税收、市场、宗教学术和部族贡赋共同支撑王权；非斯后来成为摩洛哥最重要的政治与文化中心之一。

### 鼎盛与统治结构

伊德里斯二世时期，王朝影响从北部山地、塞斯平原延伸至大西洋和部分内陆，但巴尔加瓦塔、锡吉勒马萨及其他地方政权并未始终臣服。其统治结构可概括为：

- **谢里夫—伊玛目合法性**：以先知家族血统吸引效忠，并为后世王朝提供政治语言。
- **部族盟约与婚姻**：王权依赖阿马齐格联盟，不是外来少数单方面征服。
- **城市财政**：非斯的市场、手工业、移民社群和宗教机构提供稳定资源。
- **王族分封**：地方由兄弟与亲属治理，可快速扩张，却也使支系拥有独立军政基础。
- **宗教与司法**：法学家、清真寺和卡迪逐渐把伊斯兰规范嵌入城市与交易。

### 分裂、衰落与直接转折

828年伊德里斯二世去世后，穆罕默德把领地分给兄弟。分封起初以宗主权调和王族，随后演变为地方支系竞争。9世纪后期，非斯宫廷政变、首领更替和城市派系冲突削弱中心；王朝难以建立独立于部族和家族的常备军与稳定全国税制。

10世纪法蒂玛哈里发从东部扩张，科尔多瓦后倭马亚从伊比利亚介入，两者通过米克纳萨、马格拉瓦等联盟争夺远西马格里布。917／919年后伊德里斯主线失去非斯，虽在925年短暂复城，并在北部延续至974年、985年再度起兵，却不再恢复全国中心。11世纪的直接转折不是“伊德里斯亡、穆拉比特立即继承”，而是数十年的泽纳塔竞争、绿洲与商路重组，最终由桑哈贾穆拉比特运动夺取锡吉勒马萨、阿格马特和非斯等节点。

| 时间 | 事件 | 结果与长期影响 |
|---|---|---|
| 前1千纪 | 腓尼基—迦太基商站在沿海发展 | 海岸进入西地中海交换，但本地阿马齐格社会保持主体。 |
| 约前3世纪 | 沃吕比利斯等中心进入毛里塔尼亚王国体系 | 地方王权、城市与布匿—希腊化影响结合。 |
| 前105年左右 | 博库斯一世交出朱古达 | 王国获得努米底亚西部，与罗马结盟深化。 |
| 前25年 | 奥古斯都任命朱巴二世与塞勒涅共同统治 | 毛里塔尼亚成为文化多元的罗马附属王国。 |
| 40—42年 | 托勒密被杀、罗马兼并并平定反抗 | 毛里塔尼亚廷吉塔纳行省形成。 |
| 3世纪末 | 罗马收缩内陆防线 | 沃吕比利斯等地转入地方化，城市与宗教传统未骤然消失。 |
| 7世纪末—8世纪初 | 阿拉伯军政网络进入远西马格里布 | 伊斯兰化开始加速，但地方自主与宗派多样持续。 |
| 739／740—743年 | 阿马齐格大起义 | 倭马亚直接统治瓦解，多个本地伊斯兰国家兴起。 |
| 757年前后 | 锡吉勒马萨形成 | 跨撒哈拉贸易与绿洲国家成为摩洛哥南部主线。 |
| 788年 | 伊德里斯一世获阿瓦拉巴支持 | 谢里夫王权与本地联盟结合。 |
| 803—828年 | 伊德里斯二世亲政并建设非斯 | 王朝达到整合高峰，城市成为长期政治文化中心。 |
| 828年后 | 王族领地分封 | 宗主权逐步转为多支系竞争。 |
| 917／919年 | 伊德里斯主线失去非斯 | 法蒂玛代理与地方联盟主导城市。 |
| 974年、985年 | 北部伊德里斯统治终结及最后复起失败 | 王朝作为统一政治力量退出，但谢里夫家族与象征延续。 |
| 11世纪中叶 | 穆拉比特夺取绿洲、城市并进攻巴尔加瓦塔 | 撒哈拉宗教—军事联盟重组摩洛哥，进入跨区域帝国阶段。 |

## 崛起、鼎盛与衰落的因果层次

| 层次 | 崛起与整合 | 衰落与转型 |
|---|---|---|
| 结构因素 | 平原农业、城市市场、海陆商路和绿洲贸易提供资源 | 资源分布多中心，任何王朝都难以持续垄断山地、平原与撒哈拉。 |
| 合法性 | 本地王权、罗马承认、伊斯兰宗派或谢里夫血统各自提供动员语言 | 单一合法性不足以约束拥有地盘和军队的地方首领。 |
| 组织机制 | 部族联盟、王族分封、城市官员与宗教精英能快速扩张 | 分封和私人效忠使继承危机直接转为领地分裂。 |
| 外部环境 | 地中海贸易、伊比利亚和伊斯兰世界联系带来人才与财富 | 罗马、法蒂玛与科尔多瓦等强权可利用地方派系。 |
| 直接触发 | 强势领袖、迁徙社群和新宗教运动集中资源 | 暗杀、幼主、宫廷政变或一次败战加速已有分裂，但不是唯一原因。 |

## 争议与辨析

- **“毛里塔尼亚”名称**：古代 Mauretania 得名于毛里人及罗马地理，核心横跨今摩洛哥、阿尔及利亚北部；不能与现代国家毛里塔尼亚等同。
- **“罗马化”尺度**：沃吕比利斯和廷吉斯高度参与罗马城市体系，山地和南部并未被同样密度控制。
- **“阿拉伯征服等于阿拉伯化”**：军事征服、伊斯兰化和语言阿拉伯化是不同过程；阿马齐格语言与政治组织长期延续。
- **早期国家材料**：巴尔加瓦塔教义主要由敌对的穆斯林作者转述，不能把所有奇异描述当作中立事实。
- **伊德里斯“第一统一王朝”**：其国家对后世意义重大，但巴尔加瓦塔、锡吉勒马萨和奈库尔等并立，疆域也随时期变化。
- **非斯建立年代**：两岸聚落与城市扩建分属伊德里斯一世、伊德里斯二世及后续移民阶段，不能缩成单一“建城日”。
