# 摩洛哥历史

## 历史主线

摩洛哥位于马格里布最西端，同时面向大西洋、地中海、撒哈拉与伊比利亚。阿马齐格社会、古代毛里塔尼亚与罗马北部行省构成古代基础；阿拉伯征服后，伊斯兰化、城市和跨撒哈拉贸易推动多个地方国家并立。伊德里斯王朝以非斯和谢里夫合法性留下长期传统，穆拉比特、穆瓦希德、马林、瓦塔斯、萨阿德与阿拉维王朝则在部族联盟、宗教纲领、军队、城市税收和商路控制之间反复重组国家。

与阿尔及利亚、突尼斯不同，近世摩洛哥没有长期成为奥斯曼行省。阿拉维王朝自17世纪延续至今，但19世纪战争、外债与列强条约逐步侵蚀主权。1912年法国和西班牙保护区建立，苏丹制度保留而实权受制；王室、民族主义、武装抵抗、工人和国际环境共同推动1956年恢复独立。现代摩洛哥是国王拥有重要战略权力的宪政君主国，西撒哈拉最终地位截至2026年7月14日仍未解决。

## 重要转折与时间节点

| 时间 | 转折 | 导航意义 |
|---|---|---|
| 40—42年 | 罗马兼并毛里塔尼亚并设廷吉塔纳行省 | 古代附属王国转为行省，北部城市进入罗马行政。 |
| 739／740年代 | 阿马齐格大起义 | 倭马亚直接控制瓦解，远西马格里布地方国家兴起。 |
| 788年 | 伊德里斯一世建立统治 | 谢里夫合法性与本地联盟结合。 |
| 11世纪中叶 | 穆拉比特运动控制撒哈拉—摩洛哥节点 | 摩洛哥进入跨区域帝国阶段。 |
| 1147年 | 穆瓦希德攻占马拉喀什 | 穆拉比特灭亡，新帝国建立。 |
| 1212—1269年 | 托洛萨会战后分裂，马林最终夺取马拉喀什 | 穆瓦希德帝国解体，政治重心回到非斯王朝。 |
| 1415年 | 葡萄牙占领休达 | 伊比利亚海上扩张持续介入摩洛哥。 |
| 1578年 | 三王战役 | 葡萄牙远征失败，萨阿德曼苏尔时期开始。 |
| 1666—1668年 | 阿拉维拉希德夺取非斯与马拉喀什 | 延续至现代的王朝完成主要统一。 |
| 1844、1859—1860年 | 对法、对西战争失败 | 军事差距、赔款、贷款和列强干预加深。 |
| 1912年 | 《非斯条约》与法西协定 | 多重保护国体制形成。 |
| 1921—1926年 | 里夫共和国及法西联合战争 | 反殖民国家建设达到高点后遭镇压。 |
| 1953—1956年 | 苏丹被废、复位和独立 | 殖民合法性崩塌，王室成为主权核心。 |
| 1961—1999年 | 哈桑二世时期 | 强势君主制、西撒哈拉战争与有限开放并存。 |
| 2011年 | 二月二十日运动与修宪 | 政府权力条文扩大，王室战略权力保留。 |
| 2020—2026年 | 西撒哈拉敌对行动复起、联合国任务延续 | 实际控制与最终地位仍须分开表述。 |

## 阅读提示

- “毛里塔尼亚”在古代指横跨今摩洛哥与阿尔及利亚的王国和罗马行省，不等于现代[毛里塔尼亚](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E9%9D%9E%E6%B4%B2/%E8%A5%BF%E9%9D%9E/%E6%AF%9B%E9%87%8C%E5%A1%94%E5%B0%BC%E4%BA%9A/README.md)。
- 王朝版图随君主和地区效忠变化，不应把最大疆域投射到整段时期。
- 保护国时期必须区分苏丹、法国驻节总监、西班牙高级专员和丹吉尔国际行政。
- 西撒哈拉必须区分实际控制、主权主张、外国立场与联合国非殖民化框架。
