# 阿尔及利亚的努米底亚、罗马北非与伊斯兰化

## 时间

古代—1516年

## 概括

今阿尔及利亚地区在古代属于阿马齐格社会、努米底亚王国和罗马北非的重要部分。沿海城市、山地共同体和草原牧业构成多样社会。阿拉伯征服以后，伊斯兰逐步传播，地方阿马齐格力量既反抗外来总督，也建立鲁斯塔姆、齐里德、哈马德等王朝。

## 统治结构

| 政治阶段 | 中心或范围 | 统治特征 |
|---|---|---|
| 努米底亚王国 | 北部高原与沿海腹地 | 骑兵、地方首领和王权联盟 |
| 罗马行省 | 沿海城市、农业平原和军事边界 | 城市自治、庄园农业、驻军与税制 |
| 汪达尔、拜占庭时期 | 沿海与重要城市 | 帝国控制有限，地方政权和山地社会并存 |
| 鲁斯塔姆王朝 | 塔赫尔特 | 伊巴德派国家、商贸和宗教学术中心 |
| 齐里德与哈马德王朝 | 阿希尔、贝贾亚等 | 地方王朝、城市贸易与马格里布竞争 |
| 特莱姆森的扎亚尼德王朝 | 阿尔及利亚西部 | 控制东西与撒哈拉商路，在邻国压力间维持独立 |

## 重要事件

- 第二次布匿战争期间，马西尼萨借罗马支持统一努米底亚，并建立强大骑兵王国。
- 朱古达战争显示罗马与努米底亚精英之间复杂的联盟和冲突。
- 罗马时期北非成为重要谷物和橄榄油产区，基督教城市与多纳徒派运动兴盛。
- 7—8世纪阿拉伯军队进入中部马格里布，卡希娜等地方领袖成为抵抗记忆的一部分。
- 8世纪末鲁斯塔姆王朝建立，塔赫尔特连接马格里布与撒哈拉商路。
- 11世纪希拉勒部族迁徙改变部分平原和牧区的语言、土地利用与政治结构。
- 穆拉比特、穆瓦希德和后继地方王朝反复重组区域权力，直至16世纪奥斯曼介入。
