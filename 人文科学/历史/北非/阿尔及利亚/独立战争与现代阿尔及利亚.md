# 阿尔及利亚独立战争与现代国家

## 时间

1954年至今

## 概括

1954年民族解放阵线发动武装起义，阿尔及利亚战争迅速扩展为游击战、城市战、法国军队反叛乱、酷刑与大规模人口迁移并存的殖民战争。1962年《埃维昂协议》和公投结束法国统治，欧洲移民和许多哈基辅助人员外流。

独立后的民族解放阵线以革命合法性建立一党国家，军队在权力结构中保持重要地位。石油和天然气支持工业化与社会政策，也加深国家对能源收入的依赖。1988年后政治开放引发选举危机，1990年代内战造成严重伤亡；此后国家恢复秩序，但政治代表、青年就业和反腐问题持续存在。

## 统治结构

| 阶段 | 权力结构 | 主要特征 |
|---|---|---|
| 独立战争 | 民族解放阵线、民族解放军与海外组织 | 军政结合、外交动员和内部路线竞争 |
| 独立初期 | 民族解放阵线一党体制 | 国有化、土地改革和国家主导工业化 |
| 布迈丁时期及其后 | 军队支持的总统制 | 能源国家、官僚体系与非结盟外交 |
| 1989年后 | 多党制度与强大安全机构 | 选举开放、中止、内战及后续和解政策 |
| 2019年以后 | 制度调整与持续社会动员 | 希拉克要求更深政治改革，国家维持总统制框架 |

## 重要事件

- 1954年11月1日，民族解放阵线发动起义。
- 1956—1957年阿尔及尔战役显示城市地下网络与法军反叛乱的残酷性。
- 1958年法国本土政治危机推动第五共和国建立。
- 1962年《埃维昂协议》生效，阿尔及利亚独立。
- 1965年布迈丁政变后，军队与革命委员会强化国家主导发展。
- 1971年碳氢资源国有化，能源收入成为国家财政核心。
- 1988年大规模骚乱后，宪法改革允许多党竞争。
- 1991—1992年选举进程中断，随后爆发长期武装冲突。
- 1999年以后通过赦免与和解政策逐步降低大规模暴力。
- 2019年希拉克抗议促使总统辞职，并持续要求政治体系更新。
