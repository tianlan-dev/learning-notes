# 阿尔及利亚历史

阿尔及利亚横跨地中海沿岸、阿特拉斯山地、高原草地和撒哈拉。其历史不能压缩成“外族王朝依次占领”：阿马齐格社会、城市与绿洲网络长期延续，并主动参与努米底亚、罗马北非、伊斯兰王朝、奥斯曼摄政和现代民族国家的形成。法国1830年占领阿尔及尔后历经数十年才征服主要内陆，定居殖民和不平等公民制度最终引发1954—1962年独立战争。独立后，革命合法性、军队、能源国家和社会动员共同塑造共和国。

## 历史主线概括

古代努米底亚在迦太基—罗马战争中实现统一，王位分割和罗马干预又使其逐步被吞并。罗马城市、农业和基督教发展同山地、草原自主社会并存。七世纪以后，军事征服、伊斯兰化和阿拉伯化以不同速度推进；阿马齐格人既抵抗外来总督，也建立伊斯兰国家并参与跨地中海扩张。

十六世纪西班牙沿海压力促使阿尔及尔引入巴巴罗萨兄弟和奥斯曼军援。摄政名义隶属奥斯曼苏丹，实际由贝勒贝伊、帕夏、阿迦和代伊先后治理，中央通过三个贝伊领与地方盟约联系内陆。法国利用1827年外交危机于1830年攻占首都，却直到十九世纪后期才逐步压服阿卜杜勒·卡迪尔国家、君士坦丁、卡比利亚和南部社会。

殖民政府把北部划为法国省，大量土地转给欧洲移民，并以双重法律和选举制度限制穆斯林多数。1945年镇压和1947年制度失信推动民族主义激进化。独立战争胜利后，民族解放阵线和边境军主导国家；1965年政变、1971年能源国有化、1989年多党开放、1992年选举中止、1999年后和解及2019年希拉克，是共和国权力结构的主要转折。

## 关键因果线索

- **王朝更替**：努米底亚、鲁斯塔姆、哈马德和扎亚尼德的终结均包含内部继承、商路变化和外部军事压力，不能用单一“被征服”概括。
- **奥斯曼关系**：阿尔及尔并非完全独立或普通省份二选一；它承认苏丹名义并使用帝国合法性，同时自行外交、征税和选择代伊。
- **殖民建立**：1830年只标志首都投降，法国实际控制通过战争、条约、土地法和移民安置逐层扩展。
- **独立战争**：殖民不平等是结构原因，1945年镇压、1947年选举失信和民族主义组织危机是直接政治条件。
- **共和国政治**：军队在关键继承中作用突出，但总统、政府、民族解放阵线、能源国企和社会运动共同构成实际权力。
- **现代核验**：截至2026年7月14日，总统为阿卜杜勒马吉德·特本，总理为西菲·格里卜；2026年议会席位仍采用选举机构初步统计，等待宪法法院最终裁定。

## 阅读提示

- 中世纪王朝跨越现代国界，本目录只展开与今阿尔及利亚中心和社会关系最密切的部分。
- 阿马齐格、阿拉伯、穆斯林、犹太、欧洲移民等身份不能跨时代固定化；语言、法律和政治制度不断重塑其边界。
- 战争死亡、1945年镇压、哈基遭遇和1990年代内战均存在统计争议，笔记以证据范围和口径说明代替单一政治数字。
