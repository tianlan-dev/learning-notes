# 北非历史

## 范围与概括

北非是地中海、尼罗河、红海、撒哈拉与西亚交汇的历史区域。本目录以埃及、利比亚、突尼斯、阿尔及利亚、摩洛哥和苏丹为现代国家主线，并把西撒哈拉作为地位未决的争议地区单列。

北非历史并非只由阿拉伯征服或欧洲殖民塑造。尼罗河文明、阿马齐格诸社会、腓尼基与希腊殖民、罗马和拜占庭统治、伊斯兰化与阿拉伯化、跨撒哈拉贸易、地方王朝、奥斯曼行省、欧洲帝国主义和现代民族国家共同构成区域长时段。

- 上级入口：[历史](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/README.md)。
- 埃及目录兼具尼罗河文明长时段与现代国家通史；其内部明确区分文明连续性、政体更替和实际控制。
- 苏丹同时属于尼罗河谷、撒哈拉、萨赫勒和红海世界；南苏丹独立后的国家史由[南苏丹](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E9%9D%9E%E6%B4%B2/%E4%B8%9C%E9%9D%9E/%E5%8D%97%E8%8B%8F%E4%B8%B9/README.md)维护。

## 历史主线

- 地中海沿岸长期连接黎凡特、南欧与西地中海；迦太基、亚历山大里亚、昔兰尼加和罗马阿非利加都是跨海网络节点。
- 阿拉伯征服后，伊斯兰逐步成为主要宗教，阿拉伯语不断扩展，但阿马齐格语言、地方共同体与尼罗河流域传统并未消失。
- 撒哈拉是商路和迁徙通道。黄金、盐、奴隶、纺织品、马匹和学术网络将马格里布与萨赫勒连接起来。
- 19—20世纪欧洲统治形式不同：阿尔及利亚经历法国定居殖民，突尼斯和摩洛哥成为保护国，利比亚遭意大利征服，苏丹处于英埃共管，埃及长期受英国实际控制。
- 独立后的各国在君主制、共和国、军人政治、资源国家与大众政治之间形成不同道路。

## 重要转折与时间节点

| 时间 | 事件或过程 | 意义 |
|---|---|---|
| 前1千纪 | 腓尼基和希腊殖民网络扩展 | 北非海岸更深卷入地中海贸易与争霸 |
| 前146年 | 罗马灭迦太基 | 罗马逐步建立北非行省体系 |
| 7—8世纪 | 阿拉伯征服北非 | 北非进入伊斯兰政治世界，长期伊斯兰化与阿拉伯化开始 |
| 11—13世纪 | 穆拉比特、穆瓦希德等跨区域王朝兴盛 | 马格里布、撒哈拉和伊比利亚政治联系加强 |
| 16世纪 | 奥斯曼控制阿尔及尔、突尼斯和的黎波里 | 北非东中部纳入奥斯曼体系，摩洛哥保持独立王朝 |
| 1830年 | 法国入侵阿尔及利亚 | 欧洲对北非的直接殖民扩张进入新阶段 |
| 1881—1912年 | 突尼斯、埃及、利比亚和摩洛哥相继受欧洲强权控制 | 区域主权与社会经济结构被殖民秩序重组 |
| 1951—1962年 | 利比亚、苏丹、摩洛哥、突尼斯和阿尔及利亚先后独立 | 现代北非国家体系基本形成 |
| 1975年 | 西班牙撤离西撒哈拉 | 地位问题转化为长期战争与外交争议 |
| 2011年 | 北非多国发生大规模抗议与政权危机 | 政治参与、国家权力与社会经济矛盾重新组合 |

## 关键辨析

- “马格里布”狭义常指摩洛哥、阿尔及利亚和突尼斯，广义有时包括利比亚与毛里塔尼亚；本目录不据此移动已归入西非的[毛里塔尼亚](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E9%9D%9E%E6%B4%B2/%E8%A5%BF%E9%9D%9E/%E6%AF%9B%E9%87%8C%E5%A1%94%E5%B0%BC%E4%BA%9A/README.md)。
- “阿马齐格”是多种语言和地方共同体的总称之一，不是自古至今完全同质的单一民族。
- 西撒哈拉的主权和最终地位尚未解决，应区分实际控制、政治主张、国际承认与联合国非殖民化进程。
- 埃及、迦太基和马格里布不是彼此替代的单线阶段，而是不同尺度的文明、政权与历史空间。
