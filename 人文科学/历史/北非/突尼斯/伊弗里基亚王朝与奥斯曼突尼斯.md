# 突尼斯的伊弗里基亚王朝与奥斯曼时期

## 时间

7世纪—1881年

## 概括

阿拉伯征服后，今突尼斯成为伊弗里基亚的核心。凯鲁万是军事、宗教与学术中心，沿海城市则连接地中海贸易。阿格拉布、法蒂玛、齐里德和哈夫斯等王朝先后以此为基地，统治范围常超出现代突尼斯边界。

1574年奥斯曼帝国最终排除西班牙势力。此后突尼斯虽属于奥斯曼体系，地方军政集团和贝伊家族逐渐取得世袭权力。19世纪改革、借债和欧洲经济渗透造成财政危机，最终为法国保护国创造条件。

## 王朝与统治结构

| 阶段 | 大致时间 | 特征 |
|---|---|---|
| 阿拉伯征服与总督统治 | 7—8世纪 | 凯鲁万建立，伊斯兰行政和军事移民扩展 |
| 阿格拉布王朝 | 800—909年 | 名义尊奉阿拔斯，实际独立，向西西里扩张 |
| 法蒂玛与齐里德时期 | 909—12世纪 | 法蒂玛从伊弗里基亚兴起后迁往埃及，齐里德统治地方 |
| 穆瓦希德与哈夫斯王朝 | 12—16世纪 | 哈夫斯以突尼斯城为中心发展商业和外交 |
| 奥斯曼时期 | 1574—1881年 | 代伊、贝伊与地方军队并存，侯赛因贝伊家族世袭化 |

## 重要事件

- 670年乌克白·本·纳菲建立凯鲁万。
- 800年阿格拉布王朝获得对伊弗里基亚的世袭统治，并于9世纪进攻西西里。
- 909年法蒂玛运动推翻阿格拉布王朝，随后建立跨区域哈里发国家。
- 11世纪齐里德与法蒂玛决裂；希拉勒部族迁徙改变乡村和牧区秩序。
- 1229年哈夫斯王朝独立，以突尼斯城为贸易、外交和学术中心。
- 1270年第八次十字军东征抵达突尼斯，因疫病和谈判结束。
- 1574年奥斯曼帝国最终控制突尼斯。
- 1705年侯赛因王朝建立，贝伊权力世袭化。
- 19世纪艾哈迈德贝伊改革军队和行政，但高额支出和债务削弱财政。
- 1869年国际财政委员会介入，为1881年法国建立保护国铺路。
