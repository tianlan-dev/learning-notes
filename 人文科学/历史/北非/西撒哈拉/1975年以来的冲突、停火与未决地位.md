# 西撒哈拉1975年以来的冲突、停火与未决地位

## 时间

1975年至今

## 概括

西班牙撤离后，摩洛哥和毛里塔尼亚分别进入原西属撒哈拉部分区域，波利萨里奥阵线则以独立和自决为目标展开战争，并于1976年宣布撒哈拉阿拉伯民主共和国。毛里塔尼亚在1979年退出战争和领土主张，摩洛哥随后控制更多地区。

1980年代摩洛哥建设防御墙，形成延续至今的分区控制。1991年联合国斡旋实现停火并部署特派团，原计划通过自决程序解决地位，但选民资格、方案选择和政治条件争议使进程长期停滞。2020年停火破裂后，敌对行动恢复，最终地位仍未确定。

## 政治与控制结构

| 参与者 | 主要立场或作用 | 实际状况 |
|---|---|---|
| 摩洛哥 | 主张对西撒哈拉拥有主权，提出自治方案 | 控制防御墙以西的大部分地区和主要城市 |
| 波利萨里奥阵线 | 主张独立和自决，领导撒哈拉阿拉伯民主共和国 | 控制防御墙以东部分地区，政治和难民机构主要位于廷杜夫附近 |
| 联合国 | 推动双方接受政治解决并维护自决原则 | 1991年部署西撒哈拉全民投票特派团 |
| 阿尔及利亚 | 支持波利萨里奥并接纳难民营 | 是地区外交竞争的重要参与者 |
| 毛里塔尼亚 | 1979年退出领土主张 | 维持边境安全并避免重新卷入战争 |
| 撒哈拉威居民与难民 | 分处摩洛哥控制区、波利萨里奥控制区及难民营 | 经历家庭分离、迁徙限制和不同治理制度 |

## 重要事件

- 1975—1976年，西班牙撤离，摩洛哥和毛里塔尼亚进入相关区域。
- 1976年波利萨里奥宣布成立撒哈拉阿拉伯民主共和国。
- 1979年毛里塔尼亚与波利萨里奥达成和平并退出领土主张。
- 1980年代摩洛哥逐步修建防御墙，战争转为围绕墙线和据点的冲突。
- 1984年撒哈拉阿拉伯民主共和国加入非洲统一组织，摩洛哥随后退出该组织；摩洛哥于2017年重返非洲联盟。
- 1988年双方原则上接受联合国—非洲统一组织解决方案。
- 1991年停火生效，联合国特派团开始部署。
- 1990—2000年代，多轮选民识别与谈判未能产生公投或最终协议。
- 2010年格德姆·伊齐克营地事件反映社会经济诉求与民族政治矛盾交叠。
- 2020年盖尔盖拉特紧张局势后，波利萨里奥宣布停火不再有效，低强度战事恢复。
- 此后外交承认、领事机构和国际法院诉讼继续影响争议，但没有形成各方共同接受的最终安排。
