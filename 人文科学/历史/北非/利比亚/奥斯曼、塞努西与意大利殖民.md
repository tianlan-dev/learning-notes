# 利比亚的奥斯曼、塞努西与意大利殖民

## 时间

1551—1951年

## 概括

1551年奥斯曼帝国夺取的黎波里，并以行省体系控制西部港口和部分内陆。18世纪卡拉曼利家族建立半独立世袭统治，依靠地中海航运、贡赋和撒哈拉贸易。1835年奥斯曼恢复直接统治，同一时期塞努西教团在昔兰尼加和沙漠绿洲建立宗教、商业和调解网络。

1911年意大利入侵，但殖民统治长期局限于沿海。法西斯时期通过大规模军事行动、强迫迁移和集中营镇压抵抗。第二次世界大战后，英法分别管理原殖民地区，联合国进程最终推动独立。

## 统治结构

| 阶段 | 权力结构 | 主要特征 |
|---|---|---|
| 奥斯曼第一统治期 | 帕夏、驻军与地方首领 | 的黎波里为行省中心，对内陆控制有限 |
| 卡拉曼利王朝 | 世袭帕夏 | 名义服从奥斯曼，实际自主，经营海运和商路 |
| 奥斯曼第二统治期 | 帝国总督与改革行政 | 坦志麦特改革、边疆驻军和地方协商 |
| 塞努西网络 | 教团扎维亚、部族和商队 | 在昔兰尼加与绿洲提供宗教、仲裁和防卫组织 |
| 意大利殖民 | 殖民总督、军队与定居机构 | 军事征服、土地占领、道路和定居计划 |
| 战后托管前过渡 | 英国管理昔兰尼加和的黎波里塔尼亚，法国管理费赞 | 地方代表与国际谈判决定国家前途 |

## 重要事件

- 1551年奥斯曼军队攻占的黎波里。
- 1711年卡拉曼利家族取得政权，建立世袭帕夏统治。
- 1801—1805年第一次巴巴里战争反映的黎波里与美国之间的海上冲突。
- 1835年奥斯曼恢复直接统治。
- 1837年前后穆罕默德·伊本·阿里·塞努西创建塞努西教团，并向昔兰尼加内陆扩展。
- 1911—1912年意土战争后，意大利宣布吞并利比亚。
- 1920—1930年代，意大利法西斯政权以集中营和强迫迁徙镇压昔兰尼加抵抗。
- 1931年欧麦尔·穆赫塔尔被捕并处决。
- 1940—1943年北非战役使利比亚成为轴心国和盟军主战场。
- 1949年联合国决定利比亚应在限定期限内独立，三地区开始协商联邦建国。
