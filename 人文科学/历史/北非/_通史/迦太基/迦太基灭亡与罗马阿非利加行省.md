# 迦太基灭亡与罗马阿非利加行省

## 时间

前146年—公元5世纪；后续延伸至汪达尔与拜占庭统治的转型。

## 概括

前146年罗马毁灭布匿迦太基，并把其在今突尼斯东北部的核心领土设为阿非利加行省。罗马最初并未立即吞并整个马格里布：努米底亚和毛里塔尼亚仍以王国、盟友或附庸形式存在，行省边界随后才随罗马战争和行政改革扩大。

政治上的迦太基被消灭，布匿社会却没有同时消失。地方城市继续使用布匿语言、宗教和行政传统，北非精英逐步取得罗马公民权并进入帝国体系。罗马重建的迦太基最终成为阿非利加行省首府、地中海大城和拉丁基督教中心。

## 罗马接管过程

1. 第三次布匿战争中，罗马围城三年；前146年小西庇阿攻陷城市，幸存者大量被奴役，城墙、住宅和公共设施遭拆毁。
2. 罗马把迦太基核心领土划为阿非利加行省，以乌提卡作为早期行政中心；原迦太基盟邦、敌对城市和努米底亚领地获得不同处置。
3. 新行省初期驻军有限，罗马主要通过总督、地方城市、税收承包和土地界定维持统治。
4. 前122年盖约·格拉古推动在迦太基旧址设殖民地，计划因罗马国内政治斗争而中止，部分殖民者仍可能留下。
5. 凯撒重新规划殖民城；其死后奥古斯都时代完成建置。新城沿用“迦太基”之名，却采用罗马殖民地法律与城市规划。

“撒盐诅咒土地”的叙述是后世传说，没有可靠古代证据。罗马确实毁城并剥夺主权，但旧址后来能够迅速成为殖民城市，正说明土地并未被永久废弃。

## 行省统治结构

| 层级 | 主要制度 | 实际作用 |
|---|---|---|
| 罗马中央 | 元老院、皇帝与帝国财政 | 任命总督、决定军政边界和税收框架 |
| 行省总督 | 共和国时期的年度长官，帝国时期阿非利加资深执政官 | 司法、税务监督、城市协调；边防军权在不同时期另有分工 |
| 殖民地与自治城市 | 地方议会、年度官员、祭司和城市精英 | 征税、公共工程、市场与地方司法，是日常统治基础 |
| 皇室及大地主庄园 | 皇室代理、承租人和佃农 | 经营谷物、橄榄及畜牧，联系港口与罗马市场 |
| 边疆军区 | 军团、辅助军和道路堡垒 | 控制奥雷斯山、撒哈拉边缘及跨区域通道 |
| 地方社群 | 布匿、努米底亚、阿马齐格等多种城市和乡村群体 | 在罗马法律、语言与本地传统之间形成多层身份 |

## 经济、城市与社会变化

北非并非只为罗马“提供粮食”的被动仓库。灌溉、蓄水、道路、港口和榨油设施扩展，使谷物、橄榄油、葡萄酒、陶器和鱼酱形成跨地中海商品链。帝国首都的粮食配给推动大规模生产，但地方城市、军队和区域市场同样重要。

罗马公民权、拉丁语教育和市政职位为地方精英提供上升途径；与此同时，布匿语在乡村和部分城市延续数百年，许多神祇通过名称对应与仪式重释进入罗马宗教。所谓“罗马化”因此是法律、消费、语言和身份选择的多向过程，不是本地文化突然被替换。

罗马迦太基拥有论坛、港口、蓄水池、剧场、竞技场和安东尼浴场等设施。城市既是行省行政中心，也是通向罗马、伊比利亚和东方地中海的贸易枢纽。

## 重要事件与阶段

| 时间 | 事件 | 影响 |
|---|---|---|
| 前146年 | 布匿迦太基灭亡，阿非利加行省建立 | 迦太基国家终结，罗马获得北非第一个正式行省核心 |
| 前122年 | 格拉古殖民计划 | 显示罗马国内土地问题开始与海外殖民结合 |
| 前46年 | 塔普苏斯战役后，凯撒兼并努米底亚王国大片领地 | 罗马在北非的直接统治显著扩大 |
| 前1世纪后期 | 罗马迦太基正式重建 | 行政中心由乌提卡逐渐转回迦太基 |
| 1—2世纪 | 城市、道路、农业庄园与港口贸易扩张 | 阿非利加成为帝国富庶行省群 |
| 203年 | 佩尔佩图阿与费利西塔殉教传统 | 反映北非基督教社群已形成 |
| 3—4世纪 | 居普良、奥古斯丁及多纳图派争论 | 北非成为拉丁基督教思想和教会冲突中心 |
| 429—439年 | 汪达尔人由伊比利亚进入北非并攻取迦太基 | 西罗马失去税源、舰队基地和重要粮油产区 |
| 533—534年 | 查士丁尼军队击败汪达尔王国 | 迦太基成为拜占庭阿非利加行政中心 |

## 罗马统治的维系与崩解

**维系条件**包括高产农业、密集城市、地中海海运、地方精英参与和相对有限的常备军成本。罗马通过赋予城市不同等级权利，把既有布匿和努米底亚中心纳入税收与荣誉体系。

**结构性裂缝**包括土地和税负不均、佃农与大庄园矛盾、城市教会争端，以及边疆群体与帝国军政机构的周期性冲突。多纳图派不能简单等同于“民族反罗马运动”，其宗教、社会与地方网络动机远比单一标签复杂。

**外部压力**在5世纪急剧增强。西罗马内战和军队争权削弱北非防务，汪达尔集团利用帝国内部分裂渡海。**直接转折**是439年盖塞里克突然夺取迦太基：这终结了西罗马在核心城市的统治，却没有摧毁城市本身。汪达尔王国继续使用罗马税制、港口和行政人才，533—534年再被拜占庭征服。

## 长期影响

- 罗马行省边界与道路网络影响后续汪达尔、拜占庭和早期伊斯兰政权的行政空间。
- 布匿传统、阿马齐格地方社会与罗马制度长期混合，说明“征服”与“文化消失”不是同一事件。
- 北非粮油与税收对西罗马具有战略价值，失去迦太基加剧其财政和海运困境。
- 北非拉丁基督教传统通过德尔图良、居普良、奥古斯丁等人物深刻影响西方基督教。
- 今日遗址同时保存布匿港口、罗马城市、早期基督教、汪达尔和拜占庭层次，不能只把迦太基理解为汉尼拔时代的废墟。
