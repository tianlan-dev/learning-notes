# 北非通史

## 概括

本目录维护以北非为核心、跨越多个现代国家的区域文明和帝国专题。国家通史仍在埃及、利比亚、突尼斯、阿尔及利亚、摩洛哥、苏丹和西撒哈拉目录维护。
