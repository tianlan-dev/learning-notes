# 北非通史

## 范围与对象

“_通史”是跨目录组织方式，不代表所有内容属于同一种知识对象。本目录只维护：

1. 跨越多个北非国家的区域共同史；
2. 社会、贸易、迁徙、殖民等跨境过程；
3. 不能由单一现代国家完整代表的跨区域文明或帝国规范入口。

现代国家通史仍在埃及、利比亚、突尼斯、阿尔及利亚、摩洛哥、苏丹和西撒哈拉目录维护。规范入口集中维护完整主线，国家页只说明本地经历并回链。

## 阅读规则

- 先判断对象是规范文明入口、区域共同史还是跨境专题。
- 国家页引用规范入口时，不复制完整文明或帝国主线。
- 同一过程在本页提供区域比较框架，在国家页解释当地时间、制度和社会结果。
