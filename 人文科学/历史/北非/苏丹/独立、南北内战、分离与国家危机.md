# 苏丹独立、南北内战、分离与国家危机

## 时间

1956年至今

## 概括

苏丹独立时继承了面积庞大、地区发展不均和政治代表失衡的殖民国家。第一次南北内战在独立前夕已经开始，1972年自治协议带来十年和平；1983年自治被削弱、伊斯兰法律与资源争议叠加，第二次内战爆发。2005年《全面和平协议》结束主要南北战争，并安排南部公投，[南苏丹](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E9%9D%9E%E6%B4%B2/%E4%B8%9C%E9%9D%9E/%E5%8D%97%E8%8B%8F%E4%B8%B9/README.md)于2011年独立。

南部分离并未解决苏丹境内所有矛盾。达尔富尔、南科尔多凡、青尼罗河和红海地区的边缘化问题持续，军队、安全机构、民兵和文官政治力量反复争夺国家权力。2019年革命终结巴希尔长期统治，但军民过渡在2021年政变后中断，2023年苏丹武装部队与快速支援部队之间爆发战争。

## 统治结构

| 阶段 | 权力结构 | 主要特征 |
|---|---|---|
| 独立议会时期 | 多党议会、宗派政党与军队 | 政府更替频繁，南部战争持续 |
| 1958年军政府 | 阿布德军人政权 | 中央集权和战争升级，1964年被群众运动推翻 |
| 尼迈里时期 | 军人总统、单一政治组织与地方自治安排 | 1972年和平，后转向伊斯兰化并重启战争 |
| 1985—1989年 | 过渡军政委员会和议会政府 | 民主恢复但未能结束战争和经济危机 |
| 巴希尔时期 | 军队、伊斯兰主义政党与安全机构 | 长期威权统治、地区战争和国际制裁 |
| 2019年后过渡 | 文官联盟与军方共同机构 | 权力分享脆弱，2021年政变破坏过渡 |
| 2023年战争 | 苏丹武装部队与快速支援部队竞争 | 国家机构分裂，人道和社会危机扩大 |

## 重要事件

- 1955年南部部队兵变，第一次南北内战展开；1956年苏丹正式独立。
- 1964年“十月革命”推翻阿布德军政府。
- 1969年尼迈里政变建立新军人政权。
- 1972年《亚的斯亚贝巴协议》给予南部自治，结束第一次内战。
- 1983年南部自治安排被重组，苏丹人民解放运动兴起，第二次内战开始。
- 1989年巴希尔发动政变，并与伊斯兰主义力量建立统治联盟。
- 2003年达尔富尔战争升级，政府军、民兵与反政府组织造成大规模暴力和流离失所。
- 2005年《全面和平协议》签署，南部获得自治和自决安排。
- 2011年公投后南苏丹独立；石油、边界和阿卜耶伊问题继续影响两国关系。
- 2019年群众抗议和军方行动推翻巴希尔，随后形成军民过渡安排。
- 2021年军事政变中断过渡。
- 2023年苏丹武装部队与快速支援部队爆发战争，冲突扩展至多地。
