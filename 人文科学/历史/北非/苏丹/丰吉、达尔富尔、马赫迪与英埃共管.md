# 苏丹的丰吉、达尔富尔、马赫迪与英埃共管

## 时间

约1504—1956年

## 概括

基督教努比亚王国衰落后，丰吉苏丹国以森纳尔为中心控制青尼罗河和尼罗河中游，达尔富尔苏丹国则在西部连接萨赫勒、乍得与跨撒哈拉商路。伊斯兰学者、苏菲教团、商人和阿拉伯语在这一时期进一步扩展，但地方语言与政治共同体保持多样。

1821年穆罕默德·阿里统治下的埃及征服森纳尔，建立征税、征兵和奴隶贸易驱动的行政体系。马赫迪起义推翻该统治并建立独立国家，1898年战败后又被英埃共管取代。

## 统治结构

| 阶段 | 权力结构 | 主要特征 |
|---|---|---|
| 丰吉苏丹国 | 森纳尔苏丹、地方首领与商人 | 控制尼罗河贸易，对边缘地区统治松散 |
| 达尔富尔苏丹国 | 凯拉王朝苏丹与省级官员 | 农业、贡赋、商路和多族群整合 |
| 土耳其—埃及统治 | 埃及总督、军队和官僚 | 征税、征兵、象牙和奴隶贸易扩张 |
| 马赫迪国家 | 马赫迪及继任哈里发、宗教军政组织 | 以宗教革命动员推翻外来统治 |
| 英埃共管 | 英国总督掌握实权，埃及保留名义共管地位 | 铁路、灌溉、间接统治和南北差异化行政 |
| 自治过渡 | 政党、议会与殖民总督 | 亲埃及统一派和独立派竞争，最终选择完全独立 |

## 重要事件

- 1504年前后丰吉苏丹国建立，森纳尔成为商业和伊斯兰学术中心。
- 17世纪达尔富尔凯拉王朝巩固，并控制向埃及、乍得湖和西非的商路。
- 1821年埃及军队征服森纳尔；喀土穆逐步成为新行政中心。
- 1874年埃及征服达尔富尔，把西部纳入更广行政体系。
- 1881年穆罕默德·艾哈迈德宣布自己为马赫迪，发动反抗。
- 1885年马赫迪军占领喀土穆；马赫迪去世后由哈里发阿卜杜拉统治。
- 1898年恩图曼战役后马赫迪国家崩溃。
- 1899年英埃共管安排建立，实际权力主要掌握在英国官员手中。
- 1924年白旗联盟起义显示早期苏丹民族主义和军人参与。
- 1938年毕业生大会成立，城市受教育精英推动自治。
- 1953年自治协议和选举启动权力移交；1956年议会宣布独立。
