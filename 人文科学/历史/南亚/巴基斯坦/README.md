# 巴基斯坦历史

巴基斯坦的历史横跨印度河文明核心区、犍陀罗佛教文化区、旁遮普与信德的伊斯兰化，以及英属印度西北省份。现代国家于1947年从英属印度分治而来，最初还包括东巴基斯坦，1971年东翼独立为孟加拉国。

## 范围与对象

本页从今巴基斯坦国境及1947年国家形成出发，提供印度河、犍陀罗、莫卧儿和英属印度等共享历史的本地视角。共享阶段的完整区域主线由南亚规范主笔记维护；本页不会把古代文明或跨国帝国倒推为现代巴基斯坦一国的专属前史。

## 历史坐标

- 地理范围：印度河流域、旁遮普、信德、俾路支和开伯尔山口一带。
- 国家形成：现代边界、行政语言和制度多受殖民、分治或山地—海洋交通网络影响，不能直接等同于古代政权。
- 区域联系：[南亚历史](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E5%8D%97%E4%BA%9A/README.md)；跨区域背景见[英属印度、分治与现代南亚](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E5%8D%97%E4%BA%9A/_%E9%80%9A%E5%8F%B2/%E8%8B%B1%E5%B1%9E%E5%8D%B0%E5%BA%A6%E3%80%81%E5%88%86%E6%B2%BB%E4%B8%8E%E7%8E%B0%E4%BB%A3%E5%8D%97%E4%BA%9A.md)。

## 关键节点

| 时间 | 事件 | 意义 |
|---|---|---|
| 约前2600年 | 印度河文明成熟 | 现代国家境内最早的城市文明核心 |
| 711年 | 阿拉伯军队进入信德 | 伊斯兰政治与社会史的重要开端 |
| 1849年 | 英国吞并旁遮普 | 西北印度纳入英属殖民体系 |
| 1947年 | 巴基斯坦独立 | 现代国家成立 |
| 1971年 | 孟加拉国独立 | 东西两翼国家结构终结 |
