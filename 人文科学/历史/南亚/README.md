# 南亚

## 概括

南亚历史以印度次大陆为核心，连接印度河、恒河、德干高原、印度洋、伊朗高原、中亚和东南亚。它既包含印度河文明、吠陀与恒河国家化、佛教和印度教传统，也包含伊斯兰王朝、莫卧儿帝国、英国殖民统治、印巴分治和现代民族国家。

当前先建立[印度](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E5%8D%97%E4%BA%9A/%E5%8D%B0%E5%BA%A6/README.md)主线，后续可继续扩展巴基斯坦、孟加拉、斯里兰卡、尼泊尔等目录。
