# 印度共和国

## 时间

1950年1月26日至今。本页涉及现任人物的信息核验截止到2026年7月：总统为德劳帕迪·穆尔穆，总理为纳伦德拉·莫迪。

## 概括

印度宪法生效后，印度自治领转为主权、民主共和国，后来在宪法序言中明确为“社会主义、世俗、民主共和国”。国家以普选、议会内阁制、联邦结构、独立司法和基本权利为核心制度，同时继承英属印度的行政、军队、铁路和普通法框架。它在语言、宗教、种姓、地区发展差异和边疆安全之间维持统一，也经历紧急状态、族群冲突、经济转型与政党体系重组。

总统是宪法国家元首，总理及获人民院多数支持的部长会议掌握日常行政；联邦制、最高法院、选举委员会、邦政府与政党联盟共同限制并塑造中央权力。

## 统治结构

| 层级 | 职权与实际权力 |
|---|---|
| 总统 | 宪法国家元首，由国会两院和各邦议会组成的选举团选出；通常依总理和部长会议建议行使职权，在组阁、退回法案和宪政危机中有有限裁量。 |
| 总理与部长会议 | 实际行政核心；总理通常是人民院多数党或联盟领袖，内阁对人民院集体负责。 |
| 国会 | 由联邦院和人民院组成；人民院直接选举并决定政府能否继续执政，联邦院代表各邦但席位并非完全平等。 |
| 联邦 | 中央、邦和中央直辖区分掌权限；宪法列有中央、邦和共同事项，财政、紧急状态和邦长制度使中央权力较强。 |
| 司法 | 最高法院及各级法院进行宪法审查；“基本结构”原则限制议会以修宪破坏宪法核心。 |
| 选举与政党 | 印度选举委员会组织全国和邦级选举；国大党、印度人民党、左翼和众多邦级政党先后构成一党优势、联盟政治与新一轮全国性政党优势。 |

## 重要事件与发展阶段

1. **1950年宪法生效与1951—1952年首次大选**：在高文盲率和巨大人口规模下实行普选，确立议会民主的群众基础。
2. **1956年语言邦重组**：以语言为主调整邦界，把强烈地方认同纳入联邦框架；此后仍持续产生新邦。
3. **边境战争与安全国家形成**：1947—1948、1965、1971和1999年印巴冲突，1962年中印战争及长期边境争议塑造军费、核政策和外交。
4. **绿色革命与国家主导发展**：灌溉、高产品种、化肥和最低收购价提升粮食供给，也造成地区、阶层与生态差异。
5. **1971年孟加拉国战争**：印度介入东巴基斯坦危机，巴基斯坦战败，南亚地缘格局重组。
6. **1975—1977年紧急状态**：政府暂停部分自由、审查媒体并大规模拘捕反对者；1977年选举使国大党下台，显示制度受损后仍能通过选举轮替。
7. **1980年代族群与中央—地方危机**：旁遮普冲突、蓝星行动、英迪拉遇刺和反锡克暴力，以及阿萨姆、克什米尔、斯里兰卡干预等问题相互交织。
8. **1991年经济自由化**：外汇危机促使降低许可管制、开放贸易投资和改革金融；服务业及城市中产增长，同时不平等、就业和农村压力继续存在。
9. **联盟政治时期（1989—2014年）**：全国性政党通常依赖邦级伙伴，地方政党在联邦资源和中央组阁中影响显著。
10. **核国家与全球定位**：1998年核试验后遭制裁，随后加强与美国、俄罗斯、日本和“全球南方”的多向关系，同时维持战略自主。
11. **2014年后人民党优势**：中央决策更集中，数字公共基础设施、福利直达、基础建设和印度教民族主义议程并行；公民权、少数群体处境、媒体与机构自主性成为争论焦点。
12. **2019—2024年关键变化**：中央取消查谟和克什米尔原有特殊地位并改设中央直辖区；公民身份修法引发抗议；新冠疫情和农法抗议考验治理。2024年人民党未单独过半，但全国民主联盟保住多数，莫迪第三次组阁。

## 延续机制与主要张力

- **制度韧性**：定期选举、政党轮替、司法审查、联邦谈判和活跃社会组织使多元诉求有制度出口。
- **统一机制**：全印行政体系、军队、铁路、公务员、税收与数字身份加强中央整合；印地语与英语并用及语言邦制度降低了单一语言国家化的冲突。
- **结构压力**：人口与就业、城乡和邦际差距、种姓歧视、宗教多数主义、环境与水资源、边境安全持续影响政治。
- **民主争议**：紧急状态是历史上最明显的中央集权越界；当代围绕调查机构、议会程序、媒体空间和少数群体权利的争论，显示选举竞争与自由主义宪政并非自动一致。
