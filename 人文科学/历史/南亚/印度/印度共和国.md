# 印度共和国

## 时间

1950年至今

## 概括

印度共和国于1950年宪法生效后成立，是南亚最大的现代国家。它采用议会民主、联邦制和世俗共和国框架，但长期面对语言、宗教、种姓、地区差异、经济发展、印巴关系和全球大国定位等问题。

## 统治结构

| 层级 | 说明 |
|---|---|
| 国家元首 | 总统为宪法国家元首。 |
| 政府首脑 | 总理和内阁掌握实际行政权。 |
| 联邦结构 | 中央政府与邦、中央直辖区构成联邦体系。 |
| 政党政治 | 国大党、人民党和地方政党长期竞争。 |

## 重要事件

- 1950年印度宪法生效。
- 印巴战争、中印边境战争和冷战不结盟外交塑造安全环境。
- 1991年后经济自由化推动印度成为新兴经济体。
