# 吠陀时代

## 时间

约前1500年-前600年

## 概括

吠陀时代是印度-雅利安人进入和扩展、吠陀文献形成、婆罗门祭祀体系和早期种姓秩序发展的阶段。政治形态从部落联盟逐渐走向恒河流域的国家化。

## 王朝世系 / 统治结构

| 层级 | 说明 |
|---|---|
| 部落共同体 | 早期以部落、氏族和首领为核心。 |
| 婆罗门祭祀 | 祭祀和吠陀知识成为社会秩序的重要基础。 |
| 瓦尔纳秩序 | 婆罗门、刹帝利、吠舍、首陀罗的等级观念逐渐成形。 |

## 重要事件

- 《梨俱吠陀》等文献传统形成。
- 雅利安人从西北向恒河流域扩展。
- 铁器、农业和定居推动国家化。
