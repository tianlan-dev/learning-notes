# 吠陀时代

## 时间

约前1500—前600年。常分为早期吠陀阶段（约前1500—前1000年）和后期吠陀阶段（约前1000—前600年）；这些界线是根据语言、文献层次和考古文化建立的分析框架，并非当时人使用的纪年。

## 概括

吠陀时代得名于以古印度—雅利安语传承的吠陀文献。早期社群主要活动于印度河上游、旁遮普和西北河谷，经济以畜牧、农业和战利品分配相结合；后期政治与祭祀中心向恒河—亚穆纳河间及更东地区扩展，村落农业、铁器和领土国家逐步发展。印度—雅利安语人群的迁入、当地后哈拉帕及其他社群的延续与融合共同塑造这一时期，不能写成单一种族突然取代全部原居民。

## 资料与年代辨析

- 《梨俱吠陀》最古老层次以口传诗歌保存，核心部分通常早于其他三部吠陀；《娑摩吠陀》《夜柔吠陀》《阿闼婆吠陀》及梵书、森林书、早期奥义书反映更晚的祭祀与思想发展。
- 文献不是按年代编写的政治编年史。诗歌中的河流、战斗、牲畜和氏族可用于重建社会，却带有祭司、赞助者和仪式语境。
- 考古中的后哈拉帕传统、赭色陶器、彩绘灰陶、黑红陶等不能一一等同于文献中的某个“民族”；语言、物质文化和政治认同的边界不完全重合。
- 古印度—雅利安语属于印欧语系印度—伊朗语支。语言学、考古和古代DNA总体支持相关人群与草原—中亚网络的迁移和交流，但规模、路线、性别结构及与本地社会的融合仍在讨论。
- “雅利安”在吠陀文本中首先是文化、语言和身份用语，不是现代生物种族概念；近现代“入侵”或本土起源论常带政治立场，需要同证据层次区分。

## 社会与政治结构

| 层级 / 制度 | 早期形态 | 后期变化 |
|---|---|---|
| 迦那、部落与氏族 | 人群以亲族、战士集团和迁徙共同体组织，罗阇更接近受贵族支持的首领 | 部落名称逐渐转为领土国家名称，俱卢、般阇罗等较稳定王国形成 |
| 罗阇 | 负责战争、战利品与祭祀赞助，权力受贵族和集会制约 | 通过灌顶、马祭等仪式强化世袭王权，官职和贡赋扩大 |
| 沙巴、萨米提等集会 | 贵族或共同体参与议事，其组成和权限并不完全清楚 | 随领土王权和大型祭祀发展，政治影响可能下降 |
| 婆罗门 | 祭司家族保存赞歌并主持仪式 | 形成专门学派、复杂祭式和知识权威，解释祭祀与宇宙秩序 |
| 刹帝利 / 罗阇尼耶 | 战士与首领阶层 | 王权和武士身份制度化，并与婆罗门在合作与竞争中互相确立 |
| 吠舍 | 一般自由成员、牧民、农民和商人 | 承担生产与贡赋，地位逐渐同上层两瓦尔纳区分 |
| 首陀罗 | 《梨俱吠陀》晚出层次中出现 | 被置于四瓦尔纳秩序下层；实际社会远比四分模式复杂 |
| 达萨、达休 | 可指敌对者、异文化群体或依附身份 | 有些被吸收、有些与奴役关系相关，不能简单等同某一固定族群 |

四瓦尔纳是规范性分类，不等同于后来遍布各地的数千阇提，也不能据一首晚期赞歌断言整个早期社会已有完全固定的种姓制度。家庭、性别、职业、财富、地区和依附关系共同决定实际地位；女性可参与部分祭祀和诗歌传统，但后期父权、继承和礼法限制总体加强。

## 经济、聚落与技术

- 早期财富以牛群衡量，畜牧仍很重要，同时已种植大麦等作物；战争赞歌中的“夺牛”反映资源和荣誉竞争。
- 随聚落向恒河—亚穆纳河间扩展，水稻等农业、清林和村落定居增加。铁器约在前2千纪末至前1千纪初逐渐使用，但铁器不是国家化的唯一原因。
- 专业手工业、交换和贡赋增长，后期文本出现更多职业、车战、金属和土地信息。真正大规模城市与打制印记货币主要在吠陀时代末端以后发展。
- 家庭生产和村落是基本单元；国王获得自愿贡礼逐步转为较固定的巴利、巴伽等份额，为雄国时期税制提供前提。

## 宗教与思想演变

- 早期祭祀围绕火、苏摩和赞歌展开，因陀罗、阿耆尼、苏摩、伐楼那等神祇同自然、战争、誓约和宇宙秩序相关；寺庙和固定神像并非核心。
- 后期大型王祭需要多个专业祭司和大量资源，既强化王权，也提高婆罗门知识权威。祭祀功效逐渐由仪式细节和正确语言解释。
- 梵书讨论仪式对应关系，森林书和奥义书把注意力转向梵、我、业与轮回、解脱等问题。思想转向不是祭祀突然消失，而是在其内部和周边产生新的解释。
- 鲁陀罗、毗湿奴、钵阇波提等神祇地位变化，为后来的印度教传统提供要素，但把后世湿婆派、毗湿奴派完整制度直接投射到早期并不准确。
- 后期吠陀社会同非吠陀祭祀、苦行者和东部文化持续互动；佛教、耆那教与其他沙门运动既批评婆罗门祭祀，也共享业、轮回和出家等更广思想环境。

## 重要过程与转折

1. **印度—雅利安语群进入西北**：多个世代的迁徙、婚姻、战争和文化吸收，使古印度—雅利安语及诗歌传统在西北扎根，并非一次可精确断年的全面入侵。
2. **《梨俱吠陀》诗歌层累形成**：不同祭司家族保存献给诸神和赞助者的赞歌，后经严格口传组织成集。
3. **“十王之战”等部落竞争**：文本记载婆罗多首领须陀斯在河流区域击败联盟，显示早期政治由流动联盟和竞争构成；细节不能当作完整战争实录。
4. **俱卢—般阇罗中心形成**：约前1千纪前半，多个早期集团重新组合，建立较稳定的领土王权和规范化祭祀传统。
5. **向恒河平原扩展**：农业、聚落和政治中心逐步东移；这包含同本地人群的融合，不是西北文化原样复制。
6. **王权与大型祭祀强化**：王祭、马祭、灌顶和贡赋使首领向世袭君主转型，也让祭司与统治者形成互赖。
7. **瓦尔纳秩序成形**：四分社会模型在后期文本中系统化，但实际地方社群、职业与婚姻边界仍不断变化。
8. **奥义书思想与沙门环境出现**：约前1千纪中叶，内在知识、业、轮回、解脱和苦行讨论扩展，构成古印度哲学转型的一部分。
9. **雄国与第二次城市化**：约前7—前6世纪起，恒河中下游出现城镇、货币、商路和更强国家；吠陀文化不消失，而被新王国和宗教运动重组。

## 从部落联盟到国家化的机制

定居农业和人口增加扩大可征收剩余，铁器、牛耕和清林改善部分地区生产，河流交通连接不同聚落；王权通过战争、贡赋和祭祀获得资源，婆罗门学派则提供跨氏族的礼仪规范。与此同时，集会和亲族结构没有立刻消失，跋耆、末罗等地区后来仍发展出贵族共和政体。国家化因此是地区不均、王国与联盟并行的过程。

## 关键辨析

- 印度河城市体系约前1900年后分散，吠陀传统核心约数世纪后形成；二者之间存在人口和区域连续性，却没有证据支持无缝的单一文明王朝继承。
- “吠陀时代”主要覆盖北部部分区域，德干、南印度、东部和东北具有各自考古与语言历史，不能被视为无人空间。
- 《摩诃婆罗多》《罗摩衍那》含有可能追忆后期吠陀社会的要素，但现存史诗经过数百年编纂，不可直接当作某场战争或王统的同时代记录。
- 吠陀宗教是后世印度教的重要来源之一，而非与后世完全相同的成品宗教。
