# 马拉塔联盟与地方诸邦

## 时间

17世纪-19世纪

## 概括

莫卧儿帝国衰落后，印度出现马拉塔联盟、锡克王国、海得拉巴、迈索尔、孟加拉等地方力量。它们既继承地方王权和军事财政结构，也在与英国东印度公司竞争中逐渐被纳入殖民体系。

## 王朝世系 / 统治结构

| 力量 | 说明 |
|---|---|
| 马拉塔联盟 | 源于德干，18世纪成为印度最重要本土军事政治力量之一。 |
| 锡克王国 | 旁遮普地区兴起，兰季特·辛格时期强盛。 |
| 海得拉巴、迈索尔等 | 地方诸邦在莫卧儿衰落和英国扩张之间维持自治或抵抗。 |
| 英国东印度公司 | 逐步通过战争、条约和财政控制扩大统治。 |

## 重要事件

- 马拉塔扩张削弱莫卧儿权威。
- 普拉西战役后英国控制孟加拉。
- 盎格鲁-马拉塔战争、盎格鲁-迈索尔战争推动英国霸权形成。
