# 巽伽、贵霜与笈多前诸政权

## 时间

前2世纪-4世纪

## 概括

孔雀王朝瓦解后，印度进入多区域政权并立阶段。巽伽、安得罗 / 百乘、印度-希腊、塞种、帕提亚和贵霜等力量在北印度、德干和西北印度交错，连接印度、中亚和伊朗世界。

## 王朝世系 / 统治结构

| 政权 / 力量 | 时间 | 说明 |
|---|---|---|
| 巽伽王朝 | 前2-前1世纪 | 在恒河流域继承部分孔雀遗产。 |
| 印度-希腊与塞种 | 前2世纪以后 | 西北印度希腊化和中亚游牧因素进入。 |
| 贵霜帝国 | 1-3世纪 | 连接中亚、犍陀罗和北印度，佛教艺术和丝路交流兴盛。 |
| 百乘 / 安得罗 | 约前1世纪-3世纪 | 德干和南印度重要政权。 |

## 重要事件

- 犍陀罗艺术和佛教传播发展。
- 贵霜帝国连接中亚、印度和中国西域。
- 北印度最终进入笈多王朝整合。
