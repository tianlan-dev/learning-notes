# 英属印度

## 范围与对象

本页是南亚共享殖民史主笔记。“英属印度”包括东印度公司和后来的英王政府在次大陆建立的殖民体系，既有英国直接统治的省，也有受条约、驻扎和宗主权约束的五百余个土邦；它不等同于今日印度共和国。印度、巴基斯坦和孟加拉国目录分别维护本地殖民经历与分治后的国家主线。

## 时间

1757—1947年；公司逐步取得领土与财政主权的关键起点为1757年普拉西战役和1765年孟加拉税收权，英王直接统治为1858—1947年。

## 概括

东印度公司原是享有王室特许的商业公司，凭借海运、孟加拉财政、训练军队、印度士兵和与本地政治集团结盟，逐步转为领土国家。1857年大起义后，英国议会取消公司统治，由英王、印度事务大臣和总督—副王体系直接控制。殖民政权重塑土地税、法律、军队、铁路、教育和印度洋贸易，也造成财富外流、产业与农业结构失衡、饥荒和种族等级。与此同时，殖民国家提供的全印行政空间、印刷与交通网络被不同社会运动和民族主义者反向利用，最终形成多条相互竞争的独立方案。

## 建立与扩张机制

- **孟加拉财政基础**：1757年公司在普拉西扶植孟加拉纳瓦布，1764年布克萨尔战役击败孟加拉、阿瓦德和莫卧儿皇帝联军，1765年取得孟加拉、比哈尔和奥里萨的税收权。当地收入由此供养公司军队和后续战争。
- **军事—财政循环**：公司以印度士兵为军队主体，用征税和借贷支付军费，再以战争扩大税源；欧洲火器训练有优势，但胜利也依赖印度盟友、情报和对手内部冲突。
- **条约与间接统治**：辅助同盟迫使土邦供养英军并放弃独立外交；“无嗣失权”等政策和“治理不善”借口则用于兼并。许多王公保留王位，却失去对外主权和部分内政自主。
- **英国国家监管**：1773年规制法、1784年印度法和历次特许状法逐步把公司置于议会监督下；1833年以后印度总督权力扩展至整个公司领地。
- **海权与全球资源**：英国舰队、伦敦资本、工业制品和跨殖民地兵力使公司能够同迈索尔、马拉塔、锡克等强国进行长期战争。

## 统治结构

| 层级 / 类型 | 权力关系 |
|---|---|
| 伦敦决策 | 公司时期由董事会与英国政府监督机构并存；1858年后印度事务大臣及其委员会代表英国内阁掌握最高殖民决策。 |
| 印度总督 / 副王 | 总督主持中央政府、军政和立法；1858年后又以“副王”身份代表英王处理土邦关系。完整任职序列见[印度总督与副王表](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E5%8D%97%E4%BA%9A/%E5%8D%B0%E5%BA%A6/%E5%8D%B0%E5%BA%A6%E6%80%BB%E7%9D%A3%E4%B8%8E%E5%89%AF%E7%8E%8B%E8%A1%A8.md)。 |
| 英属省 | 由总督、省督或专员及印度文官系统直接治理；省级自治经1909、1919、1935年改革逐步扩大，但总督保留否决和紧急权。 |
| 土邦 | 王公名义内治，英国驻扎官监督继承、外交和重大政策；土邦不是完全主权国家，也不属于普通英属省。 |
| 军队与警察 | 军官高层长期由英国人垄断，士兵多数来自南亚；1857年后调整招募地区和族群比例，以防再次联合兵变。 |
| 地方中介 | 地主、税吏、商人、村社精英和受教育专业阶层参与征税、司法与市政；殖民统治从未只靠少量英国官员直接覆盖全部人口。 |

## 土地、经济与社会机制

- 孟加拉永久地税制把柴明达尔塑造成固定纳税地主；马德拉斯、孟买的莱特瓦尔制直接向耕作者定额；北印度部分地区采用村社或地主共同负责的马哈尔瓦尔制。三者均因高额、僵化评估和市场波动影响农民。
- 铁路、电报、港口和统一货币首先服务军事调动、原料出口和殖民市场，也降低跨区交通成本，后来被商人、移民与民族运动使用。
- 印度被纳入棉花、黄麻、茶、鸦片和小麦等全球商品链；部分港市工业和商业增长，但英国工业品、关税政策和利润汇回造成不对称发展。
- 19世纪多次严重饥荒与季风、作物和疾病相关，殖民税制、出口导向、粮价、救济理念和行政迟缓显著放大死亡。
- 英语教育、大学、法学和印刷培养新专业阶层；社会改革者围绕寡妇再婚、萨蒂、教育、种姓和宗教身份展开不同改革，不能只视作殖民政府单向“现代化”。

## 重要事件与阶段过程

1. **1757年普拉西战役**：公司将领克莱武联合孟加拉宫廷反对派击败西拉杰·乌德·道拉，开始控制纳瓦布任免与财政。
2. **1764—1765年布克萨尔与税收权**：公司获得稳定收入基础，从商站势力转成领土统治者；早期“双重统治”导致责任和征收分离。
3. **迈索尔、马拉塔与锡克战争**：18世纪末至1849年，公司经多轮战争消灭或降服最主要区域对手，旁遮普被兼并后次大陆大部进入英国宗主体系。
4. **1793年永久地税与19世纪法律重组**：殖民政府试图使收入可预测，建立法院、警察和成文法；种族化官僚仍把高级职位主要留给欧洲人。
5. **1857—1859年大起义**：步枪弹传闻触发士兵兵变，德里、阿瓦德和中印度的王公、地主、农民等以不同目标参战；南部和旁遮普部分力量支持或未加入起义。英国镇压后实施大规模报复。
6. **1858年政权转移**：英王接管，公司解散统治职能；殖民政府承诺尊重土邦和宗教习俗，实际加强军队、情报与种族边界。
7. **1885年印度国民大会党成立**：早期以请愿、预算批评和扩大印度人参政为主；经济民族主义提出“财富外流”批判。
8. **1905年孟加拉分治与反分治运动**：抵制英货、国货运动和群众动员扩大；宗教、阶级与地区政治也随代表权竞争重新组合。1911年分治撤销。
9. **第一次世界大战与1919年危机**：印度提供军队、税收和物资；战后罗拉特法延长强制措施，阿姆利则惨案摧毁改革承诺的信誉。甘地领导的不合作运动把民族主义推向群众政治。
10. **1930年代公民抗命与宪制改革**：食盐进军象征挑战殖民垄断；1935年印度政府法建立省自治和计划中的联邦，1937年选举使印度政党取得实际执政经验。
11. **第二次世界大战与退出印度运动**：副王未征得印度领导人同意即宣布参战；国大党领导层1942年被拘捕，苏巴斯·钱德拉·鲍斯组织印度国民军，穆斯林联盟则扩展组织并推动巴基斯坦方案。
12. **1946—1947年权力移交**：皇家印度海军兵变、战后经济压力和英国国内政治促使加速撤离；内阁使团联邦方案失败，国大党与穆斯林联盟围绕中央权力和代表权无法妥协。蒙巴顿方案接受分治，仓促划界伴随大规模迁徙和暴力。

## 殖民体系的维系条件

殖民统治依靠的不只是英国军队。孟加拉和后来的全印税收、印度士兵、土邦合作、地主与商人中介、海权和伦敦信用共同降低了统治成本。殖民政府又以人口普查、法律分类和差异化代表制度固定某些宗教、种姓与“尚武族群”边界，使社会身份成为行政资源。这些合作关系会随战争、税负和政治机会改变，不能把殖民时期简化成英国人与统一印度社会的二元对立。

## 危机、撤离与终结

- **结构矛盾**：殖民国家宣称法治与代议改革，却保持种族等级和最终否决权；经济增长与基础设施并未消除财富外流、土地压力和饥荒责任。
- **政治压力**：国大党、穆斯林联盟、革命组织、工农运动、达利特政治、地方党派及土邦改革力量提出不同未来方案。群众不合作使维持秩序的行政和声誉成本上升。
- **国际压力**：两次世界大战消耗英国财政，反殖民规范扩散，美国和苏联崛起；战后印度军队、警察和文官是否会长期服从也不再确定。
- **直接过程**：1946年政治谈判失败和宗派暴力使统一联邦方案崩溃。英国选择在1947年提前移交权力，以印度和巴基斯坦两个自治领取代英属印度；殖民统治终结，但边界、军队、资产和土邦归属问题立即成为新国家危机。

## 历史影响与辨析

英属印度建立了跨次大陆的官僚、法律、铁路和军事框架，却以殖民提取和政治不平等为目标运行。“现代化”成果与掠夺、饥荒和压制并存，不能相互抵消。1947年的印度、巴基斯坦和后来的孟加拉国都继承部分殖民机构；任何一国都不能把整个英属印度历史视为自身独占前史。
