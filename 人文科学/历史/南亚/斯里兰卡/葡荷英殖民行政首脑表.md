# 葡荷英殖民行政首脑表

## 统治结构

```mermaid
flowchart LR
    A["葡萄牙堡垒与科特保护关系"] --> B["1597 葡萄牙王室属地"]
    B --> C["1658 荷兰东印度公司海岸殖民地"]
    C --> D["1795—1796 英国军事占领"]
    D --> E["1798 英属锡兰王室殖民地"]
    E --> F["1815 吞并康提后全岛殖民行政"]
    F --> G["1948 锡兰自治领"]
```

- 葡萄牙早期首脑只是科伦坡堡垒或海岸军事据点长官，随后升为总长官、总督／总司令；1597 年科特 Dharmapala 遗赠之后，葡萄牙才主张对科特领地的直接王室统治。
- 荷属锡兰由荷兰东印度公司治理，总督受巴达维亚总督及政务会节制，控制海岸堡垒、肉桂垄断、司法和税收；康提仍是独立王国，双方以条约、战争和边境贸易相互制衡。
- 英国 1795—1796 年先以战争占领荷兰据点，初期军事长官与东印度公司驻地监督并行；1798 年改为王室殖民地，1815 年吞并康提后，总督成为全岛最高行政首脑。1833 年以后立法、行政机关逐渐制度化，但总督直到 1948 年前仍代表英国王权掌握最终行政权。
- 同一人多次任职、临时代理或在不同制度下续任均分别列行；“任期条目”不等于不同人物数。
