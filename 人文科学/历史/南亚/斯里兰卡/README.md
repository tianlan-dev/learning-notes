# 斯里兰卡历史

斯里兰卡历史由季风水利、佛教王权、印度洋贸易、南印度联系以及僧伽罗、泰米尔、穆斯林和种植园社群的互动共同塑造。古代阿努拉德普勒—波隆纳鲁沃国家以水库、寺院和王室网络整合北中部；13世纪后贾夫纳与西南、山地诸国并立。葡萄牙、荷兰先后控制海岸，英国于1815年吞并康提并建立全岛行政。1948年独立后，国籍、语言和权力分配冲突演变为内战；2009年军事终战后，和解、强总统制与2022年经济危机继续重塑国家。

## 历史主线

- 古代国家形成不能只按《大史》王统理解：考古证明早期城市化和灌溉发展，最早几代王名与精确年代仍属传统编年。
- 所谓“统一王国”常与鲁胡纳、Vanni、贾夫纳和地方首领并存；13世纪后更应按多中心王权理解。
- 欧洲殖民是堡垒、贸易垄断、地方联盟与逐步吞并的过程。1815年前，任何欧洲政权都未稳定统治全岛。
- 现代冲突既有殖民制度遗产，也有独立后国籍、语言、教育、宪法与安全政策的累积作用，不能简化为古代族群仇恨。

## 重要转折与时间节点

| 时间 | 事件 | 意义 |
|---|---|---|
| 约前3世纪 | 佛教传入传统与王室护持 | 上座部佛教王权形成 |
| 993—1017年 | 朱罗分阶段征服 | 阿努拉德普勒失去政治中心地位 |
| 1070年前后 | Vijayabahu I复国 | 波隆纳鲁沃时代开始 |
