# 不丹的宗萨、英属印度与保护关系

## 时间

17世纪中叶—1949年

## 概括

夏仲阿旺·朗杰建立的体制以宗教首领和世俗首领分掌权力，宗萨作为地方行政军事中心。18—19世纪内部分裂和与西藏、阿萨姆的边境冲突加剧；英国为控制杜阿尔低地与贸易通道，发动战争并通过条约取得南部领土。1907年旺楚克王朝建立。

## 主要政权与政治阶段

| 政权或阶段 | 时间 | 特征 |
|---|---|---|
| 夏仲政教双重体制 | 17世纪 | 法王与世俗执政者共同治理，宗萨网络支撑国家 |
| 内部分裂与地区压力 | 18—19世纪 | 地方总督、教派和外部势力竞争 |
| 英不战争与条约 | 1864—1865年 | 英国夺取杜阿尔低地，规定边界与补贴 |
| 旺楚克王朝 | 1907年起 | 乌颜·旺楚克被推为首任世袭国王 |

## 重要事件

- 17世纪夏仲建立廷布、普那卡等宗萨体系。
- 18世纪后夏仲去世引发继承和地方权力竞争。
- 1864—1865年英不战争后签订《辛楚拉条约》，不丹割让杜阿尔。
- 1907年各宗教与世俗精英推举乌颜·旺楚克为国王。
- 1910年《普那卡条约》规定英国在不丹对外事务上提供“指导”，不丹保留内政。
- 1949年印不条约取代英国安排，确认友好与安全关系。
