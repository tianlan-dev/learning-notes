# 不丹的早期山地社会与佛教国家形成

## 时间

有文字记载以前—17世纪中叶。早期年代多来自寺院传记、伏藏文献和后世国家叙事，历史人物活动、宗教记忆与神圣地理需要分层理解。

## 概括

今日不丹地区由喜马拉雅东部多个河谷、山口和南部丘陵组成，早期并无统一“不丹王国”。居民使用藏缅语族多种语言，以河谷农业、放牧、森林资源和跨山贸易维生。佛教约在7—8世纪后经西藏、印度和喜马拉雅网络逐步传播，宁玛、竹巴噶举等传统同地方神灵、首领和寺院结合。13—17世纪，各河谷的宗教家族、地方头人和宗萨相互竞争。1616年夏仲阿旺·朗杰自西藏到来，以竹巴噶举合法性、堡垒网络和军事联盟整合西部与中部河谷，才形成可延续的国家框架。

国家形成不是佛教“传入”后的自动结果，而是宗教合法性、宗堡军政中心、税役、河谷粮食、跨山贸易和对外战争共同作用。早期人物与年代多来自后编宗教传记，需标明约年和传统说法。

## 资料与范围辨析

- 不丹湿润气候、木石建筑和有限考古发掘使早期物质证据较少；寺院史传是重要资料，却以弘法、伏藏和神迹为叙事目的。
- 传统把藏王松赞干布在7世纪建造基楚寺、降巴寺同佛教初传相连，现存建筑经历多次重修，具体建造层次不能只依传说确定。
- 莲花生在8世纪进入布姆唐、降伏地方神灵的故事是不丹宗教认同核心；其中可保存跨喜马拉雅传播记忆，但行程细节难用同时代文献逐项验证。
- “洛门”“门域”“南部山谷”等旧称范围变化，不等同现代国界；现代“不丹人”身份是在国家整合后逐步形成。
- 早期地方信仰常被笼统称为苯教或泛灵信仰，实际包括山神、水神、祖先和地方仪式，后来许多被佛教仪轨吸收。

## 社会、经济与地方权力

| 层级 / 网络 | 作用 |

| 河谷聚落 | 水稻、荞麦、大麦等农业同畜牧结合；山脊和河谷限制交通，使地方语言和身份多样。 |
| 跨山商路 | 盐、羊毛、谷物、布、金属、药材等在西藏、阿萨姆、杜阿尔与不丹河谷间交换。 |

| 寺院与喇嘛家族 | 保存文字和仪式，接受土地与劳役，调解争端；一些教派领袖同时掌握堡垒和军事资源。 |
| 宗 / 宗萨 | 兼具堡垒、行政中心、寺院、粮仓和避难所功能；大型宗网络在17世纪国家整合中制度化。 |

## 佛教传播与教派格局

- 宁玛传统通过莲花生崇拜、伏藏师和地方寺院深入布姆唐等地，常把地方神灵转化为护法。
- 13世纪竹巴噶举上师帕久·珠贡西波进入西部不丹，其弟子和后裔建立寺院、庄园及教法网络，为后来夏仲整合提供宗教基础。
- 拉巴噶举、宁玛、竹巴及其他教派并立，教派竞争同时涉及土地、转世、寺院与跨西藏政治联盟，不能视为纯粹教义争论。
- 南部和东部社群同阿萨姆、阿鲁纳恰尔及孟加拉平原保持联系，国家形成并非只有“西藏文化南传”一条路线。

## 重要过程与事件

1. **早期山地定居与贸易**：河谷农业和季节放牧形成地方社会，喜马拉雅山口把高原盐毛贸易同南部稻米、棉布和金属连接。
2. **7—8世纪佛教记忆形成**：松赞干布寺院和莲花生入境传统把不丹纳入吐蕃—印度佛教神圣地理。
3. **中世纪寺院网络扩大**：不同噶举支派和宁玛伏藏传统建立寺院、收徒和土地关系，地方首领借宗教赞助增强合法性。
4. **帕久·珠贡西波活动（13世纪）**：竹巴噶举在西部河谷扎根，其宗教家族逐渐成为重要政治力量。
5. **西藏教派政治外溢**：16—17世纪藏巴汗支持噶玛噶举，竹巴教主转世继承发生争议，阿旺·朗杰面临对手追索圣物与地位。
6. **1616年夏仲入不丹**：阿旺·朗杰离开热龙寺进入不丹，获得部分竹巴寺院和地方领主支持，也遭敌对教派反抗。
7. **宗萨建设**：1629年辛托卡宗及其后普那卡、旺迪颇章、通萨等宗成为军政、宗教和粮储节点，使跨河谷统治可持续。
8. **1634年前后统一战争**：夏仲联盟在“五喇嘛之战”等冲突中击败西藏支持的对手，巩固西部控制；此后仍多次抵抗藏军入侵。
9. **政教二元体制建立**：夏仲位居最高宗教政治权威，下设杰堪布主持中央僧团、竹第悉处理世俗行政，并以地方宗本执行治理。
10. **国家法与认同**：传统法典“扎依”规范官员、宗教、税役和行为；共同服饰、宗堡与竹巴宗教逐步塑造“竹域”国家认同。
11. **1651年夏仲去世后的隐瞒**：为防政局崩溃，其死讯被隐瞒多年；后来出现身、语、意多个转世系统，最高权威分散，为地方竞争埋下伏笔。

## 国家形成机制

夏仲的成功并非单靠宗教声望。他把转世合法性、地方竹巴家族、宗萨工程、税粮与民兵组织结合起来，又以抵抗西藏军队强化共同政治边界。二元体制允许僧团与世俗官员分工，但夏仲本人去世后，缺少同等权威的单一继承者，地方宗本和地区总督逐步取得自主权。

## 关键辨析

- 不丹国家形成是多河谷联盟和军事行政整合，不是一个古代统一王朝自然延续。
- 佛教传播包含本地吸收与再创造，不能把前佛教信仰简单称为被“消灭”的迷信。
- 宗萨并非只有寺院或城堡功能，而是国家征税、司法、仓储、仪式和防御的综合节点。
- 夏仲政权确立了现代不丹的重要制度祖型，但今日国界、民族认同和世袭王朝在其后数百年才逐步固定。
