# 不丹历史

不丹历史以高山河谷、藏传佛教传统和地方宗萨网络为基础。17世纪夏仲阿旺·朗杰建立政教双重体制，近代旺楚克王朝在英国与西藏、印度之间保持自治。20世纪后不丹从君主主导的渐进改革转向宪政民主。

## 历史坐标

- 地理范围：喜马拉雅东部山地、布拉马普特拉河谷北缘。
- 国家形成：现代边界、行政语言和制度多受殖民、分治或山地—海洋交通网络影响，不能直接等同于古代政权。
- 区域联系：[南亚历史](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E5%8D%97%E4%BA%9A/README.md)；跨区域背景见[英属印度、分治与现代南亚](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E5%8D%97%E4%BA%9A/%E8%8B%B1%E5%B1%9E%E5%8D%B0%E5%BA%A6%E3%80%81%E5%88%86%E6%B2%BB%E4%B8%8E%E7%8E%B0%E4%BB%A3%E5%8D%97%E4%BA%9A.md)。
