# 孟加拉国历史

孟加拉国的历史以水网三角洲、稻作和河海贸易为基础。古代孟加拉先后受孔雀、笈多、帕拉和塞纳王朝影响；中世纪孟加拉苏丹国、莫卧儿与地方地主共同推动三角洲开发。英国殖民、孟加拉分区和1947年东巴基斯坦地位，最终在1971年独立战争中转化为孟加拉国。

## 历史坐标

- 地理范围：恒河—布拉马普特拉—梅克纳河三角洲及孟加拉湾北岸。
- 国家形成：现代边界、行政语言和制度多受殖民、分治或山地—海洋交通网络影响，不能直接等同于古代政权。
- 区域联系：[南亚历史](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E5%8D%97%E4%BA%9A/README.md)；跨区域背景见[英属印度、分治与现代南亚](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E5%8D%97%E4%BA%9A/_%E9%80%9A%E5%8F%B2/%E8%8B%B1%E5%B1%9E%E5%8D%B0%E5%BA%A6%E3%80%81%E5%88%86%E6%B2%BB%E4%B8%8E%E7%8E%B0%E4%BB%A3%E5%8D%97%E4%BA%9A.md)。
