# 孟加拉国的古代孟加拉、帕拉与伊斯兰苏丹国

## 时间

古代—1576年

## 概括

孟加拉三角洲不是一个边界固定的古代国家。恒河、布拉马普特拉河和梅克纳河不断改道，政治中心、港口和垦殖前沿也随之移动；普恩德拉、瓦迦、乔荼、罗陀、萨摩塔塔等名称首先指向不同区域，不能直接等同于现代孟加拉国。8世纪帕拉王朝从伐连陀罗兴起，以孟加拉—比哈尔为核心参与北印度争霸，并支持佛教大寺院网络；11—13世纪塞纳王朝利用帕拉衰落建立婆罗门教王权。13世纪突厥军事集团进入西北孟加拉，14世纪独立苏丹最终把拉赫瑙蒂、萨特冈和孙那罗迦等中心置于同一王权下。1576年卡拉尼末主战败后，独立苏丹国终结，但莫卧儿对东部水乡的实际征服仍持续了数十年。

孟加拉各区域并非始终由一个中心统治。王朝更替在西部乔荼、北部奔那和东南三角洲常不同步，表中的统一年代需结合地方延续与并立理解。

## 地理与早期政治背景

- **区域多中心**：普恩德拉大致位于北部，瓦迦和萨摩塔塔位于东部与东南部，乔荼常与西部和西北部相连；同一名称在不同时代的范围会变化。
- **帝国统治并不均质**：孔雀、笈多等北印度帝国曾把孟加拉部分地区纳入税赋、道路和官僚体系，但河口、森林与海岸地带经常由地方王族、部落首领和港口网络维持较强自主性。
- **河海交通决定国家能力**：季风、河道和船运使稻米、盐、布匹及海外商品得以流通，也增加了中央军队进入三角洲腹地的成本。控制渡口、堤岸、港口和可征税耕地，往往比划定连续陆地边界更重要。
- **宗教传统并存**：佛教、婆罗门教和地方神祇崇拜长期共存。13世纪以后伊斯兰教通过宫廷、驻军、商人、苏菲网络、新垦殖聚落和地方社会关系逐步扩展，不能归结为一次征服或单一强制改宗。

## 帕拉王朝：建立、扩张与中兴

### 建立背景与崛起机制

约8世纪中叶，后笈多时代的孟加拉长期缺乏稳定霸权。铭文称地方权贵为终止“强者吞食弱者”的混乱而拥立瞿波罗一世。这里的“推举”更接近武装封臣和区域精英的协议，不是全民选举。帕拉王权以伐连陀罗为重要基地，通过授地、婚姻、封臣效忠和机动军队扩大控制；“波罗”是统治者名号的一部分，王朝整体以此得名。

达摩波罗和提婆波罗把势力推进比哈尔与恒河中游。帕拉、瞿折罗—普腊蒂哈罗和罗湿陀罗拘陀围绕曲女城及北印度政治威望展开长期“三方争霸”。帕拉的直接行政区主要在孟加拉—比哈尔，远方诸王朝在颂文中被说成臣服，并不等于长期驻军和直接征税。王朝的强盛来自三角洲农业、恒河交通、战象与骑兵、封臣动员，以及佛教寺院和婆罗门受地者共同构成的地方网络。

### 统治结构与文化网络

| 层级 | 作用 | 局限 |
|---|---|---|
| 国王与王族 | 任命高官、发动战争、颁发土地与宗教施舍 | 继承秩序不总是长子制，旁支和兄弟可能争位 |
| 封臣与地方首领 | 提供军队、管理地方、参与拥立与叛乱 | 中央衰弱时可转为独立王权 |
| 官僚与税收单位 | 管理村落、地区和土地收入，维持交通与军需 | 河道变化和边疆开垦使实际控制不均 |
| 寺院与受地社群 | 经营土地、教育、抄经并连接商旅和学术网络 | 宗教施舍也会分散中央可直接支配的收入 |

帕拉诸王保护那烂陀、超戒寺、欧丹多富梨等东印度佛教中心，并在今孟加拉国帕哈尔普尔兴建大型苏摩普利大寺。僧侣、经卷和艺术风格经尼泊尔、西藏与东南亚传播。王朝虽以佛教保护者著称，仍向婆罗门和印度教机构授地；把帕拉时代描述为单一宗教国家并不准确。

### 衰落、中兴与终结

提婆波罗死后，摩醯因陀罗波罗、苏罗波罗一世和毗伽罗诃波罗一世的次序及并立关系存在争议，反映王位从瞿波罗直系向旁支转移。普腊蒂哈罗、钱德拉、甘波阇等势力乘机扩张，帕拉缩回核心地区。摩醯波罗一世在约10世纪末收复伐连陀罗和部分孟加拉，形成中兴；但后续诸王又受到卡拉丘里、朱罗远征及地方封臣的压力。

摩醯波罗二世囚禁兄弟并与封臣冲突，凯瓦尔塔首领迪维耶在伐连陀罗起兵，建立地方统治。罗摩波罗依靠一批封臣和亲族击败迪维耶的后继者毗摩，重新控制北孟加拉；这次复兴本身也说明中央必须以让利换取地方军事支持。罗摩波罗死后，阿萨姆等外围代理人独立，塞纳势力从西南和东南扩张。摩檀波罗是最后一位有明确帝王铭文的帕拉君主；其后戈文陀波罗等是否属于同一王统存在争议。

## 塞纳王朝：区域统一与东南残存

塞纳家族自称来自南方“羯罗拏吒”，但其迁入时间、族源和早期身份仍有争论。萨曼塔塞纳是可考祖先，赫曼塔塞纳可能是帕拉体系内的地方首领；真正建立独立王朝的是毗阇耶塞纳。他利用帕拉内乱，先后压制西部、北部与东南部对手，使孟加拉大部分区域短期处于同一王权之下。

塞纳政权以婆罗门教王权、土地施舍和梵文宫廷文化著称。跋罗罗塞纳常被后世传统与“俱利那”社会等级制度相连，但具体制度是否完全由一位君主创设，不能仅凭后世谱牒断定。罗叉摩那塞纳时期文学和宗教赞助兴盛，宫廷却面临地方首领分离和西北防御不足。

约1204年，伊赫蒂亚尔丁·穆罕默德·巴赫蒂亚尔·卡尔吉以小规模骑兵快速进入纳迪亚，罗叉摩那塞纳撤往东南。突袭摧毁了西部政治中心的威信，却没有在一天内征服整个孟加拉。罗叉摩那塞纳及其子毗湿缚鲁波塞纳、吉舍婆塞纳仍在毗讫罗摩补罗和瓦迦一带颁发文书，塞纳后裔可能延续至约1245年；西北突厥政权、东南塞纳残余和其他地方王权曾长期并立。

## 从突厥征服到独立孟加拉苏丹国

### 13—14世纪的政权形成

巴赫蒂亚尔的征服建立了拉赫瑙蒂的突厥军政据点。此后一个多世纪中，孟加拉有时名义臣属德里，有时由总督、军人和地方集团独立；德里很难持续控制东部河网。1338年孙那罗迦的法赫鲁丁·穆巴拉克沙阿自立，拉赫瑙蒂、萨特冈和孙那罗迦出现并立苏丹。

沙姆斯丁·伊利亚斯沙阿先取得萨特冈与拉赫瑙蒂，1352年击败孙那罗迦末主，建立覆盖孟加拉主要区域的独立王权。他和继承者抵御德里苏丹的远征，使用“孟加拉苏丹”等称号，使“孟加拉”逐渐成为统一政治空间的名称。首都在潘杜阿、高尔等地转移，地方则由省区、军政长官、城市官员、土地控制者与宗教机构共同治理。

### 发展阶段与鼎盛条件

| 阶段 | 时间 | 主要特征 |
|---|---|---|
| 第一伊利亚斯沙希 | 1342—约1415年 | 统一三大中心、抵御德里、发展铸币和城市建筑 |
| 罗阇·迦内沙家族 | 约1415—1435年 | 地方大地主掌权，改宗、继承与宫廷合法性重新组合 |
| 复辟伊利亚斯沙希 | 1435/1436—1487年 | 王统恢复，地方精英和宫廷卫队影响增强 |
| 阿比西尼亚人苏丹 | 1487—1493/1494年 | 宫廷卫队频繁废立，财政与军权斗争激化 |
| 胡赛因沙希 | 1493/1494—1538年 | 政局相对稳定，领土扩张、孟加拉语文化和跨宗教官僚合作发展 |
| 苏尔、穆罕默德沙希与卡拉尼 | 1538—1576年 | 阿富汗政权重组，在莫卧儿东进压力下走向终结 |

苏丹国的繁荣来自可铸银货币、稻作开垦、棉纺织、河海贸易和多个城市中心。清真寺、陵墓、道路与铸币把王权带入新聚落；苏菲、地方地主、农民和市场网络共同参与森林边缘的开发。伊斯兰人口在东部三角洲的增长与新垦殖、地方宗教网络和政治资源分配有关，不宜用“征服后立即改宗”解释。胡赛因沙希时期广泛任用不同社群的官员，并支持孟加拉语文学，是苏丹国整合能力较强的阶段。

### 衰落因素与直接灭亡过程

苏丹国的衰落不是一条直线。结构上，地方军政长官和大地主拥有独立资源，王位又缺乏稳定继承规则；宫廷卫队和宰相可以废立苏丹。外部方面，北印度的阿富汗苏尔势力和莫卧儿帝国先后东进，奥里萨、阿萨姆、特里普拉与阿拉干也持续牵制边疆。直接触发则是1530年代胡赛因沙希宫廷分裂，末帝吉亚斯丁·马哈茂德无力阻止舍尔汗控制高尔。

苏尔帝国瓦解后，孟加拉总督再度自立，随后卡拉尼家族取得政权。苏莱曼·卡拉尼名义承认阿克巴而保持自治；达乌德·卡拉尼则公开对抗莫卧儿。1575年图卡罗伊战役后达成的安排很快破裂，1576年7月12日拉杰马哈尔战役中达乌德败亡，独立苏丹国的国家层面终结。此后东部“十二地主”等集团仍继续抵抗，莫卧儿直到17世纪初才把三角洲腹地纳入较稳定的省级统治。

## 重要事件

| 时间 | 事件 | 过程与意义 |
|---|---|---|
| 约750年 | 瞿波罗一世受拥立 | 地方权贵以建立秩序为目标形成帕拉王权。 |
| 8—9世纪 | 北印度三方争霸 | 达摩波罗、提婆波罗把帕拉带入帝国高峰，但远方影响并非持续直辖。 |
| 约1070年代 | 凯瓦尔塔起义 | 地方社会和封臣击败摩醯波罗二世，暴露帕拉统治的结构性脆弱。 |
| 约1097—1160年 | 毗阇耶塞纳扩张 | 塞纳利用帕拉衰落，整合孟加拉大部。 |
| 约1204年 | 巴赫蒂亚尔突袭纳迪亚 | 西部王权中心崩溃；塞纳政权转入东南延续，并非全境同时易手。 |
| 1338—1352年 | 独立苏丹并立与统一 | 法赫鲁丁首先自立，伊利亚斯沙阿随后统一拉赫瑙蒂、萨特冈和孙那罗迦。 |
| 1353—1359年 | 抵御德里远征 | 伊利亚斯父子依托三角洲地形和坚城保持独立。 |
| 1410—1430年代 | 罗阇·迦内沙家族掌权 | 地方地主、宗教身份和苏丹合法性重新组合，王统一度中断。 |
| 1493/1494—1519年 | 胡赛因沙阿统治 | 结束卫队政变期，推动扩张、城市经济和多社群官僚合作。 |
| 1538年 | 舍尔汗占领高尔 | 胡赛因沙希王朝灭亡，孟加拉进入阿富汗诸政权阶段。 |
| 1576年 | 拉杰马哈尔战役 | 达乌德·卡拉尼败亡，莫卧儿在名义上吞并孟加拉。 |

## 演变关系

- 前期背景：普恩德拉、瓦迦、乔荼和萨摩塔塔等区域王权，并非现代国界内的单一国家。
- 帕拉衰落后，塞纳、钱德拉等地方势力并起；塞纳失去西北中心后仍在东南延续。
