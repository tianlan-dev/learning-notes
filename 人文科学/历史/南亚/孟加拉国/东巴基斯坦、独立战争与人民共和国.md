# 孟加拉国的东巴基斯坦、独立战争与人民共和国

## 时间

1947年至今

## 概括

东巴基斯坦人口占巴基斯坦多数，却在中央政府、军队、财政和语言政策中处于不利地位。1952年语言运动以孟加拉语地位为核心，1970年选举危机和1971年军队镇压促成独立战争；此后国家经历一党化、军人统治、议会民主和发展转型。

## 主要政权与政治阶段

| 政权或阶段 | 时间 | 特征 |
|---|---|---|
| 东巴基斯坦与语言运动 | 1947—1969年 | 乌尔都语国策引发孟加拉语抗争，地区民族主义形成 |
| 选举危机与独立战争 | 1970—1971年 | 人民联盟胜选未获交权，战争与印度介入后建国 |
| 建国、政变与军政 | 1972—1990年 | 议会宪法、一党化、1975年政变和军人统治 |
| 多党共和国 | 1991年至今 | 议会民主、出口制造与政治两极化 |

## 重要事件

- 1952年2月21日达卡学生为孟加拉语地位示威并遭枪击。
- 1970年大选中人民联盟在东巴取得压倒性胜利，却未能组阁。
- 1971年3月巴基斯坦军队发动镇压，孟加拉独立战争开始。
- 1971年12月巴军投降，孟加拉国独立。
- 1975年穆吉布·拉赫曼遇刺，随后出现长期军政。
- 1991年宪法修正恢复议会制。
