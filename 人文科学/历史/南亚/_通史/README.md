# 南亚通史

## 概括

本目录整理跨越印度次大陆、喜马拉雅山地、印度洋岛屿和现代国家边界的共同历史专题。各国通史仍在对应国家目录维护；这里突出文明与宗教传统、海陆网络、跨国帝国、殖民统治、分治与建国。

## 职责边界

- 本目录负责区域共同史、跨境机制和多国共享的前史，不把古代文明或跨国帝国倒推为现代单一国家的专属历史。
- 印度、巴基斯坦、孟加拉国等国家 README 负责从本国现有地域、国家形成和现代政治的角度承接共同史。
- 同一对象只指定一篇规范主笔记；区域专题给出比较框架，国家页通过链接补充本地事实，不重复维护另一套完整通史。
