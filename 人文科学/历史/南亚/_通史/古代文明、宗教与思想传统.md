# 南亚古代文明、宗教与思想传统

## 时间

约前2600年—公元1千纪中期；若追溯聚落起源，可上溯至前7000年以后。

## 概括

南亚古代史不是从单一民族、宗教或王朝连续展开的直线。印度河流域曾形成横跨今巴基斯坦和印度西北部的城市网络；城市体系收缩后，西北、恒河流域、德干、南印度和斯里兰卡分别出现新的聚落、语言、祭祀、商业与王权。吠陀传统、婆罗门教、奥义书思想、佛教、耆那教及后来的宗派印度教在竞争、吸收和地方化中共同演变，不能简单排列为彼此取代的阶段。

## 分期与具体过程

### 哈拉帕城市网络

成熟哈拉帕阶段约在前2600—前1900年。摩亨佐-达罗、哈拉帕、多拉维拉等城市拥有规划街区、排水、水利、标准化度量衡和广域手工业交换；聚落之间共享若干物质文化，却没有证据证明存在一位统治整个流域的“皇帝”。印章文字仍未可靠释读，政治制度、语言和宗教只能依据考古材料谨慎推断。

前2千纪初，大城市人口下降，聚落向东部和较小节点重组。河道变化、季风减弱、贸易结构和地方生产方式转变可能共同作用；“一次外族入侵摧毁文明”已不能概括现有证据。城市传统并非完全消失，农业、工艺和聚落经验在不同区域以新形式延续。

### 吠陀社会与恒河国家化

约前2千纪后半，印欧语系的早期印度—雅利安语文献传统在西北形成。《梨俱吠陀》反映以畜牧、祭祀、赠礼和部族首领为重要组织方式的社会，后期文献则显示聚落向恒河上游扩展、农业加强、王权仪式复杂化和瓦尔纳秩序逐渐理论化。语言与文化迁移不等于可被简单描述为单一“种族征服”。

前6世纪前后，铁器使用、稻作扩展、道路和货币化推动恒河中下游的“第二次城市化”。十六雄国竞争中，摩揭陀凭借河运、农业腹地、象兵和对交通节点的控制逐渐占优。城市商人、游方思想者和新王权共同构成佛教、耆那教及奥义书讨论的社会背景。

### 帝国整合与区域再分化

旃陀罗笈多·孔雀约在前321年建立孔雀王朝；宾头沙继续扩张，阿育王在羯陵伽战争后以敕令阐述“法”的治理伦理。孔雀帝国依靠王室中心、行省、道路、税收和地方中介维系统治，阿育王的多语种石刻也说明其臣民并不均质。前185年前后王朝终结后，印度—希腊、塞种、贵霜、巽伽、百乘等政权并立，西北、德干和海港通过中亚与印度洋贸易相连。

约4世纪兴起的笈多王朝整合恒河核心区，并通过封赠、婚姻和地方王权建立分层宗主关系。梵语文学、数学、天文学、寺院和造像传统发展显著，但“黄金时代”不能遮蔽统治范围有限、社会等级与地区差异。6世纪以后，匈人冲击、财政与地方权力变化促成再分化，北印度、德干、泰米尔地区和孟加拉进入多国并立。

### 南印度与斯里兰卡

泰米尔地区的早期历史见于铭文、港口遗址和桑伽姆文学。朱罗、哲罗、潘地亚等王权连接内陆农业、战士集团和印度洋港口；罗马金币、陶器与本地商品说明远程交换，却不意味着南印度受罗马政治控制。斯里兰卡的阿努拉德普勒王权、佛教僧团、灌溉工程和港口网络又形成独立而相互联结的历史主线。

## 政治、经济与知识机制

| 机制 | 运作方式 | 地区差异与长期影响 |
|---|---|---|
| 农业与水利 | 河谷稻麦农业、井渠、蓄水池和土地开垦支撑城市与税收 | 印度河依赖河道与季风，恒河受稻作扩展推动，德干和斯里兰卡更重蓄水灌溉 |
| 王权与地方中介 | 征税、贡纳、军役、封赠和仪式把宫廷与村社、商人、寺院相连 | 帝国直辖程度有限，地方首领和社群往往保留实际权力 |
| 商业网络 | 陆路连接伊朗、中亚，海路连接红海、东非、东南亚 | 港口兴衰随季风航行、政治保护和商品需求改变 |
| 语言与文字 | 梵语、巴利语、普拉克里特语、泰米尔语等并存；婆罗米及其派生文字扩展 | 宫廷语言、宗教语言和日常语言并不总是一致 |
| 宗教赞助 | 国王、商人、行会和家庭向祭司、僧团与寺院捐赠 | 赞助既表达信仰，也组织土地、教育、信用和政治合法性 |

## 宗教与思想的演变

- **吠陀—婆罗门传统：** 祭祀、赞歌和祭司知识逐步发展出关于宇宙秩序、社会义务与王权的体系；奥义书把讨论推进到自我、业、轮回和解脱。
- **沙门传统：** 佛教、耆那教及其他团体质疑祭祀中心论，以出家、戒律、禅修、非暴力或苦行提出不同解脱道路。
- **宗派印度教：** 毗湿奴、湿婆、女神等崇拜与地方神祇、史诗、往世书、寺庙礼仪和虔信实践结合，形成多种传统，而非一个时点上突然出现的统一宗教。
- **跨区域传播：** 僧侣、朝圣者、商人、译者和王权赞助共同推动佛教等思想移动；传播过程中教义、艺术和制度不断被当地社会重塑。
- **社会秩序与争论：** 瓦尔纳是规范性分类，不等于各地实际职业、亲族和阇提结构的完整写照；性别、身份与劳动关系也随地区和时代变化。

## 重要事件与转折

| 时间 | 事件 | 结果与长期影响 |
|---|---|---|
| 约前2600年 | 成熟哈拉帕城市体系形成 | 标准化生产、城市水利和跨区域交换达到高峰 |
| 约前1900年后 | 大城市收缩、聚落重组 | 政治经济中心东移并区域化，原因具有多重性 |
| 约前1200—前800年 | 后期吠陀社会向恒河上游扩展 | 农业、王权仪式和社会分类进一步制度化 |
| 前6—前5世纪 | 十六雄国与第二次城市化 | 摩揭陀崛起，佛教、耆那教等沙门传统形成 |
| 约前321年 | 孔雀王朝建立 | 首次把次大陆大部纳入同一帝国框架 |
| 约前261年 | 羯陵伽战争及阿育王政策转向 | “法”的敕令治理和佛教赞助获得帝国尺度 |
| 前185年前后 | 孔雀王朝终结 | 西北、恒河、德干出现多政权竞争 |
| 1—3世纪 | 贵霜、百乘及印度洋贸易繁荣 | 南亚与中亚、罗马世界、东南亚联系加强 |
| 约320年后 | 笈多王朝兴起 | 恒河核心再整合，梵语宫廷与学术传统扩展 |
| 5—6世纪 | 外部冲击与地方权力上升 | 笈多秩序瓦解，区域国家成为主导 |

## 争议与阅读原则

古代年代常来自后世文献、铭文、钱币和考古层位的综合重建。印度河文字、早期吠陀年代、某些王统及宗教传统的起点均存在争议，应使用“约”“传统认为”或“材料不足”。现代国界也不能倒投到古代：犍陀罗、旁遮普、孟加拉、泰米尔地区和斯里兰卡都跨越或超出现代国家叙事。
