# 波利尼西亚

## 范围

波利尼西亚通常指横跨萨摩亚、汤加、库克群岛、塔希提、夏威夷、拉帕努伊、托克劳和阿奥特阿罗瓦 / 新西兰等地的广阔海洋三角区；其内部社会和政治经历并不相同。

## 概括

波利尼西亚航海者以复杂的航行知识定居广阔岛群，形成多种王权、首领制、村落和亲属政治。汤加等地发展较强的王权传统；夏威夷形成群岛国家并被美国吞并；萨摩亚、库克群岛和法属波利尼西亚则有不同的殖民和自治路径。新西兰毛利人历史属于波利尼西亚世界，同时具有独特的温带环境和条约历史。

## 主要线索

| 区域 | 主线提示 |
|---|---|
| 汤加 | 保持本土君主制传统，19世纪与欧洲列强建立保护关系。 |
| 萨摩亚 | 德、美、英竞争后分治，西萨摩亚1962年独立。 |
| 夏威夷 | 原住民王国、美国商人影响、1893年政变与1898年并入美国。 |
| 法属波利尼西亚 | 法国海外集体，核试验与自治、自决议题相连。 |
| 库克群岛、纽埃 | 与新西兰自由联合，拥有不同于独立国家和海外属地的地位。 |
| 拉帕努伊 | 与智利联系的岛屿社会，考古叙事与原住民土地权利须共同理解。 |

## 重要说明

- 波利尼西亚并非同一种“航海民族”，语言相近并不等于政治制度或历史遭遇相同。
- 欧洲疾病、传教、劳工贸易、军事和商人资本改变各岛人口与权力结构。
- 夏威夷、新西兰和法属波利尼西亚显示，殖民地、吞并、条约和自治的法律形式差异很大。
- 传统航海、语言复兴和海洋保护在当代是活跃的文化与政治行动。
