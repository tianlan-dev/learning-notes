# 太平洋岛屿

## 概括

太平洋岛屿历史包括波利尼西亚、密克罗尼西亚、美拉尼西亚航海网络，欧洲殖民、传教、太平洋战争和现代岛屿国家独立。

## 整理建议

- 先按原住民 / 航海传统、欧洲接触、殖民统治、战争与现代国家阶段组织。
- 太平洋岛屿避免只按现代小国碎片化，可先按波利尼西亚、密克罗尼西亚、美拉尼西亚分组。
