# 美拉尼西亚

## 范围

通常包括巴布亚新几内亚、所罗门群岛、瓦努阿图、斐济和新喀里多尼亚等地；这一分类内部语言、地理、殖民与政治差异极大。

## 概括

美拉尼西亚拥有极高的语言和社会多样性。新几内亚高地的农业历史、沿海岛屿航海网络、土地与习惯权利、村落和首领制度均不能用单一模式描述。殖民分割、传教、劳工招募和二战改变了社会，20世纪后半叶的独立、资源开发、族群政治和气候风险成为共同但不同步的主题。

## 主要区域

| 区域 | 主线提示 |
|---|---|
| 巴布亚新几内亚 | 高地与海岸社会多样，1975年独立，习惯土地权与资源开发关系突出。 |
| 所罗门群岛 | 岛屿社会、英国保护国、二战战场、1978年独立与后续政治危机。 |
| 瓦努阿图 | 英法共管体制罕见，1980年独立，语言与土地权利多样。 |
| 斐济 | 原住民斐济人、印裔契约劳工后代和殖民遗产共同塑造政治。 |
| 新喀里多尼亚 | 法国海外领地，卡纳克民族自决与镍矿资源问题持续。 |

## 重要过程

- 殖民前农业、园艺、猪、贝壳交换和岛际航海在各地形成不同的财富与政治结构。
- 19世纪“黑鸟掠工”和契约劳工把大量岛民带往种植园，也带来暴力、人口流失与跨岛社群。
- 殖民边界常把语言和亲属网络切分为不同管辖区。
- 独立后，国家建设需面对地形破碎、基础设施、土地习惯权利、资源项目和外部援助依赖。
- 自然灾害、海平面变化和海洋资源治理使区域合作具有现实必要性。
