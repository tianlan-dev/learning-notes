# 殖民分割、传教与劳工贸易

## 时间

18世纪末至20世纪中叶。

## 概括

欧洲和美国在太平洋的扩张通过传教、贸易、海军、条约、保护国和公司而非单一征服模式展开。英国、法国、德国、美国、西班牙、日本等在不同岛群建立殖民或战略控制。传教既带来教育、文字和跨岛联系，也参与文化改造和土地政治；劳工招募、包括暴力欺骗性的“黑鸟掠工”，把岛民卷入澳大利亚、斐济、秘鲁等地的种植园和矿业经济。

## 殖民分割

| 力量 | 主要影响区域 |
|---|---|
| 英国 | 斐济、所罗门群岛、吉尔伯特和埃利斯群岛及多处保护国。 |
| 法国 | 新喀里多尼亚、法属波利尼西亚、瓦利斯和富图纳等。 |
| 德国 | 一战前在萨摩亚、新几内亚东北部、马绍尔和加罗林部分岛屿拥有殖民地。 |
| 美国 | 夏威夷、关岛、东萨摩亚及战后托管地区。 |
| 日本 | 一战后在多处密克罗尼西亚岛屿实施委任统治，二战中形成军事网络。 |

## 重要过程

- 传教士和地方首领的关系并不只是单向皈依；宗教、教育、权威和贸易经常被重新组合。
- “黑鸟掠工”涉及绑架、欺骗和强制招募，不能与自愿契约劳动混同；其后果至今体现在劳工后代和跨岛社群中。
- 殖民地边界经常脱离既有航海和亲属网络，造成跨岛群、跨国家的社会联系。
- 19世纪末的殖民分割与欧洲列强竞争、全球商品市场和海军补给点需求密切相关。
- 殖民行政、习惯土地制度和教育语言政策为独立后的国家建设留下复杂遗产。
