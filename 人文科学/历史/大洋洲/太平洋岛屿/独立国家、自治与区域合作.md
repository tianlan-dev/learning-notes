# 独立国家、自治与区域合作

## 时间

1960年代至今。

## 概括

战后去殖民化使萨摩亚、斐济、巴布亚新几内亚、所罗门群岛、瓦努阿图、基里巴斯、图瓦卢、马绍尔群岛、密克罗尼西亚联邦、帕劳等先后取得独立或主权安排。与此同时，库克群岛、纽埃、关岛、新喀里多尼亚、法属波利尼西亚等拥有不同自治或海外领地地位。太平洋岛屿论坛等机制将国家、自治体、澳大利亚和新西兰连接起来，海洋、渔业、气候、核遗产与大国竞争成为共同议题。

## 政治路径

| 路径 | 例子 | 说明 |
|---|---|---|
| 独立国家 | 萨摩亚、斐济、巴布亚新几内亚、瓦努阿图等 | 取得国际法上的国家主权，但发展与安全资源有限。 |
| 自由联合 | 库克群岛、纽埃、马绍尔群岛、密克罗尼西亚联邦、帕劳等 | 与新西兰或美国存在各不相同的条约与公民、援助、防务安排。 |
| 海外领地 / 自治体 | 关岛、新喀里多尼亚、法属波利尼西亚等 | 主权国家之外的宪政地位，自决议题持续。 |
| 区域合作 | 太平洋岛屿论坛等 | 处理贸易、渔业、海洋、气候、安全和共同外交。 |

## 重要过程

- 1962年西萨摩亚独立是战后太平洋去殖民化的重要早期节点；1970-1980年代多座岛屿相继独立。
- 1971年太平洋岛屿论坛成立，反映岛屿国家和澳新围绕共同事务进行区域协调。
- 专属经济区使许多小岛国家拥有广阔海域权益，渔业、海底资源与海洋保护成为经济和主权核心。
- 气候变化、海平面上升、热带气旋和珊瑚礁退化并非单纯环境问题，也关联领土、迁移、文化与国际法。
- 外部大国在港口、安全、援助、通信和资源上的竞争，为岛屿国家带来谈判空间，也带来主权和依赖风险。
- 原住民、习惯土地权利、语言、女性与青年政治在国家建构中持续发挥作用。
