# 密克罗尼西亚

## 范围

通常包括加罗林群岛、马绍尔群岛、马里亚纳群岛、帕劳、瑙鲁和基里巴斯等岛群；岛屿、环礁和航海网络的历史各不相同。

## 概括

密克罗尼西亚社会以环礁和小岛环境中的航海、渔业、园艺、首领政治和岛际交换为基础。西班牙、德国、日本和美国先后介入不同岛群，二战后联合国托管和美国自由联合关系重塑政治地位。核试验、军事基地、海平面上升和海洋专属经济区是理解现代密克罗尼西亚不可忽略的历史延续。

## 主要线索

| 地区 | 主线提示 |
|---|---|
| 马绍尔群岛 | 环礁航海、日治与美军统治、比基尼和埃内韦塔克核试验遗产。 |
| 密克罗尼西亚联邦 | 加罗林群岛多岛政治体，与美国自由联合。 |
| 帕劳 | 海洋社会、日美统治遗产、自由联合与环境保护。 |
| 北马里亚纳、关岛 | 美国关联领地，军事基地与查莫罗权利议题突出。 |
| 瑙鲁、基里巴斯 | 磷矿、环礁脆弱性、海洋权利和气候风险。 |

## 重要过程

- 传统导航者以星图、海浪和岛屿知识组织跨岛航行，不能视为没有远洋能力的孤立社会。
- 19世纪末德国、西班牙和其他列强分割岛屿；一战后日本取得多处委任统治。
- 二战太平洋战场造成严重破坏，美国战后取得多地托管管理。
- 美国在马绍尔群岛进行核试验，健康、土地和赔偿问题持续影响当地社区。
- 自由联合协定给予若干国家不同于完全独立或殖民属地的政治经济安排。
