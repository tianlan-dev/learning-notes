# 毛利人定居与社会

## 时间

约13-14世纪至今。

## 概括

毛利人是波利尼西亚航海者及其后代在阿奥特阿罗瓦形成的原住民族群。定居者适应温带环境，发展园艺、渔猎、林木利用和跨区域贸易；以whānau（大家庭）、hapū（亚部族）和iwi（部族）等亲属政治网络组织社会。毛利人社会并非静止或统一，地区、资源、战争、婚姻与迁徙持续塑造各共同体。

## 社会结构

| 层级 / 概念 | 说明 |
|---|---|
| Whānau | 扩展家庭，是亲属、照护和日常经济关系的重要单位。 |
| Hapū | 由多个家庭构成的亚部族，常是土地、政治和战争动员的关键层级。 |
| Iwi | 更广泛的部族网络，拥有共同祖先、领地与政治身份。 |
| Mana、whakapapa、whenua | 权威、谱系与土地之间相互联系的概念，不能简单译为近代私有财产或主权。 |

## 重要过程

- 毛利定居与更广泛的波利尼西亚航海传统相连，日期和迁徙路径仍随考古、语言和口述传统研究而讨论。
- 园艺、储藏、鸟类与海洋资源利用因北岛、南岛和沿海环境而异。
- 部族之间存在贸易、婚姻、仪式和战争关系，不能以“部落冲突”一语替代复杂政治。
- 18世纪末欧洲接触后，金属工具、贸易品、疾病和火器迅速改变力量平衡；所谓“火枪战争”具有地区差异和多重原因。
- 19世纪殖民国家扩张、战争和土地购买 / 没收改变了毛利社会，但语言、艺术、政治组织和条约权利持续存在。
