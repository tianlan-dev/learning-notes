# 欧洲接触、怀唐伊条约与殖民战争

## 时间

1769-1870年代。

## 概括

库克到访后，捕鲸者、海豹猎人、传教士、商人和定居者逐步扩大在阿奥特阿罗瓦的活动。1840年签署的怀唐伊条约试图建立王室政府与毛利首领之间的关系，但两种文本并非逐字对应：毛利语文本中的kāwanatanga和tino rangatiratanga与英语文本中的主权概念存在重要差异。殖民政府以条约、购买、没收和战争取得土地，19世纪中后期的殖民战争使这种冲突制度化。

## 统治与条约结构

| 要素 | 说明 |
|---|---|
| 英国王室与总督 | 建立殖民政府并主张治理权。 |
| 毛利首领、hapū与iwi | 以亲属、土地与权威关系参与谈判、支持或抵抗。 |
| 怀唐伊条约 | 1840年签署，存在毛利语和英语文本；意义与履行长期有争议。 |
| 殖民议会与定居者政府 | 逐步取得自治，推动土地、移民和基础设施政策。 |

## 重要事件

- 1769年库克航行后欧洲接触增加，贸易、疾病和火器改变毛利社会与区域关系。
- 1840年2月6日，首批毛利首领与王室代表在怀唐伊签约，后有更多首领在其他地点签署。
- 条约并未解决土地、主权和政府权力的解释问题；部分首领未签，或对条约含义有不同理解。
- 1840-1870年代的新西兰战争涉及北岛多地的王室军队、定居者、毛利政治体和联盟。
- 土地没收、法院制度和定居扩张削弱许多毛利共同体的土地基础。
- 1870年代以后殖民政府加强铁路、移民和土地开发，国家制度逐步覆盖岛屿。
