# 大洋洲历史

## 概括

大洋洲历史包括澳大利亚、新西兰和太平洋岛屿世界。其主线既有澳大利亚原住民、毛利人和波利尼西亚航海传统，也有英国、法国、德国、美国、日本等外来殖民和太平洋战争、现代岛屿国家独立等近现代变化。
