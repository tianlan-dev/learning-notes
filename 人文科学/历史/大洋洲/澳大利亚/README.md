# 澳大利亚

## 概括

澳大利亚历史包括原住民社会、英国流放殖民地、殖民自治、澳大利亚联邦、两次世界大战和现代澳大利亚。

- 先按原住民 / 航海传统、欧洲接触、殖民统治、战争与现代国家阶段组织。
