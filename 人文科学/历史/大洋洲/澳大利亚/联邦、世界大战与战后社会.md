# 联邦、世界大战与战后社会

## 时间

1901-1945年。

## 概括

1901年澳大利亚联邦成立后，新的联邦政府逐步控制关税、移民、防务和国家市场。白澳政策、劳工仲裁和社会改革共同构成早期联邦秩序。澳大利亚在两次世界大战中作为英帝国和后来的盟国体系成员参战；第二次世界大战的日本南进和太平洋战场促使澳大利亚安全战略更紧密转向美国。

## 统治结构

| 机构 / 力量 | 作用 |
|---|---|
| 联邦议会与内阁 | 制定移民、关税、防务和全国性社会经济政策。 |
| 州政府 | 保留教育、警务、土地和大量公共服务权力。 |
| 英国王室与总督 | 宪政象征与形式权力，外交安全关系仍深受帝国纽带影响。 |
| 工会、农场主与企业 | 影响工资仲裁、贸易保护和政党政治。 |

## 重要事件

- 1901年联邦成立并推动限制非白人移民的法律，白澳政策长期影响国家人口和身份想象。
- 1914-1918年澳大利亚参加第一次世界大战，加里波利战役在国家记忆中具有突出地位；征兵公投两次失败。
- 1920年代经济增长与国际贸易依赖并存，1930年代大萧条造成失业和社会压力。
- 第二次世界大战初期澳大利亚在欧洲和北非作战；1941-1942年日本南进、达尔文空袭和新几内亚战役使太平洋防务成为核心。
- 战争扩大工业动员、女性劳动和国家规划，也强化澳大利亚与美国的军事合作。
