# 原住民与托雷斯海峡岛民社会

## 时间

至少数万年前至今；具体定居年代与迁徙路线仍随考古和环境研究更新。

## 概括

澳大利亚原住民与托雷斯海峡岛民并非单一文化或语言群体。不同民族在沙漠、森林、河流、海岸和海峡环境中形成复杂的土地关系、亲属制度、法律、仪式、贸易和知识体系。殖民后的人口损失、土地剥夺、边疆暴力、传教、保护制度和强制同化造成深刻伤害，但各族群持续维护文化、语言、社区和政治权利。

## 社会与区域

| 区域 / 群体 | 历史特点 |
|---|---|
| 大陆原住民社会 | 数百种语言与方言、国家 / 民族共同体和广泛贸易网络，土地与祖先故事密切相连。 |
| 托雷斯海峡岛民 | 海洋航行、渔业、园艺和岛际网络突出，与巴布亚新几内亚及北澳海域关系密切。 |
| 城市与边境地区 | 殖民接触更早、更强烈，但原住民家庭、劳工、政治组织与文化实践持续存在。 |
| 偏远地区 | 受牧业、矿业、传教和行政控制影响，同时保留或复兴多种语言和土地管理知识。 |

## 重要过程

- 土地、海洋与文化知识由具体民族、亲属关系、仪式与管理责任构成，不是抽象的“传统”。
- 1788年后定居殖民扩张导致疾病、暴力、驱逐与资源剥夺，影响因地区、时间和民族不同而异。
- 19-20世纪保护、隔离和同化政策限制流动、家庭和语言；“被偷走的一代”是这一历史的组成部分。
- 1967年公投、土地权判例和原住民土地权立法改变联邦法律与公共政治，但未解决所有主权、条约、社会服务和土地争议。
- 原住民与托雷斯海峡岛民的艺术、语言复兴、土地管理、医疗、教育和政治组织持续塑造当代澳大利亚。
