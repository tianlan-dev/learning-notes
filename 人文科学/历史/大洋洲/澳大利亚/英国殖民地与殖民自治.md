# 英国殖民地与殖民自治

## 时间

1788-1901年。

## 概括

1788年英国在悉尼湾建立新南威尔士殖民地，最初以流放制度为核心，后逐步发展为牧业、农业、港口和城市社会。定居殖民不断侵占原住民土地，引发边疆暴力和长期法律政治后果。19世纪淘金和移民推动人口与经济增长，多个殖民地获得自治；1890年代联邦运动最终促成1901年澳大利亚联邦。

## 殖民统治结构

| 层级 | 角色 | 说明 |
|---|---|---|
| 英国王室与殖民部 | 宗主国权力 | 任命总督，管理外交、防务与殖民宪政框架。 |
| 总督 | 殖民行政首脑 | 早期权力较集中，自治后仍代表王室。 |
| 殖民议会与内阁 | 地方自治 | 19世纪中叶起逐步获得责任政府。 |
| 定居者、牧场主与矿业利益 | 经济政治力量 | 依靠土地占有、劳工与出口经济扩大影响。 |
| 原住民与托雷斯海峡岛民 | 被殖民的原有社会 | 面对土地丧失、暴力和法律排除，同时持续抵抗、谈判与生存。 |

## 重要事件

- 1788年“第一舰队”抵达并建立殖民地；英国“无主地”主张否认原住民既有土地关系。
- 19世纪牧业向内陆扩张，边疆冲突、疾病和驱逐广泛发生。
- 1850年代金矿发现吸引全球移民，也激发劳工组织和政治改革。
- 1854年尤里卡栅栏事件成为淘金者、选举权和民主政治的重要象征。
- 1850-1890年代各殖民地逐步取得责任政府。
- 排华和限制非白人移民的政策在殖民时期已出现，后来影响联邦“白澳政策”。
- 1901年六个英国殖民地组成澳大利亚联邦；原住民并未作为平等的制宪主体被纳入。
