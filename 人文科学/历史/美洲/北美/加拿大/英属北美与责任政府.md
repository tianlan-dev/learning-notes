# 英属北美与责任政府

## 时间

1763年-1867年。

## 概括

1763年以后，英国取得法国在加拿大地区的主要殖民地，却必须治理以法语和天主教徒为主的圣劳伦斯河社会，同时面对第一民族的条约权利、美国革命后的保皇派移民，以及大西洋、太平洋和哈得孙湾地区彼此分散的殖民地与公司领地。英国没有把这些地区一次性合并为统一国家，“英属北美”始终是多个殖民地、领地和贸易区的集合。

19世纪前半叶，民选议会与由总督任命的行政集团之间冲突加剧。1837年至1838年叛乱、达勒姆报告和1840年《联合法》推动制度重组；1848年前后，多个殖民地形成行政部门对民选议会负责的责任政府。美国扩张、殖民地财政与铁路需求、防务焦虑和长期政治僵局，最终推动1867年加拿大联邦建立。

## 主要殖民地与领地

| 殖民地 / 地区 | 主要阶段 | 说明 |
|---|---|---|
| 魁北克殖民地 | 1763年-1791年 | 英国统治原新法兰西核心区；1774年后保留法国民法和天主教信仰空间。 |
| 上加拿大与下加拿大 | 1791年-1841年 | 上加拿大以英语殖民者为主，下加拿大以法语人口为主，二者各有民选议会。 |
| 加拿大省 | 1841年-1867年 | 上、下加拿大合并为加拿大西部和加拿大东部；议席、语言与政府组成长期引发政治僵局。 |
| 新斯科舍、新不伦瑞克、爱德华王子岛 | 18世纪-1867年 | 大西洋殖民地各有独立议会和政治发展；新斯科舍率先形成稳定的责任政府。 |
| 纽芬兰 | 英国殖民地；未参加1867年联邦 | 渔业社会拥有独立殖民政治，直到1949年才加入加拿大。 |
| 鲁珀特地 | 哈得孙湾公司控制至1870年 | 覆盖哈得孙湾流域的大面积贸易特许地，不等同于普通定居殖民地。 |
| 温哥华岛与不列颠哥伦比亚 | 1849年、1858年分别建殖民地，1866年合并 | 淘金潮和太平洋贸易推动英国加强西海岸统治；1871年加入加拿大联邦。 |

## 宪制转型

| 时间 | 法律 / 事件 | 制度影响 |
|---|---|---|
| 1763年 | 《皇家公告》 | 建立英国殖民行政，并在英王所称领地中确认一套涉及原住民土地和王室购地的原则；实际执行长期充满争议。 |
| 1774年 | 《魁北克法》 | 扩大魁北克边界，允许天主教信仰，并在私法领域恢复法国民法传统。 |
| 1791年 | 《宪法法》 | 分设上加拿大与下加拿大，各设民选议会，但行政权仍主要掌握在总督及任命机构手中。 |
| 1837年-1838年 | 上、下加拿大叛乱 | 暴露代表制与殖民行政之间的冲突；下加拿大还包含法语民族政治和土地制度矛盾。 |
| 1839年 | 达勒姆报告 | 建议合并两加拿大并推行责任政府，也带有同化法裔加拿大人的明确意图。 |
| 1840年生效于1841年 | 《联合法》 | 上、下加拿大合并为加拿大省，分别称加拿大西部和加拿大东部。 |
| 1848年前后 | 责任政府形成 | 新斯科舍和加拿大省等地的行政部门开始依赖民选议会多数支持。 |
| 1864年 | 夏洛特敦会议与魁北克会议 | 殖民地代表讨论联邦方案并形成主要制度原则。 |
| 1866年 | 伦敦会议 | 最终拟定提交英国议会的联邦方案。 |
| 1867年 | 《英属北美法》生效 | 加拿大省分为安大略、魁北克，并与新斯科舍、新不伦瑞克组成联邦。 |

## 殖民统治结构

### 责任政府形成以前

| 角色 / 机构 | 权力来源 | 主要职能 |
|---|---|---|
| 英国君主与英国议会 | 帝国主权 | 制定殖民地根本法律并任命或授权殖民高级官员。 |
| 总督 / 副总督 | 英国政府与王室任命 | 代表王室主持行政，可任命行政委员会并处理对外和军事事务。 |
| 行政委员会 | 总督任命 | 协助行政，早期不必取得民选议会信任。 |
| 立法委员会 | 任命产生 | 审议法案，常代表殖民精英利益。 |
| 民选议会 | 有限选举 | 审议税收和法案，但选举资格受财产、性别和身份限制。 |

### 责任政府形成以后

| 角色 / 机构 | 制度变化 | 说明 |
|---|---|---|
| 总督 | 逐步减少日常党派统治 | 通常任命能获得民选议会支持的政治领袖组阁。 |
| 行政委员会 / 内阁 | 对民选议会负责 | 失去多数信任时原则上应辞职或寻求重新选举。 |
| 殖民地议会 | 财政和立法中心地位增强 | 仍受帝国法律和总督保留权等限制。 |
| 殖民地政治领袖 | 形成政党与联合内阁 | 罗伯特·鲍德温、路易-伊波利特·拉封丹等合作体现英法政治联盟。 |

## 原住民族、土地与殖民扩张

- 《皇家公告》在英王体系内承认某些原住民土地不得由私人直接取得，但它没有终止殖民扩张，也不能概括各原住民族与王室之间的全部条约关系。
- 美国独立战争后，大批保皇派迁入英属北美，推动上加拿大和新不伦瑞克等政治空间形成，同时加大土地需求。
- 19世纪殖民政府通过土地让渡条约、定居计划和道路建设扩大控制。不同地区的条约实践差异很大，太平洋沿岸许多土地并未通过历史条约取得。
- 教会和殖民政府开办面向原住民儿童的寄宿学校，为联邦时期更加制度化的强制同化体系奠定基础。
- 毛皮贸易中的原住民、法裔和混合家庭网络促成梅蒂共同体发展；红河地区逐渐成为重要的梅蒂政治与经济中心。

## 社会、战争与经济

- 1775年至1783年美国独立战争改变边境和人口结构；英属北美没有整体加入美国革命。
- 1812年至1815年战争中，英国正规军、殖民地民兵和多个第一民族盟友共同抵抗美国入侵。战争不能简单写成现代加拿大独自击败美国。
- 1821年，哈得孙湾公司与西北公司合并，重组内陆毛皮贸易。
- 19世纪中叶，运河、铁路、木材和跨大西洋移民推动殖民经济整合。
- 1854年英美互惠条约扩大与美国的贸易；1866年条约终止后，殖民地更重视内部市场。
- 美国南北战争、芬尼亚袭击和英国减少驻军的趋势增强了殖民地联合防务的诉求。

## 重要事件

| 时间 | 事件 | 影响 |
|---|---|---|
| 1784年 | 新不伦瑞克从新斯科舍分出 | 回应保皇派移民增长，形成新的大西洋殖民地。 |
| 1812年-1815年 | 1812年战争 | 强化英属北美的防务合作与地方政治记忆，也使部分第一民族盟友付出重大代价。 |
| 1837年-1838年 | 上、下加拿大叛乱 | 推动英国重新评估殖民治理，但叛乱者诉求和社会基础并不相同。 |
| 1848年 | 新斯科舍和加拿大省责任政府确立 | 内阁对民选议会负责成为自治政治的核心惯例。 |
| 1864年-1866年 | 三次联邦会议 | 形成联邦与省分权、两院议会和君主立宪的基本方案。 |
| 1867年7月1日 | 加拿大联邦成立 | 标志殖民地联合国家建立，但加拿大的独立和疆域形成仍是长期过程。 |
