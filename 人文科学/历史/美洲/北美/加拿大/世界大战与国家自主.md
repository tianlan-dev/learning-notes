# 世界大战与国家自主

## 时间

1914年-1945年。

## 概括

两次世界大战使加拿大付出巨大人员代价，也加速工业化、联邦权力扩张和国际地位提升。1914年英国对德宣战时，加拿大作为大英帝国自治领随之进入战争，但加拿大政府自行决定军事贡献；1939年，加拿大议会经过辩论后单独对德宣战，显示其对外决策已发生根本变化。

1919年加拿大参与《凡尔赛条约》和国际联盟，1926年帝国会议承认自治领地位平等，1931年《威斯敏斯特法令》确认加拿大议会的立法自主。然而自主化不是一次完成的“独立日”：宪法修改仍依赖英国议会，英国枢密院司法委员会也继续受理加拿大上诉。战争时期的征兵危机、限制公民自由、日裔加拿大人拘禁与财产剥夺，以及延续的原住民同化政策，显示国家能力扩张也可能造成严重权利侵害。

## 第一次世界大战

| 方面 | 内容 |
|---|---|
| 参战方式 | 1914年英国宣战后加拿大随帝国参战；联邦政府决定组建加拿大远征军并调动国内资源。 |
| 海外作战 | 加拿大军队参加第二次伊普尔、索姆河、维米岭、帕斯尚尔和“百日攻势”等战役。 |
| 维米岭记忆 | 1917年维米岭战役常被视为国家认同的重要象征，但加拿大自主化是长期政治、法律和社会过程，不能归结为一场战役。 |
| 国内动员 | 《战时措施法》扩大政府权力；工业、税制和劳动力结构变化，女性更多进入有薪劳动。 |
| 征兵危机 | 1917年征兵政策在英语加拿大与法语魁北克之间造成严重分裂，也影响农民、劳工和少数族群。 |
| 选举权 | 1917年至1918年联邦女性选举权扩大，但许多原住民、亚裔及其他群体仍被排除。 |
| 原住民参战 | 多个第一民族、梅蒂和因纽特成员服役；退伍后仍面对土地、身份和福利方面的歧视。 |

## 战间期与自主化

| 时间 | 事件 | 自主化意义 |
|---|---|---|
| 1919年 | 加拿大单独列名签署《凡尔赛条约》并加入国际联盟 | 国际人格增强，但仍处于大英帝国共同体内。 |
| 1923年 | 加拿大自行签署大比目鱼条约 | 在未由英国共同签署的情况下处理对美条约，是独立外交实践的重要一步。 |
| 1926年 | 金—宾宪政危机 | 总督拒绝总理麦肯齐·金解散议会的请求，引发总督角色和加拿大自主的争论。 |
| 1926年 | 《贝尔福宣言》 | 帝国会议确认英国与自治领地位平等、彼此不相隶属并共同效忠王室。 |
| 1931年 | 《威斯敏斯特法令》 | 加拿大议会取得关键立法自主，英国法律原则上不再自动适用于加拿大。 |
| 1939年 | 加拿大单独对德宣战 | 议会在英国宣战一周后自行作出战争决定，体现外交自主。 |

《威斯敏斯特法令》没有把加拿大宪法修改权完全移交国内，因为联邦与各省尚未就修改公式达成一致；这一问题直到1982年才解决。

## 大萧条

- 1929年经济危机导致失业、农产品价格崩跌和草原旱灾，地方救济能力迅速不足。
- 联邦与省之间围绕失业救济和财政责任发生冲突，暴露联邦制度缺乏全国社会保障机制的问题。
- R·B·贝内特政府设立救济营，并在任期后期提出带有“新政”色彩的改革；部分措施遭法院否定或未及实施。
- 1935年加拿大银行成立，联邦货币与金融管理能力增强。
- 1935年昂塔里奥至渥太华长征在里贾纳遭镇压，体现失业者对救济营制度的不满。
- 大萧条推动此后失业保险、财政转移和福利国家的制度讨论。

## 第二次世界大战

| 方面 | 内容 |
|---|---|
| 参战与战场 | 1939年9月10日加拿大对德宣战；军队参与大西洋海战、香港保卫战、迪耶普突袭、意大利战役、诺曼底登陆和西北欧战役。 |
| 空军训练 | 英联邦空军训练计划在加拿大建立大规模飞行人员训练网络。 |
| 工业与社会 | 军工生产、农业出口和女性就业扩大；联邦通过价格、工资和资源配置加强经济管理。 |
| 征兵问题 | 政府先承诺限制海外征兵，1944年因兵员压力派部分义务兵出国，再次引发政治分裂。 |
| 日裔加拿大人 | 政府强制迁移和拘禁太平洋沿岸日裔居民，并没收、出售其财产；这些措施后来被正式承认和赔偿。 |
| 其他限制 | 德裔、意大利裔等部分居民被登记或拘禁，战时权力压缩公民自由。 |
| 原住民与少数族群 | 许多人参军或参加国内生产，却仍承受选举、身份、土地和就业歧视。 |

## 国家治理

### 国家元首与代表

| 角色 | 本阶段特点 |
|---|---|
| 加拿大君主 | 依次为乔治五世、爱德华八世、乔治六世；1931年以后加拿大王冠逐渐成为与英国王冠法律上可区分的制度。 |
| 总督 | 从兼具英国政府代表性质，逐步转向主要依加拿大部长建议履行加拿大宪政职能。 |

### 政府与议会

| 角色 / 机构 | 本阶段特点 |
|---|---|
| 总理与内阁 | 罗伯特·博登、阿瑟·米恩、麦肯齐·金和R·B·贝内特等政府处理战争、经济危机和自主化；详见[加拿大总理表](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E7%BE%8E%E6%B4%B2/%E5%8C%97%E7%BE%8E/%E5%8A%A0%E6%8B%BF%E5%A4%A7/%E5%8A%A0%E6%8B%BF%E5%A4%A7%E6%80%BB%E7%90%86%E8%A1%A8.md)。 |
| 联邦议会 | 通过战争拨款、税收、兵役和紧急法律；少数政府与党派重组考验责任政府惯例。 |
| 省政府 | 承担救济、教育和医疗等事务，与联邦围绕财政和管辖权持续谈判。 |
| 司法机关 | 审查联邦与省权限；英国枢密院司法委员会仍是最终上诉机构。 |

## 社会与文化变化

- 1919年温尼伯大罢工显示战后劳资冲突和社会改革诉求。
- 1923年《华人移民法》几乎阻断华人移民，是排斥性移民制度的高峰之一。
- 城市化、广播、电影和大众消费文化发展；加拿大广播公司于1936年成立。
- 魁北克、草原和大西洋地区对帝国、联邦和经济政策的看法差异继续影响政党政治。
- 寄宿学校和《印第安人法》下的强制同化持续，原住民族政治组织同时发展并抵抗压迫。

## 重要事件

| 时间 | 事件 | 影响 |
|---|---|---|
| 1914年-1918年 | 第一次世界大战 | 军事牺牲、联邦权力扩张和社会分裂并存。 |
| 1917年 | 征兵危机 | 深化英法政治裂痕，促成联合政府。 |
| 1919年 | 加入国际联盟 | 加拿大国际地位提高。 |
| 1926年 | 金—宾危机与帝国会议 | 推动总督角色和自治领平等原则重塑。 |
| 1931年 | 《威斯敏斯特法令》 | 立法自主取得关键法律基础。 |
| 1929年-1939年 | 大萧条 | 推动国家重新思考财政联邦制和社会保障。 |
| 1939年-1945年 | 第二次世界大战 | 加拿大以更自主方式参战，工业能力和国际影响显著上升。 |
| 1945年 | 联合国成立 | 加拿大成为创始会员国，战后外交进入多边主义阶段。 |
