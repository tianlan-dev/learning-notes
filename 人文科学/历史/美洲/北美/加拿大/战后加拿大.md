# 战后加拿大

## 时间

1945年-1982年。

## 概括

第二次世界大战以后，加拿大从战时工业经济转向长期增长、城市化和大众消费社会，并通过公民身份、社会保障、医疗保险、养老金和财政转移建立更完整的公共制度。加拿大加入联合国、北约和北美防空体系，在与美国关系日益密切的同时发展多边外交与维和传统。

国内政治的关键变化来自三个方向：魁北克“寂静革命”和法语权利推动双语联邦重构；移民制度改革与多元文化政策改变国家认同；原住民族反对同化政策、争取土地和自治，迫使联邦重新讨论条约与固有权利。1982年宪法回归和《权利与自由宪章》把这些变化带入新的宪政阶段。

## 国家与疆域完善

| 时间 | 变化 | 意义 |
|---|---|---|
| 1947年 | 《加拿大公民身份法》生效 | 首次建立独立于英国臣民身份的加拿大公民身份。 |
| 1949年 | 纽芬兰加入联邦 | 成为第十省；2001年省名改为“纽芬兰与拉布拉多”。 |
| 1949年 | 终止向英国枢密院司法委员会提出新的上诉 | 加拿大最高法院成为国内最终上诉法院。 |
| 1965年 | 枫叶旗启用 | 形成区别于英国帝国旗帜的现代国家象征。 |
| 1982年 | 宪法回归 | 加拿大取得国内宪法修改程序，不再依赖英国议会立法。 |

## 经济增长与福利国家

### 社会制度

| 制度 | 关键时间 | 内容 |
|---|---|---|
| 失业保险 | 1940年建立，战后扩展 | 把经济风险管理更多纳入联邦责任。 |
| 家庭津贴 | 1944年建立，战后实施 | 向有子女家庭提供全国性现金支持。 |
| 均等化制度 | 1957年 | 通过联邦转移帮助财政能力较弱省份提供公共服务。 |
| 医院保险 | 1957年联邦立法 | 联邦分担符合条件的省级医院保险成本。 |
| 加拿大养老金计划 / 魁北克养老金计划 | 1965年立法、1966年实施 | 建立缴费型公共养老金；魁北克运行独立但协调的计划。 |
| 医疗保险 | 1966年联邦立法 | 通过联邦与省分担机制推动全民医疗服务在各省普及。 |
| 失业保险扩展 | 1971年 | 覆盖范围和福利扩大，体现战后社会政策高峰。 |

医疗、教育和社会服务主要由省负责，联邦通常通过税收、转移支付和全国标准发挥影响。因此“加拿大福利国家”不是完全由中央政府统一管理的制度。

### 经济与区域关系

- 战后资源、制造业和城市建设快速增长，南安大略—魁北克工业带与美国市场联系加深。
- 1965年《汽车产品协定》整合加美汽车产业，成为后来自由贸易安排的重要前奏。
- 大西洋地区、北部、草原和工业中心发展不均，促使联邦加强区域发展与财政转移。
- 1970年代石油危机扩大阿尔伯塔能源利益与联邦政策之间的冲突；1980年“国家能源计划”进一步激化西部不满。
- 工会、公共部门和女性就业扩大，家庭结构和高等教育也发生显著变化。

## 外交与安全

| 时间 | 事件 / 机制 | 说明 |
|---|---|---|
| 1945年 | 联合国创始会员国 | 加拿大把多边外交作为中等强国战略的重要部分。 |
| 1949年 | 加入北约 | 在冷战中成为西方军事联盟成员。 |
| 1950年-1953年 | 参加朝鲜战争 | 在联合国框架下投入军队。 |
| 1956年 | 苏伊士运河危机 | 外长莱斯特·皮尔逊推动紧急部队方案，强化加拿大维和国家形象。 |
| 1958年 | 北美防空司令部协定 | 与美国共同建设大陆防空体系，体现安全依赖与主权协调。 |
| 冷战时期 | 北极预警线、北约部署与核武争论 | 国防政策不断在美加合作、联盟义务和国内政治之间平衡。 |

“维和国家”是战后加拿大认同的重要部分，但加拿大同时参与北约、北美防空和多场战争，不能把其外交简化为只参与维和。

## 魁北克、语言与国家统一

- 1960年代魁北克“寂静革命”推动教育、经济和社会服务世俗化，省政府能力与法语民族主义同步增强。
- 1963年至1969年皇家双语与双文化委员会调查英法关系，并推动联邦语言改革。
- 1969年《官方语言法》确立英语和法语在联邦机构的官方地位；它不意味着所有省和居民都必须日常使用双语。
- 1970年“十月危机”中，魁北克解放阵线绑架英国外交官和魁北克部长；联邦动用《战时措施法》，引发安全与公民自由争论。
- 1976年魁北克人党执政，1977年《法语宪章》强化法语在魁北克公共生活中的地位。
- 1980年魁北克第一次主权公投中，“主权—联系”方案被否决，但宪政和民族认同争议并未结束。

## 移民、多元文化与权利

- 1947年以后，加拿大逐步取消以种族和国籍为基础的排斥性移民规则。
- 1967年积分制更多依据教育、技能和语言等指标选择经济移民，移民来源日益多元。
- 1960年，具有法定“印第安人”身份者获得联邦选举权，而无需为投票放弃身份。
- 1960年《加拿大权利法案》以联邦普通法律形式保障部分权利，但适用范围和效力有限。
- 1971年联邦宣布官方多元文化政策，试图在英法双语框架内承认更广泛的文化多样性。
- 1976年《移民法》重整移民类别、难民政策和联邦—省合作。

## 原住民族政治与土地权利

| 时间 | 事件 | 影响 |
|---|---|---|
| 1950年代-1980年代 | “六十年代强制带离”及相关儿童福利政策 | 大量原住民儿童被带离家庭并由非原住民家庭收养或寄养，造成身份和社区联系断裂。 |
| 1969年 | 联邦“白皮书” | 提议取消特殊法律身份和《印第安人法》框架；遭原住民族广泛反对，被迫撤回。 |
| 1973年 | 卡尔德案 | 最高法院承认原住民所有权在殖民法律之前已存在，推动联邦建立综合土地权利谈判政策。 |
| 1975年 | 《詹姆斯湾与北魁北克协议》 | 因纽特人和克里人与政府达成现代土地权利协议，成为现代条约的重要范例。 |
| 1982年 | 《宪法法》第35条 | 确认现有的原住民族与条约权利，并把第一民族、因纽特人和梅蒂人纳入宪法所称原住民族。 |

寄宿学校仍在运行，联邦同化政策没有因战后民主化自动结束；原住民组织、妇女和地方社群持续推动制度改变。

## 1982年宪法重组

### 国家元首

| 角色 | 本阶段变化 |
|---|---|
| 加拿大君主 | 伊丽莎白二世于1952年即位，1953年获得单独的“加拿大女王”正式称号；加拿大王冠作为独立宪政角色更加明确。 |
| 总督 | 主要依加拿大总理和内阁建议履行职权；1952年文森特·梅西成为首位加拿大出生的总督。 |

### 宪法成果

- 《1982年宪法法》建立国内修宪公式，使加拿大不再请求英国议会修改核心宪法。
- 《加拿大权利与自由宪章》约束联邦和省政府，保障基本自由、民主权利、迁徙权、法律权利、平等权和语言教育权等。
- 第35条确认现有原住民族和条约权利，但权利的具体范围仍需谈判、立法和法院判例不断界定。
- 魁北克政府没有正式同意1982年宪法安排，成为此后宪政政治的重要争议来源。

## 重要事件

| 时间 | 事件 | 影响 |
|---|---|---|
| 1949年 | 纽芬兰加入、最高法院最终上诉地位确立 | 完善疆域与司法自主。 |
| 1956年 | 苏伊士危机 | 强化多边外交和维和认同。 |
| 1960年 | 原住民联邦选举权扩大 | 取消投票必须放弃法定身份的要求。 |
| 1965年 | 枫叶旗、养老金与汽车协定 | 国家象征、社会保障和北美经济整合同步变化。 |
| 1969年 | 《官方语言法》与“白皮书”争议 | 双语联邦加强，同时原住民族反对同化方案。 |
| 1970年 | 十月危机 | 国家安全、魁北克民族主义与公民自由发生尖锐冲突。 |
| 1980年 | 魁北克第一次主权公投 | 联邦方案获胜，宪法改革压力增加。 |
| 1982年 | 宪法回归 | 开启以宪章、原住民族权利和国内修宪为核心的新阶段。 |
