# 当代加拿大

## 时间

1982年至今。

## 概括

1982年以后，加拿大政治以《权利与自由宪章》、联邦与省关系、魁北克国家认同和原住民族权利为核心。宪章增强法院审查公共政策的作用，国内修宪公式终结了依赖英国议会修改宪法的制度；但魁北克政府未正式同意1982年安排，随后两次宪法协议失败和1995年独立公投使国家统一问题长期居于政治中心。

同一时期，加拿大通过自由贸易更深入融入北美经济，移民持续改变人口结构，多元文化、性别平等和性少数权利获得更强法律保障。寄宿学校幸存者、原住民族政治组织和法院判例推动和解、土地权利和自治议程，但殖民制度的长期影响、失踪与被害原住民女性和女孩、饮水住房与土地争端等问题仍未解决。

## 1982年宪政框架

| 组成 | 内容 | 长期影响 |
|---|---|---|
| 《权利与自由宪章》 | 保障基本自由、民主、迁徙、法律、平等和语言教育等权利 | 法院可审查政府法律与政策；第1条允许符合理性限制，第33条允许部分权利在一定条件下被暂时排除适用。 |
| 第35条 | 确认现有原住民族与条约权利 | 成为土地、捕鱼、自治和协商义务等案件的重要宪法基础。 |
| 国内修宪公式 | 按事项采用全体同意、一般“七省且人口过半”等不同程序 | 加拿大不再请求英国议会修宪，但高门槛也使重大制度改革困难。 |
| 联邦制延续 | 联邦与省各有宪法权限 | 医疗、教育、资源、环境、移民和语言政策仍需跨层级协调。 |

《宪章》第15条平等权延至1985年生效。魁北克省议会没有同意1982年宪法方案，但该宪法在魁北克同样具有法律效力。

## 魁北克与国家统一

| 时间 | 事件 | 结果与意义 |
|---|---|---|
| 1987年-1990年 | 米奇湖协议 | 试图取得魁北克对宪法的正式认同并调整联邦制度，因未获全部省份批准而失败。 |
| 1992年 | 夏洛特敦协议公投 | 包含魁北克、原住民族自治和联邦机构改革的方案被全国公投否决。 |
| 1995年 | 魁北克第二次独立公投 | 反对独立一方以极小差距胜出。 |
| 1998年 | 最高法院“魁北克分离案”意见 | 认为单方面分离不符合加拿大宪法，但明确多数支持清晰问题时各方负有谈判义务。 |
| 2000年 | 《清晰法》 | 规定联邦众议院评估公投问题和多数是否清晰的程序。 |

魁北克语言和世俗主义政策持续引发省权、少数语言权利、宗教自由和加拿大多元文化模式之间的争论。

## 原住民族权利、自治与和解

| 时间 | 事件 / 进程 | 意义 |
|---|---|---|
| 1985年 | 《印第安人法》修正 | 纠正部分因性别歧视而失去法定身份的问题，但后续仍需继续修法。 |
| 1990年 | 奥卡危机 | 莫霍克人保卫土地，暴露殖民土地争端和警务、军事回应问题。 |
| 1996年 | 原住民皇家委员会报告；最后一所受联邦管理的寄宿学校关闭 | 系统说明殖民政策影响，并推动重建民族间关系的讨论。 |
| 1999年 | 努纳武特成立 | 源于因纽特土地权利协议，建立以因纽特人口为主体的公共领地政府。 |
| 2008年 | 联邦政府就寄宿学校正式道歉；真相与和解委员会开始工作 | 幸存者证词和历史责任进入全国公共议程。 |
| 2015年 | 真相与和解委员会最终报告 | 提出94项行动呼吁，把寄宿学校制度界定为文化灭绝政策的一部分。 |
| 2019年 | 失踪与被害原住民女性和女孩全国调查最终报告 | 指出长期、系统性的暴力与制度失灵。 |
| 2021年 | 《联合国原住民族权利宣言法》 | 要求联邦法律逐步与联合国宣言保持一致并制定行动计划。 |

和解并不是已经完成的历史阶段。现代条约、自治协议、土地返还、资源开发协商、儿童福利、语言复兴和公共服务仍在持续谈判。

## 经济与北美一体化

| 时间 | 安排 / 变化 | 影响 |
|---|---|---|
| 1989年 | 加美自由贸易协定生效 | 降低双边关税，引发产业调整、经济主权与市场机会争论。 |
| 1994年 | 北美自由贸易协定生效 | 把墨西哥纳入加美自由贸易体系，形成更紧密的大陆供应链。 |
| 2020年 | 加美墨协定生效 | 取代北美自由贸易协定，更新汽车、劳工、数字贸易和争端规则。 |
| 20世纪90年代 | 联邦削减赤字与财政重组 | 改变联邦项目和对省转移支付，后来恢复财政空间。 |
| 21世纪 | 能源、矿产、科技与服务业并行 | 资源省份、制造业中心和大城市之间利益差异继续影响联邦政治。 |

加拿大经济高度依赖对美贸易，同时通过跨太平洋和跨大西洋协定分散市场。能源开发、管道、碳定价和气候政策经常引发联邦、省、原住民族和产业之间的管辖权争论。

## 社会与权利变化

- 1988年《加拿大多元文化主义法》把联邦多元文化政策写入法律。
- 《宪章》推动性别平等、刑事程序、少数语言教育和性少数权利等领域的判例发展。
- 2005年加拿大在全国范围承认同性婚姻。
- 移民和难民继续成为人口增长的重要来源，多伦多、温哥华、蒙特利尔等都会区高度多元。
- 1989年蒙特利尔工程学院枪击案、针对原住民女性的暴力和其他事件推动公共讨论性别暴力与枪械政策。
- 2020年起的新冠疫情考验公共卫生、边境、财政和联邦—省协调，也放大长期护理、住房和不平等问题。
- 人口老龄化、住房负担、法语与英语关系、北部主权、气候变化和生产率成为当代国家治理的重要议题。

## 主要政治阶段

| 时间 | 总理 / 政治阶段 | 主要议题 |
|---|---|---|
| 1982年-1984年 | 皮埃尔·特鲁多后期 | 宪法实施、能源与联邦关系。 |
| 1984年 | 约翰·特纳短期政府 | 自由党权力交接与大选。 |
| 1984年-1993年 | 布赖恩·马尔罗尼 | 加美自由贸易、税制改革、米奇湖和夏洛特敦协议。 |
| 1993年 | 金·坎贝尔短期政府 | 首位女性总理，进步保守党在大选中惨败。 |
| 1993年-2003年 | 让·克雷蒂安 | 财政整顿、1995年魁北克公投与《清晰法》。 |
| 2003年-2006年 | 保罗·马丁 | 少数政府、同性婚姻和联邦—原住民族政策谈判。 |
| 2006年-2015年 | 斯蒂芬·哈珀 | 保守党少数及多数政府、2008年寄宿学校道歉、资源与外交政策。 |
| 2015年-2025年 | 贾斯廷·特鲁多 | 多数及少数政府、气候与社会政策、新冠疫情和北美贸易协定更新。 |
| 2025年至今 | 马克·卡尼 | 2025年就任自由党政府总理；后续任期持续更新。 |

完整任期和历任顺序见[加拿大总理表](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E7%BE%8E%E6%B4%B2/%E5%8C%97%E7%BE%8E/%E5%8A%A0%E6%8B%BF%E5%A4%A7/%E5%8A%A0%E6%8B%BF%E5%A4%A7%E6%80%BB%E7%90%86%E8%A1%A8.md)。

| 角色 | 现任 | 时间 | 说明 |
|---|---|---|---|
| 加拿大总理 | 马克·卡尼 | 2025年至今 | 领导内阁并维持众议院信任。 |

### 议会与联邦

| 机构 | 主要职能 |
|---|---|
| 总督 | 代表君主履行多数联邦宪政和礼仪职责，通常依总理与内阁建议行事。 |
| 众议院 | 民选议院，决定政府能否维持信任。 |
| 参议院 | 任命产生的上院，按区域分配席位并审议立法。 |
| 省与地区政府 | 管理各自权限领域；地区权力主要来自联邦立法并持续权力下放。 |
| 最高法院 | 最终解释宪法、联邦制和宪章权利。 |

## 重要事件

| 时间 | 事件 | 影响 |
|---|---|---|
| 1982年 | 宪法回归 | 开启宪章司法、国内修宪和第35条权利的新阶段。 |
| 1990年 | 米奇湖协议失败与奥卡危机 | 国家统一和原住民土地问题同时进入危机。 |
| 1994年 | 北美自由贸易协定 | 大陆经济整合加深。 |
| 1995年 | 魁北克独立公投 | 联邦以极小差距维持统一。 |
| 1999年 | 努纳武特成立 | 因纽特土地权利与自治取得重要制度成果。 |
| 2005年 | 同性婚姻全国合法化 | 宪章时代平等权发展的标志之一。 |
| 2008年-2015年 | 寄宿学校道歉与真相和解委员会 | 国家承认殖民教育制度伤害，和解成为长期公共议程。 |
| 2020年 | 加美墨协定生效、新冠疫情暴发 | 贸易制度更新与公共卫生危机同时考验政府。 |
| 2022年 | 查尔斯三世即位 | 加拿大君主完成王位继承。 |
| 2025年 | 马克·卡尼就任总理 | 联邦政府进入新的领导阶段。 |
