# 加拿大联邦建立与西部扩张

## 时间

1867年-1914年。

## 概括

1867年，加拿大省、新斯科舍和新不伦瑞克组成联邦，加拿大省同时分为安大略和魁北克。新国家采用君主立宪、责任政府和联邦制，在统一国防、贸易和交通的同时，把教育、财产与民权等重要事务留给各省。此时的加拿大仍处于大英帝国之内，英国继续掌握重要外交和宪法权力。

联邦随后取得鲁珀特地和西北地区，吸收不列颠哥伦比亚与爱德华王子岛，并建立曼尼托巴、育空、阿尔伯塔和萨斯喀彻温。铁路和移民把横跨大陆的市场连接起来，但“西部扩张”同时意味着对第一民族和梅蒂人领土的占有、条约承诺与违背、保留地制度、《印第安人法》和寄宿学校体系扩张。国家建设、经济增长和殖民压迫是同一过程的不同侧面。

## 联邦建立

1867年《英属北美法》建立的不是单一制国家，而是联邦。其核心设计是：联邦政府负责国防、贸易、货币、刑法和“印第安人及为印第安人保留的土地”等事务；各省负责教育、地方制度、财产与民权等领域。具体权限后来不断由法院判例、政治谈判和宪法修正塑造。

### 国家元首体系

| 角色 | 位置 | 说明 |
|---|---|---|
| 加拿大君主 | 国家元首 | 1867年联邦在维多利亚女王统治下建立；君主序列见[加拿大君主表](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E7%BE%8E%E6%B4%B2/%E5%8C%97%E7%BE%8E/%E5%8A%A0%E6%8B%BF%E5%A4%A7/%E5%8A%A0%E6%8B%BF%E5%A4%A7%E5%90%9B%E4%B8%BB%E8%A1%A8.md)。 |
| 总督 | 君主在联邦层面的代表 | 任命总理、召集和解散议会、给予御准；此时仍兼具帝国代表性质。 |

### 政府与议会

| 角色 / 机构 | 组成与职能 | 说明 |
|---|---|---|
| 总理与内阁 | 领导联邦行政并对众议院负责 | 首任总理是约翰·A·麦克唐纳；完整序列见[加拿大总理表](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E7%BE%8E%E6%B4%B2/%E5%8C%97%E7%BE%8E/%E5%8A%A0%E6%8B%BF%E5%A4%A7/%E5%8A%A0%E6%8B%BF%E5%A4%A7%E6%80%BB%E7%90%86%E8%A1%A8.md)。 |
| 众议院 | 按选区选举产生 | 政府必须维持众议院信任；选举权最初受性别、财产、种族和原住民身份等限制。 |
| 参议院 | 由总督依总理建议任命 | 代表区域并审议法案，制度设计也用于平衡人口较少地区。 |
| 省政府与省议会 | 在宪法列举的省级领域立法和行政 | 省长由联邦任命的副总督代表君主；联邦早期曾频繁保留或否决省级法律。 |
| 司法机关 | 联邦与省级法院 | 加拿大最高法院于1875年建立，但英国枢密院司法委员会仍长期是最高上诉机构。 |

## 疆域与省份扩展

| 时间 | 变化 | 说明 |
|---|---|---|
| 1867年 | 安大略、魁北克、新斯科舍、新不伦瑞克组成联邦 | 四省是联邦创始成员。 |
| 1870年 | 鲁珀特地和西北地区转归加拿大；曼尼托巴建立 | 土地转让未充分征得当地第一民族和梅蒂人同意，引发红河抵抗。 |
| 1871年 | 不列颠哥伦比亚加入 | 联邦承诺修建跨大陆铁路。 |
| 1873年 | 爱德华王子岛加入 | 铁路债务和财政条件促成入联邦。 |
| 1880年 | 英国把北极群岛主权主张移交加拿大 | 加拿大获得北部广阔岛屿的帝国法理控制，但实际行政与因纽特社会生活之间仍有巨大距离。 |
| 1898年 | 育空从西北地区分设 | 克朗代克淘金潮促使联邦强化北部行政。 |
| 1905年 | 阿尔伯塔与萨斯喀彻温建立 | 草原南部主要地区形成两个新省。 |

纽芬兰在这一时期仍是独立于加拿大的英国殖民地，1949年才成为加拿大第十省。

## 梅蒂人、第一民族与殖民政策

### 红河与西北抵抗

- 1869年至1870年，联邦准备接收鲁珀特地时，没有充分处理红河梅蒂人的土地、语言、宗教和政治权利。路易·里尔领导临时政府，通过谈判促成《曼尼托巴法》。
- 《曼尼托巴法》承诺省级地位、语言与宗教保障以及处理梅蒂人土地权利，但土地分配延误、投机和定居压力促使许多梅蒂人向西迁移。
- 1885年，萨斯喀彻温河流域的梅蒂人再次组织临时政府；部分第一民族也因饥饿、条约执行和殖民管制参与抵抗。联邦军队镇压西北抵抗，里尔随后被处决。
- 1885年不是原住民和梅蒂政治的终点。土地、自治、条约与身份权利继续成为加拿大政治议题。

### 条约与《印第安人法》

| 政策 / 制度 | 时间 | 内容与影响 |
|---|---|---|
| 编号条约 | 1871年起，本阶段签署至第10号条约 | 王室与多个第一民族签署土地共享、年金、保留地和援助等安排；双方对条约含义的理解常不一致，政府履约长期存在争议。 |
| 《印第安人法》 | 1876年 | 把既有殖民法规整合为联邦管制框架，规定法定身份、保留地管理和部落治理，并支持同化政策。 |
| 文化禁令 | 19世纪后期起 | 法律限制夸富宴等宗教与文化实践，干预原住民族社会制度。 |
| 寄宿学校体系 | 1880年代后加速扩张 | 联邦与教会合作把儿童从家庭和社区带走，以同化为目标，造成虐待、死亡、语言流失和跨代创伤。 |

## 铁路、经济与移民

- 1879年“国家政策”以保护性关税、铁路建设和西部定居为核心，试图形成东西向国内市场。
- 加拿大太平洋铁路于1885年贯通。它加强联邦控制和商品流动，也便利军队镇压1885年抵抗及定居者进入原住民族领土。
- 大批华工参与太平洋铁路西段建设。1885年联邦开始征收华人人头税，显示国家移民政策中的种族歧视。
- 1896年以后，草原小麦种植、欧洲和美国移民、城市工业及矿业迅速发展。
- 克朗代克淘金潮推动育空行政建立，也加剧对原住民土地、资源和生活方式的冲击。
- 加拿大参与1899年至1902年南非战争引发帝国忠诚与法裔加拿大民族主义之间的争论。
- 1910年《海军法》建立加拿大海军，同样引起“支持帝国”与“自主防务”之争。

## 重要事件

| 时间 | 事件 | 影响 |
|---|---|---|
| 1869年-1870年 | 红河抵抗 | 推动曼尼托巴以省的形式加入联邦，并暴露梅蒂土地权利冲突。 |
| 1873年 | 西北骑警建立 | 联邦以警务和行政力量加强对草原与西北地区的控制。 |
| 1876年 | 《印第安人法》 | 建立延续多代的联邦原住民管制制度。 |
| 1885年 | 西北抵抗、铁路贯通与华人人头税 | 同一年集中体现国家整合、军事镇压、基础设施与排斥性移民政策。 |
| 1896年-1899年 | 克朗代克淘金潮 | 育空人口和行政急剧变化。 |
| 1905年 | 阿尔伯塔、萨斯喀彻温建省 | 草原省份格局基本形成。 |
| 1914年 | 第一次世界大战爆发 | 加拿大以大英帝国自治领身份参战，国家自主与社会分裂进入新阶段。 |
