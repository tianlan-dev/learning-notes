# 波菲里奥统治与墨西哥革命

## 时间

1876年-1920年

## 概括

波菲里奥·迪亚斯通过军政联盟、地方精英、选举控制和“秩序与进步”政策长期掌权。铁路、矿业、出口和外资增长推动经济整合，却加剧土地集中、劳工冲突和政治排斥。1910年反连任运动引发革命，马德罗、萨帕塔、比利亚、卡兰萨和奥夫雷贡等力量先后合作与交战。

## 国家元首

| 人物 | 任期 | 说明 |
|---|---|---|
| 波菲里奥·迪亚斯 | 1876-1880年、1884-1911年 | 通过连续连任和精英联盟形成长期统治。 |
| 曼努埃尔·冈萨雷斯 | 1880-1884年 | 迪亚斯盟友，在迪亚斯两段总统任期之间执政。 |
| 弗朗西斯科·马德罗 | 1911-1913年 | 反连任运动领袖，在政变中被杀。 |
| 维克托里亚诺·韦尔塔 | 1913-1914年 | 通过政变掌权，遭宪政派和其他革命力量反对。 |
| 贝努斯蒂亚诺·卡兰萨 | 1914-1920年间为宪政派最高领导，1917-1920年任总统 | 推动1917年宪法，后在反对派起事中被杀。 |

## 革命阵营与诉求

| 阵营 / 领袖 | 核心区域 | 主要诉求或特点 |
|---|---|---|
| 马德罗派 | 全国反连任网络 | 反对迪亚斯连任，主张选举政治。 |
| 萨帕塔派 | 莫雷洛斯等南部地区 | 以《阿亚拉计划》要求土地归还和乡村自治。 |
| 比利亚派 | 奇瓦瓦等北部地区 | 依托北方军队、地方社会和土地改革诉求。 |
| 卡兰萨与宪政派 | 科阿韦拉、索诺拉等 | 反对韦尔塔并重建宪政国家，内部亦有派系。 |
| 奥夫雷贡派 | 索诺拉及宪政派军队 | 以军事和政治联盟击败主要对手，1920年掌权。 |

## 重要事件

| 时间 | 事件 | 说明 |
|---|---|---|
| 1910年 | 《圣路易斯波托西计划》与起义 | 墨西哥革命爆发。 |
| 1911年 | 迪亚斯辞职 | 长期统治结束，马德罗当选。 |
| 1913年 | “悲惨十日”政变 | 马德罗被推翻并遇害，韦尔塔掌权。 |
| 1914年 | 韦尔塔倒台 | 革命联盟随后分裂为相互战争的阵营。 |
| 1917年 | 新宪法 | 纳入土地、劳工、教育和国家资源权等重要条款。 |
| 1920年 | 阿瓜普列塔起义 | 卡兰萨倒台，奥夫雷贡阵营掌权。 |

## 关键辨析

- 波菲里奥时期不是完全停滞的独裁时代；经济现代化真实存在，但收益、土地和政治权利分配高度不均。
- 墨西哥革命没有单一开端、统一领导或所有参与者共同接受的终点。
- 1917年宪法规定的社会权利需要在后续几十年通过政治斗争和国家建设逐步落实。
