# 独立、第一帝国与早期共和国

## 时间

1810年-1855年

## 概括

墨西哥独立战争从伊达尔戈、莫雷洛斯等领导的群众起义开始，后由阿古斯丁·德·伊图尔维德与反叛力量结盟完成独立。第一帝国很快倒台，联邦共和国与中央集权体制反复转换。财政薄弱、军人政治、地区自治、教会与军队特权、得克萨斯分离和美墨战争使国家长期处于危机。

## 重要事件

| 时间 | 事件 | 影响 |
|---|---|---|
| 1810年 | 伊达尔戈发出多洛雷斯呼声 | 独立战争爆发。 |
| 1813-1815年 | 莫雷洛斯的制宪与战争 | 提出独立、主权和社会改革纲领，后被捕处决。 |
| 1821年 | 《伊瓜拉计划》与《科尔多瓦条约》 | 独立军进入墨西哥城，新西班牙统治终结。 |
| 1822-1823年 | 墨西哥第一帝国 | 伊图尔维德称帝，随后被共和派推翻。 |
| 1824年 | 联邦宪法 | 建立墨西哥合众国和联邦共和国。 |
| 1836年 | 得克萨斯分离 | 北部边疆矛盾和移民问题升级为战争。 |
| 1846-1848年 | 美墨战争 | 墨西哥战败并失去大片北部领土。 |
| 1854-1855年 | 阿尤特拉革命 | 推翻圣安纳，开启自由改革。 |

## 国家元首与统治结构

### 帝国

| 统治者 | 在位时间 | 说明 |
|---|---|---|
| 阿古斯丁一世 | 1822-1823年 | 原独立军领袖伊图尔维德，第一帝国唯一皇帝。 |

### 共和国主要总统与阶段

| 人物或阶段 | 时间 | 说明 |
|---|---|---|
| 瓜达卢佩·维多利亚 | 1824-1829年 | 首任宪法总统，完成完整任期。 |
| 维森特·格雷罗 | 1829年 | 独立战争领袖，任期在政变中结束。 |
| 安纳斯塔西奥·布斯塔曼特 | 1830年代至1840年代多次任职 | 代表保守和中央集权政治的重要军人领袖。 |
| 安东尼奥·洛佩斯·德·圣安纳 | 1833-1855年间多次任职 | 在联邦、中央制、得克萨斯和美墨战争等危机中反复掌权。 |
| 临时和代总统政府 | 多次 | 政变、战争和财政危机导致行政更替频繁。 |

### 实际权力结构

| 力量 | 作用 |
|---|---|
| 总统与军队 | 军人领袖和地方武装频繁通过政变改变政府。 |
| 州与地方集团 | 联邦主义与中央集权冲突反映地区利益和国家能力不足。 |
| 教会与军队特权 | 财产、司法特权和政治地位成为自由派与保守派争论中心。 |

## 关键辨析

- 独立初期的“自由派”“保守派”内部都不断变化，不能直接套用后来的政党定义。
- 圣安纳并非整个时期唯一统治者；他多次上台、离任和复出，临时政府数量很多。
- 美墨战争后的领土变化还涉及当地原住民族、墨西哥居民和美国定居者，不能只看国家地图。
