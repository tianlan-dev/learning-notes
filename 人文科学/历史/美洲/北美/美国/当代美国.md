# 当代美国

## 时间

1991年至今。

## 概括

冷战结束后，美国一度拥有显著的全球军事、金融和技术优势。全球化、互联网和服务经济创造新产业，也推动制造业地域变化、财富集中和地区不平等。2001年九一一袭击把国家安全与海外战争置于政治中心；2008年金融危机、种族正义运动、移民争论、新冠疫情和党派极化进一步改变社会。美国仍是全球重要强国，但国内制度信任、民主规则、社会公平以及对外承诺成为持续争议。

## 主要阶段

### 冷战后与全球化

- 1991年海湾战争显示美国在冷战后组织多国军事行动的能力。
- 1994年北美自由贸易协定生效，扩大美加墨贸易与供应链，也引发就业、工资和产业转移争论。
- 互联网、个人计算机和金融创新推动1990年代增长，技术行业和知识经济地位上升。
- 克林顿政府时期财政一度出现盈余，但福利改革、大规模监禁和金融放松等政策后来受到重新评价。
- 1998年众议院弹劾克林顿，参议院未将其定罪。

### 九一一与反恐战争

- 2001年9月11日恐怖袭击造成近三千人死亡，美国随后发动阿富汗战争并建立国土安全部。
- 《爱国者法》等措施扩大监控与执法权，引发安全和公民自由争论。
- 2003年美国以伊拉克拥有大规模杀伤性武器等理由发动战争，未发现预期库存，长期占领造成巨大人员、财政和地区代价。
- 2005年卡特里娜飓风暴露灾害治理、基础设施和种族阶级差距。

### 金融危机与社会重组

- 2007-2009年住房与金融危机导致严重衰退，政府采取银行救助、货币宽松和经济刺激。
- 2010年医疗改革扩大保险覆盖并重塑医疗市场，同时成为长期党争焦点。
- 数字平台、智能手机和社交媒体改变新闻、商业、选举与社会关系，也带来隐私、垄断和虚假信息问题。
- 2015年最高法院判决使同性婚姻在全美合法化。
- 黑人生命也是命等运动使警察暴力、刑事司法和结构性种族不平等成为全国政治议题。

### 极化、疫情与新产业政策

- 2016年特朗普当选反映对移民、贸易、文化和政治建制的强烈分歧；其第一任期推动减税、去监管和更具对抗性的贸易政策。
- 特朗普在任内两次遭众议院弹劾，均未被参议院定罪。
- 2020年新冠疫情造成大规模死亡、教育与经济中断；联邦、州和地方在公共卫生措施上分歧显著。
- 2020年乔治·弗洛伊德死亡引发全国和国际抗议，也伴随对治安、抗议秩序与制度改革的争论。
- 2021年1月6日国会大厦遇袭，围绕选举结果、权力和平移交和政治暴力的争论持续。
- 拜登政府推动基础设施、半导体、清洁能源和制造业投资，并在俄乌战争中协调盟友支持乌克兰。
- 2024年特朗普再次当选，2025年开始第二个不连续任期，显示美国政党竞争和社会分裂仍处高强度状态。

## 长期议题

| 主题 | 核心问题 |
|---|---|
| 联邦民主 | 选举人团、选区划分、投票规则、最高法院与总统权力如何保持制衡。 |
| 经济结构 | 技术创新与生产率如何同工资、住房、医疗、教育成本和地区不平等协调。 |
| 移民与边境 | 劳动力需求、人道庇护、无证移民、家庭团聚和边境执法如何平衡。 |
| 种族与公民权 | 法律平等之后，财富、教育、住房、健康和刑事司法差距如何处理。 |
| 原住民主权 | 条约、部落司法、土地与资源、儿童福利和文化保护如何落实。 |
| 全球角色 | 同盟、战争授权、贸易、气候与大国竞争之间如何分配资源。 |
| 技术治理 | 人工智能、平台权力、隐私、网络安全和信息生态如何监管。 |
