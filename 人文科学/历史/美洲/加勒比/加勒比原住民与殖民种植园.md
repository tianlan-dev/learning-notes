# 加勒比原住民与殖民种植园

## 时间

1492年前后的原住民社会至19世纪奴隶制废除；其社会遗产延续至今。

## 概括

大安的列斯和小安的列斯群岛在欧洲到来前已有人口迁徙、航海、农耕、渔猎、贸易和政治网络。西班牙征服带来疾病、战争、奴役和人口崩溃；随后英国、法国、荷兰、丹麦等争夺岛屿。糖、咖啡、烟草和靛蓝种植园依靠被奴役的非洲人劳动，形成极端不平等的社会。逃奴社群、起义、宗教与文化混合持续挑战殖民秩序。

## 殖民体系

| 力量 | 主要区域 / 影响 |
|---|---|
| 西班牙 | 伊斯帕尼奥拉、古巴、波多黎各及早期加勒比核心。 |
| 法国 | 圣多明各、马提尼克、瓜德罗普等，糖业和奴隶制高度发展。 |
| 英国 | 牙买加、巴巴多斯及多座小安的列斯岛，形成英属西印度群岛。 |
| 荷兰 | 库拉索、阿鲁巴、苏里南及贸易网络。 |
| 丹麦等 | 部分小岛与港口贸易，后多有主权转移。 |

## 重要过程

- 泰诺、卡利布等名称涉及不同语言与历史传统，不能把所有岛屿原住民概括为单一民族。
- 1492年后西班牙在伊斯帕尼奥拉建立殖民据点，恩科米恩达、矿业和强迫劳动造成巨大伤害。
- 17世纪糖业技术、欧洲消费和奴隶贸易结合，使加勒比成为大西洋种植园经济中心。
- 被奴役者通过日常抵抗、逃亡、宗教和武装起义争取自主；牙买加、苏里南等地的马龙社群形成持久共同体。
- 英帝国1830年代废除奴隶制，法属殖民地1848年废奴，西班牙加勒比的废奴更晚；不同殖民地的日期和过渡制度不能混为一谈。
- 契约劳工制度在废奴后引入印度、华人等劳工，进一步改变人口结构。
