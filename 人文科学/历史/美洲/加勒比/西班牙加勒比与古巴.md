# 西班牙加勒比与古巴

## 时间

16世纪至今；19世纪末独立战争与1898年美西战争为关键转折。

## 概括

古巴、波多黎各和伊斯帕尼奥拉东部长期处于西班牙加勒比体系之中。古巴在19世纪仍保留奴隶制和糖业，并经历多次独立战争；1898年美西战争后，西班牙帝国在加勒比的主导地位结束。古巴于1902年建立共和国，1959年革命后走上不同发展道路；波多黎各成为美国属地，具有特殊宪政地位；多米尼加共和国则经历独立、外部占领和共和国政治的复杂路径。

## 主要节点

| 时间 | 事件 | 意义 |
|---|---|---|
| 1492年后 | 西班牙殖民加勒比 | 伊斯帕尼奥拉、古巴和波多黎各成为帝国早期据点。 |
| 1868-1878年 | 古巴十年战争 | 古巴独立运动的重要前史。 |
| 1895年 | 古巴独立战争重启 | 何塞·马蒂等推动新的民族独立运动。 |
| 1898年 | 美西战争 | 西班牙失去古巴、波多黎各等地的帝国主导地位。 |
| 1902年 | 古巴共和国建立 | 形式独立但受到美国强烈影响。 |
| 1959年 | 古巴革命 | 革命政府建立，冷战中形成与美国对立的社会主义国家。 |

## 说明

- 古巴糖业、奴隶制、华工和非洲后裔文化共同塑造岛屿社会。
- 波多黎各不是独立国家，也不是美国州；其政治地位和自决问题持续存在。
- 多米尼加共和国与海地共享伊斯帕尼奥拉岛，但两国的殖民语言、独立和国家形成路径不同。
- 冷战、美国禁运、移民和旅游业使古巴史具有跨加勒比与全球维度。
