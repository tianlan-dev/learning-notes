# 中美洲与中部美洲

## 范围与概括

本目录同时处理两个相互重叠但不能混同的范围：中部美洲（Mesoamerica）是以玉米农业、历法、文字、城市与宗教传统相互联系的文化历史区，涵盖墨西哥中南部和中美洲北部；地理中美洲则是危地马拉、伯利兹、洪都拉斯、萨尔瓦多、尼加拉瓜、哥斯达黎加和巴拿马组成的陆桥。中部美洲古代文明与新西班牙核心在这里集中整理；连续的[墨西哥国家通史](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E7%BE%8E%E6%B4%B2/%E5%8C%97%E7%BE%8E/%E5%A2%A8%E8%A5%BF%E5%93%A5/README.md)按地理归入北美并回链本目录。

## 文化史主线：中部美洲

中部美洲是文化历史区，主线跨越今日墨西哥中南部、危地马拉、伯利兹及洪都拉斯、萨尔瓦多等部分地区。它以多中心文明、城市、文字、历法、市场与宗教网络为线索，不以现代国界为边界。

```mermaid
    Olmec --> Cities["特奥蒂瓦坎、玛雅、萨波特克与多中心城市网络"]

    Cities -->|玛雅等社会并未消失| Continuity["语言、社区、土地与文化传统延续"]

```

## 地理主线：中美洲

地理中美洲是墨西哥以南、哥伦比亚以北的陆桥，通常包括危地马拉、伯利兹、洪都拉斯、萨尔瓦多、尼加拉瓜、哥斯达黎加和巴拿马。其近现代主线以殖民行政、独立、联邦解体、国家形成、外部干预、内战、和平进程和跨洋交通为核心。

```mermaid
flowchart TD
    Indigenous["地理中美洲多种原住民社会"] --> Colonial["危地马拉总督辖区与其他殖民安排"]
    Colonial -->|1821 独立| Independence["脱离西班牙统治"]
    Independence --> Federation["中美洲联合省 / 中美洲联邦"]
    Federation -->|1838-1841 解体| States["各国国家形成"]

    Intervention --> Peace["内战、革命与和平进程"]
    States --> Panama["巴拿马、运河与跨洋交通"]
    Peace --> Contemporary["当代中美洲区域合作与迁移"]
    Panama --> Contemporary
    Indigenous -->|民族与社区延续| Contemporary
```

## 重要辨析

- 玛雅不是单一帝国；不同城邦、区域与后古典政治体的时间并不一致。
- 阿兹特克是对墨西加及其三方联盟的常用称呼，核心位于今墨西哥中部，不属于地理中美洲各国。
- 巴拿马地理上属于中美洲，但其哥伦比亚、运河和加勒比—太平洋联系使其历史具有跨区域性质。
- “中美洲联邦”不是今日七国的天然前身，成员、边界和实际政治整合不断变化。
