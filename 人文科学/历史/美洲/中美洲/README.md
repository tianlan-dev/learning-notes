# 中美洲

## 概括

中美洲历史以奥尔梅克、玛雅、特奥蒂瓦坎、阿兹特克、西班牙征服、新西班牙和现代中美洲国家为主线。

## 整理建议

- 前哥伦布阶段优先按文明和区域整理；殖民以后可按殖民帝国、独立运动和现代国家整理。
- 美洲现代国家很多，先建立区域主线，再逐步拆分美国、加拿大、墨西哥、巴西等国家目录。
