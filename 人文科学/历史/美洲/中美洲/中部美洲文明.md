# 中部美洲文明

## 时间

约公元前2000年至16世纪；语言、社区和文化传统延续至今。

## 概括

中部美洲是一个文化历史区，而不是一个单一帝国。玉米农业、历法、文字、城市、市场和宗教网络在不同地区相互影响，但奥尔梅克、特奥蒂瓦坎、玛雅、萨波特克、米斯特克、托尔特克和墨西加等社会各有政治结构与时间线。西班牙征服改变了国家与人口结构，并未抹去原住民语言、社区土地、宗教实践和知识传统。

## 主要传统

| 传统 / 政治体 | 时间 | 区域与特点 |
|---|---|---|
| 奥尔梅克 | 约前1500-前400年 | 墨西哥湾沿岸的重要早期复杂社会，不能简单称为所有后继文明的“母国”。 |
| 特奥蒂瓦坎 | 约1-7世纪 | 墨西哥高原大型城市，影响广泛，但其族群和政治结构仍有争议。 |
| 玛雅 | 前古典至后古典时期 | 覆盖尤卡坦、危地马拉、伯利兹和周边地区，多城邦及区域政治体并存。 |
| 萨波特克、米斯特克 | 长期发展 | 瓦哈卡高地的重要城市、文字和贵族传统。 |
| 托尔特克及后古典网络 | 约10-12世纪 | 图拉等中心与贸易、军事、宗教传统相关。 |
| 墨西加 / 三方联盟 | 1428-1521年 | 以特诺奇蒂特兰为核心的贡赋帝国，常被称为阿兹特克。 |

## 重要特征

- 特奥蒂瓦坎在1-7世纪成为美洲最大的古代城市之一，其城市规划影响后续中部美洲。
- 玛雅文字、历法和碑铭保存大量王朝政治信息，但“玛雅崩溃”只适用于部分古典期中心的转变，不能概括所有玛雅社会。
- 市场、贡赋、战争与婚姻联盟把城市与农村连接，地方社区并非只服从城市中心。
- 1521年特诺奇蒂特兰失守是西班牙征服的重要节点，但西班牙依赖特拉斯卡拉等原住民盟友；征服不是少数欧洲人单独完成。
- 疾病、战争、强制劳役与社会破坏共同造成严重人口灾难，程度和过程因地区不同而异。
