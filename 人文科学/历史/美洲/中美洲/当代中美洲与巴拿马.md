# 当代中美洲与巴拿马

## 时间

19世纪后期至今，重点为20世纪的出口经济、革命与内战、和平进程和跨境迁移。

## 概括

中美洲各国在咖啡、香蕉、糖、运河与跨洋交通的世界市场中发展，常面临土地集中、外国企业与外部干预、军队政治和社会不平等。危地马拉、萨尔瓦多和尼加拉瓜在冷战时期发生革命、内战或反叛乱战争；哥斯达黎加形成较稳定的文人民主传统；巴拿马则因运河、美国影响和服务经济具有特殊位置。和平协议和选举恢复并未终结暴力、贫困、气候灾害和迁移压力。

## 区域比较

| 地区 | 主线提示 |
|---|---|
| 危地马拉 | 土地不平等、玛雅原住民权利、长期内战与和平协议。 |
| 萨尔瓦多 | 寡头经济、内战、和平协议与侨汇社会。 |
| 洪都拉斯、尼加拉瓜 | 香蕉经济、外国干预、军政冲突与革命遗产。 |
| 哥斯达黎加 | 1948年内战后废除常备军，发展文人宪政和社会政策。 |
| 巴拿马 | 1903年建国、运河区历史、1999年收回运河管理与服务经济。 |
| 伯利兹 | 英属殖民遗产、1981年独立，与中美洲和加勒比均有联系。 |

## 重要主题

- 铁路、港口和香蕉公司把中美洲经济与美国市场连接，也形成“香蕉共和国”式的外部依赖与政治干预印象；不同国家经历不可一概而论。
- 冷战时期武装冲突造成大规模死亡、失踪和流离失所；和平协议、真相调查和司法追责成为民主化的一部分。
- 飓风、干旱、森林砍伐、火山和地震等环境风险与贫困、粮食安全和跨境迁移相互叠加。
- 巴拿马运河把加勒比—太平洋航运、全球贸易和国家主权争议联结起来。
- 中美洲与加勒比、墨西哥、美国和南美之间的侨汇、贸易、犯罪网络与人道迁移，使区域边界具有高度流动性。
