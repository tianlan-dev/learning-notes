# 新西班牙与墨西哥中南部

## 时间

1521-1821年为新西班牙核心时期；1821年至今为墨西哥国家史的中南部视角。

## 概括

1521年特诺奇蒂特兰失守后，西班牙以墨西哥城为中心建立新西班牙总督区。它涵盖的范围远超今日墨西哥，曾延伸至中美洲北部、加勒比、北美西南和太平洋航路。银矿、庄园、城市、教会、原住民贡纳和非洲奴隶劳动共同构成殖民社会。1821年墨西哥独立后，中南部成为国家政治核心；而其北部边疆的另一路径见[墨西哥北部边疆](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E7%BE%8E%E6%B4%B2/%E5%8C%97%E7%BE%8E/%E5%A2%A8%E8%A5%BF%E5%93%A5%E5%8C%97%E9%83%A8%E8%BE%B9%E7%96%86.md)。

## 统治结构

| 层级 | 角色 | 说明 |
|---|---|---|
| 西班牙王室 | 宗主权、贸易与任命 | 通过印度事务委员会、总督和财政体系管理海外领地。 |
| 新西班牙总督 | 殖民地最高行政 | 驻墨西哥城，兼具军事、财政和司法协调作用。 |
| 主教区、修会与教区 | 宗教与社会组织 | 参与传教、教育、土地和慈善，也与国家权力互动。 |
| 城市市政会与地方精英 | 城市政治经济 | 克里奥尔、商人和土地精英在地方具有影响。 |
| 原住民共和国与社区 | 地方共同体 | 在殖民法框架下保留部分自治与土地组织，同时承担贡纳和劳役。 |

## 重要事件

- 1521年后西班牙在原住民盟友协助下建立殖民首都；征服和治理在各地区持续数十年。
- 16-18世纪银矿和跨太平洋马尼拉大帆船贸易使新西班牙连接亚洲、欧洲和美洲。
- 1810年伊达尔戈起义启动独立战争；1821年《科尔多瓦条约》与军政联盟促成墨西哥独立。
- 独立并未立即消除原住民土地压力、教会影响、区域主义和社会等级。
- 墨西哥中南部的中部美洲文明遗产与国家认同密切相连，但不能将现代墨西哥等同于古代墨西加政体。
