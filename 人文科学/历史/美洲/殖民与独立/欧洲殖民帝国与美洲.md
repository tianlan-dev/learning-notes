# 欧洲殖民帝国与美洲

## 时间

15世纪末至19世纪；若干殖民地和海外属地延续至今。

## 概括

欧洲在美洲的殖民体系不是同一模式的复制。西班牙以总督区、城市、教会、矿业和原住民贡纳建立广阔帝国；葡萄牙在巴西发展种植园、矿业与内陆扩张；英国在北美和加勒比形成不同的定居殖民地与种植园殖民地；法国、荷兰和丹麦等则以岛屿、港口、毛皮、糖业和贸易网络参与竞争。各体系均依赖或对抗原住民政治共同体，并在不同程度上使用非洲奴隶劳动。

## 主要殖民体系

| 帝国 | 主要区域 | 制度特点 |
|---|---|---|
| 西班牙 | 墨西哥、中美洲、加勒比、安第斯、拉普拉塔及北美边疆 | 总督区、总督辖区、城市市政会、教会、银矿和原住民贡纳。 |
| 葡萄牙 | 巴西 | 王室总督、种植园、金矿、奴隶贸易和班代兰特内陆扩张。 |
| 英国 | 北美大西洋沿岸、加拿大部分、加勒比 | 定居殖民地、特许公司、地方议会、种植园和海军贸易网络。 |
| 法国 | 加拿大、密西西比河流域、加勒比 | 毛皮贸易、联盟外交、圣多明各种植园和岛屿殖民。 |
| 荷兰 | 新尼德兰、苏里南、安的列斯 | 贸易公司、港口、奴隶贸易和种植园经济。 |
| 其他欧洲政权 | 若干加勒比岛屿和边境地区 | 丹麦、瑞典、库尔兰等小型殖民尝试及主权转移。 |

## 重要说明

- 殖民者的“占有”声明常先于实际控制。河谷、草原、雨林和高地的原住民政治力量长期限制或重塑殖民边界。
- 宗教传教既可能在局部保护原住民免受奴役，也服务于定居、劳动控制和文化改造；不能只作正面或负面单一概括。
- 贸易限制、走私、海盗和跨帝国婚姻使殖民地社会远比官方地图流动。
- 欧洲列强之间的战争在美洲形成不同战场，例如北美英法战争、加勒比海战和西属美洲的入侵。
- 独立后殖民权力并未完全消失：加拿大、加勒比、圭亚那、波多黎各、法属地区等拥有不同的去殖民化或属地路径。
