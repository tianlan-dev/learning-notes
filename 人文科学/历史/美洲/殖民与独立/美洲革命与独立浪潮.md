# 美洲革命与独立浪潮

## 时间

1775-1825年为核心，去殖民化与国家形成延续至20世纪。

## 概括

美洲独立革命共享启蒙、帝国战争、税收、奴隶制、种族等级和地方自治等背景，但不是同一场运动。美国革命产生联邦共和国却保留奴隶制；海地革命将反奴役与独立结合；西属美洲在1808年王权危机后爆发多场地区战争；巴西在葡萄牙王室迁移的条件下走向君主制独立。革命者、被奴役者、原住民、自由有色人种、妇女、城市民兵和地方精英的目标并不相同。

## 比较

| 过程 | 核心时间 | 结果与差异 |
|---|---|---|
| 美国革命 | 1775-1783年 | 1776年宣言、1783年承认、1789年宪政运行；奴隶制和原住民土地扩张未被解决。 |
| 海地革命 | 1791-1804年 | 由被奴役者领导的革命建立独立共和国，震动整个奴隶制世界。 |
| 西属美洲独立 | 1810-1825年 | 多个副王区解体为新国家，大哥伦比亚等跨国方案又迅速面临地区分裂。 |
| 巴西独立 | 1822年 | 佩德罗一世建立巴西帝国，领土统一与君主制延续较突出。 |
| 加勒比去殖民化 | 20世纪 | 多个英属领地独立，其他地区保留不同属地或自治地位。 |

## 重要影响

- 海地革命促使奴隶主社会恐惧扩散，也为废奴主义和黑人政治提供强大参照。
- 西属美洲独立军相互支援，但地区主义、财政和交通限制使统一方案难以持久。
- 英国、法国、美国和其他外部力量以贸易、贷款、承认、军事干预或外交压力影响独立后国家。
- 新国家通常继承殖民行政边界和社会等级，公民权、土地、奴隶制与原住民主权仍需长期斗争。
- “独立日”是重要政治象征，但国家实际控制、国际承认和宪政稳定往往发生在不同时间。
