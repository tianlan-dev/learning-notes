# 19世纪帝国主义与门罗主义

## 时间

1823年至20世纪初；相关外交观念和干预遗产延续至今。

## 概括

殖民帝国解体后，美洲并未成为没有外部力量的空间。1823年门罗主义反对欧洲在美洲重新殖民或干预，最初能力有限，后来逐渐与美国大陆扩张、加勒比政策和“保护性”干预相连。英国、法国、西班牙和其他欧洲力量仍通过贸易、债务、战争、殖民地和外交影响拉美与加勒比。新国家的主权常受军力、财政和全球市场限制。

## 主要节点

| 时间 | 事件 | 说明 |
|---|---|---|
| 1823年 | 门罗主义 | 反对欧洲扩张的外交声明，早期实际执行依赖英国海军和国际力量平衡。 |
| 1833年 | 英国占领马尔维纳斯群岛 | 后续成为阿根廷与英国长期主权争议。 |
| 1846-1848年 | 美墨战争 | 美国大规模扩张领土，重塑北美和拉美关系。 |
| 1861-1867年 | 法国干预墨西哥 | 在美国内战背景下扶植帝制，最终失败。 |
| 1898年 | 美西战争 | 美国取代西班牙成为加勒比和太平洋的重要力量。 |
| 1904年 | 罗斯福推论 | 把门罗主义扩展为美国可在拉美“预防性干预”的主张。 |
| 1903-1914年 | 巴拿马与运河建设 | 美国支持巴拿马脱离哥伦比亚并取得运河区控制，改变区域战略。 |

## 说明

- 门罗主义不是国际法授权，也不是拉美国家共同认可的安全体系；不同政府和社会对其有不同理解和批评。
- 债务、关税、铁路、矿业和贸易协定常与武力威胁交织，主权问题并不只发生在战场。
- 美国的大陆扩张和加勒比干预与欧洲的殖民遗留共同塑造“泛美”关系。
- 拉丁美洲国家并非只有被动反应，也通过联盟、国际法、泛美主义和民族主义争取外交空间。
