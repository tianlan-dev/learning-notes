# 美洲历史

## 概括

美洲历史可分为前哥伦布文明、欧洲殖民、独立建国和现代国家几个大阶段。北美、中美洲、加勒比和南美洲各有不同主线：玛雅、阿兹特克、印加等文明与西班牙、葡萄牙、英国、法国等殖民体系共同构成近现代美洲的基础。
