# 联合王权与丹麦统治时期的挪威

## 时间

1319年—1814年

## 概括

14世纪以后，挪威先后参与瑞典联合、卡尔马联盟和丹麦—挪威复合君主国。黑死病、贵族衰落和行政中心外移削弱本土政治力量，但挪威的法律传统、地方社会和王国身份并未消失。

## 历史走向

- 1319年挪威与瑞典因王朝继承共戴君主，此后王位和贵族政治越来越嵌入斯堪的纳维亚联合。
- 1349年前后黑死病造成严重人口损失，土地、教会和贵族结构受到长期冲击，本土精英力量明显削弱。
- 1397年挪威进入卡尔马联盟。丹麦王室逐渐成为联合核心，挪威在外交和高层政治中的自主空间收缩。
- 1536—1537年宗教改革后，挪威议政机构被取消，行政和教会进一步受哥本哈根控制；但挪威仍保有王国名称、法律和地方行政传统。
- 木材、矿业、航运和渔业在近世发展，城市商人、官僚和农民社会形成不同于丹麦本土的利益结构。
- 1814年《基尔条约》拟将挪威从丹麦国王转让给瑞典国王，引发挪威制宪与独立行动，结束丹麦统治。

## 关键辨析

- “丹麦统治时期”不等于挪威社会和制度完全丹麦化；地方农民、法律区、经济网络和王国认同持续存在。
- 卡尔马联盟与1536年以后的丹麦—挪威不是同一种联合结构。
- 冰岛、法罗群岛和格陵兰在1814年后留在丹麦王权之下，没有随挪威转入瑞典联合。

## 从中世纪强国到共主体系

13世纪挪威王室把冰岛、格陵兰诺斯聚落、法罗、马恩岛和苏格兰西部岛屿纳入不同程度的宗主网络。1263年对苏格兰远征未获决定胜利，1266年《珀斯条约》放弃赫布里底和马恩岛，保留奥克尼、设得兰等北方领地。马格努斯六世以全国法典、城市法和外交取代持续扩张，形成中世纪行政高峰。

1349年起黑死病多轮流行导致人口、教士和地方精英锐减，税收与庄园经济衰退。王朝继承把挪威先与瑞典、再与丹麦联结；这不是瘟疫“自动导致亡国”，而是人口冲击、贵族资源不足和婚姻继承共同作用。1380年奥拉夫兼丹麦王，玛格丽特摄政；卡尔马联盟后，王廷和外交中心更多移向丹麦。

1536年克里斯蒂安三世宣布挪威更紧密隶属丹麦王冠，1537年镇压大主教奥拉夫·恩格尔布雷克特松并推行路德宗改革，挪威王国议会消失。挪威仍保有王国称号、地方司法、地方法律与1687年《挪威法典》，因此既不是平等联合，也不是普通省份。木材、铜矿、鱼类和商船使挪威经济在17—18世纪增长，本地官员、商人和农民社会亦逐渐增强。

拿破仑战争中英国封锁造成粮食危机和贸易中断。1814年丹麦国王被迫割让挪威，挪威精英拒绝直接转交，遂召开制宪会议。联合的终结由列强条约、战争失败与挪威国内国家能力共同造成。

## 重要事件与统治层次

| 时间 | 事件 | 过程与影响 |
|---|---|---|
| 1262—1264年 | 冰岛接受挪威王权 | 通过盟约而非一次征服并入 |
| 1266年 | 《珀斯条约》 | 放弃苏格兰西部岛屿，北大西洋重心北移 |
| 1274、1276年 | 全国法典与城市法 | 王国法律和行政统一程度提高 |
| 1349年起 | 黑死病 | 人口与税基崩落，社会结构长期调整 |
| 1380年 | 丹挪共主 | 王朝继承连接两王国，各自法律仍存 |
| 1397年 | 卡尔马联盟 | 挪威纳入三国共主框架，资源和政治影响相对较弱 |
| 1468—1469年 | 奥克尼、设得兰被抵押给苏格兰 | 婚姻欠款导致领土实际转移，未被赎回 |
| 1536—1537年 | 宗教改革和议会撤销 | 教产、教会任命和中央行政转向哥本哈根 |
| 1660—1665年 | 绝对君主制 | 贵族选王制度终结，官僚和军队集中化 |
| 1687年 | 《挪威法典》 | 全国法律整理，同时体现独立法律领域 |
| 1807—1814年 | 封锁、饥荒与联盟解体 | 海路切断及战败促成制宪和新联合 |

完整共同君主及挪威法律地位见[挪威君主与政府首脑表](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E6%AC%A7%E6%B4%B2/%E5%8C%97%E6%AC%A7/%E6%8C%AA%E5%A8%81/%E6%8C%AA%E5%A8%81%E5%90%9B%E4%B8%BB%E4%B8%8E%E6%94%BF%E5%BA%9C%E9%A6%96%E8%84%91%E8%A1%A8.md)。
