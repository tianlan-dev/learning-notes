# 瑞典帝国

## 时间

1611年—1721年

## 概括

瑞典帝国是瑞典通过军事财政改革、波罗的海扩张和欧洲大战成为北欧强权的时期，常称“大国时代”。其统治范围包括或影响芬兰、爱沙尼亚、利沃尼亚、英格里亚和波美拉尼亚等地，但这些地区的法律、等级和地方制度并不完全相同。

## 历史走向

- 瓦萨王朝建立的税收、教会和军事基础在古斯塔夫二世·阿道夫与大法官阿克塞尔·乌克森谢纳时期进一步制度化。
- 瑞典通过征兵、地方登记、铜铁出口和较有效的中央行政支撑长期战争，人口规模较小却能维持欧洲强军。
- 在三十年战争中，瑞典以新教强国身份进入德意志战场。1648年《威斯特伐利亚和约》使瑞典获得波美拉尼亚等领地并取得帝国政治影响。
- 17世纪中叶对丹麦战争使瑞典取得斯科讷等地区，控制范围环绕波罗的海多个海岸；但扩张也造成财政、兵役和地方整合压力。
- 帝国并非单一民族国家。瑞典本土、芬兰、波罗的海省份和德意志领地分别保有不同等级结构、法律与语言社会。
- 1680年代卡尔十一世通过土地收回、军队编制和行政整顿强化王权，为卡尔十二世时代战争提供资源。
- 1700—1721年大北方战争中，瑞典先取得战场优势，后在俄国纵深和多国联盟压力下失败。
- 1721年《尼斯塔德和约》后，瑞典失去爱沙尼亚、利沃尼亚、英格里亚等关键领地，俄罗斯崛起为波罗的海主导强权，瑞典大国时代结束。

## 统治结构

| 层次 | 主要特征 |
|---|---|
| 君主 | 战争、任官与财政的核心；17世纪后期绝对王权加强 |
| 枢密院与中央官署 | 形成较专业的行政分工，管理财政、战争、司法和地方 |
| 等级议会 | 贵族、教士、市民和农民四等级参与税收与政治协商 |
| 地方与省份 | 瑞典—芬兰核心区和海外省份采用不同程度的既有法律与特权 |
| 军事财政体系 | 征兵、土地配置、税收与出口收入共同支持常备军事力量 |

## 重要事件

| 时间 | 事件 | 影响 |
|---|---|---|
| 1611年 | 古斯塔夫二世·阿道夫即位 | 军事与行政改革加速 |
| 1630年 | 瑞典进入三十年战争 | 取得德意志和欧洲政治影响 |
| 1648年 | 《威斯特伐利亚和约》 | 确认瑞典在帝国及波罗的海的强国地位 |
| 1658年 | 《罗斯基勒条约》 | 从丹麦取得斯科讷等地区 |
| 1680年代 | 土地收回与军事整顿 | 王权和财政能力加强 |
| 1700—1721年 | 大北方战争 | 波罗的海霸权丧失，俄罗斯崛起 |

## 区域影响

- 对[瑞典历史](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E6%AC%A7%E6%B4%B2/%E5%8C%97%E6%AC%A7/%E7%91%9E%E5%85%B8/README.md)而言，本阶段连接瓦萨国家形成与18世纪宪制重组。
- 对[芬兰历史](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E6%AC%A7%E6%B4%B2/%E5%8C%97%E6%AC%A7/%E8%8A%AC%E5%85%B0/README.md)而言，芬兰是王国核心部分和重要兵员、行政区域，不应仅称为海外殖民地。
- 对[波罗的海历史](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E6%AC%A7%E6%B4%B2/%E6%B3%A2%E7%BD%97%E7%9A%84%E6%B5%B7/README.md)而言，瑞典统治是爱沙尼亚和利沃尼亚等地区多重帝国统治中的一个阶段。
- 对丹麦、波兰—立陶宛和俄罗斯而言，瑞典扩张改变了17世纪北欧与东北欧的力量平衡。
