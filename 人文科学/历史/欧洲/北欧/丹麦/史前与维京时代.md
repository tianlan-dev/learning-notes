# 史前与维京时代的丹麦

## 时间

史前—约1050年

## 概括

丹麦历史的早期主线形成于日德兰半岛、丹麦群岛与厄勒海峡之间。农业社会、铁器时代首领网络和海上交通逐步汇合，至维京时代出现更稳定的王权、城镇与跨海统治。

## 历史走向

- 石器时代和青铜时代遗存说明日德兰与群岛很早便嵌入北海—波罗的海交换网络。
- 罗马铁器时代以后，地方首领通过军事随从、贡赋和海上贸易积累权力，但尚不能直接对应现代丹麦国家。
- 8—11世纪的丹麦维京集团频繁进入英格兰、法兰克沿岸和波罗的海；赫泽比等贸易中心连接大陆与北海。
- 10世纪后半叶，耶灵王朝加强王权。老戈姆和蓝牙哈拉尔常被视为王国形成中的关键君主；基督教化把王权进一步接入西欧教会秩序。
- 11世纪克努特大帝建立的跨海统治把丹麦、英格兰和挪威暂时纳入同一王权网络，但这一复合结构没有长期维持。

## 统治结构

| 层次 | 主要形态 | 演变 |
|---|---|---|
| 地方社会 | 农庄、地方集会与首领集团 | 维持习惯法和区域动员 |
| 王权 | 从竞争性首领向较稳定的世袭君主发展 | 通过要塞、铸币、征税和战争加强控制 |
| 宗教 | 北欧多神信仰逐渐转向基督教 | 教会组织帮助王权整合领土 |

## 形成过程深化

冰盖退却后，日德兰和群岛先由流动渔猎群体定居。约前4000年起农业、畜牧和巨石墓葬扩展，但狩猎与海洋资源继续重要。北欧青铜时代依靠海路交换铜、锡、琥珀与礼仪物品；岩画、太阳车和大型墓冢显示精英以远程交换和宗教仪式建立权威。铁器时代聚落扩大，罗马货币、玻璃和武器经间接贸易进入丹麦，沼泽祭献和武器沉积又记录了战争与仪式。

5—8世纪的大厅、港市和船葬说明地区性首领网络已形成。里伯、赫泽比等节点把北海、波罗的海和大陆道路连接起来；丹尼维尔克防御工程与环形要塞则反映更大规模的劳役动员。所谓“丹麦人”在法兰克和英格兰文献中逐渐成为政治称呼，但其范围随王权竞争而变化。

## 关键事件与转折

| 时间 | 事件 | 过程与意义 |
|---|---|---|
| 约前4000年 | 农业与巨石墓扩展 | 定居、土地权和祖先纪念加强，未完全取代渔猎经济 |
| 前1700—前500年 | 北欧青铜时代 | 琥珀与金属长途交换支撑地方精英和海上交通 |
| 约500—750年 | 大厅、港市和首领中心发展 | 贡礼、工艺和武装随从网络为维京时代奠基 |
| 约710年起 | 里伯贸易中心形成 | 季节性市场逐渐城市化，铸币和专业手工业出现 |
| 793年以后 | 西向袭击与移民扩大 | 丹麦舰队参与英格兰和法兰克沿岸战争、贡金与定居 |
| 865年 | “大异教军”进入英格兰 | 多年征服取代部分季节性劫掠，丹法区逐渐形成 |
| 约965年 | 耶灵碑与王权基督教化 | 蓝牙哈拉尔把宗教转变与王朝合法性结合 |
| 974—980年代 | 丹尼维尔克与环形堡垒扩建 | 证明王权具备跨地区征役能力，但控制仍不均衡 |
| 1013—1035年 | 斯文与克努特的英格兰王权 | 税收、舰队、教会与地方精英共同维持跨海统治 |
| 1042—1047年 | 王朝断裂与马格努斯统治 | 北海王权解体，斯文二世最终建立新王朝 |

## 王国形成的动力与局限

- **海峡位置**使国王能够干预北海—波罗的海交通，却也使丹麦暴露于萨克森、斯拉夫、挪威和瑞典竞争。
- **战争与贡金**为王族提供银和随从，失败或继承危机又会迅速削弱中央控制。
- **基督教会**带来书写、教区和跨国合法性；地方祭祀与习惯没有在一次洗礼后消失。
- **港市、铸币和要塞**把季节性动员转为较稳定的征税和行政，但国王仍须与地方首领、自由农户和集会协商。
