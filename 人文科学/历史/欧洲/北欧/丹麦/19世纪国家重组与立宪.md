# 19世纪丹麦的国家重组与立宪

## 时间

1814年—1914年

## 概括

失去挪威后，丹麦从复合北欧强权转向规模较小的民族国家。石勒苏益格—荷尔斯泰因问题、1849年宪法和1864年战败构成政治重组的核心，此后农业合作、工业化和议会政治逐步发展。

## 历史走向

- 1814年以后，丹麦王国仍同时统治丹麦本土以及石勒苏益格、荷尔斯泰因和劳恩堡等地，王国与公国的法律、语言和民族认同并不一致。
- 1848年革命和第一次石勒苏益格战争推动绝对君主制终结；1849年《六月宪法》建立君主立宪框架和两院议会。
- 丹麦民族自由派与德意志民族运动对石勒苏益格归属存在冲突。1864年第二次石勒苏益格战争失败后，丹麦失去三个公国。
- 1864年以后，国家发展重心从大陆领土竞争转向本土农业、教育与社会组织。“向内赢回外部所失”概括了农业改良、合作社和国民教育的时代方向。
- 19世纪后期铁路、城市工业和工人运动发展。自由派、农民政党与保守派围绕政府是否必须获得议会多数支持长期斗争。
- 1901年议会制原则得到确认，政府责任逐渐以议会多数为基础。

## 重要转折

| 时间 | 转折 | 影响 |
|---|---|---|
| 1849年 | 《六月宪法》 | 从绝对君主制转向君主立宪 |
| 1864年 | 第二次石勒苏益格战争失败 | 领土大幅缩小，国家发展方向改变 |
| 1901年 | 议会制突破 | 内阁形成开始服从议会多数逻辑 |

## 公国问题的具体结构

1814年后的“丹麦君主国”并不等于丹麦王国本土。石勒苏益格是丹麦王冠封地，德语与丹麦语人群交错；荷尔斯泰因和劳恩堡属于德意志邦联，三者又通过继承和行政与国王相连。丹麦民族自由派主张把石勒苏益格更紧密并入王国，德意志民族派主张两公国共同加入统一德国。国际条约、王位继承和当地语言政治互相缠绕。

1848年哥本哈根政治动员与基尔临时政府同时出现，第一次石勒苏益格战争遂成为内战、王朝冲突和国际战争的复合体。1849年宪法在丹麦王国内建立选举议会和公民权利，但公国宪制仍未解决。1852年列强确认格吕克斯堡的克里斯蒂安继承整体君主国；1863年“十一月宪法”试图把丹麦和石勒苏益格结合，普鲁士与奥地利遂于1864年进军。迪伯尔和阿尔斯失守后，丹麦割让三公国。

## 社会与议会化

1866年修宪提高地主和富裕阶层在上院的影响。1870年代起左翼农民政党控制下院，埃斯特鲁普政府却依靠国王和上院以临时财政法长期执政，形成“宪法斗争”。1894年妥协和1901年国王任命获下院支持的德恩策政府，使不违背议会多数的原则成为常规。

经济方面，廉价谷物冲击促使农民转向乳品、猪肉和合作社加工；民众高等学校、农会与合作社把教育、信用和政治组织结合。哥本哈根工业和工人运动成长，1899年九月妥协确认雇主管理权与工会组织、谈判权，为此后劳资制度奠基。海外移民和城市化也改变人口结构。

## 重要事件

| 时间 | 事件 | 结果与长期影响 |
|---|---|---|
| 1814年 | 《基尔条约》 | 挪威脱离，丹麦保留北大西洋领地和公国 |
| 1834年 | 咨议等级会议 | 绝对王权下有限代表制度，为政治公开化铺路 |
| 1848—1851年 | 第一次石勒苏益格战争 | 丹麦暂时维持整体君主国，民族冲突未解决 |
| 1849年 | 《六月宪法》 | 丹麦王国内绝对君主制结束 |
| 1852年 | 《伦敦议定书》 | 列强规定整体君主国继承，未消除制度矛盾 |
| 1863—1864年 | 十一月宪法与第二次战争 | 普奥获胜，丹麦割让石勒苏益格、荷尔斯泰因和劳恩堡 |
| 1866年 | 保守修宪 | 上院选举制度偏向大土地所有者 |
| 1875—1894年 | 埃斯特鲁普长期执政 | 下院多数与王室—上院政府持续冲突 |
| 1880—1900年代 | 合作社和工业化 | 农产品出口转型，现代阶级与组织社会形成 |
| 1899年 | 九月妥协 | 劳资双方建立集体谈判基本规则 |
| 1901年 | 议会制突破 | 政府必须避免遭下院多数反对 |
