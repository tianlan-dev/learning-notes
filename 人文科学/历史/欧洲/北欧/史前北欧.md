# 史前北欧

[返回北欧历史总览](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E6%AC%A7%E6%B4%B2/%E5%8C%97%E6%AC%A7/README.md)

## 时间

约公元前11000年—公元750/800年

## 概括

“史前北欧”涵盖末次冰期结束后至维京时代前夕的斯堪的纳维亚半岛、日德兰半岛、芬兰与北大西洋邻近区域。这个长时段没有统一国家，也不能把考古文化直接等同于后来的丹麦人、挪威人、瑞典人、芬兰人或萨米人。其主线是环境开放、狩猎采集网络、农业与畜牧业北传、金属和远距离交换扩展，以及铁器时代权力中心和海上军事组织逐渐形成。

## 分期与具体过程

### 冰后定居与中石器时代

- 约公元前11000年以后，冰盖退缩、海岸线不断变化，驯鹿猎人和沿海采集者由欧洲大陆与东北方向进入可居住地带。聚落多依赖渔猎、海兽、森林资源和季节迁徙。
- 南斯堪的纳维亚先后出现马格勒莫斯、孔厄莫斯和埃特伯勒等考古文化；北部与芬诺斯坎迪亚则形成适应寒地河流、湖泊和海岸的区域传统。燧石、石英、骨角器和独木舟反映出高度专业化的生计。
- 海平面回升和地壳反弹反复改变海湾、岛屿与航路，聚落迁移不是简单的“文明衰落”，而是对生态边界变化的持续适应。

### 新石器时代的农业扩展

- 约公元前4000年起，农业、畜牧、定居村落和巨石墓葬进入丹麦及南部斯堪的纳维亚，漏斗杯文化是重要代表。农业没有立即替代渔猎，沿海和北部长期保持混合生计。
- 坑纹陶文化等群体继续倚重海洋资源；约公元前2800年以后战斧文化及相关的绳纹器传统进入本区。考古与古 DNA 说明人口迁移、婚姻网络和技术传播共同发生，但文化器物不能直接证明单一民族迁徙。
- 北部芬兰、萨米区域及高纬地带的农业化更晚、更不均衡，驯鹿利用、捕鱼和狩猎在许多社群中长期重要。

### 北欧青铜时代

- 约公元前1700—前500年，铜和锡主要依靠远距离输入，琥珀、毛皮及其他北方产品则进入跨区域交换。控制交换网络的家族通过大型墓冢、武器、礼仪器物和宴饮建立地位。
- 岩画、剃刀和金属器上的船、太阳、车轮与武士图像显示海上交通和宇宙观的重要性。船只是运输工具，也是权力、祭仪与共同体身份的象征。
- 青铜资源依赖外部供应，因此区域权力中心既因贸易繁荣，也容易受交换路线变化影响；青铜时代不是一个统一“王国时代”。

### 铁器时代与权力中心形成

- 前罗马铁器时代约始于公元前500年，本地沼铁冶炼降低了对进口金属的依赖。农业聚落、长屋和区域性物品风格增加，但气候与生计压力在不同地方影响不一。
- 罗马铁器时代约公元1—400年，北欧并未并入罗马帝国，却通过贸易、雇佣兵和战争接触获得钱币、玻璃、武器与奢侈品。湿地祭献和战利品沉积显示跨部落冲突与军事集团扩张。
- 约400—550年的民族迁徙时期，金器、精英大厅和随从武士组织增多。罗马秩序瓦解改变了财富来源，也促使本地首领重新控制土地、贡赋和航路。
- 约550—750/800年的文德尔时期及挪威的梅罗文加时期出现船葬、华丽武具、布料和远程贸易中心。比尔卡、里伯等早期商贸节点的前身以及更成熟的帆船技术，为维京时代的远航提供条件。

## 重要转折与辨识点

| 时间 | 转折 | 历史意义 |
|---|---|---|
| 约前11000年以后 | 冰盖退缩后重新定居 | 打开新的海岸、森林和湖区生态空间 |
| 约前7000—前4000年 | 海岸渔猎网络成熟 | 航行、交换和季节性聚落成为长期传统 |
| 约前4000年 | 农业进入南部北欧 | 土地利用、人口密度和墓葬组织改变 |
| 约前2800年以后 | 战斧与绳纹器传统扩展 | 人群迁移与社会分层加深 |
| 约前1700年 | 北欧青铜时代形成 | 远距离金属—琥珀交换和精英权力上升 |
| 约前500年 | 铁器普及 | 本地冶铁扩大，聚落和武装组织重组 |
| 公元1—400年 | 与罗马世界密切接触 | 输入财富、军事经验和身份象征 |
| 约550—750/800年 | 船葬、精英大厅与贸易点发展 | 海上王权和维京远航的制度、技术条件成熟 |

## 形成维京时代的机制

- **技术条件**：帆船、龙骨、铁工具和航海经验持续积累，使跨海运兵与大宗运输更可靠。
- **政治条件**：首领通过大厅、宴饮、赠礼和武装随从集中资源，区域竞争促成更大规模的军事联盟。
- **经济条件**：琥珀、毛皮、铁、奴隶和奢侈品交换把北欧接入波罗的海、北海与欧陆网络。
- **社会条件**：土地继承、声望竞争和人口流动推动部分人外迁、经商、服役或劫掠。
- **文化条件**：船、战士身份、祖先纪念和共同祭仪为远征组织提供象征语言，但不同地区仍保有显著差异。

## 争议与地区差异

- 考古文化名称是研究分类，不等同于有共同自称、语言和政治组织的民族。
- 萨米人及其他北方人群的形成涉及长期互动、语言扩散与生计变化，年代和路径仍存在争议，不宜画成单一血缘直线。
- “维京时代开始于793年”是以英格兰书面记录为标志的常用断代；斯堪的纳维亚内部的城镇、航海和权力变化在此之前已经展开。

## 演变关系

史前北欧各区域传统并未同时消失，而是在海上交通、商贸中心、军事随从和早期王权扩大的过程中逐渐进入[维京时代](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E6%AC%A7%E6%B4%B2/%E5%8C%97%E6%AC%A7/%E7%BB%B4%E4%BA%AC%E6%97%B6%E4%BB%A3.md)。更长时段的地区脉络见[北欧历史总览](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E6%AC%A7%E6%B4%B2/%E5%8C%97%E6%AC%A7/README.md)。
