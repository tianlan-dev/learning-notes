# 史前、维京时代与中世纪瑞典王国

## 时间

史前—1397年

## 概括

现代瑞典所在地区由斯韦阿、约塔等多个区域社会逐步整合。维京时代的波罗的海与罗斯贸易、基督教化、地方集会和王朝竞争共同塑造中世纪王国，统一过程并没有单一明确的完成时点。

## 历史走向

- 史前瑞典包括不同生态区和文化网络；梅拉伦湖、乌普兰、约特兰和波罗的海岛屿的发展并不一致。
- 维京时代，瑞典方向的商人与武装集团主要沿波罗的海和东欧河道活动，与罗斯、拜占庭和伊斯兰世界的贸易相连。
- 乌普萨拉等中心保留北欧宗教传统，基督教在11—12世纪逐步制度化。教区和修道院把王国纳入拉丁教会体系。
- 斯韦阿与约塔地区的王朝、法律区和贵族通过战争、婚姻和选王制度逐步结合；“瑞典统一”应理解为长时段政治整合。
- 13世纪比耶尔·亚尔及其后继者加强王权、城市和税收，并向芬兰方向扩展教会与政治控制。
- 14世纪王朝继承使瑞典与挪威等国共戴君主；贵族对王权、税收和外国官员的不满推动新的联盟政治。
- 1397年瑞典加入卡尔马联盟，但仍保留本国法律、贵族会议和政治共同体。

## 统治结构

| 层次 | 主要作用 |
|---|---|
| 地方法律区与集会 | 维持地区法律、自由民参与和选王传统 |
| 王权 | 逐步发展税收、铸币、城堡和行政职位 |
| 贵族与教会 | 掌握土地和政治资源，也制约君主 |
| 城市与贸易网络 | 连接汉萨同盟、波罗的海和内陆资源区 |
