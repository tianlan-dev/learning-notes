# 史前、维京时代与中世纪瑞典王国

## 时间

史前—1397年

## 概括

现代瑞典所在地区由斯韦阿、约塔等多个区域社会逐步整合。维京时代的波罗的海与罗斯贸易、基督教化、地方集会和王朝竞争共同塑造中世纪王国，统一过程并没有单一明确的完成时点。

## 历史走向

- 史前瑞典包括不同生态区和文化网络；梅拉伦湖、乌普兰、约特兰和波罗的海岛屿的发展并不一致。
- 维京时代，瑞典方向的商人与武装集团主要沿波罗的海和东欧河道活动，与罗斯、拜占庭和伊斯兰世界的贸易相连。
- 乌普萨拉等中心保留北欧宗教传统，基督教在11—12世纪逐步制度化。教区和修道院把王国纳入拉丁教会体系。
- 斯韦阿与约塔地区的王朝、法律区和贵族通过战争、婚姻和选王制度逐步结合；“瑞典统一”应理解为长时段政治整合。
- 13世纪比耶尔·亚尔及其后继者加强王权、城市和税收，并向芬兰方向扩展教会与政治控制。
- 14世纪王朝继承使瑞典与挪威等国共戴君主；贵族对王权、税收和外国官员的不满推动新的联盟政治。
- 1397年瑞典加入卡尔马联盟，但仍保留本国法律、贵族会议和政治共同体。

## 统治结构

| 层次 | 主要作用 |
|---|---|
| 地方法律区与集会 | 维持地区法律、自由民参与和选王传统 |
| 王权 | 逐步发展税收、铸币、城堡和行政职位 |
| 贵族与教会 | 掌握土地和政治资源，也制约君主 |
| 城市与贸易网络 | 连接汉萨同盟、波罗的海和内陆资源区 |

## 形成过程深化

冰后居民在湖海、森林和河谷建立狩猎与渔业社会；约前4000年后农业逐渐进入南部，但北部生计、萨米和其他芬兰—乌戈尔人群的历史不能套入同一农业直线。青铜时代的岩画、金属器与船形图像显示波罗的海—北海交换；铁器时代乌普兰墓冢、精英大厅和哥特兰银藏反映地方权力与远程贸易。

维京时代的比尔卡、哥特兰和梅拉伦地区通过拉多加、第聂伯和伏尔加路线连接罗斯、拜占庭与伊斯兰世界。所谓“罗斯”网络由北欧、斯拉夫、芬兰—乌戈尔与草原人群共同形成，不能写成现代瑞典的国家殖民。约1000年奥洛夫·舍特康努格受洗并铸币，但乌普萨拉传统祭祀、地方庭会议和区域王族继续制约王权。

12世纪斯韦尔克尔和埃里克两家交替以婚姻、丹麦支持、教会和武力争位。13世纪比耶尔·亚尔及其子瓦尔德马建立更稳定行政，1252年斯德哥尔摩传统建城年代标志梅拉伦出海口的重要性。1280年阿尔斯内敕令塑造免税骑士贵族；1350年前后《马格努斯·埃里克松国法》提高全国法律一致性。王族内斗、贵族反对、瘟疫和财政压力最终让梅克伦堡阿尔布雷希特被引入，继而为玛格丽特建立联盟创造条件。

## 重要事件

| 时间 | 事件 | 意义 |
|---|---|---|
| 约前1700—前500年 | 北欧青铜时代 | 金属和礼仪网络连接南部斯堪的纳维亚 |
| 约750—970年 | 比尔卡兴盛 | 东西方贸易、工艺和王权保护结合 |
| 约995—1022年 | 奥洛夫统治 | 基督教、铸币和跨国王室关系加强 |
| 约1080年代 | 英格一世被逐并复位 | 基督教王权与传统祭祀冲突仍激烈 |
| 1130—1210年 | 两王族竞争 | 王位继承依赖地方和教会联盟 |
| 1248年 | 教会会议改革 | 教区纪律与拉丁教会制度深化 |
| 1250年 | 瓦尔德马即位 | 比耶尔·亚尔摄政，王国行政整合 |
| 1280年 | 阿尔斯内敕令 | 骑士贵族特权制度化 |
| 1319年 | 马格努斯四世兼挪威王 | 王朝联合形成但两国法律分立 |
| 1349—1350年 | 黑死病 | 人口、土地与财政结构剧变 |
| 1364年 | 阿尔布雷希特即位 | 贵族借外来王室制衡马格努斯 |
| 1389年 | 玛格丽特击败阿尔布雷希特 | 瑞典进入三国联盟框架 |
