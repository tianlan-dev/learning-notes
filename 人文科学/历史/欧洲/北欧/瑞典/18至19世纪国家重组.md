# 18至19世纪瑞典的国家重组

## 时间

1718年—1905年

## 概括

大北方战争和卡尔十二世之死终结了瑞典大国时代。此后瑞典经历议会主导的自由时代、古斯塔夫王权复兴、1809年失去芬兰与宪制重建，并在1814年后同挪威形成联合；工业化、人口增长和社会运动逐步改变国家。

## 历史走向

- 1718—1719年以后，王权受到新宪制限制，等级议会和枢密机构主导政治，形成所谓“自由时代”。
- 自由时代的党派竞争、商业政策和对俄战争显示议会政治扩大，但政治参与仍受等级、财产和性别限制。
- 古斯塔夫三世1772年政变恢复较强王权，文化与行政改革并行；1792年遇刺后王朝危机持续。
- 1808—1809年对俄战争失败，瑞典失去芬兰。1809年宪法重新分配君主、议会和行政权，成为近代宪制转折。
- 贝尔纳多特王朝建立后，瑞典参加反拿破仑联盟。1814年挪威在保留宪法的条件下与瑞典共戴君主。
- 19世纪瑞典避免再次参加欧洲大战，对外谨慎逐渐发展为和平时期的不结盟传统。
- 农业改革、人口增长、北美移民、铁路、木材和工业化重塑社会；自由派、工人运动和复兴宗教团体推动政治参与扩大。
- 1905年瑞典接受与挪威和平解体，联合君主制终结。

## 重要转折

| 时间 | 转折 | 影响 |
|---|---|---|
| 1718—1719年 | 大国时代终结与自由时代开始 | 等级议会权力扩大 |
| 1809年 | 失去芬兰并制定新宪法 | 疆域和政治制度同时重组 |
| 1814年 | 与挪威联合 | 形成共戴君主、制度分立的联合 |
| 1905年 | 联合和平解体 | 瑞典国家边界与外交路线重新稳定 |

## 帝国之后的政体转向

卡尔十二世阵亡后，乌尔丽卡·埃利奥诺拉和弗雷德里克一世接受新宪法，国王权力受枢密院和四等级议会限制。“自由时代”由帽子党、便帽党等精英派系竞争，出版自由、科学和贸易发展，但选举贿赂、贵族主导和列强资助也很明显。1741—1743年对俄战争失败与1760年代财政危机削弱议会制度声誉。

古斯塔夫三世1772年借政变恢复较强王权，改革刑法、宗教宽容和文化，却压制反对；1789年进一步削弱贵族，1792年遭刺杀。古斯塔夫四世坚持反拿破仑路线，俄军1808年进攻芬兰。军事失败、财政和宫廷不满促成1809年政变，国王被废，瑞典永久失去芬兰。

## 1809年宪制、贝尔纳多特与联合

1809年《政府法》在国王、大臣和等级议会之间重新分权。王储卡尔·奥古斯特去世后，议会选择法国元帅贝尔纳多特为继承人；他以反拿破仑外交换取列强支持，1814年迫使挪威进入个人联合。瑞典国内从此避免大战，联合却因两国宪法、外交与领事权矛盾持续紧张。

19世纪人口增长、土地改革、木材与铁业、铁路和城市工业改变社会，大量人口移民北美。1842年初等教育法、1846年行会改革和1864年经济自由化推动市场社会。1866年四等级议会改为两院制，财产与收入门槛仍限制选举。工会、自由派和社会民主党随后推动普选。

1905年联合因挪威领事事务危机和平解体。瑞典军政强硬派最终接受谈判，卡尔斯塔德协议确立边境安排；这不是瑞典国家衰落终结，而是共主体制无法容纳两个议会民族国家的结果。

| 1721年 | 《尼斯塔德和约》 | 波罗的海帝国终结，议会自由时代开始 |
| 1741—1743年 | “帽子战争” | 芬兰遭俄占，瑞典再次割地 |
| 1766年 | 出版自由法 | 公共政治和信息权的重要里程碑 |
| 1772年 | 古斯塔夫三世政变 | 议会主导被较强王权取代 |
| 1792年 | 国王遇刺 | 摄政与王权不稳 |
| 1808—1809年 | 芬兰战争 | 失去芬兰，旧王朝被废 |
| 1809年 | 新政府法 | 权力分立和大臣责任重构 |
| 1810年 | 贝尔纳多特当选王储 | 新王朝和新外交路线确立 |
| 1814年 | 挪威联合 | 共享君主和外交，挪威保留本国宪法 |
| 1842年 | 普及初等教育 | 国家与教区教育扩展 |
| 1866年 | 两院制改革 | 等级议会终结，代表制仍受财产限制 |
| 1876年 | 首相职位设立 | 现代政府首脑制度成形 |
| 1905年 | 联合解体 | 瑞典接受挪威独立，长期和平外交延续 |
