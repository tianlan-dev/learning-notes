# 瑞典统治时期的芬兰

## 时间

约12世纪—1809年

## 概括

中世纪至1809年，今芬兰大部分地区逐步成为瑞典王国的东部区域。教会、法律、贵族与地方行政将其纳入王国，居民参与共同政治和战争；“瑞典统治”因此既包含中心—边缘关系，也不能简单等同于近代海外殖民。

## 历史走向

- 12—13世纪以后，瑞典王权和天主教会在西南芬兰逐步建立教区、城堡和税收网络；同诺夫哥罗德的竞争影响东部边界。
- 奥布和维堡等城市、沿海瑞典语定居以及内陆芬兰语社群共同构成多语言社会。
- 芬兰地区使用瑞典王国法律，贵族、教士、市民和农民可通过王国等级制度参与政治，但高层行政和书写语言长期以瑞典语为主。
- 16世纪宗教改革使教会转为路德宗体系。米卡埃尔·阿格里科拉的翻译和著述奠定书面芬兰语的重要基础。
- 芬兰是瑞典与俄罗斯战争的主要边疆和兵员来源，多次遭受占领、征税与人口损失。
- 17世纪瑞典大国时代，芬兰属于王国核心组成部分；图尔库学院于1640年建立，行政、教育和教会制度发展。
- 大北方战争期间俄罗斯占领芬兰，1721年后瑞俄边界西移；18世纪后期芬兰的战略与身份问题更突出。
- 1808—1809年芬兰战争中俄罗斯占领全境，瑞典在和约中割让芬兰，数百年共同王国关系终结。

## 统治结构

| 层次 | 地位 |
|---|---|
| 王国 | 芬兰地区属于瑞典王国，并非一个统一的“芬兰殖民地”行政单元 |
| 教会与法律 | 采用瑞典王国教会、法律和地方行政体系 |
| 等级政治 | 芬兰地区的贵族、教士、市民和农民参与王国等级议会 |
| 语言社会 | 瑞典语居行政和高等教育优势，芬兰语为多数居民日常语言之一 |
| 东部边疆 | 与诺夫哥罗德、莫斯科和俄罗斯的战争持续塑造边界 |
