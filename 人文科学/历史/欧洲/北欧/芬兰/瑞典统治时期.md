# 瑞典统治时期的芬兰

## 时间

约12世纪—1809年

## 概括

中世纪至1809年，今芬兰大部分地区逐步成为瑞典王国的东部区域。教会、法律、贵族与地方行政将其纳入王国，居民参与共同政治和战争；“瑞典统治”因此既包含中心—边缘关系，也不能简单等同于近代海外殖民。

## 历史走向

- 12—13世纪以后，瑞典王权和天主教会在西南芬兰逐步建立教区、城堡和税收网络；同诺夫哥罗德的竞争影响东部边界。
- 奥布和维堡等城市、沿海瑞典语定居以及内陆芬兰语社群共同构成多语言社会。
- 芬兰地区使用瑞典王国法律，贵族、教士、市民和农民可通过王国等级制度参与政治，但高层行政和书写语言长期以瑞典语为主。
- 16世纪宗教改革使教会转为路德宗体系。米卡埃尔·阿格里科拉的翻译和著述奠定书面芬兰语的重要基础。
- 芬兰是瑞典与俄罗斯战争的主要边疆和兵员来源，多次遭受占领、征税与人口损失。
- 17世纪瑞典大国时代，芬兰属于王国核心组成部分；图尔库学院于1640年建立，行政、教育和教会制度发展。
- 大北方战争期间俄罗斯占领芬兰，1721年后瑞俄边界西移；18世纪后期芬兰的战略与身份问题更突出。
- 1808—1809年芬兰战争中俄罗斯占领全境，瑞典在和约中割让芬兰，数百年共同王国关系终结。

## 统治结构

| 层次 | 地位 |
|---|---|
| 王国 | 芬兰地区属于瑞典王国，并非一个统一的“芬兰殖民地”行政单元 |
| 教会与法律 | 采用瑞典王国教会、法律和地方行政体系 |
| 等级政治 | 芬兰地区的贵族、教士、市民和农民参与王国等级议会 |
| 语言社会 | 瑞典语居行政和高等教育优势，芬兰语为多数居民日常语言之一 |
| 东部边疆 | 与诺夫哥罗德、莫斯科和俄罗斯的战争持续塑造边界 |

## 整合过程和地位

12—13世纪的“十字军”叙事来自较晚文献，具体远征时间和规模有争议。更可靠的是瑞典王权、乌普萨拉教会、商人和地方精英通过图尔库教区、城堡、税收及沿海移民逐步把芬兰西南、海梅等地纳入王国。诺夫哥罗德则在卡累利阿和东部征贡、贸易与传教。1323年《内特堡和约》首次大体划定瑞典与诺夫哥罗德势力界线，但边疆活动继续流动。

芬兰不是海外殖民地，也没有独立“芬兰王国”；它是瑞典王国东部若干法律区、省份和教区。瑞典语成为高级行政、城市和海岸社会的重要语言，芬兰语在农村和教会基层广泛使用。宗教改革后米卡埃尔·阿格里科创制和规范芬兰语书面传统，既服务路德宗教化，也为后世民族文化提供工具。

大国时代芬兰承担沉重征兵、税役和边境战争。17世纪官僚、学校、图尔库学院和法庭扩大，农民作为四等级之一参加议会。1690年代饥荒、大北方战争期间1713—1721年“大愤怒”俄军占领及1742—1743年“小愤怒”造成严重人口和财产损失。18世纪农业、铁业、城镇和启蒙改革发展，芬兰语文化与地方精英增加，但国家认同仍首先是瑞典王国臣民。

## 重要事件

| 时间 | 事件 | 影响 |
|---|---|---|
| 约1150—1250年 | 教会与王权逐步整合 | 传统“十字军”需与长期制度扩展区分 |
| 1323年 | 《内特堡和约》 | 瑞典与诺夫哥罗德边界首次成文 |
| 1362年 | 芬兰地区代表获参与国王选举地位 | 表明其属于王国政治共同体 |
| 1527年以后 | 宗教改革 | 教产收归王室，芬兰语教会文本发展 |
| 1548年 | 芬兰语《新约》 | 阿格里科奠定书面芬兰语 |
| 1640年 | 图尔库学院建立 | 高等教育和官僚培养中心 |
| 1695—1697年 | 大饥荒 | 气候、收成与救济不足造成高死亡 |
| 1700—1721年 | 大北方战争 | 征兵、流亡与俄军占领重创芬兰 |
| 1741—1743年 | 再次战争与占领 | 东南边界西移，俄国压力持续 |
| 1788—1790年 | 古斯塔夫三世对俄战争 | 安亚拉军官反对显示边疆精英政治化 |
| 1808—1809年 | 芬兰战争 | 瑞典失去东部省份，俄国建立大公国 |

瑞典君主与芬兰地位不另造一套王系；参见[芬兰大公、总督、国家元首与政府首脑表](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E6%AC%A7%E6%B4%B2/%E5%8C%97%E6%AC%A7/%E8%8A%AC%E5%85%B0/%E8%8A%AC%E5%85%B0%E5%A4%A7%E5%85%AC%E3%80%81%E6%80%BB%E7%9D%A3%E3%80%81%E5%9B%BD%E5%AE%B6%E5%85%83%E9%A6%96%E4%B8%8E%E6%94%BF%E5%BA%9C%E9%A6%96%E8%84%91%E8%A1%A8.md)。
