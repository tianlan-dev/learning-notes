# 维京时代

[返回北欧历史总览](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E6%AC%A7%E6%B4%B2/%E5%8C%97%E6%AC%A7/README.md)

## 时间

约公元750/793年—1066/1100年

## 概括

维京时代是北欧航海者在北海、北大西洋、波罗的海、东欧河道和地中海广泛经商、迁徙、服役、殖民与战争的时期。793年林迪斯法恩修道院遇袭和1066年斯坦福桥战役常被用作西欧叙事的起讫点；北欧考古通常把变化放在约750—1100年。它不是一个统一“维京帝国”，也不存在与现代国界完全对应的单一维京民族。

## 建立背景

- 维京时代承接[史前北欧](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E6%AC%A7%E6%B4%B2/%E5%8C%97%E6%AC%A7/%E5%8F%B2%E5%89%8D%E5%8C%97%E6%AC%A7.md)晚期的帆船技术、港市、铁器生产、精英大厅和武装随从网络。
- 北海与波罗的海贸易增长，阿拉伯银币、法兰克钱币、毛皮、蜂蜡、铁、琥珀、奴隶和奢侈品形成跨区域交换链。
- 地方首领需要银、土地和战利品维系随从；王权竞争、继承冲突与个人寻求财富共同推动远征。人口增加可能是背景之一，但不能单独解释扩张。
- 西欧沿海修道院财富集中、防御薄弱；法兰克与不列颠诸国分裂，也给小规模舰队和后来更大的军队提供机会。

## 社会与统治结构

| 层面 | 主要结构 | 说明 |
|---|---|---|
| 家庭与农庄 | 自由农户、依附者与奴隶 | 多数人务农、捕鱼和手工业；被奴役者承担劳动，也被跨区买卖 |
| 地方政治 | “庭”（Thing）会议、地方习惯法 | 自由人参与裁判与协商，但财富、性别和地位限制实际权力 |
| 军事组织 | 首领、雅尔、武装随从与临时舰队 | 依靠赠礼、战利品、贡税和人身忠诚，规模从季节性队伍发展到常备征服军 |
| 王权 | 竞争中的区域国王 | 丹麦、挪威和瑞典王权逐渐整合，但边界、继承和控制力长期不稳定 |
| 宗教 | 北欧多神传统与基督教并存 | 传教、婚姻、洗礼和王权建制推动转变，地方信仰并非一次性消失 |
| 城镇与贸易 | 里伯、赫泽比、比尔卡、考庞等节点 | 商人、工匠、铸币、税收和王权保护相互促进 |

## 扩张路线与具体过程

### 西向与北大西洋

来自挪威沿岸及北海岛屿的航海者经设得兰、奥克尼、赫布里底和爱尔兰进入不列颠群岛，并在法罗、冰岛、格陵兰建立聚落，约1000年前后短暂到达北美。移民、通婚、奴役和当地反抗并存，不能把所有西向行动归于一个“挪威国家”。

### 北海与法兰克世界

许多以丹麦为基地的舰队袭击英格兰和法兰克海岸，后来建立越冬营地、征收丹麦金并夺取领土。911年法兰克国王承认罗洛在塞纳河口的地位，诺曼底逐渐形成；英格兰的丹法区则经历定居、法律互动和王权再征服。11世纪克努特把英格兰、丹麦和挪威王位暂时联结为[北海帝国](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E6%AC%A7%E6%B4%B2/%E5%8C%97%E6%AC%A7/%E5%8C%97%E6%B5%B7%E5%B8%9D%E5%9B%BD.md)。

### 波罗的海与东欧河道

以瑞典中东部和哥特兰为重要基地的航海者经芬兰湾、拉多加湖、第聂伯河与伏尔加河前往罗斯、拜占庭和伊斯兰世界。他们经商、征贡、担任瓦兰吉卫队，也参与罗斯政权形成；这不是单向“殖民”，而是与斯拉夫、芬兰—乌戈尔、波罗的人群和草原政权共同塑造的网络。

## 重要事件

| 时间 | 事件 | 过程与意义 |
|---|---|---|
| 793年 | 林迪斯法恩遇袭 | 因英格兰文献记载充分，成为传统开端标志，但远航活动此前已存在 |
| 841年 | 都柏林长期据点形成 | 袭击者开始越冬并建设贸易、军事中心 |
| 845年 | 舰队进入巴黎 | 塞纳河路线显示内河航行和勒索能力扩大 |
| 865年 | “大异教军”进入英格兰 | 季节性袭击转为多年征服，约克及丹法区随后形成 |
| 约870—930年 | 冰岛定居与全岛议会形成 | 移民建立地方议会网络，930年传统上视为阿尔庭创立年 |
| 911年 | 诺曼底政治安排 | 罗洛获承认并承担防御责任，后裔逐渐法兰克化和基督教化 |
| 约965年 | 耶灵碑与丹麦王权基督教化 | 哈拉尔“蓝牙”以王权、纪念碑和新宗教强化统合，实际皈依是渐进过程 |
| 约985年 | 格陵兰诺斯聚落建立 | 北大西洋航线抵达高纬边疆，聚落依赖畜牧、猎获和远程交换 |
| 约1000年 | 北美短期据点 | 兰塞奥兹牧草地遗址证明诺斯人抵达北美，但未形成持久殖民国家 |
| 1016—1035年 | 克努特统治英格兰并联结三国 | 远征王权转为税收、教会和地方精英合作支撑的跨海君主政体 |
| 1030年 | 斯蒂克莱斯塔德战役 | 奥拉夫·哈拉尔松战死，其圣王崇拜后来加强挪威基督教王权 |
| 1066年 | 斯坦福桥战役 | 挪威国王哈拉尔三世战败，常被视为大型北欧征服英格兰时代的终点 |

## 兴盛条件

- **船舶与航海**：浅吃水长船可登陆、入河和快速撤离，货船则维持移民与贸易。
- **银和信用网络**：钱币、银锭、称量银及赎金让远征收益可以分配，城镇又把财富转为武器和船只。
- **政治竞争**：首领以远征积累声望，成功者再用财富建舰、笼络贵族并挑战王位。
- **目标地区分裂**：不列颠、法兰克和斯拉夫诸地的内战使北欧军队既能劫掠，也能作为盟军和雇佣兵介入。
- **移民能力**：家庭、牲畜、工匠和法律习惯随船移动，使部分航线由袭击转为定居社会。

## 转型与结束原因

### 结构变化

- 丹麦、挪威和瑞典王权逐渐建立税收、铸币、港口控制与区域军役，私人首领更难独立发动大规模远征。
- 教区、教会法和基督教王位观把统治者纳入欧洲政治网络；远征没有突然停止，但目标和合法性发生变化。
- 赫泽比、都柏林、约克等贸易节点制度化，商贸逐渐与王权保护、关税和城市治理结合。

### 外部压力

- 英格兰和法兰克诸政权修筑堡垒、组织舰队、支付或拒绝贡金，并提高沿海防御和快速动员能力。
- 东欧与拜占庭的权力格局变化，使瓦兰吉贸易和军事服务转入更稳定的国家框架。

### 直接标志

1066年挪威远征在斯坦福桥失败常作为西向征服阶段的象征性终点；约1100年前后，斯堪的纳维亚诸王国的教会、城市和王朝制度趋于稳定。因此“维京时代结束”更应理解为航海武装、贸易和移民被国家化、基督教化，而不是北欧人停止远航。

## 长期影响

- 北欧人参与塑造英格兰、爱尔兰、诺曼底、罗斯与北大西洋聚落的政治和语言文化。
- 奴隶贸易、战争屠杀和强迫征贡是扩张的重要组成，不能只以探险和贸易浪漫化。
- 远航积累的财政、舰队和统治经验促进区域王权，随后进入中世纪诸王国和[卡尔马联盟](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E6%AC%A7%E6%B4%B2/%E5%8C%97%E6%AC%A7/%E5%8D%A1%E5%B0%94%E9%A9%AC%E8%81%94%E7%9B%9F.md)等新的政治框架。
