## 概括

现代英国是议会君主制和责任内阁制国家。君主依法任命首相、批准法律并履行国家礼仪，但通常依大臣建议行事；能在下议院获得信任的首相领导政府。制度的关键不在一部统一宪法，而在议会立法、普通法、司法解释、王室特权和政治惯例的组合。

## 权力结构

| 主体 | 权力与制约 |
|---|---|
| 君主 | 国家元首；按惯例政治中立，绝大多数权力依首相和大臣建议行使。 |
| 下议院 | 民选立法机关；决定政府信任、预算和主要立法。 |
| 上议院 | 审议、修正和延缓法案，一般不能永久阻止下院主导的立法。 |
| 首相与内阁 | 实际行政核心；控制政府议程，但受党内、议会、法院、媒体和选举制约。 |
| 最高法院 | 解释法律、审查行政是否合法并裁决权力下放问题；不能以普通违宪审查废除议会法。 |
| 下放政府 | 苏格兰、威尔士、北爱在法定领域自治；法律上权限源于英国议会。 |
| 文官体系 | 政治中立、常任行政，为历届政府执行政策。 |

## 制度演变与重要事件

- 1911、1949年议会法限制上议院否决权，巩固民选下院优先。
- 两次世界大战扩大行政国家、财政和社会动员；1945年后福利国家与国民保健制度确立。
- 20世纪工党取代自由党，与保守党构成主要竞争；小党和区域政党在悬浮议会及下放政治中仍具影响。
- 1973年加入欧洲共同体使部分规则与欧洲机构共享；2016年公投和2020年脱欧重新强调议会与国内法自主。
- 1998年人权法把《欧洲人权公约》权利纳入国内司法；法院可宣告法律不相容，但最终修法仍由议会决定。
- 1999年后权力下放形成不对称联合；2014年苏格兰公投与北爱协议表明领土宪制依民主同意维系。
- 2009年最高法院取代上议院司法委员会，使最高司法职能与立法机构分离。
- 2019年最高法院裁定议会休会建议违法，显示法院可审查行政如何运用王室特权。

## 稳定机制与风险

制度稳定来自渐进改革、政党轮替、专业文官和议会负责制；灵活性也带来规则依赖惯例、行政在多数政府下权力集中、上下两院代表性不均等问题。脱欧、区域民族主义、上院改革和选举制度争论，是现代宪制持续调整的主要压力。
