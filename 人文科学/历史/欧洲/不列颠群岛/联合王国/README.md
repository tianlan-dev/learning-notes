# 联合王国

[返回不列颠群岛](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E6%AC%A7%E6%B4%B2/%E4%B8%8D%E5%88%97%E9%A2%A0%E7%BE%A4%E5%B2%9B/README.md)

## 历史主线

联合王国的国家主线从1707年开始：英格兰王国与苏格兰王国通过条约及两国联合法案组成大不列颠王国；1801年爱尔兰王国并入，形成大不列颠及爱尔兰联合王国；1922年爱尔兰自由邦成立，国家实际范围转为大不列颠及北爱尔兰联合王国。18—19世纪财政信用、海军、工业和殖民网络支持全球强国地位，20世纪则经历两次世界大战、福利国家、去殖民化、欧洲一体化、权力下放与脱欧。现代英国是议会君主制：君主为国家元首，能获下议院信任的首相领导实际行政。

## 重要转折

| 时间 | 转折 | 意义 |
|---|---|---|
| 1707年 | 英苏联合法案 | 共同议会、贸易、外交和财政，保留苏格兰法律、教会与教育。 |
| 1714年 | 汉诺威继承 | 议会法定新教继承落实；詹姆斯党发起军事挑战。 |
| 1721年后 | 沃波尔长期执政 | 首相与内阁惯例逐渐形成。 |
| 1756—1763年 | 七年战争 | 英国取得加拿大和印度优势，战争债务加剧美洲税争。 |
| 1775—1783年 | 美国独立战争 | 十三殖民地脱离，帝国重心转向亚洲和其他地区。 |
| 1801年 | 爱尔兰并入 | 国家扩大，但宗派与主权问题没有解决。 |
| 1832—1884年 | 三轮议会改革 | 选举代表性逐步扩大，现代政党和责任内阁成熟。 |
| 1911年 | 《议会法》 | 限制上院否决，确认民选下院优先。 |
| 1914—1918年 | 第一次世界大战 | 总体战、普选扩大、债务和爱尔兰危机重塑国家。 |
| 1922年 | 爱尔兰自由邦成立 | 26郡脱离，北爱留在联合王国。 |
| 1939—1945年 | 第二次世界大战 | 战争胜利与国力透支并存。 |
| 1945—1948年 | 福利国家建设 | 国民保健、社会保障和关键产业国有化。 |
| 1947年后 | 去殖民化 | 帝国转为主权国家与英联邦联系。 |
| 1973年 | 加入欧洲共同体 | 国内法与欧洲制度深度结合。 |
| 1998—1999年 | 北爱协议与权力下放 | 联合内部形成不对称自治。 |
| 2016／2020年 | 脱欧公投／正式退出 | 贸易、北爱安排和领土认同重新谈判。 |

## 现代权力结构

| 主体 | 角色 |
|---|---|
| 君主 | 国家元首；按惯例政治中立，绝大多数职权依大臣建议行使。 |
| 首相与内阁 | 实际行政核心，对下议院负责。 |
| 下议院 | 决定政府信任、预算和主要立法。 |
| 上议院 | 审议、修正和延缓法案，通常不能永久阻止下院意志。 |
| 最高法院 | 解释法律、审查行政合法性和权力下放争议，不以普通违宪审查废除议会法。 |
| 苏格兰、威尔士、北爱机构 | 在法定下放领域自治；权限范围不对称。 |
| 地方政府与英格兰地区机构 | 承担地方服务；英格兰没有与苏格兰等对应的全国下放议会。 |

截至2026年7月，国家元首为查理三世，首相为基尔·斯塔默。
