# 联合王国时期的爱尔兰

## 时间

1801年—1922年

## 概括

1801年联合法案撤销都柏林的爱尔兰议会，把爱尔兰并入大不列颠及爱尔兰联合王国。一个世纪内，天主教解放、土地制度改革、大饥荒、人口外流、文化复兴与自治运动相互叠加，使“在联合王国内改革”逐渐与“建立独立国家”两条路线分化。

## 国家与实际权力结构

| 层级 | 权力与限制 |
|---|---|
| 英国君主与威斯敏斯特议会 | 掌握主权、立法、税收和军队；爱尔兰选出议员进入下议院。 |
| 爱尔兰总督与都柏林城堡行政 | 代表英国政府统治爱尔兰，首席秘书逐渐成为实际行政负责人。 |
| 地方地主与治安体系 | 大量土地由地主出租，地方治安官、警察和法院维持秩序。 |
| 民族主义组织 | 从议会自治党、土地联盟到新芬党与爱尔兰志愿军，策略由议会改革扩展到武装独立。 |
| 联合派 | 尤其集中于工业化的阿尔斯特东北部，维护与英国的政治经济联系。 |

## 发展过程与重要事件

- 1803年罗伯特·埃米特起义失败；国家安全体系继续压制共和秘密组织。
- 1829年丹尼尔·奥康奈尔领导的群众运动促成天主教解放，使天主教徒可进入议会，但其废除联合运动未成功。
- 1845—1852年马铃薯疫病、单一作物依赖、贫困租佃制度与救济政策失效共同造成大饥荒。约百万人死亡、更多人移民，人口长期下降。
- 1869年爱尔兰圣公会取消国教地位；1870年以后多轮土地法逐步扩大佃农权利，最终促成地主土地向农民所有转移。
- 帕内尔领导爱尔兰议会党，以议会阻挠和土地联盟推动自治。1886、1893年自治法案均未生效。
- 19世纪末盖尔联盟、盖尔运动协会和文学复兴重建语言、体育与历史认同。
- 1912年第三次自治危机促成阿尔斯特志愿军与爱尔兰志愿军对立；1914年自治法因第一次世界大战暂停实施。
- 1916年复活节起义的军事失败及随后处决改变舆论；1918年新芬党胜选并拒赴威斯敏斯特，另组爱尔兰议会。

## 联合体制失去合法性的原因

结构原因包括土地与宗派不平等、重大政策由伦敦决定、饥荒创伤和跨大西洋民族主义网络。改革确实扩大参政和土地所有，却未能解决主权问题；阿尔斯特联合派反对接受都柏林自治又使统一自治方案难以实施。第一次世界大战、1916年镇压和1918年选举是路线转向独立的直接触发点。
