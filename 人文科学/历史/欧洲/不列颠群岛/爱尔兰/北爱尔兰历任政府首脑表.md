# 北爱尔兰历任政府首脑表

## 口径说明

北爱尔兰的实际行政首脑随宪制阶段改变：1921—1972年由北爱尔兰首相领导多数制政府；1972年以后英国北爱尔兰事务大臣在直接统治期掌握主要行政责任；1998年《贝尔法斯特协议》建立第一部长与副第一部长共同领导的权力分享制度。两职法律权力相等，必须联合履职，“副”不表示从属。行政停摆期由英国政府、北爱部门常任官员和后续临时立法共同维持有限治理，不应把某一人误列为完整自治政府首脑。

## 北爱尔兰事务大臣

北爱尔兰事务大臣是英国政府阁员。1972—1999年及下放暂停期，其部门对北爱事务承担直接行政与立法责任；正常下放期仍负责宪制、国家安全、选举、遗留问题等保留或例外事项。

## 现代实际权力结构

| 主体 | 权限 |
|---|---|
| 第一部长与副第一部长 | 联合领导行政办公室，必须跨联合派—民族派共同履职；共同协调政府政策。 |
| 北爱尔兰行政委员会 | 各部门由按党派比例分配的部长负责，集体权力分享而非单一多数内阁。 |
| 北爱尔兰议会 | 对卫生、教育、农业、地方经济、司法等下放领域立法，并受跨社群表决机制影响。 |
| 英国政府与议会 | 负责宪制、国防、外交、移民、国家安全等；下放暂停时可扩大直接治理。 |
| 爱尔兰政府与南北机构 | 不统治北爱，但依协议在跨境合作、咨询和共同机构中发挥作用。 |
| 英国君主 | 联合王国国家元首，不参与北爱日常党派行政。 |
