# 英格兰

[返回不列颠群岛](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E6%AC%A7%E6%B4%B2/%E4%B8%8D%E5%88%97%E9%A2%A0%E7%BE%A4%E5%B2%9B/README.md)

## 历史主线

英格兰从罗马撤军后的东南与中部政治重组中形成。盎格鲁、撒克逊、朱特及本地布立吞社群建立多个王国，威塞克斯王室在维京冲击下幸存并于10世纪统一英格兰。1066年诺曼征服更换土地与统治精英，却继承郡政、税收和王室文书基础；金雀花时期普通法、议会及跨海战争发展。15世纪兰开斯特、约克支系争位，都铎王朝重建秩序并发动宗教改革。1603年英苏王冠联合，1649—1660年君主制中断，1688—1689年议会革命后，1707年英格兰王国与苏格兰王国合为大不列颠王国，英格兰不再是独立主权国家。

## 重要转折

| 时间 | 转折 | 意义 |
|---|---|---|
| 878年 | 埃丁顿战役 | 威塞克斯幸存并建立对丹麦法区长期反攻的制度基础。 |
| 927年 | 埃塞尔斯坦取得约克 | 统一英格兰王权的通常起点。 |
| 1066年 | 斯坦福桥与黑斯廷斯 | 多重继承战争结束盎格鲁—撒克逊王室统治。 |
| 1154年 | 亨利二世继位 | 诺曼继承内战以安茹血统和平接续。 |
| 1215年 | 《大宪章》 | 内战和约逐步转化为王权受法律约束的传统。 |
| 1295年 | 模范议会 | 骑士与城市代表的税收协商制度化。 |
| 1399年 | 亨利四世夺位 | 金雀花支系竞争转为兰开斯特王朝。 |
| 1485年 | 博斯沃思战役 | 都铎取代约克，王朝战争进入收束。 |
| 1534年 | 《至尊法案》 | 英格兰教会脱离教宗管辖，国家与土地结构重组。 |
| 1603年 | 王冠联合 | 英格兰与苏格兰共享君主但仍为两个国家。 |
| 1649年 | 查理一世被处死 | 君主制和上院被废，展开共和国试验。 |
| 1688—1689年 | 光荣革命 | 议会决定王位条件，财政和责任政治进一步发展。 |
| 1707年 | 联合法案生效 | 英格兰王国并入大不列颠王国。 |
