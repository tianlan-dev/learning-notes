## 概括

苏格兰宗教改革把王国从天主教体系转向改革宗新教，并发展出长老会治理传统；斯图亚特君主则试图维持主教制和王权统一。两种教会政治观的冲突与英格兰、爱尔兰局势交织，引发盟约运动、三国战争、克伦威尔占领和复辟后的迫害，最终在1689—1690年革命安排中确立长老会制度。

## 宗教改革过程

- 16世纪初人文主义、路德宗和加尔文宗著作经港口与大学传播，王室仍严厉打击异端。
- 1542年玛丽女王幼年即位后，亲法、天主教摄政与亲英、新教贵族对立。英格兰“粗暴求婚”战争加剧社会动荡。
- 约翰·诺克斯等受日内瓦改革宗影响；1557年新教贵族组成“会众领主”联盟。
- 1559—1560年起义与英格兰干预迫使法国军队撤离。1560年苏格兰议会否定教宗权威、通过改革宗信条并禁止弥撒。
- 改革并未一次完成：旧教财产分配、牧师供给和教会治理长期争论。长老会主张由牧师与长老组成的教会法庭自治，不接受国王任命主教控制教会。

## 斯图亚特王权与教会冲突

| 阶段 | 统治与冲突 |
|---|---|
| 玛丽一世（1542—1567） | 天主教君主在新教化王国统治；婚姻、贵族叛乱与政治谋杀使其退位。 |
| 詹姆斯六世（1567—1625） | 在长老会与主教制间调整；1603年继承英格兰王位后形成王冠联合。 |
| 查理一世（1625—1649） | 强推主教制和新祈祷书，引发1638年《民族盟约》与主教战争。 |
| 盟约派与三国战争 | 苏格兰议会和教会限制王权，军队介入英格兰内战；盟约派内部随后分裂。 |
| 共和国占领（1651—1660） | 克伦威尔军队击败苏格兰，王国被纳入共和政体。 |
| 复辟（1660—1688） | 查理二世恢复主教制，政府镇压拒绝服从的盟约派聚会。 |
| 革命安排（1689—1690） | 詹姆斯七世被废，议会承认威廉与玛丽并恢复长老会国教。 |

## 重要事件

- 1560年宗教改革议会奠定新教国家框架。
- 1567年玛丽退位，幼子詹姆斯六世在新教摄政下继位。
- 1603年王冠联合使英苏共享君主，但议会、法律和教会仍分立。
- 1637年爱丁堡反对新祈祷书的骚乱触发全国动员；1638年民族盟约把宗教承诺转化为政治联盟。
- 1643年《庄严同盟与圣约》使苏格兰军队加入英格兰内战；1649年查理一世被处死后，苏格兰承认查理二世，引来共和国入侵。
- 1660年复辟后主教制恢复，“杀戮时代”的镇压强化长老会殉道记忆。
- 1689年权利主张否定詹姆斯滥权，1690年法律最终确立长老会；高地仍有詹姆斯党抵抗。

## 转型与长期影响

宗教冲突之所以持续，是因为教会治理同时决定谁能组织地方社会、任命牧师并约束君主。王冠联合扩大了君主推行三王国一致政策的诱因，却使苏格兰制度更担忧被英格兰吸收。1707年政治联合后，长老会的独立地位被条约保障，成为苏格兰在失去独立议会后仍保持制度连续性的核心。
