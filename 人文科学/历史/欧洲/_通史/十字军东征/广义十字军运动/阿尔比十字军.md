# 阿尔比十字军

## 时间

1209年－1229年

## 别称

卡特里派十字军

## 概括

阿尔比十字军是教宗意诺森三世号召的反异端战争，目标是法国南部朗格多克地区的卡特里派及其世俗保护者。它不仅是宗教镇压，也推动了法国王权向南部扩张，削弱了图卢兹伯国等地方贵族的独立性。

## 说明

- 背景：卡特里派在法国南部具有较强影响，被罗马教会视为严重异端。
- 导火索：1208年，教宗使节皮埃尔·德·卡斯泰尔诺遇刺，教宗意诺森三世号召十字军讨伐被认为纵容异端的南方领主。
- 主要参与者：西蒙·德·蒙福尔、法国北部贵族、图卢兹伯雷蒙六世及其继承人、教廷代表等。
- 贝济耶屠城：1209年十字军攻陷贝济耶，发生大规模屠杀，成为阿尔比十字军最著名的暴力事件之一。
- 卡尔卡松与图卢兹：十字军占领卡尔卡松，西蒙·德·蒙福尔取得大量南方领地；图卢兹长期抵抗。
- 后续镇压：军事战争之后，教会审判和宗教裁判机制加强，对卡特里派进行持续清剿。
- 结果：1229年《巴黎条约》削弱图卢兹伯国，法国王权对朗格多克的控制加强。

## 参与者与冲突层次

| 参与者 | 目标 | 需要区分之处 |
|---|---|---|
| 教宗意诺森三世及教会使节 | 消除被界定为异端的卡特里信仰，迫使领主配合 | 最初长期尝试宣讲，1208年后才全面军事化 |
| 北法十字军与西蒙·德·蒙福尔 | 获取赎罪，也争取领地、封号和战利品 | 40日服役期使军队季节性解散，蒙福尔须建立个人统治 |
| 图卢兹伯雷蒙六世、雷蒙七世 | 保存领地和地方贵族权利 | 并非都信奉卡特里教，但被指纵容或保护异端 |
| 阿拉贡王佩德罗二世 | 维护比利牛斯以北附庸和地区影响 | 本为教廷认可的天主教君主，却在米雷与十字军作战并阵亡 |
| 卡特里“完美者”、信徒及普通居民 | 维持宗教社群或地方生活 | “阿尔比派”是外部统称，信仰分布和参与程度不一致 |
| 法国卡佩王权 | 后期把南方纳入更稳定的王室秩序 | 王室初期谨慎，1226年后直接介入改变战争性质 |

## 具体过程

教会把卡特里二元论、拒绝部分圣礼和独立宗教权威视为对正统与教区秩序的威胁。熙笃会宣讲、辩论和多次教令未能迫使地方领主根除异端。1208年使节皮埃尔·德·卡斯泰尔诺遇刺成为直接导火索，责任归属并不完全清楚，但教宗把雷蒙六世及其政治环境作为讨伐对象。

1209年十字军攻陷贝济耶并发生无差别屠杀；中世纪记载给出的死亡数字带有修辞和争议，不宜当作精确统计。卡尔卡松投降后，特朗卡韦尔家族领地转给蒙福尔。北方贵族的短期服役与当地城堡网络使占领反复，战争很快从镇压宗教异端转成谁拥有朗格多克领地的政治战争。

1213年米雷战役中，蒙福尔击败图卢兹—阿拉贡联军，佩德罗二世战死。第四次拉特朗会议把雷蒙部分领地判给蒙福尔，但图卢兹居民和地方贵族重新反攻；1218年蒙福尔围城时死亡。其子无力维持全部征服，最终把权利交给法国王室。路易八世1226年亲征后，多座城市屈服；1229年条约迫使雷蒙七世割地、拆堡并安排继承，使图卢兹最终并入王室的路径制度化。

军事结束没有立即消灭卡特里信仰。教宗格列高利九世时期的宗教裁判官通过调查、证言和处罚追查社群；蒙塞居尔1244年陷落及后续审判是长期清剿的一部分。多明我会与异端斗争有关，但宗教裁判制度的形成不能简单写成一次战争当天“创立”。

## 兴起原因、终结与影响

- **宗教结构：** 教会担忧正统、圣礼和主教权威受挑战，并把拒绝合作的世俗领主纳入异端政治。
- **政治利益：** 北方贵族有获取封地的机会，卡佩王权则在后期把松散宗主权转为直接领土影响。
- **直接触发：** 使节遇刺提供全面动员理由，但战争持续二十年是因为占领、继承和地方抵抗，而非仅因宗教教义。
- **暴力后果：** 围城、屠杀、没收和流离影响卡特里与天主教居民；不能把所有南方人都视为卡特里信徒。
- **国家形成：** 条约、婚姻继承和王室行政比早期十字军胜利更持久地削弱图卢兹独立性。
- **知识与记忆：** 后世既把卡特里浪漫化为现代自由思想先驱，也把十字军简化为纯粹北方掠夺；可靠叙述须区分同时代神学、地方政治和后世身份建构。
