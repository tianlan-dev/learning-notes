# 后期反奥斯曼十字军

## 时间

14世纪－15世纪

## 概括

后期反奥斯曼十字军是奥斯曼帝国在巴尔干和东地中海扩张后，教廷、匈牙利、波兰、威尼斯、勃艮第等力量组织或参与的防御与反攻行动。它们延续了十字军的宗教动员语言，但主要目标已从收复耶路撒冷转为阻止奥斯曼进入中欧和控制东地中海。

## 说明

- 背景：14世纪后，奥斯曼帝国跨过达达尼尔海峡进入巴尔干，逐步削弱拜占庭和巴尔干诸国。
- 尼科波利斯十字军：1396年，匈牙利国王西吉斯蒙德与法国、勃艮第等西欧骑士组成联军，在尼科波利斯被奥斯曼苏丹巴耶济德一世击败。
- 瓦尔纳十字军：1443年－1444年，波兰-匈牙利国王瓦迪斯瓦夫三世、匈雅提·亚诺什等发动远征，1444年在瓦尔纳败于奥斯曼苏丹穆拉德二世，瓦迪斯瓦夫三世战死。
- 贝尔格莱德围城战：1456年，匈雅提·亚诺什与方济各会士卡皮斯特拉诺的若望组织防御，击退穆罕默德二世对贝尔格莱德的进攻，短期延缓奥斯曼向中欧推进。
- 地中海战线：威尼斯、热那亚、医院骑士团等海上力量持续与奥斯曼争夺爱琴海、希腊、罗得岛和东地中海据点。
- 与传统东征差异：后期反奥斯曼十字军更强调欧洲防御和巴尔干地缘政治，而不再以建立圣地十字军国家为核心。
- 结果：多次远征未能阻止奥斯曼长期崛起；1453年君士坦丁堡陷落后，反奥斯曼战争成为欧洲政治和宗教动员的长期主题。

## 参与者与联盟难题

| 参与者 | 战略目标 | 主要矛盾 |
|---|---|---|
| 教宗与使节 | 以赎罪、募款和外交组织跨国防御 | 无法消除王朝利益、教会分裂和军队指挥差异 |
| 匈牙利、波兰及巴尔干统治者 | 保护多瑙河边界、王位与附庸网络 | 对进攻时机、停战和领土分配意见不同 |
| 法国—勃艮第骑士及西欧志愿者 | 履行骑士和宗教誓愿、获取声望 | 不熟悉巴尔干地形与奥斯曼战术，纪律受贵族荣誉影响 |
| 威尼斯、热那亚、医院骑士团 | 保卫岛屿、港口、贸易和海峡 | 彼此商业竞争，可能同时与奥斯曼谈判 |
| 拜占庭与东正教政权 | 求援以保存领土 | 1204年记忆和教会合一争议妨碍拉丁—希腊互信 |
| 奥斯曼苏丹及边疆军人 | 整合巴尔干附庸、税源和交通 | 也受继承战争、安纳托利亚诸侯和帖木儿冲击 |

## 主要阶段

奥斯曼夺取加里波利后，以边疆军人、附庸兵、蒂玛尔骑兵和苏丹常备军逐步进入色雷斯、保加利亚与塞尔维亚。马里查和科索沃等战役使若干巴尔干统治者死亡或接受附庸关系。西方动员因此从“援救圣地”转为保卫匈牙利、多瑙河和君士坦丁堡，但参战者仍使用十字、赎罪和教廷授权。

1396年尼科波利斯联军沿多瑙河推进。法勃骑士不愿等待侦察和匈牙利步兵，贸然冲击奥斯曼前线，在越过防御后又遭巴耶济德预备队和塞尔维亚附庸骑兵反击。失败来自指挥、地形、纪律和奥斯曼纵深，不能归结为“骑士落后”一个因素。

1402年帖木儿在安卡拉击败并俘虏巴耶济德，奥斯曼进入内战，巴尔干获得喘息。穆拉德二世恢复统一后，匈雅提·亚诺什等人在1443年长途远征取得进展。1444年和议刚订，教廷使节又推动恢复战争；瓦迪斯瓦夫三世在瓦尔纳战场突击苏丹中军时战死，联军崩溃。1448年第二次科索沃失败进一步削弱大规模陆上反攻可能。

穆罕默德二世1453年攻陷君士坦丁堡，却在1456年贝尔格莱德遭匈雅提、卡皮斯特拉诺及守军击退。胜利延缓奥斯曼直接进入匈牙利腹地，说明反奥斯曼行动并非全然失败；但它未恢复巴尔干旧王国。此后教廷号召、威尼斯战争、罗得岛防御和地中海联盟继续沿用十字军语言。

## 成败因果与长期影响

- **奥斯曼优势：** 苏丹能结合常备步兵、蒂玛尔骑兵、火炮、附庸部队和连续税源，并在胜利后保留部分地方精英。
- **联盟弱点：** 远征军来自多个王国，军令、语言、补给和战争期限不同；王位、贸易与教会分裂使同盟难以持久。
- **直接触发失误：** 尼科波利斯的冒进、瓦尔纳撕毁或放弃停战后的仓促会战，都把可控战略转成高风险决战。
- **外部变量：** 帖木儿冲击一度逆转局势，证明奥斯曼崛起并非不可阻挡；西欧百年战争等冲突也限制援军。
- **长期结果：** 巴尔干成为多瑙河堡垒、附庸国、边防军和难民流动构成的军事边疆；“十字军”逐渐与王朝联盟、海军战争和早期近代外交混合。
- **概念边界：** 并非每场基督教—奥斯曼战争都正式获得十字军特许，具体行动应分别核对教宗号召、赎罪和募款制度。
