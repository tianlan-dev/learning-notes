# 东罗马帝国与拜占庭帝国

## 时间

395年-1453年

## 别称

Eastern Roman Empire、Byzantine Empire

## 概括

以君士坦丁堡为中心的东部帝国延续罗马法统，后世常称拜占庭帝国，直到1453年被奥斯曼攻灭。

## 统治结构 / 主要阶段

东罗马帝国以君士坦丁堡为中心延续罗马帝国法统；后世称拜占庭帝国。

| 阶段 | 时间 | 身份 / 关系 | 说明 |
|---|---|---|---|
| 早期东罗马 | 395-527 | 君士坦丁堡中心 | 承接罗马行政和基督教帝国传统。 |
| 查士丁尼时代 | 527-565 | 查士丁尼一世 | 编纂《民法大全》，一度收复意大利和北非。 |
| 中期拜占庭 | 7-11世纪 | 军区制和希腊化帝国 | 抵御阿拉伯、保加尔和斯拉夫压力。 |
| 后期帝国 | 1204-1453 | 十字军、拉丁帝国、复国后衰落 | 第四次十字军重创帝国，最终被奥斯曼攻灭。 |

### 皇帝世系 / 王朝概览

东罗马皇帝世系极长，完整皇帝表可单独成篇；此处先按主要王朝和统治集团概括。

| 顺序 / 王朝或阶段 | 时间 | 代表皇帝 | 关键事件 / 说明 |
|---|---|---|---|
| 狄奥多西王朝东部支系 | 395-457 | 阿卡狄乌斯、狄奥多西二世、马尔基安 | 东部以君士坦丁堡为中心稳定延续。 |
| 利奥王朝 | 457-518 | 利奥一世、芝诺、阿纳斯塔修斯一世 | 476年后成为唯一罗马皇帝法统中心。 |
| 查士丁尼王朝 | 518-602 | **查士丁尼一世**、查士丁二世、莫里斯 | 编纂《民法大全》，收复北非、意大利部分地区。 |
| 希拉克略王朝 | 610-711 | **希拉克略**、君士坦斯二世、君士坦丁四世 | 抵御波斯和阿拉伯扩张，帝国希腊化加深。 |
| 伊苏里亚王朝及圣像破坏时代 | 717-802 | 利奥三世、君士坦丁五世、伊琳娜 | 稳住阿拉伯攻势，圣像破坏争议影响政教关系。 |
| 马其顿王朝 | 867-1056 | 巴西尔一世、利奥六世、**巴西尔二世** | 中期复兴，法制、军事和文化影响扩大。 |
| 科穆宁王朝 | 1081-1185 | 阿历克塞一世、约翰二世、曼努埃尔一世 | 十字军时代维持帝国，但财政和军事压力上升。 |
| 拉斯卡里斯王朝 / 尼西亚帝国 | 1204-1261 | 狄奥多尔一世、约翰三世 | 第四次十字军后保存拜占庭法统并收复君士坦丁堡。 |
| 巴列奥略王朝 | 1261-1453 | 米海尔八世、约翰五世、**君士坦丁十一世** | 复国后长期衰落，1453年君士坦丁堡陷落。 |

## 重要事件

- 330年，君士坦丁堡成为帝国新都。
- 537年，圣索菲亚大教堂建成。
- 1204年，第四次十字军攻陷君士坦丁堡。
- 1261年，拜占庭复国。
- 1453年，君士坦丁堡陷落，东罗马帝国灭亡。
