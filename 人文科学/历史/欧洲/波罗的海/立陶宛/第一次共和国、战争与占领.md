# 第一次共和国、战争与占领

## 时间

1918—1940年

## 概括

1918年2月16日独立法案只是国家重建的起点。德国撤军后，立陶宛必须在布尔什维克、西俄志愿军和波兰三条战线上建立军队、征税与地方行政；维尔纽斯被波兰方面控制，使考纳斯成为“临时首都”。1920—1926年共和国以议会民主、土地改革、货币和教育建设巩固国家，1926年军事政变后则转为安塔纳斯·斯梅托纳的总统威权体制。1938—1940年波兰、德国和苏联连续最后通牒暴露小国安全困境；1940年6月苏联以驻军为后盾强迫改组政府并吞并立陶宛，第一共和国的国内统治终结，但外交使团和法律连续性主张没有消失。

## 从宣言到政府

### 德国战败后的权力真空

1918年11月德国革命和停战使军事占领迅速松动。立陶宛委员会改称国家委员会，撤销未实施的“明道加斯二世”君主方案，采用共和临时宪制。奥古斯蒂纳斯·沃尔德马拉斯于11月11日组建首届政府，但最初认为无需大规模军队，且许多行政岗位、铁路和通信仍由德军掌握。

苏俄宣布废除布列斯特和约并向西推进后，米科拉斯·斯莱热维丘斯政府于12月号召志愿者参军，承诺土地改革，迁往考纳斯继续运作。国家的实际建立依靠三件事同时推进：

1. 由志愿兵、前俄军士兵和德国志愿部队协同阻挡红军；
2. 在县、镇建立税收、警察和地方委员会；
3. 寻求协约国承认、贷款、武器和边界支持。

## 独立战争

### 对布尔什维克战争

1918年底红军进入东部，支持温察斯·米茨克维丘斯-卡普苏卡斯建立立陶宛苏维埃共和国，后与白俄罗斯苏维埃共和国合并为“立陶宛—白俄罗斯苏维埃共和国”。1919年初红军控制维尔纽斯并逼近考纳斯。

立陶宛军与仍在撤退的德国及萨克森志愿部队合作守住核心，随后在乌克梅尔盖、帕涅韦日斯等方向反攻。1919年夏红军被逐出立陶宛语核心区，战线向东移动。1920年7月立陶宛与苏俄签订和平条约，俄方承认独立并承认包括维尔纽斯在内的广泛边界；这份条约没有解决波兰—立陶宛实际控制冲突。

### 西俄志愿军 / 贝尔蒙特军

1919年，帕维尔·贝尔蒙特-阿瓦洛夫领导的西俄志愿军由俄国白军名义和德军残余组成，占据拉脱维亚、立陶宛北部铁路和城市，既不服从波罗的海新国家，也不优先攻击布尔什维克。立陶宛军在拉德维利什基战斗中击败其部队，缴获大量装备，至年底迫使其撤离。

### 与波兰的战争和维尔纽斯问题

立陶宛民族国家以维尔纽斯为历史首都，波兰国家则强调当地波兰语人口、旧联邦传统和约瑟夫·毕苏斯基的联邦方案。1919年双方在苏瓦乌基和东部地区冲突，波兰军事组织还策划在考纳斯政变，被立陶宛当局破获。

1920年波苏战争改变控制线。苏军撤出后，立陶宛进入维尔纽斯；10月双方刚签苏瓦乌基协议，波兰将领卢齐安·热利戈夫斯基即以“兵变”名义占领维尔纽斯，建立中立陶宛，1922年经有争议的选举并入波兰。波兰政府与行动保持可否认距离，但毕苏斯基后来承认策划。

结果是：

- 立陶宛政府以考纳斯为临时首都，宪法继续把维尔纽斯列为首都；
- 两国直到1938年才建立正常外交关系；
- 边界和民族少数问题成为两国安全合作的长期障碍；
- 立陶宛的外交资源大量用于“维尔纽斯问题”。

## 议会共和国

### 制宪议会与1922年宪法

1920年制宪议会经普选产生，女性享有选举权。议会确认共和国、通过土地改革、建立中央银行和立塔斯货币，并制定1922年宪法。制度以议会为中心，总统由议会选出，内阁须获多数支持。

| 建国领域 | 主要措施 | 影响 |
| --- | --- | --- |
| 土地 | 拆分大庄园，优先分给无地农民和志愿兵 | 削弱波兰化大地主，扩大自耕农和国家认同。 |
| 货币财政 | 1922年引入立塔斯、建立中央银行 | 结束战后多种货币混用，稳定财政。 |
| 教育 | 扩大小学、建立立陶宛语中学和考纳斯大学 | 培养本国语官僚与专业阶层。 |
| 外交 | 加入国际联盟，争取大国承认 | 主权获承认，但维尔纽斯边界未获本国接受。 |
| 少数群体 | 早期给予犹太事务自治及学校空间 | 后来自治缩减；波兰、犹太、德意志等社群处境随政治变化。 |

议会政府更替频繁，但不是“国家完全失控”：行政、货币、土地和教育改革取得实质成果。政党围绕教会与世俗教育、土地、少数群体、外交和劳工政策竞争。

### 克莱佩达地区

凡尔赛安排把德国的梅梅尔地区置于协约国管理。1923年立陶宛支持的武装行动夺取该区，随后国际协议承认立陶宛主权并赋予克莱佩达自治、议会和特殊港口地位。克莱佩达为立陶宛提供重要不冻港和工业，却有大量德语人口，德国影响始终存在。

## 1926年政变与威权化

### 政变过程

1926年议会选举产生以农民人民联盟和社会民主党为主的中左政府，卡齐斯·格里纽斯任总统、斯莱热维丘斯任总理。政府解除部分戒严、允许更广政治活动，并与苏联签不侵犯条约。保守军官、民族主义者和基督教民主派以“共产主义威胁”、军队利益和社会失序为由反对。

12月17日军队控制考纳斯机关，拘押政府成员。奥古斯蒂纳斯·沃尔德马拉斯任总理，议会在压力下选安塔纳斯·斯梅托纳为总统。政变初期仍使用宪法程序外观，但很快转为新制度。

### 权力集中

- 1927年总统解散议会，此后近九年没有正常议会；
- 1928年宪法扩大总统权力，未经制宪议会批准；
- 民族主义联盟成为统治核心，其他政党逐步被禁止或限制；
- 警察、审查和行政控制压制共产党、左翼及政权内部竞争者；
- 沃尔德马拉斯1929年被斯梅托纳解除，铁狼派后遭镇压；
- 1936年重开经严格控制的议会，1938年宪法进一步确认总统至上。

斯梅托纳体制比纳粹或斯大林政权的群众极权程度低，保留私人财产、教会与有限社会组织，但不再是竞争性议会民主。

### 威权政权的稳定条件

| 因素 | 作用 |
| --- | --- |
| 农业自耕农社会 | 土地改革使大量家庭与国家秩序利益相连。 |
| 官僚与教育建设 | 新一代立陶宛语公务员、教师和军官支持国家化。 |
| 反共与安全焦虑 | 苏联威胁、边界争端被用于正当化强政府。 |
| 经济政策 | 合作社、农产品出口和相对稳健货币支撑日常秩序。 |
| 反对派分裂 | 基督教民主派、左翼、沃尔德马拉斯派和共产党难形成共同替代。 |

## 社会、经济与文化

考纳斯从边缘要塞城市变成临时首都，现代主义建筑、大学、剧院、出版和行政机关快速发展。农业仍占主导，奶制品、肉类和亚麻面向欧洲市场；大萧条使价格和农民收入下降，1935年苏瓦尔基亚农民抗议遭镇压。

社会具有明显多样性：

- 犹太人是考纳斯、维尔纽斯之外许多城镇商业、手工业与文化的重要群体，存在希伯来语、意第绪语学校和政治组织；
- 克莱佩达德语人口享自治，但纳粹运动在1930年代增长；
- 留在立陶宛边界内的波兰语人口与国家教育、土地和忠诚政策摩擦；
- 罗姆人、俄罗斯人、白俄罗斯人、卡拉伊姆人与鞑靼人等规模较小社群也存在。

国家推动立陶宛语公共文化，同时从早期少数文化自治逐渐转向更统一的民族国家模式。不能用后来大屠杀的结果倒推战前所有社群关系，但经济竞争、民族政治和反犹主义确实存在，并在占领战争中被极端化。

## 1938—1940年外交危机

### 波兰最后通牒

1938年边境事件后，波兰要求立陶宛在短期内建立外交关系，暗含军事压力。立陶宛接受，结束长期“无战无和”，却没有正式承认维尔纽斯归波兰。事件暴露军力与盟友不足，也损害威权政权声望。

### 德国吞并克莱佩达

1939年3月，希特勒德国发出最后通牒，要求交还克莱佩达。缺乏外援的立陶宛被迫接受。德国取得港口和工业区，立陶宛丧失主要出海设施之一，经济安全进一步恶化。

### 苏德协议与苏联基地

1939年8月苏德互不侵犯条约秘密议定书最初把立陶宛大部划入德国势力范围，9月修订后转入苏联范围。苏联击败波兰东部后，以交还维尔纽斯及部分地区为条件，迫使立陶宛签互助条约并允许苏军驻扎。

“维尔纽斯归还”与主权受限同时发生：政府获得历史首都，却让一支远强于本国军队的外国武装进入境内。芬兰拒绝类似要求后爆发冬季战争，更使立陶宛领导层担心军事毁灭。

## 1940年苏联占领与第一共和国终结

### 直接过程

1. 苏联指控立陶宛绑架士兵、违反互助条约；
2. 1940年6月14日发出最后通牒，要求改组政府、逮捕官员和无限增兵；
3. 斯梅托纳主张抵抗，但内阁多数认为无胜算；总统离境；
4. 苏军6月15日进入并控制全国；
5. 苏联特使弗拉基米尔·杰卡诺佐夫指导组建帕莱茨基斯“人民政府”；
6. 7月举行只有受控名单参选的“人民议会”选举；
7. 新议会请求加入苏联，8月3日苏联最高苏维埃宣布接纳；
8. 国有化、逮捕、政治清洗和制度苏维埃化展开。

### 为何迅速失去主权

| 类型 | 因素 |
| --- | --- |
| 外部力量 | 苏联驻军已在境内，军事规模和地理位置使常规抵抗极困难。 |
| 国际环境 | 法国将败、英国自保、德国与苏联仍合作，立陶宛无可执行安全保证。 |
| 地区孤立 | 与波兰长期冲突、波罗的海三国军事合作不足，未形成共同防御。 |
| 国内决策 | 威权体制把重大决策集中于小圈子，政府与总统对抵抗意见分裂。 |
| 直接触发 | 6月最后通牒和无限增兵要求；接受后苏联立即控制任命和选举。 |
| 法理辨析 | 武力威胁下的改组、受控选举和吞并请求不构成自由主权转移。 |

### 没有终结的国家连续性

驻华盛顿、伦敦、罗马等地的立陶宛外交机构拒绝承认吞并，部分西方国家依据不承认政策继续承认其法律人格。1940年占领终结了本土政府，却没有得到普遍国际法承认；这成为1990年使用“恢复独立”而非“新建国家”的基础。

## 重要事件

| 时间 | 事件 | 转折 | 长期影响 |
| --- | --- | --- | --- |
| 1918-11 | 首届政府成立 | 德国败退，委员会转为实际行政 | 从独立宣言进入国家建构。 |
| 1918—1920 | 三线独立战争 | 击退红军、贝尔蒙特军并与波兰交战 | 保住核心领土，但失去维尔纽斯。 |
| 1920-07 | 对苏和平条约 | 苏俄承认独立和广泛边界 | 成为外交依据，实际边界受波兰战争改变。 |
| 1920-10 | 热利戈夫斯基夺取维尔纽斯 | 波兰支持的“兵变”突破停战安排 | 考纳斯成为临时首都，两国长期敌对。 |
| 1920—1922 | 制宪议会和宪法 | 普选议会完成土地、货币和制度改革 | 建立议会共和国。 |
| 1923 | 克莱佩达行动 | 本地武装和立陶宛支持改变协约国管理 | 获得港口与自治地区。 |
| 1926-12 | 军事政变 | 中左政府被推翻，斯梅托纳复位 | 民主倒退，威权体制形成。 |
| 1938 | 波兰最后通牒 | 被迫建交 | 显示安全孤立。 |
| 1939-03 | 德国夺取克莱佩达 | 军事威胁下割地 | 港口和战略地位受重创。 |
| 1939-10 | 苏联互助条约 | 以维尔纽斯换驻军 | 苏联取得占领前置条件。 |
| 1940-06—08 | 最后通牒、占领和吞并 | 苏军控制、傀儡政府、受控议会 | 第一共和国本土政权终结，法律连续性转入外交与地下。 |
