# 俄罗斯帝国统治与民族复兴

## 时间

1795—1918年

## 概括

1795年波兰—立陶宛联邦第三次瓜分后，立陶宛大部分并入俄罗斯帝国，涅曼河以西一部分先归普鲁士，拿破仑战争后进入俄属波兰王国体系。帝国统治并未把原大公国作为一个完整政治单位保存，而是拆入多个省份，以总督、官僚、军队、东正教机构和俄语教育逐步整合。1830—1831年与1863—1864年起义失败后，旧贵族主导的联邦复兴路线遭重创；农奴制废除、识字扩展、天主教抵抗、秘密学校和书籍走私又推动以立陶宛语言和农民社会为基础的新民族运动。第一次世界大战的德军占领摧毁帝国行政，1917—1918年立陶宛委员会利用俄德崩溃恢复国家。

## 瓜分与帝国行政重组

### 大公国为何灭亡

[立陶宛大公国](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E6%AC%A7%E6%B4%B2/%E6%B3%A2%E7%BD%97%E7%9A%84%E6%B5%B7/%E7%AB%8B%E9%99%B6%E5%AE%9B%E5%A4%A7%E5%85%AC%E5%9B%BD.md)在18世纪后期尝试用五三宪法、扩军和财政改革摆脱俄国干预。保守贵族邀请俄军镇压改革，随后俄、普、奥通过三次瓜分消灭联邦。1794年科希丘什科起义失败是直接军事转折，1795年末代共同君主退位。

原大公国没有整体成为“俄罗斯治下的立陶宛王国”。帝国把土地拆入维尔纳、科夫诺、格罗德诺、明斯克等省；今日立陶宛西南部的苏瓦乌基地区先归普鲁士，1807年进入华沙公国，1815年成为俄国控制的波兰王国一部分。现代边界地区因此在法律、土地制度和行政经历上并不一致。

### 统治结构

| 层级 | 机构 / 人员 | 主要职能 | 对地方社会的影响 |
| --- | --- | --- | --- |
| 帝国皇帝与中央部 | 圣彼得堡政府、内务部、教育部 | 法律、军队、税收、任命和宗教政策 | 取消原大公国主权机构，将地方视为帝国“西北边疆”。 |
| 维尔纳总督体系 | 总督 / 西北边疆总督 | 协调多省军政、警察和镇压 | 起义后权力扩大，穆拉维约夫时期以处决、流放和没收著称。 |
| 省与县官僚 | 省长、警察、法院、税务机关 | 日常行政、征兵、土地和治安 | 俄语官僚逐步取代旧联邦机构。 |
| 波兰化贵族 | 大地主、天主教教会和旧地方精英 | 庄园、文化、慈善和地方关系 | 早期反抗多由其领导，但与立陶宛语农民社会存在文化距离。 |
| 农民与村社 | 庄园劳役、村社、教区 | 农业生产、地方互助与宗教生活 | 19世纪后期成为立陶宛语言民族运动的主要社会基础。 |

## 拿破仑战争与旧联邦想象

1812年拿破仑大军进入立陶宛，许多贵族把它视为恢复大公国或波兰—立陶宛国家的机会。维尔纽斯设临时委员会，组织军队和供给，但法军征发沉重，国家重建方案从未稳定落实。拿破仑撤退后俄国恢复统治，并以更强警察监控防止贵族秘密结社。

这一事件显示早期政治认同仍主要围绕旧联邦和贵族国家，不等同后来以立陶宛语、民族教育和独立共和国为目标的运动。

## 1830—1831年起义及其后果

波兰王国起义传入立陶宛，地方贵族、学生、神职人员和农民武装袭击俄军据点。起义者目标并不统一：部分要求恢复1772年前联邦，部分强调农民减负，立陶宛独立民族纲领尚未成熟。俄军以正规军和交通优势逐步镇压。

### 镇压措施

- 1832年关闭维尔纽斯大学，削弱本地高等教育和知识中心；
- 没收起义贵族财产，处决或流放参与者；
- 限制天主教修会和学校，把部分教产转给国家或东正教会；
- 加强俄语行政和对出版、学校的审查；
- 把“立陶宛—鲁塞尼亚历史地区”行政话语改称帝国“西北边疆”。

镇压削弱旧贵族政治，却没有立即创造现代民族运动。天主教教区、乡村读写传统和流亡知识圈保存了反帝国记忆。

## 社会转型与农奴制废除

1861年亚历山大二世改革在帝国西部废除农奴制。农民获得人身自由和分配土地，但须承担赎买、地块零碎和庄园纠纷。改革产生两面效果：

- 地主对农民劳役和司法控制下降；
- 农民需要现金、教育和市场信息，识字需求增长；
- 农民家庭分化，部分迁入城市或移民美国等地；
- 立陶宛语农民不再只是贵族政治的动员对象，而逐步形成自己的教师、神职人员和知识分子。

西南苏瓦乌基地区因受波兰王国不同法律影响，农民土地状况和识字发展又与帝国直属省份有所区别。

## 1863—1864年起义

“ 一月起义”从波兰王国扩展到立陶宛和白俄罗斯。齐格蒙特·谢拉科夫斯基、安塔纳斯·马茨凯维丘斯、卡斯图斯·卡利诺夫斯基等领导者组织游击队，试图把贵族国家复兴与农民土地诉求结合。不同地区的语言、社会和政治目标仍有差异。

俄军总督米哈伊尔·穆拉维约夫以公开处决、军事法庭、流放西伯利亚、村庄罚款和没收财产镇压，因而被反抗者称为“绞刑者”。1864年主要组织被摧毁。起义失败造成三个长期转折：

1. 旧联邦式武装起义路线暂时破产；
2. 帝国进一步压制波兰语贵族文化和天主教组织；
3. 立陶宛民族活动转向语言、出版、教育、合作社和社会组织。

## 俄化、书禁与文化抵抗

### 书禁

1864—1904年，帝国禁止以拉丁字母印刷和公开传播立陶宛语出版物，企图推广西里尔字母版本。政策没有消灭语言出版，反而形成跨境网络：

- 书籍在东普鲁士蒂尔西特、拉格尼特等地印刷；
- “书籍走私者”越境运输祈祷书、报纸、历书和教科书；
- 秘密乡村学校由教师、神父或家庭教授立陶宛语；
- 天主教信仰、拉丁字母和民族语言逐渐被结合为抵抗身份。

书籍走私既是文化运动也是社会组织训练：筹款、印刷、运输、分销和警戒网络后来可转化为政党与自治运动。

### 民族复兴的主要流派

| 流派 / 媒介 | 代表 | 主张与作用 |
| --- | --- | --- |
| 文化民族主义 | 约纳斯·巴萨纳维丘斯、《黎明》 | 强调语言、民俗、历史和立陶宛独立文化主体。 |
| 自由民族主义 | 温察斯·库迪尔卡、《钟声》 | 推动公民、经济自立和政治自治；其诗后来成为国歌歌词。 |
| 天主教社会运动 | 神职人员、秘密学校和合作组织 | 保护教会与语言教育，动员乡村社会。 |
| 社会民主主义 | 1896年成立的立陶宛社会民主党 | 连接工人、土地和民族自治诉求，部分派别主张独立。 |
| 基督教民主主义 | 天主教知识分子 | 以宗教社会原则和民族权利组织政治。 |

语言标准化以东高地—西南方言为基础，逐步形成现代书面立陶宛语。民族运动的“立陶宛”更多以语言共同体为中心，与波兰语贵族的旧大公国认同、白俄罗斯民族运动和维尔纽斯多族城市现实发生竞争。

## 1904—1914年：合法政治空间扩大

1904年书禁撤销，1905年俄国革命迫使帝国短暂开放。约两千名代表参加维尔纽斯大议会，要求立陶宛在民族边界内获得广泛自治、以维尔纽斯为中心，并主张在地方行政和学校使用立陶宛语。会议没有建立政府，却显示跨阶层、跨党派全国政治组织已形成。

此后：

- 合法报刊、学校、文化社团和合作社快速增长；
- 立陶宛代表进入帝国国家杜马；
- 政党围绕自治、土地、教会和社会改革竞争；
- 维尔纽斯的波兰人、犹太人、白俄罗斯人和立陶宛人对城市与地区归属形成不同政治方案。

## 第一次世界大战与德国占领

### 俄国撤退与“东方军政区”

1915年德军占领立陶宛，纳入“东方军政区”。德国军方以交通、征发和安全为中心治理：

| 领域 | 政策 | 后果 |
| --- | --- | --- |
| 行政 | 军事官员直接管理，严格证件和旅行制度 | 地区交通与社会联系被切割。 |
| 经济 | 征收粮食、牲畜、木材和劳工 | 农村负担沉重，城市供应困难。 |
| 文化 | 起初限制政治，后允许有限民族学校和委员会 | 立陶宛精英获得组织空间，但必须规避德国控制。 |
| 外交 | 德国考虑建立附庸国家和经济军事联盟 | 促使立陶宛委员会在“依德”与真正独立之间反复谈判。 |

### 维尔纽斯会议与立陶宛委员会

1917年9月维尔纽斯会议选出20人立陶宛委员会。德国要求独立声明与永久联盟、军事和关税条约绑定；委员会内部则希望利用德国承认，同时保留制宪议会决定国家制度的权利。

1918年2月16日，委员会通过独立法案，宣布恢复以维尔纽斯为首都的民主立陶宛国家，断绝过去对其他国家的一切国家联系，并由民选制宪议会决定最终制度。它成为现代国家法律连续性的核心文件。

## 1918年的君主方案

德国仍希望建立可控制的君主国。委员会多数于1918年7月邀请符腾堡的威廉·冯·乌拉赫担任“明道加斯二世”，并拟定限制君权、要求居住立陶宛和学习语言的条件。该方案从未实际施政或完成加冕：

- 德国政府没有形成一致支持；
- 委员会内左派反对；
- 战局逆转使依德君主制失去基础；
- 1918年11月委员会正式撤销邀请，转向共和国临时宪制。

因此，威廉不列为实际在位的立陶宛国王；该事件反映小国精英在德俄帝国崩溃间寻找外部保障的策略。

## 帝国统治衰落与独立成功的原因

| 类型 | 因素 |
| --- | --- |
| 帝国结构 | 分省统治和俄化未能建立地方认同，反而使语言、教会和民族权利结合。 |
| 社会转型 | 农奴制废除、教育和市场扩展产生立陶宛语知识分子、合作社和政党。 |
| 组织积累 | 书籍走私、秘密学校、合法报刊与1905年议会训练了全国网络。 |
| 战争冲击 | 1915年俄国行政崩溃，1918年德国战败，两个控制中心先后退出。 |
| 国际环境 | 民族自决话语、俄国革命、德国求和与战后秩序给小国建国窗口。 |
| 直接机制 | 立陶宛委员会用2月16日法案确立连续性，再在德国崩溃时组建政府和军队。 |

## 重要事件

| 时间 | 事件 | 过程 | 长期影响 |
| --- | --- | --- | --- |
| 1795 | 第三次瓜分 | 原大公国被俄、普等分割 | 立陶宛国家机构消失，进入帝国分省统治。 |
| 1812 | 拿破仑入侵 | 贵族建立临时委员会，期待恢复国家 | 旧联邦政治想象短暂复兴后再受镇压。 |
| 1830—1831 | 第一次大起义 | 波兰王国起义扩展到立陶宛 | 大学关闭、贵族财产没收和俄化加强。 |
| 1861 | 废除农奴制 | 农民获得人身自由和土地安排 | 形成民族运动的新社会基础。 |
| 1863—1864 | 一月起义 | 贵族、神职人员和农民游击失败 | 武装复国路线受挫，文化民族运动转强。 |
| 1864—1904 | 拉丁字母书禁 | 帝国压制出版，跨境书籍走私兴起 | 语言、天主教和民族认同更紧密。 |
| 1883、1889 | 《黎明》《钟声》创刊 | 海外印刷并秘密传播 | 塑造现代民族历史、语言和政治词汇。 |
| 1905 | 维尔纽斯大议会 | 各派代表要求广泛自治 | 从文化复兴转入群众政治。 |
| 1915 | 德军占领 | 俄国行政撤退，德国军管和征发 | 帝国连续性断裂，民族委员会获得有限空间。 |
| 1917 | 维尔纽斯会议 | 选出立陶宛委员会 | 建立独立声明和临时国家机关的组织核心。 |
| 1918-02-16 | 独立法案 | 宣布民主国家与法律关系断绝 | 成为共和国恢复独立的法理基础。 |
| 1918-11 | 德国战败、君主方案撤销 | 委员会转向共和国并组阁 | 独立从宣言进入国家建构与战争阶段。 |
