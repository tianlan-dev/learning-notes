# 十字军征服、丹麦与骑士团统治

## 时间

1208—1561年

## 概括

13世纪初，里加主教、宝剑骑士团、受洗的利沃尼亚盟军、丹麦王权和瑞典远征者从南北两线进入爱沙尼亚。征服历时约二十年，经过结盟、受洗、反叛和再征服，而非一次战役完成。此后爱沙尼亚被分入丹麦属爱沙尼亚公国、利沃尼亚骑士团领地以及塔尔图、萨雷—莱内等主教领；汉萨城市和波罗的德意志封建等级成为权力核心。1343—1345年圣乔治之夜起义失败后，丹麦出售北部属地，骑士团势力扩大。16世纪宗教改革、莫斯科进攻和内部财政军事脆弱共同终结旧利沃尼亚秩序。

## 征服过程

### 南线：里加主教与宝剑骑士团

里加主教阿尔伯特以十字军、商人、传教士和宝剑骑士团为支柱，先控制道加瓦河口与利沃尼亚人地区，再向乌甘迪和萨卡拉推进。征服者利用已受洗的利沃尼亚人与拉特加利亚盟军，也利用爱沙尼亚各县之间缺乏统一指挥。地方势力则反复袭击城堡、拒绝贡赋，并在不同外援之间转换。

1217年圣马太日战役中，萨卡拉长老伦比图与十字军一方的利沃尼亚领袖考波均战死。战役没有立即结束抵抗，却削弱南部联盟的协调中心。此后乌甘迪、萨卡拉等地在骑士团、主教与本地反抗之间数度易手。

### 北线：丹麦与瑞典竞争

1219年丹麦国王瓦尔德马二世在林达尼瑟登陆并击败当地军队，建立后来发展为塔林的城堡和主教体系，控制哈留、雷瓦拉和维鲁等北部地区。1220年瑞典远征莱内，但驻军很快被萨雷马人消灭，未形成长期统治。

宝剑骑士团一度在丹麦国内危机时占据北部。1236年骑士团在绍莱战役惨败，1237年残部并入条顿骑士团，成为利沃尼亚分团。1238年《斯滕斯比条约》恢复丹麦对北部爱沙尼亚的权利，同时确认骑士团和教会领的既成利益。

### 塔尔巴图与萨雷马的最后抵抗

1224年，十字军围攻并攻陷塔尔巴图城堡，罗斯援军与守军大部被杀；塔尔图主教领随后建立。1227年萨雷马岛主要堡垒被征服，古代县的独立政治结构至此基本瓦解。岛民此后仍多次起义，受洗也经历长期的制度化过程。

## 复合统治结构

“中世纪爱沙尼亚”不是单一国家。它属于更大的[利沃尼亚](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E6%AC%A7%E6%B4%B2/%E6%B3%A2%E7%BD%97%E7%9A%84%E6%B5%B7/%E5%88%A9%E6%B2%83%E5%B0%BC%E4%BA%9A.md)政治空间，各领主共同防御又彼此竞争。

| 权力中心 | 主要地区 | 权力与限制 |
| --- | --- | --- |
| 丹麦王与爱沙尼亚公国 | 以塔林、哈留、维鲁为中心，1219—1346年间断统治 | 国王通过总督、封臣和教会统治；距丹麦本土远，财政与军事依赖本地贵族。 |
| 利沃尼亚骑士团 | 中部、南部及1346年后的北部大片地区 | 掌城堡、军队、贡赋和封地；隶属条顿骑士团体系，但分团有较强自主性。 |
| 塔尔图主教领 | 东南部 | 主教兼世俗领主，以教会法、封臣和城堡治理。 |
| 萨雷—莱内主教领 | 西部和岛屿部分地区 | 领地分散，常受骑士团、丹麦和城市势力夹击。 |
| 塔林、塔尔图等城市 | 港口和贸易节点 | 依城市法、行会和市政会自治；加入汉萨网络，德语商人占优势。 |
| 波罗的德意志封臣与教士 | 各领地庄园和教会 | 逐步控制土地、司法和农民义务，构成跨领地等级精英。 |
| 爱沙尼亚农民与乡村共同体 | 广大农村 | 保留生产和部分村社惯例，但政治权利下降、劳役和人身依附逐步加重。 |

教会建立教区、修道院和什一税制度，拉丁基督教礼仪与本地习俗长期并存。德语成为城市、领主与上层行政语言，爱沙尼亚语则主要维持于乡村、口传和基层教化。

## 圣乔治之夜起义

1343年4月，哈留地区起义者袭击庄园和修道院，推举四名首领，并寻求瑞典方面支援。起义迅速扩及北部和萨雷马。利沃尼亚骑士团以谈判诱捕并杀死起义首领，再以军事行动镇压；萨雷马抵抗延续到1345年。

起义原因包含劳役和领主压力、丹麦统治薄弱、地方精英权力被侵蚀以及对外援的期待。它不是现代民族国家式独立战争，但体现本地社会反抗征服秩序。丹麦无力、也无意继续承担北部防务，1346年将爱沙尼亚公国售给条顿骑士团，后由利沃尼亚分团实际接管。结果是骑士团领土更集中，波罗的德意志封建控制加深。

## 经济、城市与农民地位

塔林位于俄国、芬兰湾与西欧海路之间，塔尔图连接普斯科夫—诺夫哥罗德内陆路线。谷物、蜡、皮毛、木材和盐等贸易使城市行会与商人富裕，城墙、教堂和市政制度发展。城市公民权主要掌握在德语商人和工匠手中，本地爱沙尼亚人多为劳工、小手工业者或郊区居民。

14—16世纪，西欧谷物需求和庄园经营扩张推动地主增加劳役。农民迁徙和诉讼权受到限制，农奴化逐渐加深；不同领地速度并不相同，也不存在一纸法令在全境同时建立农奴制。

## 宗教改革与旧秩序危机

1520年代路德宗思想经汉萨城市传播。塔林和塔尔图出现反圣像行动，市政会接管部分教会财产；城市与贵族逐步改宗，主教领和修道院的权威削弱。宗教改革推动以本地语言讲道和教理教育，却没有立即改善农民的政治地位。

旧利沃尼亚危机来自多重因素：

| 类型 | 因素 |
| --- | --- |
| 政治结构 | 骑士团、主教、城市和封臣没有统一财政、继承与最高指挥。 |
| 军事变化 | 中世纪骑士城堡体系难以负担火器、雇佣军和长期边境战争。 |
| 宗教分裂 | 路德宗城市与贵族侵蚀天主教主教和骑士团的合法性、财源。 |
| 社会压力 | 农民依附加重，统治者难以形成广泛动员。 |
| 外部竞争 | 莫斯科、瑞典、丹麦和波兰—立陶宛都试图控制港口和通道。 |
| 直接触发 | 1558年伊凡四世军队入侵，纳尔瓦和塔尔图等要地失守。 |

1561年，北部爱沙尼亚贵族和塔林向瑞典效忠；萨雷—莱内主教领已转入丹麦王弟马格努斯之手；骑士团末任团长哥特哈德·凯特勒向波兰—立陶宛臣服并世俗化南部领地。爱沙尼亚由此进入瑞典、丹麦、波兰—立陶宛和莫斯科争夺的时代。

| --- | --- | --- |
| 1208 | 南部战事扩大 | 十字军、拉特加利亚盟军与爱沙尼亚诸县进入长期战争。 |
| 1217 | 圣马太日战役 | 伦比图和考波战死，南部爱沙尼亚联盟受挫。 |

| 1236—1238 | 绍莱战败、骑士团重组与《斯滕斯比条约》 | 宝剑骑士团并入条顿体系，丹麦恢复北部属地。 |
| 1343—1345 | 圣乔治之夜起义 | 起义失败，地方自主进一步削弱。 |

| 1520年代 | 宗教改革传播 | 城市和贵族转向路德宗，旧教会领权威下降。 |
| 1558 | 利沃尼亚战争爆发 | 莫斯科军夺取东部重镇，复合政体军事崩溃。 |
| 1561 | 利沃尼亚旧秩序瓦解 | 北部归瑞典，南部与岛屿分属不同列强。 |
