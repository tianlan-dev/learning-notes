# 利沃尼亚瓦解与瑞典、俄罗斯帝国统治

## 时间

1561—1918年

## 概括

利沃尼亚战争瓦解中世纪复合政体后，今日爱沙尼亚先由瑞典、波兰—立陶宛和丹麦分割。瑞典在17世纪逐步统一北部、南部和萨雷马岛，推行路德宗教会、司法、教育与王室土地“收归”改革，但庄园和农奴制仍支配农村。大北方战争、瘟疫和饥荒摧毁人口与财政，1710年各地向俄军投降，1721年正式并入俄罗斯帝国。俄帝保留波罗的德意志贵族和城市特权，以其治理地方；19世纪农奴制废除、学校、报刊、社团、城市化和民族运动逐步把爱沙尼亚语农民转化为现代政治民族。第一次世界大战和1917年革命最终击穿帝国统治。

## 利沃尼亚瓦解与列强分割

1558年莫斯科军队入侵，纳尔瓦、塔尔图等地失守。利沃尼亚骑士团、主教领和城市不能形成统一财政与指挥，只能分别寻求保护。1561年，北部爱沙尼亚贵族和塔林向瑞典国王效忠；南部原塔尔图主教区和利沃尼亚地区主要转入波兰—立陶宛；萨雷马及西部部分地区受丹麦王弟马格努斯控制。

战争持续到1580年代，乡村人口、庄园和贸易遭严重破坏。17世纪初瑞典与波兰—立陶宛再战，1629年《阿尔特马克停战协定》使瑞典取得利沃尼亚大部，今日爱沙尼亚南部由此纳入瑞典。1645年《布勒姆瑟布鲁条约》又使丹麦让出萨雷马岛。瑞典王权才基本统一今日爱沙尼亚范围，但仍分别设爱沙尼亚公国、瑞属利沃尼亚等行政单位。

## 瑞典统治

### 权力结构

| 层级 | 机构 / 群体 | 作用与限制 |
| --- | --- | --- |
| 瑞典国王与中央机关 | 王室、总督和中央法令 | 掌战争、外交、税收与最高任命；需要地方等级配合。 |
| 爱沙尼亚与利沃尼亚总督体系 | 总督、省级官署和法院 | 执行王令、征税、司法和军事征集；两省制度不完全相同。 |
| 波罗的德意志骑士团体 | 地主议会、封臣和庄园主 | 控制地方土地、司法和农民劳役，对王权改革具有议价能力。 |
| 路德宗教会 | 主教、牧师、教区学校 | 推行教理、婚姻登记和基层教育，逐渐使用爱沙尼亚语。 |
| 塔林、塔尔图等城市 | 市政会、行会、港口 | 保有城市法和商业特权，但受王室财政与战争政策约束。 |
| 爱沙尼亚农民 | 庄园辖下村社和佃户 | 承担地租、劳役和征兵，政治权利很少。 |

瑞典统治并非取消波罗的德意志等级，而是试图把其纳入更有秩序的王国行政。司法编纂、教会巡视和官员监督提高王权存在；地方贵族仍通过等级议会、庄园法庭和人身依附控制农村。

### 教育、教会与王室土地收归

古斯塔夫二世·阿道夫于1632年创办塔尔图大学，培养牧师、官员和学者。17世纪后期，本特·戈特弗里德·福塞柳斯推动乡村学校和教师训练，爱沙尼亚语教理与识字传播。改革目的主要是建立合格路德宗信徒和行政秩序，却为后来民族语言公共文化积累了条件。

查理十一世时期，王室把大量此前授予贵族的土地收回，称为“土地收归”。王室希望增加财政、限制贵族并减轻部分过度劳役；贵族认为传统财产权和自治受到侵犯。后世“美好的瑞典时代”记忆由这种相对于后期庄园压迫的比较形成，但瑞典时期并未废除农奴制，农民仍缺乏政治代表。

## 灾难与俄国征服

1695—1697年连续歉收造成大饥荒，饥饿和疾病使人口锐减。1700年大北方战争爆发，瑞典最初在纳尔瓦击败俄军，但彼得一世重建军队后逐步占领东波罗的海。塔尔图于1704年陷落，居民遭迁徙；1710年瘟疫与围城使塔林、派尔努及地方等级先后投降。

投降协定保障路德宗信仰、城市法、波罗的德意志贵族土地和等级特权，以换取向俄国沙皇效忠。1721年《尼斯塔德条约》正式确认瑞典把爱沙尼亚和利沃尼亚等地让给俄罗斯。瑞典统治衰落的结构因素是王国同时在多条战线作战、东波罗的海防御线漫长、地方人口和财政已受饥荒摧毁；俄军围城和1710年瘟疫是直接崩溃机制。

## 俄罗斯帝国的波罗的地方秩序

俄帝没有立即以俄罗斯官僚全面取代地方精英，而是通过“波罗的特殊秩序”统治。爱沙尼亚省和利沃尼亚省由总督、骑士团体、城市市政会与庄园法庭共同运作；德语长期是行政、司法和高等教育语言，路德宗仍占主导。贵族向帝国提供军官和官员，沙皇则确认其土地与农民控制权。

| 权力中心 | 实际职能 | 历史后果 |
| --- | --- | --- |
| 皇帝与帝国中央 | 任命总督、掌军队外交、审核地方制度 | 主权属于俄帝，地方自治可被中央收回。 |
| 爱沙尼亚省、利沃尼亚省官署 | 税收、治安、司法和征兵 | 今日爱沙尼亚南北继续处于不同省制。 |
| 波罗的德意志骑士团体 | 地方等级议会、庄园和基层司法 | 维持德语精英与农民社会之间的巨大权力差距。 |
| 城市商人和工匠 | 港口、制造业、行会和市政 | 19世纪铁路与工业化后城市社会日益多族化。 |
| 路德宗教会与学校 | 教区管理、识字和道德纪律 | 在精英统治同时保存、标准化爱沙尼亚语。 |

18世纪庄园经济扩大，谷物和烈酒生产增加，农民劳役、人身限制和惩戒权达到高点。启蒙知识分子、教士和帝国改革者开始批评农奴制，但改革首先服务于提高生产和行政效率。

## 农奴制废除与社会转型

亚历山大一世时期，爱沙尼亚省于1816年、利沃尼亚省于1819年废除农奴身份，早于俄帝腹地。农民获得人身自由，却没有普遍无偿取得耕地；土地仍主要掌握在庄园主手中，租佃、劳役契约和迁徙条件使依附延续。19世纪中期后，货币租、购买农庄和帝国进一步改革才扩大自耕农层。

社会转型包括：

- 农民获得姓氏、契约和更多迁徙可能，法律主体地位逐渐增强；
- 教区学校和识字率上升，爱沙尼亚语读物、历书和报刊扩大；
- 铁路、港口和工厂吸引农村人口进入塔林、纳尔瓦和塔尔图；
- 波罗的德意志庄园主仍掌握大量土地和地方职位，引发土地与民族矛盾；
- 正教改宗运动、移民和多语城市使宗教与身份关系更复杂。

## 民族觉醒

19世纪中叶，弗里德里希·克罗伊茨瓦尔德整理史诗《卡列维波埃格》，约翰·沃尔德马尔·扬森创办报刊并推动社团，卡尔·罗伯特·雅各布松强调农民经济和政治权利。1869年塔尔图首届全爱沙尼亚歌咏节把教区合唱传统转化为跨地区公共仪式；“爱沙尼亚人”逐步从被外部称呼的农民类别成为自我认同。

民族运动并非单线统一。温和派重视教育、文化和与地方精英合作，激进派要求土地、平等和政治权利；城市工人又受到社会主义影响。1880年代内部路线分裂、帝国审查和经济压力一度削弱组织，但报刊、学生社团、农业协会和歌咏节已形成持续网络。

## 俄化、1905年革命与1917年自治

亚历山大三世时期，帝国在1880—1890年代取消或限制部分波罗的等级特例，俄语进入行政、法院和中高等教育，塔尔图大学也被俄语化。俄化削弱波罗的德意志垄断，却不是为爱沙尼亚自治服务；它同时压缩爱沙尼亚语高等公共空间。民族运动因此既反对德意志庄园优势，也反对帝国中央化。

1905年革命中，工人罢工、农民会议和民族政治集会扩大。部分农村庄园遭焚毁，帝国以军事法庭、处决和鞭刑镇压。革命失败，却促成政党、合作社、自治纲领和议会经验发展。

1917年二月革命推翻沙皇。俄国临时政府把爱沙尼亚省与利沃尼亚省北部爱沙尼亚语地区合并为自治省，并设民选临时省议会“马佩夫”。布尔什维克夺权后，马佩夫于1917年11月宣布自己是召集制宪会议前的最高权力，随即被解散并转入地下。1918年2月德军推进造成短暂权力真空，救国委员会借此宣布共和国。
