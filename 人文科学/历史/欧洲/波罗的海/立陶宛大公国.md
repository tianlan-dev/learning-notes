# 立陶宛大公国

## 时间

约13世纪中叶—1795年

## 概括

立陶宛大公国由波罗的海东南部的立陶宛诸部在条顿骑士团、利沃尼亚骑士团和鲁塞尼亚诸公国压力下整合而成。明道加斯于1253年加冕为王，但其死后王号未能延续；格迪米纳斯、阿尔吉尔达斯和维陶塔斯时期，大公国通过战争、婚姻和地方公臣服扩展到今天的白俄罗斯、乌克兰和俄罗斯西部，成为以立陶宛统治家族为核心、人口和书写文化以东斯拉夫—鲁塞尼亚为主的多族群国家。1385年后它与波兰形成共同君主关系，1569年卢布林联合后成为波兰—立陶宛联邦的组成国，仍保有自己的法律、财政、军队和官职，直至1795年第三次瓜分灭亡。

## 建立背景

### 外部压力促成联合

12—13世纪，立陶宛地区由多个公和部族首领控制。德意志骑士团在普鲁士、利沃尼亚宝剑骑士团在北方发动十字军，试图征服仍保持本土宗教的波罗人。蒙古入侵又破坏基辅罗斯旧秩序，使东部鲁塞尼亚诸公国需要新的保护者。立陶宛统治者由此同时面对西方军事修会与东方政治真空。

1219年立陶宛诸公与加利西亚—沃里尼亚签约，文献列出二十余名公，明道加斯已居重要地位。此后他通过兼并、婚姻、流放宗亲和战争成为最高统治者。其统一不是建立一套成熟官僚制，而是把宗亲封地、部族军队和贡赋关系纳入个人网络。

### 王冠与宗教选择

明道加斯为削弱国内对手、取得利沃尼亚骑士团支持，于1251年受洗，1253年由教皇授权加冕为国王。作为交换，他承诺向骑士团让渡部分土地。王冠使立陶宛首次获得拉丁基督教世界承认，却没有使全国立即基督教化；地方贵族担心领土让渡，萨莫吉希亚人继续抵抗骑士团。1260年杜尔贝战役中骑士团失败后，明道加斯重新靠近反骑士团阵营。1263年他被杀，王国称号与稳定继承一同中断。

## 国家重组与格迪米纳斯王朝

### 1263—1316年：危机后恢复

特雷尼奥塔、瓦伊什维尔卡斯和什瓦尔恩相继上台，王位通过暗杀、宗亲联盟和外来支持更替。特赖德尼斯恢复中央权威并抵抗利沃尼亚骑士团。13世纪末布蒂盖迪斯、布特维达斯和维特尼斯建立较连续的家族统治，沿涅曼河修筑或强化堡垒，与里加商人和大主教结盟，在基督教列强之间寻求空间。

这一阶段的关键不是“异教国家拒绝一切欧洲联系”，而是统治者反复利用受洗承诺、贸易特权和外交联盟，同时避免让军事修会以传教名义取得领土控制。

### 格迪米纳斯的扩张机制

格迪米纳斯于1316年继位，以维尔纽斯为中心。他向教皇、汉萨城市和修会写信，邀请工匠、商人和教士进入大公国；虽讨论受洗，他最终没有正式皈依。向东扩张主要依靠：

1. 把格迪米纳斯家族成员与鲁塞尼亚王公联姻；
2. 保留当地东正教、法律和精英，只要求贡赋、军役与政治忠诚；
3. 利用金帐汗国影响减弱和鲁塞尼亚诸公之间竞争；
4. 让宗亲治理重要城市，形成大公家族的分层封地网络。

这种“少改地方制度、先建立宗主权”的方式，使人口规模远小于鲁塞尼亚人的立陶宛统治家族能够管理广阔领地。

## 阿尔吉尔达斯—凯斯图蒂斯双核心

1345年，阿尔吉尔达斯和凯斯图蒂斯推翻弟弟尧努蒂斯，形成事实分工：阿尔吉尔达斯从维尔纽斯主导东部扩张，凯斯图蒂斯从特拉凯防御条顿骑士团。两人并非现代意义上的平等共治者，但长期合作避免了宗亲分裂。

### 向东与向南

- 1362年前后的蓝水之战击败金帐汗国势力，立陶宛控制基辅、波多利亚等地；
- 大公国与莫斯科争夺鲁塞尼亚政治中心，阿尔吉尔达斯三次进攻莫斯科但未能摧毁其崛起；
- 鲁塞尼亚语成为大公国文书语言，东正教贵族在军政体系中占重要位置；
- 黑海方向的控制多为宗主和要塞网络，不能等同现代国界内的均匀行政。

### 西部生存战

条顿骑士团从普鲁士、利沃尼亚骑士团从北方夹击，长期袭击萨莫吉希亚和立陶宛核心。凯斯图蒂斯利用谈判、短期受洗承诺和反击维持防线。骑士团的军事压力促使大公国寻找一个能提供王冠、正式受洗和战略纵深的联盟，这成为与波兰联合的深层背景。

## 约盖拉、基督教化与波兰联合

### 内战与克雷沃安排

1377年阿尔吉尔达斯死后，其子约盖拉继位，引发与叔父凯斯图蒂斯及堂兄维陶塔斯的冲突。凯斯图蒂斯一度夺位，1382年被拘押后死去；维陶塔斯逃往条顿骑士团控制区。约盖拉既要应对宗亲战争，也要选择与莫斯科或波兰结盟。

1385年克雷沃文件规定约盖拉迎娶波兰女王雅德维加、受洗并成为波兰国王，同时推动立陶宛基督教化。1386年约盖拉在克拉科夫加冕为瓦迪斯瓦夫二世，1387年在立陶宛核心建立教区、教堂和天主教贵族特权。

### 基督教化的范围与后果

- 它主要从立陶宛语核心区和统治精英开始；东部鲁塞尼亚居民早已信奉东正教；
- 萨莫吉希亚因仍在与骑士团争夺，正式基督教化较晚，到1413—1417年才系统建立教会；
- 天主教贵族最初获得较优政治待遇，东正教贵族随后通过特权逐渐缩小差距；
- 骑士团失去“征服异教徒”的主要合法化理由，却仍以萨莫吉希亚和条约争端继续战争。

## 维陶塔斯大帝时期

### 权力形成

1392年奥斯特鲁夫协议结束约盖拉与维陶塔斯的主要内战：维陶塔斯取得立陶宛实际统治，名义承认约盖拉的最高地位；1401年联合文件确认其大公身份。他削弱地方格迪米纳斯宗亲公，把重要领地交给可撤换的总督，使大公权力更集中。

### 扩张、战争与制度

- 1398年萨利纳斯条约暂把萨莫吉希亚让给骑士团，以换取东征空间；
- 1399年沃尔斯克拉河战役惨败于金帐汗国力量，限制向黑海和草原扩张；
- 1409年萨莫吉希亚起义引发波兰—立陶宛与条顿骑士团大战；
- 1410年格伦瓦尔德 / 坦能堡战役中，约盖拉与维陶塔斯联合军击败骑士团主力；
- 1422年梅尔诺和约最终把萨莫吉希亚大部留给立陶宛，稳定西部边界；
- 1413年霍罗德沃联合进一步制度化波兰—立陶宛贵族联系，同时保留两个政治实体；
- 1430年维陶塔斯筹备加冕为立陶宛国王，因王冠运输、波兰反对和外交冲突未成，去世后计划终止。

### 鼎盛条件与局限

| 鼎盛条件 | 局限 |
| --- | --- |
| 波兰联盟提供西部军事伙伴 | 联盟中的主权层级始终有争议。 |
| 鲁塞尼亚地方精英与宗教制度大体保留 | 广阔领地行政密度低，宗亲与贵族自主性强。 |
| 大公削弱世袭地方公，改派总督 | 集权依赖统治者个人，继承后易反复。 |
| 蒙古诸汗国分裂提供南扩机会 | 1399年惨败说明草原军事边界难以突破。 |
| 条顿骑士团在1410年遭重创 | 骑士团未立即灭亡，战争和外交仍延续。 |

## 1430—1569年：贵族国家与莫斯科压力

### 继承战争

维陶塔斯无子，什维特里盖拉试图保持大公国独立并依靠东正教鲁塞尼亚贵族；1432年西部贵族推举西吉斯蒙德·凯斯图泰蒂斯。双方内战在1435年帕巴伊斯卡斯战役后大体结束。1440年西吉斯蒙德被杀，立陶宛贵族推举约盖拉之子卡齐米日，未先征得波兰同意，显示大公国仍能独立选择统治者。1447年卡齐米日兼任波兰国王后，共同君主关系恢复。

### 制度成熟

| 机构 / 制度 | 功能 | 演变 |
| --- | --- | --- |
| 大公会议 | 高级贵族和主教参与外交、任命和继承 | 15世纪后权力上升，成为贵族政治核心。 |
| 地方总督、长老 | 管理旧公国、城堡和司法 | 逐步取代半独立宗亲公。 |
| 立陶宛军队 | 贵族骑兵、领地征召与雇佣兵 | 与波兰军队分立，战时协同。 |
| 国库与官职 | 大法官、司库、统帅等 | 形成独立于波兰的立陶宛官僚职位。 |
| 《立陶宛法典》 | 1529、1566、1588年三版法典 | 以鲁塞尼亚文书传统成文，1588年版在联邦成立后仍维持大公国法律独立。 |

### 莫斯科—立陶宛战争

莫斯科以“聚集罗斯诸地”和东正教保护者自居，持续争夺上奥卡、斯摩棱斯克和第聂伯河流域。1490年代以后，大公国多次失地；1514年虽在奥尔沙战役取胜，斯摩棱斯克仍归莫斯科。16世纪中叶利沃尼亚战争又使立陶宛直接面对伊凡四世军队，国防成本和兵源压力迫使贵族寻求与波兰更紧密的共同体。

## 1569年卢布林联合

### 形成过程

西吉斯蒙德二世无子，雅盖隆王朝即将绝嗣；立陶宛需要波兰军事和财政支持，波兰贵族则要求更完整的政治联合。立陶宛代表担心失去独立地位，一度退出卢布林会议。国王随后把波德拉谢、沃里尼亚、基辅和布拉茨拉夫等大公国南部领地划入波兰王冠领，迫使立陶宛贵族返回谈判。

1569年联合建立一个共同选举君主和共同议会的联邦，同时保留：

- 立陶宛大公国名称和领土；
- 独立法律体系与1588年法典；
- 自己的军队、国库、中央官职和地方行政；
- 与波兰王冠领有区别的贵族政治共同体。

因此，1569年不是立陶宛“大公国立即灭亡”，而是从王朝联合转为复合联邦中的组成国。

## 联邦时期的立陶宛大公国

### 政治与社会

鲁塞尼亚贵族逐渐采用波兰语和波兰贵族文化，1596年布列斯特联合又建立与罗马共融的鲁塞尼亚礼教会。波兰化主要发生于精英文化和公共政治，不等同所有居民改为波兰族；立陶宛语农民、鲁塞尼亚语人口、犹太社群、德意志城市居民和鞑靼军役社群长期共存。

### 17世纪战争与破坏

1648年赫梅利尼茨基起义、俄波战争和瑞典入侵把联邦拖入长期危机。1655年莫斯科军攻占维尔纽斯，发生屠杀、火灾和掠夺，大公宫殿严重毁坏；沙皇阿列克谢自称立陶宛大公，但占领未获本国宪制承认。1661年联邦军收复维尔纽斯，战争仍造成人口、贸易和税源重大损失。

北方战争期间，萨皮耶哈等立陶宛大贵族集团内斗，瑞典、俄国与萨克森军队反复进入。1717年以后俄罗斯以“调停者”身份影响联邦政治，军力和财政改革受到限制。

## 衰落、改革与灭亡

### 结构性困境

| 类型 | 因素 |
| --- | --- |
| 宪制 | 自由否决权使单个议员可破坏议会决定；邦联和贵族私兵使中央执行困难。 |
| 社会 | 大贵族拥有庞大领地和客户网络，农民农奴负担加重，城市和王室财政较弱。 |
| 军事财政 | 常备军规模远小于俄、普、奥，税收改革受既得利益阻碍。 |
| 外部压力 | 俄罗斯把联邦视为缓冲和保护国，普鲁士、奥地利也利用内部分裂。 |
| 认同与代表 | 贵族“政治民族”享有自由，但绝大多数农民和城镇居民缺乏政治参与。 |

### 改革尝试

1773年成立国家教育委员会，常被视为欧洲较早的国家教育管理机构。四年议会推动扩军、税制和城市权利改革，1791年五三宪法试图取消自由否决、建立较有效的世袭君主制和政府责任。立陶宛代表随后通过《两国相互保证》，强调波兰与立陶宛在共同机构中的对等。

保守贵族组成塔戈维查联盟并请求俄罗斯干预。1792年战争失败后，俄普实施第二次瓜分；1794年科希丘什科起义在维尔纽斯等地爆发，雅库布·雅辛斯基领导立陶宛起义，但被俄军镇压。

### 直接灭亡

- 1772年第一次瓜分削去联邦边疆；
- 1793年第二次瓜分使国家失去大部分领土和战略能力；
- 1794年起义失败摧毁最后军事抵抗；
- 1795年俄、普、奥第三次瓜分，斯坦尼斯瓦夫·奥古斯特退位，大公国国家机构被占领者拆分；
- 立陶宛大部分领土并入俄罗斯帝国，西部小部分归普鲁士。

大公国灭亡不能归因于单一“贵族自由”或单一外国阴谋，而是宪制执行困难、军事财政不足、社会排斥与周边绝对主义强国干预相互强化，改革启动又直接触发既得利益与俄国的军事反击。

| 1655 | 维尔纽斯被占 | 莫斯科军掠夺首都并作竞争性大公宣称 | 人口经济重创，宫殿不再恢复。 |
| 1791 | 五三宪法与相互保证 | 改革中央制度并确认两国对等 | 现代化尝试触发内外反改革联盟。 |
| 1794—1795 | 起义失败与第三次瓜分 | 俄普奥镇压后彻底分割 | 大公国灭亡，立陶宛进入帝国统治。 |
