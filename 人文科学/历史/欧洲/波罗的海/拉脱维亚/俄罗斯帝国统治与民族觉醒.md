# 俄罗斯帝国统治与民族觉醒

## 时间

里加、维泽梅自1710/1721年起；拉特加尔自1772年起；库尔兰自1795年起；至1918年共和国成立

## 概括

俄罗斯不是一次兼并“整个拉脱维亚”。大北方战争后先取得里加和维泽梅，第一次瓜分波兰时取得拉特加尔，第三次瓜分才并入库尔兰。帝国把这些地区分别置于利沃尼亚省、库尔兰省和维捷布斯克省，继续利用波罗的德意志贵族、路德宗教会和城市制度；拉特加尔则处于不同的法律、宗教和行政环境。19世纪农奴制废除、商品农业、港口工业化、学校和报刊扩大了社会流动，青年拉脱维亚人由此把分散方言区想象为共同民族。俄化、阶级冲突和土地不平等又推动新思潮、社会民主与1905年革命。第一次世界大战最终摧毁帝国框架，为独立建国创造机会，也留下难民、军队分裂和多重占领。

## 分期并入与行政结构

| 地区 | 并入时间与方式 | 帝国行政归属 | 延续的地方结构 |
| --- | --- | --- | --- |
| 里加、维泽梅 | 1710年投降，1721年和约确认 | 利沃尼亚省 | 德语贵族等级、路德宗教会、城市市法和庄园司法得到相当程度确认。 |
| 拉特加尔 | 1772年第一次瓜分波兰 | 先后属波洛茨克、白俄罗斯等行政单位，后主要属维捷布斯克省 | 天主教、波兰化贵族与东部边疆社会延续；与波罗的海省法律不同。 |
| 库尔兰、瑟米加利亚 | 1795年公国臣服并被兼并 | 库尔兰省 | 公爵被取消，贵族议会、庄园特权和德语行政传统大量保留。 |
| 里加港与城市圈 | 随利沃尼亚省，但战略地位特殊 | 总督、省机关、市政与帝国军事机构交叠 | 德语商人优势逐渐被俄语官僚、犹太商人和拉脱维亚工商业人口共同改变。 |

所谓“波罗的海自治”主要是等级与地区特权，并非现代民主自治。受益者首先是贵族、城市精英和路德宗教会；拉脱维亚农民没有与之相等的政治代表权。帝国中央通过确认地方特权换取忠诚，19世纪后期又以俄化和统一行政逐步削弱这些特权。

## 农奴制、土地与农村社会

库尔兰1817年、利沃尼亚1819年废除农奴制，早于俄罗斯帝国1861年改革。农民获得人身自由，却通常没有同步无偿取得耕地，庄园主仍掌握大部分土地和地方权力。租佃、劳役过渡、迁徙限制和购地成本使改革初期生活未必改善。

19世纪中叶后，法律逐渐允许农民购买土地，商品农业、乳业和亚麻生产发展。出现自耕农、佃农、无地劳工和外出工人等分化。拉特加尔因行政制度、人口压力与土地碎片化，改革路径不同，贫困和季节迁移较突出。

农村社会变化带来三种长期后果：

1. 识字、教区学校和路德宗虔信运动培养能读写的农民社群；
2. 购地自耕农成为民族文化和后来共和国土地秩序的重要基础；
3. 大庄园与无地人口并存，使土地问题成为1905年革命和1918年后改革的核心。

## 城市化与工业化

里加凭借道加瓦港、铁路、帝国市场和技术人才，在19世纪后期成为俄罗斯帝国主要工业城市之一。机械、铁路车辆、电工、橡胶、木材和食品工业吸纳大量农村移民；利耶帕亚、文茨皮尔斯发展为海军、铁路与出口港。

| 变化 | 推动力 | 社会后果 |
| --- | --- | --- |
| 铁路与港口 | 连接俄罗斯腹地、里加湾和不冻港 | 谷物、木材与工业品流量扩大，城市战略价值上升。 |
| 大型工厂 | 帝国关税市场、外资和技术 | 拉脱维亚工人阶级扩大，俄语、德语、意第绪语等多语社群共居。 |
| 城市人口增长 | 农村剩余劳动力与教育机会 | 拉脱维亚人逐步进入商业、专业和市政领域，挑战旧德语精英垄断。 |
| 报刊与社团 | 印刷、铁路邮政和识字率 | 文化运动可跨地区组织，政治思想传播加快。 |

工业化没有自动消除民族不平等。工厂纪律、住房拥挤、低工资和缺少合法工会使社会民主主义迅速传播；帝国官僚俄化与波罗的德意志经济优势又形成双重政治压力。

## 民族觉醒

### 青年拉脱维亚人

19世纪中叶，受教育的拉脱维亚知识分子开始反对把拉脱维亚语仅视为农民方言。他们整理词汇、民歌和历史，创办报刊，鼓励航海、商业、教育和购地。克里斯亚尼斯·瓦尔德马斯、尤里斯·阿卢南斯、阿蒂斯·克龙瓦尔兹、克里斯亚尼斯·巴龙斯等人的活动彼此不同，但共同推动“一个民族、一个语言文化空间、一个历史家园”的想象。

运动既反对波罗的德意志等级垄断，也一度把俄国中央视为可能的制衡者；随着俄化加深，这种策略局限显现。民族觉醒不是从一开始就要求独立国家，早期更多争取语言、教育、经济和社会平等。

### 社团、歌咏节与民俗

1868年里加拉脱维亚协会成立，组织讲座、戏剧、出版和科学活动。1873年首届全国歌咏节把地方合唱团汇集到里加，形成可重复的大规模民族仪式。民歌收集、文学、戏剧和节庆把不同方言区的文化材料编入共同传统。

这些活动既建立民族认同，也反映城乡和阶层差异。拉特加尔的天主教、书写传统与方言长期受到不同限制；“统一民族文化”需要协调，而非自然存在。

## 俄化、新思潮与政治现代化

1880年代后，帝国强化俄语行政和教育，削弱德语大学、法院和学校地位。俄化在某些领域打破波罗的德意志垄断，却没有给予拉脱维亚语平等地位。教师、学生和公务员必须使用俄语，本地文化组织受到审查。

1890年代“新思潮”把马克思主义、女性解放、现实主义和欧洲社会理论引入拉脱维亚公共领域。赖尼斯、阿斯帕齐娅等知识分子及报刊网络把民族问题与工人、农民和民主权利联系起来。1897年镇压迫使许多活动者入狱或流亡，反而推动地下社会民主组织发展。

## 1905年革命

### 结构原因

- 沙皇专制缺少合法政治参与；
- 大庄园与无地、少地农民矛盾尖锐；
- 里加工人劳动条件恶劣；
- 俄化、德意志等级特权和民族不平等叠加；
- 日俄战争失败削弱帝国权威；
- 社会民主网络能够组织罢工和宣传。

### 过程

1905年1月里加工人示威遭军警射击，罢工迅速扩大。春夏农村出现集会、教会冲突和拒绝服从庄园权威；秋季总罢工与帝国危机使地方执行委员会、武装小组和农民会议一度掌权。部分庄园被焚毁，革命者袭击警察、官员与地主，暴力也波及个人复仇和族群紧张。

年底沙皇军队恢复控制。惩罚远征未经充分审判处决、鞭打、焚毁房屋，大量参与者被监禁、流放或逃往西欧、美国。革命失败，却产生长期影响：政治组织获得战斗经验，土地和民主成为不可回避的问题，民族运动分化为自由派、社会民主派和保守农业派。

## 第一次世界大战

### 德军推进与难民

1915年德军占领库尔兰并逼近里加。俄方撤迁工厂、设备和劳动力，数十万居民逃往俄罗斯内地；农村、港口和工业遭严重破坏。前线把拉脱维亚历史地区再次分割：库尔兰在德军占领下，里加和维泽梅一度仍属俄军后方，拉特加尔则承受军需与难民压力。

### 拉脱维亚步枪兵

俄国于1915年同意组建拉脱维亚步枪营，利用本地动员保卫里加。部队在死亡岛、圣诞战役等作战中伤亡惨重。对俄军指挥、补给和战果的不满，加上1917年革命，使相当部分步枪兵支持布尔什维克；另一些人后来加入反布尔什维克军队或拉脱维亚共和国军。不能把全部步枪兵视作同一政治阵营。

## 1917—1918年的权力竞争

| 组织 / 政权 | 主要基础 | 目标与行动 |
| --- | --- | --- |
| 拉脱维亚临时民族委员会 | 难民组织、地方议会和民族政治人物 | 主张拉脱维亚各区统一、自决，逐步走向完整独立；开展对外外交。 |
| 民主集团 | 德占里加的政党与公共人物 | 在占领条件下协调非布尔什维克政治力量，后来参与人民委员会。 |
| 伊斯科拉特 | 工兵苏维埃与布尔什维克 | 在未被德军占领地区建立苏维埃权力，反对民族共和国方案。 |
| 拉特加尔大会 | 拉特加尔代表 | 1917年支持从维捷布斯克省分出并与其他拉脱维亚地区统一，但内部仍有自治与政治路线争论。 |
| 德国军政与联合波罗的公国计划 | 占领军、部分波罗的德意志贵族 | 试图在德国保护下建立君主制复合国家，缺乏广泛本地授权。 |

1917年俄国二月革命打开组织空间，十月革命又造成权力分裂。1918年《布列斯特—立托夫斯克条约》使德国控制扩大；德国战败和十一月革命随即摧毁占领体系。拉脱维亚临时民族委员会与民主集团的力量在人民委员会中汇合，于11月18日宣布共和国。

## 帝国统治终结的原因

### 结构因素

- 分省统治无法阻止语言、教育和市场形成跨区民族空间；
- 等级特权和土地不平等持续制造社会反对；
- 俄化既削弱地方自治，也疏远本地知识精英；
- 工业化产生有组织的工人政治；
- 帝国对多民族自决诉求缺少可接受的宪制方案。

### 外部与直接因素

第一次世界大战造成军事失败、财政崩溃、难民和行政瓦解。1917年两次革命消灭沙皇并使中央权力碎片化；1918年德国先占领、后战败撤退，形成短暂权力真空。民族组织能在这一窗口把文化共同体转化为国家主张。

## 重要事件

| 时间 | 事件 | 结果与长期影响 |
| --- | --- | --- |
| 1710/1721 | 里加、维泽梅转入俄国 | 俄罗斯取得东波罗的海门户并确认地方贵族特权。 |
| 1772 | 拉特加尔被并入 | 东部与波罗的海省继续分治。 |
| 1795 | 库尔兰公国灭亡 | 现代拉脱维亚各区均受俄国统治。 |
| 1817、1819 | 废除农奴制 | 人身自由扩大，土地问题保留。 |
| 1850—1860年代 | 青年拉脱维亚人兴起 | 语言文化运动进入现代公共领域。 |
| 1868 | 里加拉脱维亚协会成立 | 社团化民族文化获得中心。 |
| 1873 | 首届歌咏节 | 跨地区民族仪式形成。 |
| 1880年代 | 俄化加强 | 德语特权受限，拉脱维亚语仍未获平等。 |
| 1890年代 | 新思潮 | 社会民主、女性解放和现实主义扩散。 |
| 1905 | 革命与惩罚远征 | 群众政治成熟，镇压促成流亡和激进化。 |
| 1915 | 德军占领库尔兰、步枪营成立 | 难民、工业撤迁和军事民族组织并存。 |
| 1916—1917 | 圣诞战役 | 高伤亡削弱对俄军指挥的信任。 |
| 1917 | 拉特加尔大会、民族委员会和伊斯科拉特并立 | 统一、自决与革命路线公开竞争。 |
| 1918-02 | 德国控制扩大 | 俄罗斯帝国统治事实上终止。 |
| 1918-11 | 德国战败、共和国宣布 | 帝国与占领双重崩溃转化为建国机会。 |
