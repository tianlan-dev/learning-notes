# 塞尔维亚王国与南斯拉夫王国时期

## 时间

1912—1941年

## 概括

1912年第一次巴尔干战争结束了奥斯曼在科索沃的统治。今日科索沃大部被塞尔维亚王国控制，西部部分地区由黑山王国取得；1918年两部分一同进入塞尔维亚人、克罗地亚人和斯洛文尼亚人王国，1929年后称南斯拉夫王国。新国家把科索沃视为中世纪国家核心和南进战略通道，多数阿尔巴尼亚居民则把1912年视为外来征服。军事占领、土地改革、殖民安置、卡查克武装抵抗、移民和教育语言政策使双方政治共同体进一步分离。

## 1912年的军事占领

塞尔维亚第一集团军在库马诺沃击败奥斯曼主力后进入普里什蒂纳及科索沃盆地，黑山军从西北进入佩奇和贾科维察方向。奥斯曼军撤退与地方武装抵抗交织，塞尔维亚和黑山部队对被视为反抗者的村庄实施报复。外交记者、救援人员和后来的调查记录了处决、焚村和人口逃亡，具体数字因资料来源与统计口径而异。

塞尔维亚政府以恢复历史领土、取得通往亚得里亚海方向战略通道和保护塞族宗教遗产为正当性；阿尔巴尼亚政治组织则主张科索沃居民参与1912年反奥斯曼起义，不应被排除在新阿尔巴尼亚国家之外。1913年列强承认阿尔巴尼亚独立，却将其边界划在科索沃以西，使阿尔巴尼亚人口大量留在塞尔维亚与黑山境内。

## 并入与早期治理

新统治首先是军事性的。塞尔维亚与黑山分别建立县、区和警察体系，解除地方武装，征收税款并要求效忠。伊斯兰宗教机构、土地契约和奥斯曼法制与新国家法律之间产生大量权属冲突。部分穆斯林地主离开，另一些人与新行政合作；农民则常在所有权、佃租和战时破坏中失去保障。

1913年科索沃并未立即享有与战前塞尔维亚本土完全相同的宪法地位。边疆治安逻辑、警察行动和对阿尔巴尼亚政治组织的限制长期存在。学校以塞尔维亚语为主，阿尔巴尼亚语公共教育空间极小，宗教学校难以替代现代母语教育。

## 第一次世界大战

1914年战争爆发后，科索沃成为塞尔维亚军队补给和退却通道。1915年秋中央同盟攻破塞尔维亚，塞尔维亚军政人员与大量平民经科索沃、黑山和阿尔巴尼亚山地向亚得里亚海撤退，途中因严寒、饥饿、疾病和袭击伤亡惨重。

占领区大体由奥匈帝国和保加利亚分管：

| 占领力量 | 主要范围与政策 | 社会反应 |
|---|---|---|
| 奥匈帝国 | 西部及西北部较多地区；利用阿尔巴尼亚语学校和地方精英削弱塞尔维亚影响 | 一部分阿尔巴尼亚居民把母语教育与行政机会视为改善，塞尔维亚居民则承受占领与拘押。 |
| 保加利亚 | 东部及东南交通区；推行保加利亚化、征用和治安控制 | 塞尔维亚官员、教士和教育人员受压，农村承担粮食与劳役征集。 |
| 地方武装 | 边界和山区出现亲阿尔巴尼亚、亲塞尔维亚及机会主义团体 | 战争旧怨、土地争议和报复循环强化。 |

1918年协约国与塞尔维亚军队北进，中央同盟撤退。统治再次转换，战时合作嫌疑与报复为战后冲突埋下伏笔。

## 进入南斯拉夫王国

1918年12月成立的塞尔维亚人、克罗地亚人和斯洛文尼亚人王国把原塞尔维亚、黑山控制区合并进中央集权国家。科索沃没有作为自治或历史区域整体存在，而被划入多个县、1922年后的州以及1929年后的泽塔、摩拉瓦和瓦尔达尔等巴诺维纳。分割行政边界有交通和中央管理考虑，也使阿尔巴尼亚人口难以形成统一地区代表。

南斯拉夫议会制度给予居民形式上的选举参与，但治安压力、政党限制、识字率低和地方官员控制削弱代表性。杰米耶特党在1920年代初组织部分穆斯林和阿尔巴尼亚选民，以宗教学校、土地和地方权利换取对中央政府的议会支持；政府完成关键表决后压制该党，显示少数族群政治被纳入的是不稳定的交换关系。

## 土地改革与殖民安置

国家把土地改革解释为消除奥斯曼封建关系、奖励退伍军人、扶植无地农民并加强边疆安全。实践中，权属证明标准、没收“弃置地”、国有地划拨和殖民政策使许多阿尔巴尼亚家庭失地，来自塞尔维亚、黑山及其他地区的塞族和黑山族殖民者获得地块。

- **政策目标**：改变被政府视为不可靠的边疆人口结构，保护铁路与道路，建立忠于国家的农户。
- **执行方式**：丈量土地、否定部分奥斯曼契约、没收离境者或被认定为武装支持者的财产、建立殖民村。
- **实际矛盾**：殖民者常得到缺少水利、住房或安全保障的土地；本地农民则诉称祖传地被重新登记。
- **长期结果**：土地问题与民族身份绑定，1941年后部分殖民者遭驱逐，1945年新政权又必须处理返还和禁止回归问题。

殖民规模不足以彻底改变科索沃人口多数结构，却强化了国家权力与族群竞争的日常接触面。

## 卡查克抵抗

“卡查克”原指脱离官方控制的武装者，战后逐渐成为反南斯拉夫统治的阿尔巴尼亚游击力量。科索沃民族防卫委员会在斯库台等地提供政治和物资支持；阿泽姆·加利察与绍特·加利察是最著名的领导者。抵抗者袭击宪兵、官员和殖民点，也向村庄征收物资；国家军警则进行围剿、集体惩罚、拘押和焚毁。

| 时间 | 过程 | 结果 |
|---|---|---|
| 1918—1919年 | 塞尔维亚军队复归后零散起义扩大 | 地方武装与阿尔巴尼亚境内委员会建立联系。 |
| 1919年 | 德雷尼察等地冲突 | 王国加强军警与边境封锁。 |
| 1921—1923年 | 加利察武装活动达到高峰 | 部分地区形成短暂、不稳定的游击控制区。 |
| 1924年 | 阿泽姆·加利察受伤身亡，阿尔巴尼亚政局转向 | 有组织抵抗主力被瓦解，残余活动继续但难以挑战国家。 |

卡查克运动包含民族自决、地方自治、土地防卫、家族复仇和武装生计等多重动机。把所有参与者描述为统一民族军队，或一概称为匪徒，都不能解释其社会基础和内部差异。

## 移民、教育与社会变化

两次世界大战之间，穆斯林和阿尔巴尼亚居民持续向阿尔巴尼亚、土耳其及其他地区迁移。动因包括安全恐惧、失地、税役、家庭网络和土耳其接纳穆斯林移民的政策。南斯拉夫与土耳其1938年草签移民公约，计划迁移大批穆斯林人口，但因财政、反对意见和第二次世界大战爆发而未正式实施。即使没有该公约，个人和家庭迁移已经改变地方社会。

国家修建道路、行政设施和少量学校，但阿尔巴尼亚语正规教育受到压制，识字率和医疗条件长期落后。普里什蒂纳等城镇出现新官僚、商人和工人阶层，矿业开发尤其是特雷普查矿区在外国资本与国家政策下恢复重要性。现代化收益分布不均，阿尔巴尼亚农民很少进入行政和专业职位。

## 统治困境与1941年崩溃

### 结构因素

- 国家以统一主义和安全治理处理民族多样性，缺少可信的地区自治渠道。
- 土地改革同时承担社会改革与人口工程目标，使财产权争议民族化。
- 教育、司法和警察缺少阿尔巴尼亚语人员，居民与国家沟通成本高。
- 科索沃经济以农业和矿产输出为主，公共投资不足，社会流动有限。

### 外部压力与直接触发

意大利长期支持阿尔巴尼亚修正边界，德国和保加利亚也把南斯拉夫内部矛盾视为战略机会。1941年3月南斯拉夫政变后，轴心国于4月入侵。王国军队迅速崩溃，科索沃未形成稳定防线；德国、意大利和保加利亚随即按各自战略分割地区。许多阿尔巴尼亚居民把意大利控制下与阿尔巴尼亚合并视为摆脱南斯拉夫统治，部分塞族与黑山族殖民者则遭驱逐或逃亡，战间期矛盾转化为战争报复。

## 重要事件

| 时间 | 事件 | 影响 |
|---|---|---|
| 1912年 | 塞尔维亚、黑山军控制科索沃 | 奥斯曼统治结束，国家与多数居民的主权认知发生冲突。 |
| 1913年 | 国际划界 | 科索沃留在阿尔巴尼亚边界之外。 |
| 1915—1918年 | 奥匈与保加利亚占领 | 母语教育、强制同化、征用和战时报复在不同占领区并存。 |
| 1918年 | 科索沃进入南斯拉夫王国 | 中央集权取代塞黑分治，未建立地区自治。 |
| 1919—1924年 | 卡查克抵抗 | 形成长期治安战争，激化国家镇压和地方不信任。 |
| 1920—1930年代 | 土地改革与殖民安置 | 财产权和人口问题深度民族化。 |
| 1938年 | 南斯拉夫—土耳其移民公约草签 | 大规模移民计划未正式执行，却反映国家人口政策方向。 |
| 1941年4月 | 轴心国入侵 | 南斯拉夫统治崩溃，科索沃被再次分割。 |
