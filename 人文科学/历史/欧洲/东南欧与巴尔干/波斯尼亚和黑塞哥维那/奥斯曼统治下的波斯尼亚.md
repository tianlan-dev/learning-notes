# 奥斯曼统治下的波斯尼亚

## 时间

1463—1878年；黑塞哥维那主要领地至1482年才全部纳入

## 概括

奥斯曼通过1463年的王国征服、其后数十年的边疆战争和1482年对科萨查残余领地的吞并，把波斯尼亚—黑塞哥维那改造为帝国西北边疆。统治从桑贾克、蒂玛尔和要塞军制发展为1580年建立的波斯尼亚行省，再经19世纪坦齐马特改革转为维拉耶特。城市、伊斯兰宗教基金与跨帝国商业兴起，多宗教共同体持续共存；与此同时，长期战争、瘟疫、迁徙、土地关系和中央集权改革不断重塑人口。1878年的终结并非奥斯曼在柏林会议上立即割让主权，而是奥匈获得占领和管理权、奥斯曼保留名义主权至1908年。

## 征服与统治建立

穆罕默德二世在1450年代已迫使波斯尼亚纳贡并扶植王位竞争者。末王斯捷潘·托马舍维奇停止纳贡、寻求西方援助后，奥斯曼军于1463年迅速夺取波博瓦茨、克柳奇和亚伊采，俘杀国王。匈牙利同年反攻，在北部建立亚伊采和斯雷布雷尼克防区；赫尔采格·斯捷潘及其子仍控制南部若干要塞。因此，征服是1463—1520年代逐段推进的边疆过程，而不是单日换朝。

1463年建立波斯尼亚桑贾克，最初隶属鲁米利亚；1470年前后另设黑塞哥维那桑贾克。亚伊采到1527/1528年才被奥斯曼夺取，西北比哈奇则于1592年陷落。新征服区以要塞、边防骑兵和蒂玛尔采邑组织，旧贵族中有人战死、流亡或进入奥斯曼军政体系。

## 行省发展与边疆社会

### 1580年行省建立

1580年，奥斯曼把波斯尼亚及周边若干桑贾克组成波斯尼亚行省，由贝勒贝伊 / 帕夏统辖。这一安排提升了本地在帝国防御中的等级，也使总督能够协调从德里纳到克罗地亚边境的军队、税收和要塞。

边疆军队包括蒂玛尔骑兵、要塞守军、地方队长与辅助部队。许多要塞辖区形成世袭化的卡佩坦体系，本地穆斯林军政家族因此获得长期影响；但总督仍由苏丹任命，波斯尼亚不是独立世袭国家。

### 城市、贸易与宗教基金

萨拉热窝在伊萨贝格·伊萨科维奇和加齐·胡斯雷夫贝格等官员捐建下发展为行政、手工业与商贸中心。莫斯塔尔、特拉夫尼克、巴尼亚卢卡、图兹拉和福查等城镇以桥梁、商队驿站、市场、浴室、清真寺、学校和慈善厨房连接。瓦克夫宗教基金把私人财产转化为公共设施的长期收入来源。

塞法迪犹太人在16世纪后进入萨拉热窝等城；天主教方济各会凭征服初期的保护文书与修道院网络延续，东正教主教区和修道院也服务不断增长或迁入的信众。宗教共同体有自身婚姻、教育和慈善组织，但司法、税负及政治地位并不完全平等。

## 伊斯兰化与人口变迁

波斯尼亚的伊斯兰化持续数代，城市、军政职位、土地权利、税制、家庭网络和社会流动共同发挥作用。皈依者主要来自本地人口，不能解释为土耳其人大规模替换。改宗速度因城市、边区和山区而异；部分家庭内部可长期保有不同信仰。

战争也改变人口分布。16—18世纪奥斯曼—哈布斯堡与威尼斯战争造成逃亡、俘虏、再定居和军事殖民；1697年欧根亲王突袭并焚毁萨拉热窝，带来严重破坏。哈布斯堡收复匈牙利后，来自失地的穆斯林难民进入波斯尼亚，东正教与天主教人口也跨越边境迁移。现代民族不能从某次迁徙或单一宗教直接倒推。

## 土地、税收与地方权力

| 层次 | 制度 / 群体 | 作用与变化 |
|---|---|---|
| 帝国中央 | 苏丹、帝国会议、任命的总督 | 决定战争、税制、官员任免和改革；远距离与战时财政使执行强弱不一。 |
| 行省 | 贝勒贝伊、后称瓦利；行省议事机构 | 协调各桑贾克、要塞与财政；总督任期常短，不能视作稳定地方王朝。 |
| 桑贾克与县 | 桑贾克贝伊、卡迪、穆特萨勒夫等 | 负责军政、司法、征税与文书；19世纪官署名称和权限反复改革。 |
| 军事土地 | 蒂玛尔持有人、要塞军与卡佩坦 | 以收入换兵役；17—18世纪后俸禄化、包税和地方家族权势增长。 |
| 城乡精英 | 阿扬、商人、乌里玛、基督教教士 | 调解中央与地方、控制信用和社区组织；既可合作也可领导反抗。 |
| 农村生产者 | 穆斯林与基督徒农户、佃农 | 承担地租、什一税和非常征发；19世纪地主—佃农矛盾成为危机根源。 |

## 重要事件

| 时间 | 事件 | 过程与结果 |
|---|---|---|
| 1463年 | 王国中枢被征服 | 末王被处决；奥斯曼建立桑贾克，匈牙利则在北部重建边防据点。 |
| 1482年 | 黑塞哥维那残余领地失守 | 科萨查末支撤离，奥斯曼控制今日波黑大部。 |
| 1527/1528年 | 亚伊采陷落 | 匈牙利在波斯尼亚北部的主要防区终结，奥斯曼边界向克罗地亚推进。 |
| 1580年 | 波斯尼亚行省建立 | 多个桑贾克被纳入统一边疆军政体系，首任总督为费尔哈德帕夏·索科洛维奇。 |
| 1592年 | 比哈奇被攻占 | 奥斯曼在西北取得最大推进之一，随后长期战争使边界逐步固定。 |
| 1683—1699年 | 大土耳其战争 | 奥斯曼在中欧败退，波斯尼亚从扩张前沿转为防御边疆；1697年萨拉热窝遭毁。 |
| 1718年 | 帕萨罗维茨和约 | 北部部分地区暂归哈布斯堡；1739年贝尔格莱德和约后边界再次调整。 |
| 1737年 | 巴尼亚卢卡战役 | 波斯尼亚地方军在总督领导下击退哈布斯堡进攻，强化本地边防认同。 |
| 1788—1791年 | 新一轮奥哈战争 | 贸易和人口再受破坏，此后边境大体稳定，但军政负担仍重。 |
| 1804—1815年 | 塞尔维亚起义 | 德里纳边界军事化，难民与地方武装流动加剧；塞尔维亚自治改变地区力量平衡。 |
| 1831—1832年 | 侯赛因·格拉达什切维奇自治运动 | 波斯尼亚卡佩坦与地方精英反对削权、征兵和割地，先胜后败；黑塞哥维那的阿里帕夏支持中央。 |
| 1833—1851年 | 黑塞哥维那行省分设 | 阿里帕夏·里兹万贝戈维奇获半自治总督地位；1851年被奥斯曼军处死，地区重新并入。 |
| 1850—1851年 | 奥马尔帕夏·拉塔斯整顿 | 中央军击败地方贵族，废除卡佩坦残余，坦齐马特中央化取得突破。 |
| 1867年 | 波斯尼亚维拉耶特成立 | 按新省制设置行政委员会、县区和官僚机构，试图加强税收、治安与基础设施。 |
| 1875年 | 黑塞哥维那起义 | 佃农负担、歉收、税收和地主关系引发起义，塞尔维亚、黑山和列强介入后升级为东方危机。 |
| 1878年 | 柏林会议与奥匈占领 | 条约授权奥匈占领管理波黑并驻军新帕扎尔桑贾克；当地武装激烈抵抗后被击败，奥斯曼只保留名义主权。 |

## 19世纪改革、反抗与统治终结

### 结构性矛盾

- 军事采邑衰退后，包税、私人债务和地主—佃农关系使农村负担更难预测。
- 地方卡佩坦、阿扬和禁卫军利益与苏丹新军、统一征税和中央任官冲突。
- 东正教、天主教和穆斯林政治诉求并不一致，但都受到列强保护政治和邻国民族项目影响。
- 交通、教育和行政改革投入不足，官员频繁调动又削弱地方信任。

### 外部压力

塞尔维亚与黑山自治、哈布斯堡边境政策、俄罗斯泛斯拉夫外交和奥匈向巴尔干扩张共同压缩奥斯曼回旋空间。1875年起义引发塞尔维亚、黑山对奥战争和1877—1878年俄土战争，使波斯尼亚问题成为列强重划秩序的一部分。

### 直接终结

柏林会议没有在1878年把波黑法理主权直接转给奥匈，而是授权其占领和管理。奥匈军在图兹拉、萨拉热窝、莫斯塔尔等地遭到以本地穆斯林为主、亦有其他反占领者参与的抵抗，至秋季控制主要城市。此后伊斯坦布尔不再行使实际行政，奥斯曼时期在地方治理意义上结束；1908年吞并才终止名义主权。

## 代表性行政人物

| 人物 | 身份与时期 | 作用 |
|---|---|---|
| 伊萨贝格·伊萨科维奇 | 15世纪波斯尼亚桑贾克早期官员 | 参与征服与萨拉热窝早期城市建设。 |
| 加齐·胡斯雷夫贝格 | 1520—1540年代多次任桑贾克贝伊 | 以瓦克夫建设清真寺、学校、市场和公共设施，塑造萨拉热窝。 |
| 费尔哈德帕夏·索科洛维奇 | 1580年首任波斯尼亚行省总督 | 整合边疆桑贾克并建设巴尼亚卢卡；遇刺身亡。 |
| 侯赛因·格拉达什切维奇 | 格拉达查茨卡佩坦；1831—1832年运动领袖 | 要求波斯尼亚自治、反对中央军改；失败流亡。 |
| 阿里帕夏·里兹万贝戈维奇 | 1833—1851年黑塞哥维那总督 | 因支持中央镇压自治运动获分治地位，后因抗拒改革被清除。 |
| 奥马尔帕夏·拉塔斯 | 1850—1851年中央整顿统帅 | 摧毁地方军事家族的制度性自主，为维拉耶特改革铺路。 |

行省总督任期常短且同一人可能多次任职，完整名单含大量代理、复任与辖区变化；本页不把这一行政轮替误写成君主世系。
