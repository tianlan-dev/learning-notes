# 联合公国、独立与王国建立

## 时间

1859—1914年

## 概括

1859年瓦拉几亚与摩尔达维亚分别选举亚历山德鲁·伊万·库扎，使有限邦联转化为共戴一君；1862年统一政府和议会后，现代国家行政框架形成。库扎以土地、教会财产、教育和法制改革削弱旧等级，却因1864年强化个人权力而被自由派与保守派联合推翻。1866年引入霍亨索伦王朝和宪法，1877—1878年战争取得独立，1881年建立王国。国家在铁路、财政、军队和教育上发展，但有限选举、地主结构、农民贫困和少数群体排斥构成长期弱点。

## 库扎的联合与改革（1859—1866年）

列强最初只承认库扎在两个公国分别任职。库扎和统一派逐步合并军队、外交、税务与行政；1861年底奥斯曼允许其任内实现完整行政联合，1862年布加勒斯特设单一政府和议会，国名逐渐固定为“罗马尼亚”。奥斯曼宗主权和列强保证仍在，国家尚未国际法独立。

| 改革 | 具体过程 | 成效与矛盾 |
|---|---|---|
| 修道院地产世俗化（1863年） | 没收大量向境外希腊圣地输送收入的“奉献修道院”土地 | 扩大国家土地与财政，触发教会、奥斯曼和国际争议。 |
| 土地改革（1864年） | 在库扎解散反对议会、以公投通过《巴黎公约扩充章程》后实施；废除部分封建义务并给予农民小块土地 | 形成自耕农，但土地不足、人口增长和大地产继续造成贫困。 |
| 法律与教育 | 采用受法国影响的民法、刑法，建立大学和公共教育体系 | 统一两公国制度，扩大官僚国家，也增加财政与执行压力。 |
| 行政中央化 | 统一部委、县制、军队和首都 | 有助国家形成，却削弱地方传统并扩大库扎亲信统治。 |

自由派希望更快扩大政治权利，保守地主反对土地和选举改革，两者组成“怪异联盟”。库扎的婚外关系、财政争议和1864年权力集中削弱支持；1866年2月军官迫使他签署退位书。政变终结个人统治，却保留主要改革与统一成果。

## 外来王朝与1866年宪制

摄政团先邀请佛兰德伯爵未果，随后选择普鲁士霍亨索伦-锡格马林根家卡尔亲王。卡尔秘密穿越哈布斯堡领土，1866年5月到达布加勒斯特，以卡罗尔一世身份宣誓。外来王朝减少本地大族争位，并为国家争取欧洲王室联系。

1866年宪法规定世袭君主、两院议会、部长责任和权利条款，形式上未提奥斯曼宗主；选举按财产和纳税等级进行，绝大多数农民没有有效政治影响。国王任命首相、解散议会并掌握军队，政府又常通过行政影响选举。国家自由党与保守党逐步形成主要党派，卡罗尔后来以“轮流执政”维持精英平衡。

## 独立战争与王国建立（1877—1881年）

1877年俄土战争爆发，罗马尼亚允许俄军过境并于5月9／21日宣布独立。俄军在普列文受阻后请求支援，卡罗尔统率罗俄联军，罗马尼亚部队参加格里维察、拉霍瓦、普列文和维丁作战。战争胜利使独立成为事实，但俄国要求收回1856年归还摩尔达维亚的南比萨拉比亚，暴露盟友目标差异。

1878年柏林会议承认独立，条件包括给予非基督徒更平等的公民权；罗马尼亚被迫把南比萨拉比亚让给俄国，以北多布罗加和多瑙河三角洲补偿。政府对犹太人入籍仍采取个案审查，平等承诺执行有限。1881年3月议会宣布王国，卡罗尔和伊丽莎白于5月加冕。

## 王国的崛起、治理与社会裂缝

- **财政与基础设施**：铁路连接布加勒斯特、雅西、多瑙河港口和特兰西瓦尼亚边界；国家银行、关税和石油业发展，使出口经济增长。
- **军队与外交**：担忧俄国扩张后，卡罗尔于1883年秘密加入三国同盟；协议长期未向公众和多数政治人物充分公开。
- **党派轮替**：国家自由党支持工业保护、官僚中央化，保守党依托地主与传统精英；竞争存在，但国王任命、政府操纵选举和财产资格限制削弱代议性。
- **农村结构**：大地产把土地租给中间承租人，农民地块碎片化、债务和劳役契约恶化。1907年摩尔达维亚北部租佃冲突扩散为全国起义，军队镇压并造成大量死亡；直接触发是契约争端，结构原因是土地不足、市场风险和政治排斥。
- **民族与宗教政策**：北多布罗加的穆斯林、保加利亚人及其他群体处于特殊行政；犹太居民长期难以集体取得国籍，显示民族国家建构的排他性。
- **文化建设**：罗马尼亚科学院、大学、学校、报刊和国家教会制度推广共同语言与历史叙事，同时加深首都同乡村、中心同边缘差异。

## 1913年扩张与1914年选择

第二次巴尔干战争中，罗马尼亚越过多瑙河迫使保加利亚求和，1913年《布加勒斯特条约》取得南多布罗加。行动显示军力和外交地位上升，但新增领土居民多为保加利亚人、土耳其人和鞑靼人，加剧边界与少数族群问题。1914年卡罗尔希望按秘密同盟支持同盟国，王室会议认为奥匈属于进攻方，决定保持中立；卡罗尔同年去世，斐迪南继位。

## 重要事件

| 时间 | 事件 | 因果意义 |
|---|---|---|
| 1859年 | 库扎双重当选 | 以共同君主突破有限邦联安排。 |
| 1862年 | 政府与议会统一 | 从共戴一君转为实际单一国家。 |
| 1863—1864年 | 世俗化、土地与宪制改革 | 建立现代财政和法制，也造成权力集中与利益反弹。 |
| 1866年 | 库扎退位、卡罗尔即位、颁布宪法 | 引入稳定世袭王朝和长期宪制框架。 |
| 1877—1878年 | 独立战争与柏林会议 | 摆脱奥斯曼宗主权，领土交换并接受公民权条件。 |
| 1881年 | 王国建立 | 独立国家地位制度化。 |
| 1883年 | 秘密加入三国同盟 | 以奥匈、德国制衡俄国，却同国内民族同情存在张力。 |
| 1907年 | 农民起义 | 暴露土地制度和有限政治参与的结构危机。 |
| 1913年 | 取得南多布罗加 | 区域地位上升，也积累与保加利亚的新矛盾。 |
