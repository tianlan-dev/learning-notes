# 达契亚、罗马行省与早期中世纪

## 时间

约公元前1世纪—13世纪

## 概括

现代罗马尼亚领土在古代并非一个连续政权：喀尔巴阡山内外分布着达契亚、盖塔等群体，黑海沿岸有希腊城市，后又分别处于罗马行省、帝国边境与多个迁徙政权影响下。罗马征服带来拉丁语、城市、矿业和道路体系，撤军后政治机构中断；罗马尼亚语及东罗曼人群的形成涉及罗马化人口、巴尔干拉丁语共同体、斯拉夫接触与长期迁徙，不能把达契亚王国直接等同于现代罗马尼亚。

## 达契亚王权的形成与崩解

约公元前82年，布雷比斯塔利用宗教权威、军事动员和对喀尔巴阡盐矿、金属及多瑙河贸易的控制，把多个盖塔—达契亚集团纳入松散王国。他介入黑海希腊城市和罗马内战，势力一度横跨多瑙河两岸。公元前44年布雷比斯塔遇刺，王国分裂为数个中心，说明统一主要依赖个人与战争联盟，尚未形成稳定继承制度。

1世纪末，德凯巴卢斯在奥勒什蒂山地要塞群基础上重建较强王权。公元85—89年达契亚军越过多瑙河，图密善派军反击；罗马虽取得部分胜利，仍以补贴和技术人员换取边境和平。德凯巴卢斯随后利用资源整军，促使图拉真发动101—102年、105—106年两次战争。萨尔米泽盖图萨失陷后德凯巴卢斯自尽，达契亚王国灭亡。

## 罗马达契亚的统治与撤离

| 层面 | 具体过程 | 长期影响与限制 |
|---|---|---|
| 行政 | 106年后罗马在喀尔巴阡山内、奥尔特尼亚与巴纳特等地设省，多次调整为若干行省和军区；多瑙河以东、北的大部分地区并未长期纳入。 | “罗马达契亚”疆域不等同于现代国界。 |
| 军事 | 军团、辅助军与多瑙河防线保护矿区、道路和城市；边境外仍有自由达契亚人及其他群体。 | 行省安全高度依赖帝国在多瑙河的整体军力。 |
| 经济 | 阿普塞尼金矿、盐、农业和跨境贸易支撑城市；殖民者、退伍军人和商人来自帝国多地。 | 经济繁荣与军事财政和强制征用相连。 |
| 文化 | 拉丁语行政、铭文、城市生活与多神教传播，后期基督教影响增加。 | 罗马化是长期混合过程，不是人口被完全替换。 |

3世纪哥特等集团越境、帝国内战和财政危机使北多瑙河行省难以防守。奥勒良在约271—275年撤出军队和省级行政，并在多瑙河以南另设“达契亚”行省。撤离不是所有居民同时迁走，但考古材料无法单独证明每一地区人口和语言的连续方式。

## 迁徙时代与早期中世纪重组

- 4世纪哥特联盟控制部分地区，基督教经多瑙河和哥特网络传播；376年匈人压力引发大规模迁徙。
- 5世纪匈人霸权以军事联盟为主，阿提拉死后瓦解；格皮德随后在特兰西瓦尼亚及蒂萨河流域建立权力，567年败于阿瓦尔—伦巴第联盟。
- 6—7世纪斯拉夫人广泛迁入多瑙河两岸，同既有人口互动；斯拉夫语深刻影响罗马尼亚语，东正教礼仪和政治词汇后来多经斯拉夫—保加利亚渠道传入。
- 第一保加利亚帝国在8—10世纪对多瑙河下游和喀尔巴阡外地区有不同程度的政治、宗教影响，但控制范围随时期变化。
- 9—11世纪马扎尔人进入盆地，匈牙利王权逐步扩展至特兰西瓦尼亚；“格拉德”“盖卢”“梅努莫鲁特”等早期统治者主要见于较晚编年史，实际身份与领地存在争议。
- 10—13世纪佩切涅格、乌古斯、库曼等突厥语草原联盟控制或影响喀尔巴阡外平原；1241—1242年蒙古入侵摧毁聚落，也促使匈牙利王权与地方领主强化石堡和边防。

## 罗马尼亚语人群形成与史学争议

东罗曼语来自巴尔干拉丁语，在斯拉夫迁徙后仍存续，并同阿尔巴尼亚语、希腊语和南斯拉夫语言形成巴尔干语言联盟特征。中世纪文献中的“弗拉赫人”可指说东罗曼语的牧民、军役群体或地方共同体，不总是现代民族标签。学界对核心形成区、罗马撤军后北多瑙河连续性、跨多瑙河迁徙规模争论长期存在；较稳妥的结论是南北巴尔干都参与了语言和人口网络，现代民族国家叙事不能替代分阶段证据。

## 重要转折

| 时间 | 事件 | 因果意义 |
|---|---|---|
| 约前82—前44年 | 布雷比斯塔统一与王国分裂 | 显示矿产、贸易和个人军事联盟可暂时整合诸部，也暴露继承薄弱。 |
| 85—89年 | 图密善时期达契亚战争 | 德凯巴卢斯获得喘息并整军，罗马边境矛盾未解决。 |
| 101—106年 | 图拉真征服 | 达契亚王权被消灭，部分地区转为罗马行省。 |
| 约271—275年 | 奥勒良撤军 | 省级行政北撤，地区进入多中心边疆秩序。 |
| 6—7世纪 | 斯拉夫迁徙 | 重塑语言、聚落、宗教和政治环境。 |
| 9—13世纪 | 匈牙利扩张与草原联盟竞争 | 形成特兰西瓦尼亚王国体系和喀尔巴阡外地方公国发展的双重背景。 |
| 1241—1242年 | 蒙古入侵 | 破坏旧边防，推动其后堡垒化、移民与地方权力重组。 |
