# 古代马其顿与罗马—拜占庭时期

## 时间

公元前1千纪—6世纪

## 范围与前提

本页以今天北马其顿国土为观察中心，同时交代古代马其顿王国和罗马“马其顿”行政区。现代国土在古代横跨佩奥尼亚、达尔达尼亚南缘、上马其顿和达萨雷提亚等不同空间；古代马其顿王国的核心在今希腊北部。两者有部分空间重叠，却不是边界、人口与国家制度的连续体。

## 古代区域格局

### 佩奥尼亚与北部共同体

佩奥尼亚人活动于阿克西奥斯河即今瓦尔达尔河中上游，政治中心曾在比拉佐拉等地。其社会由多个部族与王权联盟构成，既同马其顿王国贸易、结盟，也受其军事压力。前4世纪腓力二世北扩后，佩奥尼亚诸王多在依附与反抗之间维持有限自主；亚历山大三世远征时仍能看到佩奥尼亚骑兵作为独立编组参战。继业者时代，佩奥尼亚王权逐渐衰落，南部地区越来越受马其顿控制。

今斯科普里周边更接近达尔达尼亚势力范围。达尔达尼亚人与马其顿王国在前3—2世纪多次交战，边界随军事胜负移动。西南的佩拉戈尼亚、林凯斯提斯和达萨雷提亚则连接奥赫里德—普雷斯帕盆地、马其顿高地与伊利里亚世界。由此可见，今日国土在古代并非单一民族或单一王国的自然单位。

### 古代马其顿王国的扩张

阿吉德王朝从下马其顿平原崛起，阿敏塔斯三世之后由腓力二世完成军制改革、贵族整合和矿产财政扩张。前358年后，腓力击败北方对手，控制上马其顿和部分佩奥尼亚地区；前338年喀罗尼亚战役又确立对希腊诸邦的霸权。亚历山大三世自前334年东征，摧毁阿契美尼德帝国，但帝国在他前323年去世后迅速分裂。

这段扩张使今日北马其顿南部进入王国或其交通腹地，却不能把帝国最大版图当作现代国家领土。古王国的王都埃盖与佩拉均在今希腊，政治精英、军队与希腊城邦世界关系密切；关于古代马其顿人的语言和身份应留在古代材料语境，不以现代民族争议预设答案。

## 罗马征服与行省化

### 征服过程

- 前215—205年的第一次马其顿战争使罗马开始直接介入巴尔干东部。
- 前200—197年的第二次战争以库诺斯克法莱战役结束，腓力五世被迫削弱海陆军并放弃域外控制。
- 前171—168年的第三次战争中，罗马在彼得那击败珀尔修斯，废除安提柯王朝，把原王国分为四个政治经济受限的共和国。
- 前149—148年安德里斯库斯起兵失败后，罗马取消分区安排。
- 前146年马其顿正式成为行省，并在其后吸收或管辖伊利里亚、伊庇鲁斯和希腊部分地区；边界后来又多次调整。

罗马征服不是一次即刻同质化。佩奥尼亚旧地、斯库皮和奥赫里德周边分别被纳入不同城市辖区和道路网络，地方语言、乡村社会与帝国行政长期并存。

### 道路、城市与经济

埃格纳提亚大道从亚得里亚海沿奥赫里德、赫拉克利亚、塞萨洛尼基通向拜占庭城，是横穿巴尔干的战略轴线。斯托比位于瓦尔达尔河与黑河汇流处，凭交通、贸易与铸币发展为重要城市；斯库皮靠近北方军事边境，弗拉维王朝时期成为殖民城市；赫拉克利亚·林凯斯提斯控制比托拉盆地与西向通道；利赫尼多斯即奥赫里德则联系湖区和埃格纳提亚大道。

农业、畜牧、矿业与城市手工业通过税收和市场接入帝国。退伍军人殖民、地方精英获得罗马公民权以及212年《安东尼努斯敕令》扩大公民资格，逐步改变政治身份，但乡村聚落和区域差异一直显著。

## 晚期古代与早期基督教

3世纪危机、哥特人入侵和帝国改革改变行政结构。戴克里先—君士坦丁时期，原马其顿行省拆分为更小单位，今北马其顿不同部分曾涉及第一马其顿、第二马其顿、达尔达尼亚和新伊庇鲁斯等省，并共同处在伊利里库姆大区或马其顿教区的变化框架中。395年后该地区归东罗马帝国，但“东西分治”并未在地方社会立刻造成全面断裂。

基督教在4世纪合法化后成为城市公共生活核心：

| 城市 | 古代位置与功能 | 基督教与考古意义 |
|---|---|---|
| 斯托比 | 瓦尔达尔河交通枢纽 | 设主教座堂，多座巴西利卡、洗礼堂和马赛克反映4—6世纪教会网络。 |
| 斯库皮 | 达尔达尼亚南部行政与军事中心 | 主教参与晚期古代教会会议；518年地震后城市格局发生重大变化。 |
| 赫拉克利亚·林凯斯提斯 | 连接爱琴海与奥赫里德的道路节点 | 大、小巴西利卡及马赛克显示城市基督教化与区域贸易。 |
| 利赫尼多斯 | 奥赫里德湖东岸城市 | 早期教堂遗址为后来奥赫里德宗教中心提供空间前史，但不能据此假定机构无间断延续。 |
| 巴尔加拉 | 今什蒂普附近的晚期古代城镇 | 城墙、主教堂与住宅展示内陆城市在5—6世纪的发展。 |

## 6世纪危机与阶段终结

查士丁尼一世时期，帝国修筑或恢复要塞，并试图维护多瑙河防线。传统上有人把“第一查士丁尼亚纳”与斯科普里附近陶尔西乌姆联系，但具体遗址定位仍有争议。541年起的查士丁尼瘟疫削弱人口与财政；地震、税赋压力和长距离贸易收缩影响城市。阿瓦尔与斯拉夫群体自6世纪后期反复越过多瑙河，围攻或袭击巴尔干城市，帝国与波斯长期战争又分散军力。

### 结构因素

- 多瑙边防需要持续军费，内陆城市高度依赖帝国税收、军需和道路安全。
- 大地产、城市—乡村分化及人口下降降低地方承受连续战争的能力。
- 行政单位屡次重组，显示帝国仍有治理能力，却也反映边境压力上升。

### 外部压力

- 哥特、匈人、阿瓦尔及不同斯拉夫群体的迁徙和军事行动并非一次“替代”，而是持续数代的冲突、定居与融合。
- 6世纪末至7世纪初东罗马与萨珊波斯大战，使巴尔干驻军和财政资源进一步紧张。

### 直接转折

602年多瑙河军队哗变与福卡斯夺位后，边防体系迅速恶化。此后斯拉夫聚落在巴尔干内陆扩展，许多古典城市缩小、迁移或转为堡垒—教会中心。东罗马并未从地区完全消失，但其控制由连续行省治理转为沿海、城堡和阶段性军事恢复。古代城市时代由此进入中世纪多中心格局。

|---|---|---|
| 前4世纪中叶 | 腓力二世扩张至上马其顿与佩奥尼亚方向 | 古马其顿王国将现代国土南缘纳入势力范围。 |
| 前168年 | 彼得那战役 | 安提柯王朝覆亡，罗马建立四区控制。 |
| 前146年 | 马其顿行省建立 | 地区进入持久罗马行政、道路与税收体系。 |
| 1—2世纪 | 斯托比、斯库皮等城市发展 | 内陆交通、市场和罗马公民共同体扩大。 |
| 4世纪 | 基督教合法化并制度化 | 主教区和巴西利卡成为城市治理、救济与身份中心。 |
| 479年 | 东哥特人进攻斯托比等地 | 显示巴尔干道路城市在迁徙时代的脆弱性。 |
| 518年 | 强烈地震破坏斯库皮 | 北部城市与行政空间重组。 |
| 6世纪后期—7世纪 | 阿瓦尔战争和斯拉夫定居扩大 | 古典城市网络收缩，中世纪人口与语言格局形成。 |
