# 教皇国教宗世系表

## 时间

752年-1870年（教宗任期表延伸至庇护九世1878年逝世）

## 概括

本表从斯德望二世开始列出与教皇国形成、延续和终结相重叠的公认教宗序列。754—756年丕平献土为罗马主教建立可持续的中部意大利领土基础；1870年意大利王国占领罗马后，教皇国作为半岛领土国家灭亡。教宗的宗教职务一直延续，1929年建立的梵蒂冈城国是后来通过条约形成的新主权安排，不应把1870年后的教宗继续当作教皇国君主。

教宗不是世袭君主。中世纪以前选举规则多次变化，1059年以后枢机选举权逐步制度化；表中“世系”指公认任职顺序。官方序列中的本笃九世三段任期分别列出；对立教宗不列入正统序号。阿维尼翁教廷与西方教会大分裂期间，教宗名义、驻地和教皇国的实际控制必须分开理解。

## 关键继承、争议与实际控制说明

| 时间 | 情形 | 名义与实际权力 |
|---|---|---|
| 754—756年 | 斯德望二世与法兰克国王丕平结盟 | “丕平献土”奠定教皇国领土基础；教宗仍需法兰克军事保护。 |
| 800年 | 良三世为查理曼加冕 | 教宗获得授冠象征，皇帝则宣称保护罗马与意大利秩序，双方权威长期竞合。 |
| 9—10世纪 | 加洛林权力衰退、罗马贵族家族控制选举 | 教宗保有名义领土，罗马及各城市常由贵族、侯爵或地方军阀实际支配。 |
| 1046—1122年 | 教会改革与叙任权斗争 | 枢机选举和教会法强化教宗独立；皇帝在意大利仍有军政力量。 |
| 1143—1193年 | 罗马公社挑战教宗世俗统治 | 多位教宗离开罗马；宗教首席与城市实际政府并不重合。 |
| 1309—1377年 | 教廷常驻阿维尼翁 | 教皇国由枢机使节、地方领主和公社分层治理；阿尔博诺兹枢机推动再整合。 |
| 1378—1417年 | 西方教会大分裂 | 罗马、阿维尼翁及比萨系统互称对立；本表沿用后来公认序列，争议者不另编号。 |
| 1492—1527年 | 亚历山大六世、儒略二世和良十世时期 | 教宗亲族、雇佣军和外交联盟推动领土国家化；1527年罗马之劫暴露军事依赖。 |
| 1798—1799年 | 罗马共和国 | 法军废除教皇国，庇护六世被俘并在流亡中去世。 |
| 1808—1814年 | 法国吞并与庇护七世被拘押 | 教皇国再次中断；维也纳秩序下才恢复。 |
| 1848—1849年 | 庇护九世出走与罗马共和国 | 法军恢复教宗统治，此后罗马的生存依赖法国驻军。 |
| 1860年 | 中北部教皇领地并入意大利王国 | 教皇国缩为罗马及拉齐奥核心。 |
| 1870年 | 意大利军队攻入罗马 | 教皇国领土主权终结；教宗与意大利国家的争议延续到1929年。 |
