# 罗马王政与共和国时期

## 时间

传统前753年-前27年

## 概括

罗马由台伯河畔的拉丁城邦发展为统一意大利半岛的共和国，再扩张为地中海霸权。其优势不只来自军团，也来自可伸缩的公民权、殖民地、道路和分级盟约：战败社群往往被纳入而非完全排除。与此同时，海外征服带来的财富、奴隶和军权集中破坏了共和国原有平衡，最终在内战中转向个人军政统治。

## 共和国统治结构

| 机构 / 群体 | 作用 | 制衡与局限 |
|---|---|---|
| 元老院 | 掌握外交、财政、行省与长期政策的连续性 | 法理上多为咨询，实际由贵族和前任高级官员主导 |
| 执政官 | 两人每年当选，统军并主持政务 | 任期短、同僚互相否决；危机时可任命独裁官 |
| 公民大会 | 选举官员、立法、宣战和审判部分案件 | 投票单位并非一人一票，财富与地域影响代表性 |
| 保民官与平民大会 | 保护平民、否决官员行为、推动平民决议 | 产生于“平民撤离”斗争，后来也成为政治竞争工具 |
| 裁判官、监察官、财务官等 | 司法、户籍、公共道德、财政和行政 | 构成逐级晋升的官职序列 |
| 意大利盟邦 | 提供军队，保有不同程度内部自治 | 长期无完整罗马公民权，最终引发同盟者战争 |

## 分阶段发展

1. **王政城市化（传统前753-前509）**：拉丁、萨宾和伊特鲁里亚因素共同塑造城市、宗教与军事制度。
2. **共和国早期与身份斗争（前509-前367）**：罗马抵御邻邦，贵族与平民围绕债务、土地和官职资格反复冲突；《十二铜表法》与保民官制度使冲突制度化。
3. **统一意大利（前367-前264）**：拉丁战争、萨莫奈战争和皮洛士战争后，罗马以公民殖民地、拉丁殖民地、道路与差等盟约构成半岛联盟。
4. **地中海扩张（前264-前133）**：三次布匿战争、马其顿战争和对希腊化世界的干预使罗马获得海外行省，意大利成为军事与政治核心。
5. **共和国危机（前133-前27）**：土地集中、奴隶经济、盟邦权利、军队个人化和寡头竞争推动暴力升级，格拉古改革、同盟者战争、苏拉独裁与连续内战相继发生。

## 重要事件

- 前509年传统纪年，末代国王被驱逐，共和国执政官制度建立。
- 前494年传统纪年，第一次平民撤离促成保民官制度。
- 前451-前450年，《十二铜表法》公开基本法律规则。
- 前390年或前387年，高卢人攻入罗马，促使罗马重建防御和军事体系。
- 前338年，拉丁战争结束，罗马重组拉丁同盟并扩大差等公民权。
- 前343-前290年，三次萨莫奈战争使罗马控制亚平宁中南部。
- 前280-前275年，皮洛士战争后罗马成为半岛最强权力。
- 前264-前146年，三次布匿战争摧毁迦太基霸权，罗马获得西地中海行省。
- 前133年，提比略·格拉古遇害，精英竞争开始更频繁诉诸政治暴力。
- 前91-前87年，同盟者战争迫使罗马向大多数意大利自由民授予公民权，半岛政治整合显著加深。
- 前82-前79年，苏拉以军事胜利建立独裁并清洗政敌，军队干预政权成为危险先例。
- 前49年，凯撒越过卢比孔河，第二轮大内战爆发；前44年凯撒遇刺并未恢复旧秩序。
- 前31年屋大维在亚克兴获胜，前27年获“奥古斯都”称号，元首制确立。

## 崛起机制

罗马的公民兵传统提供广泛动员基础；战败者可获完全公民权、无投票权公民权或盟邦地位，使人力储备不断扩大。殖民地保护交通节点，道路加快军队调动，元老院则在年度官员更替中维持政策连续性。失败后仍能重新征兵和结盟，是罗马在布匿战争等长期冲突中胜出的关键。

## 共和国衰落与终结

- **结构因素**：海外战争扩大贫富差距，公共土地和奴隶劳动向大地产集中；年度官职难以管理广阔帝国，行省统治又制造寻租空间。
- **社会矛盾**：意大利盟邦承担战争义务却缺乏公民权，小农长期服役，城市民众依赖粮食与政治庇护。
- **权力机制变化**：将领以战利品、土地承诺和私人忠诚维系军队，马略、苏拉、庞培、凯撒、安东尼与屋大维均借军权介入政治。
- **直接触发**：凯撒遇刺后的权力真空、后三头同盟清洗和安东尼—屋大维内战，使恢复旧共和国失去现实基础。前27年的安排保留共和国名义，却把关键军政权集中于奥古斯都。
