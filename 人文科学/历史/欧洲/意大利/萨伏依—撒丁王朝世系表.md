# 萨伏依—撒丁王朝世系表

## 时间

约1003年-1861年（此后续入意大利王国世系）

## 概括

萨伏依家族从阿尔卑斯通道上的伯国扩张为公国，1713年先取得西西里王号，1720年以西西里交换撒丁，形成通常以都灵和皮埃蒙特为行政核心的撒丁王国。1847年的“完美融合”统一大陆与岛屿制度，1861年维托里奥·埃马努埃莱二世又以原有王号序数成为意大利国王。本表按公认主权者顺序列全，早期纪年因文书稀少以“约”表示。

## 摄政、共治与权力说明

| 时间 | 摄政 / 实际掌权者 | 原因与影响 |
|---|---|---|
| 1060-1091年 | 苏萨的阿德莱德 | 先后为奥多内遗孀及两位儿子的监护人；她去世后皮埃蒙特领地出现继承战争。 |
| 1391年后 | 博讷·德·波旁及贵族集团 | 阿梅迪奥八世幼年；宫廷派系围绕摄政发生冲突。 |
| 1472-1478年 | 约兰德·德·法兰西等 | 菲利贝托一世幼年，法国和家族亲王争夺影响。 |
| 1637-1648年 | 法兰西的克里斯蒂娜 | 两位幼主之母；“亲王战争”中抵抗萨伏依叔父支系并依靠法国。 |
| 1675-1684年 | 玛丽亚·乔瓦娜·巴蒂斯塔 | 维托里奥·阿梅迪奥二世幼年；成年后夺回实权。 |
| 1798-1814年 | 法国控制大陆、王室留在撒丁岛 | “撒丁国王”仍保有王号与岛上实权，但不控制皮埃蒙特。 |
| 1821年 | 卡洛·阿尔贝托短暂摄政 | 维托里奥·埃马努埃莱一世退位、卡洛·费利切不在都灵；摄政承诺宪法，后被新王废止。 |

## 崛起、转型与统一

萨伏依崛起的结构条件是控制阿尔卑斯山口、能在法国与帝国之间转换联盟，以及较早形成以都灵为中心的军政官僚。1713—1720年取得王号提高了国际法地位；真正的税收、军队和人口核心仍在皮埃蒙特，而非撒丁岛。法国革命战争一度摧毁大陆政权，却也让复辟后的王国取得热那亚。

1848年宪章使撒丁成为革命失败后少数保留宪政的意大利邦国。加富尔的财政、铁路与外交改革、法国军事援助、中意公投和加里波第南征共同扩大王国。1861年不是萨伏依王朝消失，而是撒丁王国的制度、国王编号和政府连续扩展为意大利王国。
