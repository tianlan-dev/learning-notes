# 世界历史通史

## 概括

本目录用于放置不适合归入单一区域的世界历史横向主题，例如世界大帝国时空对照、跨区域帝国比较、全球贸易体系、殖民扩张、世界大战和冷战格局。具体区域史仍放回各区域目录维护。

## 整理建议

- 世界级专题只在本目录保留横向框架，具体地区细节链接到区域目录。
- 如果某个帝国有明确核心区域，例如迦太基、阿拉伯帝国、罗马帝国，应优先放在对应区域通史下，本目录只做对照索引。
- 俄罗斯国家史放在欧洲 / 斯拉夫 / 东斯拉夫；草原汗国和中亚互动放在中亚 / 草原汗国，不单独建立跨区域一级目录。
