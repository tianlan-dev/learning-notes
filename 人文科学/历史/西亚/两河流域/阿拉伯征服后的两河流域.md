# 阿拉伯征服后的两河流域

## 时间

约633—945年；本页重点是征服、正统哈里发与倭马亚治理，以及阿拔斯以巴格达—萨迈拉为中心的阶段。945年白益王朝进入巴格达后，哈里发仍延续，但区域实际权力结构转入地方王朝与军人政权时代。

## 概括

阿拉伯征服使两河从萨珊帝国财政和王都核心转为新哈里发国家的东部基地。穆斯林没有把全部萨珊灌溉土地分给征服军，而是保留“萨瓦德”土地税源，在巴士拉、库法建立军营城市并以名册俸禄安置部族军队。库法很快成为第一次内战、什叶派和法学传统中心；倭马亚通过伊拉克总督管理整个东部。阿拔斯革命依靠呼罗珊军进入伊拉克，762年建立巴格达，把两河再次变成欧亚帝国首都和商业学术中心。

## 征服与阶段发展

- **征服萨珊两河（约633—642年）**：早期战事的年份和路线主要由后世阿拉伯叙事保存，细节互异。卡迪西亚通常置于636或637年，泰西封在637年失守；巴士拉、库法随后成为继续征服胡齐斯坦和伊朗高原的基地。
- **正统哈里发军镇（约638—661年）**：征服军按部族编入 dīwān 名册领取俸禄，萨瓦德农民继续耕作并缴地税。阿里656年把政治中心移到库法，使伊拉克成为第一次内战主战场。
- **倭马亚东部总督区（661—750年）**：大马士革哈里发通过巴士拉、库法和后来瓦西特的总督统管伊拉克、伊朗与呼罗珊。齐亚德、哈查只等强势总督重整税收、军队和阿拉伯语行政，也面对哈瓦利吉、什叶派和部族叛乱。
- **阿拔斯建都（750—809年）**：阿拔斯在库法附近宣布新王朝，曼苏尔于762年建“和平之城”巴格达。呼罗珊军、波斯—伊拉克书吏、商人和宗教学者使首都迅速扩张。
- **内战与萨迈拉（809—892年）**：阿明与马蒙战争围困并破坏巴格达；穆尔台绥姆836年迁都萨迈拉，以突厥军人解决首都冲突，却使禁卫军更能左右废立。892年宫廷回到巴格达。
- **实际权力地方化（9—10世纪）**：赞吉起义、财政困难、军人争权和各省独立削弱中央；945年白益军进入巴格达，哈里发保留宗教—礼仪地位，行政军事权落入埃米尔手中。

## 统治结构与最高权力

## 土地、财政与社会

| 机制 | 内容 |
|---|---|
| 萨瓦德土地 | 征服后大面积土地保留原耕作者，由国家征收 kharāj 地税；具体土地究竟属于共同体、国库或私人，在法学和实际执行中长期变化。 |
| 军镇与俸禄 | 巴士拉、库法按部族街区安置征服军及家属，以战利品和税收发放 ʿaṭāʾ；军镇后来发展为商业、语言、法学和政治中心。 |
| 旧官僚延续 | 早期税务仍使用中古波斯语和旧有书吏，7世纪末才逐步阿拉伯语化；萨珊渠网、地籍和税名被调整而非全部废除。 |
| 城市网络 | 瓦西特连接巴士拉与库法，巴格达连接呼罗珊路、叙利亚路和波斯湾，萨迈拉则是为宫廷军队新建的线性首都。 |
| 商业与知识 | 巴格达市场汇集纸张、纺织、香料和金属；宫廷翻译、天文、医学与哲学依靠叙利亚语基督徒、波斯书吏、犹太学者及多地知识传统。 |
| 宗教政治 | 库法形成阿里派、什叶派和哈乃斐法学重要传统，巴士拉孕育语言学、穆尔太齐赖与伊巴德派网络；宗派形成与城市政治、部族和 mawālī 身份交织。 |

## 重要事件

1. 633—634年前后穆斯林军进入伊拉克南部，与拉赫姆旧地、萨珊驻军和地方部族交战；早期战役次序存在史源争议。
2. 636 / 637年卡迪西亚战役击溃萨珊主力，统帅鲁斯塔姆战死；次年泰西封失守，萨珊王廷东撤。
3. 约638年前后巴士拉、库法逐步建成军镇，成为向伊朗高原、海湾和中亚扩张的组织基地。
4. 656年骆驼之战发生在巴士拉附近，阿里获胜并以库法为首都；657年隋芬战役及仲裁造成阵营分裂。
5. 658年阿里在纳赫拉万击败哈瓦利吉，661年又在库法遇刺；倭马亚王朝建立。
6. 680年侯赛因及随从在卡尔巴拉被倭马亚军杀害，成为什叶派历史记忆和宗教礼仪核心。
7. 685—687年穆赫塔尔在库法起义，以为侯赛因复仇和支持伊本哈乃菲叶为号召，动员大量非阿拉伯归信者，最终被祖拜尔派击败。
8. 697—702年前后哈查只平定伊本阿什阿斯特大叛乱、推进税务文书阿拉伯语化，并修建瓦西特安置叙利亚军。
9. 749年阿拔斯支持者在库法拥立萨法赫，750年大扎卜河战役结束倭马亚在东方的统治。
10. 762年曼苏尔建巴格达圆城；宫廷、军队和市场很快扩展到城墙外，形成跨河大都市。
11. 哈伦拉希德时期帝国财政和宫廷文化达到高峰；803年巴尔马克家族失势显示哈里发仍可清洗强势官僚集团。
12. 809—813年阿明—马蒙内战，塔希尔军长期围攻巴格达；阿明被杀，首都遭严重破坏，呼罗珊军事集团权势上升。
13. 836年穆尔台绥姆迁都萨迈拉；861年穆塔瓦基勒被禁卫军杀害后进入“萨迈拉无政府”阶段，哈里发频繁废立。
14. 869—883年赞吉起义控制巴士拉沼泽和南部多城，暴露大庄园、劳工、税政与中央军力问题；阿拔斯历经多年才镇压。
15. 892年穆塔迪德时期宫廷回到巴格达，但埃及、伊朗和中亚的地方王朝已掌握各自税源。
16. 945年白益王朝艾哈迈德进入巴格达，获封“穆伊兹道莱”；阿拔斯哈里发转为受军人埃米尔保护和制约。

## 成为帝国中心的条件

- 征服者保留萨瓦德税源和当地生产者，避免把灌溉农业一次性瓜分耗尽。
- 巴士拉、库法把部族军队与平民隔离安置，既便于继续扩张，也形成新的阿拉伯—伊斯兰城市文化。
- 倭马亚总督把伊拉克与伊朗东部纳入同一军政轴线，东方税赋和兵源由此集中。
- 阿拔斯革命的呼罗珊联盟需要一个连接伊朗与阿拉伯核心的首都，巴格达地理位置优于大马士革。
- 运河农业、波斯湾海运、丝路陆路及纸张传播共同支撑人口、贸易和知识机构。
- 阿拔斯借先知家族合法性、波斯官僚传统和多民族军队重构帝国，而不是简单“波斯化”或“阿拉伯化”。

## 结构压力、衰落与阶段终点

- **结构因素**：两河繁荣依赖持续清淤、堤防和安全贸易；内战与税收承包会使地方官追求短期收入，渠网和乡村恢复能力下降。
- **军事财政压力**：俸禄军、禁卫军和宫廷开支持续扩大；哈里发为解决旧部族军政治问题引入突厥等新军，又产生将领控制废立的新风险。
- **地方化**：行省总督掌握本地税收后形成世袭王朝，巴格达获得的净收入减少；塔希尔、萨法尔、萨曼等政权名义承认哈里发却自行用兵。
- **社会与外部压力**：赞吉、卡尔马特和部族运动切断南部交通，拜占庭边境和地方战争继续消耗军费；宗派冲突不是唯一原因。
- **直接转折**：财政破产和军队欠饷让巴格达政权依赖外来军事强人。白益进入时没有废除哈里发，而是接管其实际行政军事权，故945年是统治结构转换而非国家突然灭亡。
- **后续**：两河地方史由[伊拉克](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A/%E4%B8%A4%E6%B2%B3%E6%B5%81%E5%9F%9F/%E4%BC%8A%E6%8B%89%E5%85%8B/README.md)继续，哈里发与跨区域伊斯兰帝国主线见[阿拔斯王朝](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A/_%E9%80%9A%E5%8F%B2/%E9%98%BF%E6%8B%89%E4%BC%AF%E5%B8%9D%E5%9B%BD/%E9%98%BF%E6%8B%94%E6%96%AF%E7%8E%8B%E6%9C%9D.md)。
