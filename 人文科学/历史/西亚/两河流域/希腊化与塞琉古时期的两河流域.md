# 希腊化与塞琉古时期的两河流域

## 时间

前331—前141年左右；前141年后塞琉古曾在前130—前129年短暂重占部分两河，未能恢复长期统治。

## 概括

亚历山大在高加米拉获胜后进入巴比伦，保留部分阿契美尼德官僚并计划把城市作为亚洲帝国中心。他前323年死于巴比伦，继业者围绕总督区、军队和王室名义反复争夺。塞琉古一世以巴比伦尼亚为根据地建立王朝，前312年重返巴比伦成为塞琉古纪元起点；新城底格里斯河畔塞琉西亚后来成为东方首都。希腊语城市机构、马其顿军事移民与巴比伦神庙、阿拉米语社群、楔形文字学者长期并存，而非“希腊文化替代东方文化”。

## 阶段过程

- **马其顿接管（前331—前323年）**：总督马扎亚斯投降后继续管理巴比伦；亚历山大修复部分神庙、征集军队和税收，并在返回巴比伦准备阿拉伯远征时病逝。
- **继业者争夺（前323—前312年）**：腓力三世和幼王亚历山大四世名义共治，实权由摄政、将领和总督掌握。塞琉古先任巴比伦总督，后被安提柯一世逼走。
- **塞琉古建国（前312—前281年）**：塞琉古在托勒密支持下重返巴比伦，击败安提柯势力，以该年为塞琉古纪元。称王后建立横跨叙利亚、两河与伊朗的帝国。
- **双都与融合（前3世纪）**：塞琉西亚吸引居民、官员和商贸，安条克成为西部中心；巴比伦仍有神庙、希腊式剧场和楔形文字学校。安条克一世曾长期作为东部共治王。
- **东方收缩（前2世纪）**：继承战争、地中海战争和东部属国独立削弱中央。米特里达梯一世前141年占领巴比伦尼亚；此后两次塞琉古反攻均以国王被俘或战死告终。

## 统治结构与社会

| 机制 | 内容 |
|---|---|
| 王与总督 | 国王任命巴比伦尼亚总督、军事长官和财政官；继业者战争中总督可决定区域归属。 |
| 希腊城市 | 塞琉西亚拥有公民团体、议事机构和铸币，安置马其顿—希腊移民；它不是简单建在巴比伦废墟上，而是新的双河交通中心。 |
| 巴比伦城市与神庙 | 埃萨吉拉及乌鲁克神庙继续持有土地、组织祭祀与教育；祭司编写王表、天文日志并用楔形文字记录希腊君主。 |
| 军事殖民与土地 | 士兵定居点和王室土地提供兵员、税收与马匹；本地村社和商业家族仍参与租佃与供给。 |
| 多元语言 | 王室与城市公民主要使用希腊语；阿拉米语广泛通行，阿卡德语楔形文字继续用于神庙、法律和天文学。 |
| 纪年制度 | 前312年成为塞琉古纪元起点，连续纪年取代每王重新起算，为西亚长期沿用。 |

## 重要事件

1. 前331年高加米拉战役后马扎亚斯交出巴比伦，亚历山大沿用其总督经验。
2. 前323年亚历山大病逝巴比伦，军队在“巴比伦会议”安排腓力三世与未出生的亚历山大四世共同继承。
3. 前321年前后塞琉古获任巴比伦总督；前316年受安提柯压力逃往埃及。
4. 前312年塞琉古率小部队重返巴比伦并击败对手，此年成为塞琉古纪元。
5. 前305年塞琉古称王，前301年伊普苏斯战役后获得叙利亚大部，形成横跨东西的帝国。
6. 塞琉西亚在前4世纪末—前3世纪初建立，前275年前后成为东方行政中心，巴比伦人口与政治地位逐渐下降。
7. 安条克一世作为共治王治理东方，在巴比伦神庙传统中采用本地王权语言。
8. 安条克三世前212—前205年东方远征暂时恢复伊朗诸属国名义臣服，回到巴比伦后使用“大王”称号。
9. 前190年马格尼西亚战败和前188年阿帕米亚和约迫使王朝支付巨额赔款，长期削弱西部财政与战略空间。
10. 前141年安息王米特里达梯一世进入巴比伦尼亚并在塞琉西亚称王。
11. 前140 / 139年德米特里二世反攻失败被俘；前130年安条克七世短暂收复两河，前129年战死后彻底退出。

## 兴盛条件

- 巴比伦尼亚是塞琉古最初根据地，提供粮赋、银钱、兵员和成熟行政。
- 塞琉西亚位于底格里斯、陆路丝路前段和波斯湾贸易交汇处，适合成为东方首都。
- 国王允许神庙和本地书吏延续，使新王朝能用楔形文字传统证明合法性。
- 希腊城市、军事移民和本地社群并存，为军队、税收和商业提供多重基础。
- 塞琉古一世与安条克一世的长期父子共治缓和了最初的继承风险。

## 衰落与终结

- **结构因素**：帝国东西跨度极大，西部安条克与东部塞琉西亚之间距离长；王室不断在多条战线和继承争夺中分配资源。
- **财政与军事压力**：对托勒密、罗马和小亚细亚诸国的战争消耗军队；阿帕米亚赔款并非立即摧毁帝国，却限制后续选择。
- **地方离心**：巴克特里亚、帕提亚、佩尔西斯等逐步自治；希腊城市与神庙能在不同征服者下继续运作，降低其为塞琉古死守的动力。
- **直接终结**：王朝内战使德米特里二世无法集中兵力，米特里达梯一世前141年占领两河；两次反攻均失败。
- **文化延续**：希腊城市、阿拉米语社会和楔形文字天文学进入[安息与萨珊时期的两河流域](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A/%E4%B8%A4%E6%B2%B3%E6%B5%81%E5%9F%9F/%E5%AE%89%E6%81%AF%E4%B8%8E%E8%90%A8%E7%8F%8A%E6%97%B6%E6%9C%9F%E7%9A%84%E4%B8%A4%E6%B2%B3%E6%B5%81%E5%9F%9F.md)，并未随王朝更替中断。
