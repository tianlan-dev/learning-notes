# 希腊化与塞琉古时期的两河流域

## 时间

前331年-前141年左右

## 概括

亚历山大征服阿契美尼德帝国后，两河流域进入希腊化世界。继业者战争后，塞琉古王朝控制巴比伦尼亚和叙利亚，塞琉西亚等新城市成为帝国东方治理与贸易中心。

## 王朝世系 / 统治结构

| 层级 | 说明 |
|---|---|
| 马其顿征服 | 亚历山大征服两河，巴比伦成为其帝国重要中心之一。 |
| 塞琉古王朝 | 塞琉古一世及后继者建立横跨叙利亚、两河和伊朗部分地区的希腊化王国。 |
| 城市网络 | 塞琉西亚等城市与巴比伦传统并存，形成希腊化和本地文化交叠。 |

## 重要事件

- 前331年高加米拉战役后，两河流域脱离波斯统治。
- 亚历山大死后，巴比伦成为继业者政治争夺中心之一。
- 帕提亚 / 安息势力兴起后，塞琉古逐渐失去东方领地。
