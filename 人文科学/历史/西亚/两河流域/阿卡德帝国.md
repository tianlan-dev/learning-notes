# 阿卡德帝国

## 时间

约前2334—前2154年（中年表传统约数）。阿卡德城遗址尚未确定，第三千纪君主的绝对在位年可因年表方案而移动，表中均应理解为“约”。

## 概括

阿卡德帝国由萨尔贡在击败卢伽尔扎格西后建立，是首个长期把苏美尔南部、阿卡德北部及其外缘置于同一王权下的两河帝国。它没有消除城邦、神庙和地方精英，而是以王族任官、驻军、贡赋、道路和统一王室意识形态把它们连接起来。阿卡德语的行政地位上升，苏美尔语仍继续用于宗教、行政和文学。

## 建立与阶段发展

- **萨尔贡建国**：萨尔贡可能先服务于基什王室，随后夺取权力，击败控制南部多城的卢伽尔扎格西。他以阿卡德为中心，把地方统治者、王族和军事长官置于帝国网络。
- **两子巩固**：里穆什即位后面对南部大叛乱，以军事镇压恢复控制；马尼什图舒继续远征并留下大规模土地交易文书，显示王室与地方土地持有者的复杂关系。
- **纳拉姆辛鼎盛**：纳拉姆辛平定“诸王大叛乱”，远征叙利亚、扎格罗斯与北部地区，自称“四方之王”并在名字前使用神性符号，帝国王权达到象征高峰。
- **沙尔卡利沙里收缩**：边疆的卢卢比、古提等压力加重，南部城市恢复自主，中央控制退回核心地区。
- **末期复兴失败**：王位争夺后，杜杜和舒图鲁勒恢复阿卡德城周边统治，却已无法重建萨尔贡时代的帝国。

## 统治结构

| 机制 | 内容 |
|---|---|
| 王族与地方官 | 王子、公主和忠诚官员被安排到关键城市；萨尔贡之女恩赫杜安娜任乌尔月神大祭司，把宗教职位与王朝合法性结合。 |
| 城邦延续 | 地方 ensi、神庙和城市档案仍运作；帝国统治是对既有机构加层控制，不是统一官僚制完全取代城邦。 |
| 军事与交通 | 核心常备随从、地方征发和驻军支撑连续远征；河运及陆路把南部粮食、北部木材和东部金属网络相连。 |
| 王室意识形态 | “四方之王”、胜利碑和神化君主宣称超越单城保护神的普遍统治；这种宣传不能等同实际边界长期稳定。 |
| 语言与书写 | 阿卡德语成为重要王室行政语言，苏美尔语继续广泛使用；双语书吏文化由此深化。 |

## 重要事件

1. 萨尔贡击败卢伽尔扎格西，俘获其人并把南部主要城邦纳入王权。
2. 里穆什即位之初镇压乌尔、乌鲁克、拉伽什等地的联合叛乱，显示继承并非自动获得地方承认。
3. 马尼什图舒组织海上或沿湾远征；其方尖碑记录大规模土地购买，反映王室扩张经济基础。
4. 纳拉姆辛平定基什、乌鲁克等多城参加的“大叛乱”，随后加强神化与“四方之王”称号。
5. 纳拉姆辛对扎格罗斯卢卢比的胜利被刻成胜利碑；对埃布拉等北方城市的征服范围仍有考古与文本争论。
6. 沙尔卡利沙里时期，古提、阿摩利和埃兰方向战争并存，苏美尔城市开始重新独立。
7. 沙尔卡利沙里死后出现四王争位，中央权力在数年内碎裂。
8. 杜杜、舒图鲁勒只能维持缩小的阿卡德政权；其后古提首领及乌鲁克、拉伽什等地方力量分掌区域。

## 鼎盛条件

- 继承了苏美尔成熟的城市、书吏、仓储和灌溉体系。
- 阿卡德靠近南北交通节点，便于连接南部粮区、迪亚拉通道和北方商路。
- 王族婚姻、祭司任命及地方官网络减少了单纯依赖征服的成本。
- 连续胜利和战利品强化王室随从，也形成一种跨城邦的帝国身份。
- 双语行政使北部阿卡德语网络与南部苏美尔机构能够在同一统治体系内运作。

## 衰落与终结

- **结构因素**：版图跨越生态差异巨大的地区，交通和驻军成本高；地方城邦保有自身机构，一遇继承危机即可脱离。王权高度依赖统治者个人威望和连续军事成功。
- **外部压力**：扎格罗斯山地的古提、卢卢比，西部阿摩利群体及埃兰方向战争不断消耗边疆。所谓约前2200年的“4.2千年干旱事件”可能加剧北部农业与人口压力，但地区影响和它在崩溃中的权重仍有争议。
- **直接触发**：沙尔卡利沙里死后的争位和地方叛离摧毁中央协调；古提人是参与者而非唯一原因。舒图鲁勒之后，阿卡德王权不再出现。
- **后续重组**：古提势力、拉伽什第二王朝和乌鲁克的乌图赫加尔先后占据部分地区，最终由[乌尔第三王朝](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A/%E4%B8%A4%E6%B2%B3%E6%B5%81%E5%9F%9F/%E4%B9%8C%E5%B0%94%E7%AC%AC%E4%B8%89%E7%8E%8B%E6%9C%9D.md)重建南部大国。
