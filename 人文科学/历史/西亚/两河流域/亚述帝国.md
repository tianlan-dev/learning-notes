# 亚述帝国

## 时间

本页重点为中亚述国家兴起至末代残余政权，约前1420—前609年；中亚述帝国化通常从前14世纪中叶算起；新亚述帝国通常约前911—前609年。前12世纪以前的绝对年代存在约十年的不同重建方案，表中均按约年理解；前911年以后可用名年官表、日食记录和编年史较可靠地固定。

## 概括

亚述从底格里斯河上游的阿淑尔城及贸易共同体发展为领土国家，继而成为覆盖两河、黎凡特、安纳托利亚部分地区、埃及和伊朗西缘的新亚述帝国。它的优势不只是“残酷军队”，而在于持续征兵、攻城工程、行省制、驿路情报、粮秣仓储和有计划的强制迁徙。王室浮雕与铭文刻意放大惩罚以威慑敌人，但实际治理同时依靠地方精英、附庸王、税收协商和多语言书吏。

## 建立与阶段发展

- **中亚述崛起（约前14—前13世纪）**：阿淑尔乌巴利特一世摆脱米坦尼支配，进入埃及和巴比伦外交体系；阿达德尼拉里一世、沙尔马那塞尔一世摧毁米坦尼残余，把北部两河改造成省区。
- **图库尔蒂尼努尔塔一世高峰与危机**：他击败加喜特巴比伦并短期统治南部，另建新都卡尔—图库尔蒂—尼努尔塔；晚年被宫廷集团杀害，继承战争削弱王权。
- **前12—前10世纪收缩**：提格拉特帕拉沙尔一世曾恢复远征，但青铜时代末期动荡、阿拉米部落扩张和内部争位使亚述退守核心；王统和城市机构仍未中断。
- **新亚述复兴（前911—前745年）**：阿达德尼拉里二世恢复边境，阿淑尔纳西尔帕二世以卡尔胡为新都，沙尔马那塞尔三世把征战推进至地中海和扎格罗斯。
- **帝国改革与鼎盛（前745—前631年）**：提格拉特帕拉沙尔三世扩大直属行省、常备军和迁徙体系；萨尔贡二世及其后继者统治巴比伦、黎凡特和一度占领埃及，尼尼微成为首都。
- **内战与崩溃（前631—前609年）**：阿淑尔巴尼拔死期及末期王序有数年争议。巴比伦的那波帕拉萨尔反叛，米底加入战争，阿淑尔和尼尼微相继陷落；阿淑尔乌巴利特二世在哈兰维持残余政权至前609年。

## 王朝世系

从恩利尔纳西尔二世到阿淑尔乌巴利特二世的51位连续君主、短暂并立者与日期争议，见[亚述君主世系表](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A/%E4%B8%A4%E6%B2%B3%E6%B5%81%E5%9F%9F/%E4%BA%9A%E8%BF%B0%E5%90%9B%E4%B8%BB%E4%B8%96%E7%B3%BB%E8%A1%A8.md)。早于本页的“帐篷中王”“祖先王”和古亚述商贸时期王序不纳入此阶段；专表说明其与中亚述王统的关系。

## 统治结构

| 层级 | 运作方式 |
|---|---|
| 国王与阿淑尔神 | 国王被视为阿淑尔神在人间的代理和最高战帅，须通过征服“恢复秩序”；实际决策依靠王室、宦官、将领、学者和高级宫廷官。 |
| 名年官制 | 每年以一位 līmu 官员命名，形成纪年链；新亚述名年官表是重建年代的重要基础。 |
| 行省与总督 | 核心领土分省，由总督征税、征兵和修路；国王会拆小过强省份，限制地方大员形成独立势力。 |
| 附庸体系 | 边缘王国宣誓、纳贡、供军并交人质；反叛者可能被废，领地改为直属行省。 |
| 军队 | 王室部队、行省征兵、附庸兵、骑兵、战车、工兵和攻城器械协同；提格拉特帕拉沙尔三世后常备化程度提高。 |
| 迁徙政策 | 被征服人口按国家计划跨区安置，用于补充农民、工匠、士兵和城市居民，也拆解反抗网络；迁入者通常成为帝国臣民，并非一概处死或奴役。 |
| 驿路与情报 | 王家道路、驿站、密封公文和边境报告使中央在数日到数周内获得军政信息。 |
| 巴比伦政策 | 亚述王或兼任巴比伦王、或扶立代理国王；尊重南部神庙与强制镇压交替，形成长期合法性难题。 |

## 重要事件

1. 约前1350年，阿淑尔乌巴利特一世与埃及法老通信，并干预巴比伦王位，标志亚述进入大国体系。
2. 阿达德尼拉里一世和沙尔马那塞尔一世击败米坦尼 / 哈尼加尔巴特，将北部两河纳入行省。
3. 约前1225年，图库尔蒂尼努尔塔一世击败巴比伦王卡什提利亚舒四世、掠取巴比伦；数年后亚述在南部统治瓦解。
4. 提格拉特帕拉沙尔一世远征地中海和幼发拉底河上游，但未能阻止阿拉米群体在叙利亚与两河边缘扩张。
5. 前883年后，阿淑尔纳西尔帕二世重建卡尔胡，迁入人口并使其成为行政和宫廷中心。
6. 前853年卡尔卡战役中，沙尔马那塞尔三世遭遇大马士革、以色列等国联盟；亚述自称胜利，却未能立即征服叙利亚。
7. 前745年提格拉特帕拉沙尔三世夺位，重组军队和行省，随后兼任巴比伦王并吞并叙利亚、以色列北部。
8. 前722年前后撒马利亚陷落；沙尔马那塞尔五世围城，萨尔贡二世完成接管并迁徙人口，具体功绩归属有争议。
9. 前710—前709年萨尔贡二世重夺巴比伦；前705年战死后，西拿基立继位并迁都尼尼微。
10. 前701年西拿基立镇压犹大和黎凡特反叛，围困耶路撒冷但未占城；前689年因长期叛乱摧毁巴比伦。
11. 埃萨尔哈东重建巴比伦并于前671年攻入埃及孟斐斯；亚述对埃及的统治短暂而昂贵。
12. 阿淑尔巴尼拔镇压其兄沙马什舒姆乌金领导的巴比伦内战，随后洗劫埃兰苏萨；尼尼微图书馆汇集大量楔形文字作品。
13. 前626年那波帕拉萨尔在巴比伦称王；亚述内战使南部永久脱离。
14. 前614年米底攻陷阿淑尔，前612年米底—巴比伦联盟攻陷尼尼微，辛沙里施昆死亡或失踪。
15. 前609年亚述残余与埃及援军未能夺回哈兰，阿淑尔乌巴利特二世退出史料，帝国终结。

## 鼎盛条件

- 北部两河拥有雨养农业、牧场、木材与金属通道，能支撑不同于南部灌溉区的军事财政。
- 从中亚述时期延续下来的王统、名年官制和省区经验，使前10世纪复兴不必从零开始。
- 铁并非亚述独占；真正优势是工兵、情报、马匹、仓储、道路和多兵种协同的组织能力。
- 把征服区改省、迁徙人口和扶植附庸三种工具组合使用，使中央可因地制宜控制不同地区。
- 对巴比伦书吏、阿拉米语文书、腓尼基工匠等多地域人才的吸收，提高帝国行政和文化整合能力。
- 提格拉特帕拉沙尔三世以来的常备军和省份拆分削弱旧贵族割据，萨尔贡王朝又通过新都建设集中资源。

## 衰落与灭亡

- **结构因素**：版图过大，需要持续战争、贡赋和迁徙维持；多个王都、庞大宫廷与边境驻军成本高。王位继承没有绝对长子制，王子与宫廷集团容易形成内战。
- **统治矛盾**：强制迁徙有利于短期整合，却制造庞大流动人口与治理压力；巴比伦具有独立王权和宗教传统，反复改省、扶王、毁城和重建都未解决合法性问题。
- **外部压力**：米底、巴比伦并非突然出现。扎格罗斯诸国和南部反抗长期存在，埃及与黎凡特战争又分散资源。
- **直接触发**：阿淑尔巴尼拔晚年及死后的继承冲突削弱中央，辛舒姆利希尔等短暂争位；那波帕拉萨尔控制巴比伦后建立稳定后方，米底的加入使亚述同时承受南、东夹击。
- **军事终结**：阿淑尔、尼尼微、哈兰先后失守，核心王都、神庙、档案和征兵区同时丧失，残余王廷无法再重建。
- **历史后果**：[新巴比伦王国](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A/%E4%B8%A4%E6%B2%B3%E6%B5%81%E5%9F%9F/%E6%96%B0%E5%B7%B4%E6%AF%94%E4%BC%A6%E7%8E%8B%E5%9B%BD.md)接收亚述西部帝国空间，[米底王国](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A/%E4%BC%8A%E6%9C%97/%E7%B1%B3%E5%BA%95%E7%8E%8B%E5%9B%BD.md)控制东部联盟成果；行省、驿路、迁徙和帝国王权模式则被后来的波斯政权选择性继承。

## 演变关系

- 南部背景：[古巴比伦王国](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A/%E4%B8%A4%E6%B2%B3%E6%B5%81%E5%9F%9F/%E5%8F%A4%E5%B7%B4%E6%AF%94%E4%BC%A6%E7%8E%8B%E5%9B%BD.md)以后，巴比伦与亚述曾长期并立、征服和反征服，二者不是简单前后王朝。
- 后续节点：[新巴比伦王国](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A/%E4%B8%A4%E6%B2%B3%E6%B5%81%E5%9F%9F/%E6%96%B0%E5%B7%B4%E6%AF%94%E4%BC%A6%E7%8E%8B%E5%9B%BD.md)；亚述是被巴比伦—米底联盟摧毁的一方。
