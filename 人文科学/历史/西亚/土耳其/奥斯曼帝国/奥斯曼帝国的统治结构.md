# 奥斯曼帝国的统治结构

## 时间

1299年-1922年

## 概括

奥斯曼帝国由苏丹、中央官僚、军队、米利特制度、行省和地方精英共同构成复杂统治体系。

## 统治结构

| 层级 | 说明 |
|---|---|
| 苏丹 / 帕迪沙 | 帝国最高统治者，兼具王朝、军事和宗教政治权威。 |
| 大维齐尔与帝国会议 | 中央行政和军政决策核心。 |
| 耶尼切里与军队体系 | 早期为中央军事支柱，后期也成为政治压力集团。 |
| 米利特制度 | 以宗教共同体管理非穆斯林臣民的制度框架。 |
| 行省与地方精英 | 贝勒贝伊、帕夏、地方名门和税收承包共同构成地方统治。 |
| 哈里发身份 | 近代尤其被强调，用于维系穆斯林臣民和国际伊斯兰合法性。 |
