# 阿拔斯哈里发世系表

## 时间与范围

- **巴格达阿拔斯哈里发**：750—1258年，通行世系为37位公认哈里发。
- **开罗阿拔斯哈里发**：1261—1517年，共17名人物、22段在位记录；复位必须重复列出。
- **中断**：1258年蒙古攻陷巴格达后，巴格达世系中断约三年。1261年马穆鲁克苏丹拜巴尔斯扶立阿拔斯宗室，不是原巴格达领土国家迁都开罗。

本表把王朝血统、哈里发名号、实际军政权和在位承认范围分开处理。巴格达后期哈里发常受突厥近卫、布韦希埃米尔或塞尔柱苏丹控制；开罗哈里发通常承担授予苏丹合法性、主持礼仪、外交背书和宗教象征等职能，实际财政与军队属于马穆鲁克苏丹。开罗哈里发并非在所有时刻都毫无政治作用：宫廷危机中，他们可能充当调停者、反对派象征或短暂的权力替代者。

## 开罗哈里发与马穆鲁克苏丹的权力分工

| 权力领域 | 开罗阿拔斯哈里发 | 马穆鲁克苏丹与埃米尔 |
|---|---|---|
| 军队与财政 | 通常没有独立常备军、领土税源和行政体系 | 掌握军队、土地收益、任官、司法行政和对外战争 |
| 继承与废立 | 原则上限于阿拔斯男性宗室，可指定亲属，但不能保证兑现 | 核验或操纵宗谱，主持册立，并能废黜、复位或改立哈里发 |
| 合法性礼仪 | 宣布或背书苏丹权位，参与效忠、诏书、礼拜和公共仪式 | 借阿拔斯血统和哈里发名号把军事夺权包装为逊尼合法统治 |
| 宗教权威 | 象征共同体统一，与乌里玛、法官和宗教机构交往 | 资助宗教机构并控制国家任命；不能仅靠哈里发授权获得所有社会认同 |
| 外交 | 诏书可用于承认远方穆斯林君主或支持马穆鲁克外交主张 | 控制使节、战争与实际盟约；哈里发文书提供附加象征资本 |
| 危机政治 | 可充当调停者、反对派象征；1412年出现哈里发兼苏丹特例 | 埃米尔联盟决定军政归属，能迅速架空或撤换哈里发 |

## 世系连续性与关键辨析

- **37位与17位不可混加成一条无断裂的领土王朝**：巴格达37位是750—1258年的政权世系；开罗17名人物是1261—1517年在马穆鲁克保护下重建的阿拔斯职位。
- **复位必须逐段记录**：穆塔瓦基勒一世三次在位，开罗穆斯台绥木两次在位，穆斯塔姆西克和穆塔瓦基勒三世各两次在位。
- **同名号不等于同一人**：巴格达末代穆斯台绥木与开罗1377、1386年在位者仅使用相同尊号。
- **“礼仪性”不等于完全无作用**：日常军政由苏丹掌握，但哈里发能影响合法性、外交和宫廷危机；穆斯塔因兼任苏丹和卡伊姆被废都是例证。
- **1517年不是有据可证的正式“转让仪式”**：后世流行穆塔瓦基勒三世把哈里发名号和先知圣物交给塞利姆一世的故事，但缺乏可靠同时代证据。可以确认的是奥斯曼征服终结了马穆鲁克国家及开罗阿拔斯职位，奥斯曼对哈里发名号的系统化主张有自身演变过程。
- **终年不等于退位年**：穆塔瓦基勒三世被带往伊斯坦布尔后仍活到16世纪中叶；本表以1517年作为制度终点，而不是他的死亡日期。

## 年代与争议说明

- 中世纪阿拉伯编年常按伊斯兰历记载，换算到公历时会跨两个年份；短期在位者尤其容易出现一年前后差异。
- 巴格达通行37位不包括未获广泛承认的并立者和一日政变领袖，但本表已单独列明，避免把争议人物抹去。
- 穆斯坦绥尔二世和哈基姆一世的具体宗谱在扶立时具有强烈政治用途；当时法官的认可证明其在制度上被接受，不等于现代谱系研究能毫无疑问地重建每一代。
- 开罗早期册立日、穆斯塔克菲一世流放与死亡的具体年月，以及1508—1517年最后两位哈里发的交替日期，在不同编年表中存在细微差异。
- “公认”指在相应政治中心和逊尼制度网络中获得主要承认，不代表同时期法蒂玛、科尔多瓦等竞争哈里发或所有穆斯林群体接受其唯一合法性。
