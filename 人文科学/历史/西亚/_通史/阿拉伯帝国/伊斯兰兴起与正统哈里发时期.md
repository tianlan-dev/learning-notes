# 伊斯兰兴起与正统哈里发时期

## 时间

610年—661年；其中正统哈里发时期为632—661年

## 概括

伊斯兰教从麦加的宗教运动发展为麦地那的政治共同体，继而在阿拉伯半岛形成新的联盟中心。穆罕默德去世后，共同体没有一套固定继承法；四位后来被逊尼传统称为“正统哈里发”的统治者，分别通过紧急协商、前任提名、委员会推举和内战中的拥立取得权位。632—651年前后，哈里发国家征服拜占庭的叙利亚与埃及行省，并摧毁萨珊王朝的中央统治，但伊斯兰化、阿拉伯化和地方社会重组远比军事征服缓慢。

这一阶段同时奠定军镇、迪万名册、征服区税制和哈里发制度，也暴露出战利品分配、总督监督、宗族竞争和先知家族继承权等矛盾。656—661年的第一次内战使政治中心分裂，最终由穆阿维叶建立以大马士革为中心的倭马亚王朝。

## 建立背景与形成过程

- **晚期古代环境**：阿拉伯半岛并非与外界隔绝。红海、也门、叙利亚和两河贸易把部落社会与拜占庭、萨珊及阿克苏姆联系起来，犹太教、基督教和阿拉伯一神信仰传统也已存在。
- **麦加传教**：610年前后，穆罕默德开始宣讲独一神、末日审判、施舍和共同体责任。古莱氏精英的反对既涉及宗教，也与家族秩序和城市权威有关。
- **迁徙与建共同体**：622年穆罕默德及追随者迁往雅斯里卜，后称麦地那。新共同体以信仰为核心，又通过盟约、婚姻和部落关系容纳不同集团；622年后来成为伊斯兰纪元起点。
- **战争、谈判与整合**：624年白德尔之战、625年吴侯德之战和627年壕沟之战改变麦加—麦地那力量对比；628年侯代比亚和约为扩大联盟提供空间，630年穆斯林控制麦加。
- **继承难题**：632年穆罕默德去世后，先知身份不可继承。哈里发承担政治、军事和共同体领导，但谁有资格、由谁选择、先知家族有何权利，从一开始就没有一致答案。

## 正统哈里发完整世系

“正统哈里发”是后来的逊尼史学称谓，并不表示当时所有穆斯林都接受同一种合法性解释。什叶传统把阿里及其后裔的伊玛目权置于核心；哈瓦利吉则强调统治者必须持续符合宗教与道德资格。

| 顺序 | 哈里发 | 在位时间 | 取得权位与继承关系 | 统治重点、废立与结局 |
|---:|---|---|---|---|
| 1 | 阿布·伯克尔 | 632—634 | 穆罕默德岳父和早期追随者；在赛吉法紧急协商中获拥立，阿里及部分先知家族成员起初未参与 | 处理各部落退出麦地那联盟、拒绝纳贡及先知竞争者问题；启动叙利亚、伊拉克方向远征；临终提名欧麦尔。 |
| 2 | **欧麦尔·本·哈塔卜** | 634—644 | 经阿布·伯克尔提名并获主要集团承认 | 建立军镇、军饷迪万和征服区治理框架；叙利亚、埃及、伊拉克与伊朗大部被征服；644年遇刺，遗命六人委员会议定继承。 |
| 3 | 奥斯曼·本·阿凡 | 644—656 | 倭马亚家族成员；由六人委员会推举 | 推动《古兰经》标准文本并继续海陆扩张；亲族总督和资源分配引发批评；656年麦地那住宅被围，遇刺身亡，未能正常指定继承人。 |
| 4 | **阿里·本·阿比·塔利卜** | 656—661 | 穆罕默德堂弟、女婿；奥斯曼死后在麦地那获拥立 | 将政治中心移至库法；先后面对骆驼之战、穆阿维叶集团和哈瓦利吉；661年遇刺。其支持者逐步发展出什叶政治传统。 |

哈桑·本·阿里在阿里死后获部分伊拉克支持者承认为领袖或哈里发，但同年与穆阿维叶和解并放弃争夺。由于其权位短暂、承认范围和称号定位在不同传统中不一，本表不把他列作四位“正统哈里发”之一，但不能据此忽略其在661年权力移交中的作用。

## 统治结构

| 层面 | 运作方式 | 历史意义与限制 |
|---|---|---|
| 哈里发与协商 | 哈里发统军、任命总督、分配公共收入；重要决定需争取迁士、辅士、古莱氏和军政首领支持 | 没有固定选举机构或长子继承法，每次继位方式均不同 |
| 军镇与征服军 | 库法、巴士拉、福斯塔特等集中驻扎阿拉伯军队及家属 | 便于动员并减少直接瓜分农地，也形成能左右中央政治的军镇社会 |
| 迪万与俸禄 | 登记军人、早期功绩和给养，按身份与资历分配国家收入 | 把征服收益制度化，也固化先后次序与分配差异 |
| 行省与地方行政 | 总督负责军政和税收；大量拜占庭、萨珊、科普特和叙利亚文书人员继续工作 | 保证税赋连续，但远方总督可能形成独立网络 |
| 土地与税收 | 多数耕地仍由原居民经营；土地税和人头税沿用并调整 | 征服不等于立即改宗，税种和身份的对应关系随后才逐步规范 |
| 宗教规范 | 《古兰经》启示、先知实践和共同体记忆提供规范 | 圣训、法学学派和成熟司法体系尚未完全定型，后世解释不能机械投射到7世纪 |

## 重要事件

| 时间 | 事件 | 过程、结果与长期影响 |
|---|---|---|
| 610年前后 | 最初启示与麦加传教 | 一神信仰和社会伦理形成新的宗教共同体，遭部分古莱氏集团抵制。 |
| 622年 | 迁徙麦地那 | 穆斯林从受迫害群体转为具有盟约、司法和军事能力的政治共同体。 |
| 624—628年 | 白德尔、吴侯德、壕沟之战与侯代比亚和约 | 军事胜负和外交妥协共同改变半岛联盟，说明统一并非单靠征服。 |
| 630年 | 控制麦加 | 古莱氏大部接受新秩序，克尔白成为伊斯兰朝觐中心；半岛联盟继续扩张。 |
| 632—633年 | 里达战争 | 麦地那击败拒绝税贡、重订盟约或追随其他先知者；“里达”不能简单等同于所有参与者都放弃伊斯兰信仰。 |
| 634—636年 | 叙利亚战役与耶尔穆克之战 | 拜占庭主力战败，大马士革、巴勒斯坦和叙利亚城市逐步纳入哈里发统治。 |
| 636—642年 | 卡迪西亚、攻取泰西封与尼哈万德之战 | 萨珊军政中枢崩解，征服从两河流域扩展至伊朗高原；地方抵抗仍延续多年。 |
| 638年前后 | 耶路撒冷归降 | 城市通过协商纳入新政权，拜占庭在黎凡特的核心统治结束。 |
| 639—642年 | 征服埃及 | 阿姆鲁·本·阿斯军队控制尼罗河谷并建立福斯塔特，拜占庭失去重要税粮区。 |
| 644—651年 | 奥斯曼时期继续扩张 | 海军进入东地中海，高加索和伊朗东部战线推进；萨珊末帝叶兹德格尔德三世死后王朝复国希望消退。 |
| 约650年前后 | 《古兰经》标准文本 | 中央向主要城市发送标准抄本并限制异本差异，成为经典传承的重要转折；具体编纂环节仍有学术讨论。 |
| 656年 | 奥斯曼遇刺与骆驼之战 | 总督任用、财政公正和复仇责任争议转为武装冲突，阿里击败阿伊莎、塔尔哈和祖拜尔一方。 |
| 657年 | 隋芬之战与仲裁 | 阿里与叙利亚总督穆阿维叶相持；仲裁削弱阿里联盟，部分反对者形成哈瓦利吉。 |
| 658年 | 纳赫拉万之战 | 阿里击败哈瓦利吉主力，但相关运动并未消失，刺杀风险和教义分歧持续。 |
| 661年 | 阿里遇刺与权力移交 | 哈桑与穆阿维叶和解，穆阿维叶取得多数地区承认并建立[倭马亚王朝](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A/_%E9%80%9A%E5%8F%B2/%E9%98%BF%E6%8B%89%E4%BC%AF%E5%B8%9D%E5%9B%BD/%E5%80%AD%E9%A9%AC%E4%BA%9A%E7%8E%8B%E6%9C%9D.md)。 |

## 扩张、鼎盛与危机原因

### 扩张条件

- **外部结构**：拜占庭与萨珊经历602—628年长期战争、瘟疫和财政耗损，东部边防与附庸体系受到破坏。
- **阿拉伯动员**：半岛联盟、轻装机动部队和按战利品—俸禄分配的激励，使连续远征成为可能。
- **治理延续**：新统治者保留大量地方税册、官员和农业关系，没有要求全体居民立即改宗，减少征服后的行政真空。
- **多样地方反应**：部分城市抵抗，部分地方精英谈判归降；宗教政策、税负与帝国忠诚在各地不同，不能把征服解释为被征服者一致“欢迎”。

### 制度危机与阶段终结

- **分配矛盾**：早期参战者、后续移民、军镇和麦地那精英争夺税收、土地收益及官职。
- **总督与宗族网络**：广阔行省需要强势代理人，中央监督不足；奥斯曼依赖亲族引发任人唯亲指控。
- **继承无定制**：协商、提名、委员会和紧急拥立没有形成可重复规则，合法性争议随统治范围扩大而加剧。
- **宗教—政治分歧**：先知家族地位、奥斯曼复仇、仲裁和统治者资格的争论，分别成为什叶和哈瓦利吉传统的重要记忆。
- **直接触发**：奥斯曼遇刺后无法就问责与继位达成一致；叙利亚军政体系保持完整，穆阿维叶最终以更稳定的地区军队取得优势。

## 争议与认识边界

- 本阶段的许多叙事由较晚的编年史、传记和教派传统保存；事件主线较清楚，具体谈话、动机和人数常难以验证。
- “阿拉伯征服”是军事与政治过程，不等同于人口立即被替换，也不等同于伊斯兰化、阿拉伯化同步完成。
- “正统”是规范性称谓；整理世系时应区分后来宗派记忆、当时的承认范围和实际控制。
- 耶尔穆克、卡迪西亚等战役的精确日期、参战人数及多阶段战斗细节在史料中存在差异，宜采用约年而不伪造精确性。
