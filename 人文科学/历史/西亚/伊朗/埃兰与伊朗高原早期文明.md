# 埃兰与伊朗高原早期文明

## 时间

约前3200年—前539年

## 概括

埃兰不是单一王朝，而是以苏萨和伊朗西南部为核心、由多个王朝和区域中心构成的长期文明。它与苏美尔、阿卡德、乌尔、巴比伦和亚述持续战争、通婚与贸易，使用埃兰语，并先后采用原始埃兰文字、线形埃兰文字和楔形文字。前6世纪后埃兰政治被阿契美尼德吸收，但苏萨行政、书写传统和地方人口继续成为波斯帝国的重要基础。

## 主要阶段

| 阶段 | 约略时间 | 政治与文化特征 |
|---|---|---|
| 原始埃兰时期 | 前3200—前2700年 | 苏萨与高原聚落形成贸易和文书网络；原始埃兰文字尚未完全释读。 |
| 阿万王朝 | 前3千纪中后期 | 与阿卡德帝国竞争；统治者名单来自后世王表，年代和顺序多有争议。 |
| 西马什基王朝 | 约前2100—前1900年 | 高原势力进入苏萨；前2004年前后参与推翻乌尔第三王朝。 |
| 苏卡尔马赫时期 | 约前1900—前1500年 | 多中心王权，统治者使用“大使者 / 摄政”等称号；与美索不达米亚广泛通婚。 |
| 中埃兰时期 | 约前1500—前1100年 | 安善、苏萨和胡齐斯坦被更紧密整合；建筑、铭文和王权扩张达到高峰。 |
| 新埃兰时期 | 约前1100—前539年 | 地方王朝并立，与亚述、巴比伦反复结盟和战争；后被亚述重创并逐渐纳入波斯势力。 |

早期文献残缺且统治中心多变，无法编出覆盖三千年的可靠“完整君主表”。将所有见于残片的名字排成连续世系会制造虚假的确定性。

## 代表统治者与统治结构

| 统治者 | 约略时期 | 主要活动 |
|---|---|---|
| 普祖尔—因舒希纳克 | 前22世纪末 | 阿万末期重要君主；在苏萨建筑并使用阿卡德文和线形埃兰文。 |
| 金达图等西马什基君主 | 前21—20世纪 | 参与高原—苏萨整合；具体继承关系不完全清楚。 |
| 西韦—帕拉尔—胡帕克 | 前18世纪 | 苏卡尔马赫时期强势统治者，与埃什努纳、马里等两河国家外交。 |
| 温塔什—纳皮里沙 | 前13世纪 | 建造杜尔—温塔什宗教中心及大型齐古拉特。 |
| 舒特鲁克—纳洪特一世 | 前12世纪 | 入侵巴比伦，把汉谟拉比法典碑等战利品运至苏萨。 |
| 希尔哈克—因舒希纳克 | 前12世纪 | 中埃兰扩张高峰，控制伊朗西南和扎格罗斯部分地区。 |
| 胡姆班—哈尔塔什三世等 | 前7世纪 | 新埃兰末期在亚述、巴比伦和国内派系之间反复争战。 |

埃兰王权常由苏萨、安善等不同中心构成，王室成员可能分别管理地区。神庙、宫廷、土地和长途贸易相连，书吏使用埃兰语与阿卡德语。山区部落和低地农业之间的关系，也是王朝强弱的重要变量。

## 重要事件

- 前3千纪苏萨发展文书和商品分配体系，与乌鲁克世界及伊朗高原相连。
- 前23—22世纪阿卡德王纳拉姆辛与埃兰统治者订立条约，是现存早期埃兰语材料之一。
- 前2004年前后西马什基和盟军攻陷乌尔，乌尔第三王朝终结。
- 前13世纪温塔什—纳皮里沙建设杜尔—温塔什，显示宗教中心与王权整合。
- 前12世纪舒特鲁克—纳洪特一世及继承者入侵巴比伦，埃兰达到军事高峰。
- 前8—7世纪埃兰支持巴比伦反亚述力量，亚述多次入侵。
- 前646年前后亚述巴尼拔洗劫苏萨，破坏王室和神庙，但埃兰人口和若干小政权并未消失。
- 前6世纪，波斯人在安善兴起；居鲁士二世征服米底和巴比伦后，苏萨成为[阿契美尼德王朝](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A/%E4%BC%8A%E6%9C%97/%E9%98%BF%E5%A5%91%E7%BE%8E%E5%B0%BC%E5%BE%B7%E7%8E%8B%E6%9C%9D.md)行政中心。

## 兴盛与消失原因

埃兰的优势来自苏萨平原农业、扎格罗斯通道和连接两河—高原—波斯湾的贸易。多中心结构能吸收不同地区，也容易因王族分裂和山区—低地利益冲突而破碎。末期衰落由长期内斗、亚述军事打击和波斯安善势力上升叠加造成。“埃兰灭亡”主要指独立王权消失，不代表语言、书吏和地方文化突然中断；阿契美尼德王室仍在行政中使用埃兰语。
