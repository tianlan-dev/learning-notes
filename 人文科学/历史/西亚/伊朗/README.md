# 伊朗

## 历史主线

伊朗历史以伊朗高原为核心，连接两河流域、中亚、印度和高加索。它从埃兰、米底和阿契美尼德波斯帝国开始，经安息、萨珊形成强烈的伊朗帝国传统；阿拉伯征服后进入伊斯兰化，但波斯语、行政文化和地方王权持续复兴；近世萨法维确立什叶派国家传统，恺加、巴列维和伊斯兰共和国构成近现代主线。

## 名称辨析：波斯、伊朗、伊朗高原

- “伊朗”适合做完整目录名，覆盖古代伊朗高原、波斯帝国传统和现代伊朗国家。
- “波斯”多是外部传统称呼，尤其常用于阿契美尼德、萨珊和前现代伊朗王朝。
- “伊朗高原”是地理文化范围，可能超出现代伊朗国界。
- “波斯帝国”通常特指阿契美尼德，广义时也可能包括萨珊等伊朗帝国，使用时需要说明。
