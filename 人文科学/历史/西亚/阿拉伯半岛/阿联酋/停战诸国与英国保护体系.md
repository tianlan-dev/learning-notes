# 停战诸国与英国保护体系

## 时间

1820—1971年

## 概括

“停战诸国”不是英国直接统治的殖民地，也不是统一国家，而是若干世袭酋长国在英国主导的海上停战、排他外交和保护安排下形成的政治集合。各酋长继续处理继承、土地、税收、警务和部落关系，英国则逐步控制海上秩序、对外关系和外部防务。条约体系压低了酋长国之间公开海战，却也冻结并重塑地方权力边界。珍珠业崩溃、油气开发、边界争端和英国撤出最终迫使这些小型政体寻找联邦方案。

英国条约主要控制海上战争、外交与对外让地，各酋长在内政、税收和部落关系上仍有自主空间；“保护国”因此是逐步形成的权力结构，不是1820年一次条约即完成的全面殖民接管。

## 条约体系的建立

1819年英国远征军攻占拉斯海马并打击沿海船队后，1820年1月起与卡瓦西姆及其他沿岸首领分别签署《海上总条约》。条约禁止英国所称的“海盗行为”、要求船只悬挂规定旗帜并登记，同时承认签约酋长作为海岸政治主体。它没有立即建立完整保护国，却构成此后约150年英国优势地位的法律起点。

| 时间 | 条约或安排 | 具体内容 | 政治后果 |
|---|---|---|---|
| 1809年 | 第一次英国远征 | 攻击拉斯海马、摧毁部分船只后撤离 | 卡瓦西姆力量受损但很快恢复，冲突未解决。 |
| 1819—1820年 | 第二次远征与《海上总条约》 | 英军占领主要据点；签约者接受海上行为规范、船籍与旗帜制度 | 英国开始以条约监督海上秩序，并选择哪些首领被视为合法签约者。 |
| 1835年起 | 珍珠季海上休战 | 在采珠季禁止海上敌对行动，逐年续订 | 保护最重要的现金经济，也扩大英国仲裁角色。 |
| 1843年 | 十年海上休战 | 把季节性安排延长为较长期停战 | 酋长国间海战进一步减少。 |
| 1853年 | 《永久海上停战》 | 争端不得自行在海上诉诸武力，应交由英国处理 | “停战海岸”“停战诸国”名称由此固定。 |
| 1892年 | 排他协议 | 未经英国同意，不得同其他国家缔约、接受代理人或割让领土 | 各酋长国内政仍自治，对外主权却受到严格限制，形成事实保护体系。 |
| 1922年 | 石油特许承诺 | 未经英国同意不得授予石油特许 | 英国把未来能源权益纳入保护体系。 |
| 1952年 | 停战诸国委员会 | 七位获承认酋长定期会商，由英国政治代理人主持早期会议 | 为跨酋长行政与后来联邦合作提供制度经验。 |

英国所谓“反海盗”既包含真实的袭船、俘虏和海上暴力，也服务于保护英属印度航线及马斯喀特盟友。条约带来更稳定的商业环境，但把谁是“合法酋长”、哪些港口可独立签约的决定权大幅交给英国。拉姆斯、迪巴、卡尔巴、哈姆里亚等曾有独立或半独立首领，却没有都成为1971年的联邦成员。

## 七个酋长国政治版图的形成

### 迪拜从阿布扎比分离

1833年，巴尼亚斯联盟阿布法拉萨支系在乌拜德·本·赛义德与马克图姆·本·布提率领下迁入迪拜。阿布扎比未能恢复直接统治，阿勒马克图姆家族由此建立独立酋长国。英国海上停战为迪拜提供外部安全，统治者则以低关税、港口开放和商人自治吸引转口贸易。1890年代末至20世纪初，来自林格等波斯岸港口的商人迁入，迪拜逐渐超过部分传统港口。

### 沙迦与哈伊马角的合并、分离

卡西米家族的沙迦和哈伊马角支系长期重叠。1869年以后哈伊马角一度独立；1900年侯迈德·本·阿卜杜拉去世后，沙迦再度控制该地并任命地方长官。苏尔坦·本·萨利姆1919年取得实际权力，英国于1921年7月7日承认哈伊马角为独立停战酋长国。因而1900—1921年不是“王位空缺”，而是地方权力存在、独立国家地位中断。

### 富查伊拉取得承认

沙尔基家族在19世纪末、20世纪初摆脱沙迦的实际控制，但英国为避免沿岸政体继续碎片化，长期拒绝承认。石油公司希望在东海岸取得清晰特许权后，英国于1952年3月21日承认穆罕默德·本·哈马德为停战酋长，富查伊拉成为最后一个获承认的七酋长国。同一时期，卡尔巴在1951年重新并入沙迦。英国的选择性承认并非单纯记录地方事实，而是参与塑造了今天七酋长国的地图。

## 珍珠繁荣与崩溃

19世纪后半叶至第一次世界大战前，天然珍珠是沿岸最重要的出口。酋长通过船只、市场和商人征费取得收入；商人向船主和潜水员提供预付款，印度市场承担加工和转售。安全改善和全球奢侈品需求扩大船队规模，但债务链也把风险压给潜水者和小船主。

珍珠危机不是单一事件：

1. 第一次世界大战和印度市场波动削弱贸易。
2. 1920年代日本养殖珍珠以更稳定、低廉的价格进入市场。
3. 1929年后大萧条使欧美奢侈品需求骤降。
4. 殖民关税、卢比与信贷变化进一步挤压商人。
5. 第二次世界大战扰乱海运，使恢复更加困难。

到1930年代，船队、税收和商人信贷同时收缩，沿岸出现贫困、迁移、负债与粮食不足。迪拜依靠转口贸易和开放港口稍有缓冲，阿布扎比、乌姆盖万等依赖采珠更深的地方承受更重冲击。珍珠业没有瞬间消失，但已失去支撑整个政治经济的能力。

## 商人、改革与地方治理

停战酋长国没有现代官僚制，统治依靠家族、部落首领、商人和伊斯兰法官之间的协商。港口商人既为统治者提供贷款与关税，也会要求财政透明和参与决策。迪拜1938年改革运动建立短命议会式机构，试图监督收入、教育和市政；统治家族在次年恢复直接控制。沙迦、迪拜的继承危机也常由英国调停，但最终仍取决于家族与地方联盟是否接受。

英国在20世纪中期扩大低成本行政：1951年建立停战阿曼征募兵团，1952年设停战诸国委员会，随后推动简易机场、医疗、教育、道路和发展基金。这些机构规模有限，却培养出跨酋长协调、边界巡逻和公共工程经验。

## 边界划分与布赖米争端

传统部落使用的牧场、绿洲和海岸并无连续国界。石油特许要求把地下资源对应到明确领土，原先可协商的效忠与季节迁徙遂变成主权争端。阿布扎比与迪拜在20世纪40年代发生边界冲突，后通过停火和1968年前后的协议逐步确定边界；北部诸酋长之间则保留飞地、反飞地和交错领土。

布赖米绿洲包含今天阿布扎比的阿莱因诸村与阿曼布赖米一侧村庄。沙特阿拉伯、阿布扎比和马斯喀特—阿曼都提出权利主张，油气预期使争端升级。

- 1952年8月，沙特代表图尔基·本·阿卜杜拉·奥泰尚率人进入哈马萨村，以金钱、效忠和行政存在巩固沙特主张。
- 英国代表阿布扎比和马斯喀特同沙特谈判，并同意仲裁；1955年日内瓦仲裁因程序、证据和干预争议崩溃。
- 1955年10月，停战阿曼征募兵团与马斯喀特部队驱逐沙特人员。绿洲西部阿莱因归阿布扎比控制，东部布赖米归阿曼控制。
- 沙特—阿布扎比更广泛边界问题并未就此完全解决，联邦成立后以1974年《吉达条约》处理，条文解释后来仍有分歧。

## 石油发现与权力重组

1930年代各酋长开始授予勘探特许，但地质、战争和交通条件使商业开发延迟。阿布扎比乌姆谢夫海上油田于1958年发现石油，1962年出口第一批原油；陆上巴卜等油田随后投产。迪拜1966年在法塔赫海上油田发现石油，1969年开始出口。其他酋长资源较少或发现较晚。

石油改变了三层关系：

- **统治者与社会**：石油收入取代关税和珍珠税，统治者能直接建设道路、学校、医院和住房。
- **酋长国之间**：阿布扎比拥有绝大多数储量，迪拜拥有更成熟的港口商业；两者互补，也需要就联邦财政和权力分配谈判。
- **英国与地方**：英国仍负责外部安全，却不再能用低成本维持海湾承诺；当地统治者的财政和外交能力迅速增长。

沙赫布特·本·苏尔坦谨慎积存收入、限制快速支出，引起家族与英国对发展速度的不满。1966年8月，家族会议在英国默许下由其弟扎耶德·本·苏尔坦接任阿布扎比酋长。扎耶德迅速扩大公共建设，并向较贫穷酋长提供援助，为联邦谈判积累政治信用。

## 九酋长联邦谈判

1968年1月英国宣布将在1971年底前撤出“苏伊士以东”。2月18日，阿布扎比酋长扎耶德与迪拜酋长拉希德在两地边界附近达成两酋长联合原则，并邀请其余五个停战酋长国、巴林和卡塔尔组成九成员“阿拉伯酋长国联邦”。

九方同意设最高委员会，却在代表权、表决方式、首都、联邦预算、议席按人口还是政体平等分配等问题上反复争执。巴林人口最多，要求与人口相称的代表；卡塔尔担心阿布扎比和迪拜主导；较小酋长国则希望一酋长国一票。巴林于1971年8月、卡塔尔于9月分别独立，九方方案告终。

七个停战酋长国继续谈判。1971年7月18日，除哈伊马角外的六位酋长同意临时宪法；12月2日阿布扎比、迪拜、沙迦、阿治曼、乌姆盖万和富查伊拉宣布成立阿拉伯联合酋长国。哈伊马角因联邦条件、安全和海峡岛屿争端暂缓加入。

1971年11月30日英国撤离前夕，伊朗控制大通布、小通布并依与沙迦谅解备忘录进驻阿布穆萨岛；哈伊马角在通布岛发生武装冲突。岛屿主权争端延续至今，也凸显单个小酋长难以独自应对区域强权。哈伊马角最终于1972年2月10日加入联邦，七酋长格局完成。

| 时间 | 事件 | 过程 | 结果与长期影响 |
|---|---|---|---|
| 1820年 | 海上总条约 | 英国军事胜利后分别与沿岸首领签约 | 英国条约秩序建立。 |
| 1833年 | 迪拜分离 | 阿布法拉萨支系迁入迪拜 | 马克图姆家族政权形成。 |
| 1835—1853年 | 休战制度逐步永久化 | 珍珠季休战、十年休战、永久停战 | 海战减少，英国仲裁权扩大。 |
| 1892年 | 排他协议 | 酋长限制对外缔约与领土处分 | 形成事实英国保护体系。 |
| 1900—1921年 | 哈伊马角并入后再独立 | 沙迦控制、地方反抗、英国承认 | 卡西米两支边界固定。 |
| 1920—1930年代 | 珍珠业崩溃 | 养殖珍珠、大萧条与信贷收缩叠加 | 传统财政和社会结构陷入危机。 |
| 1952年 | 富查伊拉获承认、停战诸国委员会成立 | 英国为行政和石油特许重组安排 | 七酋长政治地图和协商机制成形。 |
| 1952—1955年 | 布赖米危机 | 沙特进入、仲裁失败、武力驱逐 | 阿莱因—布赖米实际分界形成。 |
| 1958／1962年 | 阿布扎比发现并出口石油 | 乌姆谢夫发现，随后商业生产 | 财政与权力结构根本改变。 |
| 1966／1969年 | 迪拜发现并出口石油 | 法塔赫油田投产 | 为港口和服务业多元化提供资本。 |
| 1968年 | 英国宣布撤军、扎耶德—拉希德会谈 | 从两酋长联合扩大到九方谈判 | 联邦成为安全与发展方案。 |
| 1971年 | 九方失败、六酋长建国 | 巴林、卡塔尔独立，哈伊马角暂缓 | 阿联酋于12月2日成立。 |
| 1972年2月10日 | 哈伊马角加入 | 接受联邦宪制 | 七酋长联邦完成。 |

## 体系形成与终结原因

### 形成条件

- 英国需要以较低成本保护印度航线和海湾贸易。
- 地方酋长需要外部海上安全，同时保留内部统治。
- 珍珠经济依赖可预测的采珠季和港口信用。
- 英国选择性承认使分散的港口与部落领地逐渐固定为较少数酋长国。

### 衰落因素

- 英国全球收缩和财政压力使保护承诺难以延续。
- 石油收入提高地方国家能力，也放大安全、边界和发展协调需求。
- 小酋长国单独面对沙特、伊朗、阿曼及海湾战争风险的能力有限。
- 阿布扎比资源与迪拜商业网络提供了可行的联邦财政核心。

### 直接终结

英国在1971年12月1日终止特殊条约关系。次日六个酋长国以临时宪法建立主权联邦，原“停战诸国”不再作为国际政治单位存在。
