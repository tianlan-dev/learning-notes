# 英国影响、国家整合与现代阿曼

## 时间

1913年至今

## 概括

20世纪阿曼的国家形成不能简单写成一条“苏丹逐步现代化”的直线。1913—1959年，马斯喀特沿海苏丹国与尼兹瓦内陆伊巴德派伊玛目政权实际并立；英国支持苏丹，沙特阿拉伯和埃及等力量又介入内陆冲突。1959年内陆伊玛目政权被军事击败后，佐法尔从1962年起爆发另一场边缘地区叛乱。1970年卡布斯废黜父亲赛义德·本·泰穆尔，以地方发展、赦免和军事反攻相结合，1975—1976年结束战争，并利用石油收入建立全国行政体系。

2020年海赛姆继位后，阿曼没有改变苏丹掌握最终行政与政治权力的基本格局，但通过2021年《国家基本法》首次明确王储机制，并以财政整顿、经济多元化和“2040愿景”回应后石油时代压力。

阿曼近现代并非单纯的苏丹父子继承史：沿海苏丹、内陆伊玛目、英国顾问、部落与佐法尔武装曾分别掌握不同权力。世系与政府首脑需结合实际控制范围阅读。

## 沿海苏丹国与内陆伊玛目国

### 1913—1920年：伊玛目制复兴与战争

1913年，内陆学者和部落推举萨利姆·本·拉希德·哈鲁西为伊玛目，反对马斯喀特对英国的依赖以及苏丹对内陆的征税和任命。伊玛目联盟夺取尼兹瓦等地，1915年进攻马斯喀特门户拜特法拉杰，但被苏丹军和英属印度部队击退。此后双方均无力彻底消灭对方。

萨利姆于1920年7月遇刺后，穆罕默德·本·阿卜杜拉·哈利利获推举。同年9月，在英国政治代表斡旋下签订《锡卜条约》：内陆部落不攻击沿海，苏丹方面不干预内陆事务，双方保持人员与贸易往来。条约没有用现代国际法清楚规定两个主权国家，而是形成“苏丹名义框架含糊、内陆实际自治明确”的妥协。

### 1920—1954年：双重权力的稳定期

穆罕默德·哈利利依靠宗教学者、瓦利和部落首领治理尼兹瓦及周边内陆；苏丹控制马斯喀特、巴提奈和若干港口，并依赖英国顾问、贷款和军事力量。两边不是完全隔绝：商贸、部落和亲属网络跨越边界，但税收、司法、官员任命和对外关系各有中心。

### 1954—1959年：杰贝勒阿赫达尔战争

1954年穆罕默德·哈利利去世，加利卜·本·阿里·希奈获推举为伊玛目。石油勘探和布赖米绿洲争端使“谁能授予内陆石油特许权”成为主权冲突核心。1955年，苏丹军在英国军官指挥和空中支持下进入尼兹瓦，加利卜退位或暂时停止抵抗；其弟塔利卜流亡后于1957年携武装返回，伊玛目阵营重新控制杰贝勒阿赫达尔高地。

英国空军轰炸补给点，苏丹军长期无法攻上高原。1959年1月，英国特种空勤团攀登防守薄弱的山崖并突破阵地，加利卜、塔利卜等人流亡沙特。1959年意味着伊玛目政权失去领土和行政控制；加利卜此后继续在海外保留伊玛目主张，直到2009年去世，但不应写成其继续统治阿曼。

| 层次 | 战争原因与结果 |
|---|---|
| 结构因素 | 沿海—内陆双重权力、《锡卜条约》主权含糊、伊巴德派自治传统。 |
| 资源因素 | 石油特许权把原本可妥协的税收与任命争议升级为领土主权问题。 |
| 外部压力 | 英国支持苏丹；沙特和埃及支持伊玛目阵营，冲突进入阿拉伯冷战背景。 |
| 直接触发 | 1954年伊玛目更替和苏丹军进入内陆。 |
| 结果 | 1959年伊玛目政权失去领土控制，苏丹取得北部内陆，但国际争议和地方不满并未立即消失。 |

## 佐法尔战争与1970年政变

### 叛乱的形成

佐法尔与北部阿曼隔着广阔荒漠，拥有独特部落、语言和季风农业传统。赛义德·本·泰穆尔实行严厉财政紧缩、限制旅行与教育，南部公共服务稀少。1962年起出现武装袭击，1965年佐法尔解放阵线正式组织化。1967年南也门建立激进政权后，叛军获得训练、武器和后方基地，并逐渐转为马克思主义主导的阿拉伯湾人民解放阵线；到1970年前后，政府在佐法尔山地处于劣势。

### 1970年宫廷政变

石油于1964年发现商业储量、1967年开始出口，但赛义德仍限制开支和行政扩张。英国官员、军方和王族越来越认为旧统治无法同时应对叛乱与国家建设。1970年7月23日，卡布斯在英国支持下废黜其父，赛义德流亡海外。这既是王朝内部权力转移，也是对财政封闭、社会停滞和军事危机的直接回应。

### 改革与军事反攻

- 卡布斯宣布赦免，吸收投诚者组成“菲尔卡特”地方部队，以部落和宗教网络争取山地人口。
- 政府迅速建设道路、井、诊所、学校和行政站，削弱叛军以边缘化动员的基础。
- 1972年米尔巴特战斗阻止叛军突袭重要据点，成为战争转折之一。
- 英国顾问、特种部队和空军继续支援；伊朗自1973年起派兵，约旦和其他盟友也提供协助。
- 政府军建立封锁线、切断来自南也门的补给，并逐步夺回山地。1975年12月主要军事抵抗被击败，1976年官方宣布战争结束。

| 层次 | 佐法尔冲突 |
|---|---|
| 结构因素 | 南部长期边缘化、地方身份强、交通和公共服务缺失。 |
| 外部压力 | 南也门的基地和援助、地区革命运动与冷战竞争。 |
| 直接触发 | 旧政权压制和1960年代武装组织化。 |
| 政府胜因 | 赦免、地方发展和政治吸纳与外援、情报、空中力量、地面封锁结合；不能只归因于军事镇压。 |
| 长期影响 | 中央行政深入佐法尔，军队和福利建设成为国家整合工具，但政治权力仍高度集中。 |

## 卡布斯时期：国家建设与外交

1. **行政整合**：1970年后废除旧有封闭限制，统一国名为阿曼苏丹国，建立部委、省级行政、常备军和全国财政体系。
2. **公共服务**：石油收入用于公路、港口、电力、供水、学校和医院，北部内陆与佐法尔被纳入同一行政网络。
3. **国际加入**：1971年加入阿拉伯联盟和联合国，1981年参与建立海湾合作委员会。
4. **制度化**：协商会议逐步扩大选举基础，1996年颁布首部《国家基本法》，并设立任命制国家委员会，组成两院制阿曼委员会。
5. **社会契约**：国家提供就业和福利，推广“阿曼化”以提高本国人在私营部门就业，但对石油收入和外籍劳工的依赖持续。
6. **2011年抗议**：苏哈尔等地围绕就业、工资、腐败和官员问责出现抗议。政府更换部分大臣、增加岗位和福利并扩大协商机构权限，同时也实施逮捕和安全管控。
7. **平衡外交**：阿曼维持与海合会邻国、伊朗、英国和美国的关系，避免卷入最激烈的阵营对抗，并在伊核问题、也门等议题上提供沟通渠道。
8. **继承问题**：卡布斯无子女，也未公开指定王储。1996年制度规定王族若不能迅速一致推举继承人，则开启苏丹密封遗书；2020年王族直接接受遗书所列海赛姆，完成和平交接。

## 海赛姆时期与现行统治结构

### 2020—2026年的主要变化

- 2020年1月11日海赛姆继位，保留平衡外交和“2040愿景”，同时重组部委、国有企业和投资管理。
- 2021年新《国家基本法》确定王位在苏丹男性后裔中按长子继承原则传递，并正式设立王储。齐亚赞·本·海赛姆成为首位明确王储，减少了卡布斯时期秘密遗书机制的不确定性。
- 政府推行财政中期计划、增值税、补贴调整和债务控制，并通过阿曼投资局整合部分国有资产；改革目标是在维持社会稳定的同时降低油气依赖。
- 就业、青年人口、私营部门竞争力、水资源、气候风险及油气价格波动仍是长期约束。
- 2026年进入第十一个五年计划阶段，政策重点继续围绕产业多元化、物流、旅游、制造业、可再生能源、数字化和社会保障。

### 截至2026年7月13日的正式与实际权力

| 机构或人物 | 正式地位 | 实际权力与限制 |
|---|---|---|
| **苏丹海赛姆·本·塔里克** | 国家元首、首相、武装力量最高统帅 | 主持内阁，拥有最终任免、法令、国防、外交和国家战略决定权，是权力核心。 |
| **王储齐亚赞·本·海赛姆** | 王位继承人、负责经济事务的副首相 | 统筹和监督部分经济政策并主持相关委员会，但不与苏丹共治，授权范围来自苏丹。 |
| 国防事务副首相希哈卜·本·塔里克 | 协助处理国防事务 | 在安全与国防协调中地位重要，仍受苏丹最高统帅权统辖。 |
| 大臣会议 | 最高行政执行机构 | 制定建议、协调部委并执行苏丹决定；不对议会产生的多数政府负责。 |
| 阿曼委员会 | 国家委员会为任命制，协商会议为选举制 | 审议法律、预算和公共政策，监督和质询能力较过去增加，但法案和重大政治方向最终受苏丹权力约束。 |
| 司法、各省与国家经济机构 | 法院、总督体系、阿曼投资局等 | 承担专业执行和资源配置；高层任命与制度边界由王室法令确定。 |

2026年王室分工还包括比拉拉布·本·海赛姆担任国务大臣兼马斯喀特省长。王族成员进入经济、国防和地方治理岗位，显示海赛姆在明确继承顺序的同时，也通过职务分工培养下一代统治团队。

## 延续条件与风险

现代阿曼尚不存在“衰亡过程”，更适合区分政权延续条件和潜在风险：

| 类型 | 内容 |
|---|---|
| 延续条件 | 布赛义德王朝合法性、伊巴德派宽容传统、全国公共服务、专业外交、军队整合和明确继承制度。 |
| 结构风险 | 权力高度集中、代表机构权限有限、青年就业压力、地区与代际期望差异。 |
| 经济风险 | 油气财政依赖、债务与福利平衡、外籍劳工结构、能源转型。 |
| 外部风险 | 霍尔木兹海峡紧张、也门冲突、海合会与伊朗关系波动。 |
| 政策回应 | “2040愿景”、财政改革、社会保障、产业多元化及王储承担经济协调职责。 |
