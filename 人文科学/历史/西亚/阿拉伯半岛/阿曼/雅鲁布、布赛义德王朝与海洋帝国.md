# 雅鲁布、布赛义德王朝与海洋帝国

## 时间

1624—1913年

## 概括

1624年建立的雅鲁布伊玛目国把伊巴德派推举传统、部落联盟、堡垒体系和海军建设结合起来，1650年收复马斯喀特，继而向东非反攻葡萄牙据点。其海上力量在赛义夫·本·苏尔坦一世时期达到高峰，但1718年以后未成年继承、复位与多名伊玛目并立引发内战，波斯军队乘机占领沿海要地。苏哈尔总督艾哈迈德·本·赛义德在1744年获推举、1749年成为无争议统治者，建立布赛义德王朝。

19世纪前半叶，赛义德·本·苏尔坦把马斯喀特、桑给巴尔和东非港口编入一个以舰队、关税、印度商人信贷、象牙、丁香和奴隶劳动维系的海洋政权。1856年后的王位争议和1861年英方仲裁使马斯喀特与桑给巴尔分立；失去东非财政后，阿曼沿海苏丹更依赖英国援助，同时难以控制内陆伊巴德派部落。

雅鲁布兼具伊玛目选举传统与家族继承趋势，后期争位不能硬排成稳定父子线；布赛义德则逐步形成世袭王朝。完整伊玛目、苏丹及并立者由世系专表维护。

## 王朝世系

雅鲁布后期存在复位、并立伊玛目和非王朝候选人；布赛义德早期又有“鲁斯塔克名义伊玛目—马斯喀特实际统治者”分离。为避免把这些权力线压成单一继承表，完整次序、共治和桑给巴尔分支另见[阿曼近世伊玛目与布赛义德苏丹世系表](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A/%E9%98%BF%E6%8B%89%E4%BC%AF%E5%8D%8A%E5%B2%9B/%E9%98%BF%E6%9B%BC/%E9%98%BF%E6%9B%BC%E8%BF%91%E4%B8%96%E4%BC%8A%E7%8E%9B%E7%9B%AE%E4%B8%8E%E5%B8%83%E8%B5%9B%E4%B9%89%E5%BE%B7%E8%8B%8F%E4%B8%B9%E4%B8%96%E7%B3%BB%E8%A1%A8.md)。

## 雅鲁布的建立与崛起

### 1624—1650年：先统一内陆，再夺回港口

- 纳西尔·本·穆尔希德在鲁斯塔克获推举后，首先制服或争取山地、尼兹瓦、巴赫拉等地方力量，并修复堡垒、组织税收。
- 他没有立即正面进攻马斯喀特，而是逐步切断葡萄牙的内陆支持，争夺苏哈尔等港口；1643年前后苏哈尔落入雅鲁布势力。
- 1649年纳西尔去世后，苏尔坦·本·赛义夫一世继位。1650年雅鲁布军夺取马斯喀特，葡萄牙在阿曼核心海岸的统治结束。
- 胜利来自内陆粮税和兵源、地方商人的海贸利益、堡垒围攻与新建舰队的结合，并非一次孤立的港口起义。

### 1650—1711年：印度洋反攻与鼎盛

- 阿曼舰队袭击波斯湾、印度西岸和东非的葡萄牙航路，试图把港口关税重新纳入本地统治。
- 1696—1698年，阿曼围攻并攻克蒙巴萨耶稣堡。此后蒙巴萨、桑给巴尔等地逐渐进入阿曼影响圈，但当地统治仍依赖驻军、总督、商人和斯瓦希里精英协商，不能理解为现代式连续领土占领。
- 赛义夫·本·苏尔坦一世投资法拉吉、椰枣园、堡垒、学校和舰队，使农业剩余与海关收入互相支撑。
- 海上扩张削弱葡萄牙在西印度洋的垄断，却没有消除东非地方王族、城市精英与阿曼总督之间的冲突。

### 1711—1749年：继承内战与波斯干预

1718年苏尔坦·本·赛义夫二世去世后，年幼的赛义夫·本·苏尔坦二世被拥立。伊巴德派“应推举有能力者”的原则与雅鲁布家族的事实世袭发生正面冲突。穆罕纳、雅鲁布·本·比拉拉布、穆罕默德·本·纳西尔、比拉拉布·本·希姆亚尔和苏尔坦·本·穆尔希德先后或同时获得不同部落支持，赛义夫二世本人四次复位。

1728年穆罕默德·本·纳西尔在苏哈尔附近战死后，赛义夫二世主要控制沿海，比拉拉布·本·希姆亚尔控制内陆。为打破僵局，赛义夫二世于1737年邀请纳迪尔沙的波斯军队介入；外军劫掠和占领反而加深危机。1742—1743年新一轮波斯远征夺取马斯喀特堡垒。苏哈尔总督艾哈迈德·本·赛义德抵抗围城、利用补给与谈判迫使波斯军撤退，1744年获一部分学者和部落推举为伊玛目；直到1749年比拉拉布·本·希姆亚尔战败身亡，他才成为无争议统治者。

## 布赛义德王朝与海洋帝国

### 1744/1749—1806年：重建中央与转向马斯喀特

- 艾哈迈德整合敌对部落、重建舰队和商贸，1775年还派船队协助解除波斯军对巴士拉的围困。
- 1783年后，其子赛义德·本·艾哈迈德在鲁斯塔克保留伊玛目名号，实际权力先转到其子哈马德，再转到叔父苏尔坦·本·艾哈迈德；政治中心由内陆鲁斯塔克转向马斯喀特。
- 苏尔坦·本·艾哈迈德面对卡瓦西姆海上力量、瓦哈比—沙特扩张和法国—英国竞争，1798年与英国缔结友好安排，1800年英国政治代表常驻马斯喀特。
- 1804年苏尔坦战死后，巴德尔·本·赛义夫以摄政者身份掌权并借助沙特势力；1806年赛义德·本·苏尔坦将其杀死，结束摄政。

### 1806—1856年：赛义德·本·苏尔坦的双中心帝国

赛义德重建海军，压制或收编沿海对手，巩固桑给巴尔、奔巴、蒙巴萨及东非若干港口的贡赋和驻军体系。约1840年以后，他长期驻留桑给巴尔，使其成为与马斯喀特并列、事实上更富庶的政治中心。

帝国经济有四个互相连接的支柱：

| 支柱 | 运作方式 | 影响 |
|---|---|---|
| 港口关税 | 马斯喀特、桑给巴尔等港口向区域贸易征税 | 提供舰队、宫廷和驻军财政。 |
| 印度商人网络 | 信贷、报关、批发和远程结算 | 增强跨洋贸易，也使统治者依赖商人融资。 |
| 象牙与内陆商路 | 从东非内陆经商队输往海岸 | 推动海岸—内陆交换，也伴随暴力和掠夺。 |
| 丁香种植和奴隶劳动 | 桑给巴尔、奔巴大种植园出口丁香 | 创造高额收益，同时建立在奴隶贸易和强制劳动之上。 |

1833年阿曼与美国缔结商贸条约，之后又与英国、法国等国建立外交联系。所谓“阿曼帝国”并不是对每一段海岸实行同样行政管理：马斯喀特王廷直接控制核心港口，对远处城市则常依靠总督、驻军、贡赋、商人社群和地方盟约。

### 1856—1913年：分立、收缩与英国影响

- 1856年赛义德去世，其子苏韦尼控制马斯喀特，马吉德控制桑给巴尔，双方都主张更多遗产。
- 1861年英国主持的“坎宁裁决”确认两个苏丹国分立，并规定桑给巴尔向马斯喀特支付补贴。东非收入从统一王廷财政中剥离，马斯喀特长期面临债务和军费困难。
- 1866年萨利姆·本·苏韦尼弑父夺位，1868年被伊巴德派联盟推举的阿赞·本·盖斯赶走。阿赞以伊玛目名义短暂重建沿海—内陆统一，1871年战败身亡。
- 图尔基·本·赛义德依靠王族盟友、部落和英国支持恢复马斯喀特王权；其后苏丹仍需以补贴、武器和政治交易维持统治。
- 1891年排他性条约限制苏丹未经英国同意转让领土或接受其他列强保护。阿曼并非被简单改为一个正式“英国保护国”，但外交选择、财政和军事能力受到英国强烈约束。
- 费萨尔·本·图尔基晚期，内陆部落、伊巴德派学者和反对马斯喀特对英依赖的力量重新联合，1913年另立伊玛目。

## 鼎盛条件

- **政治整合**：雅鲁布早期用推举合法性协调部落，布赛义德早期则利用反波斯声望重建秩序。
- **陆海互补**：绿洲农业、法拉吉和内陆兵源支撑港口，海关与远程贸易反过来支付舰队和堡垒。
- **海权真空**：葡萄牙衰退、波斯湾政局动荡，为阿曼舰队扩张提供窗口。
- **商人和侨民网络**：阿拉伯、印度、波斯和斯瓦希里商人提供信贷、航海知识和地方中介。
- **双中心调度**：赛义德·本·苏尔坦能在马斯喀特与桑给巴尔之间调配舰队、关税和王族成员。

## 衰落与终结原因

### 雅鲁布政权

| 层次 | 因素 |
|---|---|
| 结构因素 | 伊玛目选举原则与家族世袭化冲突；部落阵营、沿海与内陆利益分化。 |
| 治理因素 | 未成年继承、频繁废立和复位使税收、军队与港口指挥割裂。 |
| 外部压力 | 波斯军队以援助一方为名进入阿曼，转而占领要地。 |
| 直接触发 | 1742—1743年波斯占据马斯喀特，艾哈迈德在苏哈尔形成独立权力中心；1749年最后对手败亡。 |

### 布赛义德海洋帝国

| 层次 | 因素 |
|---|---|
| 结构因素 | 领地跨海、统治方式不一，且没有固定且被普遍接受的继承规则。 |
| 经济因素 | 舰队和宫廷成本高；经济高度依赖港口关税、丁香、象牙和奴隶劳动。 |
| 外部压力 | 英国海权、废奴政策和条约体系压缩自主空间，欧洲蒸汽海军改变力量对比。 |
| 直接触发 | 1856年赛义德无明确统一继承安排而去世，两个儿子分别掌握马斯喀特和桑给巴尔；1861年仲裁把事实分裂制度化。 |
| 长期后果 | 马斯喀特失去东非税源后债务加深，依赖英国补贴与军事支持，内陆反对力量复兴。 |
