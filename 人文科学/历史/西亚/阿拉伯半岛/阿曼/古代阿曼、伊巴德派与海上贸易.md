# 古代阿曼、伊巴德派与海上贸易

## 时间

古代-17世纪初

## 概括

阿曼山地拥有铜矿、绿洲和可防御的内陆聚落，沿海港口则面向阿拉伯海与印度洋季风航线。古代两河文献中的“马干”通常与今阿曼及邻近地区相联系。伊斯兰化后，阿曼较早接受伊巴德派传统，形成伊玛目、学者和部落协商并存的政治结构；马斯喀特等港口又持续吸引波斯、印度和东非商人。

## 统治结构

| 阶段 | 主要结构 | 说明 |
|---|---|---|
| 马干与古代聚落 | 矿业、绿洲和港口网络 | 铜及海运连接两河与印度洋。 |
| 伊斯兰初期 | 总督、地方部落和商人 | 阿曼纳入早期伊斯兰世界，但地方自治传统较强。 |
| 伊巴德派伊玛目制 | 伊玛目、宗教学者和部落协商 | 伊玛目原则上凭资格和推举取得合法性，并非单纯世袭。 |
| 纳卜哈尼王朝 | 12-17世纪 | 世袭王朝与伊玛目传统并存，内陆和沿海权力分散。 |
| 葡萄牙港口体系 | 1507-1650年 | 葡萄牙控制马斯喀特等战略港口，但未统治整个内陆。 |

## 重要事件

- 铜矿与海运使古代阿曼连接苏美尔、迪尔蒙、伊朗南岸和印度河方向。
- 7世纪阿曼接受伊斯兰，8世纪形成早期伊巴德派伊玛目制。
- 阿曼船只和商人活跃于波斯湾、印度西岸和东非。
- 葡萄牙于1507年占领马斯喀特，以堡垒和舰队控制海路节点。
- 内陆部落与伊巴德派政治传统为17世纪雅鲁布王朝的动员提供基础。
