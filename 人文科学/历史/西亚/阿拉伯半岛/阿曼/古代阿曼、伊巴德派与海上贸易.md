# 古代阿曼、伊巴德派与海上贸易

## 时间

约前3000年—1624年

## 概括

阿曼古代史不是一个连续王朝的故事，而是铜矿、绿洲农业、季风航海、部落联盟和港口竞争不断重新组合的过程。哈杰尔山地提供铜矿和可防御聚落，法拉吉灌溉支撑内陆绿洲，马斯喀特、苏哈尔、盖勒哈特等港口则把内陆物产接入两河、伊朗、印度和东非。伊斯兰化以后，伊巴德派为这种分散社会提供了推举伊玛目、缔结效忠和约束统治者的政治语言，但伊玛目制并未始终连续存在。

## 地理、生产与航路基础

| 要素 | 作用 | 历史限制 |
|---|---|---|
| 哈杰尔山地铜矿 | 支撑青铜时代冶炼和对外交换 | “马干”大体对应今阿曼及邻近阿联酋，不能直接等同现代国界。 |
| 巴提奈沿海与苏哈尔等港口 | 连接波斯湾、阿拉伯海和印度洋季风航线 | 港口控制不等于对山地和绿洲的全面统治。 |
| 尼兹瓦、巴赫拉等内陆绿洲 | 依靠法拉吉灌溉形成农业、宗教和部落中心 | 水权与土地分配也会加剧地方竞争。 |
| 佐法尔 | 乳香、季风草场和面向亚丁湾的航路 | 与北部阿曼的政治整合长期较弱。 |
| 海商网络 | 阿拉伯、波斯、印度和东非商人共同维持转口贸易 | 海上繁荣受霍尔木兹、葡萄牙等外部海权制约。 |

考古所见的乌姆安纳尔文化、巴特—哈特姆—艾因遗址群和大型墓葬，说明前3—前2千纪已经存在跨绿洲协作与远程交换。前1千纪以后，法拉吉灌溉的扩展提高了绿洲承载力；阿契美尼德、帕提亚和萨珊势力在不同时期影响或控制部分海岸节点，但不能概括为对全境持续不间断的直接统治。

## 伊斯兰化的具体过程

1. **约630年接受伊斯兰**：先知穆罕默德派阿姆鲁·本·阿斯前往阿曼，朱兰达家族的杰伊法尔和阿卜德兄弟接受伊斯兰，并在地方继续掌权。阿曼因而主要通过谈判和政治归附进入早期伊斯兰共同体。
2. **632年后的里达战争**：先知去世后，迪巴一带发生反叛与拒缴宗教税的冲突。麦地那政权和忠于朱兰达家族的力量镇压反叛，显示阿曼的归附并非一次完成。
3. **总督统治与地方自治并存**：倭马亚、阿拔斯等哈里发王朝曾派遣总督或发动远征，但山地、绿洲和海上距离使中央控制常呈间歇性。
4. **伊巴德派传播**：8世纪巴士拉伊巴德派学者网络把教法与政治思想传入阿曼；贾比尔·本·宰德、阿布·乌拜达等人物的学术传统，为阿曼地方社群提供组织资源。
5. **早期独立伊玛目制**：朱兰达·本·马苏德约749—750年获推举为伊玛目，约751年在阿拔斯干预中战死。793年以后又建立较持久的伊玛目政权，直到893年阿拔斯军事介入和内部冲突使之瓦解。
6. **反复复兴**：此后数世纪，伊玛目会在部落联盟能够达成妥协时复兴，又会因继承、宗派、区域和部落冲突而中断。它是一种可被重新启动的政治制度，而不是一条无间断世袭王朝。

## 伊巴德派伊玛目制的机制

| 环节 | 原则 | 实际运行 |
|---|---|---|
| 产生 | 由具资格的学者和地方领袖协商推举，并以效忠宣誓确认 | 强大部落联盟的支持往往决定推举能否落实。 |
| 资格 | 重视宗教学识、公正、治理和军事能力 | 候选人常来自有声望的家族，但血统本身不是充分条件。 |
| 权力 | 兼有宗教裁判、行政、税收和战争领导责任 | 港口、山地和远方部落对伊玛目的服从程度并不一致。 |
| 约束 | 理论上失德或无力履职者可被废黜 | 废立常与部落战争结合，容易造成多个候选人并立。 |
| 延续 | 通过重新推举而非自动父子继承 | 雅鲁布后期逐渐家族化，正是制度合法性危机的重要来源。 |

## 纳卜哈尼王权与间歇伊玛目

1154年以后，巴尼·纳卜汉家族以“马利克”即王的身份控制巴赫拉及部分内陆贸易路线。第一阶段大致延续至1500年，第二阶段约1500—1624年；中间和后期多次出现伊玛目复兴、地方部落割据和家族内部争斗。巴赫拉堡及其绿洲体系体现了王权对灌溉、农业和商路的组织能力。

纳卜哈尼时期没有一份能被现代研究一致接受、覆盖五个世纪且年代连续的完整君主表。较明确的节点包括首期建立者法拉赫·本·穆赫辛、15世纪诗人王苏莱曼·本·苏莱曼·本·穆扎法尔，以及1624年前后被雅鲁布联盟取代的晚期纳卜哈尼统治者。与其用推测补齐空白，更应把这个时期理解为“家族王权占优势，但并非始终统一全境”的两阶段结构。

## 霍尔木兹、葡萄牙与沿海据点

- 15世纪以前，盖勒哈特、马斯喀特等港口已受霍尔木兹王国的贡赋、贸易和政治影响。
- 1507年阿方索·德·阿尔布克尔克率葡萄牙舰队袭取马斯喀特，并把港口纳入印度洋堡垒—关税体系。
- 1515年葡萄牙控制霍尔木兹后，其对阿曼港口的影响进一步制度化；马斯喀特堡垒和舰队用于补给、护航和征收海上贸易收益。
- 1552年奥斯曼舰队一度攻取马斯喀特，随后还有多次袭击，但未建立稳定内陆统治。
- 葡萄牙始终主要控制战略港口和海路节点。内陆绿洲、部落与伊巴德派网络保留自主空间，成为后来统一和反攻的政治纵深。
- 1624年纳西尔·本·穆尔希德在鲁斯塔克获推举为伊玛目，先整合内陆和争夺苏哈尔等港口，再由继任者完成对马斯喀特的收复。

## 重要事件

| 时间 | 事件 | 过程与影响 |
|---|---|---|
| 约前3千纪 | 马干铜进入两河交换体系 | 矿产、冶炼和海运形成跨区域经济链。 |
| 约前2500—前2000年 | 乌姆安纳尔文化和大型墓葬发展 | 显示绿洲之间已有组织化协作和社会分层。 |
| 前1千纪 | 法拉吉灌溉扩展 | 内陆农业和定居聚落增长，水权成为政治资源。 |
| 约630年 | 朱兰达兄弟接受伊斯兰 | 地方统治者经谈判归附，减少了全面征服成本。 |
| 632—633年 | 迪巴冲突 | 早期哈里发权力与地方反叛发生军事碰撞。 |
| 约749—751年 | 第一位清楚可考的独立伊巴德派伊玛目兴亡 | 朱兰达·本·马苏德的失败显示阿拔斯扩张压力。 |
| 793—893年 | 较持久的伊玛目政权 | 学者—部落联盟建立行政秩序，最终因内部分裂与外部干预瓦解。 |
| 1154年 | 纳卜哈尼王权兴起 | 世袭家族王权取代持续的选举伊玛目统治。 |
| 1507年 | 葡萄牙夺取马斯喀特 | 阿曼沿海卷入葡萄牙与霍尔木兹、奥斯曼的海权竞争。 |
| 1624年 | 纳西尔·本·穆尔希德获推举 | 分散的内陆力量开始在雅鲁布伊玛目领导下统一。 |

## 兴起、衰落与政权转换原因

| 类型 | 伊巴德派伊玛目制得以反复复兴 | 纳卜哈尼和葡萄牙体系为何被取代 |
|---|---|---|
| 结构因素 | 山地—绿洲社会分散，需用协商和宗教合法性协调部落；学者网络能跨地方维持规范。 | 纳卜哈尼内部家族斗争、港口与内陆分裂；葡萄牙依赖少数堡垒和海军，缺乏内陆社会基础。 |
| 经济与军事因素 | 绿洲税收、商路和港口收入可支撑获得广泛效忠的伊玛目。 | 葡萄牙封锁和关税损害地方商人利益；内陆为反攻提供人口、粮食和堡垒纵深。 |
| 外部压力 | 哈里发、波斯和海上强权干预常使伊玛目政权中断。 | 霍尔木兹衰落、葡萄牙资源分散，奥斯曼和欧洲竞争削弱其垄断。 |
| 直接触发 | 危机时期的部落妥协会重新启动推举。 | 1624年纳西尔获得跨部落效忠，建立可持续征税和作战的统一联盟。 |
