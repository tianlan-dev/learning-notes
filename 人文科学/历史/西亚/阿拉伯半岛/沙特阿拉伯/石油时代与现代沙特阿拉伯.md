# 石油时代与现代沙特阿拉伯

## 时间

1932年至今（现状核验截至2026年7月13日）

## 概括

现代沙特阿拉伯以沙特家族君主制、两圣地管理和石油财政为制度核心。1932年建国时，王国仍依赖朝觐、海关、地方征收与外部贷款；1938年达曼7号井商业产油、1939年首次出口后，国家获得前所未有的财政能力。石油收入支持官僚机构、基础设施、教育、医疗和福利，也形成对能源周期、外籍劳动力和安全伙伴的长期依赖。

王国并非静态“传统君主制”。沙特—费萨尔权力斗争、1979年宗教与地区冲击、1990年海湾战争、2001年后反恐、2011年地区动荡和2015年后的集中化改革，反复调整王室、宗教学者、官僚、安全机构和社会之间的关系。2016年后2030愿景加快非石油产业、公共投资、旅游和社会开放；与此同时，重大决策更集中于王储兼首相穆罕默德·本·萨勒曼及其领导的机构。

## 统治结构

| 层级 | 组成与职能 | 权力边界 |
|---|---|---|
| 国王与王储 | 国王为国家元首和各权力最终参照；王储负责继承并承接国王委托事项 | 不是议会制下的礼仪君主；国王保留王令、任免和最终批准权。 |
| 首相与内阁 | 首相领导政府；内阁制定并执行内政、外交、财政、经济、教育和国防政策 | 内阁成员由王令任免，决议需国王认可；2022年后首相不再必然由国王兼任。 |
| 战略委员会 | 经济与发展事务委员会、政治与安全事务委员会统筹重大跨部门事务 | 由王储领导，增强政策集中和执行协调。 |
| 协商会议 | 主席和150名成员均由国王任命，审议法规、计划、条约和部门报告 | 有建议和审议功能，不产生对王室负责的民选政府。 |
| 宗教与司法机构 | 高级宗教学者、宗教事务机构及普通和行政法院 | 沙里亚仍是法源核心；成文法、专业法院和行政规则持续扩展。 |

| 国有经济体系 | 财政部、能源部、沙特阿美和公共投资基金等 | 石油收入仍决定财政空间；公共投资基金成为多元化和大型项目的重要工具。 |

## 历史过程

### 建国、财政困境与石油突破（1932—1953年）

阿卜杜勒阿齐兹统一的领地在行政传统、社会结构和财政来源上差异很大。利雅得王权以王子担任地方长官，在汉志保留并改造部委、协商和海关制度；朝觐收入却受世界经济萧条影响，中央财政长期紧张。1933年政府向美国加利福尼亚标准石油公司授予石油特许权，1938年达曼7号井显示商业价值，1939年首批原油出口。第二次世界大战限制早期开发，战后产量和收入才快速增加。

1944年企业更名为阿拉伯美国石油公司。1945年阿卜杜勒阿齐兹与美国总统罗斯福在“昆西”号会面，象征安全与能源关系上升，但双边关系并非一份固定不变的“石油换安全”条约。1950年政府取得石油利润五五分成安排，财政能力大幅提高；1952年建立货币管理机构，1953年创建内阁，中央国家由王室家政式管理逐步转向部委官僚制。

### 沙特—费萨尔权力斗争与国家整顿（1953—1975年）

沙特国王继位后扩大教育、医疗和基础设施，但宫廷支出、项目管理和收入波动造成严重财政赤字。王储费萨尔1954—1960年任首相，试图整顿财政；双方在预算、任命和外交路线上的冲突导致首相职位数度易手。1962年费萨尔重掌政府，提出改革方案；1964年王族主要成员和高级宗教学者支持废黜沙特，费萨尔即位。这不是普通父子继承，而是王室共识、宗教认可和国家财政危机共同推动的宫廷废立。

费萨尔冻结不受控制的支出，强化内阁和专业部委，扩张男女教育并在1962年正式废除奴隶制。他在也门内战中支持王党，同纳赛尔主义埃及竞争；1967年战争后又以财政援助支持埃及和约旦。1973年第四次中东战争期间，沙特参与阿拉伯产油国减产与对特定国家禁运，油价上涨显著增强财政和外交影响。1975年费萨尔被侄子刺杀，哈立德国王继位。

### 石油繁荣、宗教转折与海湾安全（1975—2005年）

1970年代后期巨额石油收入支持道路、港口、大学、医院、新城和补贴体系，公民就业和福利更依赖国家，外籍劳工成为建设与服务业支柱。政府1973年取得阿美25%股权、1974年增至60%，1980年实现完全所有；1988年改组为沙特阿美。

1979年出现三重冲击：伊朗革命挑战地区领导与君主制安全；朱海曼·欧泰比集团占领麦加禁寺，王国经两周战斗夺回；东部省什叶派抗议暴露地区与教派不平等。此后国家一方面强化宗教保守规范和宗教机构，另一方面扩大安全体系。1981年参与建立海湾合作委员会。1980年代油价下跌迫使财政收缩，显示福利和发展模式对石油周期的依赖。

1990年伊拉克吞并科威特后，沙特邀请美国及多国部队驻扎防卫，成为解放科威特的主要后方基地。决定解决了直接军事威胁，却因外国非穆斯林军队进入圣地所在国而引发国内宗教反对。1992年《基本法》《协商会议法》和《地区法》把王位、内阁、协商及地方行政制度成文化。1995年法赫德中风后，王储阿卜杜拉承担多数日常国务，呈现“国王在位、王储摄政”的实际分工。

15名9·11袭击者为沙特公民使王国受到巨大国际压力。2003—2006年基地组织在沙特发动袭击，政府通过安全行动、情报合作、宗教再教育和金融监管压制其国内网络。安全合作加强的同时，关于宗教教育、政治参与和人权的争论延续。

### 阿卜杜拉时期的渐进改革（2005—2015年）

阿卜杜拉正式即位后扩大高等教育和海外奖学金，建立阿卜杜拉国王科技大学，推进经济城和司法改革。2005年开始有限的市政选举；2013年女性进入协商会议，2015年女性首次参加市政选举并可当选。2006年效忠委员会被制度化，意在管理建国者诸子之后的王位继承，但国王在任命和王室政治中仍居核心。

2011年阿拉伯起义期间，政府以住房、工资和就业支出回应国内压力，同时禁止街头抗议，并支持海湾合作委员会部队进入巴林。区域政策在伊朗、叙利亚、也门和穆斯林兄弟会问题上趋于安全化。高福利、青年就业、住房供给、私营部门对外籍劳力的依赖以及油价波动，构成继任者必须处理的结构问题。

### 萨勒曼与穆罕默德·本·萨勒曼时期（2015年至今）

萨勒曼2015年即位后取消和合并多个委员会，建立经济与发展事务委员会、政治与安全事务委员会。其子穆罕默德先任国防大臣和副王储，2017年取代穆罕默德·本·纳伊夫成为王储，标志王位继承首次明确转向建国者之孙一代。同年利雅得丽思卡尔顿酒店拘押行动以反腐名义处理王子、官员和商人，也显著压缩传统王室支系的独立权力。

2015年沙特主导干预也门内战，试图阻止胡塞武装巩固政权；战争造成长期人道危机、跨境导弹与无人机威胁，军事目标并未迅速实现。2017—2021年沙特、阿联酋、巴林和埃及封锁卡塔尔，最终以欧拉和解结束。2018年记者贾迈勒·卡舒吉在伊斯坦布尔沙特领馆遇害，引发严重国际危机，并使权力集中与问责问题受到持续关注。

2016年公布的2030愿景以减少石油依赖、扩大私营部门、发展旅游娱乐、提高女性就业和重塑政府绩效为目标。女性2018年获准驾驶，监护制度的若干行政限制随后放宽；电影院、演出和国际旅游迅速扩展。改革扩大生活与就业空间，但政治异议、媒体、司法透明和死刑等问题仍有争议，社会开放不等于政治权力分散。

2019年沙特阿美上市，2020年阿美收购沙特基础工业公司多数股权，公共投资基金则投资国内大型项目和全球资产。新未来城、红海旅游、利雅得建设及体育文化投资推动非石油活动，但规模、融资、执行能力、水资源、劳动力技能和油价周期都是约束。2023年沙特与伊朗在中国斡旋下恢复外交关系，显示在美国安全合作之外寻求地区降温和多向外交；同年也门停火谈判有所推进，但持久政治解决仍未完成。

## 国家延续条件与结构性风险

| 类型 | 有利条件 | 风险与约束 |
|---|---|---|
| 政治 | 王位继承目前明确、决策链集中、国家机构深入各地区 | 过度集中降低内部纠错和透明度，继承与社会协商机制仍主要在王室内部。 |
| 财政 | 低成本石油、庞大储备和国有能源资产提供投资能力 | 油价与产量波动影响预算，大型项目需要持续融资并可能挤压常规公共支出。 |
| 社会 | 教育普及、女性就业增加、娱乐旅游开放扩大社会支持面 | 青年就业、住房、地区与教派差异、外籍劳工权利仍是长期议题。 |
| 安全 | 强大安全机构、海湾与西方伙伴关系、地区调停能力 | 也门、伊朗关系、导弹无人机威胁和大国竞争可能产生外部冲击。 |
| 经济转型 | 非石油服务业、旅游、物流和公共投资增长 | 私营生产率、技能匹配、水资源和项目回报决定多元化能否持续。 |

现代王国尚未进入“衰亡”阶段，因此不套用灭亡模板。其稳定取决于能否把个人化集中决策转化为可持续制度，并在石油收入、社会期望、王室继承和地区安全之间维持新平衡。
