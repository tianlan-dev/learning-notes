# 古代南阿拉伯、绿洲与商路

## 时间

约前3千纪—7世纪初

## 概括

伊斯兰兴起以前的阿拉伯半岛不是与外界隔绝的沙漠。也门高地凭季风降水、梯田和大型水利形成农业核心，萨巴、米奈、卡塔班、哈德拉毛和希木叶尔等王国竞争乳香产区及商路；阿曼和海湾的马干、迪尔蒙连接两河、伊朗和印度河；汉志、西北与内志绿洲则依靠椰枣农业、商队、游牧交换和圣地网络维系。海路增长没有简单“终结”陆路，两者随战争、航海技术、需求和港口安全反复消长。

## 地理与经济结构

| 区域 | 生态和代表节点 | 历史作用 |
|---|---|---|
| 也门高地和沙漠边缘 | 马里卜、希卜沃、提姆纳、扎法尔；梯田、季风雨和河谷洪水 | 以灌溉农业支撑人口、寺庙、军队和铭文行政；连接乳香产地与红海。 |
| 哈德拉毛与佐法尔 | 乳香树产区、沿岸港湾和内陆谷地 | 向地中海、两河、印度和东非输出乳香、没药等高价值商品。 |
| 阿曼和东南海岸 | 铜矿、绿洲与港口；古代文献中的马干 | 青铜时代向两河输出铜石，后持续参与印度洋航行。 |
| 巴林和东部海湾 | 迪尔蒙、泰勒穆恩等不同阶段港口 | 作为两河、阿曼、伊朗和印度河之间的转运、淡水补给和珍珠中心。 |
| 汉志和西北绿洲 | 德丹、利哈延、泰马、麦加、叶斯里卜 | 商队补给、椰枣农业、文字和地方宗教中心；与纳巴泰、黎凡特相连。 |
| 中北部绿洲和沙漠边缘 | 杜马特·詹达勒、盖尔耶特·法奥等 | 连接也门、海湾、两河和叙利亚，兼具定居、游牧与王国交界性质。 |
| 红海和阿拉伯海 | 季风港口、曼德海峡和近岸航线 | 船舶可绕过部分陆路，将南阿拉伯卷入罗马、阿克苏姆、印度和萨珊竞争。 |

## 国家与社会如何运作

南阿拉伯铭文显示，“王”之外还有部族、城市、神庙、土地共同体和议事组织。统治者依靠灌溉工程、战争、贡赋、神庙祭祀和商路保护，但权力并非始终高度中央化。王号、领土和都城会变化，同一王国也可能由不同王族或共治者统治。

| 要素 | 运作方式 |
|---|---|
| 水利 | 河谷洪水经坝、闸和渠道分配；维护需跨聚落协调，工程失修会影响税粮与合法性。 |
| 神庙 | 接受奉献、管理土地和誓约，铭文记录战争、建设与朝圣；宗教机构也是经济组织。 |
| 部族和城市 | 提供兵力、劳役和商队，能与王室协商或反叛；“部落”不等于完全游牧。 |
| 商路 | 国家通过护路、关税、驿站和港口获益，也可能争夺产地和中转节点。 |
| 文字与语言 | 古代南阿拉伯字母和多种语言用于铭文；北部还有达达尼、萨法伊等书写传统。 |
| 外交 | 南阿拉伯王国同阿克苏姆、托勒密—罗马世界、纳巴泰和萨珊往来，接受影响但不是其被动附庸。 |

## 分阶段发展

### 青铜时代海湾网络

约前3千纪，南美索不达米亚文献提到马干和迪尔蒙。马干通常与今阿曼及阿联酋部分地区联系，以铜石和海运著称；迪尔蒙的核心在不同时期涉及巴林和邻近海岸，是两河—印度河贸易中介。古代地名与现代国界并非完全对应，不能把一个文献名称直接等同今日国家。

海湾贸易随印度河文明衰落、两河王朝变化和航道需求而起伏。巴林仍在后世作为珍珠、椰枣和转口节点，阿曼铜矿及港口也多次复兴。

### 南阿拉伯诸王国

前1千纪，萨巴以马里卜和塞尔瓦为中心，凭灌溉、神庙和乳香路线成为强国。米奈王国在焦夫地区发展商贸侨团，卡塔班控制贝汉谷地，哈德拉毛以希卜沃和东方产区为核心。各王国的强盛时期互有重叠，传统线性“萨巴以后依次轮替”会掩盖长期并立。

马里卜坝不是一次建成的单一现代水坝，而是多阶段堤坝、闸门和渠道系统。它把季节洪水导入大片农田，维护能力体现国家与社群协作。萨巴影响也曾跨越红海，埃塞俄比亚高地的南阿拉伯文字和文化联系反映迁徙与交流，但阿克苏姆文明不能简单称为萨巴殖民地。

### 陆海贸易重组

乳香和没药用于地中海、两河、印度等地宗教、医药和香料消费。商队从哈德拉毛、也门经汉志至加沙等港口，沿途由多个群体分段经营；红海和阿拉伯海船运则借季风连接埃及、东非和印度。罗马控制埃及后更积极进入红海贸易，前26—前25年埃利乌斯·加卢斯远征南阿拉伯失败，补给、地形和疾病显示陆上征服的困难。

1—3世纪海运增长、北部政治变化和地方战争改变乳香陆路收益，但南阿拉伯经济并非突然衰亡。农业、近海贸易和区域市场继续支撑国家，希木叶尔正是在此背景下扩张。

### 希木叶尔统一与宗教转型

希木叶尔约在前1世纪形成，至3世纪后期击败或吸收萨巴、卡塔班和哈德拉毛等主要竞争者，以扎法尔为中心统治也门大部。王号逐渐扩大为“萨巴、祖赖丹、哈德拉毛和也门诸部之王”，显示政治整合。传统多神祭祀在4世纪后明显转向一神信仰，王室倾向犹太教；基督教社群和阿克苏姆、拜占庭联系也增长。

这一转变既有宗教原因，也与红海两岸帝国竞争有关。犹太教能提供不同于基督教拜占庭—阿克苏姆和祆教萨珊的政治定位，但地方改宗速度及王室信仰细节仍有争议。

### 阿克苏姆、萨珊与古代秩序终结

523年前后，犹太教希木叶尔王祖·努瓦斯迫害奈季兰基督徒，引发阿克苏姆干预。约525年阿克苏姆军队征服也门，扶植基督教统治；阿布拉哈后来独立掌权并修建教堂、整修水利。传统“象年”叙事与具体远征年代、目标之间需区分宗教记忆和可核实铭文。

约570年代，也门地方势力向萨珊求援，波斯军队驱逐阿克苏姆统治者，建立附庸和军政殖民。萨珊在也门、阿曼、巴林多依赖地方盟友，而非严密覆盖全半岛。7世纪初萨珊—拜占庭长期战争削弱外部控制；约628—630年间，也门波斯总督和半岛多个集团接受穆罕默德领导，进入伊斯兰时代。

## 重要事件

| 时间 | 事件 | 结果与影响 |
|---|---|---|
| 前3千纪 | 马干、迪尔蒙见于两河文献 | 半岛东部已参与铜、木材和印度河商品的海湾网络。 |
| 前1千纪初 | 南阿拉伯灌溉国家发展 | 农业剩余、神庙和铭文行政支撑王权。 |
| 约前8世纪—3世纪 | 萨巴主导马里卜地区 | 控制乳香路线并建设大型水利，影响遍及半岛和红海。 |
| 前1千纪中后期 | 米奈、卡塔班、哈德拉毛并立 | 多国竞争商路、绿洲和产区，而非单一帝国顺序替代。 |
| 前26—前25年 | 罗马埃利乌斯·加卢斯远征 | 远征因补给和环境失败，罗马转以红海贸易和外交为主。 |
| 1世纪前后 | 《红海航行记》时代海贸繁荣 | 季风航线把南阿拉伯港口与埃及、东非和印度连接。 |
| 3世纪后期 | 希木叶尔统一也门大部 | 古代南阿拉伯由多王国并立转向更大政治整合。 |
| 4世纪后 | 希木叶尔王室一神化 | 犹太教、基督教和地区帝国竞争改变合法性。 |
| 523年前后 | 奈季兰基督徒遭迫害 | 成为阿克苏姆军事干预的直接导火索。 |
| 约525年 | 阿克苏姆征服也门 | 红海非洲强国控制南阿拉伯，基督教政权建立。 |
| 6世纪中叶 | 阿布拉哈统治 | 尝试稳定也门、维护水利并参与半岛政治。 |
| 约570年代 | 萨珊军进入也门 | 阿克苏姆统治结束，波斯附庸和移民军人形成。 |
| 约628—630年 | 多地接受穆罕默德领导 | 古代王国与外部附庸体系转入穆斯林共同体框架。 |

## 兴盛与衰落因素

### 南阿拉伯王国兴盛

- 高地降水、坝渠和梯田创造稳定农业税基。
- 乳香、没药和转口货物提供高价值贸易收入。
- 神庙、铭文、部族和城市组织能协调劳役与军事。
- 位于红海—印度洋—陆路交界，可在不同路线间调整。

### 结构性衰落与政治替代

- 王国争夺水源、商道和附属部族，长期战争削弱地方中心。
- 海运和消费变化降低部分陆路中转优势，但并未单独造成崩溃。
- 希木叶尔的统一是内部政治胜出，不是整个南阿拉伯文明立即消失。
- 6世纪宗教冲突使阿克苏姆和萨珊介入；外部军队成为直接改朝因素。
- 萨珊—拜占庭大战削弱旧保护体系，伊斯兰共同体提供新的跨部落政治中心。

## 关键辨析

- “示巴女王”属于《圣经》《古兰经》和后世传统的重要叙事，但她与某位已知萨巴统治者的历史对应无法确定。
- “乳香之路”是多条可替换路线的总称，商品常由不同商人分段运输。
- “阿拉伯人”在古代文献中的范围不固定，南阿拉伯语言、部族和北部群体不能用现代统一民族概念覆盖。
- 古代南阿拉伯王表年代和共治关系仍有铭文解释争议；完整地方世系应在也门专页按可证程度列示，本跨区域页不复制推测性长表。
- 迪尔蒙、马干等历史地名与现代国家范围有重叠，但不是今日巴林、阿曼或阿联酋的古代同一国家。
