# 迪尔蒙、贸易网络与伊斯兰化

## 时间

约前5000年—16世纪初

## 概括

巴林古代史以淡水和海运为轴。晚期新石器遗址及欧贝德式器物说明群岛很早参与两河交换；前3千纪末，迪尔蒙中心移到巴林，卡拉特巴林、巴巴尔神庙和庞大墓葬群显示港口、王权、宗教与社会分层。长途贸易衰落后，巴林仍在巴比伦、波斯与希腊化海湾体系中承担港口作用。7世纪伊斯兰化以后，“巴林”常指含哈萨、卡提夫和群岛的东阿拉伯大区，不能把所有以巴林为名的政权事件都局限在今天的岛国内。

## 分阶段发展

### 史前交换与迪尔蒙形成

- 杰贝勒杜汉一带约前5000—4000年的遗址显示较稳定的渔猎和定居；欧贝德式陶器反映同南部两河的接触，不代表两河城市直接殖民。
- 两河文字约前3300年已提到迪尔蒙并联系铜贸易。早期政治中心可能位于今天沙特东部；约前2300—2000年才明显转移到巴林。
- 卡拉特巴林的城墙、宫殿、港口和街区，萨尔聚落、巴巴尔神庙以及数以万计的墓冢，显示前2千纪初人口增长、专业劳动和精英权力。
- 迪尔蒙商人转运马干铜、印度河商品、木材、石料及两河制品，并使用本地印章。其繁荣来自控制交换节点和淡水补给，而非大面积农业帝国。
- 印度河城市衰退、两河需求与航线变化使早期迪尔蒙约前18世纪后收缩；这是一种贸易体系转型，不是所有居民突然消失。

### 中晚期迪尔蒙与帝国网络

- 约前15—前14世纪，加喜特巴比伦在卡拉特巴林修建宫殿并派驻官员，试图控制海路和高价值贸易；宫殿火灾及帝国退缩后，地方秩序再次变化。
- 前8世纪亚述文献记载迪尔蒙统治者乌佩里进贡，说明远方帝国可主张优势，却未必日常统治岛屿。晚期迪尔蒙文化仍同巴比伦和阿曼保持密切交换。
- 阿契美尼德时期巴林很可能纳入波斯海湾体系，考古材料显示联系，但具体行政从属仍有不确定性。
- 亚历山大舰队约前325年考察群岛。希腊化时代的“提洛斯”在塞琉古及后来卡剌肯王国网络中保持港口和珍珠贸易，并出现希腊化器物、文字和墓葬文化。
- 安息和萨珊时期，巴林由海湾帝国与阿拉伯附庸势力共同影响。4—7世纪文献中的马斯马希格主教区显示东叙利亚基督教社群存在，伊斯兰化前社会并非单一宗教。

### 伊斯兰化、卡尔马特与地方王朝

- 7世纪初，东阿拉伯统治者和阿卜杜勒盖斯等部族接受伊斯兰。穆罕默德去世后发生叛离与效忠冲突，阿拉·哈德拉米率军恢复哈里发控制；朱瓦萨等主要战场多在东阿拉伯大陆，需同岛屿史区分。
- 正统哈里发和倭马亚时期，大巴林省因港口、农业绿洲、税收及通往法尔斯和印度的航线重要。哈瓦利吉和祖拜尔派势力曾利用中央内战争夺该区。
- 阿拔斯前期恢复直接任命，但赞吉起义和边疆财政削弱中央。899年阿布·赛义德·真纳比在哈萨建立卡尔马特国家，控制群岛和大陆，930年袭击麦加并夺走黑石，后因内部分裂和贸易权力变化衰落。
- 乌尤尼王朝约1077年在塞尔柱支持下结束卡尔马特统治，此后乌斯富里、贾尔万、贾布里等地方家族轮替。它们的核心常在哈萨或卡提夫，依赖珍珠税、绿洲农业和海湾航线。
- 15世纪贾布里势力与霍尔木兹王国争夺岛屿。霍尔木兹借海军与贡赋体系控制巴林珍珠收入，葡萄牙介入后于1521年击败贾布里统治者穆克林，开启下一阶段。

## 社会、贸易与统治机制

| 要素 | 运作方式 | 历史变化 |
|---|---|---|
| 淡水泉与绿洲 | 支持椰枣、聚落和过航船只补给。 | 使巴林相较干旱海岸更适合长期城市化。 |
| 港口与转运 | 连接两河、伊朗、阿曼和印度洋。 | 航线和大帝国需求变化决定繁荣周期。 |
| 珍珠床 | 提供长期高价值出口品。 | 从古典时代延续到20世纪，税权反复吸引外来征服者。 |
| 帝国与地方代理 | 帝国任命总督、收贡或借附庸治理。 | 名义主权、驻军控制与商人自治经常并存。 |
| 宗教社群 | 古代多神信仰、东叙利亚基督教、逊尼与什叶传统先后共存。 | 伊斯兰化是长期过程，教派结构又受卡尔马特与伊朗联系影响。 |

## 兴衰原因

- **迪尔蒙崛起**：淡水港湾、海湾中点和跨区域商品互补形成中转优势；政治中心移岛后，港口和祭祀机构把贸易收益集中化。
- **早期迪尔蒙衰退**：印度河城市体系与两河贸易需求变化是结构因素，帝国竞争和航线重组是外部压力；不能以单次入侵解释数百年收缩。
- **卡尔马特扩张**：阿拔斯边疆控制薄弱、哈萨绿洲财政和宗教军事组织相结合；其衰落则来自内部继承分裂、商路与盟友变化以及乌尤尼进攻。
- **地方王朝频繁更替**：岛屿财富足以吸引征服，却不足以长期供养大帝国驻军，因而常由大陆王朝、海上商业国和本地精英共同治理。
