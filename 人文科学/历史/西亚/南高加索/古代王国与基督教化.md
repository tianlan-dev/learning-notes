# 南高加索古代王国与基督教化

## 时间

约前9世纪至7世纪

## 概括

南高加索古代史由高原、河谷和黑海沿岸多个政体构成。乌拉尔图控制亚美尼亚高原部分地区；科尔基斯活跃于黑海东岸；高加索伊比利亚形成于今格鲁吉亚东部；高加索阿尔巴尼亚位于库拉河下游和东高加索。此后亚美尼亚诸王朝、格鲁吉亚政体与东西方帝国共同塑造区域格局。

## 主要政体与文化

| 名称 | 大致范围 | 历史意义 |
|---|---|---|
| 乌拉尔图 | 凡湖、亚美尼亚高原及周边 | 建立堡垒、灌溉和楔形文字王权体系；不能简单等同于后世亚美尼亚国家 |
| 科尔基斯 | 黑海东岸 | 连接黑海贸易、希腊殖民网络与地方社会 |
| 高加索伊比利亚 | 今格鲁吉亚东部一带 | 格鲁吉亚王权与基督教传统的重要古代基础 |
| 高加索阿尔巴尼亚 | 今阿塞拜疆北部及邻近地区 | 东高加索多族群王国，后受伊朗、基督教和伊斯兰政权影响 |
| 亚美尼亚王国 | 亚美尼亚高原及不同时期周边 | 阿尔塔什斯、阿尔萨息等王朝在罗马—伊朗竞争中维持王权 |

## 重要事件

- 前9—前6世纪乌拉尔图在亚美尼亚高原建立区域王国，后在帝国竞争和政治重组中消失。
- 前4世纪后亚美尼亚、伊比利亚和阿尔巴尼亚等政体在希腊化、罗马与伊朗世界之间发展。
- 提格兰二世时期，亚美尼亚王国一度扩张为跨高原与北美索不达米亚的强国。
- 亚美尼亚传统以4世纪初确立基督教王国；格鲁吉亚东部也在4世纪完成王权基督教化。
- 亚美尼亚字母约在405年由梅斯罗普·马什托茨创制；格鲁吉亚文字传统也在晚期古代逐步形成。
- 387年后亚美尼亚长期处于罗马 / 拜占庭与萨珊伊朗分割和竞争之中。
