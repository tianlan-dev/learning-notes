# 格鲁吉亚统一王国、分裂与帝国竞争

## 时间

1008年至1801年

## 概括

巴格拉季昂王朝在1008年前后统一多个格鲁吉亚政体。大卫四世和塔玛尔时期，王国扩张并形成宫廷、教会、修道院和城市文化高峰。蒙古入侵后，王国逐步分裂为卡特利、卡赫季、伊梅列季等王国和公国，并在奥斯曼与伊朗诸王朝之间周旋。

## 君主与统治结构

| 阶段 | 代表统治者或结构 | 特点 |
|---|---|---|
| 王国统一 | 巴格拉特三世 | 合并主要王位与领地，建立统一王权 |
| 强盛时期 | 大卫四世、塔玛尔 | 军事改革、领土扩张、教会与文化繁荣 |
| 蒙古统治与分裂 | 蒙古征服者、地方王室和贵族 | 贡赋、王位分裂与地方化 |
| 多王国时期 | 卡特利、卡赫季、伊梅列季等 | 王室、贵族和教会在奥斯曼—伊朗竞争中求存 |
| 俄国保护与吞并 | 卡特利—卡赫季王国、俄罗斯帝国 | 1783年保护条约后俄国于1801年吞并东格鲁吉亚王国 |

## 重要事件

- 1008年前后巴格拉特三世成为统一格鲁吉亚王国的首位君主。
- 1121年迪德格里战役后，大卫四世巩固王国并收复第比利斯。
- 塔玛尔统治时期，格鲁吉亚影响扩展到南高加索和黑海东部。
- 13世纪蒙古征服迫使王国纳贡并加深政治分裂。
- 15世纪后统一王国瓦解为多个王国与公国。
- 1555年《阿马西亚和约》及后续战争使奥斯曼与伊朗反复划分格鲁吉亚影响区。
- 1783年《格奥尔吉耶夫斯克条约》使卡特利—卡赫季接受俄罗斯保护。
- 1801年俄罗斯正式吞并卡特利—卡赫季王国。
