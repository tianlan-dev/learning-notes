# 现代黎巴嫩

## 时间

1943年至今

## 概括

现代黎巴嫩史以法国委任统治后的独立国家建设、宗派权力分配、贝鲁特商业文化、巴勒斯坦问题、内战、叙利亚影响、以色列冲突和区域力量竞争为主线。黎巴嫩的现代政治不能只从古代腓尼基传统理解，更要放在奥斯曼山地社会、法国委任统治和现代宗派政治框架中理解。

## 统治结构

| 主题 | 说明 |
|---|---|
| 宗派政治 | 国家职位和政治代表长期受宗派分配框架影响。 |
| 国家与民兵 | 内战及其后续使国家机构、政党、民兵和外部力量关系复杂。 |
| 区域关系 | 叙利亚、以色列、巴勒斯坦组织、伊朗、海湾国家和西方国家均长期影响黎巴嫩政治。 |
| 经济与社会 | 贝鲁特金融、港口、侨民网络和周期性危机共同塑造现代黎巴嫩。 |

## 重要事件

- 1943年黎巴嫩独立，形成以宗派权力分配为基础的国家结构。
- 1975-1990年黎巴嫩内战重塑国家政治和社会秩序。
- 内战后叙利亚影响、以色列撤军、真主党兴起和周期性政治危机成为重要线索。
- 21世纪以来，金融危机、港口爆炸、区域冲突外溢和国家治理困境持续影响黎巴嫩。
