# 奥斯曼边疆与外约旦酋长国

## 时间

1516—1946年

## 概括

奥斯曼在1516年击败马穆鲁克后接管叙利亚，但没有建立一个名为“外约旦”的单一行省。阿杰隆、拜勒加、卡拉克、马安等地区在不同时期隶属不同桑贾克、穆塔萨里夫区或大马士革行政中心。帝国早期主要保护朝觐路线、登记可征税村庄并同部落谈判；19世纪后期才借行政区、土地登记、驻军、移民定居和铁路显著加强直接统治。

1916—1918年阿拉伯起义和英国战争行动摧毁奥斯曼秩序。费萨尔在大马士革的阿拉伯政府于1920年被法国终结后，东岸出现地方政府。阿卜杜拉从汉志北上，1921年同英国达成安排，建立外约旦酋长国。此后的国家建设不是“英国划线后自动出现”，而是哈希姆王权、英国资金与军官、地方城镇精英、部落首领和新军队持续协商及冲突的结果。

## 奥斯曼治理的三个阶段

| 阶段 | 主要结构 | 具体特点 |
|---|---|---|
| 1516—1831年 | 大马士革总督、地方官、朝觐堡垒、税农和部落首领 | 直接控制集中于城镇、耕地和朝觐节点；偏远牧区常通过补贴或协议维持通行。 |
| 1831—1840年 | 埃及易卜拉欣帕夏占领 | 征兵、征税和集中化激起反抗；1840年后奥斯曼恢复统治。 |
| 1840—1918年 | 坦志麦特行政区、土地登记、驻军、自治市和铁路 | 国家进入乡村和部落社会的能力增强，同时引发征兵、税收和土地冲突。 |

16世纪税册曾把广阔地区归入阿杰隆桑贾克，但此后不少南部地区长期缺乏持续普查。1866—1871年前后，大马士革总督重建阿杰隆、萨尔特等行政区；1872年以马安为中心整合南部若干地区，1895年卡拉克成为特殊行政区中心。边界和隶属多次变化，因此不能把现代约旦行政区直接套入奥斯曼地图。

## 19世纪的社会与城市变化

- **土地与税收**：1858年土地法及后续登记把部分共同使用地转为私人名下，便利征税，也让城镇商人、部落首领和农民围绕所有权重新竞争。
- **移民定居**：1878年前后切尔克斯难民在安曼定居，后来又有车臣人在扎尔卡等地建立聚落。奥斯曼提供土地和保护，移民则加强道路、农业和行政据点。
- **城镇增长**：萨尔特原本是北部重要市场和行政中心；卡拉克、伊尔比德、马代巴、安曼逐步扩展。安曼从小聚落成长为铁路城镇，但到一战前仍不必然压倒萨尔特。
- **部落协商**：中央既使用武力和征兵，也给首领职务、土地、补贴和护路责任。部落不是国家的简单对立面，而是行政扩张的重要合作者和反抗者。
- **汉志铁路**：1900—1908年从大马士革通至麦地那，安曼、卡拉克以东和马安一线获得车站。铁路降低朝觐和军队运输成本，也使游牧护路收益、地方贸易和征税方式改变。

1910年卡拉克起义由人口登记、征兵、缴械和税收措施触发。奥斯曼军队镇压起义并处决、监禁参与者，表明中央化已深入地方，但也揭示强制改革缺乏稳定认同。

## 第一次世界大战与阿拉伯起义

1. 1914年奥斯曼参战后，征兵、牲畜和粮食征用加重，铁路成为军事生命线。
2. 1916年谢里夫侯赛因在麦加发动阿拉伯起义，其子费萨尔领导北方部队；英国提供武器、资金、顾问和海军支援。
3. 1917年7月，奥达·阿布·塔伊等部落力量与费萨尔军队从陆路攻取亚喀巴，使协约国获得红海北端补给港。
4. 起义军持续袭击汉志铁路桥梁、列车和小型驻地，但马安等坚固据点长期未被正面攻克。
5. 1918年，起义军与英军沿不同轴线北上，大马士革失守，外约旦奥斯曼政权瓦解。
6. 战时阿拉伯独立承诺、英法秘密划分和贝尔福宣言彼此冲突，为战后边界与合法性争议埋下根源。

## 1918—1921年的政权过渡

外约旦最初同费萨尔的大马士革阿拉伯政府保持联系，地方官、旧奥斯曼人员和民族主义者维持行政。1920年7月法国在迈萨伦击败费萨尔后，大马士革政府终结；东岸没有立即形成统一中央，而是出现阿杰隆、萨尔特、卡拉克等地方政府或行政委员会，英国官员以有限补助和顾问方式介入。

阿卜杜拉于1920年11月抵达马安，宣称准备北上恢复其兄费萨尔的叙利亚王位；1921年3月进入安曼。英国既不愿同法国冲突，也需要一个成本较低、能控制边境与部落的伙伴。开罗会议及阿卜杜拉同殖民大臣丘吉尔的会谈，把其北伐计划转化为在英国监督下治理东岸的安排。1921年4月11日通常视为酋长国建立日。

## 委任统治与酋长国的法律位置

1922年国际联盟批准的巴勒斯坦委任统治文本第25条允许英国在约旦河以东暂停实施与“犹太民族家园”有关的若干条款；同年英国备忘录把东岸置于不同制度。其含义是**在同一委任框架内分开行政**，而不是外约旦已经独立或英国不再拥有最高权力。

1923年英国承认阿卜杜拉领导的外约旦政府，但把正式独立同“具备自治能力”等条件相连。1928年英外约旦条约和《基本法》建立更明确的行政、内阁和立法委员会制度，英国仍控制外交、军事、重大财政和顾问任用。1929年举行首届立法委员会选举，选举范围和议会权力均有限。

## 实际行政结构

| 权力中心 | 正式地位 | 实际职能与限制 |
|---|---|---|
| **埃米尔阿卜杜拉** | 外约旦统治者 | 任命政府、经营部落和城镇精英、主持内部政治；外交、军事和重大财政受英国约束。 |
| 巴勒斯坦高级专员 | 英国委任统治最高官员 | 对伦敦负责，对东岸保留最终监督；通过批准、预算和条约限制酋长国。 |
| 英国驻地代表及顾问 | 驻安曼监督机构 | 深度介入财政、司法、行政和警务；早期与埃米尔、首相常有权力摩擦。 |
| 首相或执行委员会主席 | 埃米尔任命的政府首脑 | 管理日常部门并在埃米尔、英国代表和地方联盟之间协调，内阁更迭频繁。 |
| 立法委员会 | 1928年基本法后成立，部分选举产生 | 审议预算和法规、表达地方利益；条约、英国否决和埃米尔权力限制其自主性。 |
| 阿拉伯军团 | 1923年起发展为警备和军队核心 | 兵员多来自东岸社群，英国军官、补贴和训练占关键地位；同时建立王室与部落的直接联系。 |
| 外约旦边防军 | 1926年设立、另属英国体系 | 负责边界和交通线，与阿拉伯军团不是同一机构。 |
| 部落首领、城镇显贵与宗教社群 | 地方中介 | 通过税收、土地、治安、议席和官职交换支持；也会在利益受损时反抗。 |

阿拉伯军团先由弗雷德里克·皮克等英国军官建设，1939年后约翰·格拉布成为主要指挥者。沙漠巡逻队、通信、工资和招募把偏远部落纳入国家。它既是公共安全机构，也是王权组织能力的核心；这种英国资助、王室指挥和东岸兵源的组合延续到独立后。

## 建国中的冲突与整合

- 1921年库拉地区反抗新政府，显示北部地方首领不愿立即接受安曼集中权力。
- 1922—1924年，来自内志的伊赫万袭击边境村庄和部落；英国飞机、装甲力量及地方部队协防，加强了居民对中央安全的依赖。
- 1923年阿德万部落起事反对官职分配、税收和叙利亚流亡官员影响，政府在英国军事支持下取胜，随后又重新谈判部落关系。
- 埃米尔吸纳来自大马士革政府、巴勒斯坦、汉志及奥斯曼行政体系的人才，但“外来官员”与本地精英之间长期竞争。
- 1936—1939年巴勒斯坦阿拉伯起义产生跨河志愿者、武器和政治声援；阿卜杜拉同时尝试充当谈判中介。
- 第二次世界大战期间，外约旦军队协助英国在伊拉克和叙利亚行动，王国的战略价值和军队能力上升。

## 独立过程

第二次世界大战削弱英国财政，也强化阿拉伯民族主义和外约旦要求平等条约的压力。1946年3月英外约旦条约承认独立，5月25日议会宣布阿卜杜拉为“外约旦哈希姆王国”国王。独立终结委任统治和英国驻地代表的最高行政地位，但英国基地、补贴和军官仍在；军事自主直至1956年撤换格拉布等英国军官、1957年终止旧英约后才更完整。

## 酋长国崛起、稳定与转型原因

| 层次 | 酋长国为何形成并延续 | 为何在1946年转为独立王国 |
|---|---|---|
| 结构因素 | 奥斯曼解体后缺乏统一中央；英国需要低成本缓冲区；地方需要边境和治安。 | 行政、军队、财政和立法机构已具备国家雏形。 |
| 合法性 | 阿卜杜拉继承阿拉伯起义和麦加哈希姆声望，并同部落缔结效忠。 | 王权把独立塑造成起义目标的实现。 |
| 财政军事 | 英国补贴、军官、飞机与通信弥补资源不足；阿拉伯军团吸纳本地兵源。 | 二战提高军团价值，也让英国直接统治成本上升。 |
| 治理 | 官职、土地、议席和保护把城镇、移民社群与部落纳入国家。 | 本地精英和立法机构要求更平等条约。 |
| 外部压力 | 伊赫万袭击、沙特边界、法属叙利亚和巴勒斯坦危机强化中央需要。 | 战后反殖民潮、阿拉伯民族主义和英国财政收缩。 |
| 直接触发 | 1921年阿卜杜拉与英国达成治理安排。 | 1946年伦敦条约及5月25日议会宣布独立。 |
