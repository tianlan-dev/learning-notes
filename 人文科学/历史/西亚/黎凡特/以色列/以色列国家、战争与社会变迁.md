# 以色列国家、战争与社会变迁

## 时间

1949年至今（当代信息核验截至2026年7月13日）

## 概括

1949年停战以后，以色列在没有单一成文宪法的情况下，以《独立宣言》、普通法律和逐步制定的基本法建立议会制国家。国家在短期内吸收大规模犹太移民，统一军队和公共行政，发展农业、工业、教育、科研与高科技产业；与此同时，土地与住房分配、不同犹太社群的社会差距、宗教与世俗关系、巴勒斯坦阿拉伯公民的平等问题以及长期战争持续塑造政治。

安全与领土问题构成现代史另一主线。1956、1967和1973年国家间战争以后，以色列同埃及、约旦实现和平，但1967年以来对西岸、东耶路撒冷和戈兰高地的控制、定居点扩张、巴勒斯坦建国、难民和耶路撒冷问题没有解决。2005年以色列撤出加沙境内定居点和常驻军队，未终结其对边境、空域、海域等控制的法律争议；2023年后以军再次长期进入加沙。

2023年10月7日袭击把冲突扩大为加沙、黎巴嫩、伊朗、红海和叙利亚相互牵连的地区战争。到2025年10月加沙形成脆弱停火，所有以色列人质在2026年1月前返回或遗体被寻回，但加沙治理、哈马斯解除武装、以军控制区、重建和巴勒斯坦政治前途仍未解决。2026年又发生美以对伊朗的“雄狮咆哮”行动以及黎巴嫩战事复燃，因而“停火”不等于地区战争已经完成政治解决。

## 国家制度与实际权力

| 机构 | 法定地位 | 实际运作 |
|---|---|---|
| 总统 | 由克奈塞特选举的国家元首 | 主要承担礼仪、外交代表、赦免及在议会选举后委托组阁等职能；不能自行决定日常外交或军事政策。 |
| 克奈塞特 | 120席一院制议会，全国单一选区比例代表制 | 通过法律、基本法和预算，监督政府；政党众多使联盟谈判、宗教政党与小党议价成为常态。 |
| 总理与政府 | 行政权主体，须获得并维持议会信任 | 总理主持政府，是行政和安全政策的核心协调者，但重大决定依法由政府或国家安全内阁集体承担。 |
| 国家安全内阁 | 由总理主持的法定部长委员会 | 审议战争目标、重大行动、停火和外交安全问题；军方和情报机关提供方案，文官内阁作政治决定。 |
| 司法系统与总检察长 | 最高法院兼高等法院审查行政行为，基本法构成宪制框架 | 在没有单一成文宪法的情况下，对政府合法性和个人权利具有重要制衡作用；其权限是2023年以来政治危机的核心。 |
| 国防军与情报机构 | 国防军受文官政府统率；辛贝特、摩萨德分别负责国内安全和对外情报 | 征兵、预备役和情报体系使安全机构深度嵌入社会；政策责任仍属于民选政府，不能把军方当作独立政权。 |
| 地方与被占领土治理 | 以色列境内由民政机关管理，西岸另有军事司令部、民政管理局和巴勒斯坦有限自治机构 | 公民法、军事命令、定居点民事法与巴勒斯坦自治权限并存，形成多层且权利不对称的治理结构。 |

截至2026年7月13日，伊萨克·赫尔佐格任总统，本雅明·内塔尼亚胡任第37届政府总理，阿米尔·奥哈纳任克奈塞特议长。行政与战争政策主要由总理、国家安全内阁、国防部门和安全机构共同形成；总统并非实际行政首脑。执政联盟已把2026年10月27日确定为下一次克奈塞特选举日期，本文截止时投票尚未举行。完整任职序列见[以色列总统与总理世系表](/%E4%BA%BA%E6%96%87%E7%A7%91%E5%AD%A6/%E5%8E%86%E5%8F%B2/%E8%A5%BF%E4%BA%9A/%E9%BB%8E%E5%87%A1%E7%89%B9/%E4%BB%A5%E8%89%B2%E5%88%97/%E4%BB%A5%E8%89%B2%E5%88%97%E6%80%BB%E7%BB%9F%E4%B8%8E%E6%80%BB%E7%90%86%E4%B8%96%E7%B3%BB%E8%A1%A8.md)。

## 政治演变

### 国家建设与马派主导（1949—1967年）

- 临时政府和犹太复国主义准国家机构转化为克奈塞特、政府、法院、地方行政和全国性公共服务。不同地下武装被整合进以色列国防军，军队由民选政府统率。
- 1950年《回归法》确立犹太人移居并取得公民身份的通道。欧洲大屠杀幸存者及来自中东、北非的大量犹太人到来，使人口迅速增加；帐篷营和过渡营、住房短缺、配给制及族群不平等随之出现。
- 留在以色列境内的巴勒斯坦阿拉伯人逐步取得公民身份，但多数地区直到1966年仍受军事管制。土地征收、流离失所者财产、教育和地方发展差距成为长期问题。
- 马派依托总工会、合作社、国有和准国有机构主导政治经济。国家建设能力很强，却也形成庇护网络、中央集权和阿什肯纳兹精英占优的结构。
- 边境渗透、难民返乡尝试、巴勒斯坦武装袭击和以色列报复行动不断升级；1956年以色列同英国、法国进攻埃及，在军事获胜后因美苏及联合国压力撤出西奈和加沙。

### 1967年胜利、占领与1973年震荡（1967—1977年）

六日战争使以色列控制西奈、加沙、西岸、东耶路撒冷和戈兰高地，获得战略纵深，也把数百万巴勒斯坦人置于其军事控制之下。政府把东耶路撒冷纳入城市和国内法律体系，在西岸、加沙、戈兰与西奈逐步建立定居点；不同政党对“以土地换和平”、永久控制和安全边界的立场分化。

1967—1970年消耗战显示军事胜利没有消除埃及和巴勒斯坦冲突。1973年埃及、叙利亚突袭造成严重损失，虽然后来战线扭转，情报判断、动员延误和领导责任仍重创工党信誉。战后抗议、经济压力、米兹拉希犹太人对社会不平等的不满和腐败争议共同促成1977年政党轮替。

### 利库德崛起、和平与黎巴嫩战争（1977—1992年）

- 贝京领导利库德首次执政，吸纳长期边缘化的米兹拉希选民、民族主义者和宗教阵营，结束工党体系垄断。
- 1978年戴维营协议和1979年埃以和平条约，以西奈全部归还换取承认与和平；1982年最后撤离西奈定居点，证明撤出占领领土与正式和平可以相连。
- 与此同时，政府加速西岸定居点建设。1982年入侵黎巴嫩的目标从清除边境威胁扩大到重塑黎巴嫩政治，战争拖延、萨布拉和夏蒂拉屠杀后的国内问责及伤亡消耗削弱政府；以军最终保留南黎巴嫩“安全区”至2000年。
- 高通胀和债务危机促成1984—1988年工党—利库德全国团结政府及总理轮换。1985年稳定计划控制通胀，也推动经济从国家主导向市场化转型。
- 1987年第一次巴勒斯坦大起义使占领的日常成本和巴勒斯坦群众政治进入以色列社会中心。1991年海湾战争和马德里和会开启新的地区谈判环境。

### 奥斯陆进程、政治暴力与制度试验（1992—2001年）

拉宾政府同巴解组织相互承认并签署《奥斯陆协议》，在加沙和西岸部分地区建立巴勒斯坦有限自治；1994年又同约旦签署和平条约。进程没有解决边界、耶路撒冷、难民、定居点和安全等最终地位问题。巴勒斯坦武装袭击、定居点继续扩张以及双方对临时义务的不同解释不断侵蚀信任。

1995年拉宾被反对和平进程的犹太极端分子刺杀，是以色列民主与社会分裂的重大转折。1996年开始实行总理直选，原意是强化行政稳定，结果选民可以把总理票与政党票分开，反而助长小党增长。内塔尼亚胡、巴拉克先后在安全事件、联盟分裂和谈判失败中失去支持；2001年特别选举后，总理直选制度被取消。

### 第二次起义、单边主义与党系重组（2001—2009年）

2000年第二次巴勒斯坦大起义带来大规模袭击、以军行动、城市封锁和严重平民伤亡。沙龙政府修建西岸隔离墙或安全屏障；其在阻止袭击方面得到许多以色列人支持，但路线深入西岸、土地征收和行动限制受到国际法律挑战。

2005年政府撤出加沙全部定居点和常驻地面部队，却未同巴勒斯坦方面达成最终协议。哈马斯2006年选举获胜、2007年控制加沙后，以色列和埃及加强边境限制，火箭袭击、封锁与周期性军事行动形成长期循环。沙龙退出利库德创建前进党，2006年重病后由奥尔默特接任；第二次黎巴嫩战争及腐败调查又使中间派新党迅速衰落。

### 内塔尼亚胡长期执政与选举僵局（2009—2021年）

内塔尼亚胡建立以利库德、宗教政党和不同中右翼伙伴为核心的灵活联盟，安全、伊朗、经济增长与外交正常化成为长期执政支柱。高科技、天然气开发和对外贸易增强国家资源，住房成本、贫富差距和公共服务压力则引发2011年社会抗议。

这一时期经历2008—2009、2012、2014和2021年加沙战争或大规模行动。2018年《犹太民族国家基本法》把犹太民族自决、国家象征和希伯来语地位写入基本法；支持者认为它确认建国宗旨，反对者担心其削弱非犹太公民平等地位。2020年以色列同阿联酋、巴林、摩洛哥等推进关系正常化，但没有解决巴勒斯坦最终地位。

内塔尼亚胡受贿、欺诈和背信案件进入司法程序，他否认指控。2019—2021年连续四次选举均难形成稳定多数；新冠疫情期间的内塔尼亚胡—甘茨轮换协议未完成轮换。2021年八党联盟最终结束其连续执政。

### 贝内特—拉皮德联盟与第37届政府（2021—2023年10月）

贝内特—拉皮德政府横跨右派、中间派、左派和阿拉伯联合名单；后者成为首个以独立阿拉伯政党身份正式支持并加入执政联盟的党派。联盟通过预算并实行轮换安排，但在巴勒斯坦、宗教与身份议题上差异过大，2022年失去多数并解散议会。

2022年选举后，内塔尼亚胡同极右翼和犹太教正统派政党组成第37届政府。政府推动限制最高法院审查和改变法官任命等方案，支持者称其恢复民选多数权力，反对者认为会削弱司法独立和少数权利。2023年出现持续大规模抗议、预备役拒服威胁和经济警告，司法改革危机在10月7日袭击前已严重撕裂社会。

| 2008—2009、2012、2014、2021年 | 加沙多轮战争 | 火箭、隧道、空袭和地面行动周期性升级，停火多由埃及等调停。 | 哈马斯未被解除武装，以色列维持封锁和威慑；加沙平民、基础设施及以色列边境社群反复受损。 |

| 2026年2月起 | “雄狮咆哮”及黎巴嫩战争复燃 | 2月28日美国与以色列共同攻击伊朗，伊朗以导弹、无人机和地区力量还击；真主党参战后，以色列在黎巴嫩实施大规模空袭和地面行动。 | 伊朗战线形成脆弱停火后仍有互袭；以黎6月达成框架性安排，但撤军、真主党解除武装和边境安全执行不完整，7月仍有攻击。 |

## 2023—2026年战争的分阶段过程

### 10月7日袭击与加沙地面战（2023—2024年）

哈马斯主导的武装人员于2023年10月7日突破加沙边界，袭击以色列城镇、社区和音乐节，杀害军民并劫持人质。袭击暴露情报、边境防御、指挥和政府判断的系统性失败。以色列宣布战争，动员预备役，以空袭和地面进攻摧毁哈马斯军事与治理能力并营救人质。

2023年11月短暂停火促成部分人质与巴勒斯坦囚犯交换。此后以军从加沙北部推进至汗尤尼斯和拉法，摧毁隧道、指挥节点和大量城市设施。哈马斯遭受重大损失但继续以分散小组和地下网络作战。战事造成加沙极高的巴勒斯坦死亡数字、反复流离失所、饥饿风险和医疗系统崩溃；具体伤亡由加沙卫生部门统计并被联合国用于人道报告，以色列质疑其中部分归因方式。

### 停火破裂、伊朗战线与加沙再停火（2025年）

2025年1月的阶段性停火释放部分人质并增加援助，3月以后战事恢复。以色列继续扩大军事控制并施压哈马斯，人质家属和反政府群体则要求优先达成交换协议。6月以色列直接攻击伊朗，冲突短暂转向国家间导弹与空袭；加沙战争并未因此结束。

2025年10月10日的新停火以人质全部返还、分阶段撤军、援助和战后治理为框架。剩余在世人质于10月13日返回以色列，最后一具遗体于2026年1月26日被寻回。人质全部返回是实质进展，但第二阶段所要求的哈马斯解除武装、替代治理、以军进一步撤离和重建机制推进缓慢。

### 脆弱停火与地区战争延续（2026年）

停火初期的“黄线”使以军继续部署在约占加沙一半以上的区域；到2026年7月，联合国仍报告该线附近推进、空袭、枪击、人员死亡以及援助准入限制。以色列政府以边境安全、地下设施和哈马斯未解除武装为由维持行动；联合国及人道机构则认为控制区扩大、平民伤亡和救援限制破坏停火。因而截止日期上的准确表述是：加沙处于正式但频繁受损的停火，而非已签署最终和平或以军完全撤离。

2026年2月28日，美以联合攻击伊朗引发新的地区战争，伊朗和真主党还击，以色列北部与黎巴嫩再次成为主战场。伊朗战线在春季形成临时停火，6—7月仍有互袭；以色列和黎巴嫩6月底的框架安排要求撤军、黎巴嫩国家垄断武力及真主党解除武装，但执行尚未完成。以色列同时继续控制叙利亚戈兰停火缓冲区内若干位置，称其为安全需要；联合国和叙利亚要求撤离。

## 占领、控制范围与法律状态

| 地区 | 以色列的控制与国内法立场 | 国际法律与政治状态 |
|---|---|---|
| 以色列1949年停战线以内地区 | 由以色列完整民政管辖，是国际承认国家的核心范围；1949年线本身原为停战线而非双方最终议定国界。 | 以色列同埃及、约旦边界后来由和平条约确定；与巴勒斯坦、叙利亚和黎巴嫩的部分边界或最终地位仍有争议。 |
| 西奈半岛 | 1967年占领，依据1979年埃以和平条约分阶段撤出，1982年完成。 | 已归还埃及，不再属于以色列控制范围。 |
| 东耶路撒冷 | 1967年后被纳入耶路撒冷市域和以色列法律体系；1980年基本法宣布完整、统一的耶路撒冷为首都。 | 联合国及多数国家不承认单方面吞并，视东耶路撒冷为被占领土和最终地位谈判核心。 |
| 西岸 | 以军维持总体安全控制；奥斯陆安排下A区由巴勒斯坦权力机构负责民政和内部安全，B区权力分担，C区由以色列控制。定居者适用以色列民事法律框架，巴勒斯坦人主要受军事命令与自治法规约束。 | 联合国、国际法院和多数国家称其为被占领巴勒斯坦领土，认为定居点违反国际法；以色列主张领土主权未定、其权利和安全要求应由谈判解决，并反对部分法律结论。 |
| 加沙 | 2005年撤出定居点和常驻军队后，以色列称占领已经结束；仍控制大部分外部通道、空域、海域、人口登记和货物流动。2023年后以军重新在境内长期部署，2025年停火后仍控制“黄线”以外大片区域。 | 联合国、国际法院等仍把加沙视为被占领巴勒斯坦领土的一部分；2025年停火没有决定主权或最终边界。 |
| 戈兰高地 | 1967年从叙利亚夺取，1981年适用以色列法律；美国2019年承认以色列主权。2024年叙利亚政局变化后，以军又进入联合国缓冲区若干位置。 | 联合国和绝大多数国家仍视戈兰为被占领叙利亚领土，不承认吞并；缓冲区部署也受到撤离要求。 |

2024年国际法院咨询意见认定以色列在被占领巴勒斯坦领土的持续存在违法，要求停止新定居活动并尽快结束该状态；咨询意见具有权威法律解释意义，但不同于双方签署的和平执行机制。以色列政府拒绝该结论的事实与法律框架，强调安全威胁、历史权利、奥斯陆协议和直接谈判。书写时应同时区分“国际机构的法律判断”“以色列官方立场”与“现场实际控制”，不能把其中任何一项代替另外两项。

围绕加沙战争，国际法院在南非诉以色列案中发布临时措施，但截至本文截止日尚未就灭绝种族指控作出最终实体判决；国际刑事法院2024年对内塔尼亚胡和时任国防部长加兰特签发逮捕令，以色列否认管辖权和指控。逮捕令不是定罪，相关程序仍在进行。

## 社会与经济变迁

- **移民国家形成**：1948—1951年人口急增，随后又有苏联及后苏联地区、埃塞俄比亚和其他地区移民。希伯来语、教育和兵役创造共同制度经验，但欧洲、米兹拉希、俄罗斯语、埃塞俄比亚及宗教社群之间的地位差异长期存在。
- **阿拉伯公民**：包括穆斯林、基督徒、德鲁兹和贝都因社群，在选举、教育、地方治理和专业领域参与国家生活；土地、预算、规划、治安、身份与巴勒斯坦民族联系仍造成不平等和政治争议。
- **宗教与国家**：婚姻离婚多由宗教法庭管理，安息日、饮食规范、皈依、圣地和正统派兵役豁免反复引发联盟危机。最高法院2024年后对征兵豁免的裁决加剧了正统派政党与战争动员需求的冲突。
- **经济转型**：早期配给和国家主导经济，经1985年稳定计划、私有化、全球化及科研投资转向高科技与服务业。创新和出口形成优势，住房、生活成本、贫富差距及阿拉伯和正统派人口就业教育差距构成结构约束。
- **战争社会**：普遍兵役、预备役和长期紧急状态强化社会动员，也使战争成本直接进入家庭、劳动力、财政和政治问责。2023年后长期征召、北南边境撤离、创伤和海外关系恶化成为新的社会问题。

## 国家韧性与结构性危机

| 维度 | 韧性来源 | 长期风险 |
|---|---|---|
| 国家能力 | 成熟行政体系、全民动员、税收、教育科研和移民吸收经验 | 战争常态化扩大国防支出并挤压公共服务。 |
| 军事与外交 | 技术优势、情报体系、美国支持、同部分阿拉伯国家和平或正常化 | 对威慑和军事手段依赖难以替代政治安排，多战线作战造成战略过伸。 |
| 经济 | 高科技、全球投资、人力资本和天然气资源 | 生活成本、不平等、人才外流风险和战争冲击。 |
| 民主制度 | 竞争性选举、活跃社会组织、司法审查和媒体争论 | 联盟碎片化、司法改革危机、领导人刑事案件及紧急权力削弱制度信任。 |
| 社会整合 | 希伯来语、教育、兵役及共同安全经验 | 犹太—阿拉伯、宗教—世俗、阿什肯纳兹—米兹拉希及中心—边缘裂痕。 |
| 巴以关系 | 同巴解组织相互承认及既有谈判文本仍提供制度基础 | 占领、定居点、暴力、难民、耶路撒冷和两地政治分裂相互强化。 |

冲突不能用单一宗教仇恨、单一领袖选择或单一外部干预解释。结构因素包括两种民族运动对同一土地的主权诉求、1948年难民问题、1967年以来的占领与定居体系、安全困境和权力不对称；外部压力包括邻国战争、伊朗及地区武装网络、美国和国际体系介入；直接触发因素则是具体袭击、暗杀、政策决定、联盟危机和谈判破裂。
