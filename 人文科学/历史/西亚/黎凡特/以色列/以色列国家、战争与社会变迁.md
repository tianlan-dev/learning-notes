# 以色列国家、战争与社会变迁

## 时间

1949年至今

## 概括

1949年停战后，以色列建立议会、政府、军队、司法和公共服务体系，并吸收来自欧洲、中东、北非、苏联地区、埃塞俄比亚等地的大量犹太移民。国家在希伯来语复兴、教育、农业、工业和科技领域迅速发展，同时不同移民群体、犹太宗教派别、世俗社会和巴勒斯坦阿拉伯公民之间存在资源、身份与制度争议。

阿以战争和巴勒斯坦问题长期主导安全与外交。1967年战争后，以色列占领西岸、东耶路撒冷、加沙、戈兰高地和西奈；西奈后来依据和平条约归还埃及，以色列2005年撤出加沙境内定居点和常驻地面部队，但对加沙边界、空域和海域控制以及封锁问题继续引发法律与政治争议。西岸定居点、耶路撒冷地位、巴勒斯坦主权、安全安排和难民问题仍未解决。

## 统治结构

| 机构 | 角色 | 说明 |
|---|---|---|
| 总统 | 国家元首 | 主要承担礼仪和象征职能。 |
| 议会 | 一院制克奈塞特 | 采用比例代表制，政党联盟和联合政府是政治常态。 |
| 总理与内阁 | 行政领导 | 负责政府施政、外交和安全政策。 |
| 司法系统 | 最高法院及其他法院 | 审查行政行为并处理基本法相关争议；司法权范围是国内政治焦点。 |
| 地方与特殊治理体系 | 地方政府、军事和民政管理机构 | 以色列境内公民治理同被占领土的军事—民政安排性质不同。 |

## 重要事件

- 1949年，以色列加入联合国并同邻国签署停战协定。
- 1956年，以色列与英国、法国共同发动苏伊士战争，后在国际压力下撤军。
- 1967年，六日战争改变领土和占领格局。
- 1973年，埃及和叙利亚发动战争；其后外交进程加速。
- 1977年，利库德集团首次领导政府，长期由工党主导的政治格局终结。
- 1979年，以色列与埃及签署和平条约并逐步归还西奈。
- 1982年，以色列入侵黎巴嫩，巴解组织主力撤出贝鲁特。
- 1987年，第一次巴勒斯坦大起义爆发。
- 1993年，以色列与巴解组织签署《奥斯陆协议》；1994年同约旦签署和平条约。
- 2000年，第二次巴勒斯坦大起义爆发。
- 2005年，以色列单方面撤出加沙定居点和常驻军队。
- 2020年，以色列同阿联酋、巴林等国推进关系正常化。
- 2023年10月7日，哈马斯等武装组织袭击以色列并劫持人质；以色列随后在加沙发动大规模军事行动，造成严重人员伤亡、破坏与人道危机。
