# 古代以色列、犹大与犹太历史传统

## 时间

约前12世纪—1517年

## 概括

铁器时代的南黎凡特出现以色列和犹大王国。相关历史来自考古材料、近东铭文和《希伯来圣经》等传统，具体年代与统一王国规模存在学术争论。前722年前后新亚述灭亡北方以色列王国，前586年新巴比伦攻陷耶路撒冷并终结犹大王国。波斯统治时期部分被掳群体返回，耶路撒冷圣殿和犹太社群制度重建。

希腊化和罗马时期，哈斯蒙尼王朝曾建立独立政权，随后罗马控制犹太地区。66—73年和132—135年两次大规模犹太战争遭镇压，第二圣殿被毁，离散社群进一步扩展；但犹太人在巴勒斯坦地区并未完全消失。拜占庭、早期伊斯兰、十字军、阿尤布和马穆鲁克时期，本地犹太社群与更广泛离散世界持续互动。

## 王朝世系 / 统治结构

| 阶段 | 主要政治或社群结构 | 说明 |
|---|---|---|
| 以色列与犹大王国 | 王权、地方贵族、圣所和部落传统 | 北方以色列和南方犹大是两个相关但独立的王国。 |
| 巴比伦与波斯时期 | 帝国行省、地方总督、祭司和社群长老 | 流亡与返回共同影响犹太身份和经典传统。 |
| 哈斯蒙尼王朝 | 祭司王朝与扩张性国家 | 马加比起义后形成独立政权，后因内斗被罗马介入。 |
| 罗马—拜占庭时期 | 罗马行省、附庸王、拉比社群和侨民共同体 | 第二圣殿毁灭后，拉比犹太教和会堂网络更为重要。 |
| 伊斯兰与中世纪 | 伊斯兰政权下受保护宗教社群及地方会众 | 法律地位和生活状况随王朝、城市与时期变化。 |

## 重要事件

- 前9世纪，外部铭文已提到以色列王国等南黎凡特政治实体。
- 前722年前后，新亚述征服北方以色列王国。
- 前586年，新巴比伦摧毁耶路撒冷第一圣殿并终结犹大王国。
- 前539年后，阿契美尼德波斯允许部分被掳者返回，犹大行省和第二圣殿制度逐步重建。
- 前2世纪，马加比起义推动哈斯蒙尼王朝建立。
- 前63年，罗马干预犹太地区。
- 70年，罗马军队摧毁第二圣殿。
- 132—135年，巴尔·科赫巴起义被镇压。
- 7世纪以后，犹太社群在伊斯兰统治下参与耶路撒冷、提比里亚等地生活，并同中东和地中海离散社群联系。
- 12—15世纪，十字军和后续穆斯林政权更替改变圣地各社群处境。
