# 锡安主义、英国委任统治与建国

## 时间

1517—1949年

## 概括

现代以色列的建国不是古代王国自然延续，而是奥斯曼帝国晚期和欧洲民族主义时代形成的政治运动，在英国殖民治理、两次世界大战、欧洲反犹主义与大屠杀、犹太移民和机构建设、巴勒斯坦阿拉伯民族运动以及国际决策共同作用下实现的结果。

19世纪以前，巴勒斯坦地区长期生活着以阿拉伯语为主的穆斯林和基督徒人口，也有耶路撒冷、希伯伦、采法特和提比里亚等地的犹太社群。19世纪末以后，近代锡安主义者以移民、购地、自治机构、希伯来教育、劳工组织和防卫力量建设“新伊舒夫”；巴勒斯坦阿拉伯社会则要求代表多数居民取得独立，反对英国把“犹太民族家园”置于当地民族自决之前。两种民族运动在同一土地上寻求主权，而英国既无法调和承诺，也不愿建立由当地多数决定移民政策的自治体制，冲突逐渐军事化。

1947年联合国大会建议分治，但没有建立执行该方案的共同政治或军事机制。委任统治地内战在英国撤离前已经爆发；1948年5月14日以色列宣布独立后，战争扩大为同邻近阿拉伯国家的国家间冲突。到1949年，以色列保住国家并扩大实际控制区，约旦控制西岸和东耶路撒冷，埃及管理加沙，分治方案中的阿拉伯国未能建立；约70万巴勒斯坦人逃离或被驱逐，难民与“纳克巴”成为此后冲突核心。

## 奥斯曼统治与近代社会背景

### 行政、土地和社群

1517年奥斯曼征服南黎凡特后，今天以色列和巴勒斯坦所在地区长期分属大马士革、西顿、贝鲁特等行省及地方行政单位，并不存在一个贯穿四百年的单一“巴勒斯坦省”。19世纪后期，耶路撒冷及其周边成为直接同伊斯坦布尔联系的独立桑贾克，北部和沿海则分别归属贝鲁特与大马士革体系。

奥斯曼改革把土地登记、征税、征兵和地方委员会纳入更统一的法律框架。1858年土地法推动土地登记，却也使未登记耕作者、地方名流、城市地主和国家之间出现复杂权利重叠。后来锡安主义机构购买的土地多经合法契约取得，但出售者、实际耕作者和租佃者未必是同一群体；部分农民失去耕作权，使土地交易同时具有法律、社会和民族政治含义。

欧洲领事保护、教会、学校和资本深入耶路撒冷、雅法、海法等城市。蒸汽航运、铁路和出口农业把沿海同世界市场连接，城市人口增加。阿拉伯语报刊、地方自治诉求和奥斯曼立宪政治推动巴勒斯坦阿拉伯身份逐渐形成，不能把当地社会写成等待外来开发的“空地”。

### 旧伊舒夫与新移民

奥斯曼时期既有以宗教学习、朝圣和慈善网络维系的“旧伊舒夫”，也有19世纪末以后以农业劳动、民族复兴和政治建国为目标的“新伊舒夫”。两者之间存在人员重叠，不能简单等同为“东方宗教犹太人”和“欧洲世俗锡安主义者”。

- **第一次阿利亚（约1882—1903年）**：俄国和东欧迫害、罗马尼亚困境及“锡安之爱”等运动推动移民。移民建立农业定居点，但疾病、资金、农业经验不足和奥斯曼限制使许多项目依赖埃德蒙·罗斯柴尔德等资助。
- **第二次阿利亚（约1904—1914年）**：俄国革命失败和迫害推动新一批青年移民。劳工锡安主义、“希伯来劳动”、集体农庄、工人政党、守卫组织和希伯来语教育发展，特拉维夫于1909年建立。
- 移民规模相对当地总人口仍有限，却建立了土地基金、学校、医疗、工会和安全组织等可扩张制度。部分组织推行只雇用犹太劳工和土地永久归民族机构所有，增强内部凝聚，也排斥阿拉伯劳动者并加剧民族分隔。

## 锡安主义的形成与内部路线

政治锡安主义产生于欧洲民族国家兴起、犹太解放受挫、现代种族反犹主义和周期性暴力之中。1896年西奥多·赫茨尔发表《犹太国》，1897年第一届锡安主义者大会通过巴塞尔纲领，目标是在巴勒斯坦为犹太人民建立受公共法律保障的家园，并创建世界锡安主义组织。

锡安主义从来不是单一路线：

| 路线 | 主要主张 | 对建国过程的影响 |
|---|---|---|
| 政治锡安主义 | 先争取列强和国际法承认，再组织大规模移民 | 形成世界锡安主义组织、外交游说和《贝尔福宣言》的国际路径。 |
| 实践锡安主义 | 通过移民、购地、农业和教育逐步建立事实基础 | 建立定居点、城市、学校和自治网络。 |
| 劳工锡安主义 | 以劳动、合作社、工会和集体农业塑造新社会 | 马派、总工会、哈加纳及建国初期国家机构的重要来源。 |
| 修正主义锡安主义 | 强调立即建国、军事自卫、自由经济及更广领土诉求 | 形成贝塔尔、伊尔贡等组织，是后来赫鲁特和利库德传统来源。 |
| 宗教锡安主义 | 把民族复兴同宗教义务结合 | 建立宗教教育、定居和政党网络，后来影响领土与定居点政治。 |
| 文化锡安主义 | 优先复兴希伯来文化、教育和精神中心 | 推动现代希伯来语、出版、大学和跨地区文化共同体。 |

1901年犹太民族基金成立，1908年巴勒斯坦事务机构运作，1920年总工会和哈加纳形成，1929年扩大的犹太事务局获得委任统治制度承认。运动的成功不仅来自外交宣言，也来自持续把人口、资金、土地、知识和组织能力转化为准国家机构。

## 第一次世界大战与相互冲突的承诺

战争期间奥斯曼军队征用物资、征兵和迁徙，加上封锁、蝗灾和疾病，使当地社会遭受严重困难；部分具有敌国国籍的犹太居民被驱逐，锡安主义组织活动受限。英军1917年占领耶路撒冷，1918年控制整个地区。

英国战时外交同时包含多组彼此张力很大的安排：同阿拉伯方面讨论反奥斯曼起义后的独立，同法国划分中东势力范围，1917年《贝尔福宣言》又赞成在巴勒斯坦建立“犹太人民的民族家园”，并声明不得损害当地非犹太社群的公民和宗教权利。宣言没有承认巴勒斯坦阿拉伯人的集体政治权利，也未说明民族家园是否最终成为国家，歧义后来被写入委任统治。

## 英国委任统治的制度矛盾

### 正式结构与实际控制

| 层级或机构 | 权力 | 实际作用 |
|---|---|---|
| 英国高级专员 | 最高行政与立法权 | 控制移民、土地、警察、军队、外交和财政，向伦敦负责而非向当地选民负责。 |
| 行政机关、法院和警察 | 日常殖民治理 | 建设基础设施和专业官僚体系，也以紧急法、拘留、审查和军事手段镇压反抗。 |
| 犹太事务局 | 委任文件承认的公共机构 | 协助移民和经济社会发展，对外代表犹太社群，并同民族委员会、总工会组成准国家网络。 |
| 犹太民族委员会与社群机构 | 教育、医疗、税费、地方代表 | 通过选举和组织网络管理伊舒夫内部事务；哈加纳承担地下防卫。 |
| 巴勒斯坦阿拉伯政治组织 | 阿拉伯执行委员会、最高穆斯林委员会、地方政党和名流网络 | 代表民族诉求，却没有获得同犹太事务局相当的国际法定地位；家族竞争和英国压制削弱连续组织。 |
| 拟议立法委员会 | 英国多次提出但未形成稳定代议机构 | 若按人口比例，阿拉伯多数可控制移民政策；若保证民族家园，阿拉伯代表又认为多数原则被架空，矛盾难解。 |

1922年国际联盟委任文件把《贝尔福宣言》纳入英国义务，并承认犹太事务局的公共角色；同时要求保护全体居民的公民和宗教权利。英国没有承诺立即建立犹太国，也没有赋予当地居民完整民族自决。英国政策因此在促进民族家园、限制移民、安抚阿拉伯多数和维护帝国战略之间反复摇摆。

### 两个社会的平行制度化

1920—1930年代，伊舒夫建立希伯来教育、医疗、工会、合作社、工商业、自治议会和防卫体系。第三次阿利亚强化劳工机构；第四次阿利亚带来中产商人和城市资本；纳粹上台后的第五次阿利亚带来大量德国及中欧移民、专业人才与资金，使特拉维夫、海法和工业体系迅速扩张。

巴勒斯坦阿拉伯社会也发展报刊、学校、工会、妇女组织、政党和地方商业，却面对殖民权力、土地失衡、城市与乡村差异、侯赛尼—纳沙希比等名流竞争，以及缺乏统一税收和军事机构的问题。英国把社群作为分离单位处理，经济和教育网络日益平行，减少了形成共同公民政治的空间。

### 暴力升级与英国政策转向

| 时间 | 事件 | 过程与影响 |
|---|---|---|
| 1920、1921年 | 耶路撒冷和雅法等地暴力 | 民族恐惧、移民和殖民治理争议转化为社群袭击；哈加纳进一步组织化。 |
| 1922年 | 丘吉尔白皮书 | 解释民族家园不等于把整个巴勒斯坦变为犹太国，并按经济吸收能力调节移民，但维持《贝尔福宣言》。 |
| 1929年 | 西墙争议与全地暴力 | 耶路撒冷宗教空间争执扩大为袭击和屠杀，希伯伦古老犹太社群遭重创；英国调查土地和移民问题。 |
| 1930年 | 帕斯菲尔德白皮书及后续修正 | 试图限制土地转移和强调阿拉伯农民处境，受到锡安主义反对后部分退让。 |
| 1936—1939年 | 巴勒斯坦阿拉伯人大起义 | 先以总罢工和抵制开始，后转为乡村游击；要求独立、停止大规模犹太移民和土地转移。 |
| 1937年 | 皮尔委员会 | 首次正式提出分治，认定两种民族诉求难以在单一委任制内调和；边界、人口转移和可行性引发争议。 |
| 1939年 | 英国白皮书 | 计划十年内建立统一独立国家，五年限额接纳犹太移民，随后需阿拉伯同意；并限制部分土地出售。阿拉伯方面不信任执行，锡安主义运动认为英国背弃民族家园。 |

大起义被英国军队、警察和同英方合作的犹太辅助力量镇压。村庄受集体惩罚，许多阿拉伯政治与军事领导人被杀、拘捕或流亡。镇压并未消除民族主义，却使巴勒斯坦阿拉伯社会在1947年内战前缺少统一指挥、武器和连续行政体系。与此同时，哈加纳获得军事经验，伊尔贡等修正主义地下组织也以袭击方式回应阿拉伯暴力和英国限制。

## 大屠杀、反英斗争与委任统治崩溃

1939—1945年纳粹德国及其盟国杀害约六百万欧洲犹太人。大屠杀摧毁欧洲犹太社会，使幸存者收容、无国籍和移民问题成为紧迫国际危机，也强化许多犹太人对主权国家和自卫能力的诉求。英国仍执行移民限额，地下海运“非法移民”与拦截、拘留成为冲突焦点。

二战期间，犹太事务局和哈加纳总体支持英国反纳粹战争；伊尔贡后期与莱希转向反英。战后哈加纳、伊尔贡和莱希一度协调破坏行动，随后因平民伤亡和政治路线分裂。1946年伊尔贡炸毁大卫王酒店英国行政与军事总部的一部分，造成大量死者；英国搜捕、处决和拘留地下成员，治理成本继续上升。

英国受财政压力、驻军伤亡、美英分歧、阿犹对立和难民危机夹击，1947年把问题交给联合国，而不是在当地建立共同制宪程序。联合国特别委员会多数方案主张两个国家和经济联盟，耶路撒冷国际化。

## 分治方案与双方选择

1947年11月29日，联合国大会第181号决议建议：

- 委任统治终止后建立一个犹太国和一个阿拉伯国；
- 耶路撒冷和伯利恒周边实行国际特别制度；
- 两国建立经济联盟，并保障少数群体权利；
- 拟议犹太国约占委任统治地一半以上，内部仍有大量阿拉伯居民；拟议阿拉伯国分布为不相连地区。

犹太事务局接受分治作为建国的法律和外交基础，虽对边界和耶路撒冷安排有保留。巴勒斯坦阿拉伯领导与阿拉伯国家拒绝方案，认为当地阿拉伯人占人口多数且拥有大部分土地，却被外部机构要求把过半领土交给移民人口建立国家；他们主张统一独立的阿拉伯多数国家。双方均没有接受一个共同强制执行机关，英国宣布撤离并拒绝以本国军队实施分治，战争遂成为决定控制范围的手段。

## 1947—1949年战争的具体过程

### 第一阶段：委任统治地内战（1947年11月—1948年5月）

联合国表决后，阿拉伯和犹太交通、城市街区及混合地区立即出现袭击。巴勒斯坦阿拉伯地方武装、阿拉伯解放军同哈加纳、帕尔马赫、伊尔贡和莱希交战。耶路撒冷犹太区道路被切断，双方围绕车队、山口和混合城市争夺。

1948年春，哈加纳从防守转向争取分治区和交通线的主动行动，实施“达列特计划”。该计划究竟主要是军事控制方案，还是为广泛驱逐提供框架，在史学上存在争论；实际战场中既有直接驱逐，也有居民因恐惧、战斗、领导撤离、经济崩溃和对屠杀消息的反应而逃亡，不能用单一原因解释全部难民。

4月代尔亚辛村民被伊尔贡和莱希人员杀害，加剧巴勒斯坦社会恐慌；犹太医疗车队等也遭阿拉伯武装杀害。提比里亚、海法、采法特和雅法相继被犹太军控制，大批阿拉伯居民离开。英国优先保护撤军路线，并未建立能接管行政和治安的统一本地权力。

### 第二阶段：宣布独立与阿拉伯国家参战（1948年5—6月）

5月14日，戴维·本-古里安代表犹太人民委员会宣布以色列国成立。5月15日起，埃及、外约旦、叙利亚、黎巴嫩和伊拉克军队进入原委任统治地；沙特等提供较小力量或援助。各国共同反对犹太国，但战略目标并不一致：

- 外约旦阿拉伯军团重点控制分治方案中的阿拉伯地区和东耶路撒冷，同巴勒斯坦独立建国目标并非完全一致；
- 埃及从南部和沿海推进，也担心外约旦扩大；
- 叙利亚、伊拉克和黎巴嫩分别从北部与东部参战，协调和后勤能力有限；
- 巴勒斯坦阿拉伯武装已经在内战中严重削弱，没有统一国家指挥体系。

阿拉伯军团占领东耶路撒冷犹太区并守住拉塔伦，以军则稳住特拉维夫、沿海和耶路撒冷西部交通替代线。5月26日正式组建以色列国防军，把哈加纳、帕尔马赫和其他武装纳入国家指挥。6月“阿尔塔莱纳号”武器船冲突显示本-古里安政府以武力拒绝政党保留独立军队，确立文官国家对武装的垄断。

### 第三阶段：停火、重整与“十日战斗”（1948年6—7月）

联合国第一次停火让双方重整。以色列利用海外采购、捷克斯洛伐克军火和国内动员缓解武器短缺，建立更完整旅级指挥；阿拉伯联军补给和共同指挥问题未根本改善。7月战斗恢复后，以军在卢德—拉姆拉、加利利和中部走廊取得进展，卢德和拉姆拉大部分巴勒斯坦居民被驱逐或在战斗中逃离。

第二次停火期间，联合国调停员福尔克·贝尔纳多特提出新方案，后被莱希成员刺杀。以色列政府取缔莱希并逮捕成员，进一步确立国家不容党派武装独立行动。

### 第四阶段：秋冬攻势与1949年停战

1948年10月以后，以军发动约阿夫行动打通内盖夫并击退埃军，希拉姆行动控制加利利，随后在何烈行动中进入西奈边缘并受国际压力撤回。1949年3月乌夫达行动控制乌姆拉什拉什，即后来的埃拉特。

1949年，以色列依次同埃及、黎巴嫩、外约旦和叙利亚签署停战协定。“绿线”标示停战部队位置，不是各方承认的永久政治边界。以色列控制原委任统治地约77%；外约旦控制西岸和东耶路撒冷，埃及管理加沙。联合国拟议阿拉伯国没有成立。

## 建国成功的机制

| 机制 | 具体作用 |
|---|---|
| 准国家机构 | 犹太事务局、民族委员会、总工会、地方议会、学校、医疗和税费网络能迅速转化为政府。 |
| 统一动员 | 伊舒夫能够征兵、分配物资、组织交通和形成集中军事指挥；地下武装最终被并入国防军。 |
| 人口与国际网络 | 离散犹太社群提供资金、外交、志愿者、技术和武器采购渠道。 |
| 大国与国际环境 | 联合国分治提供外交合法性；美国和苏联较早承认以色列，捷克斯洛伐克军火在关键阶段发挥作用。 |
| 对手分裂 | 巴勒斯坦领导层在1939年后受重创，阿拉伯国家之间目标竞争，联军缺乏统一指挥。 |
| 战争中的学习 | 停火期间重组、武器输入和从防御转向旅级攻势，使战力差距扩大。 |

这些机制解释以色列为何能在战争中存续，但不能把结果写成预定胜利。1948年初犹太社区曾面临道路中断、武器短缺和多线战争；阿拉伯军团在耶路撒冷等地也有明显优势。战争结果来自组织能力、国际环境、战场选择和对手协调失败共同作用。

## 难民、人口与国家整合

战争造成约70万巴勒斯坦人逃离或被驱逐，数百个村庄和城市社区人口流失。原因随地点和阶段不同，包括直接驱逐、战斗和屠杀恐惧、经济与行政崩溃、临时避战以及部分地方领导撤离；把全部归因于单一统一命令或全部描述为自愿离开都不符合复杂事实。

以色列拒绝大规模难民返回，担忧安全和犹太人口多数，并以《不在者财产法》等制度接管大量土地财产。仍留境内的巴勒斯坦人后来成为以色列公民，但许多成为“境内流离失所者”。同一时期及随后几年，大量来自阿拉伯和穆斯林国家的犹太人因迫害、战争、民族政策、财产压力和锡安主义动员等不同原因迁入以色列；两类迁徙的历史条件不同，不能互相抵消难民权利。

## 崛起、冲突升级与委任统治终结原因

### 结构因素

- 两个民族运动都把同一地区视为不可替代的故土，并要求集体自决；
- 欧洲迫害推动犹太移民，当地阿拉伯多数则担心人口、土地和政治权力被逐步改变；
- 英国以殖民权力管理移民与治安，却没有建立获得双方承认的共同主权制度；
- 社群在教育、经济、安全和政治组织上平行发展，日常交往没有转化为共享国家认同。

### 外部压力

- 欧洲民族主义、反犹主义、俄国迫害和纳粹大屠杀不断增加建国紧迫性；
- 英国帝国战略、美国国内外政策、苏联对英国势力的竞争和联合国决策改变国际条件；
- 阿拉伯国家的竞争性战略使巴勒斯坦问题地区化，却未形成统一国家建设方案。

### 直接触发因素

- 1939年白皮书、战后难民危机和地下组织反英行动使委任统治失去可持续性；
- 英国决定退出而不执行分治，造成权力真空；
- 第181号决议获得犹太方面接受而被阿拉伯方面拒绝，双方转向武装控制；
- 1947年末社群内战和1948年5月阿拉伯国家参战把政治争端变成全面战争。

委任统治的“灭亡”不是某一方单独击败英国，而是帝国财政和军事实力下降、殖民合法性衰退、相互矛盾的承诺、双方武装反抗及国际化失败共同造成。以色列建国解决了犹太主权和避难问题，却没有建立巴勒斯坦阿拉伯国，因而把一项民族国家建设的成功同另一民族的灾难和长期无国籍状态连接在一起。

|---|---|---|

| 1917年 | 《贝尔福宣言》与英军占领耶路撒冷 | 英国支持“犹太民族家园”，同时留下当地多数民族权利矛盾。 |
| 1920—1922年 | 英国民政与国际联盟委任确立 | 殖民行政和民族家园义务制度化。 |
| 1920年 | 总工会与哈加纳成立 | 劳工和防卫体系成为准国家支柱。 |
| 1929年 | 西墙争议及社群暴力 | 宗教地点、民族恐惧和安全组织进一步军事化。 |
| 1936—1939年 | 巴勒斯坦阿拉伯人大起义 | 反殖民和反移民运动遭镇压，阿拉伯政治组织严重削弱。 |
| 1939年 | 英国白皮书 | 限制移民和土地交易，锡安主义运动转向反对英国政策。 |
| 1939—1945年 | 纳粹大屠杀 | 约六百万犹太人遇害，建国与难民安置的紧迫性剧增。 |
| 1946年 | 大卫王酒店爆炸 | 反英地下斗争升级，委任统治成本与危机加深。 |
| 1947年11月29日 | 联合国大会第181号决议 | 建议阿犹分治和耶路撒冷国际化，但没有被共同接受或执行。 |
| 1947年末—1948年5月 | 委任统治地内战 | 道路、城市和村庄易手，巴勒斯坦难民潮已经开始。 |
| 1948年5月14日 | 以色列宣布独立 | 犹太准国家机构转化为主权政府。 |
| 1948年5月15日起 | 阿拉伯国家参战 | 地方内战扩大为第一次阿以战争。 |
| 1948年5—6月 | 国防军成立与阿尔塔莱纳事件 | 国家取得对犹太武装的集中控制。 |
| 1948年9月 | 贝尔纳多特遇刺 | 联合国调停受挫，以色列取缔莱希。 |
| 1949年 | 四份停战协定 | 绿线和新的实际控制格局形成，巴勒斯坦阿拉伯国未建立。 |
