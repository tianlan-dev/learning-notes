# 以色列王国与犹大王国

## 时间

约前11世纪—前586年；其中北方以色列王国约在前722年灭亡。

## 概括

以色列王国与犹大王国是青铜时代末期南黎凡特政治重组的产物。中央山地在约前12—前11世纪出现大量村落和新的身份表达，随后在非利士城邦、外约旦王国与区域强权竞争中形成王权。扫罗、大卫、所罗门构成早期王权的传统主线，但前10世纪是否存在疆域广阔、行政成熟的统一王国，学界仍有争论；前9世纪以后，北方以色列与南方犹大作为两个国家的证据明显增多。

北国人口、农业与贸易条件较优，暗利王朝时期成为区域强国，却因频繁政变、反亚述联盟和帝国干预于前722年前后灭亡。南国犹大规模较小，以耶路撒冷、大卫王室和圣殿为核心，在亚述附庸体系下延续更久；新巴比伦崛起后，统治者反复改变宗主路线，最终在前586年遭毁灭。两国覆亡不是单一宗教或君主品德造成，而是国家能力、精英竞争、帝国扩张和具体反叛决策共同作用的结果。

## 史料与历史重建

重建这一时期需同时使用聚落考古、城市破坏层、本地碑铭、亚述和巴比伦王室铭文、摩押石碑、印章与宗教文献。《希伯来圣经》保存了王名、政变、战争、宗教改革和流亡记忆，但其材料经过数百年编纂，具有神学与王朝立场。外部铭文同样是胜利者宣传，不能视为完全中立。

前1208年前后的梅尔奈普塔赫石碑已提到“以色列”，表述更像一群人而非国家。前9世纪的特尔丹碑提到“大卫家”，摩押王米沙碑与亚述铭文则能同北国君主、战争和贡赋交叉核对。证据强弱随时期变化：早期王权的疆域和年代应写“约”与“存在争议”，暗利王朝以后则可建立较连续的政治史。

## 建立背景与崛起机制

### 山地聚落与王权形成

青铜时代末期，埃及对南黎凡特的控制收缩，沿海和内陆旧城邦体系重组。中央山地人口增加，新村落以旱作、畜牧和亲族网络维持生活。这些居民在物质文化上同迦南传统有明显连续性；早期以色列人的形成更可能是本地人口重组、边缘群体聚合和有限迁入，而非一次外来征服完整取代原居民。

非利士城邦拥有较成熟的城市、铁器与军事组织，迫使山地联盟形成更稳定的指挥。扫罗传统反映军事首领向王权转化。大卫据传统先在希伯仑建立基础，再控制耶路撒冷；把首都设在原有支派核心之外，有利于建立王朝中心。所罗门传统把圣殿、宫廷、贡赋和国际贸易同王权制度化联系起来，但其实际规模需要谨慎评估。

### 分裂与两个国家

所罗门死后，北方集团反对徭役和财政负担，耶罗波安一世在示剑被拥立；罗波安保有犹大与耶路撒冷。分裂不只是一次宫廷争吵，也反映南北人口、经济、地方精英和圣所网络的长期差异。北国先后以示剑、得撒和撒马利亚为中心，王朝更替频繁；犹大则依靠大卫家、首都与圣殿的结合维持名义连续。

## 北方以色列王国

### 暗利王朝的扩张

暗利在内战中取胜并建设撒马利亚。新都靠近肥沃农业区和国际通道，宫殿、仓储和奢侈品显示王廷动员能力。暗利、亚哈通过同推罗联姻、控制外约旦部分地区及参与反亚述联盟，使北国成为前9世纪的重要国家。前853年卡尔卡战役的亚述记录列出亚哈的庞大战车和步兵，虽然数字可能夸张，仍说明其军事地位。

王朝崛起依赖人口与农业基础、沿海贸易、宫廷建设、婚姻外交和职业军队；同时也加重税役与地方不满。宗教文本把耶洗别与巴力崇拜置于冲突中心，但政治斗争还涉及王室土地、地方家族、先知群体和对外联盟。

### 耶户政变与亚述压力

前841年，军官耶户发动政变，杀死北国约兰和犹大亚哈谢，清洗暗利王朝。耶户获得亚述承认并纳贡，却失去部分区域影响。耶户王朝在耶罗波安二世时期利用大马士革衰弱与亚述暂缓西进恢复疆域，农业和贸易再次繁荣；先知文本所批判的土地集中、司法不公与贫富分化也在此背景下加深。

前8世纪后期，提革拉特帕拉沙尔三世把叙利亚—巴勒斯坦纳入直接扩张。北国宫廷连续发生政变，米拿现靠重贡保位，比加参与反亚述联盟并联合大马士革攻击拒绝加入的犹大。亚述吞并加利利等边区并扶立何细亚。何细亚后来停止纳贡、寻求埃及援助，触发围攻。

### 灭亡过程

撒马利亚约在前722年陷落，行动横跨撒缦以色五世和萨尔贡二世时期。亚述把部分王室、官员、工匠和军户迁往帝国其他地区，又迁入其他被征服人口，原北国核心被重组为行省。人口迁移服务于压制反叛、配置劳动力与削弱地方精英，但不是全体居民被一次迁空。北方村落继续存在，后来的撒马利亚共同体也同本地人口和耶和华崇拜传统有关。

## 南方犹大王国

### 亚述附庸下的国家化

犹大早期人口和资源少于北国。前8世纪，北方难民南迁、亚述贸易体系和耶路撒冷王廷扩张共同促进城市、书写与行政发展。亚哈斯在叙利亚—以法莲战争中向亚述求援，避免被北国—大马士革联盟推翻，却使犹大成为附庸。

希西家后来参与反亚述活动并加强城防、水利和仓储。前701年西拿基立攻占拉吉等大量城镇，掳走人口，耶路撒冷未陷但付出重贡。国家空间更集中于首都，王室和圣殿也更容易控制税赋与宗教资源。玛拿西长期服从亚述，虽然被后世宗教史家批判，其统治为战后恢复和精英重建提供了相对稳定期。

### 约西亚改革与帝国真空

亚述衰落后，约西亚试图扩展耶路撒冷影响并推动圣所中央化。宗教改革、律法传统和王权整合互相支持，但具体执行范围及同《申命记》的关系仍有争议。前609年，约西亚在米吉多遭遇北上的埃及军并死亡。埃及废黜约哈斯，册立约雅敬；数年后新巴比伦在卡尔凯美什击败埃及，犹大又转入巴比伦宗主权。

### 两次攻城与直接灭亡

约雅敬在大国竞争中反叛。前597年，尼布甲尼撒二世攻占耶路撒冷，约雅斤投降，王室、官员、军人和工匠中的一部分被迁往巴比伦；巴比伦改立西底家。西底家在埃及支持预期和国内主战派影响下再次反叛。前589 / 588年起巴比伦围城，埃及援军未能改变战局；前586年城墙被破，宫殿和第一圣殿被毁，西底家被俘。

巴比伦设置地方行政并任命基大利治理留居人口，基大利不久被刺杀，又有部分人逃往埃及。犹大国家机构和大卫王朝本地统治终结，但人口并未全部消失。流亡者、留居者和周边移民随后共同参与波斯时期“耶胡德”行省与第二圣殿共同体的形成。

## 统治结构

| 层面 | 北方以色列 | 南方犹大 | 历史作用 |
|---|---|---|---|
| 王权基础 | 军队、地方精英和多处区域中心 | 大卫王朝、耶路撒冷与圣殿结合 | 北国较易政变换朝，南国以王朝神学和首都集中维持连续 |
| 首都 | 示剑、得撒，后固定于撒马利亚 | 耶路撒冷 | 都城控制宫廷、仓储、外交与军事动员 |
| 宗教结构 | 伯特利、但、撒马利亚及地方圣所并存 | 耶路撒冷圣殿地位逐步上升 | 宗教中央化既是信仰路线，也是王权整合资源的工具 |
| 军事与财政 | 北国人口多、战车与农业资源较强 | 依靠山地防御、附庸外交与首都集中 | 两国都无法同亚述、新巴比伦的帝国动员规模长期抗衡 |
| 地方社会 | 土地家族、城镇、村社、先知与军官 | 王室官员、祭司、书吏、地主和村社 | 精英竞争、税役与社会不平等影响王朝稳定 |

## 重要事件

| 时间 | 事件 | 过程与结果 |
|---|---|---|
| 约前1208年 | 埃及铭文提到“以色列” | 证明南黎凡特已有以色列人群称谓，但尚不能据此推断成熟王国。 |
| 约前11—前10世纪 | 早期王权形成 | 非利士压力、山地联盟和耶路撒冷中心化促成扫罗、大卫、所罗门传统。 |
| 约前10世纪后期 | 南北分立 | 北方拥立耶罗波安，犹大由罗波安统治。 |
| 前9世纪前半 | 暗利王朝崛起 | 撒马利亚建都、腓尼基联姻和区域战争提高北国国家能力。 |
| 前853年 | 卡尔卡战役 | 亚哈参与叙利亚诸国反亚述联盟，暂时阻挡撒缦以色三世。 |
| 前841年 | 耶户政变 | 暗利家被推翻，北国转而向亚述纳贡。 |
| 前734—732年 | 叙利亚—以法莲战争 | 北国与大马士革攻击犹大，亚述介入并吞并北国边区。 |
| 约前722年 | 撒马利亚陷落 | 北国灭亡，亚述实施行省化和人口迁移。 |
| 前701年 | 西拿基立入侵犹大 | 拉吉等城被毁，耶路撒冷保存但重回亚述附庸。 |
| 前609年 | 约西亚死亡 | 埃及短暂控制犹大并干预王位。 |
| 前597年 | 第一次攻陷耶路撒冷 | 约雅斤和部分精英被迁巴比伦，西底家被册立。 |
| 前586年 | 耶路撒冷与第一圣殿被毁 | 犹大王国终结，进入流亡与后王国共同体阶段。 |

## 兴衰与灭亡原因

| 因素类型 | 北方以色列 | 南方犹大 |
|---|---|---|
| 崛起条件 | 人口与农业优势、国际交通、撒马利亚新都、腓尼基贸易和职业军队 | 首都—王朝—圣殿结合、山地防御、吸收北方人口与书写行政增长 |
| 结构性弱点 | 王位继承制度不稳，军官政变频繁，地方与宫廷利益冲突 | 资源有限，财政和兵力不足，首都集中使全国易受一次围城重创 |
| 外部压力 | 大马士革竞争与新亚述直接扩张 | 先后夹在亚述、埃及和新巴比伦之间 |
| 直接触发 | 何细亚停止纳贡并寻求埃及支持 | 约雅敬、西底家在不利形势下反叛巴比伦 |
| 灭亡方式 | 首都陷落、行省化、精英和部分人口迁移 | 两次攻城、王室废立、首都圣殿毁灭和地方行政重组 |

把灭亡解释为某一君主“不敬神”属于古代宗教史观，能说明后世共同体如何理解灾难，却不能替代对帝国兵力、贡赋压力、联盟误判、国内派系和战争过程的分析。
