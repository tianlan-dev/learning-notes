# 古代叙利亚与伊斯兰时代

## 时间

约前3千纪—1516年

## 概括

现代叙利亚境内位于地中海、安纳托利亚、两河流域和阿拉伯草原之间。埃卜拉、马里和乌加里特等古代城市参与青铜时代外交与贸易网络；铁器时代的阿拉米城邦以及腓尼基沿海网络又受到亚述、巴比伦和波斯帝国控制。亚历山大东征后，塞琉古王朝以叙利亚为核心之一，罗马于前64年建立叙利亚行省，安条克、帕尔米拉和大马士革成为重要城市。

7世纪阿拉伯征服后，大马士革成为倭马亚王朝首都。阿拔斯迁都巴格达后，叙利亚不再是统一帝国的政治中心，却继续处于地方王朝、拜占庭边疆、十字军国家、阿尤布王朝和马穆鲁克苏丹国的竞争中。1516年奥斯曼击败马穆鲁克，开启新的统治阶段。

## 统治结构

| 阶段 | 主要政治结构 | 特点 |
|---|---|---|
| 青铜与铁器时代 | 城邦、宫廷和区域帝国附属国 | 城市网络密集，常在埃及、安纳托利亚与两河强权之间结盟或臣服。 |
| 希腊化与罗马时期 | 王朝行省、罗马总督与自治城市 | 安条克为区域都会，商队城市帕尔米拉连接地中海与内陆。 |
| 拜占庭时期 | 帝国行省、主教区和边防体系 | 基督教教派、修道传统及阿拉伯边疆盟友共同存在。 |
| 倭马亚及后续伊斯兰时期 | 哈里发、地方总督、苏丹与军事封地 | 大马士革一度为帝国首都，后成为地区行政和宗教学术中心。 |
| 十字军—马穆鲁克时期 | 十字军领地、阿尤布诸侯与马穆鲁克行省 | 沿海堡垒、内陆城市和商路控制反复易手。 |

## 重要事件

- 前3—前2千纪，埃卜拉、马里、乌加里特等城市进入跨区域外交和贸易体系。
- 前64年，庞培将叙利亚纳入罗马行省体系。
- 3世纪，帕尔米拉势力一度在芝诺比娅领导下挑战罗马。
- 636年雅穆克河战役后，拜占庭失去叙利亚大部。
- 661—750年，倭马亚王朝以大马士革为首都。
- 1098年，十字军建立安条克公国；此后叙利亚沿海和内陆长期争夺。
- 1174年后，萨拉丁逐步控制大马士革及叙利亚多地。
- 1260年后，马穆鲁克控制叙利亚并抵御蒙古势力。
- 1516年，马穆鲁克在达比克草原战败，叙利亚并入奥斯曼帝国。
