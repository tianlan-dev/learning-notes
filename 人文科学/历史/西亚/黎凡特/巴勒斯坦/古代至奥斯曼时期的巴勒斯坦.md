# 古代至奥斯曼时期的巴勒斯坦

## 时间

约前3千纪—1917年

## 概括

今以色列、巴勒斯坦领土及周边在古代属于南黎凡特。青铜时代存在迦南城邦和村落社会；铁器时代出现以色列、犹大、非利士城市及其他政治实体。此后地区先后进入亚述、巴比伦、波斯、希腊化王国和罗马统治。罗马和拜占庭时期，“巴勒斯坦”逐渐成为行政与地理称呼之一，犹太人、撒玛利亚人、希腊语城市居民和基督徒等多种社群共同生活。

7世纪阿拉伯征服后，地区进入伊斯兰政治体系，阿拉伯语和伊斯兰文化逐步占主导，但基督徒、犹太人和其他社群继续存在。十字军、阿尤布、马穆鲁克和奥斯曼先后统治。奥斯曼时期当地并非名为“巴勒斯坦”的单一行省，而由耶路撒冷、纳布卢斯、阿卡等不同行政区治理；城市名流、乡村家族、贝都因部落、宗教机构和农民共同构成社会。

## 统治结构

| 阶段 | 主要结构 | 特点 |
|---|---|---|
| 青铜与铁器时代 | 城邦、王国、村落与部落网络 | 处在埃及、两河和安纳托利亚帝国竞争之间。 |
| 波斯—希腊化—罗马时期 | 帝国行省、自治城市、圣殿和地方王朝 | 行政边界与社群身份多次重组。 |
| 拜占庭与早期伊斯兰 | 行省、主教区、军区和地方社群 | 基督教圣地、伊斯兰圣地和多宗教城市共同发展。 |
| 十字军至马穆鲁克 | 拉丁王国、阿尤布诸侯、马穆鲁克行政 | 城堡、圣地、港口和道路成为争夺重点。 |
| 奥斯曼时期 | 行省和桑贾克官员、宗教法庭、地方名流与村社 | 地方权力分散，19世纪行政和土地改革加强国家介入。 |

## 重要事件

- 青铜时代，耶路撒冷、拉吉、夏琐等城邦处在区域外交网络中。
- 前8—前6世纪，亚述和新巴比伦征服改变南黎凡特人口与政治结构。
- 前4世纪后，地区进入希腊化王国竞争。
- 70年，罗马摧毁耶路撒冷第二圣殿；135年前后行政和城市秩序再次重组。
- 4世纪以后，耶路撒冷和伯利恒等地成为基督教朝圣中心。
- 638年前后，耶路撒冷向阿拉伯军队投降，地区进入早期伊斯兰统治。
- 1099年，十字军攻占耶路撒冷；1187年萨拉丁重新控制该城。
- 1517年，奥斯曼帝国征服巴勒斯坦地区。
- 19世纪，坦志麦特、土地登记、欧洲领事和宗教机构以及新的交通网络改变地方社会。
- 第一次世界大战期间，奥斯曼统治因战争、征发、饥荒和英军进攻而终结。
