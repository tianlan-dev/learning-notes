# 纳巴泰字母

## 概括

纳巴泰字母是亚兰字母的地方化分支，主要与纳巴泰王国和北阿拉伯地区有关。它的草写化字形是阿拉伯字母形成的重要中介。

## 演变关系

```mermaid
graph TD
  Aramaic["亚兰字母"] --> Nabataean["纳巴泰字母"]
  Nabataean --> Arabic["阿拉伯字母"]
```

## 说明

- 纳巴泰字母保留亚兰字母的辅音字母性质。
- 后期纳巴泰书写越来越连笔化，形成早期阿拉伯字母的关键字形基础。
