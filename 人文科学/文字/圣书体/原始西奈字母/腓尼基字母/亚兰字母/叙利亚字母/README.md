# 叙利亚字母

## 概括

叙利亚字母是亚兰字母的后继分支，主要用于书写叙利亚语，并通过基督教、商贸和中亚交通影响更东部的文字传统。

## 说明

- 叙利亚字母属于辅音字母传统，有不同书体分化。
- 它在中亚文字传播史中常作为亚兰传统东传的重要一环。
