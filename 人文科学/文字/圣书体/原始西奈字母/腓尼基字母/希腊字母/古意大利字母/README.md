# 古意大利字母

## 概括

古意大利字母是古代意大利半岛若干文字的统称，受希腊字母影响形成，包括伊特鲁里亚字母等。拉丁字母通常经由这一传统发展出来。

## 说明

- “古意大利字母”不是单一民族文字，而是一组相关文字传统。
- 拉丁字母的形成与伊特鲁里亚字母、中意大利地区语言和罗马早期书写实践有关。
