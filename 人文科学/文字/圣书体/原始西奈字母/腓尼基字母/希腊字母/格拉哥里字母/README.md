# 格拉哥里字母

## 概括

格拉哥里字母是9世纪为斯拉夫语基督教文献创造的文字传统，与希腊字母、拜占庭传教和古教会斯拉夫语密切相关。西里尔字母后来成为东斯拉夫、南斯拉夫等许多斯拉夫语社群更广泛采用的书写体系。

## 说明

- 格拉哥里字母与西里尔字母不是简单的“一字形改一字形”关系，而是同处斯拉夫基督教书写传统中的两个体系。
- 西里尔字母大量借用希腊字母形体，并加入表示斯拉夫语音的字母。
