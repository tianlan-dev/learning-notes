# Learning Notes

这个库用于把不同学科的知识整理成可导航、可复习、可扩展的结构。总览页承担两件事：一是进入子页面，二是说明该层级的知识框架。

## 使用方法

- 对历史类主题，优先按时间线整理和阅读。
- 对生物分类、语言文字等体系类主题，优先按层级关系整理和阅读。
- 对建筑、哲学、宗教等文化类主题，优先按时代演变、地域传统和概念差异整理和阅读。
- 对自然科学主题，优先分清尺度和层级：微观、宏观、生命、地球、宇宙分别有不同解释框架。

## 整理原则

- 每个目录尽量有一个 `README.md` 作为总览页，用于说明子页关系。
- 总览页不只是链接清单，还应给出时间线、层级概括、或主线总结。
- 具体笔记保留事实、概念、易混点和相关笔记，避免只堆资料。
