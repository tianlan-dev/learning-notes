# 蓝细菌门

## 范围

蓝细菌门属于细菌域，现行拉丁名常写作 Cyanobacteriota，常见名为 Cyanobacteria。

## 概括

蓝细菌能够进行产氧光合作用，是地球早期氧气积累和生态系统演化中的关键类群。真核植物和藻类的叶绿体通常被认为来源于蓝细菌内共生事件。
