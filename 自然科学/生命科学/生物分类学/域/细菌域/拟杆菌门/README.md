# 拟杆菌门

## 范围

拟杆菌门属于细菌域，现行拉丁名常写作 Bacteroidota，常见旧名为 Bacteroidetes。

## 概括

拟杆菌门成员广泛存在于土壤、水体和动物肠道等环境中，许多类群擅长分解复杂有机物。
