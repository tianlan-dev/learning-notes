# 厚壁菌门

## 范围

厚壁菌门属于细菌域，现行拉丁名常写作 Bacillota，常见旧名为 Firmicutes。

## 概括

厚壁菌门包含许多低 G+C 革兰氏阳性细菌，其中一些类群能形成芽孢，也包括大量与土壤、发酵、肠道微生物群和病原相关的细菌。
