# 变形菌门

## 范围

变形菌门属于细菌域，现行拉丁名常写作 Pseudomonadota，常见旧名为 Proteobacteria。

## 概括

变形菌门包含大量革兰氏阴性细菌，生态位非常广，包括自由生活、共生、病原、光合和化能营养等多种类型。
