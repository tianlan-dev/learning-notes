# 植物界

## 范围

植物界属于真核生物域，是以光合作用为重要特征的真核生物类群。常见植物包括苔藓类、蕨类、裸子植物和被子植物等。

## 概括

植物通常具有叶绿体，能通过光合作用把光能转化为有机物中的化学能。植物界的边界在不同分类体系中可能有差异，尤其是藻类是否纳入植物界的问题。
