# 腕足动物门

## 范围

腕足动物门属于动物界，常见代表为腕足类。

## 概括

腕足动物通常具有两片壳，外形容易与双壳类软体动物混淆，但二者身体结构和亲缘关系不同。腕足动物在古生代海洋中曾非常繁盛。

## 说明

- 外壳通常为背腹两瓣，而双壳类软体动物多为左右两瓣。
- 多数营海生固着生活。
- 化石记录丰富，是古生物学中常见类群。
