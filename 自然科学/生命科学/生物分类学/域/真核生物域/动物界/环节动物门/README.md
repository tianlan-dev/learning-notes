# 环节动物门

## 范围

环节动物门属于动物界，常见代表包括蚯蚓、水蛭和多毛类等。

## 概括

环节动物以身体分节为重要特征，具有较明显的体腔和器官系统。它们生活环境多样，包括土壤、淡水和海洋。

## 说明

- 蚯蚓常作为陆生环节动物代表。
- 水蛭部分种类以吸血或捕食为生。
- 多毛类多见于海洋环境。
