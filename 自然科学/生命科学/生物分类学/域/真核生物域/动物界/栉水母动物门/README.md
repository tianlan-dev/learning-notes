# 栉水母动物门

## 范围

栉水母动物门属于动物界，常见代表为栉水母。

## 概括

栉水母动物多为海生，体表具有成列的栉板，用纤毛结构游动。它们常呈透明胶质体，与刺胞动物门的水母外形相似但分类不同。

## 说明

- 运动结构以栉板为典型特征。
- 多数为捕食性海洋动物。
- 与刺胞动物不同，栉水母没有刺胞动物那种典型刺细胞。
