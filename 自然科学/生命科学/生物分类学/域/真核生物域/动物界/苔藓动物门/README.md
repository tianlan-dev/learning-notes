# 苔藓动物门

## 范围

苔藓动物门属于动物界，常见代表为苔藓虫。

## 概括

苔藓动物多为水生群体动物，个体小，常形成附着在基质上的群体结构。名称中的“苔藓”容易误导，它们不是植物中的苔藓。
