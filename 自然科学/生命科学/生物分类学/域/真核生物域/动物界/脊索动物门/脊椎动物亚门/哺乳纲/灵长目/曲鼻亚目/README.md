# 曲鼻亚目

## 范围

曲鼻亚目是灵长目下的主要亚目之一，通常包括狐猴类、懒猴类和婴猴类等类群。

## 概括

曲鼻亚目相对保留较多早期灵长类特征，嗅觉作用较突出，许多类群具有湿润鼻端。现生代表主要分布在马达加斯加、非洲和亚洲热带地区。
