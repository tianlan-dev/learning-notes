# 皮翼目

## 范围

皮翼目属于哺乳纲、胎盘类。

## 概括

皮翼目包括鼯猴类，又称飞狐猴。它们具有发达皮膜，可在树林间滑翔，但不能像蝙蝠那样主动飞行。

## 说明

- “飞狐猴”不是猴类，皮翼目也不属于灵长目。
- 皮翼目与灵长目、树鼩目常在灵长总目相关讨论中出现。
