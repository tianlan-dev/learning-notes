# 啮齿目

## 范围

啮齿目属于哺乳纲、胎盘类。

## 概括

啮齿目包括鼠、松鼠、豪猪、河狸等，是现生哺乳动物中物种数最多的目。其典型特征是持续生长的门齿和啮咬适应。
