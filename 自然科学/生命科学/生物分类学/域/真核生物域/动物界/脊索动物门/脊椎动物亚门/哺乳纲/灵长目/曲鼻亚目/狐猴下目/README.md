# 狐猴下目

## 范围

狐猴下目属于曲鼻亚目，主要整理马达加斯加灵长类及其近缘类群。

## 概括

狐猴下目常包括狐猴科、鼬狐猴科、大狐猴科、鼠狐猴科和指猴科等。它们多为树栖或半树栖，体型、食性和活动节律差异很大。

## 说明

- 狐猴类的多样化与马达加斯加长期隔离有关。
- 指猴形态特殊，常作为狐猴类中辨识度很高的分支。
