# 海牛目

## 范围

海牛目属于哺乳纲、胎盘类。

## 概括

海牛目包括海牛和儒艮等水生草食性哺乳动物。它们长期适应水生生活，但仍属于哺乳动物。

## 说明

- 海牛目与鲸类不是同一目。
- 海牛目常被放在非洲兽总目相关讨论中。
