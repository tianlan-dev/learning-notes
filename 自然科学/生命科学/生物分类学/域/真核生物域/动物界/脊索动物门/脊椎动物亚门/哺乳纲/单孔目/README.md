# 单孔目

## 范围

单孔目属于哺乳纲，是现生原兽类的代表目。

## 概括

单孔目包括鸭嘴兽和针鼹等卵生哺乳动物。它们具有哺乳动物的乳腺和毛发等特征，但繁殖方式保留卵生特点。

## 说明

- 单孔目不是有袋类，也不是胎盘类。
- “单孔”指泄殖腔开口结构，与其繁殖和排泄系统有关。
