# 真猴下目

## 范围

真猴下目属于简鼻亚目，包括新大陆猴、旧大陆猴、猿类和人类。

## 概括

真猴下目通常分为阔鼻小目和狭鼻小目。阔鼻小目主要对应美洲新大陆猴；狭鼻小目包括旧大陆猴和人猿类，是人类所在的主干分支。
