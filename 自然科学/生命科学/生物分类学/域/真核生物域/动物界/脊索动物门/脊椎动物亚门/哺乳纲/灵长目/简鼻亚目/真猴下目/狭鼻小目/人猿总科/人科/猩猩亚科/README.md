# 猩猩亚科

## 范围

猩猩亚科属于人科，现生代表为猩猩属。

## 概括

猩猩亚科代表亚洲大型猿类分支。下级为猩猩属，包括苏门达腊猩猩、打巴奴里猩猩和婆罗洲猩猩。
