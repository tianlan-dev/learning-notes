# 真盲缺目

## 范围

真盲缺目属于哺乳纲、胎盘类。

## 概括

真盲缺目包括鼩鼱、鼹、刺猬等劳亚食虫类群。它是旧“食虫目”拆分后常用的现代目级入口之一。

## 说明

- 代表类群包括鼩鼱、鼹和刺猬。
- 非洲猬目中的马岛猬、金毛鼹不属于真盲缺目。
