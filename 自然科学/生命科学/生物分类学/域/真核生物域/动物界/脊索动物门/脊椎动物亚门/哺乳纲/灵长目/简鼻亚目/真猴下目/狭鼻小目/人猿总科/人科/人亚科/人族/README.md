# 人族

## 范围

人族属于人亚科，包含人亚族和黑猩猩亚族。

## 概括

人族是人属和黑猩猩属所在的分支。人族分为人亚族和黑猩猩亚族，分别对应人属与黑猩猩属。

- 人族不是只包含人类，也包括与人类关系最近的黑猩猩属分支。
