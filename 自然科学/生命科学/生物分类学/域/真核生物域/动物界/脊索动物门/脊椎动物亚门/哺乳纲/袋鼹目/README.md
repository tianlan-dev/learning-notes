# 袋鼹目

## 范围

袋鼹目属于哺乳纲、有袋类。

## 概括

袋鼹目包括澳洲地下生活的袋鼹类。它们外形和生活方式与胎盘类鼹鼠有相似处，但属于有袋哺乳动物。
