# 偶蹄目

## 范围

偶蹄目属于哺乳纲、胎盘类。

## 概括

偶蹄目包括鹿、牛、羊、猪、河马等传统偶蹄类；现代系统分类通常把鲸类也纳入广义偶蹄目，因此也常见“鲸偶蹄目”这一名称。

## 说明

- 传统“偶蹄目”常不含鲸类；现代系统分类通常把鲸类放入偶蹄类谱系中。
- 本目录使用“偶蹄目”作为目录名，并在正文中说明其广义口径包含鲸类。
- 奇蹄目和偶蹄目都属于有蹄类相关讨论，但承重趾和系统关系不同。
