# 黑猩猩属

## 范围

黑猩猩属属于黑猩猩亚族，是与人属关系最近的现生大型猿类分支。

## 概括

黑猩猩属包括黑猩猩和倭黑猩猩。

## 说明

- 黑猩猩属属于人族，但不属于人亚族。
- 黑猩猩和倭黑猩猩是两个现生近缘物种。
