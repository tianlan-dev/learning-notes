# 狭鼻小目

## 范围

狭鼻小目属于真猴下目，包括旧大陆猴、猿类和人类。

## 概括

狭鼻小目通常分为猴总科和人猿总科。猴总科主要是旧大陆猴；人猿总科包括长臂猿科和人科，人科下再包含猩猩、大猩猩、黑猩猩、人类及其近缘类群。
