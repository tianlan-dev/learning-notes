# 肉鳍鱼纲

## 范围

肉鳍鱼纲属于脊椎动物亚门，是硬骨鱼类中的一支。

## 概括

肉鳍鱼纲包括肺鱼和腔棘鱼等现生类群。其偶鳍具有肉质叶状结构，在脊椎动物演化中与四足动物肢体起源关系密切。

## 说明

- 现生代表包括肺鱼和腔棘鱼。
- 肉鳍鱼纲与四足动物起源关系密切；若按严格系统发育，四足动物可视为肉鳍鱼谱系中的后裔。
- 作为学习目录，本笔记暂把肉鳍鱼纲作为鱼类形态入口，不把两栖纲、爬行纲、鸟纲、哺乳纲嵌入其下。
