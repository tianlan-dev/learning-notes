# 大猩猩属

## 范围

大猩猩属属于大猩猩族，是非洲大型猿类分支。

## 概括

大猩猩属包括西部大猩猩和东部大猩猩，并进一步列出西部低地大猩猩、克罗斯河大猩猩、东部低地大猩猩、山地大猩猩。
