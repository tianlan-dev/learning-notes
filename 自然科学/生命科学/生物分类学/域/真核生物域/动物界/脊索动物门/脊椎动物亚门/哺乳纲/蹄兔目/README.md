# 蹄兔目

## 范围

蹄兔目属于哺乳纲、胎盘类。

## 概括

蹄兔目包括蹄兔类。它们外形像小型啮齿动物，但系统关系上与象类、海牛类更接近，常放在非洲兽总目中讨论。

## 分类关系

```mermaid
graph TD
    Mammalia["哺乳纲"] --> Eutheria["胎盘类"]
    Eutheria --> Hyracoidea["蹄兔目"]
```

## 说明

- 代表类群为蹄兔。
- 不要因外形把蹄兔归入啮齿目或兔形目。
