# 鳞甲目

## 范围

鳞甲目属于哺乳纲、胎盘类。

## 概括

鳞甲目包括穿山甲。穿山甲体表具有角质鳞片，主要以蚂蚁、白蚁等昆虫为食。

## 说明

- 穿山甲不是爬行动物，而是哺乳动物。
- 鳞甲目与食肉目关系较近，常在猛兽总目相关讨论中出现。
