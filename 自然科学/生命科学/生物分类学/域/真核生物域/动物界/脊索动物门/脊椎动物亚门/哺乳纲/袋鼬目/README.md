# 袋鼬目

## 范围

袋鼬目属于哺乳纲、有袋类。

## 概括

袋鼬目包括袋鼬、袋獾等澳洲肉食性有袋类。它们在澳洲生态系统中承担类似小型到中型食肉动物的角色。
