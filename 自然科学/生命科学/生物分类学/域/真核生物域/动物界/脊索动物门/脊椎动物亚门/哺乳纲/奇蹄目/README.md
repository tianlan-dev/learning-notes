# 奇蹄目

## 范围

奇蹄目属于哺乳纲、胎盘类。

## 概括

奇蹄目包括马、犀和貘等。它们通常以中轴趾承重为主，与偶蹄目的偶数趾承重模式相对。

## 说明

- 代表类群包括马、犀、貘。
- 奇蹄目和偶蹄目都属于有蹄类相关讨论，但不是同一目。
