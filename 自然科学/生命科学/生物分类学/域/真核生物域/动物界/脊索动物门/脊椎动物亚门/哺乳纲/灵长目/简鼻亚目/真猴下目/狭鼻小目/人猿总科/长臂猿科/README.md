# 长臂猿科

## 范围

长臂猿科属于人猿总科，是与人科并列的小型猿类分支。

## 概括

长臂猿科包括长臂猿属、白眉长臂猿属、黑冠长臂猿属、合趾猿属等。本页整理到属级类群，暂不继续展开物种。

## 说明

- 长臂猿类体型较小，臂行适应明显。
- 它们属于猿类，但不属于人科。
