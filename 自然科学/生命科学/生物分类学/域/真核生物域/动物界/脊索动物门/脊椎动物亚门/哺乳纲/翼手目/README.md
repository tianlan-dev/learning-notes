# 翼手目

## 范围

翼手目属于哺乳纲、胎盘类。

## 概括

翼手目包括蝙蝠，是唯一真正具备主动飞行能力的哺乳动物类群。其前肢特化为翼，生态位非常多样。

## 分类关系

## 说明

- 蝙蝠不是鸟类，而是哺乳动物。
- 滑翔的皮翼目动物不能主动飞行，和翼手目不同。
