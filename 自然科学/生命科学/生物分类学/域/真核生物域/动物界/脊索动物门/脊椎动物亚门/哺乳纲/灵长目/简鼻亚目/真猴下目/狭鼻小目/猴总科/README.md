# 猴总科

## 范围

猴总科属于狭鼻小目，主要包括旧大陆猴类。

## 概括

猴总科现生代表通常归入猴科，包含猕猴、狒狒、长尾猴、疣猴、叶猴等类群。它们与人猿总科并列，区别于猿类和人类。

## 说明

- 猴总科通常有尾，而人猿总科现生类群一般无尾。
- 猴科内部常分为猴亚科和疣猴亚科，食性和消化结构差异明显。
