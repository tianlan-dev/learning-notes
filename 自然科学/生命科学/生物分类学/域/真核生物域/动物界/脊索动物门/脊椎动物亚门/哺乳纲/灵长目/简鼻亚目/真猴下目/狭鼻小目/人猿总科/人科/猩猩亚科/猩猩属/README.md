# 猩猩属

## 范围

猩猩属属于猩猩亚科，是亚洲大型猿类的现生属。

## 概括

猩猩属包括三个现生代表：苏门达腊猩猩、打巴奴里猩猩、婆罗洲猩猩。

## 说明

- 猩猩属属于人科，但不属于人亚科。
- 三个现生种主要分布于苏门达腊和婆罗洲。
