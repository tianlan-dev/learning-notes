# 七鳃鳗纲

## 范围

七鳃鳗纲属于脊椎动物亚门，是现生无颌类脊椎动物的一支。

## 概括

七鳃鳗身体细长，缺少真正颌，口部常呈吸盘状。部分种类营寄生生活，也有非寄生类型。

## 说明

- 七鳃鳗与盲鳗常被合称为圆口类。
- 七鳃鳗是理解无颌脊椎动物和有颌脊椎动物差异的重要类群。
- 与盲鳗相比，七鳃鳗在一些结构上更接近传统意义的脊椎动物。
