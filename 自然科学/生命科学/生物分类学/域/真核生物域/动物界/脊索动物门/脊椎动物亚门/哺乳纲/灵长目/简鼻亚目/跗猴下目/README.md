# 跗猴下目

## 范围

跗猴下目属于简鼻亚目，现生代表主要为跗猴科。

## 概括

跗猴类体型小，眼大，后肢和跗骨发达，适合夜间捕食和跳跃。它们与真猴类同属简鼻亚目，但不是猴、猿或人类的直接成员。

## 说明

- 跗猴类常被用于说明简鼻亚目内部早期分支和真猴类的差异。
- 现生跗猴主要分布在东南亚岛屿和邻近地区。
