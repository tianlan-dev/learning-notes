# 兔形目

## 范围

兔形目属于哺乳纲、胎盘类。

## 概括

兔形目包括兔、野兔和鼠兔等。它们与啮齿目相近但不是同一目，典型区别之一是上颌门齿结构。

## 说明

- 代表类群包括兔和鼠兔。
