# 微兽目

## 范围

微兽目属于哺乳纲、有袋类。

## 概括

微兽目是现生有袋类中的小型类群，代表为山猴负鼠。虽然现生种类很少，但在理解南美有袋类和澳洲有袋类关系时有重要意义。

## 说明

- 代表类群为山猴负鼠。
- 微兽目名称中的“微兽”是分类名，不表示所有个体都极小。
