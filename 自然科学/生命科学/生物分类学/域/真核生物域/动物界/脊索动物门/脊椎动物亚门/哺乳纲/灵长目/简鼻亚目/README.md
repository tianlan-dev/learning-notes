# 简鼻亚目

## 范围

简鼻亚目是灵长目下的主要亚目之一，包括跗猴类和真猴类，后者包含猴类、猿类和人类。

## 概括

简鼻亚目的视觉系统和脑部发育较突出，鼻端不像曲鼻亚目那样保留湿润鼻镜。真猴下目是简鼻亚目中物种和形态多样性很高的分支。
