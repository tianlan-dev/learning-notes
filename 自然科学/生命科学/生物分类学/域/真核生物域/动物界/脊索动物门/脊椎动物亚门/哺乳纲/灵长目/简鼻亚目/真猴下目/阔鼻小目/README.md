# 阔鼻小目

## 范围

阔鼻小目属于真猴下目，主要对应新大陆猴类。

## 概括

阔鼻小目主要分布在中南美洲，鼻孔较宽且多向侧方开口，尾部形态和生活方式多样。部分类群具有可缠绕尾，适合树栖生活。
