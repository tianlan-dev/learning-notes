# 象鼩目

## 范围

象鼩目属于哺乳纲、胎盘类。

## 概括

象鼩目包括象鼩类，是非洲分布的小型哺乳动物。它们名称中有“鼩”，但不属于真盲缺目中的鼩鼱类。
