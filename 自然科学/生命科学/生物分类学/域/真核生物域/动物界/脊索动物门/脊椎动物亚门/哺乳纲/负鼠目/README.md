# 负鼠目

## 范围

负鼠目属于哺乳纲、有袋类。

## 概括

负鼠目是美洲有袋类的重要代表，常见代表为负鼠。它们与澳洲有袋类同属有袋哺乳动物，但分布和演化支系不同。

## 说明

- 负鼠目主要分布于美洲。
- 不要把“负鼠”与澳洲常说的袋貂类或其他有袋类混为同一分类单元。
