# 软骨鱼纲

## 范围

软骨鱼纲属于脊椎动物亚门，是有颌鱼类中的一支。

## 概括

软骨鱼纲包括鲨、鳐、魟和银鲛等。它们的骨骼主要由软骨构成，而不是像多数硬骨鱼那样高度骨化。

## 说明

- 鲨类、鳐类和魟类是常见代表。
- 软骨鱼属于有颌脊椎动物，与无颌的盲鳗、七鳃鳗不同。
- “软骨鱼”不是指身体柔软，而是指骨骼系统以软骨为主。
