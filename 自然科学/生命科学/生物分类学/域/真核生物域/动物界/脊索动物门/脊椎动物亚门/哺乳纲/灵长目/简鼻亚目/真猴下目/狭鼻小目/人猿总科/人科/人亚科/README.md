# 人亚科

## 范围

人亚科属于人科，包含人族和大猩猩族。

## 概括

人亚科是人类、黑猩猩和大猩猩所在的分支。人亚科分为人族和大猩猩族；人族下再列人亚族和黑猩猩亚族。
