# 袋狸目

## 范围

袋狸目属于哺乳纲、有袋类。

## 概括

袋狸目包括袋狸和兔耳袋狸等澳洲有袋类。它们多为杂食或食虫性，体形和生态位与部分小型胎盘类动物相似。
