# 浴古菌门

## 范围

浴古菌门常用拉丁名为 Bathyarchaeota，是主要通过沉积物、缺氧环境和宏基因组研究认识的古菌候选门级类群。

## 概括

浴古菌门成员常见于海洋沉积物、淡水沉积物和其他缺氧环境，可能参与碳循环、氮循环或有机质降解等过程。它体现了现代古菌分类中环境测序的重要性。
