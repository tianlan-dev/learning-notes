# 纳古菌门

## 范围

纳古菌门常用拉丁名为 Nanoarchaeota，部分命名体系中可见 Nanobdellota 等名称。

## 概括

纳古菌门通常指体型和基因组都较小的一类古菌，许多成员与其他古菌存在共生、依附或寄生式关系。它常被放在古菌小型基因组和共生演化的语境中理解。
