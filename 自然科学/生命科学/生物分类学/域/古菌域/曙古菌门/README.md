# 曙古菌门

## 范围

曙古菌门常用拉丁名为 Aigarchaeota，是主要由环境基因组资料支持的古菌候选门级类群。

## 概括

曙古菌门常与 TACK 相关古菌谱系一起讨论。它反映了现代古菌分类中大量类群来自环境测序而非传统培养观察的特点。
