# 生物分类学

生物分类学研究生物的命名、归类和系统关系。传统分类使用阶元体系来组织生物多样性，现代分类更强调系统发育和共同祖先，因此同一类群在不同资料中可能出现层级调整。

## 分类阶元

```mermaid
graph TD
    Life["生物"] --> Domain["域"]
    Domain --> Kingdom["界"]

    Phylum --> Class["纲"]
    Class --> Order["目"]
    Order --> Family["科"]
    Family --> Genus["属"]
    Genus --> Species["种"]
```

## 关键易混点

- “原核生物”是细胞结构层面的概括，不适合作为现代系统分类中的单一自然类群。
- 细菌和古菌都没有细胞核，但二者在遗传机制、生化性质和演化关系上差异很大。
- “界、门、纲、目、科、属、种”是整理层级，不一定精确反映所有演化分支的时间先后或亲缘距离。
